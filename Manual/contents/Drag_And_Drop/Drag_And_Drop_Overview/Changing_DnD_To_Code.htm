当你添加、改变或删除GML 视觉动作时，GML 实时预览将更新，向你展示正在 "幕后 "创建的实际代码。实时预览中的代码不能直接编辑，但你可以选择部分并复制它们以粘贴到GML scripts 或代码动作（例如）。

如果你在看到预览后决定要将GML Visual转换为代码，那么可以再次通过点击鼠标右键 ，在任何事件workspace ，并选择转换为GML 代码来完成。

当你第一次这样做的时候，你会得到一个警告信息，说这是一个单向的转换，因为你可以把动作转换为代码，但你以后不能再把它们转换为单个 动作。点击这里的 "好 "将为你执行转换。
