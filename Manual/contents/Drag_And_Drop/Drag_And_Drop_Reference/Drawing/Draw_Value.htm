提取值

通过这个动作，你可以在屏幕上画出一个string 的标题和/或一个变量或值。标题参数是一个string ，你想把它加在值的前面，如果需要，可以留空，但如果你想包括文本，应该用引号括起来。 值可以是一个变量、一个固定值或另一个string ，在这种情况下也应该用引号括起来，但它不能留空。然后你要给出string 的位置，可以是一个绝对位置 或相对于绘制文本的实例的位置。请注意，文本在绘制时的样子将取决于所使用的font 、所使用的颜色和对齐方式。所有这些都可以在绘制前改变，如下面的例子所示。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，则不会抽到任何东西。
