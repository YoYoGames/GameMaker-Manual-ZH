绘画动作库

绘图动作库是你可以找到绘制sprites 、文本或形状以及设置某些绘图属性所需的动作的地方。 形状以及设置某些绘图属性所需的动作。这些动作大多只在 object 的各种绘图事件中使用，如果在绘图事件之外使用，可能无法工作。 绘制事件之外使用可能无法工作。这方面的例外是设置动作，它可以被添加到任何事件中，并且会影响之后所有实例的所有绘制。

需要注意的是，如果你在一个object 的主绘图事件中添加任何动作，那么它将不会绘制已经分配给该实例的sprite ，除非你明确告诉GameMaker绘制它，使用一个动作 绘制自己。基本上，GameMaker将默认绘制分配给实例的任何sprite ，只有在Draw Event中没有 其他东西的情况下。
