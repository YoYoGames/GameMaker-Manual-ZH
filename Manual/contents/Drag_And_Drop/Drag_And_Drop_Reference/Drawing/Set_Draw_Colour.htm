设置绘图颜色

通过这个动作，你可以设置任何形状（矩形/椭圆/直线）或随后绘制的任何文本所使用的颜色。请注意，这个动作不一定要在每个绘制动作之前调用，只能在游戏开始时设置一次。 例如，如果你的所有文本都是黄色的，你可以在游戏的第一个room 的创建事件中调用这个动作，然后所有文本（和形状）将从那时起被画成黄色。但是如果你需要使用多种不同的颜色。 那么你就需要在每个需要不同颜色的文本或形状之前调用这个动作。

这个动作在底部有一个使用阿尔法的复选框。这可以使颜色的阿尔法分量也被用来设置绘图阿尔法（与使用设置绘图阿尔法的作用相同。如果你不希望包括 如果你不想包括阿尔法分量而只是有一个纯色，这可以被禁用（默认是启用的）。
