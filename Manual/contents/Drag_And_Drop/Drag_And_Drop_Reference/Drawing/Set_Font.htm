设置字体

通过这个动作，你可以设置font ，以用于所有后续的绘制文本动作。点击对话将打开以下窗口，然后你可以选择你希望使用的font 。

请注意，这将设置font ，用于所有后续的绘制文本调用，因此可以在游戏开始时调用一次--在任何情况下，不仅仅是绘制事件--所有的绘制文本动作将使用设置的 。 所以可以在游戏开始时调用一次--在任何情况下，不仅仅是绘制事件--所有绘制文本的动作都将使用设定的font 。然而，如果你在游戏中需要多个fonts ，那么你将需要在绘制任何带有所需 的文本之前调用这个动作。 任何带有所需文字的font 。
