绘制叠加精灵

这个动作将在sprite ，在sprites ，一个接一个的给定数量的room 。你给出要画的sprite ，以及堆叠顺序，可以是行（水平方向，从左到右），也可以是列（垂直方向，从上到下）。 从上到下），以及要绘制的sprites 的数量和位置。位置可以是room 中的绝对位置，也可以是相对于进行绘制的实例的位置，而图像之间的间隔将基于 sprite 的宽度或高度。注意，这只是绘制一个静态图像--给定的sprite 的初始单幅图像（第 0 帧）--而任何其他的帧都将被忽略，任何通过改变实例变量（如 image_xscale 或 image_blend ）而添加的变换也将被忽略。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，则不会抽到任何东西。
