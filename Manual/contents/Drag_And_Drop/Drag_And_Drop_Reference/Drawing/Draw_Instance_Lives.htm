绘制实例寿命

这个动作将绘制一个sprite ，代表实例当前拥有的每一条生命。要绘制的数字来自于本地实例变量 lives 的当前值，这是一个特殊的变量，如果你使用任何需要它的动作，它就会在实例中自动创建。 如果你使用任何需要它的操作，这个特殊的变量会在实例中自动创建。你给sprite ，为生命和堆栈顺序，可以是行（水平方向，从左到右），或列（垂直方向，从上到下），以及位置。 位置可以是room 中的绝对位置，也可以是相对于进行绘制的实例的位置，图像之间的间隔将基于sprite 的宽度或高度。注意，这只是绘制一个静态图像 - 最初的单一图像（第0帧）的给定sprite ，任何进一步的帧都将被忽略，任何通过改变实例变量（如 image_xscale 或 image_blend ）添加到实例的变换也将被忽略。 标题文本将被画在生命值的前面，如果它是文本，应该用""来包装，两者都将使用当前的绘制font 、颜色和对齐方式进行格式化。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，则不会抽到任何东西。
