如果音频被暂停

这个动作可以用来检查任何声音资源的实例目前是否在你的游戏中暂停了。你从asset 探索器中给出要检查的声音资源，这个函数将返回 true if it is paused and false 如果它不是，尽管如果你检查 "不是 "flag ，这将是相反的，函数将返回 true ，如果声音没有暂停， false 。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true ，这些操作将被运行，而 在其他地方放弃的任何行动将在 "if "块之后执行。
