获取音频音量

这个动作返回给定声音的音量。你从asset explorer中挑选声音，并给一个目标变量来返回音量值（你可以flag ，作为一个临时的局部变量）。返回的值将在0和 1之间，其中0为无声，1为全音量。

例子。

上面的动作块代码检查一个全局变量，看它是 true 还是假的。如果是 true ，那么给定声音的音量将被检索并存储在一个临时的局部变量中。然后检查该值是否不等于0.5，如果不等于0.5，则声音的音量被设置为0.5。如果 全局变量的值为 false ，那么这个声音的音量就被设置为1。
