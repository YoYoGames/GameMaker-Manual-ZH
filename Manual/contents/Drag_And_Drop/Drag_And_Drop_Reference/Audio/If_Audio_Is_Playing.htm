如果正在播放音频

这个动作可以用来检查任何声音资源的实例是否在你的游戏中正在播放。你从asset 探索器中给出要检查的声音资源，该函数将返回 true if it is playing and false 如果不是，尽管如果你检查 "不是 "flag ，这将是相反的，如果声音没有在播放，函数将返回 true ，否则返回 false 。
