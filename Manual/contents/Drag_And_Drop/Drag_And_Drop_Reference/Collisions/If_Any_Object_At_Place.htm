这个动作用于检查并查看是否有任何 object 的实例放置在基于调用实例的碰撞掩码的位置上。你给出位置，可以是一个绝对位置，也可以是一个相对于实例的位置。 如果有碰撞，该动作将返回 true ，如果没有，则返回 false 。你也可以检查 "Not "flag ，以检查在给定的位置是否有碰撞，如果没有发现碰撞，动作将返回 true ，否则返回 false 。碰撞的计算是基于调用实例的碰撞掩码与该位置的任何实例的碰撞掩码相重叠（就像它被 "放置 "在该位置一样）。

重要的是! sprite_index sprite 碰撞将只登记那些具有有效碰撞掩码的实例，即：他们有一个分配给sprite ，或一个分配给 mask_index 。 碰撞中的任何一个实例没有碰撞掩码，那么这个碰撞将不会被检测到，不管这个实例当时正在画什么。
