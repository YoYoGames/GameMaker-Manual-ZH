复制文件

通过这个动作，你可以复制任何由你的游戏保存的文件，或者作为包含文件的一部分被捆绑的文件。你给出要复制的文件的名称（作为string ），然后给出要创建的文件的名称，作为原始文件的副本（也是 作为一个string ，并包括文件extension ）。给定的文件将被复制并赋予新的名称。

争论。

争论 描述
Filename 要复制的文件的名称（以string ）。
Copy To 将被创建为副本的文件的名称（以string ）。
