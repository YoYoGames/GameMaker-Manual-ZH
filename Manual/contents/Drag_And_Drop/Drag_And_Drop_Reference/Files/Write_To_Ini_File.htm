通过这个动作，你可以向一个Ini文件写一个数值或string 。你必须事先用打开Ini文件的动作打开Ini文件进行读/写，然后用这个动作给出要写的数据类型（可以是实数，也可以是用""括起来的。 数据的类型（实数或用""括起来的string ），以及部分标题、该部分的键和键的值。为了帮助直观地了解Ini文件的结构，请看这个图片。

正如你所看到的，Ini文件被分割成几个部分--每个部分都有自己的头文件名--然后每个部分又被分割成各种键/值对（很像地图数据结构）。 然后每个部分随后被分割成各种键/值对（很像一个地图数据结构）。请注意，你可以使用任何文本处理软件（如记事本上的）自己创建Ini文件。 Windows)，然后把它们作为一个包含文件，在你的游戏中读取和写入。

重要的是!试图从一个以前没有使用 "打开Ini文件"动作打开的Ini文件中读取数据将导致一个严重的错误并使你的游戏崩溃。

重要的是!这只是将数据写入内存，要最终将数据写入磁盘上的文件，你必须调用关闭Ini文件，否则数据将无法保存。
