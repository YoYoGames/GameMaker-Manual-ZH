重命名文件

通过这个动作，你可以重新命名任何由你的游戏保存的文件。你给出要改变的文件名（以string ），然后给出新的名字（也以string ，包括文件extension ），该文件就会被重新命名。注意，你不能重命名任何与你的游戏捆绑在一起的文件（更多信息见文件系统部分）。

行动语法。

争论。

争论 描述
Filename 要重命名的文件的名称（以string ）。
New Name 文件的新名称（作为string ）。
