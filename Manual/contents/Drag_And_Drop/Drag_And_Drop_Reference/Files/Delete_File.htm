删除文件

通过这个动作，你可以删除一个以前创建的文件。你提供要删除的文件的文件名（以string ）。注意，你不能删除任何包含在游戏包中的文件，只能删除那些已经创建的文件。 使用Close Ini或Save Buffer动作创建的文件（更多信息见文件系统一节）。
