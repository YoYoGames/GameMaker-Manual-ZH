如果文件存在

通过这个动作，你可以检查一个给定名称的文件是否存在。你选择要检查的文件（作为一个刺，包括extension ），如果它存在，该函数将返回 true ，如果不存在则返回 false 。注意 如果该文件作为一个包含的文件存在于游戏包中，或者如果该文件在游戏运行时被保存出来，该函数将返回 true （更多信息请参见文件系统部分）。
