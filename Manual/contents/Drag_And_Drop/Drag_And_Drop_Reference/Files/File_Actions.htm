文件操作处理两种不同的文件类型--Buffer 文件和Ini文件--也有某些通用的文件操作来重命名或复制现有文件。

缓冲文件是通过保存你以前创建的buffer 中的数据来创建的，然后这些数据可以在将来的任何时候再次加载到buffer 。你可以在这里找到关于buffers 的更多信息。缓冲行动。

Ini文件是小而轻的文件，与大多数平台兼容。它们是存储小块信息的理想选择，如界面偏好、本地高分、关卡数据等......而且非常容易使用。Ini文件不需要 你可以从一个不存在的Ini文件中读取，你会得到一个 default （你指定的）返回值，但是我们建议你至少创建一个 "基础" ini文件，以便在使用这些动作之前打开和修改。这个基础ini文件可以通过简单地调用Open Ini File动作和Close Ini File动作来创建，因为关闭文件会将其写入磁盘。 文件将把它写入磁盘，或者你可以在Asset 浏览器的包含文件中包含一个文件。如果你使用Asset 浏览器中包含的文件作为你的基本Ini，你还应该阅读手册中关于如何使用Ini的部分。 本手册中关于文件系统如何工作的章节。

注意：对于带有本地化的游戏来说，可以从外部文件读取重音字母是非常重要的。这意味着你应该先创建UTF8格式的ini文件，然后把它作为一个内含文件添加到GameMaker中。 这样它就会在运行游戏时被导出，并代替默认的ANSI格式ini文件，该文件是由ini函数在之前没有提供时创建的。通过这种方式，你可以正确地读和写它，所有的重音和非罗马字母都被保留。 字母被保留下来。
