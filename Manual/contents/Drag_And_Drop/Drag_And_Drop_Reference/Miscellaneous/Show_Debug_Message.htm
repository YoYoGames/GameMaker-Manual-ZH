显示调试信息

通过这个动作，你可以向编译器输出发送一条调试信息。这个信息可以是一个string ，也可以是一个实数，但如果你想同时使用这两个数字，那么必须将数字转换为字符串（见动作Number To String）。请注意，你可以使用这个动作输出多个信息，点击 图标来扩展这个动作的更多输出域。
