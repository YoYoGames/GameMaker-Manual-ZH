如果游戏手柄按钮被释放

这个动作检查给定的游戏手柄索引的游戏手柄按钮何时被释放。它将在按钮被释放的单一游戏时间内返回 true ，在所有其他时间返回 false 。如果你启用 not 修改器。 这个动作将被反转，并检查按钮是否被释放，在按钮被释放的游戏时间内返回 false ，在其他时间返回 true 。

如果被检查的按钮是一个模拟按钮，那么只有在按钮值低于使用操作Set Gamepad Button Threshold设置的阈值时，才能检测到释放。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "的评估值为 true 的情况下，这些动作将被执行，而在其他地方放弃的任何动作将在 "if "块之后被执行。
