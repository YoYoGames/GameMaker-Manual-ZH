如果按了游戏手柄按钮

通过这个动作，你可以轮询游戏板，看是否有任何按钮目前正在被按下。该动作将在游戏板按钮被按下（并按住）的每一步返回 true ，否则返回 false ，不过如果你点击 not 修改器，你就可以检查按钮是否被按下，即：如果没有按钮被按下，动作将返回 true ，如果有按钮，则返回 false 。如果你只需要检查是否有单个按钮被按下，那么 使用动作if gamepad button down。

如果被检查的按钮是一个模拟按钮，那么只有在按钮值超过了用操作Set Gamepad Button Threshold设置的阈值时，才会检测到按下。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
