设置游戏手柄按钮阈值

这个动作可以用来设置游戏板模拟按钮的 "阈值"。你指定要设置的游戏板索引，然后设置一个从0到1的值，如果模拟按钮的输入量低于给定的值，游戏板按钮就会被认为是0。 注意，这是一个全局 设置，将影响所有连接到游戏板的模拟按钮。这个值将用于给定游戏板的所有向下、按下和释放的检查，但会被Get Gamepad Trigger动作所忽略。
