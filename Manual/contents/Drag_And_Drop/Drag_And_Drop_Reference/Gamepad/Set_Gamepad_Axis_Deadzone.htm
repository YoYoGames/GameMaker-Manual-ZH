设置游戏手柄轴的死区

这个动作可以用来设置游戏手柄杆轴的 "死区"。你指定要设置的游戏手柄索引，然后设置一个从0到1的值，如果输入量低于给定的值，游戏手柄杆轴就被认为是在0。 注意，这是一个全局设置，将影响所有连接的摇杆轴。

请注意，这并不修改Get Gamepad Axis动作的输出，它将总是返回-1和0，以及0和1之间的数值。 只有当原始轴值超过0.2时才开始返回0-1的值--所以当原始值是0.2时，返回值是0，当原始值是0.5时，返回值是0.375，或者如果原始值是0.9，那么返回值是0.875。 返回值将是0.875。
