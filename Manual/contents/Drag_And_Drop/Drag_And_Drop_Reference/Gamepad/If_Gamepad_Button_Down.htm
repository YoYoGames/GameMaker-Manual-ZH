如果游戏手柄按钮向下

通过这个动作，你可以对游戏板进行调查，看某个按钮是否被按住了。这个动作会在按钮被按下的每一步返回 true ，否则返回 false ，不过如果你点击 not 修改器 你就可以检查按钮是否被按住了，即：当没有按钮被按住时，动作将返回 true ，如果有的话，则返回 false 。如果你只需要检查一个按钮是否被按下，那么就使用If Gamepad Button Pressed这个动作。

如果被检查的按钮是一个模拟按钮，那么直到按钮的值超过了用动作Set Gamepad Button Threshold设置的阈值，它才会被认为被按住。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true ，这些动作将被运行。 而在其他地方放弃的任何行动将在 "if "块之后执行。
