获取视图变量

这个动作允许你获取与给定视图端口相关的某些内置变量的值。你选择要检索的视图端口变量，然后给出目标视图端口（从0到7），最后给出目标变量来保存返回的 值（这可以被标记为一个临时的局部变量）。视图端口是屏幕上用于绘制给定摄像机视图的区域（如在room 编辑器中设置的那样），你可以通过这种方式获得不同的变量 在下面的表格中概述。

变化的 描述
Camera 返回的值将是分配给该端口的摄像机的唯一ID值（更多信息见这里）。
Visibility 这将返回一个 true 或 false 的值，其中 true 是指视图端口是可见的（即：它被绘制到屏幕上），而 false 意味着没有任何东西被绘制。
View Port X Coordinate 这是游戏窗口中视口的X位置。
View Port Y Coordinate 这是在游戏窗口中视口的Y位置。
View Port Width 这是视图端口的宽度（单位：像素）。
View Port Height 这是视口的高度（单位：像素）。
View Port Surface ID 这将返回分配给视图端口的表面的唯一ID值，如果没有分配表面，则返回-1。
