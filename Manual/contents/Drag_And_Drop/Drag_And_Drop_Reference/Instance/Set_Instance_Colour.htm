设置实例颜色

这个动作块设置 image_blend 的颜色，以便与实例sprite"混合"。默认值是十六进制代码$FFFFFFFF（使用ARGB格式表示白色），这将使用无混合和全alpha绘制sprite 。任何其他的值（包括内部颜色常量，如 c_red ，或 c_aqua ）将把指定的颜色与原始的sprite ，以及设置 image_alpha 分量，但只有当使用阿尔法复选框被选中时。这将覆盖之前用Set Instance Alpha动作设置的任何alpha值。

请注意，为了使这个动作的变化可见，这个实例应该没有绘制事件（因此GameMaker会默认绘制sprite ），或者使用绘制自我动作来绘制。这一点很重要 还需要注意的是，你应该尽量限制HTML5平台上的混合（除非使用WebGL），因为每个混合的sprite ，必须单独进行缓存，所以有许多混合的sprites ，可能会对性能产生不利影响。
