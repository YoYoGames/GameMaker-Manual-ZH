设置实例Alpha

这个动作块设置实例的 image_alpha 值。图像alpha是绘制任何分配给实例的sprite 的 "透明度 "值，其值在0和1之间，其中0是完全透明的，1是 是完全不透明的。请注意，如果sprite 有透明或半透明的像素，那么它们的透明度将按比例放大，所以如果你在sprite 中有一些50%的透明像素，并将实例的alpha设置为0.5，那么最终的alpha 将会是25%的透明像素。你可以把这个值设置为一个相对量

请注意，要使这个变量的变化可见，实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用Draw Self 动作绘制。
