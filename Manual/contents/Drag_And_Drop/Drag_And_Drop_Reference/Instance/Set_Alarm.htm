设置报警倒计时

这个动作块设置一个给定的报警倒计时，当报警达到零时，相关的报警事件将被触发。你可以提供从0到11的报警号码，以及 然后是设置报警的值。所用的值是你希望在触发报警事件之前通过的 "游戏步骤 "的数量，你可以将其设置为相对于当前的报警值（在当前值的基础上加或减）。 值）。注意，这个值必须是一个整数，你可以把它设置为-1来停止报警。注意，当一个报警器倒计时时，当它达到0时，事件就会被运行，但报警器随后会倒计时到-1，以显示它已经完成。将一个 警报设置为0不会运行事件，因为它将在下一步倒数到-1，所以要立即运行警报，应该设置为1。注意，你可以通过点击动作旁边的加号图标 ，在同一个动作中设置额外的警报。
