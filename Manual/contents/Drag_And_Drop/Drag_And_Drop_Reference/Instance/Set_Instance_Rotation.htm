设置实例旋转

这个动作块设置实例的 image_angle （旋转）。角度以度为单位，右边是0°，向上是90°，左边是180°，向下是270°。将这个变量设置为0来重置实例，意味着它将 绘制任何sprite ，因为它是在sprite 编辑器中定义的。请注意，为了使这个动作的变化可见，这个实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用绘制自我动作来绘制 使用Draw Self动作。
