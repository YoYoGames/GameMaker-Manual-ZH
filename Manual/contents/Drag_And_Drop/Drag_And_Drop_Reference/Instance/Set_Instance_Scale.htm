设置实例规模

此操作块为实例设置 image_xscale 和 image_yscale 值。这里的输入值是修改器，将应用于分配给实例的sprite ，其中1的比例（默认值）表示没有 缩放（1:1），较小的值将会缩小（例如，0.5，将是sprite 的一半宽度），较大的值将会放大。你可以使用负值来翻转sprite ，并缩放它，除非使用的值正好是-1（在这种情况下 sprite 只是围绕原点翻转或镜像，没有缩放）。你可以选择设置相对于当前值的水平或垂直比例，在这种情况下，你将从当前比例值中添加或减去新的值。

注意，改变这些值也会改变实例检测碰撞的方式，除非你在对象编辑器中提供了一个单独的 mask_index （碰撞屏蔽）。请注意 为了使这个变量的变化可见，这个实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用Draw Self动作来绘制。
