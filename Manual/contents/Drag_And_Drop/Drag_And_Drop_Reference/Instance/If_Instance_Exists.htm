如果实例存在

这个动作可以用来检查一个特定的object 的实例是否存在于游戏中room ，或者一个特定的实例本身是否存在。你从object 探索器中给出asset ，如果任何实例存在于room ，那么该动作将返回 true ，否则将返回 false 。请注意，你可以在这里使用一个变量，为特定的实例提供唯一的ID（例如，由动作CreateObject Instance返回），甚至是由 编辑器返回的实例ID 值，甚至是在你添加实例时由room 编辑器返回的值。

如果你flag "Not "参数，那么动作将检查是否没有实例存在，如果没有发现，将返回 true ，如果甚至存在一个，将返回 false 。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true ，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
