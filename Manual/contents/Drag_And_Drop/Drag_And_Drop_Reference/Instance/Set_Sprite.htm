设置精灵

这个动作块将实例的 sprite_index 设置为sprite 浏览器中的另一个Asset 。你选择要改变的sprite ，然后给出要显示的图像（动画帧）。注意，如果你不希望sprite ，在设置了框架之后再做动画，你就需要将 。 设置框架后，你需要使用设置动画速度动作，并将其设置为0，或者如果你想让sprite 继续动画，那么设置值使用 image_index 内置变量变量 来代替。

改变sprite ，也会改变实例的碰撞掩码，除非你在对象编辑器中也单独指定了一个 mask_index 。请注意，为了使这个动作中的变化 的变化是可见的，该实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用Draw Self动作绘制。
