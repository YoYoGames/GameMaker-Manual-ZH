重新启动游戏

通过这个动作，你可以重新启动游戏。这类似于第一次运行游戏，因此将触发其他--游戏开始事件，以及其他--游戏结束事件。 然而，应该注意的是，全局变量不会被重新初始化，除非有明确的编码--这意味着，如果你不在游戏的第一个room ，它们将保留以前的值--以及 数据结构和粒子也不会被释放，所以应该在调用重启游戏动作之前 或在游戏结束事件中完成。还要注意的是，游戏不会在这个动作被调用的那一刻重启，而是在放置这个动作的script 或事件结束时重启，所以如果在这个动作之后还有任何动作，它们仍然会被调用，游戏也会在这个块的最后重启。
