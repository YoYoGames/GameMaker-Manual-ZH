加载游戏

这个动作将加载一个先前用 "保存游戏"动作保存的游戏。你给出要加载的文件名（以string ，并包括extension ），游戏将被加载并继续运行。 从保存点继续运行。请注意，这不是一个通用的保存/加载系统，试图加载一个从以前的项目运行中保存的游戏可能会出现错误（特别是如果你使用了像数据结构或粒子这样的东西），因此，你应该只尝试加载在一个游戏过程中保存的游戏（像检查点）。还要注意的是，在当前事件或script ，文件才会被加载，所以在加载动作被调用后的任何动作仍会被执行。关于加载保存的游戏数据的更全面的方法，见文件动作。
