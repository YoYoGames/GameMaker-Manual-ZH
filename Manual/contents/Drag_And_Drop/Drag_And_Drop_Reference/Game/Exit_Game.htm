通过这个动作你可以结束游戏，触发其他--游戏结束事件。请注意，这并不是在调用该动作的那一刻发生的，而是在当前事件或，所以在调用该动作后的任何动作仍将运行，直到达到事件/脚本的终点。 事件或script ，所以在这个动作被调用后，你的任何动作仍然会运行，直到达到事件/脚本的终点。

请注意，这一行动有以下限制。

  • 它将在iOS目标上报告一个错误，因为这违反了他们的市场条件。
  • 在HTML5和Windows UWP目标上调用时，它将没有任何效果。
  • 它可以在Windows（包括为Steam ）、Android、Ubuntu（Linux ）和Mac上运行。
