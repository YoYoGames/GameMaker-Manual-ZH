设置粒子精灵

这个动作可以用来设置粒子在绘图时可以使用的sprite 。你提供要设置sprite 的类型的唯一ID值（正如你用动作Create Particle Type创建该类型时返回的那样） 和要使用的sprite 资源。如果sprite 有子图像，那么粒子将以图像编辑器中为sprite 设置的速度通过这些帧制作动画。还要注意，如果你为粒子设置了一个颜色值（使用Set Particle Colour动作），那么这个颜色将与使用的sprite 混合，就像在实例上设置图像混合将与实例sprite 混合一个颜色（使用白色表示不混合）。

注意：如果你使用这个动作，那么你就不应该使用设置粒子形状这个动作，因为你只能有一个sprite ，或者一个形状，但不能同时拥有。
