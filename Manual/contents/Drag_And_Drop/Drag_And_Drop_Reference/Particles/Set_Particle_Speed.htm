设置粒子速度

这个动作用于设置粒子的移动速度，它可以是静态的，也可以设置为随时间增加/减少。你为要设置大小的类型提供唯一的ID值（就像你用动作Create Particle Type创建类型时返回的那样），然后给出一个初始速度范围，然后是增加量，最后是 "摆动"。粒子创建的速度是以每游戏帧的像素数来计算的，所以最小设置为2，最大设置为5，将发出每游戏帧2到5像素之间的粒子。

增加粒子速度的选项可以让你输入一个值，这个值将被添加到粒子生命周期的每一帧基本速度值上。因此，将这个值设置为，例如0.1，将在每个游戏帧上增加0.1的速度。请注意，你也可以在这里使用负值来使粒子变慢，尽管速度永远不会低于0。请注意，接受的最小速度递减是-0.01，最小增量是0.01。

最后你可以设置 "摇摆 "选项。这是一个将在0和你给的值之间随机产生的值，然后在粒子寿命的每一步上加减速度，导致绘制的图像在速度上波动。很明显，大的数值比小的更明显，这个数值甚至可以是负数，最大范围在-20和20之间。

行动语法。

争论。

争论 描述
Type 粒子类型的唯一ID值，用于设置其大小。
Min Speed 粒子类型的最小速度（默认为0.1）。
Max Speed 粒子类型的最大速度（默认为1）。
Increase 每一游戏帧的粒子类型速度的附加值（可以是负值，最小+/-0.01）。
Wiggle 每步随机加入或减去颗粒速度的最大值（+/-20）。
