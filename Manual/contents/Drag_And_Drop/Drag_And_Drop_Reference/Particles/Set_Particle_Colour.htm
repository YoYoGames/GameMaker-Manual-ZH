设置粒子颜色

这个动作将设置在你的sprite ，并在其生命周期内使用的颜色。你首先指定粒子类型的ID值（它应该与你使用创建粒子类型动作创建粒子时返回的值相同），然后你可以设置要使用的颜色。

你可以选择三种颜色来使用，而sprite ，在粒子的生命周期内，它们之间会相互融合。例如，如果一个粒子有红色、绿色和黄色，并且有60个游戏帧的寿命，那么当它被创建时（第0帧），它将是 红色，然后在其寿命的一半时（第30帧）会渐渐变成绿色，在其寿命结束时（第60帧）会渐渐变成黄色。如果你使用一个自定义的sprite ，而不是一个粒子形状，那么除了白色以外的任何颜色都会与 与所使用的sprite 。

注意：颜色的阿尔法分量不被使用，只使用颜色选择器的RGB分量。要设置阿尔法，请使用设置粒子阿尔法动作。
