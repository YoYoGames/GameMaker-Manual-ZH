新结构

这个动作 只在使用已被标记为构造函数的函数创建structs 时使用（更多信息请参见声明一个新函数）。这个动作接收先前定义的函数的方法，以及一些参数，如果需要的话，你可以使用左边的 图标来扩展添加更多参数。这些参数应该对应于函数所需的输入，并将用于填充正在创建的struct 。struct 将被返回到目标变量，该变量可以被标记为临时局部变量或不标记。

行动语法。

争论。

争论 描述
Function 要调用的script 或用户定义的函数的名称。
Argument0 ... Argument3 要传递给script 或函数的不同参数（值）（未使用的参数可以留空）。
Target 任何返回值的目标变量的名称（可以留空）。
