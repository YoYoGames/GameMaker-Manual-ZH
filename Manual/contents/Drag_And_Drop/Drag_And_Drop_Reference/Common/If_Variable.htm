如果变量

这个动作是用来用一个特定的表达式来检查一个给定变量的状态和一个给定的值。一个 "if "语句将总是返回 true 或 false ，用于检查的表达式可以是以下任何一个 以下任何一种。

  • 等于- 变量和值都完全相等
  • 小于- 该变量小于该值
  • 大于- 该变量大于该值
  • 小于或等于- 该变量小于或等于该值
  • 大于或等于- 该变量大于或等于该值

如果你flag "Not "参数，那么上面的表达式将被否定，例如 "等于 "变成"不等于"。还需要注意的是，变量和值参数都不限于数字值 可以是一个string 、一个指针、一个资源ID或其他任何函数可以返回或使用的东西（更多信息见数据类型一节），但你不能比较两个 例如，如果变量持有一个string ，而你试图检查它是否等于一个数字，那么你将得到一个语法错误。

重要提示：由于浮点精度问题，检查两个值是否完全相等可能会返回 false ，因为其中一个可能正好是1，而另一个可能是1.00000000000001。 另一个可能是1.00000000000001。这可以通过在检查前使用小数到整数的操作或使用 "大于 "或 "小于 "表达式来避免。
