返回

这个动作是在创建动作块函数时使用的，用来向调用script 的实例返回一个值。 像一个string ，一个数值，一个指针或一个资源ID，或者它可以是一个变量本身。请注意，当我们在一个函数中调用返回动作时，该函数将被退出--返回给定的值--并且在返回后不会有进一步的动作被调用。 尽管如果你在调用函数后在事件中还有动作，这些动作仍将被运行。
