执行脚本

这个动作可以用来调用一个script 函数或用户定义的函数。一般来说，这个动作只用于传统的支持，不应该真正用于当前或新项目。要使用这个 动作，你要给出要调用的script asset ，或者要调用的用户定义的函数变量名，然后提供它所需要的任何参数，即：你希望输入到脚本中的任何值（strings 、整数、资源指数等......）。 或函数，以便它能够使用它们。你可以通过点击动作旁边的加号图标，根据需要添加额外的参数 旁边的加号图标。

一些scripts 和用户定义的函数也可以返回一个值，以便以后在动作块列表中使用，因此你可以指定应该存储这个返回值的目标变量。这个变量可以是以前创建的，也可以是一个新的临时的 本地变量（如果你选中"Temp"复选框）。如果不需要的话，目标和参数字段都可以留空。

注意，一般来说，你最好使用动作Function Call来调用用户定义的函数，一般不应该需要使用这个动作，或者在代码中直接调用script asset 。
