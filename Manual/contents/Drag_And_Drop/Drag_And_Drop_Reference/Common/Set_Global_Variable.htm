设置全局变量

这个动作用于为全局变量赋值（在这个过程中创建它），或者用于改变现有全局变量的值。全局变量是 "属于 "任何特定实例的变量，而是属于整个项目的变量。 项目，因此可以被任何实例、script 或代码所访问和改变。关于全局变量的更多高级信息，请看这里。

你给全局变量的名字，然后给值，如果需要的话，可以相对于原值添加。如果你给了一个新的变量名称（即：一个在整个游戏中从未被分配过的变量），那么这 将创建新的全局变量，并将你给的值赋给它，而如果你给的是一个现有的全局变量的名字，它将简单地将值改为新的。请注意，勾选 "相对"将把这个值添加到现有的 全局变量的值（使用负数来减去），你也可以使用之前创建的表达式和变量作为值。

值得注意的是，当我们谈论 "值 "时，我们并不只是指数字值，因为变量可以是一个string 、一个指针、一个资源ID或任何其他函数可以返回或使用的东西。在string ，检查Relative会将值与变量之前的值连接起来，所以如果你的变量是 "Hello"，你用相对检查将 "World "分配给它，最后的变量值将是 "Hello World"。必须注意 然而，必须注意确保变量值都是相同的数据类型，因为试图添加，例如一个整数和一个string ，将导致编译器错误。

你也可以在同一个动作中添加额外的变量，方法是点击动作旁边的加号图标 旁边的加号图标，并给出另一个全局变量的 名称和值。关于变量的更多高级信息，请参见 "变量和变量范围"部分。
