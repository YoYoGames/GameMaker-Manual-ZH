如果未定义

这个动作用来检查一个变量或表达式是否返回 "未定义 "的值，如果是，那么这个动作将返回 true ，如果不是，它将返回 false 。"未定义 "的返回原因有很多。 "未定义 "的返回有很多原因，一个undefined 的值可能会导致项目的崩溃或严重的错误，所以你可以用这个动作来检查并在它发生之前抓住它。如果你flag "Not "参数，那么你要检查的是一个值是否是 不是"未定义"。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true ，这些操作将被运行，而 其他地方的任何动作将在 "if "块之后执行。注意，你可以在同一个动作中 "堆叠 "undefined 检查，方法是点击 "加号 "图标 旁边的加号图标 并给出另一个要检查的变量。这将在一次检查中检查所有给定的变量是否是undefined ，如果是，则只执行添加的动作代码。
