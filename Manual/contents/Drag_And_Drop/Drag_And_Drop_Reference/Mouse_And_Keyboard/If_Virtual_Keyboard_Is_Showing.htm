如果显示的是虚拟键盘

这个动作可以用来检查操作系统的虚拟键盘是否被显示。当键盘开放或打开时，该动作将评估为 true ，如果虚拟键盘关闭或关闭，则 false 。如果你启用了 not 修改器，这个动作将被反转，你可以将返回的 true / false 值存储在一个目标变量中，该变量可以被标记为temp（本地）。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评价为，这些动作将被运行，而其他地方的任何动作将在 "if "块之后执行。 评估为 true ，而在其他地方放弃的任何行动将在 "if "块之后执行。
