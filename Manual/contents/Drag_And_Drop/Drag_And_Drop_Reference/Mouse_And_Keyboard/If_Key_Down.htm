如果键下

通过这个动作，你可以轮询键盘，看是否有任何键正在被按住。该动作将在键盘按键被按下的每一步返回 true ，否则返回 false ，不过如果你点击 not 修改器，你可以检查按键是否没有被按下，即：该动作将在没有按键时返回 true ，在有按键时返回 false 。如果你只需要检查一次按键，那么就使用If Key Pressed这个动作。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

在其他地方放弃的任何行动将在 "if "之后执行。 块之后执行。
