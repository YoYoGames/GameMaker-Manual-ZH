如果鼠标向下

通过这个动作，你可以对鼠标进行轮询，看看所选的按钮目前是否被按住。该动作将在按钮被按下的每一步返回 true ，否则返回 false ，不过如果你点击 not 修改器，你可以检查按钮是否被按下，即：当没有按钮被按下时，该动作将返回 true ，如果有，则返回 false 。如果你只需要检查单个按钮是否被按下，那么可以使用If Mouse Pressed动作。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true ，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
