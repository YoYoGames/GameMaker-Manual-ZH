如果钥匙被释放

这个动作检查键盘按键何时被释放。它将在按键被释放的单一游戏步骤（帧）中返回 true ，在所有其他时间返回 false 。如果你启用了 not 修改器，这个动作将被反转。 如果你启用了 true 修改器，这个动作将被反转，并检查按键是否被释放，在它被释放的游戏步骤（帧）上返回 false ，在所有其他时间返回 。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true ，这些动作将被运行。 而在其他地方放弃的任何行动将在 "if "块之后执行。
