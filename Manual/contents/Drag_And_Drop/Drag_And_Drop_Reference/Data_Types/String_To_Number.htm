字符串转数字

这个动作可以将一个string 转换成一个数字。你提供要改变的string ，然后给出一个目标变量，将真实的数字返回到（这个变量可以被标记为临时的局部变量）。请注意，如果string 中有任何字符 如果string 是以数字和小数点开始的，它将不会被转换，但是如果 是以数字开始的，那么这些数字将被转换，而string 的其余部分将被排除。重要的是要注意，当把小数点传入这个 动作时，返回的string 将被四舍五入到两个小数位。下面是一个输入值和你期望的返回值的表格

字符串 返回的值
"123" 123
"123 Hello World" 123
"Hello World" 0（默认情况下，如果string 无法转换，将返回0。）
"Hello World 123" 0
"123.45678" 123.46
