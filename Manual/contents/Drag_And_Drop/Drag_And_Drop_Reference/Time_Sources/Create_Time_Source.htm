创建时间源

这个动作创建一个新的时间源，并将其存储在目标变量中。请阅读时间源的概述。

这是以 time_source_create()函数。阅读其页面，了解有关参数的详细信息。

你需要使用"开始时间源"来激活一个创建后的时间源。

当你不再需要一个时间源时，你必须使用"销毁时间源"来销毁它。

行动语法。

争论。

争论 描述
Target 新的时间源将被存储的变量
Parent 父时间源：可以是内置的时间源，也可以是自定义的。
Period 时间源的周期长度，它需要多长时间才能过期
Units 周期所表示的单位（秒或帧）。
Callback 的 method或脚本函数，以便在时间源到期时调用
Arguments OPTIONAL一个包含传入回调函数的参数的数组。
Repetitions OPTIONAL 时间源总共应该运行多少次，或者-1表示无限期重复。
Expiry Type OPTIONAL 时间源的到期类型

例子。

这个代码块做了以下工作。

  • 它创建了一个新函数，并启用了 "Temp "选项。这将创建一个本地方法变量。
    • 这个方法需要一个参数，即msg，它会将其打印到输出日志。
  • 然后它创建一个新的时间源，周期长度为1秒。
    • 它指定callback_method变量作为 "回调"，以及一个有一个参数的数组作为该函数。
    • 这个时间源被设置为无限期地重复，因为在 "重复 "中指定了-1。
  • 然后启动时间源。

这个时间源将每秒执行其回调，在输出日志中打印"一秒已过！"。
