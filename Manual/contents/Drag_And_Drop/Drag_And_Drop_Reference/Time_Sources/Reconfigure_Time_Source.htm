重新配置时间源

这个动作允许你改变在 "时间源 "字段中指定的现有时间源的设置。

这里提供的设置与 "创建时间源"相同，除了父级。

它将重置并停用时间源，所以你必须重新启动它。

行动语法。

争论。

争论 描述
Time Source 的 时间源ID来重新配置
Period 时间源的周期长度，它需要多长时间才能过期
Units 周期所表示的单位（秒或帧）。
Callback 的 method当时间源过期时调用
Arguments OPTIONAL一个包含传入回调函数的参数的数组。
Repetitions OPTIONAL 时间源总共应该运行多少次，或者-1表示无限期重复。
Expiry Type OPTIONAL 时间源的到期类型
