设置方向随机

该动作用于将实例的移动方向设置为从8个基本点中选择的一个或多个随机方向。你可以在动作中点击8个方向箭头中的一个或多个，动作将随机选择其中的一个方向 来作为该实例的方向。如果你没有选择任何一个箭头，那么这个动作将不会对实例的当前方向产生影响。
