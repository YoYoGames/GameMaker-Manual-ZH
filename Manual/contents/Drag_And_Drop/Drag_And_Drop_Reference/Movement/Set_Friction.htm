设置摩擦力

这个动作是用来设置实例的 "摩擦 "的。摩擦力只是意味着每一帧游戏都会从速度矢量中扣除一个特定的量，在这里你指定要扣除的量。通常，较小的数值效果最好，比如 0.2，但实际的数额将取决于你项目的设置。如果你设置了相对flag ，那么给定的数额将被添加到当前的摩擦值中，而不是设置它。
