设置速度

这个动作是用来设置实例的移动速度。速度的计算方法是一个实例在一个游戏帧（步）内所要移动的像素数。默认情况下，速度值将被添加到方向矢量中--因此，如果你设置了 速度为2，方向为45°，实例将在每一游戏帧中向上和向右移动2个像素--但你也可以点击 direction ，选择水平或垂直速度。

选择其中任何一个，都只为水平或垂直向量设置速度。注意 你可以flag ，作为相对于当前速度的值，这将从实例的实际速度中增加或减少该值，你也可以提供负数，这将使它在与当前的 方向的值。
