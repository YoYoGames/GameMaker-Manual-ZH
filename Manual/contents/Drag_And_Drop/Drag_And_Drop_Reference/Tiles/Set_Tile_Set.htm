瓷砖套装

通过这个操作，你可以将一个图层设置为使用与在Asset 浏览器中创建的不同的瓷砖组。你给该层的名称（一个string ，如在Room 编辑器中定义的），然后是要使用的 资源。 要使用的tile set 资源，该层上的所有tiles 中的room 都将用新的tile set 绘制。

行动语法。

争论。

争论 描述
Layer 设置新tile set 的层。
Tileset 该层要使用的tile set 。
