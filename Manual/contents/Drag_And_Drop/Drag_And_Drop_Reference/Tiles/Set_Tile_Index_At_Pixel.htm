设置 像素的索引Tile

通过这个动作，你可以在一个给定的图层上设置一个tile ，以获得一个新的索引。图层是由给定的图层名称指定的（一个string ，在Room 编辑器中用来命名图层），然后你给出一个x和y的位置来获得要设置的tile 。x和 y位置可以被标记为相对于调用实例的位置，被设置的tile 将是给定位置在room 中的那个，而使用的tile set 将是在房间中分配给该层的那个。 编辑。最后你提供tile 的索引来设置tile （tiles 的索引是从左到右从上到下，0在左上角）。注意，你可以随时将tile 索引设置为0来清除tile 。
