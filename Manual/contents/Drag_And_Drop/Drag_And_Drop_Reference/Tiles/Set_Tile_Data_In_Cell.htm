在单元格中设置 数据Tile

通过这个动作，你可以为瓦片地图层上的一个tile 单元设置tile 数据。tile 数据是一个简单的值，反映了tile 的索引以及翻转、镜像和旋转的值。 以及你所使用的任何自定义tile 掩码。在这个操作中，你首先要使用在string 编辑器中定义的图层名称（room ）来提供目标图层，然后在tile 地图图层上的列和行来设置瓦片数据。 的列和行。单元 "是指tile 地图网格上容纳你想设置数据的tile 的区域，所以如果你的tiles 是16x16，例如，room 是1024x768，tile 地图将有64列和48行。

注意：所选图层必须在Tile 编辑器中被定义为room 地图层，否则可能会出现错误。

这个动作的最后一个参数是瓦片数据本身。通常情况下，你会用动作Gettile Data In Cell来检索tile 地图单元的Tile 数据，然后用动作SetTile Data Transform来操作它，再用这个动作来设置它。关于tile 数据的更多信息，请参见GML "瓦片地图功能"部分。
