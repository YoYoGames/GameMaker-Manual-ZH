获取瓦片图变量

通过这个动作，你可以为一个图层上的瓦片地图元素检索给定数量的变量中的任何一个。当你在tile 编辑器中创建一个room 图层时，这个图层持有一个瓦片地图元素，然后用瓦片集的tiles 填充。这个tile 地图元素有一些与之相关的值，如偏移位置、正在使用的瓷砖集、宽度、高度等...。你可以得到的完整的数值列表是。

  • X坐标。 tile 地图元素在图层上沿X轴所处的位置。
  • Y坐标。 tile 地图元素在图层上沿Y轴所处的位置。
  • 列。 tile 地图元素的宽度（定义为tile 单元的列数）。
  • 行数。 tile 地图元素的高度（定义为tile 单元的行数）。
  • 瓦片宽度：单个tile 单元的宽度。
  • 瓦片高度。单个tile 单元的高度。
  • 瓦片集资源。已经分配给tile 地图元素的瓦片集资源，用于绘制。
  • 当前帧。正在为动画绘制的当前帧tiles 。
  • 屏蔽： tile 地图元素的屏蔽数据。
  • 全局屏蔽：所有tile 地图元素的全局屏蔽数据。

使用这个动作时，你要提供要获取string 地图元素数据的图层名称（一个Room ，在tile 编辑器中定义），然后是你要检索的数据类型（如上面列表所示）。然后，返回的值将被存储在 目标变量，该变量可以是以前创建的，也可以是一个新的临时变量（如果你选中 "Temp "复选框）。请注意，你可以通过点击 "加号 "图标来检索更多的值。 旁边的加号图标 动作，选择另一个变量，给另一个变量来存储返回的值，就可以获取更多的值。
