获取 数据转换Tile

通过这个动作，你可以发现一个tile 的tile 数据是否已经以三种方式之一进行了转换：翻转、镜像或旋转。tile 数据是一个简单的值，它反映了tile 的索引以及所应用的变换，你通常会想先用获取 数据来获取。 在使用这个动作之前，你通常要先使用获取单元格中的tile 数据或获取像素处的Tile 数据来获得Tile 数据。如果所选择的变换已经应用到该单元格，该操作将返回 true 。 如果所选择的变换已经应用于瓷砖数据，则该操作将返回 false ，否则将返回 ，然后返回的值将存储在目标变量中，该变量可以是之前创建的，也可以是一个新的临时变量（如果你选中 "Temp" 复选框）。关于tile 数据的更多信息，请参见GML "瓦片地图功能"部分。
