在像素处设置 数据Tile

通过这个动作，您可以为瓦片地图层上特定位置的tile 单元设置tile 数据。tile 数据是一个简单的值，它反映了tile 的索引以及 翻转、镜像和旋转的值，以及你所使用的任何自定义tile 掩码。在这个动作中，你首先要使用在string 编辑器中定义的图层名称（room ）来提供目标图层，然后在房间的X和Y轴上设置 来设置tile 的数据。请注意，tile 数据将被设置为给定位置所在的tile 地图 "单元"。例如，如果你的tile set tiles 是16x16的，那么tile 地图层将在16x16像素的单元中保存tiles ，所以 如果你使用这个动作针对room 的位置（8，8），你将会改变tile 地图的单元格（0，0）中的tile 。

注意：所选图层必须在Tile 编辑器中被定义为room 地图层，否则可能会出现错误。

该动作的最后一个参数是tile 数据本身。通常情况下，你会使用动作Gettile Data At Pixel为tile 地图单元检索Tile 数据，然后使用动作SetTile Data Transform对其进行操作，再使用此动作再次设置。关于tile 数据的更多信息，请参见GML "瓦片地图功能"部分。
