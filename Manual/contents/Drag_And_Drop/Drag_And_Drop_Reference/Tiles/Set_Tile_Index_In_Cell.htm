设置 单元格中的索引Tile

通过这个动作，你可以把一个给定图层上的tile ，设置成一个新的索引。图层是由给定的图层名称（string ，在Room 编辑器中用来命名图层）指定的，然后你给出图层中的单元格的行和列，以获取 要设置的tile 。最后，你提供tile 的索引来设置tile （tiles 的索引从左到右，从上到下，0在左上角）。注意你可以随时将tile 索引设置为0，以清除tile 。
