如果得分

这个动作用于使用一个特定的表达式来检查实例变量 score 的值。你给出要检查的表达式类型和要检查的当前 score 的值，"if "语句将 总是返回 true 或 false ，这取决于使用的表达式和值。可用的表达式有。

  • 等于- 变量和值都完全相等
  • 小于- 该变量小于该值
  • 大于- 该变量大于该值
  • 小于或等于- 该变量小于或等于该值
  • 大于或等于- 该变量大于或等于该值

如果你flag Not参数，那么上面将是否定的表达式，例如 "等于 "变成"不等于"，所以你将检查 score 值是否不等于给定值。

重要提示：由于浮点精度问题，检查两个值是否完全相等可能会返回 false ，因为其中一个可能正好是1，而另一个可能是1.00000000000001。 另一个可能是1.00000000000001。这可以通过在检查前使用小数到整数的操作或使用 "大于 "或 "小于 "表达式来避免。
