设置实例时间线

这个动作从timeline 浏览器中分配一个Asset 到实例中。你给timeline ，然后你可以选择在它到达最后一刻时让timeline loop （最后一刻运行后的下一帧将loop ，再回到 timeline 的第一时刻），以及是否立即启动timeline 。如果你不立即启动timeline ，你可以使用动作SetTimeline State来启动它，在另一个 事件或script 。

Start Timeline 标记，以使timeline ，或不立即启动（默认为标记）。
