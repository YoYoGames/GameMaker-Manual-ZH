这个动作被用来将一个timeline ，从当前时刻跳到一个新的时刻。Timelines ，通过不同的 "时刻 "向前移动，每个时刻通常占用一个游戏帧（就像一个object 的步骤事件）。一旦一个timeline 已经开始 loop 它将在各个时刻中前进，直到到达终点，这时它将回到timeline 中的第一个时刻或完全停止。然而，这种线性的timelines ，可以用这个动作来修改，在任何时候设置一个不同的 在任何时候设置不同的时刻，允许你创建你自己的timeline loops ，或者在某些情况下跳过timeline 的部分，等等。有了这个动作，你只需在timeline 中设置要跳到的时刻，下一个游戏帧就会运行该时刻。 那个时刻（注意，你需要使用动作Set Instance Timeline为实例指定一个timeline ）。
