设置 速度Timeline

这个动作可以用来设置一个timeline ，在它所包含的瞬间移动的速度。通常情况下，在每个游戏帧中，你希望timeline 中的位置增加1，但是你可以通过设置这个 动作设置为一个不同的值。你选择的值将被视为一个乘数，因此，例如，一个2的值将使timeline 以两倍的速度运行，而一个0.5的值将使它以一半的速度运行。 这对时刻来说意味着，如果数值大于1，那么在同一游戏帧中可能会出现几个时刻（没有时刻被跳过），如果速度小于1，那么在一个时刻运行后，将有一些帧 如果你将timeline 速度设置为0.1，例如，你将在一个游戏帧上得到一个时刻，然后有9个游戏帧没有时刻，而在第10个游戏帧上将运行下一个时刻。

如果你用Set Instance Timeline或Settimeline State的动作来设置Timeline ，但没有设置 速度，那么timeline 将默认为每游戏帧1时刻。
