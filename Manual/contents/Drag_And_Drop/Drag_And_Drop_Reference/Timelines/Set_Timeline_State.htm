设置 状态Timeline

这个动作可以用来启动、停止、暂停或恢复一个timeline 。timeline ，首先必须使用动作Set Instance Timeline为实例设置，然后你可以在任何时候使用这个动作来改变其运行状态。 从以下选项中改变其运行状态

选项 描述
开始 启动分配给实例的timeline （如果timeline 被暂停了，这将再次把开始位置设置为时刻0）。
简历 恢复一个暂停的timeline （如果timeline 已经停止并且没有暂停，这将不做任何事情）。
暂停使用 暂停分配给实例的timeline （当暂停时，timeline 中的时刻将不会随着游戏帧的推进而推进）。
停止 停止分配给实例的timeline 。
