设置 头Sequence

通过这个动作，你可以在sequence ，将给定的room 元素的播放头位置设置为指定的帧。你提供sequence 元素的ID（由动作Create Sequence返回）和 新的播放头位置（以帧为单位）。关于播放头的信息，见《使用Dope Sheet》。

注意，位置是在 帧 ，如果你设置的值大于总帧数（或小于0），那么实际的最终播放头的位置将取决于已经选择的sequence 播放类型。 遵循这些规则。

  • 没有循环。播放头将夹在loop （如果给定的值是负的，则夹在开始）的末端。
  • 循环播放。播放头将回绕到sequence ，再加上额外的帧数，例如：如果你在一个30帧的sequence ，将播放头设置为第45帧的位置，实际播放头的位置最终将是15。如果你给 负值，那么sequence ，播放头将以相反的方向绕行。
  • 乒乓式循环。播放头将前进到最后，然后返回超过总帧长的额外帧数，反转播放方向，例如。如果sequence 是30帧，你把播放头设置为 到40，最终播放头的位置将是20，方向将是相反的（向下到0）。请注意，你不应该 ，也不应该 ，因为它可能会产生不确定的结果，所以不要给乒乓播放的sequence 的负值。 结果。

以这种方式设置磁头位置不会停止播放，除非暂停播放，否则sequence ，就会从新的播放磁头位置继续播放。
