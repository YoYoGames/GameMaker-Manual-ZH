获取 长度Sequence

通过这个动作，你可以检索到sequence 中给定的room 元素的长度（以帧为单位）。你提供sequence 元素的ID（由动作Create Sequence返回），这个动作将返回 它在目标变量中的长度。这将是Sequence 运行的帧数。

注意，目标变量可以被标记为临时（本地），它将为该动作创建，并可以在该动作所在事件的所有后续动作中使用。

行动语法。

争论。

争论 描述
Variable sequence 元素的ID，以获得其长度。
Target 保存长度的目标变量。

例子。

上述动作块检索了存储在变量 "seq_id" 中的sequence 元素 ID 的长度。然后，该值被用来设置相同的sequence 元素的头部位置到其结束前的10帧。
