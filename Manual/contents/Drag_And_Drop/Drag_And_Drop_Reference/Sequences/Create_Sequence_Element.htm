创建序列

通过这个动作，你可以从sequence 浏览器中创建任何sequence 的元素Asset 。你给出sequence 的名称和在room 中要创建的X/Y位置，以及要创建的层（这个层必须是资产层）。你可以勾选 "相对 "flags ，将sequence 相对于运行动作代码块的实例定位。每当你以这种方式在游戏中创建一个sequence ，它就会被赋予一个独特的元素ID值，然后你可以在进一步的动作中识别这个sequence ，因此这个动作允许你定义一个 "目标 "变量来保持这个ID值（如下图所示）。注意，目标可以被标记为一个临时（本地）变量，它将为该动作创建，并可用于该动作所在事件的所有后续动作。
