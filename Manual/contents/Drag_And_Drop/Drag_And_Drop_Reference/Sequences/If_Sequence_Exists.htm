如果 存在Sequence

这个动作可以用来检查给定的sequence 元素是否存在于游戏中room 。你给出元素的ID，就像创建序列这个动作所返回的那样，如果sequence 元素存在于room ，那么 该动作将返回 true ，否则它将返回 false 。

如果你flag "不是"参数，那么动作将检查是否有这样的sequence 元素存在，如果没有发现，它将返回 true 。

请注意，要把动作添加到 "if" 块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if" 评价为 " " ，这些动作将被运行。 评估为 true ，而在其他地方放弃的任何行动将在 "if" 块之后执行。
