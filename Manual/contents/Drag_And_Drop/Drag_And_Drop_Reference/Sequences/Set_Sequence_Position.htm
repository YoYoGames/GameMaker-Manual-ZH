设置 位置Sequence

通过这个动作，你可以为给定的sequence 元素在room 。你提供sequence 元素的ID（由动作Create Sequence返回），然后是新的X和Y位置。如果你 勾选 "相对 "复选框，那么给定的X和Y值将被添加到当前sequence 的位置，否则它们将是一个绝对位置。
