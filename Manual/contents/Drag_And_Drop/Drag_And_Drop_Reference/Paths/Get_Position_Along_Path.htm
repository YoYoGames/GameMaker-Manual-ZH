沿着路径获取位置

这个动作将获得一个实例沿path 的位置。返回的值从0-1归一，所以如果你得到一个结果，例如0.5，那么这个实例正好在path 的中间。注意，你必须调用Start Following Path动作为实例分配一个path ，这个动作才会返回一个0以外的值。返回的值将存储在你给出的目标变量中，或者你可以把flag 这个变量作为一个临时局部变量来创建它并分配值。
