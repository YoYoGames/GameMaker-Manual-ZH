沿着路径设置位置

这个动作将设置一个实例沿path 的位置。数值从 0 - 1 归一，所以如果你把它设置为，例如，0.5，实例将被移动到path 的正中间。注意，你必须调用Start Following Path动作来给实例分配一个path ，这样才有效果。
