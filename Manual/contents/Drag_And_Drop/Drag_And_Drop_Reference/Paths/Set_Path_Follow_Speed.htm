设置 跟随速度Path

这个动作将设置一个实例跟随path 的速度，单位是每游戏帧的像素（注意，你必须调用Start Following Path动作来为实例分配一个path ，这样才能有 任何效果）。你可以使用负值来表示实例应该反向跟随path ，值得注意的是，你在这里设置的速度可以被path 编辑器中为每个path 点设置的速度修改器所修改，所以如果你 设置一个path 速度为2，而在path ，其中一个点的速度修改器为200%，则该点的实际像素/游戏帧速度将为4。
