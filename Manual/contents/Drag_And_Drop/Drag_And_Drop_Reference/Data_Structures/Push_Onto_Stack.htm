推到堆栈上

通过这个动作，你可以把一个值推到一个堆栈数据结构上。你提供持有堆栈索引的变量（由动作Create Stack返回），然后给出要推到顶部的值。对于堆栈数据 结构，值总是被添加到堆栈的 "顶部"，而不是在末端或中间（如果你需要这个功能，那么使用一个列表数据结构）。
