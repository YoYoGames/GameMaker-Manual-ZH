如果数据结构存在

通过这个动作，你可以检查一个变量，看它是否索引了一个数据结构。你首先需要选择要检查的数据结构类型（列表、地图、网格或堆栈），然后你提供持有数据结构的变量 索引（如创建列表、创建地图、创建网格和创建堆栈等动作所返回的）。如果数据结构存在，该操作将返回 true ，如果不存在，则返回 false 。注意，你可以在动作中flag "not "来检查一个变量是否持有数据结构的索引。
