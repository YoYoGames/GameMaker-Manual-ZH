获取地图值

通过这个动作，你可以检索与一个地图数据结构中的给定键相关的值。你提供了持有地图索引的变量（由动作Create Map返回），然后给出一个 "key"（这是一个值在地图中的标识符）。 是地图中一个值的标识符），同时提供一个目标变量来保存返回的值（可以标记为一个临时的局部变量）。

注意：如果你给了一个在地图中不存在的键，那么该动作将返回 undefined 。你可以用If Undefined动作来检查这一点。

争论。

争论 描述
Map 从地图上获取数值的索引（存储在一个变量中）。
Key 获取的关键（真实或string ）
Target 保存返回的地图值的目标变量
