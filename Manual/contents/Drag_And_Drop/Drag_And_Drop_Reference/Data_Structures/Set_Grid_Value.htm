通过这个动作，你可以将网格数据结构中的一个特定单元设置为一个值。你要提供保存网格索引的变量（由动作Create Grid返回），然后给出要设置的单元格的x和y位置（列和行）。 和行），然后给出要设置的单元格的x和y位置，最后是该单元格的值。值可以是任何有效的数据类型，但x和y位置必须是网格范围内的整数。
