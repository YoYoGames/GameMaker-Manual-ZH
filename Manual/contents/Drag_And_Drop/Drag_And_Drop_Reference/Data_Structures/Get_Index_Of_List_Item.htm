获取列表项的索引

通过这个动作，你可以恢复一个项目在给定列表中的索引位置。你提供存储列表索引的变量（由动作Create List返回）和你想在列表中找到的索引位置的值。 以及一个目标变量，用来存储返回的项目的索引位置（可以标记为临时的局部变量，直到script 或事件结束）。请注意，如果你要找的值不存在于列表中 不存在于列表中，那么将没有列表索引位置可以返回，因此返回值将是-1。
