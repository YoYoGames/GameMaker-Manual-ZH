获取网格值

通过这个动作，你可以将网格数据结构中的一个特定单元设置为一个值。你提供了保存网格索引的变量（由动作Create Grid返回），然后给要设置的单元格的x和y位置（列 和行），然后给出要设置的单元格的x和y位置，最后是该单元格的值。你还必须设置一个目标变量（可以标记为临时局部变量）来存储返回的单元格值，该值可以是任何有效的数据类型。需要注意的是 单元的x和y位置必须是网格边界内的整数。

争论。

争论 描述
Grid 要获得的网格的索引（存储在一个变量中）。
x 沿着X轴获得的单元格位置（列）。
y 要得到的单元格沿y轴的位置（行）。
Target 用来存储返回值的目标变量
