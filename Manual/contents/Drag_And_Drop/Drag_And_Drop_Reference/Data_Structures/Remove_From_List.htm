从列表中删除

这个动作可以用来从给定的列表中删除一个索引项。你提供了一个存储列表索引的变量（由动作Create List 返回）和要删除的列表中的索引位置，其中 索引位置在 1 和（列表长度-1）之间。这个动作不返回任何东西，所以如果你需要索引位置的值，你应该在删除索引之前使用Get List Item At。

行动语法。

争论。

争论 描述
List 要删除的列表的索引（存储在一个变量中）。
Index 要删除的列表中的索引
