获取输入（回滚）。

这个动作返回一个struct ，包含一个玩家的输入值。它需要一个可选的参数，指定要返回输入值的玩家的ID。

建议在多人游戏中使用If Key Down (Rollback),If Key Pressed (Rollback)和If Key Up (Rollback)来检查输入。

当在一个由定义球员（回滚）创建的球员实例中使用时，你不需要指定一个球员ID，所以你可以直接指定player_id。

当用于任何其他实例，或手动创建的球员实例时，你必须指定一个球员ID。

有关哪些输入被返回以及如何改变它们的信息，请参见定义输入。
