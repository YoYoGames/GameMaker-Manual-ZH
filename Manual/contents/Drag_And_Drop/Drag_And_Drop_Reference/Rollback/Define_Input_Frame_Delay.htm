定义输入帧延时（回滚）。

这个动作定义了本地播放器输入的固定帧延迟。

默认情况下，你的游戏将使用一个自适应的输入延迟，它是基于玩家之间的延迟。这几乎适用于所有的游戏，然而，一些特定类型的游戏（如格斗游戏）可能更喜欢使用固定的输入延迟，而不是可能随时改变的自适应延迟。

除非你清楚地知道你在做什么，否则不建议改变这个。

指定的延迟值必须是帧数。如果你指定-1，游戏将恢复到使用自适应延迟。你可以设置的最大延迟是10帧。
