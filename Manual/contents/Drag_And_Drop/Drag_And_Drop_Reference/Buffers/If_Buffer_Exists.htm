如果 存在Buffer

这个动作可以用来检查一个给定的buffer ID是否有效，并且buffer 在内存中实际存在。如果buffer ID是有效的并且存在，该动作将返回 true ，否则将返回 false 。你可以用flag "not" 来使动作检查是否有一个buffer 不存在。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
