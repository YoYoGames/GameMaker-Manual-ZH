如果缓冲区结束

这个动作可以用来检查读/写位置是否已经到达buffer 。你提供由动作Create Buffer返回的唯一的buffer ID值，然后如果读/写位置已经到达buffer 中所有可用数据的末端，该动作将返回 true ，如果没有，则返回 false 。你可以flag "not "来使动作检查读/写位置是否在buffer 的最后。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
