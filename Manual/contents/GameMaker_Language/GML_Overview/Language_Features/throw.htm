抛出

throw runtime 语句允许你生成你自己的运行时异常，结束游戏并显示错误信息，使用以下语法。

throw (<expression>);

使用的表达式可以是一个值或一个string 或任何其他数据类型，然后这将产生一个异常错误--默认情况下--显示在屏幕上，关闭错误信息后游戏将结束。例如，调用这个。

throw ("Hello World!");

将导致显示以下未处理的异常错误。

然而，你可以接管这个错误信息并使用你自己的处理代码，方法是调用 函数 exception_unhandled_handler().这个运行时函数允许你提供一个自定义的 method 来使用，当你的游戏中出现任何未处理的异常时，它将被调用。
