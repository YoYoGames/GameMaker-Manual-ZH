新

new 操作符可以用来创建一个struct ，该函数被调用，并给出参数。然后该函数可以用参数中的变量来填充struct --就像一个实例的创建事件将为该实例设置初始变量一样--然后 new 操作符将返回struct 。在使用该操作符之前，必须注意给定的函数必须被标记为 constructor 函数，否则 new 操作符将不会创建struct （下面的代码示例显示了这一点，更多信息请参见关于结构的页面）。

注意你可以检查一个struct ，以了解哪个函数被用来创建它，使用runtime 函数 instanceof().

该操作符的语法如下。

<variable> = new <function> (<argument>, <argument>, ...);

当你在使用函数的同时使用new操作符时，它将返回一个struct 引用，该引用存储在给定的变量中，允许你访问struct 和任何由函数在其中创建的变量。

下面的例子显示了一个被定义为构造器的函数。

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

然后，这个函数可以和 new 操作符一起使用，以创建一个struct ，并将变量设置为函数中使用的参数值来填充它，像这样。

mystruct = new init_struct(10, 100, "Hello World");

需要注意的是，在任何没有被标记为构造函数的函数上调用new将导致运行时异常。
