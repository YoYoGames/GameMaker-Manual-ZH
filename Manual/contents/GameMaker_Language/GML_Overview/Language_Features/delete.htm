删除

delete 操作符用于取消对结构的引用，其语法如下。

delete <variable>;

这意味着，你在调用删除操作符的同时，还要调用存放struct 的变量，它将删除存储在给定变量中的对struct 的特定引用 （引用只是一个指向内存区域的值 存储struct 及其内容）。如果这个引用是你游戏中对struct 的最后一个引用，那么垃圾收集器将在接下来的迭代中从内存中删除struct 。 在接下来的迭代中，通常是在下一个步骤的开始。

注意：delete ，只能与变量一起使用，而不是表达式。

默认情况下，structs ，在代码中没有进一步的引用，将在使用后的步骤中自动进行垃圾收集，但使用这个操作符flag ，让垃圾收集器在下一步开始时从内存中删除，是很好的做法。 在下一步骤开始时从内存中删除，这样可以快速、更有效地释放内存。

下面的例子显示了一个正在创建的struct - 例如在一个实例的创建事件中。

mystruct =
{
    a : 0,
    b : 100,
    c : "Hello World"
}

这个struct ，然后像往常一样在实例事件中使用，最后在Clean Up事件中被标记为垃圾收集，像这样。

delete mystruct;
