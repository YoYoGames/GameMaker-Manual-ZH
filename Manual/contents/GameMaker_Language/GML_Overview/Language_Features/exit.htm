退出

exit 语句的语法如下。

exit;

exit 简单地结束当前脚本函数的执行。 method，或者 event.注意这里的用法根据作用域的不同而略有不同。

  • 如果你在一个script 函数或方法中使用 exit ，它将简单地退出函数并返回到调用该函数的代码。
  • 如果你在一个对象的事件中的代码块中使用 exit ，它将退出整个事件，即使在 exit 被调用后还有各种独立的代码块。
  • 如果你在一个父事件中使用 exit ，并在子事件中使用 event_inherited()在子事件object ，只有父事件将退出，而子事件（调用 event_inherited() ）将继续。

当在一个事件中使用时， exit ，通常用于避免一个实例在满足特定条件（或不满足）时运行任何进一步的代码。下面的代码概述了一个如何使用它的例子，在这种情况下是在一个碰撞事件中，尽管它可以在任何事件中使用。

if (!visible)
{
    exit;
}

other.hp -= attack;
other.coins -= 4;
coins += 4;

上面的代码检查当前实例是否不可见，在这种情况下，它退出代码块，否则它继续运行其余的代码。

注意：它不会结束游戏的执行。为此，你需要使用函数 game_end().
