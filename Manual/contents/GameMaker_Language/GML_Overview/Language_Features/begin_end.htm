开始/结束

GameMaker Language 允许你在创建代码块时使用关键字 begin 和 end ，而不是更常用的大括号 {} 。下面的代码例子显示了这是如何工作的。

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

请注意，使用这些关键字并不典型，作为语言的一部分，更多的是为了遗留支持，而不是其他，而且在未来的任何时候，它们都可能被废弃。
