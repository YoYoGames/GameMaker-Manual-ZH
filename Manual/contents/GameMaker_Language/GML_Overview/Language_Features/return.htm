返回

return 语句的语法如下。

return (<expression>)

你只在 脚本函数和方法中使用 return 语句，它被用来从函数中返回一个值，以便在进一步的代码或函数调用中使用。应该注意的是，函数的执行在 return 语句处结束，也就是说，任何在 return 被调用后的代码都不会被运行。这里有一个简短的例子script ，称为 sqr_calc ，它计算传递给它的任何数值的平方，它包括错误捕捉，以防传递给它的参数不是实数。

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if !is_real(value)
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

要在一段代码中调用一个函数，只需使用与调用runtime 函数时相同的方法--即把函数名称与参数值写在括号中。因此，上述函数将被这样调用。

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

这里的变量 val ，要么是0（因为变量 amount 不是实数），要么是 value * value 的总数。
