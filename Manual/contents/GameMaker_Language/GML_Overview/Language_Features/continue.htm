continue 语句的基本语法如下。

continue;

如果在构成loop 的语句中使用(for, repeat, while或 do / untilloop)，它将立即结束当前的迭代，并跳回到loop 的开头，开始新的迭代，并省略 continue 中的任何代码。它也可以用在 with语句中使用，它将导致代码跳到下一个实例并再次运行。请注意，如果在这些情况之外使用 continue ，将会出现错误。

下面是一个在 for 循环中使用的例子。

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

下面是一个在 while 循环中使用的例子。

var file = file_text_open_read("Game_Data.txt");
var _num = 0;
