重复

GameMaker Language 有许多方法可以进行循环，即：让一个或多个语句在自己身上迭代一定的次数。其中最简单的是 repeat 语句，它的形式是：

repeat (<expression>)
{
    <statement>;
    <statement>;
    ...
}

通过 repeat ，给定的语句被重复的次数由表达式的四舍五入值表示。例如，下面的语句在随机位置创建了五个球。

repeat (5)
{
    instance_create_layer(random(400), random(400), "Instances", obj_ball);
}

这对于避免多次输入相同的代码非常有用，或者对于使用数组，或者对于通过一些操作计数等等。你也不局限于使用单个语句，可以通过将多个语句括在大括号内重复使用 {} 。比如说。

var _x = 32;
repeat (global.p_lives)
{
    draw_sprite(spr_heart, 0, _x, 32);
    _x += sprite_get_width(spr_heart);
}

上面的例子重复大括号中的语句，无论 "lives "全局变量有多少次 迭代，每次迭代都在 _x ，然后根据心脏sprite 的宽度沿位移动位置，画出心脏sprite 。

什么时候应该使用 repeat loop ？任何时候你想在一个或多个语句上重复固定的次数，而不需要特别保持迭代的次数。

值得注意的是，你可以在一个循环中使用特殊的 break 和 continue 语句。 重复循环中使用。使用 break ，将立即退出loop ，并在loop 应该完成后，转入事件或函数中的任何代码，例如。

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

上面的代码loops ，通过10个数组值，并将它们添加到一个局部变量中。如果本地变量的总和大于 max_total ，则使用break终止loop ，否则将继续loop 。

在 repeat loop ，一个使用 continue 的例子是。

repeat(10)
{   
    var _x = random(room_width);
    var _y = random(room_height);
    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }
    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}

这段代码将重复10次，生成一个随机的room ，然后检查该位置是否存在一个object obj_Enemy 的实例。如果存在，则使用 continue 终止当前的loop 迭代，并开始新的迭代；如果不存在，则在随机位置创建一个object obj_Enemy 的实例。
