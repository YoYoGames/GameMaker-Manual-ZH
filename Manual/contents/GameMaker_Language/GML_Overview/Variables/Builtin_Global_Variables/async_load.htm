异步加载

这个变量是全局范围的，在异步事件中使用时，用来保存DS地图，其他时候都是-1。DS地图的实际内容将取决于异步事件回调的类型，以及用于触发事件的函数，所以请参考这些事件的各个页面，了解所有可能的DS地图内容的完整细节。

语法。

async_load;

返回。

DS Map ID

扩展实例。

sprite =  sprite_add("site.com/path/image.png", 16, true, true, 0, 0);

上述代码将在一个事件中被调用，以从外部URL加载一个sprite 。然后，这将触发图像加载的异步事件，在那里你将解析 async_load 地图。

if ds_map_find_value(async_load, "id") == sprite
{
    if (ds_map_find_value(async_load, "status") >= 0)
    {
        sprite_index = sprite;
    }
}

上面的代码将首先检查 async_load 地图的ID，然后检查回调的状态。如果该值大于或等于0（标志着成功），那么回调的结果将被用来将实例的sprite 索引设置为新加载的图像。

返回。全局变量
