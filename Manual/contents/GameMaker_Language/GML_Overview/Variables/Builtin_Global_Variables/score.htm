得分

这个变量是全局范围的，用来保存一个数字值，通常用来表示玩家的得分。这个变量只是为了支持GameMaker以前版本的遗留项目，并且应该 不用于新项目中因为它在未来可能会被废弃。

语法。

score;

返回。

Real (single precision floating point value)

例子

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

上面的代码使用a检查运行代码的实例的object 索引。 switch，然后根据不同的object ，向 score 变量添加不同的金额。

返回。全局变量
