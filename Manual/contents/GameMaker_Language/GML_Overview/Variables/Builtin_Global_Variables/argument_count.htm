论据数

这个只读变量返回传递给脚本函数的 "参数 "数量，或者是一个 method.通常与一个参数数组 (argument[0...15])一起使用，以允许一个给定的函数有不同的输入参数。

注意：这不再是严格意义上的要求，更多的是为了支持传统的东西而提供。所有用户定义的函数将接受变量参数，任何传入函数的参数，如果不是命名的输入的一部分，将默认初始化为未定义，可以使用is_undefined()函数检查。

语法。

argument_count;

返回。

例子。

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

上面的代码使用 argument_count 变量以及 argument 数组来初始化变量，是一个允许用户定义的函数接受可变数量参数的例子。

返回。全局变量
