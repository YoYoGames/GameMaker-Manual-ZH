健康

这个变量是全局范围的，用来保存一个数字值，通常用来表示玩家的健康状况。这个变量只是为了支持GameMaker以前版本的遗留项目，并且应该 在新项目中不使用因为它在未来可能会被废弃。

语法。

health;

返回。

Real (single precision floating point value)

例子

if (health <= 0)
{
    global.state = "Game Over";
    instance_destroy();
}

上述代码检查 health 变量，如果它小于或等于0，则设置一个全局变量并销毁实例。

返回。全局变量
