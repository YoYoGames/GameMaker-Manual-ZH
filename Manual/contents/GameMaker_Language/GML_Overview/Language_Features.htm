语言特色

GameMaker Language (GML)有许多特性，它们是所有用该语言编写的程序的基本结构的一部分。每一个都会在你写的任何代码中多次使用，所以在开始项目编程之前，你应该知道它们是什么，以及它们如何工作。

  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
