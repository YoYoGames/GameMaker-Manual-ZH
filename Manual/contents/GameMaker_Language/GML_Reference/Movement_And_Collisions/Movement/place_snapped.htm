地点_抢占

通过这个函数，你可以检查并查看一个实例的原点（它的x和y位置）是否与你指定的hsnap和vsnap值的网格对齐。

语法。

place_snapped(hsnap, vsnap);

争论类型 描述
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

返回。

例子。

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

上面的代码检查 "obj_Pieces "的所有实例，看它们是否被固定在一个32x32像素的网格上，如果不是，就把它们固定在该网格中最近的位置。
