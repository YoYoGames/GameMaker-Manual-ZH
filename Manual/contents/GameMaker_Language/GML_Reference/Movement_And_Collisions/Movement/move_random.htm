移动_随机

通过这个函数，你可以将实例设置到room 中的任何位置，但是要对准一个 "看不见的 "网格。因此，hsnap和vsnap的值为32，将把实例设置为一个随机的位置，该位置与32x32的方格对齐。你可以把这些值设置为1，以获得room 中的任何位置。

注意：由于GameMaker每次都会生成相同的初始随机种子，使debugging ，所以这个函数在每次重新运行游戏时都会捕捉到相同的位置，这是很容易的事情。为了避免这种行为，请在游戏开始时使用 randomise()在你的游戏开始时使用。这只是在测试和debugging ，因为最终的可执行包不会显示这种行为，每次游戏都是随机的。

语法。

move_random(hsnap, vsnap);

争论类型 描述
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

返回。

例子。

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

当按下空格键时，这将把实例移动到room 中的任意位置。
