运动_添加

这个函数将修改运行代码的实例的当前 direction 和 speed ，将新的值添加到当前值中。如果你想简单地改变这些值，你应该使用函数 motion_set() 。

语法。

motion_add(dir, speed);

争论类型 描述
dir The added direction.
speed The added speed.

返回。

N/A

例子。

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

上述代码将在与另一个object 碰撞事件中被调用。它在运动方向和实例的速度上添加一个基于参与碰撞的其他实例的位置和速度的矢量。如果速度超过8（每步像素），它就会限制速度。
