向点移动

这个函数简单地告诉运行代码的实例以一个设定的速度向一个设定的点移动。请记住，你要告诉它到达目的地后该怎么做，因为它不会自己停下来。这意味着你可以得到一个 "振动 "的实例，因为它向前移动（例如）3个像素，超出X/Y目标位置2个，然后再向后移动3个像素，超出目标点1个，如此反复......有很多方法可以避免这种情况，比如说用 distance_to_point()来查看实例在移动前离指定位置有多远。

注意：这个函数设置了实例的速度，所以即使你停止使用这个代码，实例也会继续朝以前的方向移动，所以如果你希望实例停止，有必要将其设置为 speed为0，如果你希望它停止。请看下面的代码示例。

语法。

move_towards_point( x, y, sp );

争论 类型 描述
x 真实的 The x position of the point to move towards.
y 真实的 The y position of the point to move towards.
sp 真实的 The speed to move at in pixels per step.

返回。

N/A

例子。

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

上述代码将以每步5像素的速度向变量 "目标 "中索引的实例位置移动。一旦它到达该位置，速度就被设置为0。
