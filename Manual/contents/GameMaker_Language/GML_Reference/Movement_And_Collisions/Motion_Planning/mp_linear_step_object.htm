阶梯对象(mp_linear_step_object)

通过这个函数，你可以告诉一个实例向一个特定的点迈出 "一步"，由xgoal和ygoal值指定。步骤的大小（也就是实例每一步应该移动多少像素）由stepsize表示，如果实例已经到了这个位置，它就会停止，不再继续移动，这与其他更简单的函数相反，如 move_towards_point().stepsize也是object ，每一步都会检查碰撞的 "前方 "距离，你可以选择实例是否在与object 或你选择的实例发生碰撞时停止。

语法。

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

争论 类型 描述
xgoal 真实的 The target x position.
ygoal 真实的 The target y position.
stepsize 真实的 The speed the instance moves in pixels per step.
obj 对象资产或实例ID The object that is to block the path. Can be an object index, an instance id or the special keyword, all

返回。

Boolean

例子。

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上述代码将使object ，以每步5像素的速度向鼠标移动，只检查与object "obj_Wall "的碰撞情况。一旦它到达鼠标的位置，它将创建一个object "obj_Explosion "并自我销毁。
