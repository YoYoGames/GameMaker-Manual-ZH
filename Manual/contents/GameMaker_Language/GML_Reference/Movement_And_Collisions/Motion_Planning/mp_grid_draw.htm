mp_grid_draw

这个函数将画出指定的MP网格（如由 mp_grid_create())，将自由单元标记为绿色，禁止单元标记为红色。这个函数作为一个调试工具是必不可少的，但应该注意的是，它非常慢，而且只有在实例的draw事件中使用时才起作用，你可以设置draw alpha来改变网格的不透明度，允许你把它作为一个覆盖物来画，同时看到room 中的实际内容。

语法。

mp_grid_draw(id);

争论 类型 描述
id MP网格ID Index of the mp_grid that is to be drawn

返回。

Boolean

例子。

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

上面的代码会把变量 "grid "中索引的mp_grid画成半透明的覆盖层（但只有当运行代码的实例的深度低于其他所有实例时）。
