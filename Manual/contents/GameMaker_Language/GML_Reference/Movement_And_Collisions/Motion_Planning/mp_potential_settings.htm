潜在设置

mp_potential_ 函数使用一些参数来完成它们的工作，这些参数可以通过这个函数来改变。总的来说，该方法的工作原理如下：它首先尝试直接向目标移动，向前看几步，这可以通过参数 "前进 "来设置（默认值为3）。减少这个值意味着实例会晚些开始改变方向，因为它没有 "看到 "那么远的前方，而增加这个值意味着它将提前开始改变方向。如果这个检查导致了碰撞，那么它就会开始查看理想的直线方向的左边和右边的方向。它以 "rotstep "的大小（默认值是10）来做这个动作，减少这个大小会给实例带来更多的运动可能性，但会更慢，因为它需要更多的处理能力。

参数 "maxrot "有点难以解释!实例有一个当前的运动方向，而maxrot（默认值是30）表示它在一个步骤中允许在当前方向的任何一侧改变多少方向。因此，即使它可以直接向目标移动，它也只能在不违反这个最大的方向变化的情况下这样做。如果你让maxrot成为一个大值，那么实例就可以在每一步中改变很多方向，这将使它更容易找到一个短的path ，但是path ，会更难看，更不自然。如果你让这个值变小，那么path 会更平滑，但它可能会走更长的弯路（有时甚至找不到目标）。最后，当不能进行任何步骤时，行为取决于参数 "onspot "的值。如果onspot为真（默认值），那么实例将在其位置上按 "maxrot "指示的数量进行旋转，如果为假，则根本不移动。设置为 "false "对于汽车等来说是很有用的，但是会降低实例找到path 。

语法。

mp_potential_settings(maxrot, rotstep, ahead, onspot)

争论 类型 描述
maxrot 真实的 The number of degrees either side of the current direction that the instance can rotate in a step
rotstep 真实的 The number of degrees either side of the current direction that the instance can check for a collision
ahead 真实的 The number of steps ahead that the instance can check for a collision. Larger values are slower than lower ones.
onspot 布尔型 Use this to allow the instance to rotate on the spot when no path is found (true) or not (false)

返回。

N/A

例子。

mp_potential_settings(45, 5, 5, 0);

上述代码将设置 mp_potential_ ，使用以下设置：实例可以向前看五步，每步改变方向45度，如果有障碍物，则在当前方向的任何一侧看5度，在有碰撞的地方不旋转。
