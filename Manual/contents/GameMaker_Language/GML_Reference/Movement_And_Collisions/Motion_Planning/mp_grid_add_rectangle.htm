mp_grid_add_rectangle

这个函数要求你在room 内定义一个矩形，然后将所有 "接触 "该矩形的MP网格单元标记为禁区，这意味着路径搜索函数不能穿过它们。下面的图片说明了这是如何工作的。

正如你所看到的，由（50，90）到（200，180）定义的矩形将它所触及的所有等效的MP网格单元都标记为禁止。

语法。

mp_grid_add_rectangle(id, x1, y1, x2, y2);

争论 类型 描述
id MP网格ID Index of the mp_grid that is to be used
x1 真实的 The x coordinate of the left side of the rectangle to check.
y1 真实的 The y coordinate of the top side of the rectangle to check.
x2 真实的 The x coordinate of the right side of the rectangle to check.
y2 真实的 The x coordinate of the bottom side of the rectangle to check.

返回。

N/A

例子。

mp_grid_add_rectangle(grid, 0, 0, 100, 200);

上述代码将把变量 "grid "中索引的mp_grid的所有单元格标记为禁区，这些单元格位于0,0到100,200区域内。
