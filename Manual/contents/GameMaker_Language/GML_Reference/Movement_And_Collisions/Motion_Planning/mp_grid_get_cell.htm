mp_grid_get_cell

通过这个函数，你可以检查mp_grid的任何给定的单元格，看它是否被标记为占用，给出mp_grid的索引（先前创建的）和要检查的单元格的x和y坐标。

如果它已经被占用或被检查的位置超出了网格的范围，该函数将返回-1，否则将返回0。

语法。

mp_grid_get_cell(id, x , y);

争论 类型 描述
id MP网格ID Index of the mp_grid that is to be used
x1 真实的 The x coordinate of the grid to check.
y1 真实的 The y coordinate of the grid to check.

返回。

Real

例子。

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

上述代码将检查与鼠标位置相对应的mp_grid单元格，如果它被占用，则将 image_blend 变量设置为红色，如果它没有被占用，则设置为绿色。
