潜在的阶梯对象(mp_potential_step_object)

这个函数让实例向由xgoal/ygoal定义的特定位置迈出一步，同时试图避开障碍物。当实例遇到由 "obj "参数指定的object ，它将改变运动方向，试图避开该实例并绕过它。这种方法不能保证有效，但在大多数情况下，它将有效地使实例向目标移动。该函数返回是否达到了目标。

语法。

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

争论 类型 描述
xgoal 真实的 The target x position.
ygoal 真实的 The target y position.
stepsize 真实的 The speed the instance moves in pixels per step.
obj 对象资产或实例ID The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

返回。

Boolean

例子。

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

上述代码首先检查room 中是否有 "obj_Enemy "的实例。如果有，它就找到最近的一个，并将其id存储在一个变量中。然后，这个变量被用来告诉 mp_potential_step_object ，以每步5像素的速度向找到的object 的x/y位置移动实例，同时只避开object "obj_Wall "的实例。
