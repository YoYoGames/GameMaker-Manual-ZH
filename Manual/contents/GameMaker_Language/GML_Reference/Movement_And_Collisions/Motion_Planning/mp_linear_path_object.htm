mp_linear_path_object

这个函数计算从当前实例位置到 "xgoal "和 "ygoal "值所指定的点的直线path 。它使用指定的步长，与函数中的 mp_linear_step().指定的path 必须已经存在，并将被新的path 覆盖，该函数将返回是否找到完整的path （真）或不（假）。只有在与指定的path 或实例没有碰撞的情况下，才会发现一个完整的object ，如果返回false，那么仍然会生成一个path ，但它只会运行到path 被阻止的位置。

注意：这个函数不会移动实例。它只设置一个path ，你必须使用路径函数来移动。

语法。

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

争论 类型 描述
path 路径资产 The index of the path to be used
xgoal 真实的 The target x position.
ygoal 真实的 The target y position.
stepsize 真实的 The speed the instance moves in pixels per step.
obj 对象资产或实例ID The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

返回。

Boolean

例子。

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

上述代码沿着运行代码的path 和鼠标的x/y位置之间的object ，检查是否与 "obj_Wall "发生碰撞。如果没有碰撞并且生成了完整的path ，那么它将沿着生成的object 开始path 。
