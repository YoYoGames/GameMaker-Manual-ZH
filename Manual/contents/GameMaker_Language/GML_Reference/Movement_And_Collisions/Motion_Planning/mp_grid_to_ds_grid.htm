mp_grid_to_ds_grid

通过这个函数，你可以把给定的MP网格复制到DS网格中。DS网格应该与MP网格的大小相同，尽管它不一定是这样的（如果它更小，数据就会丢失，如果它更大，所有额外的网格单元都是0）。如果一个单元格在MP网格中被标记为占用，那么DS网格中的单元格将包含数值-1，如果没有，则为0。

语法。

mp_grid_to_ds_grid(source, destination);

争论 类型 描述
source MP网格ID Index of the mp_grid that is to be used
destination DS网格ID Index of the ds_grid that is to be used to copy the grid data to.

返回。

Boolean

例子。

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

上述代码将创建一个新的DS网格，然后将变量 "mp_grid "中包含的MP网格数据复制到新的DS网格中。
