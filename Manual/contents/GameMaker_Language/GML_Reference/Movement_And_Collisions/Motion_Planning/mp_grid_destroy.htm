mp_grid_destroy

你可以使用这个函数来销毁指定的mp_grid（如函数所定义的 mp_grid_create()）并释放分配给它的内存。每当MP网格完成后，你必须调用这个函数，否则你有可能得到一个内存泄漏，这意味着你的游戏会随着时间的推移而变慢，最终崩溃。

注意：在网格被销毁后，在网格上使用 mp_grid_* 功能会导致错误。

语法。

mp_grid_destroy(id);

争论 类型 描述
id MP网格ID Index of the mp_grid that is to be destroyed

返回。

N/A

例子。

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

如果变量 "计时器 "等于0，上述代码将销毁在变量 "grid "中索引的mp_grid，然后转入另一个room 。
