mp_grid_clear_all

通过这个功能，你可以清除一个MP网格中所有 "禁止 "的单元。

语法。

mp_grid_clear_all(id);

争论 类型 描述
id MP网格ID Index of the mp_grid that is to be used

返回。

Boolean

例子。

if !instance_exists(obj_Player)
{
    mp_grid_clear_all(grid);
}

如果object "obj_Player "的实例不再存在于room ，上述代码将清除变量 "grid "中索引的mp_grid，将所有单元格标记为空闲。
