mp_grid_clear_cell

这个函数可以用来清除MP网格的一个特定 "单元"。单元不是作为room ，而是作为网格坐标计算的，其中(0,0)是网格的顶角。这意味着要清除room 中特定位置的单元，我们必须将x和y坐标改为单元坐标，并将其除以MP网格的分辨率。下面的代码例子显示了这是如何工作的。

语法。

mp_grid_clear_cell(id, h, v);

争论 类型 描述
id MP网格ID Index of the mp_grid that is to be used
h 真实的 Horizontal position of the cell to clear
v 真实的 vertical position of the cell to clear

返回。

Boolean

例子。

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

上面的代码将使所有的 "obj_Box "自我销毁，并让它们把它们在变量 "grid "中索引的mp_grid中占据的单元格标记为空闲。在这个例子中，我们通过获取object 的x/y坐标并除以网格的分辨率（使用 floor ，以保持数值为整数）来找到合适的单元。
