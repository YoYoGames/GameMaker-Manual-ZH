位置_变化

这个函数将检查一个位置是否与给定点的任何实例发生碰撞，如果有的话，它将把所有发生碰撞的实例改为所选择的实例object 。你可以将 "perf "参数设置为 true ，这将迫使GameMaker执行找到的实例的Destroy和Clean Up事件，以及新实例的Create事件，或者 false ，不执行任何这些事件。请注意，如果你选择不执行Destroy、Clean Up和Create事件，任何使用通常在create事件中定义的变量而创建的实例将使游戏崩溃，因为该变量将不再存在。

语法。

position_change(x, y, obj, perf);

争论 类型 描述
x 真实的 The x coordinate of where to change colliding instances.
y 真实的 The y coordinate of where to change colliding instances.
obj 对象资产 The new object the calling object will change into.
perf 布尔型 Whether to perform that new object's Create event (true) or not (false).

返回。

N/A

例子。

position_change(200, 200, obj_Bird, true);

这将把所有在(200,200)点碰撞的实例变成 "obj_Bird "的实例，在这个过程中为每个实例执行 "obj_Bird "的创建事件。

返回。碰撞
