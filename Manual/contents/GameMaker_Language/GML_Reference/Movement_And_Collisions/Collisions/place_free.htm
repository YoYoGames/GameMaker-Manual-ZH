place_free

你可以使用这个函数来检查并查看调用的实例是否会与你游戏中任何标记为实体的实例发生碰撞。现在，应该注意的是，为了让这个函数起作用，运行该代码的实例必须有一个有效的碰撞掩码（无论是为sprite ，还是通过 mask_index)，并且它将只登记与那些也有有效掩码的实体标记实例的碰撞。

该函数本身的工作原理是，当把实例放在x/y参数指定的位置时，只测试与实体标记的实例的碰撞。碰撞检查可以是精确的，也可以是基于实例的边界框的，这取决于选择了什么样的碰撞掩码。

注意，在进行检查之前，给定的x/y坐标将被浮动到最接近的整数。

语法。

place_free(x, y);

争论类型 描述
x The x position to check.
y The y position to check.

返回。

例子。

if place_free(mouse_x, mouse_y)
{
    x = mouse_x;
    y = mouse_y;
}

如果调用的实例被放置在与鼠标相同的位置，上述代码将检查是否与任何实体标记的实例发生碰撞。如果没有检测到碰撞，那么该实例的x/y坐标就会被设置为鼠标的坐标。

返回。碰撞
