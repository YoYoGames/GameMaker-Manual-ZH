地方_空

你可以使用这个函数来检查并查看调用的实例是否会与object 的任何其他实例或你游戏中的所有实例发生碰撞。为了正确解决碰撞问题，运行该代码的实例必须有一个有效的碰撞掩码（无论是对sprite 本身，还是通过 mask_index)，并且它将只登记与那些也有一个有效掩码的实例的碰撞。

该函数正在测试如果调用的实例被放置在一个特定的位置上是否有碰撞，你可以提供一个可选的参数来细化检查，只检查一个位置是否与指定类型的实例没有碰撞。注意，如果没有提供可选的object ID，检查将针对room 内的所有实例进行。碰撞检查将是精确的，或者基于实例的边界框，这取决于选择了什么样的碰撞掩码，但是为了使精确的碰撞正常工作，检查中的所有实例应该有精确的碰撞掩码。

注意，在进行检查之前，给定的x/y坐标将被浮动到最接近的整数。

语法。

place_empty(x, y, [object_id]);

争论 类型 描述
x 真实的 The x position to check.
y 真实的 The y position to check.
[object_id] 对象资产 OPTIONAL 要检查的对象。

返回。

Boolean

例子。

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
    x = mouse_x;
    y = mouse_y;
}

上述代码将检查是否与任何其他object "obj_Enemy "的实例发生碰撞，就像调用的实例被放置在与鼠标相同的位置。如果没有检测到碰撞，那么该实例的x/y坐标就会被设置为鼠标的坐标。

返回。碰撞
