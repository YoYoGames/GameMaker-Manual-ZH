职位_会议

object通过这个函数，你可以检查一个位置是否与另一个实例或一个room 的所有实例发生碰撞。当你使用这个函数时，你是在检查一个实例或一个object 中的一个点。这个检查将针对该实例的边界框进行，如果该实例有精确碰撞检查，则针对该实例的掩码。如果你需要获得唯一的实例 id如果被碰撞的object ，你应该使用 instance_position().这个函数也接受特殊关键字 all和other。

语法。

position_meeting(x, y, obj);

争论 类型 描述
x 真实的 The x position to check.
y 真实的 The y position to check.
obj 对象资产或实例ID The object (or instance id, or the keywords "all" or "other") to check for a collision with.

返回。

Boolean

例子。

if mouse_check_button(mb_left)
{
    if (!position_meeting(mouse_x, mouse_y, all))
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

上面的代码检查鼠标左键，如果它被按下，它就检查鼠标的X/Y位置是否与任何实例发生碰撞。如果没有，那么就会创建一个 "obj_Wall "的实例。

返回。碰撞
