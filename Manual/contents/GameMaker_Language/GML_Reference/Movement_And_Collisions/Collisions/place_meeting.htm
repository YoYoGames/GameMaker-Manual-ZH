place_meeting

通过这个函数，你可以使用运行检查代码的实例的碰撞掩码来检查一个位置是否与另一个实例或一个object 的所有实例发生了碰撞。当你使用这个功能时，你实际上是要求GameMaker将实例移动到新的位置，检查是否有碰撞，然后移动回来，并告诉你是否发现了碰撞。这对精确的碰撞是有效的，但只有当实例和被检查的object ，都选择了精确的碰撞掩码，否则只应用边界框碰撞。这个函数也接受特殊关键字all和other。请注意，如果你需要获得被碰撞的object 的唯一实例ID，你应该使用 instance_place().

注意，在进行检查之前，给定的x/y坐标将被浮动到最接近的整数。

语法。

place_meeting(x, y, obj);

争论 类型 描述
x 真实的 The x position to check.
y 真实的 The y position to check.
obj 对象资产或实例ID The instance or object to check for.

返回。

Boolean

例子。

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

上述代码检查实例左侧是否有碰撞，如果没有则移动实例。

返回。碰撞
