碰撞线

Collision_line 沿着从点x1,y1到点x2,y2的直线检查是否与参数 "obj "指定的 的object 任何实例发生碰撞。这种检查可以是精确的，也可以不是，但是要启用精确碰撞，你要检查的 或实例也object 必须为其 启用精确碰撞。如果不是，默认检查是基于边界框的。下面的图片说明了这是如何工作的。sprite

记住，为了考虑精确碰撞，object sprite 和碰撞函数都必须将精确标记为开启。还应该注意的是，函数的返回值可以是沿线被认为是碰撞的任何一个实例的id，所以如果有三个实例与定义的线重叠，它们中的任何一个id都可以成为函数的返回值。

注意，你可以提供一个实例来代替object 索引，以检查特定的实例。 id来检查一个特定的实例，或者实例关键字 all ，或者 other （取决于事件和当前的代码范围）。

语法。

collision_line(x1, y1, x2, y2, obj, prec, notme);

争论 类型 描述
x1 真实的 The x coordinate of the start of the line.
y1 真实的 The y coordinate of the start of the line.
x2 真实的 The x coordinate of the end of the line.
y2 真实的 The y coordinate of the end of the line.
obj 对象资产或实例ID The object to check for instance collisions.
prec 布尔型 Whether the check is based on precise collisions (true，速度较慢）或其一般的边界盒（false ，速度较快）。
notme 布尔型 Whether the calling instance, if relevant, should be excluded (true)或不(false)。

返回。

Instance ID or noone

例子。

with (obj_Enemy)
{
    if collision_line(100, 400, 100, 600, id, false, false) instance_destroy();
}

这段代码让 "obj_Enemy "的所有实例沿着100x，400y到100x，600y的线检查是否与自己相撞，如果有的话，它们就被销毁。

返回。碰撞
