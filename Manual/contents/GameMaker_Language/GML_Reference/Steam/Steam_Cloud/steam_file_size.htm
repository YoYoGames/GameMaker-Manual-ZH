蒸汽文件大小

通过这个函数，你可以检查存储在Steam 云中的文件的大小。返回的实数是该文件的大小，单位是字节。

语法。

steam_file_size(filename);

争论 描述
filename 要检查大小的文件名称。

返回。

Real

例子。

file_bytes = steam_file_size("Save.txt");

上述代码在变量 "file_bytes "中存储了来自Steam 云的文件的大小。

返回。蒸汽云
