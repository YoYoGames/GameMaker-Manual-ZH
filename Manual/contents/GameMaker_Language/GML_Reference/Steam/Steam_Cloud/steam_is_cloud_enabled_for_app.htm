蒸汽_是云启用的应用程序

通过这个函数，你可以检查是否为你的游戏启用了Steam 云服务。如果是，它将返回true，否则返回false。switch 请注意，这并不自动意味着你可以使用云功能，因为用户可以从他们的Steam 客户端关闭云同步功能。你可以使用函数来检查这一点 steam_is_cloud_enabled_for_account()但是，即使对用户来说关闭了（对游戏来说启用了），这些功能仍然可以从所有文件的本地副本中存储和检索数据，只是在游戏结束时不会将它们上传到云端，也不会在游戏开始时进行同步。

语法。

steam_is_cloud_enabled_for_app();

返回。

Boolean

例子。

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_total();
}

上面的代码检查是否为游戏启用了steam 云，如果是，它就会获得存储配额的大小，并将其存储在一个变量中。

返回。蒸汽云
