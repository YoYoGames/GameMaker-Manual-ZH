帐户是否启用了云计算？

通过这个函数，你可以检查以确保用户在他们的Steam 客户端设置中启用了Steam 云服务。如果是，它将返回true，否则返回false。请注意，这并不自动意味着你可以将数据存储到云端，因为它还必须为你的游戏启用（你可以使用函数检查这一点 steam_is_cloud_enabled_for_app()).如果你的游戏启用了Steam 云，但用户在本地关闭了它，你仍然可以使用云功能来存储和检索所有文件的本地副本的数据，只是不会在游戏结束时将它们上传到云，也不会在游戏开始时进行同步。

语法。

steam_is_cloud_enabled_for_account();

返回。

Boolean

例子。

if steam_is_cloud_enabled_for_account
{
    steam_file_share("Save.txt");
}

上面的代码检查用户是否启用了Steam 云，如果返回为真，它就会同步给定的文件。

返回。蒸汽云
