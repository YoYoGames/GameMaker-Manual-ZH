蒸汽_获取配额_自由

通过这个函数，你可以了解到Steam Cloud配额的用户还有多少自由空间。返回的值是以字节为单位。

语法。

steam_get_quota_free();

返回。

Real

例子。

if steam_is_cloud_enabled_for_app()
{
    quota = steam_get_quota_free();
}

上面的代码检查是否为游戏启用了steam 云，如果是的话，它就会获得自由存储空间的大小并将其存储在一个变量中。

返回。蒸汽云
