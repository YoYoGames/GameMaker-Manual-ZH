删掉文件

这个函数将从Steam 云中删除给定的文件。如果由于某种原因失败，该函数将返回一个0的值，如果成功则返回一个大于0的值。

语法。

steam_file_delete(filename);

争论 描述
filename 文件删除的名称。

返回。

Real

例子。

if steam_file_exists("Save.txt")
{
    steam_file_delete("Save.txt");
}

上面的代码将检查一个文件是否存在，如果存在，它将从Steam 云中删除该文件。

返回。蒸汽云
