蒸汽文件共享（steam_file_share

通过这个功能，你可以强制你的游戏将给定的文件与Steam 云同步。由于游戏将在玩家会话结束时自动同步，这通常不是必要的，也不是Steam ，但它对于确保敏感信息被立即同步很有用。该函数如果因为任何原因而失败，将返回一个0的值，如果成功，将返回一个大于0的值。

语法。

steam_file_share(filename);

争论 描述
filename 同步的文件的名称。

返回。

Real

例子。

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

上面的代码将检查一个文件是否已经存储到Steam 云中，如果没有，它就会同步它。

返回。蒸汽云
