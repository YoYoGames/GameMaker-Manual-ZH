蒸汽文件阅读

这个函数将把给定文件的内容读入一个string ，以后可以在你的游戏中进行解析。

语法。

steam_file_read(filename);

争论 描述
filename 要读取的文件的名称。

返回。

String

例子。

if steam_file_exists("Save.txt")
{
    save_str = steam_file_read("Save.txt");
}

上面的代码检查Steam 云上是否存在一个文件，如果存在，就打开它并将其内容读入变量 "save_str"。

返回。蒸汽云
