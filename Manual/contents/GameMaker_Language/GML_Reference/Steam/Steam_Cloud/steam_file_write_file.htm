蒸汽_文件_写文件

通过这个函数，你可以将本地保存的文件的内容复制到与Steam 云同步的文件中。在使用这个函数之前，本地文件必须存在，如果由于某种原因失败，它将返回一个0的值，如果成功，则返回一个大于0的值。 如果成功，则返回一个大于0的值。

语法。

steam_file_write_file(steam_filename, local_filename);

争论 描述
steam_filename Steam 云文件，以复制过来。
local_filename 用来复制的本地文件。

返回。

real

例子。

steam_file_write_file("rm_koala.png", "Koala2.png");

上述代码将把文件 "Koala2.png "的内容复制到Steam 云文件 "rm_koala.png "中。

返回。蒸汽云
