蒸汽文件的使用情况

通过这个函数，你可以检查给定的文件，看它是否已经与Steam 云同步。返回值为 "true"意味着它是，而"false"意味着它不是。

语法。

steam_file_persisted(filename);

争论 描述
filename 要检查的文件的名称。

返回。

Boolean

例子。

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

上面的代码将检查一个文件是否已经存储到Steam 云中，如果没有，它就会同步它。

返回。蒸汽云
