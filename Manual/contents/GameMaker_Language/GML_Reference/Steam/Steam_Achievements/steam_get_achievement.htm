蒸汽_获取成就

通过这个函数，你可以检查Steam API，看看是否已经获得了某个特定的成就。该成就应该已经在你的游戏的Steamworks控制面板账户页面上定义过，传递给该函数的string 应该与控制面板上的API名称一致。

语法。

steam_get_achievement(ach_name);

争论 描述
ach_name 要获得的成就名称(string)。

返回。

Boolean

例子。

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

如果全局变量 "死亡数 "等于10，并且成就还没有被授予，那么上述代码将奖励玩家一个成就。

返回。成就和统计数据
下一步。 steam_clear_achievement
