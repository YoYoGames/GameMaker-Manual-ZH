蒸汽数据统计表

通过这个函数，你可以得到一个特定的浮点统计值。这个统计值应该已经在你的游戏的Steamworks控制面板账户页面上定义过了，传递给函数的string ，应该与控制面板上的API名称相匹配。

语法。

steam_get_stat_float(stat_name);

争论 描述
stat_name 要得到的统计数字的名称（一个string ）。

返回。

Real

例子。

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
{
    steam_set_stat_int("Travelled", dist_pc);
}

上面的代码根据旅行的距离变量 "dist "和你能旅行的最大距离 "dist_max "计算出一个百分比。然后，它调查统计数字 "Travelled "的当前值，如果它小于计算值，它将再次设置统计数字。

返回。成就和统计数据
