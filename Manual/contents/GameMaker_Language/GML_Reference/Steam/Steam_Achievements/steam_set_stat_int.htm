蒸汽_设置统计表_int

通过这个函数，你可以将一个特定的统计值设置为一个新的、有符号的整数值。这个统计值应该已经在你的游戏的Steamworks控制面板账户页面上定义过了，而且传递给这个函数的string 应该与控制面板上的API名称一致。你可以用它来跟踪玩家死亡的次数或跟踪成就的进展。

语法。

steam_set_stat_int(stat_name, value);

争论 描述
stat_name 要设置的统计数字的名称（一个string ）。
value 要设置统计的值（一个32位的整数）。

返回。

N/A

例子。

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

上面的代码设置了一个统计量，然后检查它的最终值，以决定是否授予成就。

返回。成就和统计数据
