蒸汽_设置_成就

通过这个函数，你可以告诉Steam API，为玩家授予（"设置"）成就。这些成就应该在你的游戏的Steamworks控制面板账户页面上定义，传递给该函数的string 应该与控制面板上的API名称一致。Steam Game Overlay将向用户显示一个通知面板，告知他们所获得的成就，除非该成就已经被授予，在这种情况下不会发生任何事情。

语法。

steam_set_achievement(ach_name);

争论 描述
ach_name 要设置的成就的名称(string)。

返回。

N/A

例子。

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

如果全局变量 "死亡数 "等于10，并且成就还没有被授予，那么上述代码将奖励玩家一个成就。

返回。成就和统计数据
