激活覆盖的用户

这个函数将打开Steam ，覆盖到与给定的用户ID有关的一个选定的对话中。可能的对话strings ，包括。

  • "steamid" - 打开Steam 社区网页浏览器，进入该用户或组的页面。
  • "聊天" - 向指定的用户打开一个聊天窗口。

请注意，Steam ID可能是大数字，因此你可能需要将你的ID值铸成 int64()然后再把它提供给函数。

语法。

steam_activate_overlay_user(dialog_name, steamid);

争论 描述
dialog_name 打开覆盖层的对话（见下文）。
steamid 要使用的Steam 用户ID或组ID。

返回。

N/A

例子。

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

上述代码调查最后一次按下的键盘键，如果是功能键1或功能键2，它将打开Steam 覆盖，以查看存储在全局变量 "GamegroupID "中的Steam 组，或者它将打开聊天窗口，与存储在全局变量 "FriendID "中的用户聊天。
