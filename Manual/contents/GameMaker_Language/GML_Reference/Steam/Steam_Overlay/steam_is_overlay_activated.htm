激活了吗？

这个函数可以用来发现用户是否激活了Steam Overlay。如果覆盖层是激活的并且对用户可见，该函数将返回true，如果不是，则返回false。这个函数的一个例子是用于轮询Steam API的覆盖层，这样你就可以在显示覆盖层的时候暂停你的游戏。

语法。

steam_is_overlay_activated();

返回。

Boolean

例子。

if steam_is_overlay_activated()
{
    global.Pause = true;
}

上面的代码将检查Steam 覆盖是否激活，如果是，它将把全局变量 "Pause "设置为true。
