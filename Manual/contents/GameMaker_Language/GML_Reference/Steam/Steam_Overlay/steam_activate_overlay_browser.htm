激活叠加浏览器

通过这个功能，你可以打开Steam 游戏覆盖到它的网络浏览器，然后让它加载指定的URL。你需要使用完整的URL作为string ，这样才能正确解决，例如："http://www.steamgames.com/"。

语法。

steam_activate_overlay(url);

争论 描述
url 要打开的覆盖层的（完整）URL。

返回。

N/A

例子。

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_browser("http://www.steamgames.com/");
}

上述代码轮询键盘上的F1键，如果有的话，Steam 覆盖将被打开并解析到给定的URL。
