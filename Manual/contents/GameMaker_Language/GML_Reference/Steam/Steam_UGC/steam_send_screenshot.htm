发送截图

通过这个函数，你可以将截图上传到当前登录的用户的Steam 社区资料页面。你提供的文件名是你使用GameMaker Studio 2 函数拍摄屏幕截图时创建的本地文件的名称 screen_save()或 screen_save_part().宽度和高度定义了图片的大小，如果由于某种原因失败，该函数将返回一个0的值，如果成功则返回一个大于0的值。

语法。

steam_send_screenshot(filename, width, height);

争论 描述
filename 要上传的图像文件的名称（一个string ）。
width 图像的宽度（一个整数）。
height 图像的高度（一个整数）。

返回。

Real

例子。

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

上述代码将轮询Steam API的截图请求，如果有的话，将为图像文件生成一个唯一的名称，进行截图，并将该文件发送到用户的Steam 社区页面。

返回。UGC
