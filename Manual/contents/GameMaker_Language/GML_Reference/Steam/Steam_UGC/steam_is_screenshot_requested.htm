蒸汽_is_screenshot_requested

该函数将轮询Steam API，以查看用于拍摄游戏截图的键是否被按下。当按键被按下时，该函数只在一个步骤（游戏时间）中返回真，而在其他时候都会返回假。

请注意，如果截图键被按下，这个函数在按下的每个步骤中只返回一次 true ，而在同一步骤中的任何后续调用则返回 false 。例如，如果在当前帧中要求进行截图，并且你在步骤事件中调用该函数来发现这一点，你将得到 true ；但是，如果你在Draw GUI中再次调用该函数来检查是否要求进行截图，你将得到 false ，因为该函数在步骤事件中已经被 "用完"。如果要在同一帧中多次使用该函数的返回值，建议将其存储在一个变量中，然后读取该变量，而不是再次调用该函数。

注意：这个函数不会为你拍摄屏幕截图。这只是信号，表明按键已被按下，你必须使用GameMaker Studio 2 函数 screen_save()或 screen_save_part()来保存将要上传的文件的本地拷贝。

语法。

steam_is_screenshot_requested();

返回。

Boolean

例子。

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

上述代码将轮询Steam API的截图请求，如果有的话，将为图像文件生成一个唯一的名称，进行截图，并将该文件发送到用户的Steam 社区页面。

返回。UGC
