围绕用户下载的分数

这个函数用于检索相对于当前用户条目的排行榜条目。range_start参数是在当前用户条目之前要检索的条目数，range_end参数是当前用户条目之后的条目数。 当前用户的条目之后的条目，并且当前用户的条目总是包括在结果中。例如，如果当前用户在某个排行榜上排名第5，那么将开始范围设置为-2，结束范围设置为 2将返回5个条目。3到7。如果在用户的条目之前或之后的排行榜上没有足够的条目，Steam ，将调整范围，试图返回所要求的条目数量。例如，如果用户是排行榜上的第1名。 start被设置为-2，end被设置为2，Steam 将返回排行榜上的前5个条目。

语法。

steam_download_scores_around_user(lb_name, range_start, range_end);

争论 描述
lb_name 你要下载分数的排行榜名称(astring)。
range_start 在排行榜上的起始位置。
range_end 在排行榜上的最终位置。

返回。

Real

例子。

score_get = steam_download_scores_around_user("Game Scores", -4, 5);

这将向Steam Server ，请求从排行榜 "游戏分数 "中获取10个分数，以玩家为中心，并将请求的异步ID存储在变量 "score_get "中。然后，这将被 中处理。 蒸汽异步事件中处理，如扩展示例中所示 steam_download_scores().

返回。排行榜
