下载朋友们的分数

通过这个函数，你可以只检索排行榜上属于那些在Steam 客户端中被标记为 "朋友 "的人的分数。因此，如果你的排行榜有200个条目，而其中50个是你的朋友，这个函数 将只检索出这50个结果。排行榜的名称是一个string ，是在你使用函数创建排行榜时定义的 steam_create_leaderboard()时定义的，而该函数将返回 一个值，该值可用于识别Steam Async事件中的回调，如果失败，则返回-1。

语法。

steam_download_friends_scores(lb_name);

争论 描述
lb_name 你要下载分数的排行榜名称(astring)。

返回。

Real

例子。

score_get = steam_download_friends_scores("Game Scores");

这将向Steam Server 发送请求，以获取用户在排行榜上的好友分数，并将请求的异步ID存储在变量 "score_get "中。然后，这将在Steam的异步事件中处理。 的扩展示例中所示。 steam_download_scores().

返回。排行榜
