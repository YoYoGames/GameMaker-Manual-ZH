steam_get_user_persona_name

这个函数可以用来检索任何特定用户ID的用户名（屏幕名称）。你给出用户ID，然后该函数返回一个唯一的异步ID值，可以用来在Steam异步事件中正确检查细节。 这个事件将有 async_loadDS地图，它将由以下键/值对填充。

  • "id" - 触发事件的函数返回的异步ID
  • "event_type" -string "user_persona_name"
  • "steamid"" - 用来获取用户名的Steam ID值。
  • "persona_name" - 给定的Steam ID 的用户名。

语法。

steam_get_user_persona_name(steamID);

争论 描述
steamID 一个用户唯一的Steam ID。

返回。

Real

例子。

request = steam_get_user_persona_name(global.UGC_UserID);

上述代码将请求存储在全局变量 "UGC_UserID "中的用户ID的用户名，将返回值存储在一个变量中，以便在Async Event中解析。

返回。一般Steam API
