蒸汽_用户是否已登录？

如果Steam 客户端目前有一个与Steam servers 的实时连接，该函数将返回true。如果它返回false，这意味着由于本地机器的网络问题，或者Steam server 被关闭或繁忙，没有活动连接。

语法。

steam_is_user_logged_on();

返回。

String

例子。

if steam_is_user_logged_on()
{
    global.user_id = steam_get_user_account_id();
}

上面的代码将检查用户是否登录了Steam server ，如果是，则将用户ID存储在一个全局变量中。

返回。一般Steam API
