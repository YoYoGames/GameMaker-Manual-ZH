steam_get_user_account_id

这个函数用来检索Steam ，为每个用户分配的唯一的用户ID，这是使用一些用户生成的内容功能所需要的。

语法。

steam_get_user_account_id();

返回。

Real

例子。

global.user_id = steam_get_user_account_id();

上面的代码获得了拥有游戏的人的唯一用户ID，并将其存储在一个全局变量中。

返回。一般Steam API
