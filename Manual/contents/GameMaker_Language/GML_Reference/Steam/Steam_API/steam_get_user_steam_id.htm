steam_get_user_steam_id

你可以使用这个函数来返回当前登录到Steam 客户端的用户的唯一Steam 用户ID。这是一个唯一的64位值，可以用来识别单个用户，而不是他们在屏幕上的用户名（这可以通过函数找到 steam_get_persona_name()).由于这是一个int64，你可能需要在随后的函数中使用它，使用 int64()函数。

语法。

steam_get_user_steam_id();

返回。

Integer (int64)

例子。

if steam_initialised()
{
    global.u_id = steam_get_user_steam_id();
}

如果Steam 客户端API被正确初始化，上述代码将设置一个全局变量为当前用户唯一的Steam ID。

返回。一般Steam API
