一般Steam API

以下一组函数都是用来检查Steam 客户端或server API的某些方面的可用性，因此根据server 是否繁忙或用户是否登录等因素返回真或假。这意味着这些函数应该在任何其他Steam API函数调用之前使用，以确保客户端/服务器设置正确并与你的游戏进行通信。

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
