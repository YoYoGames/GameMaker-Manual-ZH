蒸汽用户拥有dlc

如果你的游戏有为其创建的DLC，你可以使用这个函数在访问与之相关的任何文件之前检查用户是否已经购买了它。Steam如果玩家拥有该内容，该函数将返回true(1)，如果他们不拥有该内容或给定的DLC ID无效，则返回false(0)，如果他们没有登录到-1 。

请注意，即使用户拥有DLC，他们也可能没有安装它，所以你应该另外使用函数 steam_user_installed_dlc()来确保它在使用前已经安装。请注意，Steam ID可能是大数字，因此你可能需要将你的ID值作为一个 int64()然后再提供给该函数。

语法。

steam_user_owns_dlc(dlc_id);

争论 描述
dlc_id 要检查的DLC的唯一标识（一个int64的整数）。

返回。

Integer

例子。

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

上述代码将检查用户是否购买并安装了ID为10354的DLC，如果是，则将一个全局变量设置为不同的值。

返回。DLC
