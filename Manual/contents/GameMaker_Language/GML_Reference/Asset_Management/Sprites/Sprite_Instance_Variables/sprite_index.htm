魉_指数

这个变量返回实例的当前sprite 的索引，如果实例没有与之相关的sprite ，则返回-1。你可以改变它，给实例一个不同的sprite ，方法是给它一个来自资源树的sprite 的名字，或者使用一个有外部加载的sprite 索引的变量。改变sprite 并不改变当前可见帧的索引，所以如果你改变了第3帧上的sprite ，新的sprite 将在该帧可见的情况下绘制（假设它有相同的帧数）。

语法。

sprite_index;

返回。

Sprite Asset

例子。

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

上述代码将loop "obj_Check "的所有实例，检查它们与运行代码的实例之间是否有碰撞线。这些实例的sprite ，将根据碰撞线的返回值（真或假）来改变。
