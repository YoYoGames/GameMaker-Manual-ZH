魉_xoffset

这个只读变量返回已经分配给实例的sprite 的本地xoffset（原点的x分量，在sprite 编辑器中定义）。这个xoffset是以像素为单位返回的，并且取决于 image_xscale所以（例如）如果一个32x32的正方形sprite ，被定义为x原点在16，当这个sprite 被缩放为2，这个变量将返回32。如果你需要未缩放的xoffset，你应该使用 sprite_get_xoffset().

语法。

sprite_xoffset;

返回。

Real

例子。

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

上面的代码检查实例的sprite 的左边缘是否在room 之外，如果是，它就移动实例，使整个sprite 可见。
