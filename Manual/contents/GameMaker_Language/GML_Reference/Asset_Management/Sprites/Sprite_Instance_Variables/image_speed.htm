图像_速度

这个变量决定了GameMaker在当前实例中循环播放子图像的速度，sprite 。给出的速度值是一个乘数，1是默认值，将其设置为0.5将使动画速度减半--如在Sprite Editor或Image Editor中设置的那样--而将其设置为2将使其加倍。如果使用的sprite ，没有子图像，这个变量将没有影响。

语法。

image_speed;

返回。

(single precision floating point value)

例子。

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

上面的代码创建了一个object "obj_Explosion "的实例，并将其image_speed设置为0.5（有效地将运行速度减半）。
