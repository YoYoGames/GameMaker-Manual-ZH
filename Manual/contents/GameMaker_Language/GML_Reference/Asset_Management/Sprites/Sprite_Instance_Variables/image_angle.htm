image_angle

这个值设置sprite 的角度（旋转），以度为单位，右边是0º，向上是90º，左边是180º，向下是270º。将此变量设置为0，可以重置sprite ，使其按照sprite 编辑器中的定义绘制。请注意，为了使这个变量的变化可见，这个实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用一个扩展的绘制函数，如 draw_self()或 draw_sprite_ext().

语法。

image_angle

返回。

Real

例子。

image_angle = point_direction(x, y, mouse_x, mouse_y);

上述代码将旋转实例的sprite ，使其始终指向鼠标的位置。
