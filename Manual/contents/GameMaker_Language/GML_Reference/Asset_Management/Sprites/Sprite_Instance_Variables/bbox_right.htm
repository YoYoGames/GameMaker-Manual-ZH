bbox_right

这个只读变量返回实例的边界框右侧在room （沿X轴）内的位置，其中边界框是由实例的遮罩的最大宽度和高度定义的（由sprite_index 或 mask_index ）。即使sprite 有一个精确的碰撞掩码，边界框也是存在的，并被用于某些事情，因此你可以用这个变量来找到它。请注意，当实例没有指定sprite ，返回的值将与实例的X位置相同。

语法。

bbox_right;

返回。

(integer)

例子。

if bbox_right > room_width
{
    x = room_width - sprite_xoffset;
}

上述代码检查实例的边界框是否在room ，如果是，则将实例设置到一个新的位置。
