Image_yscale

这个值设置了应用于分配给当前实例的sprite 的垂直比例（沿y轴）。1表示不缩放（1:1），较小的值将会缩小（例如0.5，将是sprite 高度的一半），较大的值将会放大，负值将会镜像sprite ，并将其缩放，除非使用的值正好是-1（在这种情况下，sprite 只是沿y轴镜像，没有缩放）。

语法。

image_yscale;

返回。

(single precision floating point value)

例子。

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上面的代码缩放了sprite ，然后一旦它被缩放到原始大小的5倍，就会创建另一个object 的新实例并销毁该实例。
