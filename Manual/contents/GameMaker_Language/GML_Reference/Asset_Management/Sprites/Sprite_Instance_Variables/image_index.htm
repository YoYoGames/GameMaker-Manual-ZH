图像_索引

一个sprite 是由一个或多个子图像组成的，当它们从一个sprite 到另一个时，可以使switch 出现动画，或者它们可以在代码中切换，以提供不同的 "状态"，就像窗口中的按钮一样。如果sprite 是动画的，那么你可以通过检查 image_index 变量来获得动画的当前帧，或者如果你想改变静态sprite 的状态，你可以通过将这个变量设置为所需的sprite 的子图像来选择一个新的子图像。 请注意，为了使这个变量的变化可见，这个实例应该没有绘制事件（因此GameMaker将默认绘制sprite ）或者使用某个绘制函数来绘制，例如 draw_self()或 draw_sprite_ext()(通过在适当的参数中提供image_index)。

请注意，将此变量直接设置为一帧，不会触发该帧上可能存在的任何广播信息。

还要注意的是，在使用骨骼动画sprites ，你仍然可以获得和设置 image_index 值--见函数 skeleton_animation_get_duration()以了解如何做到这一点的例子。

语法。

image_index;

返回。

(single precision floating point value)

例子。

if (image_speed > 0)
{
    if (image_index >= image_number - 1) instance_destroy();
}

上面的代码检查sprite 是否在做动画，如果是，则检查当前image_index是否在最后一帧，在这种情况下，就会销毁这个实例。
