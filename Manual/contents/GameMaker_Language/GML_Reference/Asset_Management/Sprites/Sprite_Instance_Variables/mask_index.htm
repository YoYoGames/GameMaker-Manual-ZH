掩码_索引

当你在GameMaker中定义一个object ，你可以给它分配一个掩码，用于碰撞，而不是对应于定义的sprite 。这个变量可以用来查找 sprite_index的掩码（或者如果没有指定sprite ，它将返回-1），或者将一个实例的掩码设置为所选择的sprite 。设置掩码索引意味着你可以有，例如，一个不规则形状的实例的sprite ，但给它一个圆形的碰撞掩码，这是从不同的sprite 。

语法。

mask_index;

返回。

Sprite Asset

例子。

mask_index = spr_Round;

上面的代码将实例的掩码设置为sprite "spr_Round "的掩码。
