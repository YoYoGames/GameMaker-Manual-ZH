魉的宽度

这个只读变量返回已经分配给实例的sprite 的宽度。这个宽度是以像素为单位返回的，并且取决于 image_xscale.如果你需要未缩放的宽度，你应该使用 sprite_get_width().

语法。

sprite_width;

返回。

Real

例子。

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

上述代码将分配给运行该代码的实例的sprite 的宽度与sprite 资源的宽度进行核对，如果不一致，就将 image_xscale 重置为1。
