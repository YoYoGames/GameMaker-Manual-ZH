鼠标左键

这个只读变量返回已经分配给实例的sprite 的本地yoffset（即在sprite 编辑器中定义的原点的y分量）。这个yoffset是以像素为单位返回的，并且取决于 image_yscale因此，（例如）如果一个32x32的正方形sprite ，被定义为y原点在16，当这个sprite 被缩放为2时，这个变量将返回32。如果你需要未缩放的yoffset，你应该使用 sprite_get_yoffset().

语法。

sprite_yoffset;

返回。

Real

例子。

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

上面的代码检查实例的sprite 的顶部边缘是否在room 的外面，如果是，它就移动实例，使整个sprite 可见。
