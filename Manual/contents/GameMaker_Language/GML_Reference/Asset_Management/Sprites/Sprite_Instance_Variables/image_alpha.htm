Image_alpha

这个变量用于获取或设置sprite 的 Alpha 值。Alpha 总是以 0 和 1 之间的数值计算，其中 0 是完全透明的，1 是完全不透明的。请注意，为了使这个变量的变化可见，这个实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用一个扩展的绘制函数来绘制，例如 draw_self()或 draw_sprite_ext().

语法。

image_alpha;

返回。

Real

例子。

image_alpha = clamp(image_alpha - 0.01, 0, 1);

上面的代码会慢慢减少image_alpha，直到它达到0。
