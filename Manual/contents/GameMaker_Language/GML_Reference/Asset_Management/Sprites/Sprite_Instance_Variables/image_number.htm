image_number

这个只读变量可以用来获取已经分配给一个实例的sprite 中的子图像的数量（如果你需要一个sprite ，而不是分配给这个实例的子图像的数量，你应该使用 sprite_get_number()).请注意，当有（例如）1个子图像时，该变量将返回 "1"，但该子图像的image_index为0。

语法。

image_number

返回。

(single precision floating point value)

例子。

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

上面的代码检查sprite 是否有动画，如果有，则检查当前image_index是否大于子图像的数量，如果是，则销毁该实例。
