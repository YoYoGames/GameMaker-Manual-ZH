图像混合

这个变量控制实例的 "着色 "sprite ，默认值是-1（但也可以是 c_white ）。任何其他值（包括内部颜色常数，如 c_red ，或 c_aqua ）都会将指定的颜色与原始的sprite 。请注意，为了使这个变量的变化可见，这个实例应该没有绘制事件（因此GameMaker将默认绘制sprite ），或者使用一个扩展的绘制函数，如 draw_self()或 draw_sprite_ext().

请注意，你应该尽量限制HTML5平台上的混合（除非使用WebGL），因为每个混合的sprite ，必须单独进行缓存，因此有许多混合的sprites ，可能会对性能产生不利影响（你也可以使用函数设置cache 的大小 sprite_set_cache_size()).

语法。

image_blend;

返回。

Colour Constant

例子。

image_blend = make_colour_hsv(255, 255, random(255));

上述代码将把为该实例定义的sprite 的image_blend设为随机颜色。
