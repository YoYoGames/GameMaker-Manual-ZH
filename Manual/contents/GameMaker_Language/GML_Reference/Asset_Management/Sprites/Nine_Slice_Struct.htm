九片式结构

一个九段结构可以用以下方法创建 sprite_nineslice_create()创建，或使用sprite 检索。 sprite_get_nineslice().这个struct 将包含可以改变的属性，并可以使用sprite 。 sprite_set_nineslice().

如果从一个sprite ，改变这个struct 中的值将 影响原始sprite 。

以下属性将在此结构中可用。

九片式结构
变化的 类型 描述
enabled 布尔型 如果sprite 启用了Nine Slicing，这个布尔值将为真 ，如果禁用则为假 。你可以修改这一点，为sprite 启用或禁用Nine Slicing。
left 真实的 这是X轴上的左导向器的偏移，相对于sprite 的左边缘。
right 真实的 这是X轴上右导向的偏移，相对于sprite 的右边缘。
top 真实的 这是在Y轴上的顶部导板的偏移量，相对于sprite 的顶部边缘。
bottom 真实的 这是Y轴上底层导轨的偏移量，相对于sprite 的底边。
tilemode 阵列的 瓦片模式常数s 这将返回一个包含边缘片和中心片的瓷砖模式的数组。

瓦片模式

可以用这些常数作为数组索引来读取或修改每个片断的Tile Mode。

九片式边缘常数
恒定 描述
nineslice_left 左边的边缘切片
nineslice_top 顶层边缘的切片
nineslice_right 右边的边缘切片
nineslice_bottom 底部边缘切片
nineslice_centre 中间的片子

分片的瓦片模式将是（或可以设置为）以下常数之一。

瓦片模式常数
恒定 描述
nineslice_stretch 该片将被拉长
nineslice_repeat 该切片将被重复使用
nineslice_mirror 该片将通过镜像重复进行
nineslice_blank 分片不会被拉伸或重复，导致其后面出现空白区域
nineslice_hide 切片根本就不会出现

由于tilemode 是一个数组，你需要使用数组的语法来改变一个切片的Tile Mode。下面的代码例子将顶部片断的Tile 模式设置为重复。

例子。

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
