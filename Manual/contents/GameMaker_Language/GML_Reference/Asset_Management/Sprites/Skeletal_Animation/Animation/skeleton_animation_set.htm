骨架_动画_设置

一旦你给你的实例指定了一个骨架动画sprite ，你就可以用这个函数定义它在任何时候应该使用哪个动画集。当你创建你的sprite （在Spine ）时，你将为不同的状态定义多个动画集，并适当地命名它们。这些名字是你用来告诉GameMaker在游戏中的任何时候使用哪个动画集的。例如，如果你的sprite 是一个平台游戏，那么你可能有一个 "跳跃 "集，一个 "运行 "集和一个 "空闲 "集都包含在一个sprite ，你可以使用这个函数在你的游戏中在它们之间switch 。
该动画循环播放，这是默认的。

注意，以这种方式设置一个新的动画将重置 image_index到0，新的动画集从它的动画开始loop 。如果这不是你想要的行为，那么你可以使用 "混合 "值设置 skeleton_animation_mix()来设置一个 "混合 "值，当你设置一个新的动画时，它将与前一组动画平滑地插值。你还应该确保在每个动画的开始都有一个keyframe ，以防止数据从以前的动画中被保留下来，除非你打算使用独立的动画轨道。

语法。

skeleton_animation_set(animname, [loop]);

争论 类型 描述
animname 字符串 The name (a string) of the animation set to use.
loop 布尔型 动画是否应该 （默认为loop true)

返回。

N/A

例子。

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

上面的代码会在按下空格键时将正在使用的动画集改为 "跳跃 "集，但前提是当前正在使用的动画集不是 "跳跃"。

返回。动画
