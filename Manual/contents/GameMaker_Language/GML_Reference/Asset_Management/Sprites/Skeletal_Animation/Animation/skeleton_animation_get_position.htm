骨架_动画_获取位置

这个函数将返回动画在指定动画轨道上的位置，数值范围是0到1。这样你就可以得到轨道位置的百分比，数值0对应0%，数值1对应100%。

如果没有动画被分配到给定的特定轨道，或者如果实例没有设置sprite ，它将返回-1。

语法。

skeleton_animation_get_position(track);

争论 类型 描述
track 真实的 The animation track to get the position of.

返回。

Real

例子。

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

上面的代码检查了分配给轨道0的骨架动画的当前位置，并将其存储在一个临时变量_position中。然后，它通过乘以100将该值转换为百分比，并将其结果存储在另一个临时变量_position_percent中。
最后，它显示一个调试信息，显示百分比值。

返回。动画
