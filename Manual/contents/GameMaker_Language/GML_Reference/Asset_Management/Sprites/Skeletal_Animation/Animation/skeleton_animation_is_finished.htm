骨架_动画_已完成

通过这个函数，你可以检查当前非循环骨骼动画是否已经在给定的轨道上播放完毕。

语法。

skeleton_animation_is_finished(track);

争论 数据类型 描述
追踪 真实的 动画轨道的索引

返回。

Boolean

例子。

var _finished = skeleton_animation_is_finished(1);
if _finished
{
    show_debug_message("The animation has finished playing");
}

上面的代码首先检查轨道1上的动画是否已经播放完毕，并将结果存储在一个局部变量 _finished 。然后它显示一个调试信息，以防动画确实播放完毕。

返回。动画
