骨架_动画_列表

通过这个函数，你可以用作为骨架动画一部分的所有动画名称填充一个（预先创建的）DS列表 sprite 。这些名字将被strings ，然后可以在其他动画函数中使用这些类型的sprite 。

语法。

skeleton_animation_list(sprite, list);

争论 类型 描述
sprite 雪碧资产 The sprite index of the Spine skeletal animation to get the list from.
list DS列表ID The ID of the DS list to populate with the animation names.

返回。

N/A

例子。

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

上面的代码为实例使用的sprite ，创建了一个所有动画名称的DS列表。然后loops ，并将所有动画的混合值与第一个动画一起设置为0.5。

返回。动画
