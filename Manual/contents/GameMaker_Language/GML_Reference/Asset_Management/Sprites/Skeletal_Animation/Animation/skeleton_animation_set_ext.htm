骨架_动画_设置文本

当你为你的骨架动画sprite ，设置一个动画集时，它在内部被分配到0轨。然而，你可以将更多的动画集分配到更多的轨道（1和更多），这样你的sprite ，同时播放两个（或更多）动画。请注意，在你的动画程序中设计sprite ，应该考虑到这一点，比如0轨的 "基本 "动画只对某些骨骼有keyframes ，而接下来的轨道对其他骨骼有keyframes 。例如，如果你有一个 "行走 "动画，它的keyframes ，只影响腿部的sprite ，那么你可以有一组头部动画和手臂动画，你可以分别在轨道1和2中设置，允许你有一个基本动作，并在游戏进行中实时调整它。
该动画循环播放，这是默认的。

语法。

skeleton_animation_set_ext(animname, track, [loop]);

争论 类型 描述
animname 字符串 The name (a string) of the animation set to use.
track 真实的 The track for the animation set to use.
loop 布尔型 动画是否应该 （默认为loop true)

返回。

N/A

例子。

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

当按下鼠标左键时，上述代码将把轨道1正在使用的动画集改为 "bodyfight "集或 "bodywalk "集。

返回。动画
