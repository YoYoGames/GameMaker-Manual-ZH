骨架_动画_设置位置

这个函数会将分配给给定轨道的动画设置到你提供的位置。如果你提供了一个超出这个范围的值，那么轨道的位置将被设置为该值的小数部分。假设你使用一个0.4的值，一个1.4的值，一个2.4的值，...。这些都会将位置设置为0.4。

语法。

skeleton_animation_set_position(track, position);

争论 类型 描述
track 真实的 The animation track to set the frame of.
position 真实的 The new position for the animation frame.

返回。

N/A

例子。

skeleton_animation_set_position(1, 0.25);

上面的代码将把骨架动画的轨道1的位置设置为0.25，或者换句话说，设置为动画开始后的25%。

返回。动画
