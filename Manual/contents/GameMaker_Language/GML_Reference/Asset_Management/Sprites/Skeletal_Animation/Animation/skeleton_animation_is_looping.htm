骨架_动画_is_looping

通过这个函数，你可以检查给定轨道上的当前动画是否被设置为loop 。

语法。

skeleton_animation_is_looping(track);

争论 类型 描述
追踪 真实的 动画轨道的索引

返回。

Boolean

例子。

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

上面的代码将检查0号轨道上的动画是否被设置为loop 。如果是这样的话，它会显示一个调试信息。

返回。动画
