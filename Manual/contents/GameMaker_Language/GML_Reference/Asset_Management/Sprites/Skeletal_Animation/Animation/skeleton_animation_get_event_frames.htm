skeleton_animation_get_event_frames

这个函数可以用来检索给定动画中给定事件的所有帧。你提供骨架动画的名称（以string ，在制作动画的程序中定义，或通过使用函数 skeleton_animation_get() 返回）和一个事件名称。该函数返回一个包含该事件的帧的数组。

如果指定的事件名称不存在，该函数将返回一个数组，其第一个（也是唯一）元素为-1。

语法。

skeleton_animation_get_event_frames(anim_name, event_name);

争论 类型 描述
anim_name 字符串 The animation name to get the frames of.
event_name 字符串 The event name to get the frames for.

返回。

Array of Reals

例子。

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

上面的代码得到了"Footstep "事件的帧。如果它返回一个包含有效帧的数组，它loops ，并检查当前帧是否等于数组中的任何帧。在这种情况下，它就会播放脚步声效果，并断开（停止）loop 。

返回。动画
