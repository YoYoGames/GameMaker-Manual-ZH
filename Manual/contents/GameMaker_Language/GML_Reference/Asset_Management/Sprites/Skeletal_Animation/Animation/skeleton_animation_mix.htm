骨架_动画_混合

你可以使用switch 动画集，很容易。 skeleton_animation_set()函数，但这可能会导致一个动画换成另一个动画时出现跳动或停顿。为了防止这种情况，你可以在两个动画集之间设置混合值，sprite ，在它们之间进行插值。通常情况下，你想在具有骨架动画的实例的创建事件中这样做，因为它只需要设置一次，GameMaker将使用该实例中的动画集对sprite 的所有进一步变化进行插值。请注意，持续时间的值从0到1，其中0.5的值会有一个 "一半和一半 "的插值，从一个集合到另一个。

注意Spine 4.0改变了旋转动画的工作方式，因为它不再使用最短的路线来插值旋转，而是使用你的keyframes 中的绝对值来插值。如果你正在从旧版本升级你的Spine 动画，这可能会导致意想不到的行为，你的Spine 项目中可能需要改变，以使旋转的行为符合要求。

语法。

skeleton_animation_mix(animfrom, animto, duration);

争论 类型 描述
animfrom 字符串 The name (a string) of the first animation set to interpolate from.
animto 字符串 The name (a string) of the second animation set to interpolate to.
duration 真实的 The duration of the interpolation (from 0 to 1)

返回。

N/A

例子。

skeleton_animation_set("walk");
skeleton_animation_mix("walk", "jump", 0.2);
skeleton_animation_mix("jump", "walk", 0.4);

上面的代码将进入一个有骨架动画的实例的创建事件sprite ，并设置动画混合持续时间，用于在两个动画集 "走 "和 "跳 "之间进行插值。

返回。动画
