骨架_动画_清空

这个函数将清除指定动画轨道上的所有动画，准备重新分配。

语法。

skeleton_animation_clear(track);

争论 类型 描述
track 真实的 The animation track to clear.

返回。

N/A

例子。

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

如果按了鼠标右键，上面的代码将清除动画轨道1。

返回。动画
