骷髅头_附件_创建

当你创建你的骨架动画sprite ，你可以分配附件槽和附件到它们里面去。这些是简单的图像，与动画无关，但当它们被附加时，会随着它们被附加的骨骼一起移动。通常你会在你的动画程序中分配这些附件(Spine)，但是有了这个函数，你可以在运行时使用你游戏中的sprite asset ，创建你自己的附件。这个函数要求你给附件一个名字（如string ），然后设置要使用的sprite_index和image_index，以及x和y原点（可以与sprite 在sprite 属性中定义的不同），你还可以设置任何你希望在附件中应用到图片的变换。如果附件创建成功，该函数将返回 1 ，如果没有创建成功（你提供了一个无效的sprite 索引，或者基础sprite 不是一个Spine sprite ），那么它将返回 -1 。

语法。

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

争论 类型 描述
name 字符串 The name (as a string) of the attachment to create.
sprite 雪碧资产 The sprite_index to get the attachment image from.
ind 真实的 The image_index to get the attachment image from.
xorigin 真实的 The x origin for the image being used.
yorigin 真实的 The y origin for the image being used.
xscale 真实的 The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale 真实的 The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot 真实的 The rotation of the image. 0=normal, 90=turned 90 degrees counter-clockwise etc.

返回。

Real (1 if successful, -1 if not)

例子。

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

上述代码将检查当前为名为 "slot_leftHand "的插槽分配的附件名称，如果返回一个空的string ，则会附加一个新的sprite 。

返回。附件
