骨架_附件_创建_颜色

当你创建你的骨架动画sprite ，你可以分配附件槽和附件去它们里面。这些简单的图像(sprites)与动画是分开的，但当它们被附加时，会随着它们所附加的骨骼一起移动。通常你会在你的动画程序中分配这些附件(Spine)，但是有了这个函数，你可以在运行时使用你的游戏中的sprite asset ，创建你自己的附件。这个函数要求你给附件一个名字（如string ），然后设置 sprite_index和 image_index使用，以及x和y原点（可以与sprite 属性中的sprite 所定义的不同），你还可以设置任何你希望在附加时应用于图像的变换，以及与图像混合的颜色和它的alpha（透明度）值。如果附件创建成功，该函数将返回 1 ，如果没有创建成功（你提供了一个无效的sprite 索引，或者基础sprite 不是一个Spine sprite ），那么它将返回 -1 。

值得注意的是，实例sprite ，可以有一个混合颜色和alpha设置(image_blend和 image_alpha)，附件槽也可以(见函数 skeleton_slot_colour_set())，因此，为槽指定的附件sprite ，最终的颜色和阿尔法将是所有这些值的综合。

语法。

skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, colour, alpha);

争论 类型 描述
name 字符串 The name (as a string) of the attachment to create.
sprite 雪碧资产 The sprite_index to get the attachment image from.
ind 真实的 The image_index to get the attachment image from.
xorigin 真实的 The x origin for the image being used.
yorigin 真实的 The y origin for the image being used.
xscale 真实的 The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale 真实的 The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot 真实的 The rotation of the image: 0 = normal, 90 = turned 90° counter-clockwise etc.
Colour 颜色 The colour value to use (A constant, integer or hex value).
Alpha 真实的 The alpha value to use (from 0 to 1).

返回。

Real (1 if successful, -1 if not)

例子。

skeleton_attachment_create_colour("sword", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1);
skeleton_attachment_create_colour("knife", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1);
skeleton_attachment_create_colour("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

上述代码将检查当前为名为 "slot_leftHand "的插槽分配的附件名称，如果返回一个空的string ，则会附加一个新的sprite 。

返回。附件
