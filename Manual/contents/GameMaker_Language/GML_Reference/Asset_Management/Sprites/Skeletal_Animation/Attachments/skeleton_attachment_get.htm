骨架_附着物_获取

一个骨架动画sprite ，可以添加其他sprites 作为附件，这些sprites 被添加到一个命名的槽中（名称在你的动画程序中创建附件槽时给出），它们将与当前sprite 的动画一起被绘制。通过这个函数，你可以得到当前分配的string 的指定槽的附件名称（作为sprite ）。请注意，附件sprites 是通过它们在Spine 中分配的名称字符串来引用的，或者当你调用 skeleton_attachment_create().

语法。

skeleton_attachment_get(slot);

争论 类型 描述
slot 字符串 The slot name (a string) to get the attachment of.

返回。

String

例子。

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

上述代码将检查当前为名为 "slot_leftHand "的插槽分配的附件名称，如果返回一个空的string ，则会附加一个新的sprite 。

返回。附件
