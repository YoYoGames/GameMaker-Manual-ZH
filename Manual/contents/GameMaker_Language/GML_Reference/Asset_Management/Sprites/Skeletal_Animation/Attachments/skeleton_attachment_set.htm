骨架_附件_设置

骨架动画sprite ，可以添加其他图像作为附件，这些图像被添加到一个命名的槽中（名称是在你的动画程序中创建附件槽时给出的），它们将与当前的动画一起被绘制sprite 。通过这个功能，你可以把附件设置到一个给定的槽中，在这里你需要给出槽和附件的名称（如strings ）。这些名字是由使用的动画程序定义的，或者（在附件的情况下）当你调用 skeleton_attachment_create().

注意，你也可以传入一个 sprite_index 作为附件，该sprite 将被使用，或者你可以使用-1来从槽中移除附件。当你传入一个sprite 索引作为参数时，它将创建一个附件槽，使用sprite 的名称作为string 来命名这个槽（所以使用 spr_sword ，例如，将创建一个附件槽 "spr_sword"），这个槽将使用sprite 的第一个图像索引（0），以及它的x和y原点偏移，比例为（1，1），旋转为0。

语法。

skeleton_attachment_set(slot, attachment);

争论 类型 描述
slot 字符串 The slot name (a string) to get the attachment of.
attachment 字符串或精灵资产 The name (as a string or a sprite_index) of the attachment image.

返回。

String

例子。

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

上述代码将检查当前为名为 "slot_leftHand "的插槽分配的附件名称，如果返回一个空的string ，则会附加一个新的sprite 。

返回。附件
