骨架_查找插槽

通过这个函数，你可以在与当前实例相关的Spine sprite ，找到哪些槽在指定的房间-空间位置上。你创建一个DS列表，并提供它的ID以及要检查的X/Y位置，该列表将为string （包括任何附件修改）的每个可用的附件槽填充名称sprite 。注意，列表总是从最上面的槽开始按降序排序。

语法。

skeleton_find_slot(x, y, list);

争论 类型 描述
x 真实的 The x position in the room to check.
y 真实的 The y position in the room to check.
list DS列表ID The ID of the DS list to populate with the DS maps.

返回。

N/A

例子。

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

上述代码创建了一个DS列表，然后用鼠标在room 的位置找到的插槽名称填充它。

返回。槽位
