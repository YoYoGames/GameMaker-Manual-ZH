骨架_槽位_列表

通过这个函数，你可以用作为骨架动画的一部分而创建的所有槽的名称填充一个（预先创建的）DS列表sprite 。这些名字将被strings ，然后可以在其他骨架动画槽函数中使用这些类型的sprite 。

语法。

skeleton_slot_list(sprite, list);

争论 类型 描述
sprite 雪碧资产 The sprite index of the Spine skeletal animation to get the list from.
list DS列表ID The ID of the DS list to populate with the bone names.

返回。

N/A

例子。

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

上面的代码创建了一个DS列表，然后用槽的名字填充它，供以后使用。

返回。槽位
