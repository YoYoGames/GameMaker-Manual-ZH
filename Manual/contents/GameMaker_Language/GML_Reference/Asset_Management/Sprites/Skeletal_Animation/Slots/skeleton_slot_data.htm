骨架_slot_data

通过这个函数，你可以用一个动画所包含的每个可用的附件槽的数据来填充一个（预先创建的）DS列表。这些数据会以DS地图的形式返回，其中包含以下键/值对。

  • "名称"。插槽的名称。
  • "骨头"。骨头的名称。
  • "附件"。如果使用了附件，则是附件的名称；如果没有指定附件，则是"（无）"。

每个键的值将被strings ，然后可以在其他骨架附件函数中使用这些类型的sprite 。注意，创建的DS地图不会被销毁，所以你需要loop ，通过DS列表，自己销毁每个创建的地图。

语法。

skeleton_slot_data(sprite, list);

争论 类型 描述
sprite 雪碧资产 The sprite index of the Spine skeletal animation to get the data from.
list DS列表ID The ID of the DS list to populate with the DS maps.

返回。

N/A

例子。

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);

for (var i = 0; i < ds_list_size(list); i++)
{
    var map = list[| i];
    if open
    {
        if map[? "attachment"] == "(none)"
        {
            open = false;
            slot_name = map[? "name"];
        }
    }

    ds_map_destroy(map);
}

ds_list_destroy(list);

上述代码创建了一个DS列表，然后用实例的插槽数据填充它sprite 。然后对这个数据进行解析，提取出带有槽位数据的单个DS地图。然后检查是否有一个空槽，如果有的话，在销毁DS地图之前，变量 "slot_name "被分配给空槽的名称。最后我们销毁DS列表，因为它不再需要了。

返回。槽位
