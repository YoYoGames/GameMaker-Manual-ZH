骨架_插槽_颜色_设置

通过这个函数，你可以设置一个附件槽的颜色和阿尔法，这样所有附在它身上的sprites ，都会用这些混合值来绘制。请记住，实例sprite 可以有一个混合颜色和alpha设置(image_blend和 image_alpha)，附件也可以(见函数 skeleton_attachment_create_colour())，因此，为该槽指定的附件sprite 的最终颜色和alpha将是所有这些值的综合。

语法。

skeleton_slot_colour_set(slot, colour, alpha);

争论 类型 描述
slot 字符串 The slot name to set, a string
colour 颜色 The colour to set, either an integer, a constant, or a hex value.
alpha 真实的 The alpha to set from 0 to 1.

返回。

N/A

例子。

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

上面的代码创建了一个DS列表，然后用鼠标在room 中的位置找到的插槽名称填充它。然后loops ，通过插槽列表，如果颜色还不是白色，就把找到的插槽的颜色重置为白色，alpha为1。

返回。槽位
