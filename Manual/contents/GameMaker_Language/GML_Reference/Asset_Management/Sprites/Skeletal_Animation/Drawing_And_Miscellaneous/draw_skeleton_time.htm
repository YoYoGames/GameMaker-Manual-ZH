draw_skeleton_time

这个函数将使用给定的皮肤在动画的特定时间内绘制给定的动画。时间值应该在0（开始）和动画的结束时间之间，你可以通过以下函数找到 skeleton_animation_get_duration().你可以将时间值设置为高于动画总持续时间的值，动画将loop ，回到开头，但你有可能在累计时间变大时失去浮点精度。

语法。

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)

争论 类型 描述
sprite 雪碧资产 The index of the sprite to draw.
animname 字符串 The name of the animation to get the frame from (a string).
skinname 字符串 The name of the skin to use for drawing (a string).
time 真实的 The time-frame to draw the animation at (from 0 to the end duration, in seconds).
x 真实的 The x coordinate of where to draw the sprite.
y 真实的 The y coordinate of where to draw the sprite.
xscale 真实的 The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale 真实的 The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot 真实的 The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour 颜色 The colour with which to blend the sprite.

返回。

N/A

例子。

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

上述代码将绘制给定的骨架动画sprite ，使用delta-time来设置被绘制的帧。

返回。绘画和杂项
