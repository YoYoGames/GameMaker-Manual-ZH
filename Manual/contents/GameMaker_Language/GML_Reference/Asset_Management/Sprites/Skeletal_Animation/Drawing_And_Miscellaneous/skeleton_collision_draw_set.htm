skeleton_collision_draw_set

通过这个函数，你可以切换打开(true)或关闭(false)绘制实例正在使用的当前骨骼动画的碰撞数据sprite 。如果这个功能被打开，边界框和精确的碰撞掩码将被绘制成围绕sprite 的轮廓。

这个函数只在骨架sprite ，由object 。在手动绘制骨架sprite ，要绘制碰撞边界框，使用 draw_skeleton_collision() 。

语法。

skeleton_collision_draw_set(flag);

争论 类型 描述
flag 布尔型 Set to true 来打开绘图， 来关闭它。 false

返回。

N/A

例子。

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

上面的代码检查游戏是否在调试模式下运行，如果是，它将切换实例的骨架碰撞数据来显示。

返回。绘画和杂项
