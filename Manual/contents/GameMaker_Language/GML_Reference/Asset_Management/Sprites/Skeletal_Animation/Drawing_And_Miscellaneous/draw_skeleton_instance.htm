draw_skeleton_instance

这个函数只适用于那些用骨架动画程序sprites ，为其分配了Spine 的实例。虽然你可以使用普通的绘制函数来绘制这些sprites （如 draw_self())，但有时你可能想绘制一个单帧或显示一个皮肤的变化texture ，而不需要实际改变实例的sprite 属性。在这些情况下，你可以用这个函数来绘制已经分配给实例的sprite ，在这里你给出唯一的实例ID，然后给出动画的名称来获取帧（一个string ，在用于制作sprite 的程序中设置）。接下来你给出要使用的皮肤的名字（同样，在制作sprite 时设置的），以及一个帧值。帧值是用来告诉GameMaker要显示动画的哪一部分，是一个介于0和 image_number.其余的参数与普通的sprite 绘制参数相同，将改变最终绘制的动画帧的比例、角度、混合颜色和alpha。

警告!由于骨架动画在不同的 "关键 "帧之间插值的方式，这个调用有一个额外的开销，因此，建议你避免使用这个方法绘制基于骨架的sprites ，除非绝对必要。

语法。

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

争论 类型 描述
instance 实例ID The index of the instance to draw the sprite from.
animname 字符串 The name of the animation to get the frame from (a string).
skinname 字符串 The name of the skin to use (a string).
frame 真实的 The animation frame to draw (from 0 to image_number - 1).
x 真实的 The x coordinate of where to draw the sprite.
y 真实的 The y coordinate of where to draw the sprite.
xscale 真实的 The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale 真实的 The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot 真实的 The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour 颜色 The colour with which to blend the sprite.
alpha 真实的 The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

返回。

N/A

例子。

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

上述代码将绘制指定给实例的骨骼动画sprite ，使用 "跳跃 "动画集和定义为 "skin1 "的皮肤，绘制第一帧。

返回。绘画和杂项
