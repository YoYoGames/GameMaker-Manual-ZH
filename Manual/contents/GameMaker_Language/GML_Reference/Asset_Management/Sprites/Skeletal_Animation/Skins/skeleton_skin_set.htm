骷髅头_皮肤_设置

骨架动画sprite ，可以用各种textures （称为 "皮肤"）来绘制，允许你将一个动画用于多种不同的目的。例如，你可能有一个通用的行走动画，这个动画可以被你的NPC角色、敌人等在游戏中使用，只要改变它的皮肤就可以了。这些皮肤需要事先在你的动画程序中创建并命名，这个名字被传递到函数中（作为string ）以设置sprite ，以便在绘制时使用该皮肤。

语法。

skeleton_skin_set(skinname);

争论类型 描述
skinname The name (a string) of the skin to use.

返回。

例子。

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

上述代码将检查当前分配给动画的皮肤，如果它是 "skin_Enemy1"，它将从三个选项中选择并设置一个新的皮肤。
