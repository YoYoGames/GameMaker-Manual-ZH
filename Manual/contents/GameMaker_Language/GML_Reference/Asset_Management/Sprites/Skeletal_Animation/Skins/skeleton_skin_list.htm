skeleton_skin_list

通过这个函数，你可以用作为骨架动画一部分的所有皮肤的名称填充一个（预先创建的）DS列表 sprite 。这些名字将被strings ，然后可以在其他骨架动画皮肤函数中使用这些类型的sprite 。

语法。

skeleton_skin_list(sprite, list);

争论 类型 描述
sprite 雪碧资产 The sprite index of the Spine skeletal animation to get the list from.
list DS列表ID The ID of the DS list to populate with the animation names.

返回。

N/A

例子。

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

上面的代码创建了一个DS列表，然后用皮肤名称来填充它。然后随机选择一个，并在列表被销毁前应用于实例。
