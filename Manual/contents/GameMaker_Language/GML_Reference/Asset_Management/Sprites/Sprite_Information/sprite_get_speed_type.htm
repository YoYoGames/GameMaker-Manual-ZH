幽灵_获取速度_类型

这个函数可以用来检索sprite ，该动画类型是在Sprite Editor中为sprite 资源定义的。返回的值将是以下常数之一。

精灵速度常数
恒定 描述
spritespeed_framespersecond sprite ，以每秒一帧的动画来定义。
spritespeed_framespergameframe sprite ，以每一游戏帧的动画来定义。

下面的两个例子说明了在计算精灵的动画速度时，这可能会带来的不同。

  • 如果你有一个每秒绘制1帧的sprite ，并将图像速度设置为0.5，它将以每秒0.5帧的速度绘制。如果你的游戏帧率是每秒60帧，那么sprite ，每120个游戏帧就会画1帧。
  • 如果你有一个sprite ，每游戏帧画1帧，并将图像速度设置为0.5，它将每游戏帧画0.5帧。如果你的游戏帧率是每秒60帧，那么sprite ，每60个游戏帧就会画30帧。

你可以用函数查出sprite 动画所用的速度值是多少。 sprite_get_speed()来查看动画的速度值，还可以用函数来设置动画的速度和类型。 sprite_set_speed().

语法。

sprite_get_speed_type(index)

争论 类型 描述
index 雪碧资产 The index of the sprite to get the animation type of

返回。

Sprite Speed Constant (see above)

例子。

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

上面的代码获得了sprite 的速度和sprite 的动画类型，并将它们存储在实例变量中供将来使用。
