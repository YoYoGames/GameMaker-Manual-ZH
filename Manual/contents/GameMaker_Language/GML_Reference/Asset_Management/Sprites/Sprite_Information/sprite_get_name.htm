sprite_get_name

此函数将返回指定的sprite 的名称，即一个字符串。这个名字是在GameMaker主窗口的资源树中为sprite 指定的名字。请注意，这只是一个string ，不能用来直接引用sprite - 为此你需要精灵索引。但是，你可以使用这个string ，用返回的string 和函数来获得精灵的索引。 asset_get_index().

语法。

sprite_get_name(index);

争论 类型 描述
index 资产 The index of the sprite to get the name of.

返回

String

例子。

str = sprite_get_name(sprite_index);

上述代码将获得运行该代码的实例的sprite 索引的名称，并将返回值存储在变量 "str "中。
