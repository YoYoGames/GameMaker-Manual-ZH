幽灵信息（sprite_get_info

这个函数用来检索给定的sprite 的信息。你提供一个sprite 索引（可以是通过资产浏览器添加的asset 或在runtime添加的精灵），该函数返回一个包含以下变量的结构。

yoffset 真正的 该精灵的Y偏移/原点（像素）。
透明的 布尔型 true 如果 被标记为透明，否则 sprite false

下面的章节包含了根据structs 类型，在返回的struct 中包含的数组和sprite 的信息。

一般Sprite 数据

本节包含了关于各种精灵的struct 中所包含的变量的信息。

messages 变量是一个数组，包含给定sprite 中存在的广播信息的信息。这个数组中的每个广播信息都是一个struct ，包含以下变量。

变量名称 数据类型 描述
框架 真正的 这个广播信息从动画开始的时间（以帧为单位）。
信息 绳子 广播信息字符串

frame_info 变量是一个数组，包含精灵的帧的时序信息。这个数组中的每一帧都是一个struct ，包含以下变量。

变量名称 数据类型 描述
框架 真正的 该帧开始的时间（以帧为单位）。

slots 变量是一个数组，包含了给定Spine sprite 中所有槽的信息。这个数组中的每个槽是一个struct ，包含以下变量。

语法。

sprite_get_info(index);

争论 类型 描述
index 雪碧资产 The index of the sprite to get the information for.

返回

Sprite Info Struct (or undefined)

例子。

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

上面的代码为实例的struct ，获得信息sprite ，然后检查它是否不是 undefined ，并且其类型是0（意味着它是一个位图sprite ）。在这种情况下，它得到该sprite 的广播信息数组，然后运行forloop ，将每条广播信息（连同其帧）打印到输出日志。
