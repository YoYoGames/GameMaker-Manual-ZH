幽灵_获取_uvs

texture 该函数返回一个数组，其中包含sprite 子图像在texture 页面上的UV坐标和其他数据。该函数返回一个包含以下8个元素的数组。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部
  • [4] =asset 编译器从sprites 左边修剪的像素数量
  • [5] =asset 编译器从sprites 顶面修剪的像素数量
  • [6] = 已保存到sprites 页面的原始texture 宽度的像素数据的正常化百分比
  • [7] = 已保存到sprites 页面的原始texture 高度的像素数据的正常化百分比

然后，这个数组可以用于其他绘制函数，特别是在3D工作或使用2D原始函数时，以及使用Shader函数时。

注意：此功能不能与矢量sprites 或骨架动画sprites 。

语法。

sprite_get_uvs(sprite, subimage);

争论类型 描述
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

返回

(1D, 8 elements)

例子。

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上述代码将把给定的UV坐标存储在本地数组中sprite ，然后把这些值分配给实例变量。
