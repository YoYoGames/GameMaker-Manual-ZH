幽灵_获取yoffset

当你在sprite 编辑器中定义一个sprite ，你需要为该sprite 设置原点。这是sprite "连接 "或 "绘制 "的点，当与一个实例一起使用时。这个函数返回原点Y轴的相对偏移量，以左上角的0,0位置为基础，+y表示向下，-y表示向上。下面的图片说明了这一点。

语法。

sprite_get_yoffset(index);

争论类型 描述
index The index of the sprite to find the yoffset of.

返回

例子。

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = sprite_get_yoffset(sprite_index);
}

上述代码将确保在room 的基础上，在sprite 内维护一个实例（所以sprite 总是可见的）。
