幽灵的数量

一个sprite ，至少要有一个子图像，这个函数将返回它有多少个。记住，一个sprite 子图像从0开始编号，所以如果这个函数返回3（被检查的sprite 有3个子图像），它们将被编号为 0、1和2。

语法。

sprite_get_number(index);

争论类型 描述
index The index of the sprite.

返回

例子。

image_index = floor(random(sprite_get_number(sprite_index)));

上述代码将根据sprite 的子图像数量，将图像索引设置为一个随机帧。
