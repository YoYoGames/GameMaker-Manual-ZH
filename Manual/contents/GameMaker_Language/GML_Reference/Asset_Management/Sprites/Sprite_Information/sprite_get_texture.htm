纹理_获取纹理

这个函数为sprite texture 页面返回一个特殊的指针。然后这个值可以在其他绘图函数中使用，特别是在2D原始函数以及Shader函数中。你可以使用这里找到的不同的 texture_ 函数获得更多关于返回的texture 页面的信息。

语法。

sprite_get_texture(spr, subimg);

争论 类型 描述
spr 雪碧资产 The index of the sprite to use.
subimg 真实的 The sub-image of the sprite to use.

返回

Texture

例子。

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

上面的代码将绘制一个4vertex 的三角形条状纹理，texture ，在 "tex "变量中保存。
