幽灵_获取_TPE

这个函数将返回一个特殊的值，它与你的游戏所使用的sprite 和子图像在texture 页面上的位置有关。通过这种方式，你可以将值传递给extensions ，以便在编译的游戏之外使用，有效地绕过GameMaker的绘图功能，并允许在DOM内容中使用sprite ，然后可以在包含游戏画布的窗口中的任何地方绘制。这个函数对于那些希望在GameMaker画布元素之外创建按钮和其他互动媒体的人来说特别有兴趣，他们可以使用函数 clickable_add().

注意：该功能仅适用于HTML5。

语法。

sprite_get_tpe(sprite, index);

争论 类型 描述
sprite 雪碧资产 The index of the sprite to find the texture page entry of.
index 真实的 The image index of the sprite.

返回

Texture Page Entry

例子。

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

上面的代码在游戏画布运行的页面的位置（32，32）创建了一个DOM按钮。sprite 该按钮使用从texture 页面引用的 "spr_MS_Home"，当点击该按钮时，将为指定的URL打开一个新窗口，并为该窗口定义属性。
