幽灵的高度

通过这个函数，你可以找到基数的高度sprite asset ，没有变换，单位是像素。

语法。

sprite_get_height(index);

争论类型 描述
index The index of the sprite to find the height of.

返回

例子。

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

上面的代码检查了当前实例中sprite 的高度，如果与原来的基础sprite 有差异，就会重设y轴的比例。
