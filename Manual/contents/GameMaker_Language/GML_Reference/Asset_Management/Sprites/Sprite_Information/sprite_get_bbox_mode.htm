幽灵_get_box_mode

这个函数将返回当前边界框计算的 "模式"。sprite 你提供要检查的sprite 的索引，该函数将返回下面所示的常数之一。

语法。

sprite_get_bbox_mode(ind);

争论类型 描述
ind The index of the sprite to check.

返回

Bounding Box Mode Constant

恒定 描述
bboxmode_automatic 自动--将根据精灵的公差设置，自动计算边界框。
bboxmode_fullimage 全图 - 界限框将被设置为使用sprite 的全部宽度和高度，而不考虑公差和 "空 "像素。
bboxmode_manual spritesprite_set_bbox())

例子。

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

上面检查了当前sprite 的bbox模式，如果不是 automatic ，则将其设置为该值。
