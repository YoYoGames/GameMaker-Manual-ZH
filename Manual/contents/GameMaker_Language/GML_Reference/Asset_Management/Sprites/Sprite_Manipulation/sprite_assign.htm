魉的分配

这个函数接收两个先前创建（或包含）的sprite 索引，并将图像从一个复制到另一个。通过这种方式，你可以将一个sprite 复制（或 "克隆"）到另一个索引。注意，你不能复制到一个游戏资源。你必须在之前创建了要复制的sprite ，使用 sprite_add()或 sprite_duplicate()功能创建了要复制的。

语法。

sprite_assign(index, sprite);

争论类型 描述
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

返回

例子。

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

上述代码检查全局变量 "Player_Sprite "是否包含一个sprite ，如果包含，则使用sprite_assign将其改为从外部文件加载的一个。如果它不包含一个sprite ，则会加载一个，并将其索引存储在该变量中。
