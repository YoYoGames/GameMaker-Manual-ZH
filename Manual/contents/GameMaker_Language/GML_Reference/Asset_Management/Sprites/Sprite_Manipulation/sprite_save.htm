幽灵_保存

此功能可用于将sprite 的任何子图像保存到光盘中，给它指定文件名。sprite 必须是在runtime 添加的（你不能保存通过sprites 添加的IDE ），并且文件必须以 *.png extension 保存。

注意：根据所选择的目标平台，你可以从哪里保存和加载文件是有限制的。更多信息请参见文件处理。

语法。

sprite_save(ind, subimg, fname);

争论 类型 描述
ind 雪碧资产 The index of the sprite to save.
subimg 真实的 The sub-image of the sprite to save.
fname 字符串 The filename for the saved sprite.

返回

N/A

例子。

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

上面的代码创建了一个曲面，并将其索引存储在本地变量 "surf "中。然后，它以该表面为目标，清除它，并在其上绘制几个sprites 。它从表面上绘制的合成图像中创建一个新的sprite ，并将其索引分配给本地变量 "spr_Custom"，然后释放表面使用的内存。最后，生成的sprite 被保存到一个文件中，然后也从内存中删除。
