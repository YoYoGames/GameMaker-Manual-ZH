萌芽_保存_条状物

这个函数将从一个sprite 中的所有子图像创建一个条状图像，并以指定的文件名将其保存到光盘中。sprite 必须是在runtime 添加的（你不能保存通过sprites 添加的IDE ），文件必须以 *.png extension 保存。

注意：根据所选择的目标平台，你可以从哪里保存和加载文件是有限制的。更多信息请参见文件处理。

语法。

sprite_save_strip(ind, filename);

争论 类型 描述
ind 雪碧资产 The index of the sprite to save.
filename 字符串 The filename for the saved sprite strip.

返回

N/A

例子。

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

上述代码从变量 "surf "中索引的表面创建了一个sprite ，将其索引分配给本地变量 "spr_Custom"，然后使用 for loop ，在表面上移动并捕捉各种部分，这些部分作为子图像被添加到sprite 。这个新的sprite ，然后被保存为png条，再从内存中删除。
