sprite_prefetch_multi

这个函数可以用来预取（放入texture 内存）一些包含给定texture 的sprites 页面。你提供一个由sprite 指数填充的数组（如你创建sprite asset 时定义的），它们所在的texture 页面将被加载到内存中。请注意，如果所选资源不支持预取，或者目标平台是HTML5，该函数将返回 -1 ；如果所有工作正常，则将返回 0 。

注意：在大多数设备上，texture 被上传到texture 内存，所以建议你在关卡开始时调用 sprite_prefetch_multi()在关卡开始时调用任何需要的图形，以避免任何停顿。

语法。

sprite_prefetch_multi

争论 类型 描述
array Sprite资产阵列 Array with the sprite indices to fetch

返回。

Real (-1 or 0)

例子。

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

上述代码创建了一个数组，其中每个元素持有一个sprite 的索引。然后这个数组被用来将这些sprite textures 到内存中。
