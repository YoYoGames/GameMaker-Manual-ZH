从表面添加精灵

这个功能的工作方式与 sprite_create_from_surface()只是它不是从你选择的索引表面的区域创建一个新的sprite ，而是将表面的定义区域作为一个新的子图像添加到先前创建的sprite （这意味着你不能直接添加到资源树中的sprite ，而只能添加到先前从表面创建的，或从资源中复制的sprite ，用 sprite_duplicate()).

应该注意的是，以这种方式添加sprite ，所使用的内存会比你预期的大。这是因为GameMaker将把sprite 作为一个texture 页面来存储，它也将被存储在GPU内存中，所以总的内存将比同样的sprite 的图像文件所预期的要大。

语法。

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

争论 类型 描述
index 雪碧资产 The index of the sprite to add the new image to.
surface 表面标识 The index of the surface from which we get the image.
x 真实的 The x position to copy from.
y 真实的 The y position to copy from.
w 真实的 The width of the area to be copied (from the x position).
h 真实的 The height of the area to be copied (from the y position).
removeback 布尔型 Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth 布尔型 Indicates whether to smooth the edges.

返回。

N/A

例子。

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i += 1)
{
    sprite_add_from_surface(spr_custom, surf, i, 0, 32, 32, true, true);
}

上述代码从变量 "surf "中索引的表面创建了一个sprite ，将其索引分配给变量 "spr_Custom"，然后使用 for loop ，在表面上移动并捕捉各种部分，这些部分作为子图像被添加到sprite 。
