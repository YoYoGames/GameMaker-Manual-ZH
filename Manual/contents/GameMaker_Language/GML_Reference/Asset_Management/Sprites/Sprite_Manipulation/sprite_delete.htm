删除精灵

这个函数将从游戏中删除一个sprite ，释放为其保留的任何内存。这是一个非常重要的函数，当你需要从外部来源创建和改变sprites （如从文件中加载一个sprite ，用 sprite_add() ，或使用 sprite_duplicate() 复制一个sprite ），并且应该总是用来删除那些不再需要的assets 。 游戏不再需要的asset ，或者在将另一个asset 重新分配给该变量之前，从一个变量中清除索引的 。这不能用来删除sprites ，因为它是作为assets 的一部分包含在游戏中的Asset 浏览器。

语法。

sprite_delete(index);

争论类型 描述
index The index of the sprite to be deleted.

返回

例子。

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

上面的代码创建了一个局部变量，然后存储了从应用表面创建的sprite 的索引。这个sprite ，然后与变量 "spr_Player "中索引的asset 合并，再从内存中删除。
