萌芽_预取

这个函数可以用来预取（放入texture 内存）一个texture 页面，并给定sprite 。你提供创建sprite sprite 时定义的asset 索引，它所在的texture 页面将被加载到内存中。请注意，该 如果所选资源不支持预取，或者目标平台是HTML5，则该函数将返回 -1 ，如果所有工作正常，则返回 0 。

注意：在大多数设备上，当texture 被上传到texture 内存时，会有性能上的影响，所以建议你调用 sprite_prefetch在 任何需要的图形，以避免任何停顿。

语法。

sprite_prefetch(ind)

争论类型 描述
ind The sprite index to fetch

返回。

(-1 or 0)

例子。

sprite_prefetch(spr_Player_Aura);

上述代码将把被引用的sprite ，放入texture 内存中准备使用。
