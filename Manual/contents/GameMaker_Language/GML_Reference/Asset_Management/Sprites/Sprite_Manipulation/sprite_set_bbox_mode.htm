萌芽_set_box_mode

这个函数可以用来设置一个sprite 。你提供sprite 索引和要使用的模式，它应该是下列常数之一。

包围盒模式常数
恒定 描述
bboxmode_automatic 自动-- 将根据精灵的公差设置，自动计算边界框。
bboxmode_fullimage 全图- 界限框将被设置为使用sprite 的全部宽度和高度，而不考虑公差和 "空 "像素。
bboxmode_manual 手动--界线盒被手动设置为用户定义的值，这需要使用函数 sprite_set_bbox()

注意这个函数会影响精灵资产 ，所以所有进一步使用这个sprite 的实例都会有相同的边界框模式。

语法。

sprite_set_bbox_mode(ind, mode);

争论 类型 描述
ind 雪碧资产 The index of the sprite to change the mode of.
mode 包围盒模式常数 The mode to set (a constant).

返回

N/A

例子。

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

上面的代码检查了当前sprite 的bbox模式，如果它不是 automatic ，那么它就被设置为这个值。
