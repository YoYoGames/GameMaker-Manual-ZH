幽灵_刷新

通过这个函数，你可以从texture 内存中删除给定的sprite 页面，用于给定的texture 。你提供sprite 的索引（在创建sprite 资源时定义的），它所分配的texture 页面将从texture 内存中删除。 注意，如果texture 页面在room 的其他地方使用（由另一个实例sprite 或一个背景，等等...），你可能会得到一个小的性能打击，因为该页面被立即重新加载，所以在使用这个函数时应该注意。请注意，该 函数将返回 -1 ，如果所选资源不支持刷新，或者它将返回 0 ，如果所有工作正常。

语法。

sprite_flush(ind)

争论类型 描述
ind The index (resource name) of the sprite asset to flush

返回。

(-1 or 0)

例子。

sprite_flush(spr_Player_Aura);

上面的代码从内存中冲掉了sprite "spr_Player_Aura"。
