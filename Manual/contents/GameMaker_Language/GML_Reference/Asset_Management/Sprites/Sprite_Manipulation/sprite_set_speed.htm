幽灵_设置速度

这个函数可以用来设置sprite 资源的基本动画速度和类型。这通常是在Sprite Editor中设置的，但有时你可能需要为一个给定的sprite 资源手动设置。你需要提供要设置的sprite 索引、动画速度和要应用的动画速度类型。动画速度类型可以是以下任何一个常数。

恒定 描述
spritespeed_framespersecond sprite ，以每秒一帧的动画来定义。
spritespeed_framespergameframe sprite ，以每一游戏帧的动画来定义。


选择用于计算动画的速度类型是非常重要的，因为两个选项之间有很大的区别。下面的两个例子说明了这种区别。

  • 如果你有一个每秒绘制1帧的sprite ，并将图像速度设置为0.5，它将以每秒0.5帧的速度绘制。如果你的游戏帧率是每秒60帧，那么sprite ，每120个游戏帧就会画1帧。
  • 如果你有一个sprite ，每游戏帧画1帧，并将图像速度设置为0.5，它将每游戏帧画0.5帧。如果你的游戏帧率是每秒60帧，那么sprite ，每60个游戏帧就会画30帧。

你可以用函数找出用于sprite 动画的速度值。 sprite_get_speed()来了解动画的速度值，你还可以用 sprite_get_speed_type().

重要的是!这个功能会影响到sprite 资源，所以你用这个功能所做的任何改变都会影响到所有正在使用sprite 的实例和所有未来的实例。

语法。

sprite_set_speed(index, speed, type)

争论 类型 描述
index 雪碧资产 The index of the sprite to set the animation speed of
speed 真实的 The speed value to set
type 精灵速度常数 The speed type to use

返回。

Constant (see above)

例子。

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

上面的代码将分配给实例的sprite 的基本动画速度设置为每秒10帧。
