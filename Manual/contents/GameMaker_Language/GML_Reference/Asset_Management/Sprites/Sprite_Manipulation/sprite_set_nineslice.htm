魉的故事

该函数用于将九宫格结构应用于sprite 。您提供sprite 索引和要应用的九宫格struct ，可以使用 sprite_nineslice_create()或 sprite_get_nineslice().

注意 这个函数会影响精灵资产，所以你用这个函数所做的任何改变都会影响所有正在使用这个sprite 的实例，也会影响所有未来的实例。

语法。

sprite_set_nineslice(ind, nineslice);

争论 类型 描述
ind 雪碧资产 The index of the sprite to modify
nineslice 九片式结构 The Nine Slice struct to apply

返回。

N/A

例子。

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

上面的代码创建了一个新的Nine Slicestruct ，启用了它并设置了导向器的偏移量。然后将struct 应用于sprite ，改变其九宫格属性。
