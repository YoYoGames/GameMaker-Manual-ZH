幽灵_存在

该函数返回正在运行的项目中是否存在一个具有指定索引的sprite 。

语法。

sprite_exists(index);

争论类型 描述
index The index of the sprite to check.

返回

例子。

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

上面的代码检查一个数组，看它是否包含一个有效的sprite 索引，如果是，就把这个sprite 赋给实例，但如果不是，就从包含的资源中赋给一个sprite 。
