从表面创建精灵

通过这个函数，你可以从之前初始化的曲面中创建一个sprite （曲面的索引ID值会在你创建曲面时使用 surface_create()).你在函数中输入的x和y坐标应该是相对于曲面的(0,0)位置（左上角），而不是游戏窗口（如果你有一个激活的视图，也是如此）。宽度和高度参数的单位是像素，定义要使用的曲面部分的宽度和高度。

将 "removeback "参数设置为 true ，将从sprite ，通过检查sprite 的左下角像素的颜色，然后将其作为要去除的颜色。

如果你选择了 "removeback "选项，你可能还希望GameMaker通过将 "smooth "参数设置为 true ，使sprite 的边缘平滑。这样做的目的是在sprite ，在它的背景被移除后，在其边缘创建一个半透明的边框。

最后，你还可以指定sprite 的x和y原点。这是sprite "固定 "在使用它的实例上的点，并且总是相对于一个sprite 子图像的0,0左上角计算。因此，例如，一个32 x 32像素的sprite ，这些值设置为(16,16)，其原点将在中心。

默认情况下，所有新的sprites ，它们的边界框都是自动计算的（确切的边界框将取决于sprite 的大小和透明度），但是你可能希望定制这个，在这种情况下，你也应该使用函数 sprite_collision_mask().

注意当你用这种方法在GameMaker中创建一个sprite ，你必须记得在不再需要的时候再次删除它（用 sprite_delete())，否则会有内存泄漏的风险，这将使你的游戏变慢并最终崩溃。

语法。

sprite_create_from_surface(index, x, y, w, h, removeback, smooth, xorig, yorig);

争论 类型 描述
index 表面标识 The index of the surface to create from.
x 真实的 The x position to copy from.
y 真实的 The y position to copy from.
w 真实的 The width of the area to be copied (from the x position).
h 真实的 The height of the area to be copied (from the y position).
removeback 布尔型 Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth 布尔型 Indicates whether to smooth the edges.
xorig 真实的 Indicates the x position of the origin in the sprite.
yorig 真实的 Indicates the y position of the origin in the sprite.

返回

Sprite Asset

例子。

var surf;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);

上面的代码创建了一个曲面，并将其索引存储在本地变量 "surf "中。然后，它以该表面为目标，将其清除，并在上面绘制了几个sprites 。最后，它从表面上绘制的合成图像中创建一个新的sprite ，并将其索引分配给 "spr_Custom"，然后释放表面使用的内存。
