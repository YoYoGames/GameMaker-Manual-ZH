创建精灵_nineslice_create

九宫格切片是一种用于缩放矩形图像以保留细节的技术，使其在缩放后保留原始形态。关于九宫格切片的信息，请参见本页面。

该函数用于创建一个九宫格结构，该结构可以被修改，然后应用到sprite 。该struct 的内容在本页面有详细介绍。

你可以将这个struct ，修改其属性，并使用 sprite_set_nineslice() ，将其应用于任何sprites 。

语法。

sprite_nineslice_create();

返回。

Nine Slice Struct

例子。

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

上面的代码创建了一个新的Nine Slicestruct ，启用了它并设置了导向器的偏移量。然后将struct 应用于sprite ，改变其九宫格属性。
