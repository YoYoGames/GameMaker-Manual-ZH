绘图_设置缓存尺寸_ext

在执行图像混合时，HTML5 无法以可执行文件的方式动态地执行它。因此，GameMaker必须临时保存图像的混合副本，并将其加载进来。这个函数设置了在旧的副本被覆盖之前可以缓存多少个给定的sprite 的混合副本。默认为4。这适用于sprite 的一个单独的给定子图像。

语法。

sprite_set_cache_size_ext(ind, index, max);

争论 类型 描述
ind 雪碧资产 The index of the sprite to change the cache size of.
index 真实的 The sub-image of ind to change the cache size of.
max 真实的 The maximum number of cached copies of the sprite that can be stored.

返回

N/A

例子。

sprite_set_cache_size_ext(sprite0, 0, 2);

这将把 sprite0 的第一个子图像的sprite cache 设置为 2 份。
