音频暂停_所有

通过这个功能，你可以暂停当前正在播放的所有声音。

语法。

audio_pause_all();

返回。

例子。

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

上面的代码检查是否有人按了键盘上的 "P "键，如果检测到了，就把全局变量 "Pause "设置为 true 或 false，然后要么暂停所有的声音，要么重新启动所有以前暂停的声音。

返回。音频
