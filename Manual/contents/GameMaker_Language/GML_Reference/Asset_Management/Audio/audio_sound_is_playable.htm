音频_声音_是可播放的

这个函数可以用来检查当前是否可以播放给定的声音索引。由于HTML5目标平台上处理流媒体和非流媒体声音播放的方式不同，所以需要这个函数，如果声音可以播放，将返回 true ，如果不能，将返回 false 。请注意，在除HTML5以外的所有其他平台上，该函数将始终返回 true 。

语法。

audio_sound_is_playable(index);

争论 类型 描述
index 声音资产 The index of the sound to check.

返回。

Boolean

例子。

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

如果声音 "snd_Shot "可以播放，上述代码将播放它。

返回。音频
