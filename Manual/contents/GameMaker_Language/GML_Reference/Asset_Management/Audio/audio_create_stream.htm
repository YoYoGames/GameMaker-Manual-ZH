audio_create_stream

通过这个函数，你可以创建一个新的声音索引，然后可以在常规的音频函数中使用，直接从外部OGG文件源流传音频。这个函数需要文件名（可以是一个包含的文件，例如），并将返回新的声音索引供使用。注意，当你不再需要这个声音时，你应该调用函数 audio_destroy_stream()来从内存中删除它，否则你可能会得到一个内存泄漏，这将减慢并最终使你的游戏崩溃。

注意：此功能不适用于HTML5目标平台。

语法。

audio_create_stream(filename);

争论 类型 描述
filename 字符串 Path to the file (OGG only) to stream the audio from.

返回。

Sound Asset

例子。

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

上述代码在变量 "snd "中从给定的文件中创建一个新的声音索引，然后播放这个声音。

返回。音频
