正在播放的音频

这个函数将检查给定的声音，看它目前是否正在播放。这个声音可以是一个单一的声音实例（正在播放的单个声音的索引可以存储在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数）或一个声音asset ，在这种情况下，所有给定的声音实例将被检查，如果其中任何一个正在播放，该函数将返回 true ，否则它将返回 false 。请注意，如果被检查的声音之前已经被暂停了，这个函数仍然会返回 true 。 audio_pause_sound()函数暂停。

语法。

audio_is_playing(index);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to check.

返回。

Boolean

例子。

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

上面的代码检查变量 "snd_Waterfall "中索引的声音是否正在播放，如果返回 false ，那么这个声音将在它的room ，落差距离为300，优先级较低。

返回。音频
