音频_声音_增益

通过这个功能，你可以在给定的时间长度内淡入或淡出一个声音，或者可以用来即时设置声音的增益。

时间是以毫秒为单位的，该函数要求你输入一个最终的增益水平，让声音在该时间结束时达到。这个增益可以在0（无声）和大于0的任何值之间，尽管通常你会认为最大音量是1。任何大于1的值都可以使用，但是，根据使用的声音和被编译的平台，你可能会在声音回放时出现失真或削波。请注意，增益比例是线性的，要即时改变增益，只需将时间参数设置为0即可。

如果索引是一个声音资源，这个函数将影响当前正在播放的所有声音实例room ，最后的音量将是所有进一步播放的声音实例的音量。但是，如果你使用的是一个函数返回的索引，如 audio_play_sound()等函数返回的索引，它将只影响该声音的一个实例。

语法。

audio_sound_gain(index, volume, time);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to set the gain for.
volume 真实的 Value for the music volume.
time 真实的 The length for the change in gain in milliseconds.

返回。

N/A

例子。

if val
{
    var snd = audio_play_sound(snd_fountain);
    audio_sound_gain(snd, 0, 0);
    audio_sound_gain(snd, 1, 5000);
}

上面的代码检查一个变量，如果它返回 true ，那么它将把要播放的声音的索引分配给本地变量 "snd"。然后，这个变量被用来将该特定声音的音量降低到0，并在5秒内渐渐提高到全音量。

返回。音频
