音频名称

这个函数将返回一个给定的音频资源的名称为string 。"索引 "值可以是资源本身的索引（如在Asset 浏览器中看到的），也可以是在你使用例如播放声音时给出的唯一ID值。 audio_play_sound().注意，返回的string ，与资源的ID不一样，不能用来访问资源本身，所以只能用于显示或检查错误。

语法。

audio_get_name(index);

争论类型 描述
index The index of the sound to check.

返回。

例子。

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

上面的代码从三个不同的声音资源中随机选择一个声音，然后显示一个带有其名称的调试信息。

返回。音频
