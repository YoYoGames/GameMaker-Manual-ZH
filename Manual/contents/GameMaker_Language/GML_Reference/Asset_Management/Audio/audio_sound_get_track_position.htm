音频_声音_get_track_position

这个函数将得到声音文件中声音的位置（以秒为单位），以便播放。声音只能是一个声音的单一实例（当使用 "或 "时，单个被播放的声音的索引可以存储在一个变量中。 audio_play_sound()或 audio_play_sound_at()函数时，单个被播放的声音的索引可以存储在一个变量中）。

语法。

audio_sound_get_track_position(index);

争论 类型 描述
index 声音实例ID The index of the sound to get the play position of.

返回。

Real

例子。

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

上面的代码检查一个轨道以获得它的起始位置，如果它不是0秒，它就把它设置为0秒。

返回。音频
