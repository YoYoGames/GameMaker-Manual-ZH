音频_set_master_gain

通过这个函数，你可以为一个特定的听众设置所有声音和音乐的全局音量的绝对值。默认的听众指数是0，但你可以使用函数 audio_get_listener_info() ，以获得目标平台的不同指数。增益值是基于一个从0（无声）到1（全音量）的线性尺度，并将影响所有通过该听众从你的游戏中播放的声音和音乐的相对音量。

语法。

audio_set_master_gain(listenerIndex, gain);

争论 类型 描述
listenerIndex 真实或音频听众ID The index of the listener to set the gain on.
gain 真实的 Value for the global volume (0 to 1).

返回。

N/A

例子。

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

上述代码loops ，通过可用的监听器，然后将其主增益设置为0.75。

返回。音频
