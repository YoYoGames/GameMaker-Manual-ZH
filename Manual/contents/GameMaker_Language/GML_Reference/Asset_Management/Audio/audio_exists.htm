音频_存在

这个函数返回一个声音是否存在(true)或不存在(false)。注意，如果你搜索的索引以前没有被初始化，这个函数会引起错误，因为它正在搜索不存在的asset 索引。要检查的声音可以是一个声音的单一实例（正在播放的单个声音的索引可以存储在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数）或一个声音asset 。

语法。

audio_exists(index);

争论 类型 描述
index 资产 The index of the sound to check the existence of.

返回。

Boolean

例子。

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

上面的代码检查是否存在一个声音。如果它存在，它将被设置为在loop 。

返回。音频
