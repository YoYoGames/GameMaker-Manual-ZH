音频恢复_同步组

如果正在播放的同步组之前被暂停，该函数将恢复该同步组（使用函数 audio_pause_sync_group).组的索引是你用函数创建组时返回的值 audio_create_sync_group().

注意：此功能不适用于HTML5目标平台。

语法。

audio_resume_sync_group(group_index);

争论 类型 描述
group_index 音频同步组ID The group index to resume.

返回。

N/A

例子。

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if (global.Pause)
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

上面的代码检查是否有 "P "键被按下，如果检测到，它将切换 "global.Pause "变量，然后检查它以暂停或恢复变量 "sg "中索引的同步组。

返回。音频同步化
