audio_sync_group_get_track_pos

这个函数返回给定同步组的当前播放位置。组的索引是你用函数创建该组时返回的值 audio_create_sync_group()时返回的值，而返回值是曲目播放的时间（秒）。

注意：此功能不适用于HTML5目标平台。

语法。

audio_sync_group_get_track_pos(group_index);

争论 类型 描述
group_index 音频同步组ID The group index to get the position of.

返回。

Real

例子。

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

上面的代码得到了在变量 "sg "中索引的同步组的时间位置，然后用这个值在屏幕上画出播放的时间。

返回。音频同步化
