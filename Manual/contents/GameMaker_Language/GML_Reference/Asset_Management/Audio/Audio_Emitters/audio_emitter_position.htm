音频发射器位置

通过这个功能，你可以改变一个音频发射器在3D音频空间中的位置。这个位置将以不同的方式影响声音，这取决于听众在音频空间中的位置（默认位置是（0，0，0）。参见 audio_listener_position()了解更多信息），所以举例来说，如果发射器的位置被设置为（100, 0, 0），而当前听众在（200, 0, 0），那么从发射器流出来的音频将出现在音频区域的左边。下面的图片显示了发射器及其与监听者的相对位置的可视化表示。

语法。

audio_emitter_position(emitter, x, y, z);

争论 类型 描述
emitter 音频发射器ID The index of the emitter to change.
x 真实的 The x position.
y 真实的 The y position.
z 真实的 The z position.

返回。

N/A

例子。

if speed > 0
{
    audio_emitter_position(s_emit, x, y, 0);
}

上面的代码检查实例的速度是否超过0，如果是，它就会将变量 "s_emit "中索引的音频发射器更新到当前的x/y位置。

返回。音频发射器
