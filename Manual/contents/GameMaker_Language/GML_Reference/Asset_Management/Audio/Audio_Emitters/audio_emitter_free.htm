语音识别器

通过这个函数，你可以从内存中删除指定的发射器。只要在room 或游戏中不再使用发射器，即：在实例的销毁事件或房间的结束事件中，就应该这样做，否则你可能会出现内存泄漏，导致游戏速度减慢并最终崩溃。

语法。

audio_emitter_free(emitter);

争论 类型 描述
emitter 音频发射器ID The index of the emitter to free.

返回。

N/A

例子。

if lives = 0
{
    audio_emitter_free(s_emit);
    room_goto(rm_Menu);
}

上述代码检查全局变量 "lives "的值，如果它返回0，它将销毁变量 "s_emit "中索引的发射器，然后转到变量 "rm_Menu "中索引的room 。

返回。音频发射器
