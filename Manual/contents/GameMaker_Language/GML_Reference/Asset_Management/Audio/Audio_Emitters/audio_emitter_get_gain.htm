音频发射器获取增益

这个函数返回给定音频发射器的当前增益（音量）设置，通常在0和1之间，其中0是无声，1是全音量。注意，在某些平台上，你可以有一个大于1的增益，尽管1的值被认为是 "全音量"，任何更大的值都可能引入音频削波。

语法。

audio_emitter_get_gain(emitter);

争论 类型 描述
emitter 音频发射器ID The index of the emitter to use.

返回。

Real

例子。

if audio_emitter_get_gain(emitter_player) < 1
{
    audio_emitter_gain(emitter_player, 1);
}

上面的代码检查一个给定的发射器的当前增益，如果它小于1，就设置为1。

返回。音频发射器
