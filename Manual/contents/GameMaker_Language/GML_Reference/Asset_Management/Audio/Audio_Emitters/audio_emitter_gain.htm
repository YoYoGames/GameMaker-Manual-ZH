audio_emitter_gain

这个函数设置声音的最大增益（音量）。声音的感知音量可以根据落差值和它相对于听众的位置而改变，但通过这个函数设置增益，全部音量将永远不会超过指定的增益值。下面的图片说明了当fall-off大于0时，增益是如何影响发射器的音量的。

这个函数将改变正在播放的声音的音量，以及通过给定发射器播放的所有后续声音。注意，在某些平台上，你可以有一个大于1的增益，尽管1的值被认为是 "全音量"，任何更大的值都可能引入音频削波或失真。

注意 最终的音量也会受到全局音频增益的影响，这个增益是由功能设置的。 audio_master_gain().

语法。

audio_emitter_gain(emitter, gain);

争论 类型 描述
emitter 音频发射器ID The index of the emitter to change.
gain 真实的 The maximum gain (default 1).

返回。

N/A

例子。

if up
{
    gain += 0.05;
    if gain >= 1 up = false;
}
else
{
    gain += 0.05;
    if gain <= 0 up = true;
}

audio_emitter_gain(s_emit, gain);

上面的代码将变量 "增益 "设置为不同的值，然后用同一个变量来设置变量 "s_emit "中索引的发射器的增益。

返回。音频发射器
