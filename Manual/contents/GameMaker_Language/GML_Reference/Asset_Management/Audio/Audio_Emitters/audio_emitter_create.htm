音频发射器_创建

这个函数创建一个新的音频发射器并返回它的索引。这个索引应该被保存在一个变量中，用于所有与这个发射器有关的后续函数，当不再需要它时，应该用函数将其从内存中删除 audio_emitter_free()以防止内存泄漏，这可能最终会使你的游戏崩溃。

音频发射器是以下列默认设置创建的。

  • 增益和间距为1
  • 一个只设置为默认监听器的监听器掩码(1)
  • 一个位置是（0，0，0）。
  • 速度为0（vx = 0, vy = 0, vz = 0）。

语法。

audio_emitter_create();

返回。

Audio Emitter ID

例子。

s_emit = audio_emitter_create();

上面的代码创建了一个新的音频发射器，并将其索引分配给变量 "s_emit"。

返回。音频发射器
