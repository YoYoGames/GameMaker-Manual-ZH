音频_发射器_set_listener_mask

这个函数可以用来设置发射器的比特掩码，这样所有通过发射器播放的声音就只能从指定的听众那里播放。你可以通过以下方式创建一个比特掩码 audio_get_listener_info()来创建一个位掩码，然后用位法 or ("|")来创建一个自定义的掩码，包括那些你需要声音播放的听众，然后把这个自定义掩码应用到发射器上。这个掩码将覆盖默认的掩码，或者你可能用函数 audio_set_listener_mask().

语法。

audio_emitter_set_listener_mask(emitterID, mask);

争论 类型 描述
emitterID 音频发射器ID The unique ID of the emitter to set the mask of
mask 真实的 The bitmask data to set for the sound

返回。

N/A

例子。

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

上面的代码创建了一个音频发射器，然后使用存储在全局变量中的掩码数据来设置听众的播放。

返回。音频发射器
