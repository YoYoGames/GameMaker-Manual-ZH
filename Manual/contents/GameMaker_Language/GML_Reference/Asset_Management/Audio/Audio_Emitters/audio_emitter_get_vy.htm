音频_发射器_获取vy

该函数返回给定音频发射器沿Y轴的当前速度。

语法。

audio_emitter_get_vy(emitter);

争论 类型 描述
emitter 音频发射器ID The index of the emitter to use.

返回。

Real

例子。

if audio_emitter_get_vy(emitter_player) != 0
{
    audio_emitter_velocity(emitter_player, 0, 0, 0);
}

上面的代码检查某个发射器的当前y速度，如果不等于0，就把它设置为0。

返回。音频发射器
