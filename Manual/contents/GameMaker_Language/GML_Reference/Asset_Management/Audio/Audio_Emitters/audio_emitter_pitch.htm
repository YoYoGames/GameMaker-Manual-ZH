音频发射器间距

这个函数可以用来改变从给定发射器发出的所有声音的音高。它是一个音高倍增器，即输入值将当前的音高乘以该量，所以默认值1是不改变音高的，而小于1的值将降低音高，大于1的值将提高音高。这个功能最好使用小的增量，因为任何低于0或高于5的值可能无论如何都听不出来。

已经在音频发射器上播放的声音不需要重新启动，就可以听到音调的变化。音调的改变会立即应用。

语法。

audio_emitter_pitch(emitter, pitch);

争论 类型 描述
emitter 音频发射器ID The index of the emitter to change.
pitch 真实的 The pitch multiplier (default 1).

返回。

N/A

例子。

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

上述代码将根据变量 "gear "的值，改变从变量 "s_emit "中索引的发射器播放的音频的音调。

返回。音频发射器
