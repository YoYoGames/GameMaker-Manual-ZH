音频销毁_流

如果你以前用函数从一个文件创建了一个音频流 audio_create_stream()创建了一个音频流，并且不再需要那个声音，你可以调用这个函数将它从内存中删除。在它被销毁后，任何对声音索引的进一步调用都会产生一个错误。应该注意的是，这将释放流，但在目标平台上，这可能不会显示在内存管理器中。这是因为GameMaker汇集了内存资源，以防止内存分配的开销，所以内存将保持分配，直到需要其他东西或重新使用一个新的流。

注意：此功能不适用于HTML5目标平台。

语法。

audio_destroy_stream(filename);

争论 类型 描述
filename 字符串 Path to the audio file that was streamed.

返回。

N/A

例子。

audio_destroy_stream(snd);

上面的代码从内存中删除变量 "snd "中索引的声音。

返回。音频
