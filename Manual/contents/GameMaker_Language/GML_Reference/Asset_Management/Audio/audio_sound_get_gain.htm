audio_sound_get_gain

这个函数将返回给定声音的当前增益值。这个声音可以是一个从正在播放的单个声音的索引中引用的声音，这个索引已经被储存在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数时，从正在播放的单个声音的索引中引用，或从asset 浏览器中的实际声音Asset 。增益通常以0到1的数值计算，但在某些平台上，你可以有一个大于1的增益，尽管1的数值被认为是 "全音量"，任何更大的数值都可能引入音频削波。

语法。

audio_sound_get_gain(index);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to get the gain of.

返回。

Real

例子。

if audio_sound_get_gain(snd_Music) != 1
{
    audio_sound_gain(snd_Music, 1, 0);
}

上述代码将改变从索引为 "snd_Music "的声音中播放的音频的增益，如果其增益不等于1。

返回。音频
