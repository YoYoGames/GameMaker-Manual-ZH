音频_get_master_gain

通过这个函数，您可以得到一个特定听众的所有声音和音乐的全局音量的绝对值。默认的听众索引是0，但你可以用这个函数 audio_get_listener_info()来获取目标平台上的不同指数。返回的增益值是基于一个从0（无声）到1（全音量）的线性刻度。注意，在某些平台上，你可以有一个大于1的增益，尽管1的值被认为是 "全音量"，任何更大的值都可能引入音频削波。

语法。

audio_get_master_gain(listenerIndex);

争论 类型 描述
listenerIndex 真实或音频听众ID The index of the listener to get the gain of.

返回。

Real

例子。

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var ind = info[? "index"];
    if audio_get_master_gain(ind) != 1
    {
        audio_set_master_gain(info[? "index"], 1);
    }
    ds_map_destroy(info);
}

上面的代码loops ，检查他们的增益是否为1，如果不是，就把每个人的增益设为1。

返回。音频
