audio_is_paused

这个函数将检查给定的声音，看它目前是否暂停了。这个声音可以是一个单一的声音实例（正在播放的单个声音的索引可以存储在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数）或一个声音asset ，在这种情况下，给定声音的所有实例将被检查，如果其中任何一个暂停，函数将返回 true ，否则将返回 false 。

语法。

audio_is_paused(index);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to check.

返回。

Boolean

例子。

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

上面的代码检查变量 "snd_Waterfall "中索引的声音目前是否暂停，如果它返回 true ，那么声音的播放将被恢复。

返回。音频
