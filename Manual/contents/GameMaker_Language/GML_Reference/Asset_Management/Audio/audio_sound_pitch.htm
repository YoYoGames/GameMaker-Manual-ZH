音频_声音_音高

这个函数可以用来改变一个给定声音的音高。这个声音可以是一个从正在播放的单个声音的索引中引用的声音，这个索引已经被储存在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数时，从正在播放的单个声音的索引中引用，或从asset 浏览器中的实际声音Asset 。如果它是一个正在播放的声音的索引，那么只有那个实例会被改变，但是当使用asset 浏览器中的一个声音Asset ，所有正在播放的声音asset 的实例都会被改变。

音高参数是一个音高乘数，即输入值将当前的音高乘以该量，所以默认值1是没有音高变化的，而小于1的值将降低音高，大于1的值将提高音高。这个功能最好使用小的增量，因为任何低于0或高于5的值可能无论如何都听不出来。值得注意的是，允许的总音高变化被限制在（1/256）-256个八度，所以任何超过或低于这个值的数值都不会被记录。

上面给出的钳制值是GameMaker试图钳制的范围，但这个值不保证在所有的目标平台上。iOS举例来说，GameMaker的钳制值是(1/256)-8，所以你可能需要在每个目标平台上进行实验，并有不同版本的声音资源，每个版本都预先进行了移位，如果你需要更高或更低的八度音值。

语法。

audio_sound_pitch(index, pitch);

争论 类型 描述
index 声音资产或 声音实例ID The index of the sound to change.
pitch 真实的 The pitch multiplier (default 1).

返回。

N/A

例子。

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
    case 1: audio_sound_pitch(s_engine, 0.8); break;
    case 2: audio_sound_pitch(s_engine, 0.9); break;
    case 3: audio_sound_pitch(s_engine, 0.95); break;
    case 4: audio_sound_pitch(s_engine, 1); break;
    case 5: audio_sound_pitch(s_engine, 1.2); break;
}

上面的代码将根据变量 "齿轮 "的值，改变从变量 "s_engine "中索引的声音播放的音调。

返回。音频
