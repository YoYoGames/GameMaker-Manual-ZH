音响_queue_sound

这个函数将把来自buffer 的数据添加到你之前用函数创建的音频queue 。 audio_create_play_queue().你指定要添加到的queue 索引和要获取数据的buffer ID，以及在buffer 中的位置（偏移）和要添加的字节数。一旦你把音频从buffer 添加到queue ，buffer 就不能被删除，除非你先用函数释放了queue 。 audio_free_play_queue()而buffer 的属性应该与你要添加声音的queue 的属性一致。

语法。

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

争论 类型 描述
queueIndex 音频Queue ID The index of the queue to add to.
bufferId 缓冲区ID The buffer to add to the queue.
bufferOffset 真实的 The offset within the source buffer to start from.
bufferLength 真实的 The length of the buffer (the number of the bytes in the buffer).

返回。

N/A

例子。

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

上述代码将在异步音频录制事件中被调用，并将一些录制的数据分配到一个buffer ，然后被添加到一个音频queue 。然后进行播放。

返回。音频缓冲器
