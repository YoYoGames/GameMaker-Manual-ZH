audio_get_recorder_count

这个函数将返回你的游戏当前可用的音频录音源（如麦克风等）的数量。因此，如果你有一个返回值，例如，4，那么你将有音频输入源0，1，2和3，其中一个值是你用来开始录制的，使用函数 audio_start_recording().当人们在设备上插拔麦克风或其他输入源时，这个值可以随时改变。请注意，你可以使用功能 audio_get_recorder_info()来获取所连接的每个设备的信息。

重要提示这个函数在Opera GX目标上总是返回1。

语法。

audio_get_recorder_count();

返回。

Real

例子。

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

上述代码检查可用的录音机，如果有1个或更多，就从第一个索引的录音机（源0）开始录音。

返回。音频缓冲器
