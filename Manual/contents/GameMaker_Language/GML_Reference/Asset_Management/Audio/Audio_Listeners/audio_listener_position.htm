音频听众位置

通过这个功能，你可以改变听众在三维音频空间中的位置。下面的示例图片显示了听众在音频空间中的默认位置。

注意如果你有多个监听器，使用函数 audio_listener_set_position().

正如你所看到的，默认位置是（0, 0, 0），但你通常会使用这个函数在你的游戏中随玩家移动监听器object ，从而改变玩家听到发射器产生的音频的方式，例如，在下面这个俯视游戏的图片中，玩家实例设置的监听器将导致来自各种发射器的音频随着玩家的移动而 "改变"。

语法。

audio_listener_position(x, y, z);

争论 类型 描述
x 真实的 The x position of the listener (default 0).
y 真实的 The y position of the listener (default 0).
z 真实的 The z position of the listener (default 0).

返回。

N/A

例子。

if speed > 0
{
    audio_listener_position(x, y, 0);
}

上面的代码检查播放器实例的速度是否超过0，如果是，它就把音频监听器更新到当前的X/Y位置。

返回。音频收听者
