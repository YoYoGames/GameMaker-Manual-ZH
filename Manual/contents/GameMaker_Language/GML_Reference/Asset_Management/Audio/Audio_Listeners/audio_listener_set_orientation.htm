音频听众设置方向

通过这个函数，你可以改变3D音频空间中给定监听器的方向。默认监听器的索引是0，但你可以使用函数 audio_get_listener_info()来获得目标平台的不同指数。

观察向量和上升向量是基于从给定的相对X、Y和Z位置解析的值，观察向量默认为（0，0，1），上升向量默认为（0，1，0），如下图所示。

用这个函数改变给定的听众方向将改变你的游戏玩家对游戏周围的音频发射器产生的声音的感知room 。在下面的例子中，当听众处于默认位置时，由发射器产生的声音似乎来自听众的下方和右侧，但随着听众的新位置和方向，它们现在将被视为来自上方和右侧。

语法。

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

争论 类型 描述
index 真正的或 音频听众ID The listener to set the orientation of.
lookat_x 真实的 The x look vector (default 0).
lookat_y 真实的 The y look vector (default 0).
lookat_z 真实的 The z look vector (default 1).
up_x 真实的 The x up vector (default 0).
up_y 真实的 The y up vector (default 1).
up_z 真实的 The z up vector (default 0).

返回。

N/A

例子。

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

上面的代码检索了摄像机视图[0]的视图矩阵，然后用它来设置音频监听器的位置和方向，监听器的ID存储在全局变量 "Player_Listener "中。

返回。音频收听者
