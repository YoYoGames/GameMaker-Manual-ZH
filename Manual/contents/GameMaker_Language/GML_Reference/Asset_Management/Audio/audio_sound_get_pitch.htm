audio_sound_get_pitch

这个函数可以用来获取一个给定的声音的音高。这个声音可以是一个从正在播放的单个声音的索引中引用的声音，这个索引已经被储存在一个变量中，当使用了 audio_play_sound()或 audio_play_sound_at()函数，或者是一个实际的声音asset ，从Asset 浏览器。

语法。

audio_sound_get_pitch(index);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to get the pitch of.

返回。

Real

例子。

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

上面的代码将改变从索引为 "snd_Explode "的声音中播放的音频的音调，如果它的音调不等于1。
