音响_set_listener_mask

这个函数可以用来设置一个声音的比特掩码，这样它就只能从指定的听众那里播放。你可以通过以下方法创建一个比特掩码 audio_get_listener_info()然后用位法 or （"|"）来创建一个自定义的掩码，其中包括那些你要求声音播放的听众，然后在你播放后将这个自定义掩码应用到声音上。这个掩码将覆盖默认的掩码，或者你可能用函数 audio_set_listener_mask().

这个函数只能在一个声音被设置为播放后使用，并且需要用于播放声音的函数所返回的声音ID。

语法。

audio_sound_set_listener_mask(soundID, mask);

争论 类型 描述
soundID 声音实例ID The unique ID of the sound to set the mask of
mask 真实的 The bitmask data to set for the sound

返回。

N/A

例子。

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

上面的代码播放了一个声音，然后用存储在全局变量中的掩码数据来设置听众的播放。
