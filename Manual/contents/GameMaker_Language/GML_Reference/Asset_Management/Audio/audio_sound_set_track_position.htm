音频_声音_设置音轨_位置

这个函数将为给定的声音ID或asset ，设置位置（以秒为单位）。

提供的 "索引 "可以是一个声音的单个实例（如从 audio_play_sound()返回的）或一个声音asset （例如，通过Asset 浏览器添加的）。这个函数的行为将取决于你所指定的索引的种类。

  • 如果是你使用的独特的声音ID，已经在播放，那么它的位置将立即改变到给定的时间。
  • 如果是声音asset ，那么所有给定的声音asset 的进一步播放将在新的时间开始。

这个函数只改变当前播放的声音的音轨位置，或者它的下一次播放（在提供声音asset 的情况下）。如果启用了循环播放的声音，以后的播放将总是从开始（0.0秒）开始，而不是从用这个函数定义的轨道位置开始。例如，在5.7秒处启动一个声音loop ，将从该点开始播放第一个声音，但之后将继续重复整个轨道，从开始到结束。

语法。

audio_sound_set_track_position(index, time);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to change.
time 真实的 The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

返回。

N/A

例子。

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

上面的代码播放了一个声音，然后使用返回的ID值，将声音的开始位置设置为32秒。

返回。音频
