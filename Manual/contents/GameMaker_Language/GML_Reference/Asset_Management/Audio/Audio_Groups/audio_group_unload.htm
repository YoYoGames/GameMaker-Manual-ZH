音频组卸载

这个函数将把所有被标记为属于给定音频组的声音卸载到内存中。如果开始卸载，该函数将返回 true ；如果组的ID无效，或者已经被标记为卸载，则返回 false 。注意，当这个函数被调用时，当前正在播放的任何音频将被停止。

语法。

audio_group_unload(groupID);

争论 类型 描述
groupID 音频组ID The index of the audio group to unload (as defined in the Audio Groups Window)

返回。

Boolean

例子。

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

上面的代码检查是否加载了一个音频组，如果加载了，就把它卸载。

返回。音频组
