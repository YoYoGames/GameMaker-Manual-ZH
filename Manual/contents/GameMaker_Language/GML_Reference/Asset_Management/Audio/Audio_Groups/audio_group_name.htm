音频组名称

这个函数将返回一个string ，包含给定的音频组的名称，用于显示或检查。当你在游戏选项中定义一个音频组时，你会给它一个独特的 "名字"，这实际上是一个常数，用作该组的ID值。这个函数所做的就是接受ID并返回一个你给的ID名称的string 。

语法。

audio_group_name(groupID);

争论 类型 描述
groupID 音频组ID The index value constant of the audio group to check (as defined in the Audio Groups Window)

返回。

String

例子。

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

上面的代码检索了给定的音频组常量的名称并在屏幕上显示。

返回。音频组
