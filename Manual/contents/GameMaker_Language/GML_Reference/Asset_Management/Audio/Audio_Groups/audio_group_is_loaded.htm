音频群组_is_loaded

这个功能将检查一个特定的音频组，看它是否已经加载到内存中，准备使用。

语法。

audio_group_is_loaded(groupID);

争论 类型 描述
groupID 音频组ID The index of the audio group to check (as defined in the Audio Groups Window)

返回。

Boolean

例子。

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

上面的代码检查是否加载了一个音频组，如果加载了，就把它卸载。

返回。音频组
