音频组_设置增益

通过这个功能，你可以在给定的时间长度内淡化一组声音，或者可以用来即时设置组的增益。

时间是以毫秒为单位的，该功能要求你输入一个最终的增益水平，以便在该时间结束时使该组达到。这个增益可以在0（无声）和1（全音量）之间，而且刻度是线性的，例如0.5的值是一半的音量。注意，在某些平台上，你可以有一个大于1的增益，尽管1的值被认为是 "全音量"，任何更大的值都可能引入音频削波。

语法。

audio_group_set_gain(groupID, volume, time);

争论 类型 描述
groupID 音频组ID The index of the audio group to stop (as defined in the Audio Groups Window)
volume 真实的 The final value for the group volume.
time 真实的 The length of the change in gain in milliseconds.

返回。

N/A

例子。

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

上面的代码检查 "空格 "键，然后在5秒内将 "audiogroup1 "的所有音频淡化为0。

返回。音频组
