音频组加载

这个函数将把所有被标记为属于给定音频组的声音加载到内存中。如果加载开始，该函数将返回 true ，如果组的ID无效，或者已经被标记为加载，则返回 false 。该函数是异步的，所以当音频在后台加载时，你的游戏将继续运行。这意味着它也会触发一个异步加载/保存事件，通知你整个组已经被加载到内存中，现在可以使用这些声音。

语法。

audio_group_load(groupID);

争论 类型 描述
groupID 音频组ID The index of the audio group to load (as defined in the Audio Groups Window)

返回。

Boolean

例子。

if !audio_group_is_loaded(audiogroup_level1)
{
    audio_group_load(audiogroup_level1);
}

上面的代码检查是否已经加载了一个音频组，如果没有，就加载它。

返回。音频组
