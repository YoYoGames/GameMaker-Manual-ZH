音频组加载进度

这个函数将检查一个音频组的加载进度，并返回一个0到100之间的（近似）值。

语法。

audio_group_load_progress(groupID);

争论 类型 描述
groupID 音频组ID The index of the audio group to check (as defined in the Audio Groups Window)

返回。

Real

例子。

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

上面的代码检查一个变量，如果它返回 true ，那么就会画出一些文本，显示正在加载的音频的进度。

返回。音频组
