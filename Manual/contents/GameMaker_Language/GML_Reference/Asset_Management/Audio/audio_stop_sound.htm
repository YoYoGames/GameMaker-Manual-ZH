音频_stop_sound

如果当前正在播放给定的声音，这个函数将停止它。这个声音可以是一个单一的声音实例（正在播放的单个声音的索引可以被存储在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数）或者是一个声音asset ，在这种情况下，所有给定的声音实例都将被停止。

语法。

audio_stop_sound(index);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to stop.

返回。

N/A

例子。

if !global.SFX
{
    audio_stop_sound(snd_Waterfall);
}
else
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

上述代码检查全局变量 "SFX"，如果它返回 false ，它将停止当前正在播放的变量 "snd_Waterfall "中索引的声音，如果它返回 true ，它将loop 。

返回。音频
