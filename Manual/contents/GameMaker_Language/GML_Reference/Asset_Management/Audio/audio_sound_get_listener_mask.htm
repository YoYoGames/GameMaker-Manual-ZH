音频_声音_get_listener_mask

这个函数将返回比特掩码数据，它定义了声音应该从哪个音频听众处播放。更多信息请参见音频监听器一节。

语法。

audio_sound_get_listener_mask(soundID);

争论 类型 描述
soundID 声音实例ID The unique ID of the sound to get the mask of

返回。

Real

例子。

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
    audio_sound_set_listener_mask(snd, global.PlayerMask);
}

上面的代码先播放一个声音，然后检查这个声音的听众掩码数据，如果它和存储在全局变量中的数据不一样，它就用存储在全局变量中的掩码数据来设置播放的听众（s）。

返回。音频
