音频_恢复_声音

有了这个功能，你可以恢复当前暂停的任何声音（在使用功能后 audio_pause_sound()).这个声音可以是一个单一的声音实例（正在播放的单个声音的索引可以存储在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数）或一个声音asset ，在这种情况下，所有给定的声音实例将被重新启动。

语法。

audio_resume_sound(index);

争论 类型 描述
index 声音资产或声音实例ID The index of the sound to resume.

返回。

N/A

例子。

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

上面的代码检查是否有键盘键 "P "被按下，如果检测到，它将全局变量 "Pause "设置为 true 或 false，然后要么暂停变量 "snd_Waterfall "中索引的声音，要么从暂停的状态恢复声音。

返回。音频
