层_序列_get_instance

通过这个函数，您可以提供sequence 元素的 ID - 如由 layer_sequence_create()或某个图层元素函数所返回的ID，它将返回sequence 实例 结构。你可以在这个页面上找到更多关于sequence 实例结构的格式。

语法。

layer_sequence_get_instance(sequence_element_id)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target

返回。

Sequence Instance Struct

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

上面的代码创建了一个新的sequence 元素，然后检索其sequence 实例ID。然后用它来改变sequence 实例的播放速度刻度属性。

返回。序列层数
