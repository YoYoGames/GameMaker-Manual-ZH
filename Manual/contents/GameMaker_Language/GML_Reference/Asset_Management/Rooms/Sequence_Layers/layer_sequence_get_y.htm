layer_sequence_get_y

通过这个函数，你提供了sequence 元素的ID--如由 或由某个层元素函数返回的--它将返回 。 layer_sequence_create()或某一图层元素函数所返回的ID，它将返回该room ，在游戏中的当前Y位置sequence 。

语法。

layer_sequence_get_y(sequence_element_id)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target

返回。

Real

例子。

if layer_sequence_get_y(title_sequence) != room_height / 2
{
    layer_sequence_y(title_sequence, room_height / 2);
}

上述代码检查存储在变量 "title_sequence "中的sequence 元素ID的Y位置，如果它没有被设置为room 高度的一半，它就被设置为这个值。

返回。序列层数
