层_序列_存在

通过这个函数，你可以检查在给定的层上是否存在一个sequence 元素。你提供的图层ID可以是图层名称的string --如在room 编辑器中定义的--或唯一的图层ID--如函数所返回的那样 layer_get_id()以及sequence 元素的ID--如由 layer_sequence_create()或由一个层元素函数返回，如果给定的元素存在，它将返回 true ，否则返回 false 。

语法。

layer_sequence_exists(layer_id, sequence_element_id)

争论 类型 描述
layer_ID 字符串 或层ID The unique ID or name of the layer to check
sequence_element_id 序列元素ID The unique ID value of the sequence element to target

返回。

Boolean

例子。

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

上面的代码检查给定的sequence 元素是否存在，如果不存在则在给定的层上创建一个新的sequence ，然后暂停它。

返回。序列层数
