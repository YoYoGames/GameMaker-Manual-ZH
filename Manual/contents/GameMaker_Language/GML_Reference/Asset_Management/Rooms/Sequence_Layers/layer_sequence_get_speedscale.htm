层_序列_获取_速度刻度

通过这个函数，你提供了sequence 元素的ID--如由 layer_sequence_create()或某一图层元素函数返回的，它将返回当前的播放速度比例。这是用于减慢或加快播放速度的乘数值。1的值是默认值，小于1的值意味着播放速度变慢，大于1的值意味着播放速度变快。

语法。

layer_sequence_get_speedscale(sequence_element_id)

争论类型 描述
sequence_element_id The unique ID value of the sequence element to target

返回。

例子。

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

上面的代码检查了存储在变量 "title_sequence "中的sequence 元素ID的当前播放头速度刻度，如果它没有被设置为1，就被设置为这个值。

返回。序列层数
