层_序列_yscale

通过这个函数，你可以为给定的sequence 元素设置Y刻度。你提供的sequence 元素ID是由 layer_sequence_create()或某一图层元素函数返回的ID，以及要在Y轴上设置的新比例，sequence 将按这个量进行缩放。1表示不缩放（1:1），较小的值将会缩小（例如0.5，将是sequence 宽度的一半），较大的值将会放大，负值将会使sequence 围绕其原点翻转并缩放，除非使用的值正好是-1（在这种情况下，sequence 只是围绕其原点翻转，没有缩放）。

非常重要的一点是，对包含任何使用旋转的实例的sequence 元素应用不均匀的缩放（例如：X轴缩放3，Y轴缩放2），可能会导致实例绘制、碰撞、剔除和许多其他问题。基本上，如果你的sequence 依赖于任何实例的属性，那么我们不建议你将不均匀的缩放与实例的旋转结合在一起。

还要注意的是，你不应该为使用这个函数的sequence 中的任何实例改变 image_xscale \ image_yscale \ image_angle \ x \ y 变量，因为在调用这个函数后，sequence 一更新，它们就会被覆盖。

语法。

layer_sequence_yscale(sequence_element_id, xscale)

争论类型 描述
sequence_element_id The unique ID value of the sequence element to target
yscale The new Y axis scale value to apply to the sequence element

返回。

例子。

if seq_scale < 2
{
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
}

上面的代码检查了 seq_scale 变量中的值，如果它小于2，那么它就把它加进去，然后用这个值来设置变量 my_seq 中引用的sequence 元素的X和Y比例。

返回。序列层数
