层_序列_get_angle

通过这个函数，你可以提供sequence 元素的ID--就像通过 "图层元素 "函数返回的那样。 layer_sequence_create()或某个图层元素函数返回的，它将返回游戏中sequence 元素的当前角度，room 。注意，角度是以度为单位返回的，0º是向右，90º是向上，180º是向左，270º是向下。

语法。

layer_sequence_get_angle(sequence_element_id)

争论类型 描述
sequence_element_id The unique ID value of the sequence element to target

返回。

例子。

var _ang = layer_sequence_get_angle(title_sequence)
if _ang > 0
{
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
}

上面的代码检索了sequence ，该元素的ID存储在变量 "title_sequence "中，如果它不是0，那么就从当前角度中减去1。

返回。序列层数
