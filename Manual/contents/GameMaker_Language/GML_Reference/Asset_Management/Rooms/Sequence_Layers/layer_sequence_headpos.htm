层_序列_头部位置

通过这个函数，你可以把一个sequence 元素的播放头位置设置为一个特定的帧。你提供sequence 元素的ID，如由 layer_sequence_create()或某个层元素函数返回的ID，以及要设置的新位置。注意，位置是以帧为单位的，如果你设置的值大于总帧数（或小于0），那么实际的最终播放头的位置将取决于已经选择的sequence 播放的类型，遵循这些规则。

  • 没有循环。播放头将夹在loop （如果给定的值是负的，则夹在开头）的末端。
  • 循环播放。播放头将回绕到sequence ，再加上额外的帧数，例如：如果你在一个30帧的sequence ，将播放头设置为第45帧的位置，实际播放头的位置最终将是15。如果你给一个负值，那么sequence ，播放头将以相反的方向卷起。
  • 乒乓式循环。播放头将前进到最后，然后返回超过总帧数的额外帧数，反转播放方向，例如。如果sequence 是30帧，你把播放头设置为40，最后的播放头位置将是20，方向将被逆转（向下到0）。请注意，你永远不应该给一个负值的乒乓播放sequence ，因为它可能有undefined 的结果。

以这种方式设置磁头位置不会停止播放，除非暂停播放，否则sequence ，就会从新的播放磁头位置继续播放。

语法。

layer_sequence_headpos(sequence_element_id, position)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target
position 真实的 The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

返回。

N/A

例子。

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

上面的代码得到了分配给运行该代码的实例层的所有元素的ID。然后，该代码检查是否有返回的元素是sequence assets ，如果是，那么它们的播放头的位置就被设置为0。

返回。序列层数
