层_序列_暂停

用这个函数你可以暂停播放给定的sequence 。你提供的sequence 元素的ID是由 layer_sequence_create()或某个层元素函数返回的ID，该函数将暂停sequence ，直到你使用函数再次开始播放。 layer_sequence_play().

语法。

layer_sequence_pause(sequence_element_id)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target

返回。

N/A

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if global.Pause
{
    layer_sequence_pause(_seq);
}

上面的代码创建了一个新的sequence ，并将其ID存储在一个局部变量中。然后，它检查游戏是否暂停，如果是，就暂停播放sequence 。

返回。序列层数
