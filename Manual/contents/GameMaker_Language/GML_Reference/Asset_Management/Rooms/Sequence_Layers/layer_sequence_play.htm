层_序列_播放

用这个函数你可以开始播放给定的sequence 。你提供的sequence 元素ID是由 layer_sequence_create()或某个层元素函数返回的ID，该函数将播放sequence ，如果需要，你可以使用函数暂停播放。 layer_sequence_pause().

重要的是!如果你的sequence 中有任何实例，这些实例不应该改变它们的image_xscale /image_yscale / image_angle /x /y 变量，因为在开始播放后，sequence 更新每一步时，它们会被覆盖。

语法。

layer_sequence_play(sequence_element_id)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target

返回。

N/A

例子。

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if layer_get_element_type(a[i]) == layerelementtype_sequence
        {
            if global.Pause
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

上面的代码在按下一个键时检查游戏是否已经暂停。如果游戏暂停了，那么它loops ，通过当前层（调用实例的层）上的所有sequence 元素，暂停它们的播放，如果游戏没有暂停，那么loop ，重新开始它们的播放。

返回。序列层数
