层_序列_角度

通过这个函数，你可以为给定的sequence 元素设置旋转的角度。你提供的sequence 元素ID是由 layer_sequence_create()或某个层元素函数返回的ID，以及要设置的新角度，sequence 将围绕其原点旋转到新位置。GameMaker 中的角度是以0º向右计算的，并按逆时针方向进行--因此90º是向上，180º是向左，270º是向下--而sequence 的默认角度是0º。

注意，你不应该为使用这个函数的sequence 中的任何实例改变 image_xscale \ image_yscale \ image_angle \ x \ y 变量，因为在调用这个函数后，一旦sequence 更新，它们将被覆盖。

语法。

layer_sequence_angle(sequence_element_id, angle)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target
angle 真实的 The new angle to rotate the sequence element to

返回。

N/A

例子。

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

上面的代码检查了 current_angle 变量中的值，如果它小于90，那么它就把它加上去，然后用这个值来设置变量 my_seq 中引用的sequence 元素的角度。

返回。序列层数
