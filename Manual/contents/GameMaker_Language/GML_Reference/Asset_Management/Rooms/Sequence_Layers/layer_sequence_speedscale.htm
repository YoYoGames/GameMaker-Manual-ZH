层_序列_速度刻度

用这个函数你可以改变给定的sequence 的播放速度。你提供sequence 元素的ID，如由 layer_sequence_create()或某个层元素函数所返回的ID，然后提供一个速度刻度值。这个值是一个乘数，其中1是默认的播放速度，小于1的值会减慢播放速度，大于1的值会加快播放速度，例如：0.5的值会是一半的播放速度，而2的值会是两倍的播放速度。

语法。

layer_sequence_speedscale(sequence_element_id, speedscale)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target
speedscale 真实的 The speed scale to use (a multiplier), default is 1

返回。

N/A

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

上面的代码在调用实例的层上创建了一个新的sequence ，然后将其速度刻度值设置为0.75（四分之三的默认播放速度）。

返回。序列层数
