layer_sequence_headdir

用这个函数你可以为给定的sequence 播放头设置方向。你提供sequence 元素的ID，如由 layer_sequence_create()或某个层元素函数返回的ID，然后给出播放头的方向，它应该是下列常数之一。

恒定 描述 价值
seq_dir_right sequence ，将按照从左到右的递增顺序播放帧。 1
seq_dir_left sequence ，将按照从右到左的递减顺序播放帧。 -1

语法。

layer_sequence_headdir(sequence_element_id, direction)

争论 类型 描述
sequence_element_id 序列元素ID The unique ID value of the sequence element to target
direction 序列方向常数 The playhead direction, a constant, listed above

返回。

N/A

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

上面的代码创建了一个新的sequence ，并将其ID存储在一个局部变量中。然后这个ID被用来设置播放头的方向，以减少帧数（从右到左播放）。

返回。序列层数
