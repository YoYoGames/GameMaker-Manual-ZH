层_序列_get_xscale

通过这个函数，你提供了sequence 元素的ID--就像由 或由某个层元素函数返回的那样--它将返回游戏中 元素沿X轴的当前比例。 layer_sequence_create()或某一图层元素函数返回的，它将返回游戏中sequence 元素沿X轴的当前比例room 。

语法。

layer_sequence_get_xscale(sequence_element_id)

争论类型 描述
sequence_element_id The unique ID value of the sequence element to target

返回。

例子。

var _xs = layer_sequence_get_xscale(title_sequence)
if (_xs < 1)
{
    _xs += 0.01;
    layer_sequence_xscale(title_sequence, _xs);
}

上面的代码检索了sequence ，该元素的ID存储在变量 "title_sequence "中，沿着X轴的当前比例，如果它小于1，那么0.01将被添加到当前的X比例。

返回。序列层数
