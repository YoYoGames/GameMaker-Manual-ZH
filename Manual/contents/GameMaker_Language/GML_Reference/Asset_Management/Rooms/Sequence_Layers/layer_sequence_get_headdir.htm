层_序列_get_headdir

通过这个函数，你提供了sequence 元素ID--如由 layer_sequence_create()或某个图层元素函数返回的ID，它将返回当前播放头的方向，它将是下面列出的常数之一。

语法。

layer_sequence_get_headdir(sequence_element_id)

争论类型 描述
sequence_element_id The unique ID value of the sequence element to target

返回。

恒定 描述 价值
seq_dir_right sequence ，按照从左到右的递增顺序播放帧。 1
seq_dir_left sequence ，按照从右到左的递减顺序播放帧。 -1

例子。

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

上面的代码检查了存储在变量 "title_sequence "中的sequence 元素ID的当前播放头方向，如果它没有被设置为 seq_dir_left ，它就被设置为这个值。

返回。序列层数
