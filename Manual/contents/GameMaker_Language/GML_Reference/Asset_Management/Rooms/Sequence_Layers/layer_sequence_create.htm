layer_sequence_create

通过这个函数，你可以在给定的层上创建一个sequence asset 的实例。你提供的图层ID可以是图层名称的string （如在room 编辑器中定义的）或唯一的图层ID（如该函数返回的）。 layer_get_id()，以及在room 中创建sequence 的X和Y位置，最后是要创建的sequence 的ID。sequence ID是你在Asset 浏览器中为sequence 定义的名称常量。该函数将返回sequence 元素的唯一ID，然后可以在所有进一步的层函数中使用sequences ，或者可以使用函数检索sequence 实例struct 。 layer_sequence_get_instance().

值得注意的是，如果sequence 包含任何object 轨道，它们的实例将在sequence 元素本身被创建后立即被创建，无论它们的asset 键在Dope Sheet上的位置如何。Sequence 控制器只是简单地切换实例的可见性，根据轨道的asset 键的位置和持续时间来隐藏和显示它，而不会 重复地创建和销毁它。因此，当sequence 元素被创建时，实例将运行它们的创建事件，而不是当它们的资产键开始时。

语法。

layer_sequence_create(layer_id, x, y, sequence_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the instance layer to target
x 真实的 The x position in the room to create the sequence at
y 真实的 The y position in the room to create the sequence at
sequence_id 序列资产 The sequence asset to use, as defined in the Asset Browser

返回。

Sequence Element ID

例子。

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

上面的代码在 "背景 "层上创建了一个新的sequence ，然后暂停它。

返回。序列层数
