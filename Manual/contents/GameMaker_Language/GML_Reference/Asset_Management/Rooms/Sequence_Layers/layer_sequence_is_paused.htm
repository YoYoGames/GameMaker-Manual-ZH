层_序列_is_paused

通过这个函数，你提供了sequence 元素ID--如由 layer_sequence_create()或某个层元素函数返回的ID--它将检查sequence 目前是否暂停，如果暂停则返回 true ，如果不暂停则返回 false 。

语法。

layer_sequence_is_paused(sequence_element_id)

争论类型 描述
sequence_element_id The unique ID value of the sequence element to target

返回。

例子。

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

上面的代码检查存储在变量 "title_sequence "中的sequence 元素ID是否暂停，如果是，就开始播放它。

返回。序列层数
