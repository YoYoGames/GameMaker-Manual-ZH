房间_goto

这个函数允许你转到你的游戏项目中的任何room ，不管是用代码还是在Asset 浏览器中创建。你提供room 的索引（存储在room 名称的变量中，或者作为一个变量从函数返回 room_add()).请注意，room ，直到调用该函数的事件结束时才会改变，所以如果在同一事件中，调用该函数后的任何代码仍将运行。这个函数也会触发房间结束事件。

注意 Room ID不是基于它们在Asset 浏览器或Room 管理器中的顺序，因此你应该避免直接提供一个数字值。相反，使用你想引用的asset 的房间常量（这只是它的名字）或通过函数检索它。

语法。

room_goto(index);

争论类型 描述
index The index of the room to go to.

返回。

例子。

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

上面的代码将检查一个全局变量，并根据它所持有的值改变room 。

返回。房间
