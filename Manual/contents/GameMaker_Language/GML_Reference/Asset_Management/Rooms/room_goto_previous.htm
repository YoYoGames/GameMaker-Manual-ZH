房间_转到_上一个

通过这个函数，你可以使你的游戏进入到游戏编译时房间管理器中列出的前一个房间。如果这个room ，将抛出一个错误，游戏将被强制关闭。注意，room ，直到调用该函数的事件结束时才会改变，所以如果在同一事件中，调用该函数后的任何代码仍会运行。这个函数也会触发房间结束事件。

语法。

room_goto_previous();

返回。

例子。

if room_exists(room_previous(room))
{
    room_goto_previous();
}

上述代码将检查在当前代码之前是否有另一个room ，如果有，它将转到该room 。

返回。房间
下一步。 room_restart
