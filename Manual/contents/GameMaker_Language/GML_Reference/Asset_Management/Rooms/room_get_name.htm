房间名称（Room_get_name

这个函数可以用来返回指定的room ，作为一个string 。请注意，这只是一个string ，不能用来直接引用room ，为此，你需要房间索引。然而，你可以使用这个string ，用返回的string ，加上函数来获得房间的索引。 asset_get_index().

语法。

room_get_name(index);

争论类型 描述
index The index of the room to check the name of.

返回。

例子。

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

上面的代码将获得当前room ，并在屏幕上画出它。

返回。房间
