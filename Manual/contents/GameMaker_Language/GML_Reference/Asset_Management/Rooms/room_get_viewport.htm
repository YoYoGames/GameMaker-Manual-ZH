房间里的视口

通过这个函数，你可以在一个room ，而不是当前的 ，检索一个视口的细节。你给出room ID和要检索的视图端口的索引（从0到7），该函数将返回一个包含5个索引的数组，其中。

  • [0] =可见 (布尔值: true / false)
  • [1] = X位置（实际）。
  • [2] = Y位置（实数）。
  • [3] = 宽度（实数）
  • [4] = 高度（实数）

语法。

room_get_viewport(rm, vind);

争论 类型 描述
rm 房间资产 The index of the room to get viewport data from
vind 真实的 The index of the view port to get

返回。

Array (5 elements: visible, x, y, width, height)

例子。

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

上面的代码为给定的room 检索视图端口数据，然后检查该端口是否被标记为可见。如果不是，则对视图端口数据进行设置，使其可见。

返回。房间
