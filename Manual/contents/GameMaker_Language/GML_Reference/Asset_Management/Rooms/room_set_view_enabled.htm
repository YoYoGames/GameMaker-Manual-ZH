启用的房间_设置视图

通过这个功能，你可以启用(true)或禁用(false)你游戏中的任何room ，除了当前的一个。

语法。

room_set_view_enabled(index, val);

争论 类型 描述
index 房间资产 The index of the room to set.
val 布尔型 Whether to enable (true)或禁用(false)视图，在给定的room 。

返回。

N/A

例子。

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

这将使room ，在 "global.myroom "中索引的视图。

返回。房间
