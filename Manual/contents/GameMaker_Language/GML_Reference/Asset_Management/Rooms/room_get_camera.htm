房间_获取摄像机

通过这个函数，你可以得到分配给特定视图的摄像机的唯一索引ID，而不是当前的room 。你给出要使用的room ，要使用的视图端口（从0到7），该函数将返回一个摄像机索引。

语法。

room_get_camera(rm, vind);

争论 类型 描述
rm 房间资产 The index of the room to get the view camera of
vind 真实的 The index of the view port to get the camera of

返回。

Camera ID

例子。

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
    room_set_camera(rm_Game, 0, global.MainCam);
}

上面的代码将一个新创建的room 中的摄像机分配给了视图端口[0]。

返回。房间
