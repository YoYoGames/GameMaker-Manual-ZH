room_last

这个只读变量返回游戏中最后一个room 的索引（这是由rooms 出现在房间管理器中的顺序决定的，而不是由它们被创建的顺序决定的）。请注意，这个变量不会识别或考虑使用room_add() 或room_duplicate() 动态添加的rooms 。

语法。

room_last

返回。

Room Asset

例子。

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

上面的代码检查是否有键被按下，如果有，它就会转到游戏中的最后一个room 。
