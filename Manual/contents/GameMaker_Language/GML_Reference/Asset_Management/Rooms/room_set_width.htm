房间的宽度

通过这个函数，你可以改变（或设置）你游戏中任何room ，除了当前的宽度。

语法。

room_set_width(index, w);

争论 类型 描述
index 房间资产 The index of the room to set the width of.
w 真实的 The new width of the room in pixels.

返回。

N/A

例子。

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

这将创建一个新的room ，并在变量 "global.myroom "中存储其索引。然后，它将设置其宽度为640像素，高度为480像素，标题为 "游戏室"，其持久性为 "false"。

返回。房间
