瓦片图_x

这个函数控制room asset 地图元素在图层上沿X轴的位置tile 。你给tile 地图元素的ID（当你用tile 地图元素创建时得到的 layer_tilemap_create()或使用函数 layer_tilemap_get_id())，然后设置要使用的x值（基于room 坐标）。

语法。

tilemap_x(tilemap_element_id, x);

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to change
x 真实的 The x position for the tile map

返回。

N/A

例子。

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

tile 上述代码获得了分配给图层 "Asset_sky "的asset 地图的ID值，然后将其X位置设置为0和room 的宽度之间的一个随机值。
