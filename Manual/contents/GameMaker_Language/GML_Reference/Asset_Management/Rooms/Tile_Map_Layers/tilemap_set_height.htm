tilemap_set_height

这个函数可以用来调整一个tile 地图元素的大小。你给tile 地图元素的ID（当你用tile 创建一个地图元素时，你会得到这个ID layer_tilemap_create()或使用函数 layer_tilemap_get_id())，以及tile 地图的新高度，单位是tile 。

语法。

tilemap_set_height(tilemap_element_id, height)

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to set the height of
height 真实的 The height value (in "cells")

返回。

N/A

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_height(map_id) != room_height div 16
{
    tilemap_set_height(map_id, room_height div 16);
}

上面的代码检查特定tile 地图的宽度，如果它不是正确的尺寸，那么宽度就会被设置。
