瓷砖_设置旋转

这个函数可以用来设置一组给定的瓷砖数据，使其旋转90度或不旋转tile 。你给出瓷砖数据，这些数据可以用函数检索到 tilemap_get()，然后将旋转参数设置为 true ，如果你想让tile 旋转，或者 false ，如果你想让tile 保持其默认的、未旋转的状态。该函数将返回一个修改后的瓷砖数据集，然后可以使用 tilemap_set()函数来应用。

语法。

tile_set_rotate(tiledata, rotate)

争论 类型 描述
tiledata 瓦片数据 The tile-data to set
rotate 布尔型 Set to true 进行旋转， ，使其保持原状。 false

返回。

Real

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

上面的代码从给定的图层中得到tile 地图ID，然后得到鼠标下tile 的x和y单元的位置。然后，这个位置被用来获取瓦片数据，该数据被旋转，然后被用来再次设置tile 。
