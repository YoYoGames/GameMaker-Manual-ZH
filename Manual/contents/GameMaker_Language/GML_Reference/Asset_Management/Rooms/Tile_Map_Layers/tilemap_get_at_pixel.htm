瓷砖地图_获取像素点（tilemap_get_at_pixel

使用这个函数，您可以从tile 地图元素的某个位置（在room ）检索tile 数据。你给tile 地图元素的ID（当你用tile 地图元素创建时，你会得到这个ID layer_tilemap_create()或当你使用函数 layer_tilemap_get_id())以及房间里的x和y位置，以获取tile 数据，该函数将返回tile 数据 "blob"。这个数据本质上是一个位掩码，包含tile 索引、翻转/旋转/镜像布尔运算和任何已经应用的掩码数据（详见 tilemap_set_mask()详细信息），然后产生的数据值可以在 tile_* 函数中使用，以改变一个tiles 属性。如果你需要从一个特定的tile 单元获取tile 数据，你应该使用函数 tilemap_get()来代替。

重要的是!如果tiles 地图中的tile 一直没有变化（即：它们没有旋转或翻转等......），那么tile set 数据的返回值将完全等于tile 上的tile set 的索引。因此，你可以使用tiles 中索引为1的一个tile 创建tile set 的 "碰撞地图" - 例如 - 然后使用此函数检查1或0（一个空tile ）来计算碰撞。

语法。

tilemap_get_at_pixel(tilemap_element_id, x, y);

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to get the tile-data of
x 真实的 The position along the x-axis to get the tile data from (in room pixels)
y 真实的 The position along the y-axis to get the tile data from (in room pixels)

返回。

Tile Data (-1 if there is an error)

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

上面的代码在 "Tiles_Sky "图层上得到tile 地图 "Clouds "的ID，然后用这个ID从tile 的位置（64, 128）得到数据。然后这个tile 的数据被翻转，再用来设置tile 地图上的tile 。
