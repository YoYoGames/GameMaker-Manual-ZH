瓷砖地图_清空

使用这个函数，你可以清除/设置一个给定瓦片地图上的所有tiles 。你给tile 地图元素的ID（当你用tile 创建一个地图元素时，你会得到这个ID layer_tilemap_create()或当你使用函数 layer_tilemap_get_id())，然后提供你希望清除该层的tile 数据。默认值为0将清除图层中所有的tiles （基本上使所有的tiles "空"），而你可以使用专用的 tile_* 函数来创建你自己的tile 数据来清除tile 地图。

语法。

tilemap_clear(tilemap_element_id, tiledata)

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to change
tiledata 瓦片数据（实数） The tile data to use to clear the layer

返回。

N/A

例子。

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

上面的代码得到了在tile 编辑器中创建的room 地图的ID值，然后用 "空 "tiles 将其清除。
