层_tilemap_exists

你可以用这个函数来检查并查看任何给定的图层上是否存在tile 地图元素。您可以提供图层的ID（您在创建图层时使用 layer_create())或图层名称(作为string - 这将对性能产生影响)和tile 地图元素ID(当您使用tile 地图元素创建时，您会得到它。 layer_tilemap_create()或者当你使用函数 layer_tilemap_get_id())，如果该元素存在，该函数将返回 true ，如果不存在，则返回 false 。

注意：该函数在当前目标room 的范围内工作--默认情况下是调用该函数的room ，可以用函数设置 layer_set_target_room().

语法。

layer_tilemap_exists(layer_id, tilemap_element_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to check

返回。

Boolean

例子。

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

上面的代码检查图层 "tilemap_trees"，看给定的tile 地图元素是否存在，如果存在，那么它就被销毁（但不包括图层）。

背面。瓦片地图层
