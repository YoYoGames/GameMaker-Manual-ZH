layer_tilemap_create

通过这个函数，你可以给一个图层分配一个瓦片集资源，作为你项目中的tile 。你可以提供图层的ID（当你使用 layer_create())或图层名称（作为string - 这将对性能产生影响），然后提供一个初始（x, y）位置，将tile 地图添加到room ，使用的tile set 资源，然后提供tile 地图的宽度和高度（单元格）（即：宽度为20，高度为10将创建一个200个单元格的tile 地图，宽度为20，高度为10），单元格本身的大小由选择的tile set 定义。值得注意的是，你不能将tiles 放在tile 地图中的负数位置，所以所有tiles 必须放在0到宽-1，0到高-1的单元格空间内。

语法。

layer_tilemap_create(layer_id, x, y, tileset, width, height)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
x 真实的 The x position of the tile map in the room
y 真实的 The y position of the tile map in the room
tileset 瓷砖套装资产 The Tile Set asset to be used
width 真实的 The width tile map (in cells)
height 真实的 The height of the tile map (in cells)

返回。

Tile Map Element ID

例子。

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

上面的代码创建了一个新的图层，然后给它添加了一个新的tile 地图元素，将tile 地图的位置设置为（0,0），以及要使用的tile set 和tile 地图的宽度和高度。
