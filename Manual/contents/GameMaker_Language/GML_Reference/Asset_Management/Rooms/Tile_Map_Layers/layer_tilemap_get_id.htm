layer_tilemap_get_id

这个函数可以用来检索一个图层上tile 地图元素的唯一ID值。你可以提供图层的ID（你在创建图层时使用 layer_create())或图层名称（作为string - 这将对性能产生影响），该函数将返回与该图层上的tile 地图元素相关的ID值。请注意，这个函数是专门为在tile 中添加的IDE 地图而设计的，因为如果你通过使用函数的代码将tile 地图添加到一个图层上 layer_tilemap_create()，那么它将返回添加的tile 地图元素的唯一ID。如果给定的瓦片地图ID不正确或瓦片地图不存在，该函数将返回-1。

语法。

layer_tilemap_get_id(layer_id)

争论 类型 描述
layer_id 绳索或 图层ID The unique ID value of the layer to target (or the layer name as a string)

返回。

Tile Map Element ID or -1

例子。

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

上面的代码将得到名为 "tilemap_trees "的图层的ID，然后用它来得到该图层上tile 地图元素的ID。这个ID然后被用来从该层中删除tile 地图。
