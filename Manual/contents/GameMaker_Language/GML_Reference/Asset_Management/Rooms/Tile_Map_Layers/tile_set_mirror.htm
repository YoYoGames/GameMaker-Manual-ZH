tile_set_mirror

这个函数可以用来设置一组给定的瓦片数据，以反映tile ，或者不反映。你给出瓷砖数据，它可以用函数来检索 tilemap_get()，然后将镜像参数设置为 true ，如果你想镜像tile ，或者 false ，如果你想让tile 处于默认的、未镜像的状态。该函数将返回一个修改后的瓦片数据集，然后可以使用 tilemap_set()函数来应用。

语法。

tile_set_mirror(tiledata, mirror)

争论类型 描述
tiledata the tile-data to set
mirror Set to true 镜像和 ，让它保持原样。 false

返回。

Tile Data (Real)

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

上面的代码从给定的图层中得到tile 地图ID，然后得到鼠标下tile 的x和y单元的位置。然后，这个位置被用来获取瓦片数据，该数据被镜像，然后被用来再次设置tile 。
