tilemap_set_at_pixel

这个函数可以用来将图层上tile 地图元素中的一个单元设置为一个新的tile ，使用tile 的实际位置来改变room 。你给tile 地图元素ID（当你使用tile 创建一个地图元素时得到的 layer_tilemap_create()或当你使用函数 layer_tilemap_get_id())，然后设置tile 数据以及在room 中的位置。你可以使用函数检索tile 数据 tilemap_get_at_pixel()然后使用 tile_ 函数来改变它，然后再使用这个函数来设置它。如果成功设置了tile ，该函数将返回 true ，如果有问题没有设置，则返回 false 。

语法。

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to change
tiledata 瓦片数据（实数） The tile set data to set
xcell 真实的 The x position (in the room)
ycell 真实的 The y position (in the room)

返回。

Boolean

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

上面的代码在 "Tiles_Sky "图层上得到了tile 地图的ID，然后用这个ID从tile 的位置（4, 4）得到数据。然后这个tile 的数据被翻转，然后再用于设置tile 地图上的tile 。
