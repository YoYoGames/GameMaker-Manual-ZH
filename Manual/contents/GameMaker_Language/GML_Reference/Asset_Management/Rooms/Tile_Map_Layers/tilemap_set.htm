瓷砖图案_设置

这个函数可以用来将图层上tile 地图元素中的任何单元（网格方块）设置为新的tile 。你给tile 地图元素ID（当你用tile 地图元素创建时得到的）。 layer_tilemap_create()或使用函数 layer_tilemap_get_id())，然后是要设置的tile 数据，以及在tile 地图中的位置。你可以使用函数检索tile 数据 tilemap_get()然后在使用此函数设置单元格之前，使用 tile_ 函数来改变它。如果成功设置了tile ，该函数将返回 true ，如果有问题而没有设置，则返回 false 。

语法。

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to change
tiledata 瓦片数据（实数） The tile data to set
xcell 真实的 The cell (grid) position to set along the x-axis
ycell 真实的 The cell (grid) position to set along the y-axis

返回。

Boolean

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

上面的代码在 "Tiles_Sky "图层上获得了tile 地图的ID，然后用这个ID从tile 的单元格（0，0）处获得数据。然后这个tile 的数据被翻转，然后再用于设置tile 地图上的tile 。
