tilemap_y

这个函数控制room asset 地图元素在图层上沿Y轴的位置tile 。你给tile 地图元素的ID（当你用tile 地图元素创建时得到的 layer_tilemap_create()或使用函数 layer_tilemap_get_id())，然后设置要使用的y值（基于room 坐标）。

语法。

tilemap_y(tilemap_element_id, y);

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to change
y 真实的 The y position for the tile map

返回。

N/A

例子。

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

上面的代码得到了分配给图层 "Asset_sky "的tile 地图asset 的ID值，然后将其Y位置设置为0和room 的高度之间的随机值。
