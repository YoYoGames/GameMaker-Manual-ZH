tilemap_tileset

使用这个函数，您可以改变分配给一个图层上的特定tile set 地图元素的asset tile 。您可以给tile 地图元素ID（当您使用tile 地图元素创建时，您会得到这个ID layer_tilemap_create()或当你使用函数 layer_tilemap_get_id())，然后提供一个tile set 的索引，tile 地图将被赋予新的sprite 。

语法。

tilemap_tileset(tilemap_element_id, tileset_index)

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to change
tileset_index 瓷砖套装资产 The new tile set index to use

返回。

N/A

例子。

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

上面的代码检查当前分配给 "Tiles_trees "层上的tile set 地图的tile ，如果不是 "ts_Nighttime"，那么该tile set 被分配给tile 地图。
