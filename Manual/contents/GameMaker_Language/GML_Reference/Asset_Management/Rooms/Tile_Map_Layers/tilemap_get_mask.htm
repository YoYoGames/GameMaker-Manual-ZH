瓷砖图谱_获取掩码

这个函数可以用来检索给定tile 地图的位掩码值。你给tile 地图元素的ID（当你用tile 创建一个地图元素时，你会得到这个ID。 layer_tilemap_create()或使用函数 layer_tilemap_get_id())，该函数将返回当前的屏蔽值，如果有错误则返回-1，如果没有指定屏蔽则返回0。关于tile 地图位掩码的进一步信息，见函数 tilemap_set_mask().

语法。

tilemap_get_mask(tilemap_element_id);

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map element to get the mask of

返回。

Real (0 for no mask, -1 for an error)

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_mask(map_id, new_mask);
}

上面的代码从给定的图层中获得tile 地图ID，然后检查与之相关的掩码值。如果它与变量 "new_mask "中定义的值不一样，那么它就被设置为该值。
