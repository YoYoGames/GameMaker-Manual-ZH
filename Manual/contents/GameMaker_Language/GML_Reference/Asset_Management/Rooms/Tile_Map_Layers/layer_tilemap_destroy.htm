layer_tilemap_destroy

这个函数将销毁给定的tile 地图元素。你提供tile 地图的ID（当你用tile 创建地图时，你会得到这个ID。 layer_tilemap_create()或在使用图层ID的同时使用 layer_get_tilemap_id())，这将会删除它。注意，这不会删除图层，只是删除其中的tile 地图，如果tile 地图是在room 编辑器中添加的，那么下次你离开room ，然后返回时，tile 地图将再次被重新创建。然而，如果room 是持久的，tile 地图将被删除，除非room 的持久性再次被关闭。

语法。

layer_tilemap_destroy(tilemap_element_id)

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tile map to be destroyed

返回。

N/A

例子。

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

上面的代码检查图层 "Tiles_trees"，看给定的tile 地图元素是否存在，如果存在，那么它就被销毁（但不包括图层）。
