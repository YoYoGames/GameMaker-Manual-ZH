瓷砖地图_获取单元格_y_at_pixel

使用这个函数，您可以通过给出tile 中的相对Y轴位置来检索单个room 地图单元的Y轴位置。您可以给出tile 地图元素的ID（当您使用tile 创建一个地图元素时，可以得到这个ID）。 layer_tilemap_create() 或使用函数 layer_tilemap_get_id())，以及房间内的x和y位置，该函数将返回该点在tile 地图中的单元格的y位置。 注意，如果该值在tile 地图区域之外，并且没有单元格可用，它将返回-1。

语法。

tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y);

争论类型 描述
tilemap_element_id The unique ID value of the tile map element to get the cell y position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

返回。

(y-axis cell position or -1 if there is an error)

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

上述代码使用检索到的tile 地图ID，从一组room 坐标中获得单元格的位置，然后将那里发现的任何tile 的数据存储在一个实例变量中。
