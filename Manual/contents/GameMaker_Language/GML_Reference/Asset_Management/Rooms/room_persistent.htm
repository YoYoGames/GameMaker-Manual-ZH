持续性的房间

这个变量可以用来获取和设置当前flag 的持久性room 。如果设置为 true ，room 被认为是持久的，在这种情况下，每次你离开room ，再次回来时，该room 中的实例的状态将被保持。然而，如果它被标记为 false ，每次你回到room ，它将被重置为其初始状态。你应该注意，一个持久的room ，比一个普通的room ，要使用更多的内存，所以不建议在你的游戏中拥有太多这样的实例。

语法。

room_persistent;

返回。

例子。

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

上述代码检查变量 "lives"，如果它小于1，它将设置room 持久性为 false ，然后改变rooms 。

返回。房间
