fx_set_single_layer

这个功能是用来启用或禁用一个滤波器/效果的 "单层 "模式。默认情况下，一个滤波器/效果会应用到它被分配到的层和该层下面的所有层，但是你可以在一个FXstruct 上启用单层模式，以确保它只应用到它被分配到的层。

你指定一个FXstruct （如从 fx_create() 或 layer_get_fx() 返回）和一个布尔值来启用(true)或禁用(false)单层模式。

下面的视觉效果显示了在禁用单层模式（这是所有FX层的默认行为）的情况下使用一个过滤器，以及启用单层模式的同一过滤器。

注意当你为一个特效启用单层模式时，请确保你把它应用在一个不是特效层的图层上，因为特效层本身不包含任何东西，所以没有滤镜/特效会被看到。

语法。

fx_set_single_layer(filter_or_effect, enable);

争论类型 描述
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) 或禁用 (false) 单层模式

返回。

例子。

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

上面的代码创建了一个新的screenshake FX，在它上面启用了单层模式，然后把它应用到room 。这意味着屏风抖动过滤器将只应用于"ShakeyThings "层。
