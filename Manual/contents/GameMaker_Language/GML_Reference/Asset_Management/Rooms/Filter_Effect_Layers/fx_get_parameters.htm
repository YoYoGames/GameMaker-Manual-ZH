fx_get_parameters

这个函数用来检索一个包含给定FXstruct 的所有参数的结构。你指定一个FXstruct （如从 fx_create() 或 layer_get_fx() 返回），该函数返回一个struct ，其中包含适用于该过滤器/效果的所有参数的变量。使用向量（即值组）的参数将以数组形式出现。

你可以通过调用fx_get_parameter_names() ，在一个单独的数组中获得所有参数的名称，或者在这个页面上找到它们。FX类型和参数。

改变这个struct 中的值不会将这些改变应用到过滤器/效果中；你需要调用fx_set_parameters() ，将修改后的struct 应用到过滤器/效果中，这样才能使任何改变生效。

语法。

fx_get_parameters(filter_or_effect);

争论 类型 描述
filter_or_effect 外汇结构 The FX struct for which the parameters will be returned

返回。

Struct (or -1 on failure)

例子。

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

上面的代码为一个假定应用了 "Colour Tint "滤镜的图层获取FXstruct ，并通过调用 fx_get_parameters() ，检索其参数struct 。之后，它通过使用 sin() 和 current_time 创建一个振荡值，然后将其用于着色效果的RGBA值，这样它就会不断地产生动画。RGBA数组被分配到参数struct 中的 g_TintCol 变量中，然后通过调用struct 将其应用到FXstruct 中。 fx_set_parameters().
