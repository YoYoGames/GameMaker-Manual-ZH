layer_clear_fx

这个函数用来清除应用于指定层的FXstruct 。你指定你要修改的层的ID或名称，这个函数将删除应用于该层的FXstruct ，完全禁用其效果。

如果你用这个函数暂时禁用一个效果，并需要恢复它的层，你应该首先通过调用 layer_get_fx() ，为该层保存FXstruct ，然后调用这个函数在该层上清除它。以后你可以通过调用 layer_set_fx() ，把它重新应用到该层，这将重新启用该效果。

语法。

layer_clear_fx(layer_name_or_id);

争论 类型 描述
layer_name_or_id 绳索或 图层ID The name or ID of the layer to clear the FX for

返回。

N/A

例子。

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

上面的代码检查 "E "键是否被按下，然后检查图层 "EffectLayer "是否有滤镜/效果；在这种情况下，它将其FXstruct 在一个临时实例变量中，并清除该图层的任何滤镜/效果。当 "F "键被按下时，它会检查临时变量中是否有-1以外的东西，在这种情况下，它会将该特效重新应用到该层，并将临时变量重置为-1。所有这些本质上允许你在需要的时候启用和禁用一个效果，而不会丢失其参数。
