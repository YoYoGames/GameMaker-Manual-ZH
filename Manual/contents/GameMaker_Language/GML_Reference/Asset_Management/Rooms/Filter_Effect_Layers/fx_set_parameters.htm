fx_set_parameters

这个函数是用来改变一个滤波器/效果器的参数。你指定一个FXstruct （从 fx_create() 或 layer_get_fx() 返回）和一个包含其参数的struct （从 fx_get_parameters() 返回）。这将使你的改变在FXstruct 所指定的层中可见。

你的参数struct ，不需要包含适用于过滤器/效果的每一个参数，只有在给定的struct 中存在的变量才会被更新为过滤器/效果。

语法。

fx_set_parameters(filter_or_effect, parameter_struct);

争论类型 描述
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

返回。

例子。

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

上面的代码为一个假定应用了 "Colour Tint "滤镜的图层获取FXstruct ，并通过调用struct ，检索其参数。 fx_get_parameters().之后，它通过使用 sin() 和 current_time 创建一个振荡值，然后将其用于着色效果的RGBA值，这样它就会不断地产生动画。RGBA数组被分配给参数struct 中的 g_TintCol 变量，然后通过调用 fx_set_parameters() ，将struct 应用回FXstruct 。
