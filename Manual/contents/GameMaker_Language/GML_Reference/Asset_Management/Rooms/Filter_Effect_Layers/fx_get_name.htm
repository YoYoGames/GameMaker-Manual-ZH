fx_get_name

这个函数是用来从一个过滤器/效果的struct 中获取其名称。你指定要读取的FXstruct （如从 fx_create() 或 layer_get_fx() 返回的），该函数将其名称作为string 。然后这个名称可以被传递到fx_create() ，以创建一个新的FXstruct ，使用相同的过滤器/效果类型。

语法。

fx_get_name(filter_or_effect);

争论类型 描述
filter_or_effect The FX struct to get the name of

返回。

例子。

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

上面的代码运行一个 for循环运行所有存在于room 的层，并检查每个层是否有特效struct 。如果一个图层有特效struct ，它通过调用 fx_get_name() ，检查该过滤器/特效的名称；如果它等于"_filter_tintfilter"，意味着它是一个 "Colour Tint "过滤器，它将把其色调改为蓝色。
