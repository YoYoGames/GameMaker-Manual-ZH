启用的是 "Layer_enable_fx"。

这个函数启用/禁用分配给一个Room 图层的滤波器/效果。你可以指定你想修改的图层的ID或名称，然后指定一个布尔值，告诉你该特效应该被启用（真）还是被禁用（假）。

传入false不会从图层中移除特效，而只是让它不可见。使用 layer_clear_fx() ，从一个图层中删除特效。

同样的，如果没有特效应用到图层，传入true也不会有任何作用。使用 layer_set_fx() 来给一个图层应用特效。

语法。

layer_enable_fx(layer_name_or_id, enable);

争论 类型 描述
layer_name_or_id 绳索或 图层ID The name or ID of the layer to modify
enable 布尔型 Whether to enable or disable the FX

返回。

N/A

例子。

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

如果实例的HP值小于或等于3，上面的代码就会启用去饱和特效层，否则就会禁用它。这表明玩家的HP很低，通过对屏幕进行去饱和处理。
