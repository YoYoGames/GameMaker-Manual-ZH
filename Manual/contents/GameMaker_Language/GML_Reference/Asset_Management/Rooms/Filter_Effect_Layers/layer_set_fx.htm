层_设_计

这个功能是用来将一个FXstruct 指派给一个Room 图层。你指定你要修改的层的ID或名称，然后指定一个FXstruct （如从 fx_create() 或 layer_get_fx() 返回），它将被应用到指定的层，使效果在room 中可见。

如果你指定的图层不是滤镜/效果图层，而是其他类型的图层（如实例图层、Asset 图层、Tile 图层等），那么效果将应用于该图层的内容，以及它下面的任何图层。

如果指定的层已经有一个FXstruct ，它将被新的层所取代。

注意在一个script 的根层调用这个函数不会有任何作用，因为scripts 是在第一个room 加载之前执行的。你可以先设置一个目标房间，然后再调用这个函数。

语法。

layer_set_fx(layer_name_or_id, filter_or_effect);

争论 类型 描述
layer_name_or_id 绳索或 图层ID The name or ID of the layer to modify
filter_or_effect 外汇结构 The FX struct to apply to the layer

返回。

N/A

例子。

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上面的代码从"_filter_tintfilter "效果中创建了一个新的FXstruct ，它是在Room Editor中找到的 "Colour Tint "过滤器。它给它的"g_TintCol "参数分配了一个值，即色调的颜色，由于它是一个 vec4 ，它需要一个包含四个值的数组（对应于它的红、绿、蓝和alpha值）。色调的FXstruct ，然后应用于一个FX层。
