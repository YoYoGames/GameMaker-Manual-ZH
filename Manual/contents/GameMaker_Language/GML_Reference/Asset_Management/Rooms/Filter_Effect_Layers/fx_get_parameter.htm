fx_get_parameter

这个函数用来从一个FXStruct 检索一个参数的值。你指定要读取的FXstruct （如从 fx_create() 或 layer_get_fx() 返回）和作为string 的参数名称，然后该函数返回其当前值。这可能是一个实值，或者是一个数组，如果该参数是一个向量（这是一组数值，比如一个存储RGBA颜色的 vec4 ）。

在这个页面上找到滤波器/效果器的参数名称。FX类型和参数。

语法。

fx_get_parameter(filter_or_effect, parameter_name);

争论 类型 描述
filter_or_effect 外汇结构 The FX struct to get the parameter from
parameter_name 字符串 The name of the parameter to get the value of (as a string)

返回。

Real or Array

例子。

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

上面的代码从一个假定应用了 "Colour Tint "过滤器的图层中检索FXstruct ，并获得其"g_TintCol "参数的值，该参数返回一个包含其色调颜色（红、绿、蓝和α）的4个值的数组。然后它把这个值打印到输出日志中。
