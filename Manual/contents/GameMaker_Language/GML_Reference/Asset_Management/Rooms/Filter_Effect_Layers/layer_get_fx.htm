层级_获取_fx

这个函数是用来检索一个图层的FXstruct 。你可以指定你想要的目标层的ID或名称，该函数将返回一个包含其应用效果信息的struct 。这个struct 将类似于你从 fx_create() 得到的struct ，而函数 fx_get_parameter/s和 fx_set_parameter/s 可以在它上面使用，以读取和修改其参数。

如果指定的图层没有应用过滤器/效果，该函数将返回-1。

语法。

layer_get_fx(layer_name_or_id);

争论 类型 描述
layer_name_or_id 字符串 或层ID The name or ID of the layer to read

返回。

FX Struct (or -1 if not found)

例子。

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

上面的代码运行一个 for循环运行所有存在于room 的层，并检查每个层是否有特效struct 。如果一个图层有特效struct ，它就会通过调用该过滤器/特效的名称来检查。 fx_get_name()如果它等于"_filter_tintfilter"，意味着它是一个 "Colour Tint "过滤器，它就将其色调改为蓝色。
