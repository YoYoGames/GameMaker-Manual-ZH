fx_create

这个功能是用来为一个给定的效果类型创建一个新的FXStruct ，然后可以应用到一个图层。

你可以传入这个函数的滤波器/效果的名称列在这一页。所有过滤器/效果类型。

使用方法

你可以通过调用以下命令来读取所创建的效果的参数 fx_get_parameter/sstruct来读取创建的效果的参数，并通过调用 fx_set_parameter/s 来为它们分配新的值。

注意，这个函数返回的struct ，将不包含任何可见的成员，只能通过一个过滤器/效果函数来修改。

功能故障

如果struct ，由于任何原因创建失败，该函数将返回-1；如果指定的FX名称错误或者FX没有加载到游戏包中，可能会发生这种情况。

为了确保这个功能正常工作，请确保你所需要的FX已经加载到你的游戏包中，这一点在 "过滤器和效果功能"页上有解释。

如果这些条件没有得到满足，即你没有传入一个string literal，并且过滤器/效果不存在于任何rooms ，那么函数将无法加载该过滤器/效果，并将返回-1。

语法。

fx_create(filter_or_effect_name);

争论 类型 描述
filter_or_effect_name 字符串 The name of the filter/effect to create (get them from All Filter/Effect Types).

返回。

FX Struct or -1 (on failure)

例子。

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上面的代码从"_filter_tintfilter "效果中创建了一个新的FXstruct ，它是在Room Editor中找到的 "Colour Tint "过滤器。它给它的"g_TintCol "参数分配了一个值，这是色调的颜色，由于它是一个 vec4 ，它需要一个包含四个值的数组（对应于它的红、绿、蓝和alpha值）。色调的特效struct ，然后应用于一个特效层。
