fx_set_parameter

这个函数用来改变给定的FXstruct 中的一个参数的值。你指定一个FXstruct （从 fx_create() 或 layer_get_fx() 返回），一个参数名称作为string ，以及它的新值。这将使你的改变在FXstruct 所分配的层中可见。

如果是向量，你可以为参数传入多个参数，但是你也可以简单地指定一个数组作为唯一的参数，它将包含你的多个值。

在这个页面上找到滤波器/效果器的参数名称。FX类型和参数。

语法。

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

争论 类型 描述
filter_or_effect 外汇结构 The FX struct to modify
parameter_name 字符串 The name of the parameter to change the value of (as a string)
val... 真实或 阵列 The new value(s) for that parameter as a real or array

返回。

N/A

例子。

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上面的代码从"_filter_tintfilter "效果中创建了一个新的FXstruct ，它是在Room Editor中找到的 "Colour Tint "过滤器。它给它的"g_TintCol "参数分配了一个值，这是色调的颜色，由于它是一个 vec4 ，它需要一个包含四个值的数组（对应于它的红、绿、蓝和alpha值）。色调的FXstruct ，然后应用于一个FX层。

请注意，你也能够把值作为单独的参数传递给向量，而不是一个数组。

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
