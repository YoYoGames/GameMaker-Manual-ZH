fx_get_single_layer

该函数返回所提供的滤波器/效果是否使用单层模式 (true) 或不使用 (false) 。参见 fx_set_single_layer()获取关于单层模式的信息。

你指定一个FXstruct （如从 fx_create() 或 layer_get_fx() 返回），该函数返回一个布尔值，告诉你FX的单层模式是否被启用。

语法。

fx_get_single_layer(filter_or_effect);

争论类型 描述
filter_or_effect The FX struct to read

返回。

Boolean

例子。

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

上面的代码从一个名为"Distort "的图层中获取FXstruct ，并检查它是否启用了单层模式。如果没有，它就为该特效启用单层模式。
