层_fx_is_enabled

该函数返回某层的过滤器/效果是否被启用。

你指定你想要的目标层的ID或名称，该函数返回一个布尔值，表明该层的FX是启用（真）还是禁用（假）。

语法。

layer_fx_is_enabled(layer_name_or_id);

争论 类型 描述
layer_name_or_id 绳索或 图层ID The name or ID of the layer to check

返回。

Boolean

例子。

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

上面的代码 "重生 "了这个实例，把它移到原来的位置并重新填充它的HP。然后，它检查去饱和特效层是否启用，如果是，就禁用它。
