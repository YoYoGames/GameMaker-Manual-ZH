room_set_viewport

通过这个函数，你可以为你游戏中的任何room ，除了当前的那个，设置视口属性。你提供要设置视口的room 索引，视口索引（从0到7），然后视口是否应该在room 开始时可见（设置为 true 使视口可见，否则为 false ）。之后，你设置x和y坐标（对应于视口的左上角位置），然后设置宽度和高度（像素）。

语法。

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

争论 类型 描述
rm 房间资产 The index of the room to set
vind 真实的 The index of the view port to set
vis 布尔型 The visibility of the view port (true 是可见的， 是不可见的) false
xport 真实的 The x position for the view port in the room
yport 真实的 The y position of the view port in the room
wport 真实的 The width (in pixels) of the view port
hport 真实的 The height (in pixels) of the view port

返回。

N/A

例子。

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

这将在room ，在变量 "global.myroom "中索引设置视图端口[0]属性。

返回。房间
