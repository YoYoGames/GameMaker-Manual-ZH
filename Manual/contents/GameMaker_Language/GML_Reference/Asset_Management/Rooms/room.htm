房间

room 这个变量持有你的游戏正在运行的当前room 的索引。这不是一个只读变量，但是改变这个变量不会改变当前room 的索引值，而是改变room ，使之与你设置的变量的索引相符。在这样做的时候应该小心，因为如果你改变变量的索引是无效的，游戏将抛出一个错误并关闭。一般来说，使用 room_goto()来改变rooms 。

注意：Room ID不是基于它们在Asset 浏览器或Room 管理器中的顺序，因此你应该避免直接提供一个数字值。相反，使用你想引用的asset 的房间常数（这只是它的名字）或通过函数检索它。

语法。

room;

返回。

Room Asset

例子。

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

上面的代码将检查当前room 是否是第一关，在这种情况下，将播放该关的音乐。

返回。房间
