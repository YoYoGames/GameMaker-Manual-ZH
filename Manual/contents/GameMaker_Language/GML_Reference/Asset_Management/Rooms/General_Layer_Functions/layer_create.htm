创建层

这个函数用于在当前的room 中创建一个新的图层。你提供新图层所占的深度，该函数返回唯一的图层ID值，然后可以用来在进一步的函数中访问该图层。你还可以提供一个可选的 "名称 "参数，这将使你能够给该层一个特定的名称（如string ），你可以使用这个名称string ，而不是使用层ID来访问该层，不过需要注意的是，使用名称strings 会比只使用ID值对性能产生更大的影响。如果你不指定图层的名称，那么创建的图层将被分配一个格式为"_layer_XXX"的名称，其中 "XXX "是一个用于给图层一个唯一的名称的十六进制值。

当给定一个深度值时，该值将是离摄像机越 "远 "的层越高，离摄像机越 "近 "的层越低，所以如果你有三个深度为-100、0、100的层，这些层将按100、0、-100的顺序绘制，所以 "顶层"（即最接近摄像机视图的层，所以绘制在其他所有层之上）将是深度为-100的层。

下图显示了深度对层的作用示意图。

值得注意的是，虽然你可以给一个深度指定一个以上的图层，但你无法保证同一深度的图层会以任何给定的顺序绘制，因此最好只给任何给定深度指定一个图层。还要注意的是，最小和最大的图层深度是-16000到16000，超出这些深度的东西将不会被渲染。如果你需要一个超出这个范围的深度，那么你将需要使用函数 layer_force_draw_depth().

语法。

layer_create(depth, [name])

争论 类型 描述
depth 真实的 The depth that the layer has to be created at.
name 字符串 OPTIONAL 赋予该层的名称（一个 ）。string

返回。

Layer ID

例子。

global.Bullet_Layer = layer_create(-100);

上述代码将创建一个深度值为-100的新图层，并将其ID值分配给一个全局变量。

返回。一般层的功能
