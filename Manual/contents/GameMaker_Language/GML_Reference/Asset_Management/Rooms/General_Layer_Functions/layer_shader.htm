图层_着色器

通过这个函数，你可以给任何给定的层分配一个shader 资源，然后该层将使用该shader 。你可以提供该层的唯一ID值或该层的名称（作为一个string - 这将对性能产生影响），以及要使用的shader 的ID。shader 必须事先在Asset 浏览器中创建，然后将shader 索引（shader 资源的名称）传递给这个函数。如果指定的图层有实例被添加到它，那么shader 将被应用到该实例使用的所有绘制事件--例如，如果该实例有一个Draw GUI Begin事件，那么shader 将被自动应用到它。shader 也会影响在该层上绘制的任何其他图形元素，如sprite assets 或tile 地图。

请注意，该函数不是为了在任何绘制事件或步骤事件中调用，而是只需要在房间创建代码中的room ，或在实例的创建事件/ 房间开始事件中调用。

语法。

layer_shader(layer_id, shader)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
shader 着色器资产 The shader index to assign to the layer

返回。

N/A

例子。

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

上述代码将把shader 资源 "shd_Sepia "分配给所有绘图的给定层。

返回。一般层的功能
