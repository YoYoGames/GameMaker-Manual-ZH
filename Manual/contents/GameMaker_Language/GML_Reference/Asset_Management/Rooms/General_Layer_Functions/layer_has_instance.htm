layer_has_instance

这个函数可以用来检查一个给定的实例是否当前被分配到给定的图层。你可以提供层的ID（当你使用 layer_create())或图层名称（作为string - 这将对性能产生影响）以及要检查的实例的ID。你也可以给一个 object_index （即：object 浏览器中的Asset 的名字），该函数将检查该object 的任何实例是否在给定的层上。如果该实例在该层上，该函数将返回 true ，如果不在，则返回 false 。

语法。

layer_has_instance(layer_id, instance_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
instance_id 实例ID The unique instance ID or the object index of the instance to check for

返回。

Boolean

例子。

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

上面的代码将检查给定的图层是否包含任何object "obj_Bullet_Parent "的实例，如果没有，它将销毁该图层。

返回。一般层的功能
