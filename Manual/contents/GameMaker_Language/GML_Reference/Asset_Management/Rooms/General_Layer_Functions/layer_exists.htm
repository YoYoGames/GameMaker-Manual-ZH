layer_exists

这个函数可以用来检查给定的图层是否存在。你可以提供图层的ID（当你使用 layer_create())或图层名称（作为string - 这将对性能产生影响），如果它存在，该函数将返回一个布尔值： true ，如果不存在，则返回 false 。

注意：该函数在当前目标room 的范围内工作--默认情况下是调用该函数的room ，可以用函数设置 layer_set_target_room().

语法。

layer_exists(layer_name)

争论 类型 描述
layer_name 字符串 或层ID The name of the layer (a string or ID value)

返回。

Boolean

例子。

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

上面的代码将检查存储在全局变量中的图层是否实际存在，如果不存在，那么就创建它。

返回。一般层的功能
