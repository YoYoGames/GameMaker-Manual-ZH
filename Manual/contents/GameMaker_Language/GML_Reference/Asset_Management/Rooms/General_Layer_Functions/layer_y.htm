层_y

你可以使用这个函数来设置图层在当前范围内的y位置room 。你可以提供图层的ID（当你使用 layer_create())或图层名称(作为string - 这将对性能产生影响)，该函数将沿着room 的纵轴将该图层移动给定的像素数。

语法。

layer_y(layer_id, y)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to set the y position of
y 真实的 The y position in the room to set the layer to

返回。

N/A

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

上面的代码检查给定的图层位置，如果它没有被设置为（0，0），那么它就被设置为该位置。

返回。一般层的功能
