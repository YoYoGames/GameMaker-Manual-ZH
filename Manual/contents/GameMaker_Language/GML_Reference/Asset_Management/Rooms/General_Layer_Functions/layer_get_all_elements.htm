层级_获取所有元素

你可以用这个函数来获取给定图层的元素ID。你提供图层ID（当你使用 layer_create())或图层名称(作为string - 这将对性能产生影响)，该函数将返回一个ID数组，其中数组中的每个条目是该图层中一个元素的唯一ID。例如，如果该层是一个Asset 层，该数组将被分配给该层的每个sprite asset 的ID值所填充。注意，使用代码来处理图层意味着你可以把不同的元素类型分配给同一个图层--所以你可以有sprite assets ，同时还有实例，例如--在这种情况下，你可以使用函数 layer_get_element_type()来获取该ID所涉及的元素类型。

语法。

layer_get_all_elements(layer_id)

争论 类型 描述
layer_id 绳索或 图层ID The unique ID value of the layer to get the elements from (or the layer name as a string)

返回。

Array of Layer Element IDs

例子。

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if (layer_get_element_type(a[i]) == layerelementtype_sprite)
    {
        layer_sprite_destroy(a[i])
    }
}

上面的代码获得分配给运行代码的实例层的所有实例元素的ID。然后，代码检查是否有返回的元素是sprite assets ，如果是，则将其销毁。

返回。一般层的功能
