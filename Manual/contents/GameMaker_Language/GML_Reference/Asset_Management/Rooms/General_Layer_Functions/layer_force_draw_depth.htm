层数_强制绘制_深度

这个函数强制所有图层以指定的z深度绘制。这并不改变图层的渲染顺序，它们仍然是按深度顺序绘制的，它只是改变了使用的Z值。一般来说，你不需要担心这个问题，但是如果你的图层的深度超出了法定范围（-16000到16000），那么它们就不会被渲染，所以你可以强制Z深度为一个合理的值--比如说0--它们都会被渲染好。请注意，这通常只适用于GameMaker以前版本的遗留项目，在这些项目中，你的绘制深度可能高于或低于允许的层范围。

语法。

layer_force_draw_depth(force, depth)

争论 类型 描述
force 布尔型 Whether to enable (set to true 或禁用（设置为 ）Z深度强制 false
depth 真实的 The new Z depth

返回。

N/A

例子。

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

上面的代码检查图层的Z深度是否被强制，如果不是，就将Z深度设置为0并启用。

返回。一般层的功能
