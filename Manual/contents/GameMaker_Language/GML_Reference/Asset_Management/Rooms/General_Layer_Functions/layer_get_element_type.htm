层级_get_element_type

你可以使用这个函数来获取给定元素的元素类型。你提供唯一的元素ID值（例如，从创建元素的函数或从room 编辑器返回），该函数将返回以下常数之一（如果元素不存在或ID值有误，则返回-1）。

层元素类型常数
恒定 描述
layerelementtype_background 该元素是一个背景。
layerelementtype_instance 该元素是一个实例。
layerelementtype_sprite 该元素是一个sprite asset 。
layerelementtype_tilemap 该元素是一个瓦片图。
layerelementtype_particlesystem 该元素是一个粒子系统。
layerelementtype_tile 该元素是一个遗留的背景tile （这只适用于从以前版本的GameMaker ）导入的项目。
layerelementtype_sequence 该元素是一个sequence asset 。

请注意，当你有多个不同的元素类型分配给一个图层时，这个函数是最有用的，你可以用函数得到任何给定图层上所有元素的列表 layer_get_all_elements().

语法。

layer_get_element_type(element_id)

争论 类型 描述
layer 层元素ID The unique ID value of the element to get the type of

返回。

Layer Element Type Constant or -1 (if element does not exist or is invalid)

例子。

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i])
    }
}

上面的代码获得分配给运行代码的实例层的所有实例元素的ID。然后，代码检查是否有返回的元素是sprite assets ，如果是，则将其销毁。

返回。一般层的功能
