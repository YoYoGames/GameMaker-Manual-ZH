层数_获取_强制深度

这个函数可以用来检索在room ，用于渲染图层的Z深度集。见。 layer_force_draw_depth()获取更多信息。

语法。

layer_get_forced_depth()

返回。

Real

例子。

if layer_get_forced_depth() != 0
{
    layer_force_draw_depth(true, 0);
}

上面的代码检查图层Z深度是否被强制为0的值，如果不是，就将图层Z深度设置为0。

返回。一般层的功能
