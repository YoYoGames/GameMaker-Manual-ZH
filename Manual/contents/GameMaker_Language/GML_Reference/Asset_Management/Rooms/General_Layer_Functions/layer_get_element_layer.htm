层_获取元素_层

你可以使用这个函数来获得给定元素所在的层ID。你提供唯一的元素ID值（例如，由创建元素的函数或从room 编辑器返回），该函数将返回该元素所处的层的唯一ID。如果给定的元素ID不是一个有效的ID，那么这个函数将返回-1。

语法。

layer_get_element_layer(element_id)

争论 类型 描述
element_id 层元素ID The unique ID value of the element to get the layer of

返回。

Layer ID or -1 (if the element is invalid)

例子。

element = layer_get_element_layer(asset_1);

上面的代码得到了元素的层ID，其ID存储在变量 "asset_1 "中，并将其存储在一个变量中。

返回。一般层的功能
