层_set_target_room

当你调用这个函数时，你是在告诉GameMaker，所有进一步的层函数都应该应用于给定的房间。room通过这种方式，你可以在一个不是当前room ，程序性地改变或生成层和层内容。当你完成了对一个room 的层或层元素的添加，调用函数 layer_reset_target_room()来重置room 目标（或者用-1的参数再次调用这个函数room ）。请注意，这个函数只能用于当前房间以外的房间，而且是为了让你在当前运行的rooms 以外的room 中添加/删除层和层元素。

警告!当针对另一个room ，你可以使用所有常规的层函数，除了你不能用 instance_create_layer()或 instance_create_depth()创建实例，也不能使用图层函数 layer_add_instance()也不能使用。

语法。

layer_set_target_room(room)

争论类型 描述
room The room to target for all further layer functions

返回。

例子。

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    var _x = irandom(1000);
    var _y = irandom(1000);
    layer_sprite_create(l, _x, _y, spr_Trees);
}
layer_reset_target_room();

上述代码将目标room 设置为room "rm_Game"，然后获得该room 中名为 "SpriteAssets "的层的ID。然后用这个图层ID向该图层添加50个随机的sprite assets ，然后图层目标被重置为当前的room 。

返回。一般层的功能
