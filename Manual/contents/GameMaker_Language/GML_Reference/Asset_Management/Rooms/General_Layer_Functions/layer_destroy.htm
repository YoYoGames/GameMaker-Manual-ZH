层_销毁

这个函数将销毁给定的图层。你可以提供图层的ID（当你使用 layer_create())或图层名称(作为一个string - 这将对性能产生影响)，这将从当前room 。如果该层是在room 编辑器中设计的，那么当你下次离开room ，然后返回时，该层将以原来的内容重新创建，但是如果room 是持久的，该层将被删除，除非room 的持久性被再次关闭。当你以这种方式销毁一个图层时，它的所有内容也将被删除，所以任何背景或tile 地图等的参考ID将不再有效，任何分配给该图层的实例将被销毁（同时执行它们的销毁事件，如果它们有一个，以及清理事件）。

语法。

layer_destroy(layer_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to destroy (or the layer name as a string)

返回。

N/A

例子。

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

上面的代码将检查object "obj_Bullet_Parent "的任何实例是否存在，如果不存在，它将用存储在全局变量中的ID销毁该层。

返回。一般层的功能
