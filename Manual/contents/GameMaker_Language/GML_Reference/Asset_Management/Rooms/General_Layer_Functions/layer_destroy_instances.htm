层_销毁物质

这个函数可以用来销毁分配给给定层的所有实例。你可以提供图层的ID（当你使用 layer_create())或图层名称(作为string - 这将对性能产生影响)，然后该图层上的所有实例将被从游戏中移除，触发它们的Destroy和Clean Up事件。

语法。

layer_destroy_instances(layer_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer (or the layer name as a string)

返回。

N/A

例子。

if global.game_over
{
    layer_destroy_instances(layer);
}

上面的代码将检查一个全局变量，如果它是 true ，那么所有处于调用实例层的实例将被销毁（包括调用实例）。

返回。一般层的功能
