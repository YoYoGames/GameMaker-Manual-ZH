层数_vspeed

你可以使用这个函数来设置当前范围内的层的垂直速度（以每游戏帧的像素为单位）room 。你可以提供图层的ID（当你使用 "创建 "图层时得到的）或图层的名字（作为 - 这将对性能产生影响）以及速度值。 layer_create())或图层名称（作为string - 这将对性能产生影响）和要设置的速度值，其中正值是向下的，负值是向上的。

语法。

layer_vspeed(layer_id, vspd)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to set the vertical speed of
vspd 真实的 The vertical speed (in pixels per game frame) to set

返回。

N/A

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上述代码检查给定的图层水平和垂直速度，如果它们不是都设置为0，那么它就将它们设置为0。

返回。一般层的功能
