层级_获取_脚本_结束

这个函数返回指定在渲染给定图层结束时运行的函数的脚本函数索引，如果没有指定函数，它将返回-1。你可以提供图层的ID（当你使用 layer_create())或者图层名称（作为string - 这将对性能有影响）。你可以用layer_script_begin() 和 layer_script_end() 为一个图层分配script 函数。

语法。

layer_get_script_end(layer_id);

争论 类型 描述
layer_id 绳索或 图层ID The unique ID value of the layer to target (or the layer name as a string)

返回。

Script Function or -1 (if no function is assigned)

例子。

if layer_get_script_end(layer) == -1
{
    layer_script_end(layer, scr_ResetShaderValues);
}

上面的代码将检查运行代码的实例所在的层是否有分配给它的script 函数，如果没有就分配一个。

返回。一般层的功能
