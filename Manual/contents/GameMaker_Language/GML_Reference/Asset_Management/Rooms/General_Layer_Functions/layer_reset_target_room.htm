layer_reset_target_room

该函数用于将层目标重置为当前的room 。参见函数 layer_set_target_room()获取更多信息。

语法。

layer_reset_target_room()

返回。

例子。

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    layer_sprite_create(l, irandom(1000), irandom(1000), spr_Trees);
}
layer_reset_target_room();

上述代码将目标room 设置为room "rm_Game"，然后获得该room 中名为 "SpriteAssets "的层的ID。然后用这个图层ID向该图层添加50个随机的sprite assets ，然后图层目标被重置为当前的room 。

返回。一般层的功能
