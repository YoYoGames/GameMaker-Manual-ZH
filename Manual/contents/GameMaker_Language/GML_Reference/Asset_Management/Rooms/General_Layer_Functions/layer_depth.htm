层深

这个函数可以用来改变给定图层的深度，改变其内容在屏幕上的渲染顺序。你可以提供图层的ID（当你使用 layer_create())或图层名称（作为一个string - 这将对性能产生影响），然后给出新的深度来设置它（一个整数值，形式为-16000到16000）。图层深度的定义是：当离摄像机 "较远 "时，深度较高；当离摄像机 "较近 "时，深度较低。因此，如果你有三个深度为-100、0、100的图层，这些图层将按照100、0、-100的顺序绘制，因此，"顶层"（即最接近摄像机视图，因此绘制在其他东西之上）将是深度为-100的那一层。

下图显示了深度对层的作用示意图。

注意，你可以在任何时候用函数检查一个图层的深度 layer_get_depth().还要注意，最小和最大的图层深度是-16000到16000，超出这些深度的东西将不会被渲染。如果你需要一个超出这个范围的深度，那么你将需要使用函数 layer_force_draw_depth().

请记住，修改一个图层的深度可能会改变应用在该图层上的滤镜和特效，因为将深度改为低于特效图层的深度将不再在该图层上应用其效果。

语法。

layer_depth(layer_id, depth)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to set the depth of (or the layer name as a string)
depth 真实的 The new depth for the layer

返回。

N/A

例子。

if layer_get_depth(layer) != -100
{
    layer_depth(layer, -100);
}

上面的代码得到了运行代码的实例所在的层的深度，如果它不是-100，那么深度就被设置为-100。

返回。一般层的功能
