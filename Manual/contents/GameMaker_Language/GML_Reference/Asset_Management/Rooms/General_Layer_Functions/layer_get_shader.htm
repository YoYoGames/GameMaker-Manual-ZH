图层_get_shader

这个函数可以用来检查给定的图层是否有分配给它的shader 。你提供图层的ID（当你使用 layer_create()string )或者图层名称（作为一个shader - 这将会对性能产生影响），该函数将返回指定的shader 的索引，如果没有指定shader ，则返回-1。

语法。

layer_get_shader(layer_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)

返回。

Shader Asset or -1 (if no shader is assigned)

例子。

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

上面的代码将检查运行代码的实例所在的层是否有分配给它的shader ，如果没有就分配一个。

返回。一般层的功能
