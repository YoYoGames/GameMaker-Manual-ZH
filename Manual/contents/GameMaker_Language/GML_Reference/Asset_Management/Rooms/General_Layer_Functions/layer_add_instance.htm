层_add_instance

这个函数可以用来把一个给定的实例从它当前所在的层移到另一个层。你可以提供图层的ID（当你创建图层时，你可以用 layer_create())或图层名称（作为string - 这将对性能产生影响）以及要在图层间移动的实例的ID。例如，假设你的播放器在一个层上，而这个层的深度比另一个层低，你想让它出现在这两个层之间的后面。你可以用这个函数将播放器从前景层移到背景层，并使它出现在正在绘制的其他层的后面，switch 。

注意这个函数不能用来给一个图层添加一个新的实例。你必须事先创建实例并将其ID存储到一个变量中。

注意：该功能仅在当前room 的范围内有效，当使用功能layer_set_target_room() 更改范围时，不能使用。

语法。

layer_add_instance(layer_id, instance_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the instance layer to target (or the layer name as a string)
instance_id 实例ID The unique instance ID value of the instance to move over to the target layer

返回。

N/A

例子。

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

上面的代码首先会得到离给定x/y位置最近的实例的索引，并将其存储在一个局部变量中。然后，它得到名为 "Instances Front "的图层的唯一实例层ID，并将找到的实例移动到该图层。

返回。一般层的功能
