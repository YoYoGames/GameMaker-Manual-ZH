层_is_draw_depth_forced

这个函数可以用来检查并查看在项目中渲染图层时是否启用了强制Z深度。参见 layer_force_draw_depth()获取更多信息。

语法。

layer_is_draw_depth_forced()

返回。

例子。

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

上面的代码检查图层的Z深度是否被强制，如果不是，就将Z深度设置为0并启用。

返回。一般层的功能
