层_获取目标房间

这个函数将返回当前的room ，被层函数所针对。参见 layer_set_target_room()获取更多信息。

语法。

layer_get_target_room()

返回。

Room Asset

例子。

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

上述代码检查当前的目标room ，如果它不是当前的room ，那么该层的目标room 被重置。

返回。一般层的功能
