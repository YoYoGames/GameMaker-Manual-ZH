层_元素_移动

你可以用这个函数把一个元素从一个层移到另一个层。你给出元素的ID，就像用来创建元素的函数或room 编辑器或函数所返回的那样。 layer_get_all_elements()，然后你给图层ID（当你使用 layer_create())或图层名称（作为string - 这将对性能有影响）。

语法。

layer_element_move(element_id, layer_id)

争论 类型 描述
element_id 层元素ID The unique ID value of the element to move
layer_id 字符串 或层ID The unique ID value of the layer to move the element to (or the layer name as a string)

返回。

N/A

例子。

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

上面的代码获取运行代码的实例被分配到的图层上的元素，然后检查它们是否是sprite assets ，如果是，那么它们就被移到ID存储在变量 "asset_layer "的图层上。

返回。一般层的功能
