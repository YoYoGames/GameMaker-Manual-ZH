层数_获取深度

你可以使用这个函数来获得与一个给定的图层相关的深度值。你可以提供图层的ID（当你使用 layer_create())或图层名称（作为string ，这将对性能产生影响），该函数将以实数返回该图层的深度。请注意，深度的定义是离摄像机越 "远 "越高，离摄像机越 "近 "越低，所以如果你有三个深度为-100、0、100的图层，这些图层将按照100、0、-100的顺序绘制，所以 "顶层"（即最接近摄像机视图，所以绘制在其他东西之上）将是深度为-100的那一层。

下图显示了深度对层的作用示意图。

注意，如果你提供了一个无效的层ID值，你会得到一个错误。

语法。

layer_get_depth(layer_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to get the depth of (or the layer name as a string)

返回。

Real

例子。

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

上述代码检查存储在全局变量中的图层ID的深度，如果它不是-100，就会破坏该图层，并在-100的深度上重新创建它。

返回。一般层的功能
