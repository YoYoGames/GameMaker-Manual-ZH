层级_get_hspeed

你可以使用这个函数来检索当前范围内的层的水平速度（以每游戏帧的像素为单位）room 。你可以提供图层的ID（当你使用 layer_create())或图层名称(作为string - 这将对性能产生影响)，该函数返回一个水平速度的实数，其中正值在右边，负值在左边。默认为0（除非在room 编辑器中设置）。

语法。

layer_get_hspeed(layer_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to get the horizontal speed from

返回。

N/A

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上述代码检查给定的图层水平和垂直速度，如果它们不是都设置为0，那么它就将它们设置为0。

返回。一般层的功能
