层的名称

你可以使用这个函数来获取给定图层的名称。你提供唯一的图层ID值，如果该图层是在room 编辑器中创建的命名图层之一，那么这个函数将返回一个带有该图层名称的string 。如果该图层不是room 编辑器中的一个（即：它是用 layer_create())，那么将返回一个空的string 。

语法。

layer_get_name(layer_id)

争论 类型 描述
layer_id 图层ID The unique ID value of the layer to get the name of

返回。

String

例子。

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

上面的代码得到了room 中所有图层的ID，然后loops ，检查是否有命名图层。如果它们是，那么它们的ID就被添加到一个列表中。

返回。一般层的功能
