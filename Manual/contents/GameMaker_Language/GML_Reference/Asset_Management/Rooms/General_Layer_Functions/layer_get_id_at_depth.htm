层数_获取深度_ID

你可以使用这个函数来获得所有分配给特定深度的图层的ID。你给出要检查的深度，该函数将返回一个有1个或多个条目的数组，这取决于在给定深度是否有任何层。如果在给定的深度没有图层，那么数组将在[0]位置有一个值为-1的条目，但是，如果在该深度有图层，那么将在数组中为找到的每个图层建立一个条目--该条目值将是一个图层的唯一ID值。

语法。

layer_get_id_at_depth(depth)

争论 类型 描述
depth 真实的 The depth to check and retrieve the layer IDs from.

返回。

Array

例子。

var a = layer_get_id_at_depth(0);
if a[0] != -1
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

上面的代码检索深度为0的层的数据，检查是否有任何层存在于这个深度，如果有，那么返回的数组被解析，找到的每个层都被销毁。

返回。一般层的功能
