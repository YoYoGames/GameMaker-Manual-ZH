层数_获取_ID

这个函数可以用来获取一个给定图层的唯一ID值。在IDE ，所有的图层都有一个名字和一个类型，为了能够通过代码编辑或改变它们，你必须给出图层的ID值。这个函数是通过使用层的名称（astring ）来检索这个ID（如在IDE ）。如果你通过代码使用函数创建一个新的图层 layer_create()那么该函数将返回唯一的ID值（动态创建的图层不会得到名字）。注意，如果你给一个在当前room 中不存在的层的名字，那么你会得到一个错误，项目会崩溃。

语法。

layer_get_id(layer_name)

争论类型 描述
layer_name The name of the layer (a string)

返回。

Layer ID or -1 (if the layer specified doesn't exist)

例子。

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

上面的代码首先会得到离给定x/y位置最近的实例的索引，并将其存储在一个局部变量中。然后，它得到名为 "Instances Front "的图层的唯一实例层ID，并将找到的实例移动到该图层。

返回。一般层的功能
