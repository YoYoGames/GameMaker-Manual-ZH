层数_sprite_exists

你可以使用这个函数来检查并查看在任何给定的图层上是否存在一个sprite 元素。你可以提供图层的ID（当你创建图层时，你会得到这个ID layer_create())或图层名称(作为string - 这将对性能产生影响)和sprite 元素ID(当你使用sprite 元素创建时，你会得到它。 layer_sprite_create()或当你使用函数 layer_sprite_get_id())，如果该元素存在，该函数将返回 true ，如果不存在，则返回 false 。

注意：该函数在当前目标room 的范围内工作--默认情况下是调用该函数的room ，可以用函数设置 layer_set_target_room().

语法。

layer_sprite_exists(layer_id, sprite_element_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id 精灵元素ID The unique ID value of the sprite element to check

返回。

Boolean

例子。

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

上面的代码检查层 "Asset_trees"，看给定的sprite 元素是否存在，如果存在，那么它就被销毁（但不是层）。

返回。精灵层
