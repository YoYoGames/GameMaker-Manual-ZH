图层_sprite_get_xscale

这个函数可以用来获取sprite 元素沿x轴的当前比例乘数。你给sprite 元素的ID（当你用sprite 元素创建一个 layer_sprite_create()或使用函数 layer_sprite_get_id())，该函数将返回x刻度的实值，其中1是没有刻度的。

语法。

layer_sprite_get_xscale(sprite_element_id);

争论类型 描述
sprite_element_id The unique ID value of the sprite element to get the information from

返回。

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_xscale(spr_id) != 1)
{
    layer_sprite_xscale(spr_id, 1);
}

上面的代码将获得名为 "sprite_sky "的层的ID，然后用它来获得该层上sprite 元素的ID。这个ID然后被用来检查xscale值，如果它不是1，就被设置为1。
