layer_sprite_blend

这个函数控制sprite sprite 的混合（或 "着色"），默认值是-1（代表常数 c_white ，也可以使用）。任何其他值（包括内部颜色常数，如 c_red, 或 c_aqua ）将把指定的颜色与原始的sprite 。你给sprite 元素ID（当你使用sprite 元素创建时，你会得到这个ID。 layer_sprite_create()或当你使用 函数 layer_sprite_get_id())，然后设置要使用的混合颜色。下面你可以看到一个被混合了不同颜色的sprite 的例子。

请注意，你应该尽量限制在HTML5平台上的混合（除非使用WebGL）。 因为每个混合的sprite ，必须单独缓存，所以有许多混合的sprites ，可能会对性能产生不利影响（你也可以使用函数设置cache 的大小 sprite_set_cache_size()).

语法。

layer_sprite_blend(sprite_element_id, blend);

争论类型 描述
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

返回。

例子。

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

上面的代码得到了分配给 "Asset_sky "层的名为 "Clouds "的sprite 的ID值，然后将其染成一种颜色。

返回。精灵层
