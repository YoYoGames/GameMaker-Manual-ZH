层数_栅栏_获取阿尔法

这个函数可以用来获取sprite 元素的alpha值。你给sprite 元素的ID（当你使用sprite 元素创建一个 layer_sprite_create()或当你使用函数 layer_sprite_get_id())，该函数将返回一个在0(完全透明)和1(完全不透明)之间的值。

语法。

layer_sprite_get_alpha(sprite_element_id);

争论类型 描述
sprite_element_id The unique ID value of the sprite element to get the information from

返回。

(from 0 to 1)

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_alpha(spr_id) < 0.1
{
    layer_sprite_destroy(spr_id);
}

上面的代码将得到名为 "sprite_sky "的层的ID，然后用它来得到该层上的sprite 元素的ID。这个ID然后被用来检查元素的alpha，如果它小于0.1，那么该层元素就会被销毁。

返回。精灵层
