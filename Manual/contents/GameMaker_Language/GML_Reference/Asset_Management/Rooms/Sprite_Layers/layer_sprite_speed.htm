layer_sprite_speed

这个函数可以用来设置已经分配给sprite 元素的动画图像的速度。你给sprite 元素的ID（当你用sprite 元素创建时，你会得到这个ID。 layer_sprite_create()或当你使用函数 layer_sprite_get_id())，然后设置动画速度。如果IDE 已经为所用的sprite 的动画速度设置为每游戏帧计算，那么给出的速度值是一个乘数，1是默认值，将其设置为0.5将使动画速度减半，而设置为1.5将使其加倍。但是，如果IDE ，将所使用的sprite 的动画速度设置为每秒帧数计算，那么给出的速度值就是每秒帧数的字面值，因此将其设置为5，例如，将把动画设置为loop ，游戏每秒运行5帧。如果使用的sprite 没有子图像，这个函数就没有效果。

语法。

layer_sprite_speed(sprite_element_id, speed);

争论 类型 描述
sprite_element_id 雪碧资产 The unique ID value of the sprite element to change
speed 真实的 The speed multiplier/value to use for the element

返回。

N/A

例子。

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

上面的代码将获得名为 "Assets_water "的图层的ID，然后用它来获得该图层上给定的sprite 的元素的ID。这个ID然后被用来改变元素的图像速度。
