层_sprite_alpha

这个函数控制sprite 图层上的asset 的alpha（透明度）。你给sprite 元素的ID（当你用sprite 元素创建一个 layer_sprite_create()或当你使用 该函数 layer_sprite_get_id())，然后设置要使用的alpha值。α值可以在0（完全透明）和1（完全不透明）之间，sprite 元素的默认α值是1。注意 如果sprite 元素被分配到的图层是不可见的--或者元素本身已经变得不可见--你将不会看到这个函数的任何区别，直到该图层或元素再次变得可见。

语法。

layer_sprite_alpha(sprite_element_id, alpha);

争论类型 描述
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

返回。

例子。

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

上面的代码得到了分配给 "Asset_sky "层的名为 "Clouds "的sprite asset 的ID值，然后将其alpha值设置为0到1之间的随机值。

返回。精灵层
