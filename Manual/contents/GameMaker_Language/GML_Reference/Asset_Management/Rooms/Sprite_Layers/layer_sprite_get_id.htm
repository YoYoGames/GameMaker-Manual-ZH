层数_栅栏_获取_ID

这个函数可以用来检索一个图层上的sprite 元素的唯一ID值。你提供图层的ID（当你用 "创建 "图层时，你会得到这个ID。 layer_create()创建图层时得到的ID，或者当你使用图层名称和 layer_get_id())和在sprite 编辑器中定义的Room 元素的名称。该函数将返回与该层上的sprite 元素相关的ID值。

请注意，这个函数是专门为在sprites 中添加到IDE 层上的asset 而设计的，如果你通过使用函数的代码添加了一个sprite 到一个层上 layer_sprite_create()，那么该函数返回的ID应该被用于所有未来的参考（因为该sprite 元素将没有名字可以传入该函数）。

如果指定的层不存在，或者找不到给定的sprite 元素，该函数将返回-1。

语法。

layer_sprite_get_id(layer_id, sprite_element_name)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target
sprite_element_name 字符串 The unique name of the sprite element on the layer as defined in the Room Editor

返回。

Sprite Element ID

例子。

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

上面的代码将获得名为 "Assets_trees "的层的ID，然后用它来检索该层上的sprite "graphic_254367CB "元素的ID。检索到的sprite 元素ID，然后用来改变该元素的sprite 。

返回。精灵层
