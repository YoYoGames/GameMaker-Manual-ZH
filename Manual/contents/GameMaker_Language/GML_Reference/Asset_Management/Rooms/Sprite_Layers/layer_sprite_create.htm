layer_sprite_create

通过这个函数，你可以把一个sprite 资源分配给你的项目中使用的层。你可以提供图层的ID（当你用 "创建 "图层时，你会得到这个ID。 layer_create()或当你使用图层名称和 layer_get_id())，一个在room 中的位置，以及一个sprite 的索引(这将是sprite 浏览器中显示的Asset 的名称)，然后它将被添加到该层。该函数返回该元素的唯一ID值，然后可以在进一步的层函数中使用sprites 。

语法。

layer_sprite_create(layer_id, x, y, sprite)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target
x 真实的 The x position to use
y 真实的 The y position to use
sprite 雪碧资产 The sprite index to be used

返回。

Sprite Element ID

例子。

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

上面的代码创建了一个新的图层，然后给它添加了10个新的sprite ，把每个元素的ID存储到一个数组中。

返回。精灵层
