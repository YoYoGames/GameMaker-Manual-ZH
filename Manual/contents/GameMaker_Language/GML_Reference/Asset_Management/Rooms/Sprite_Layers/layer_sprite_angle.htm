层级_sprite_angle

使用这个函数，你可以改变一个层上给定的sprite 元素的角度。你给sprite 元素的ID（当你使用sprite 元素创建时，你会得到这个ID layer_sprite_create()或当你使用函数 layer_sprite_get_id())，然后设置角度值，从0到359逆时针，其中0是向右，90是向顶，180是向左，270是向底。如果你设置的值大于360，这将被循环使用，使其在0-359范围内。

语法。

layer_sprite_angle(sprite_element_id, angle);

争论 类型 描述
sprite_element_id 精灵元素ID The unique ID value of the sprite element to change
angle 真实的 The angle of the sprite (default is 0)

返回。

N/A

例子。

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
    layer_sprite_angle(asset_sprite, 0);
}

上述代码将检查分配给运行代码的实例所在层的sprite 元素，如果它的角度不是0，它将把它设置为0。

返回。精灵层
