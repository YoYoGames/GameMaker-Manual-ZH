破坏层_破坏层

这个函数将销毁给定的sprite 元素。你提供了sprite ID（当你使用sprite 创建时，你会得到这个ID。 layer_sprite_create()或当你使用图层ID和 layer_get_sprite_id()时得到的），这将会删除它。请注意，这并不删除图层，只是删除其中的sprite ，如果sprite 是在room 编辑器中添加的，那么下次你离开room ，然后返回时，sprite 将再次被重新创建。然而，如果room 是持久的，sprite 将被删除，除非room 的持久性被再次关闭。

语法。

layer_sprite_destroy(sprite_element_id)

争论类型 描述
sprite_element_id The unique ID value of the sprite to be destroyed

返回。

例子。

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
    layer_sprite_destroy(lay_id);
}

上面的代码检查图层 "Asset_Trees"，看给定的sprite 元素是否存在，如果存在，那么它就会被销毁（但不是图层）。

返回。精灵层
