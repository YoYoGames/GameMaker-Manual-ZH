layer_sprite_index

这个函数可以用来设置一个sprite asset 的图像索引，它在一个图层上有多个子图像。你给sprite 元素的ID（当你使用sprite 元素创建时，你会得到这个ID layer_sprite_create()或当你使用函数 layer_sprite_get_id())，然后设置要使用的图像索引。如果你设置了一个超出子图像范围的值，那么图像索引将在loop 。

语法。

layer_sprite_index(sprite_element_id, image_index);

争论 类型 描述
sprite_element_id 雪碧资产 The unique ID value of the sprite element to set
index 真实的 The image index to use for the sprite

返回。

N/A

例子。

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

上面的代码将获得名为 "sprite_trees "的层的ID，然后用它来获得该层上给定的sprite 元素的ID。这个ID然后被用来改变元素的图像索引。
