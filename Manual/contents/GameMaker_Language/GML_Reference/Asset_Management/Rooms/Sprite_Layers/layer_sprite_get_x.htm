layer_sprite_get_x

这个函数可以用来获得sprite 元素在room 中的X位置。你给sprite 元素的ID（当你创建一个sprite 元素时，你会得到这个ID，用 layer_sprite_create()或者当你使用函数 layer_sprite_get_id())，该函数将返回x位置值。

语法。

layer_sprite_get_x(sprite_element_id);

争论类型 描述
sprite_element_id The unique ID value of the sprite element to get the information from

返回。

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_x(spr_id) < 0)
{
    layer_sprite_x(spr_id, room_width);
}

上面的代码将获得名为 "sprite_sky "的层的ID，然后用它来获得该层上的sprite 元素的ID。这个ID然后被用来检查元素的X位置，如果它小于0，那么该层元素就被移到一个不同的X位置。

返回。精灵层
