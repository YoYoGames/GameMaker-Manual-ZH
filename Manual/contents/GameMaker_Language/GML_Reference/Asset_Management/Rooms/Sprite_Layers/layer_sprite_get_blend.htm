层级_栅栏_获取混合

这个函数可以用来获取sprite 元素的混合颜色。你给sprite 元素的ID（当你使用sprite 元素创建时，你会得到这个ID layer_sprite_create()或当你使用函数 layer_sprite_get_id())，该函数将返回代表指定颜色的实值。

语法。

layer_sprite_get_blend(sprite_element_id);

争论类型 描述
sprite_element_id The unique ID value of the sprite element to get the information from

返回。

Colour Constant

例子。

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
{
    layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上面的代码将获得名为 "sprite_sky "的图层的ID，然后用它来获得该图层上sprite 元素的ID。这个ID然后被用来检查元素的混合颜色，如果它等同于常数 c_white ，那么元素的混合就被设置为随机颜色。

返回。精灵层
