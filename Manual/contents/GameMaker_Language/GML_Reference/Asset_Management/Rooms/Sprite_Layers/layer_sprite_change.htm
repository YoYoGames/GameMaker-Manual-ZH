层数的改变

使用这个函数，你可以改变分配给一个图层上的特定sprite 元素的sprite 资源。你给sprite 元素的ID（当你用这个函数创建一个sprite 元素时，你会得到这个ID。 layer_sprite_create()或当你使用函数 layer_sprite_get_id())，然后提供一个sprite 索引元素asset 将被改变。注意，如果sprite 有子图像，那么它也会有动画（这可以用 layer_sprite_index()和 layer_sprite_speed()函数来控制）。请注意，你可以指定一个-1的值作为新的sprite 索引，并且不会显示sprite ，尽管该元素仍然存在，并且以后仍然可以再次改变。

语法。

layer_sprite_change(sprite_element_id, sprite_index)

争论 类型 描述
sprite_element_id 精灵元素ID The unique ID value of the sprite element to change
sprite_index 雪碧资产 The new sprite index to use

返回。

N/A

例子。

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

上面的代码用存储在全局变量 global.Asset_sprite 中的ID检查元素的sprite 索引，如果它不是 "spr_Nighttime"，那么该sprite 被分配给该元素。

返回。精灵层
