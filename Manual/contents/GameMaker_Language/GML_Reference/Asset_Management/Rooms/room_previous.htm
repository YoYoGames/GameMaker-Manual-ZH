前一个房间

room 通过这个函数，你可以检索输入到函数中的room 前的索引。例如，你可以使用变量 room来获取当前room 的索引，然后用这个函数来查找在它之前的room ，如房间管理器中所列。如果在你输入的那一个之前没有room ，那么将返回-1。注意，这个函数不会识别或考虑使用room_add() 或room_duplicate() 动态添加的rooms 。

语法。

room_previous(numb);

争论类型 描述
numb The index of the room to get the one before from.

返回。

Room Asset

例子。

if room_previous(room) != -1
{
    room_goto_previous();
}

上面的代码将检查前一个room 是否存在，如果存在，它将转到它。
