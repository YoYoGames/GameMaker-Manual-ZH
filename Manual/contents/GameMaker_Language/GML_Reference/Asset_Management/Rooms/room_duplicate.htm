房间_重复的

这将复制一个给定的room ，并返回复制的索引，在所有进一步的调用中使用，以引用新的room 。

注意：新的rooms ，不属于通常的room 排序，所以它们没有 "上一个 "或 "下一个 "room （意味着函数 room_next()和 room_previous()将不起作用）。要从添加的room 跳到另一个，你必须使用room 本身的索引。

语法。

room_duplicate(index);

争论类型 描述
index The index of the original room to be duplicated.

返回。

Room Asset

例子。

global.myroom = room_duplicate(rm_level);

room 这将复制由 "rm_level "索引的room ，并将这个新的room 的索引分配给变量 "global.myroom"。

返回。房间
