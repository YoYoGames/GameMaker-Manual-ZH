房间_instance_add

通过这个函数，你可以将一个实例添加到除当前实例之外的任何room ，并且添加到该room 中的任何位置。该函数返回实例的唯一 id实例，然后可以在进一步的函数调用中使用，以设置该实例的属性等，但只有在游戏进入指定的房间后才能使用。如果你想在当前的room ，你应该使用这个函数来创建一个实例 instance_create_layer().请注意，在room 浏览器中创建的asset Asset 上调用这个函数将永久地将该实例添加到房间中，即使调用 game_restart() 也不会将room 返回到它的原始状态（只有结束游戏并再次打开游戏才会再次以原始状态开始room ）。

语法。

room_instance_add(index, x, y, obj);

争论类型 描述
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

返回。

Instance ID

例子。

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

上述代码将向游戏添加一个新的room ，然后将索引为 "rm_Base "的room 的内容复制到其中。然后，它将在这个新的object 的100,100位置添加一个room "obj_player "的实例。

返回。房间
