层_背景_平铺

使用这个函数，你可以改变一个图层上给定的背景元素是否应该被水平铺设。你给背景元素的ID（当你用 "创建背景元素 "或 "使用函数 "时，你会得到这个ID）。 layer_background_create()或当你使用函数 layer_background_get_id())，然后设置平铺值。如果设置为 true ，那么该元素将被水平平铺显示在room ，而当设置为 false ，该元素将不被平铺。这个功能只适用于2D项目，当使用3D摄像机投影时，将不能正常工作。

语法。

layer_background_htiled(background_element_id, tiled);

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to change
tiled 布尔型 The horizontal tiling toggle, which can be true 或 false

返回。

N/A

例子。

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

上面的代码将检查分配给运行代码的实例所在层的背景元素，如果它在任何一个方向上都没有平铺，它将把它设置为tile 纵向和横向。
