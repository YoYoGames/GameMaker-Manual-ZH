layer_background_get_id

这个函数可以用来检索一个图层上的背景元素的唯一ID值。你提供图层的ID（当你使用图层名称和 layer_get_id())，该函数将返回与该层上的背景元素相关的ID值。请注意，这个函数是专门为在IDE 中添加的背景而设计的，因为如果你通过使用该函数的代码将一个背景添加到一个图层上 layer_background_create()，那么它将返回添加的背景元素的唯一ID。

语法。

layer_background_get_id(layer_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target

返回。

Background Element ID

例子。

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

上面的代码将获得名为 "Background_trees "的图层的ID，然后用它来获得该图层上背景元素的ID。这个ID然后被用来改变元素sprite 。
