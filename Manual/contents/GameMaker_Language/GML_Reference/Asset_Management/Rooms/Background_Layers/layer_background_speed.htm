layer_background_speed

这个函数可以用来设置已经分配给一个背景元素的动画sprite 的速度。你给背景元素的ID（当你用 "创建背景元素 "或 "使用函数 "创建背景元素时，你会得到这个ID）。 layer_background_create()或当你使用函数 layer_background_get_id())，然后设置动画速度。如果IDE 已经将用于sprite 的动画速度设置为每游戏帧计算，那么给出的速度值是一个乘数，1是默认值，将其设置为0.5将使动画速度减半，而设置为2将使其加倍。但是，如果IDE ，将用于sprite 的动画速度设置为每秒帧数计算，那么给出的速度值就是每秒帧数的字面值，因此将其设置为5，例如，将把动画设置为loop ，在游戏运行的每一秒钟设置5帧。如果背景元素没有附加sprite ，或者使用的sprite 没有子图像，这个函数将没有效果。

语法。

layer_background_speed(background_element_id, speed);

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to change
speed 真实的 The speed multiplier/value to use for the element

返回。

N/A

例子。

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

上面的代码将得到名为 "Background_water "的层的ID，然后用它来得到该层上的背景元素的ID。这个ID然后被用来改变元素的图像速度。
