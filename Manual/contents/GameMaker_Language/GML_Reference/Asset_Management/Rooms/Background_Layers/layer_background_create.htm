创建图层_背景

通过这个函数，你可以把一个sprite 资源分配给一个图层，作为你项目的背景使用。你可以提供图层的ID（当你使用 layer_create())或图层名称(作为一个string - 这将对性能产生影响)和一个sprite 索引(这将是sprite 浏览器中显示的Asset 的名称)，它将被添加到该图层。该函数返回该元素的唯一ID值，然后可以在进一步的图层函数中用于背景。

语法。

layer_background_create(layer_id, sprite)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
sprite 雪碧资产 The sprite index to be used

返回。

Background Element ID

例子。

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

上面的代码创建了一个新的图层，然后给它添加了一个新的背景元素，设置一个sprite ，作为使用的背景图片。
