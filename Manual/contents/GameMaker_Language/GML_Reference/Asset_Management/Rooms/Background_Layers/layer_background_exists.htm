层_背景_存在

你可以使用这个函数来检查并查看任何给定的图层上是否存在一个背景元素。你可以提供图层ID（当你创建图层的时候，你可以用 layer_create())或图层名称(作为一个string - 这将对性能产生影响)和背景元素ID(当你使用 layer_background_create()或当你使用函数 layer_background_get_id())，如果该元素存在，该函数将返回 true ，如果不存在，则返回 false 。

注意：该函数在当前目标room 的范围内工作--默认情况下是调用该函数的room ，可以用函数设置 layer_set_target_room().

语法。

layer_background_exists(layer_id, background_element_id)

争论 类型 描述
layer_id 字符串 或层ID The unique ID value of the layer to target (or the layer name as a string)
background_element_id 背景元素ID The unique ID value of the background element to check

返回。

Boolean

例子。

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

上面的代码检查图层 "Background_trees"，看给定的背景元素是否存在，如果存在，那么它就被销毁（但不包括图层）。
