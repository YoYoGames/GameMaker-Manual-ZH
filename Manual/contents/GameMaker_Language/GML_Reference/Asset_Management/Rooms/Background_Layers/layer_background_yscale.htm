layer_background_yscale

这个函数可以用来设置一个背景元素沿Y轴的比例。你给背景元素的ID（当你用 "创建一个背景元素 "时，你会得到这个ID）。 layer_background_create()或当你使用函数 layer_background_get_id())，然后设置比例值。你给的比例值是一个乘数，将用于改变背景元素的显示方式，0.5的值将以一半的比例显示该元素，2的值将以两倍的比例显示。请注意，负值是有效的，它将在(0,0)的位置上 "翻转 "元素，所以一个-1的X比例将显示图像的反面。

语法。

layer_background_yscale(background_element_id, yscale);

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to change
yscale 真实的 The speed multiplier/value to use for the element

返回。

N/A

例子。

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

上面的代码将得到名为 "Background_water "的层的ID，然后用它来得到该层上的背景元素的ID。这个ID然后被用来改变元素的yscale。
