层_背景_斯普利特

使用这个函数，你可以设置背景元素的sprite 索引。你可以提供背景元素的ID（当你使用 "创建背景元素 "时，你会得到这个ID。 layer_background_create()或当你使用函数 layer_background_get_id())，然后给出一个要使用的sprite 索引。背景元素的图像将被替换成新的sprite 。如果你给一个-1的值，该元素将没有分配到sprite （但仍然存在，以后可以再分配到一个sprite ）。

这个函数还有一个别名layer_background_change() 。

语法。

layer_background_sprite(background_element_id, sprite_index)

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to change
sprite_index 雪碧资产 The sprite index of the sprite to use for the background element

返回。

N/A

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

上面的代码将获得名为 "Background_sky "的图层的ID，然后用它来获得该图层上背景元素的ID。这个ID然后被用来检查分配给该元素的sprite ，如果它还没有被设置为sprite "spr_Clouds"。
