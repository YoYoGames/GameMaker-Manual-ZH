layer_background_visible

使用这个函数，你可以切换一个背景元素的可见性。你提供背景元素的ID（你在使用 "创建背景元素 "时得到的ID）。 layer_background_create()或当你使用函数 layer_background_get_id())，然后将可见性参数设置为 true 或 false 。当设置为 true ，该元素将被显示，而当设置为 false ，该元素将不被显示。注意，这取决于层的可见性，即使背景元素被标记为可见，如果它所在的层被标记为不可见，它也不会被画出来。

语法。

layer_background_visible(background_element_id, visible)

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to change
visible 布尔型 The visibility toggle, which can be true 或 false

返回。

N/A

例子。

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_visible(back))
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

上面的代码检查鼠标按键是否被按下，如果检测到，它将切换分配给运行该代码的实例所在层的背景元素的可见性。
