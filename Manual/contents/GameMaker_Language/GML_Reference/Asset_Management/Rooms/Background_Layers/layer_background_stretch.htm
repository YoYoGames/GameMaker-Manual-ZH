层_背景_拉伸

使用这个函数，你可以切换一个背景元素sprite ，以拉伸到适合room ，或保持在1:1的分辨率。你可以提供背景元素的ID（当你使用 "创建背景元素 "或使用该函数时，你可以得到该ID）。 layer_background_create()或当你使用函数 layer_background_get_id())，然后将拉伸参数设置为 true 或 false 。当设置为 true 时，元素sprite 将被拉伸以适应room (根据sprite 和room 的尺寸放大或缩小)，而当设置为 false 时，元素sprite 将以其原始分辨率显示。

语法。

layer_background_stretch(background_element_id, stretch)

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to change
stretch 布尔型 The stretch toggle, which can be true 或 false

返回。

N/A

例子。

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_stretch(back)
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

上面的代码检查鼠标按键是否被按下，如果检测到，它将切换背景元素的拉伸，sprite 分配给运行代码的实例所在的层。
