层_背景_获取_斯普雷特

这个函数可以用来获取背景元素的当前sprite 索引值。你给背景元素的ID（当你用AlphaGo创建一个背景元素时，你会得到这个ID。 layer_background_create()或当你使用函数 layer_background_get_id())，该函数将返回一个实值，代表正在显示的sprite 索引。如果该元素没有指定sprite ，该函数将返回-1。

语法。

layer_background_get_sprite(background_element_id);

争论类型 描述
background_element_id The unique ID value of the background element to get the information from

返回。

Sprite Asset or -1

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

上面的代码将获得名为 "Background_sky "的图层的ID，然后用它来获得该图层上背景元素的ID。这个ID然后被用来检查分配给该元素的sprite ，如果它还没有被设置为sprite "spr_Clouds"。
