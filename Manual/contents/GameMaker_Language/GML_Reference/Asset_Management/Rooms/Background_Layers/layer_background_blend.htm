layer_background_blend

这个函数控制背景的混合（或 "着色"）sprite ，默认值是-1（代表常数 c_white ，也可使用）。任何其他值（包括内部颜色常数，如 c_red, 或 c_aqua ）将把指定的颜色与原始的sprite 。你给背景元素的ID（当你用 "创建一个背景元素 "时，你会得到这个ID。 layer_background_create() 或当你使用函数 layer_background_get_id())，然后设置要使用的混合颜色。下面你可以看到一个被混合了不同颜色的sprite 的例子。

请注意，你应该尽量限制HTML5平台上的混合（除非使用WebGL），因为每个 混合的sprite ，都必须单独缓存，因此有许多混合的sprites ，可能会对性能产生不利影响（你也可以使用函数设置cache 的大小 sprite_set_cache_size()).

语法。

layer_background_blend(background_element_id, blend);

争论类型 描述
background_element_id The unique ID value of the background element to change
blend The colour to blend with the background sprite (default is c_white)

返回。

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

上面的代码得到了分配给图层 "Background_sky "的背景的ID值，然后把它染成一种颜色。
