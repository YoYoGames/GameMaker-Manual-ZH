层_背景_销毁

这个函数将销毁给定的背景元素。你提供背景ID（当你使用 layer_background_create()或当你使用图层ID和 layer_get_background_id()时得到的），这个函数将删除它。注意，这不会删除图层，只是删除其中的背景，如果这个背景是在room 编辑器中添加的，那么下次你离开room ，然后返回时，这个背景将再次被重新创建。然而，如果room 是持久的，背景将被删除，除非room 的持久性再次被关闭。

语法。

layer_background_destroy(background_element_id)

争论 类型 描述
background_element_id 背景元素ID The unique ID value of the background element to be destroyed

返回。

N/A

例子。

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if layer_background_exists(lay_id, bck_id)
{
    layer_background_destroy(bck_id);
}

上面的代码检查图层 "Background_trees"，看给定的背景元素是否存在，如果存在，那么它就被销毁（但不是图层）。
