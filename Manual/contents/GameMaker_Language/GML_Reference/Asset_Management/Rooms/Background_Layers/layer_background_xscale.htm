图层_背景_xscale

这个函数可以用来设置一个背景元素沿X轴的比例。你给背景元素的ID（当你用 "X "创建一个背景元素时，你会得到这个ID）。 layer_background_create() 或当你使用函数 layer_background_get_id())，然后设置比例值。你给的比例值是一个乘数，它将被用来改变背景元素的显示方式。 一个0.5的值将以一半的比例显示元素，一个2的值将以两倍的比例显示。请注意，负值也是有效的，它将在(0,0)的位置上 "翻转 "元素，所以一个-1的X比例将显示 因此，X比例为-1时，将显示反转的图像。

语法。

layer_background_xscale(background_element_id, xscale);

争论类型 描述
background_element_id The unique ID value of the background element to change
xscale The scale value to use (1 is no scaling)

返回。

例子。

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

上面的代码将得到名为 "Background_water "的层的ID，然后用它来得到该层上的背景元素的ID。这个ID然后被用来改变元素的xscale。
