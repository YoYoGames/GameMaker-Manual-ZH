房间_存在

通过这个函数，你可以检查并查看你指定的room 是否存在。这个函数接受room 索引（一个实数），而不是 room 名称（一个string ）。当你使用函数动态地创建rooms 时，这个函数是最有用的。 room_add()的时候，这个函数最有用，但你也可以使用只读变量 room_first和 room_last或函数 room_next()和 room_previous()来获得一个特定的room 索引，或者提供一个变量来存储你游戏中任何其他room 的索引。

语法。

room_exists(index);

争论类型 描述
index The index of the room to check.

返回。

例子。

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

上述代码检查 "global.rm[]"数组中索引的room 是否存在，如果存在，则转到该room 。

返回。房间
下一步。 room_add
