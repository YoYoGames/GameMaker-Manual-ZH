实例_count

通过这个只读变量，你可以得到room 中的所有活动实例的数量。这将包括运行代码的实例，但不包括那些已经使用实例停用函数停用的实例。请注意，这个变量只能给出步骤开始时的实例数量，所以步骤开始后对room 中的实例所做的任何改变都不会被考虑在内。

语法。

instance_count;

返回。

例子。

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

上述代码将创建多个object "obj_Star "的实例，直到总实例数达到100。

返回:实例
