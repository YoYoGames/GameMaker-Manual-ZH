instance_id

这个只读 数组保存了room 内每个活动 实例的所有id。这意味着，如果你使用了任何一个实例停用函数，那些已经停用的实例将不包括在这个数组中（如果你以前使用过这个数组中的一个值，现在它将返回关键字 noone).

语法。

instance_id[num];

返回。

Instance ID

例子。

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

上述代码将loop ，通过room 内的所有实例，并在其速度上增加0.1。

返回:实例
