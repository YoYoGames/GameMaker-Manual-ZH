实例

实例是由Objects创建的，一旦创建，就可以通过GML 代码和GML Visual动作来改变和操作。与实例有关的、可用的、最常用的主要实例功能是。

  • instance_create_layer
  • instance_create_depth
  • instance_destroy
  • instance_exists
  • instance_change
  • instance_copy
  • instance_find
  • instance_furthest
  • instance_nearest
  • instance_number
  • instance_place
  • instance_place_list
  • instance_position
  • instance_position_list

请注意，有两个额外的内置变量（这些是全局范围的变量）和一个与使用实例有关的函数，这对检索某些细节很有用。

  • instance_id_get
  • instance_id
  • instance_count

object 的每个实例中都有一些内置变量，可以用来控制实例的显示方式、如何使用其他assets ，以及如何对碰撞作出反应等不同方面，这些变量都在这里列出。

  • 实例变量

实例中有一个数组变量，可以用来获取或设置报警事件值，但也有两个函数可以用来做同样的事情。

  • alarm_get
  • alarm_set

最后，你也有可能停用实例，这意味着它们不会处理任何事件，尽管它们仍然存在于游戏中。与此相关的功能可以在下面的页面中找到。

  • 停用实例
