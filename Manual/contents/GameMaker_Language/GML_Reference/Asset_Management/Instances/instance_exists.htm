实例_存在

这个函数有两种使用方法，取决于你想检查什么。你可以给它一个要检查的object_index，在这种情况下，如果在当前的object 中存在任何指定的room 的活动实例，这个函数将返回 true ，或者你也可以给它一个实例id，在这种情况下，如果该特定实例存在并且在当前的room 中是活动的，这个函数将返回 true 。

请注意，这个功能不考虑那些已经用实例停用功能停用的实例。

语法。

instance_exists(obj);

争论 类型 描述
obj 实例ID或对象资产 The object or instance to check for the existence of.

返回。

Boolean

例子。

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

上述代码检查是否有任何object "obj_Enemy "的实例存在，如果没有，它将添加到变量 "score "并改变room 。

返回:实例
