实例_最远

这个函数将检查给定的object 的所有实例，看哪个实例离给定的x/y点最远。所有检查将从给定的x/y位置到指定的object 的实例的原点（x/y位置）。如果不存在object 的实例，该函数将返回关键字 noone，但如果有实例存在，那么它将返回 id的实例。请注意，如果运行代码的实例与被检查的object 具有相同的object 索引，那么它将被包括在检查中（这包括对父objects 的检查，如果调用的实例也是父的一个孩子）。

语法。

instance_furthest(x, y, obj);

争论类型 描述
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

返回。

Instance ID or noone

例子。

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}

上述代码将找到与运行代码的实例共享相同的object 索引的最远的实例，并将其id存储在一个变量中。然后检查这个变量是否与调用实例的id相同，如果不相同，就在两个实例之间画一条线。
