闹钟_设置

这个功能可以用来设置一个警报。你提供从0到11的报警号码，然后提供要设置报警的值。这个值必须是一个整数，你可以把它设置为-1来停止报警（非整数值将被四舍五入到最接近的整数）。这是一种直接设置报警数组的替代方法。

语法。

alarm_set(index, value);

争论 类型 描述
index 真实的 The alarm index to set, from 0 to 11.
value 真实的 The value (an integer) to set the alarm to.

返回。

N/A

例子。

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

上面的代码检查了调用实例中的所有警报，如果它们大于0，就把它们设置为-1，阻止它们继续倒计时。
