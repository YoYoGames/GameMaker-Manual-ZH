例子_拷贝

通过这个函数，你可以 "克隆 "一个实例，因为这将创建一个新版本的实例，在其相同的位置运行代码。参数 "perf "用于指示这个新实例是否执行创建事件。这个函数返回 id的参数，然后可以将其存储在一个变量中，或者用来访问该实例。

注意：如果你选择不执行创建事件，如果实例依赖于该事件中初始化的任何变量，你可能遇到错误。

语法。

instance_copy(perf);

争论 类型 描述
perf 布尔型 Whether to perform the new instance's Create event (true) or not (false).

返回。

Instance ID

例子。

var inst = instance_number(object_index);
if inst < 10
{
    instance_copy(true);
}

object上面的代码创建了一个局部变量，并使用它来存储room 中运行代码的实例的数量。如果这个数字小于10，那么这个实例就会对自己进行复制。

返回:实例
