实例_激活_所有

通过这个函数，你可以告诉GameMaker激活所有先前在room 中被停用的实例。注意，激活不是即时的，以这种方式激活的实例不会被认为是激活的，直到调用函数的事件结束。 在调用该函数的事件结束之前不会被认为是激活的。

语法。

instance_activate_all();

返回。

例子。

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

上面的代码激活了room 内的所有实例，然后停用了那些在当前相机视图限制之外64px的实例。
