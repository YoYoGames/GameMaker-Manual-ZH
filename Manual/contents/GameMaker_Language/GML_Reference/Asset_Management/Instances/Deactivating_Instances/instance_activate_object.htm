实例_激活_对象

通过这个功能，你可以从之前被停用的所有实例中激活一个单一的实例或一个特定object 的所有实例。请注意，如果你已经停用了一个被标记为持久性的实例或object ，那么 你将需要在改变room 之前用这个函数重新激活它，否则它将不会被延续，而是被丢弃。还需要注意的是，激活并不是即时的，以这种方式激活的实例不会被认为是激活的。 方式激活的实例在调用该函数的事件结束前不会被认为是激活的。

语法。

instance_activate_object(obj);

争论类型 描述
obj The object or instance to activate (the keyword all can also be used).

返回。

例子。

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);
instance_activate_object(obj_Lights);

上面的代码激活了room 内的所有实例，然后停用了那些在当前摄像机视图限制之外的实例，除了object "obj_Lights"，它们在最后又被重新激活。
