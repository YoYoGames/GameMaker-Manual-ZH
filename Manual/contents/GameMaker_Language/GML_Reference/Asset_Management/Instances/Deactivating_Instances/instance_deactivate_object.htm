instance_deactivate_object

通过这个功能，你可以从之前激活的所有object ，停用一个单一的实例或所有特定的实例。请注意，如果你已经停用了一个被标记为持久性的实例或object ，那么 你将需要在改变 之前用函数重新激活它。 instance_activate_object()room来重新激活它，否则它将不会被延续，而会被丢弃。还要注意，停用 并不是瞬间完成的，一个以这种方式被停用的实例在调用该函数的事件结束之前不会被认为是不活跃的。

注意：如果你在room 开始时停用一个实例（即：从room 创建代码，或从room 内的实例创建事件），所有从room 编辑器放置在room 内的实例在停用前仍将运行其创建事件。

警告：停用启用了物理的实例不会停止它们的灯具在物理模拟中的互动。为此，你应该设置它们的 phy_active变量 为 true 或 false ，因为你激活/停用了这些实例。

语法。

instance_deactivate_object(obj);

争论类型 描述
obj The object or instance to deactivate (the keyword all can also be used).

返回。

例子。

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上述代码停用了object "obj_Enemy "的所有实例，然后激活了room 中的一个区域。
