instance_deactivate_layer

通过这个函数，你可以停用分配给某个特定层的所有实例。你需要提供图层ID，它可以是代码编辑器中写的图层名称（作为string ），也可以是实际的图层ID值，由 layer_create()，并注意你只能用这个函数停用实例层。请注意，如果你已经停用了一个有objects 实例标记为持久性的层，那么你将需要在改变 之前用函数重新激活该层。 instance_activate_layer()room来重新激活该层，否则该层上的任何持久化实例将不会被带入并被丢弃。还要注意的是，停用并不是即时的，以这种方式停用的实例在调用该函数的事件结束前不会被认为是不活跃的。

注意：如果你在room 开始时停用一个实例（即：从room 创建代码，或从room 内的实例创建事件），所有从room 编辑器放置在room 内的实例在停用前仍将运行其创建事件。

警告：停用启用了物理的实例不会停止它们的灯具在物理模拟中的互动。为此，你应该在激活/停用实例时将它们的 phy_active变量设置为 true 或 false ，因为你激活/停用了这些实例。

语法。

instance_deactivate_layer(obj);

争论 类型 描述
layer 图层ID或字符串 The layer name string (or ID value) to be used

返回。

N/A

例子。

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上述代码停用了分配给 "Enemy_Layer "层的所有实例，然后激活了room 内的一个区域。
