实例_激活_区域

通过这个功能，你可以在room 内定义一个区域来激活以前被停用的实例。这个区域可以被标记为 "内部 "或 "外部"，如下图所示。

你可以在上图中看到，"苹果 "实例总是处于活动状态，因为，即使sprite 本身没有与该区域重叠，但其边界框却在重叠。所以，当实例的边界框与之重叠时，就被认为是在指定的区域内，而碰撞掩码的状态（精确与否）不被考虑。请注意，激活并不是瞬间完成的，以这种方式激活的实例在调用该函数的事件结束前不会被认为是激活的。

语法。

instance_activate_region(left, top, width, height, inside);

争论 类型 描述
left 真实的 The x coordinate of the left of the rectangular region to activate.
top 真实的 The y coordinate of the top of the rectangular region to activate.
width 真实的 The width of the region to activate.
height 真实的 The height of the region to activate.
inside 布尔型 Whether to activate instances on the inside of the region (true) or the outside (false).

返回。

N/A

例子。

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上面的代码停用了所有的实例，除了正在运行代码的那个，然后激活了room 内的一个区域。
