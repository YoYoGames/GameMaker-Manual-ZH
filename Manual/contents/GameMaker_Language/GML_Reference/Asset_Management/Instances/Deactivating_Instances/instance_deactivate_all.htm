实例_停用_所有

这个函数将停用所有在运行代码时在room 的实例。如果 "notme "flag 被设置为 false ，这可能包括运行代码的实例，但通常你希望该实例处于活动状态，在这种情况下，"notme "flag 应该被设置为 true 。注意，停用不是即时的，以这种方式停用的实例在调用该函数的事件结束前不会被视为不活动。

注意：如果你在room 开始时停用一个实例（即：从room 创建代码，或从room 内的实例创建事件），所有从room 编辑器放置在room 内的实例在停用前仍将运行其创建事件。

警告：停用启用了物理的实例不会停止它们的灯具在物理模拟中的互动。为此，你应该在激活/停用实例时将它们的 phy_active变量设置为 true 或 false ，因为你激活/停用了这些实例。

语法。

instance_deactivate_all(notme);

争论 类型 描述
notme 布尔型 Whether to keep the calling instance activated (true) or not (false).

返回。

N/A

例子。

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上面的代码停用了所有的实例，除了正在运行代码的那个，然后激活了room 内的一个区域。
