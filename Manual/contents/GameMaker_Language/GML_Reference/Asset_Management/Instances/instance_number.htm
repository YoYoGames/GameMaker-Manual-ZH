instance_number

通过这个函数，你可以找出在object 中存在多少个指定的room 的活动实例。使用这个函数检查时，如果object 是一个父级，那么所有的子objects 也将包括在返回值中，同时注意，那些已经用实例停用函数停用的实例将不包括在这个检查中。

语法。

instance_number(obj);

争论类型 描述
obj The object to total the number of instances of.

返回。

例子。

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

上述代码将检查与当前实例相同的object ，然后如果少于50个，就在room 内的随机位置再创建一个实例。

返回:实例
