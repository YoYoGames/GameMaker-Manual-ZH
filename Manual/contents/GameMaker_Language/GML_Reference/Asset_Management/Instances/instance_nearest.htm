实例_最近的

这个函数将检查给定的object 的所有实例，看哪个离给定的x/y点最近。所有检查将从给定的x/y位置到指定的object 的实例的原点（x/y位置）。如果不存在该object 的实例，该函数将返回关键字 noone，但如果有实例存在，那么它将返回 id的实例。请注意，如果运行代码的实例与被检查的object 具有相同的object 索引，那么它将被包括在检查中（这包括对父objects 的检查，如果调用的实例也是父的一个孩子）。

语法。

instance_nearest(x, y, obj);

争论类型 描述
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

返回。

Instance ID or noone

例子。

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

上面的代码将当前实例移动10000像素，然后检查它之前的位置，找到最近的相同类型的实例object 。如果这个实例是它自己，它将做的只是移回它的原始位置，但如果找到的实例是不同的，它将在两者之间画一条线。
