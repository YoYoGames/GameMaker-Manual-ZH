实例_位置

object通过这个函数，你可以检查一个位置是否与另一个实例或一个room 的所有实例发生碰撞。当你使用这个函数时，你是在检查一个实例或一个object 中的一个点。这个检查将针对实例的边界框进行，如果该实例有精确的碰撞检查，则针对该实例的掩码，并将返回唯一的实例。 id.如果你不需要碰撞的实例的id，你应该考虑使用 position_meeting()来代替。这个函数也接受特殊的关键字 all 和 other ，如果没有发生碰撞，将返回关键字 noone ，如果发生了碰撞，将返回找到的实例的唯一ID值。

语法。

instance_position( x, y, obj );

争论 类型 描述
x 真实的 The x position to check for instances.
y 真实的 The y position to check for instances.
obj 对象资产或实例ID The object to check for instances of.

返回。

Instance ID or noone

例子。

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst) image_index=1;
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

上述代码将检查在鼠标位置是否与 "obj_Pause_Button "的实例发生碰撞，如果有的话，它将使用返回的id将其image_index设置为一个新的值，然后创建一个新的object "obj_Menu "的实例。
