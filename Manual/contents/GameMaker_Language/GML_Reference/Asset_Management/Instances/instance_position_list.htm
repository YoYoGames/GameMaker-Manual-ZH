实例_位置_列表

通过这个函数，你可以检查一个位置是否与另一个object 的所有实例发生碰撞。当你使用这个功能时，你是在检查room 中的一个单点与一个object 的所有实例的碰撞情况，检查将针对实例的边界框进行，如果该object 有精确碰撞检查，则针对实例的掩码进行。

你也提供了一个DS列表，所以 id的值将被添加到给定列表的末尾。你可以在调用这个函数之前清除这个列表，这样它就只包含这个函数调用的结果。你还可以选择根据被检查的位置到被发现有碰撞的实例的原点的距离来排列列表。

请注意，该函数也接受特殊关键字 all，在这种情况下，所有被发现有碰撞的实例将被列出。该函数返回找到的实例的数量，如果没有找到，则返回0。

语法。

instance_position_list(x, y, obj, list, ordered);

争论 类型 描述
x 真实的 The x position to check for instances.
y 真实的 The y position to check for instances.
obj 对象资产或实例ID The object to check for instances of.
list DS列表ID The DS list to use to store the IDs of colliding instances.
ordered 布尔型 Whether the list should be ordered by distance (true)或不(false)。

返回。

Real (The number of instances found to be in collision)

例子。

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

上面的代码将检查与鼠标位置上发现的所有实例的碰撞情况。这些将被添加到一个DS列表中，然后通过循环来销毁每个碰撞的实例。
