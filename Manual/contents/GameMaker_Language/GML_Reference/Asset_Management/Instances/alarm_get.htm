报警_获取

这个函数可以用来获取给定警报的当前值。你提供从0到11的报警号码，这个函数将返回报警当前的值。这是一种替代直接获得报警数组值的方法。

语法。

alarm_get(index);

争论 类型 描述
index 真实的 The alarm index to get, from 0 to 11.

返回。

Real (integer)

例子。

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

上面的代码检查了调用实例中的所有警报，如果它们大于0，就把它们设置为-1，阻止它们继续倒计时。
