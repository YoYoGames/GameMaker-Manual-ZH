实例_地点

通过这个函数，你可以使用运行检查代码的实例的碰撞掩码来检查一个位置是否与另一个实例或一个object 的所有实例发生了碰撞。当你使用这个功能时，你实际上是要求GameMaker将实例移动到新的位置，检查是否有碰撞，然后返回，并告诉你是否发现了碰撞。

这将对精确的碰撞起作用，但只有当实例和被检查的object ，都选择了精确的碰撞掩码，否则只应用边界框碰撞。

这个函数将返回被碰撞的唯一实例 idobject 的唯一实例，但如果不需要的话，使用这个函数会更快一些。 place_meeting().这个函数也接受特殊的关键字 all 和 other ，如果没有发生碰撞，将返回关键字 noone ，如果发生碰撞，将返回找到的唯一实例ID值。

语法。

instance_place(x, y, obj);

争论 类型 描述
x 真实的 The x position to check for instances.
y 真实的 The y position to check for instances.
obj 对象资产或实例ID The object to check for instances of.

返回。

Instance ID or noone

例子。

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

上述代码将检查与 "obj_Enemy "实例的碰撞，如果有的话，它将把 "hp "变量减少，减少的量存储在碰撞实例的 "dmg "变量中，然后销毁碰撞实例。
