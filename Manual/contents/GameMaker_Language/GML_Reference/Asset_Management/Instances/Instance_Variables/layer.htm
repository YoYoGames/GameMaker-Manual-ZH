层

这个内置变量是为room 中的每一个实例创建的，包含了该实例被分配到的层的ID值。这个值可以在其他函数中使用，比如 layer_get_depth()或者可以改变它来将实例移动到另一个层，但是请注意，如果被分配的层在当前的room ，那么你会得到一个错误，会迫使你的游戏关闭。当分配一个图层时，你必须提供由函数返回的唯一的图层IDlayer_get_id()(当使用命名的room 图层时)，或由函数返回的 layer_create()(当你在运行时创建你自己的层时)。

重要的是，如果你已经用 instance_create_depth()函数创建了实例，或者手动改变了 depth变量，分配给实例的图层就会变成一个 "被管理的 "图层，也就是GameMaker自动控制和管理的图层。在这些情况下， layer 变量将返回-1。

语法。

layer;

返回。

Layer ID

例子。

layer = layer_create(-1000);

上述代码将创建一个深度为-1000的新图层，然后将实例 layer 变量设置为返回的图层ID，将实例从当前所在的图层移动到正在创建的新图层。
