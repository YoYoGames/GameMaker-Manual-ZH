实例变量

当你创建一个新的object ，它将带有某些已经初始化为默认值的变量。object这些变量我们称之为实例变量，因为它们将为你在游戏中放置的每一个rooms 的实例创建，一旦一个实例被创建，这些变量的值对该实例来说是唯一的，而且只有该实例。其中一些变量你会经常使用，比如 x 和 y ，而其他变量则需要较少，这取决于你想做什么，但总的来说，它们非常有用，在可能的情况下，你应该寻求使用它们而不是创建你自己的自定义变量。

大多数实例变量可以被设置和读取，允许你仅仅通过调整某个变量的值来改变一个实例的属性和行为--例如，你可以阻止一个实例的绘制，只需将 visible 内置变量设置为 false 。下面你可以找到不同的变量，这些变量对你游戏中所有objects 的所有实例都进行初始化。

重要的是，如果一个实例是在一个 sequence那么这些变量中的一些--比如image_xscale /image_yscale / image_angle /x /y --在开始播放后，sequence 更新每一步时，将被覆盖。

这些变量处理的是一般的实例属性。
