速度

GameMaker中的所有实例都有某些 "内置 "属性，你可以使用和设置这些属性来管理它们的外观和行为。 speed 是这些属性之一，它定义了实例每一步将移动多少像素。 不同于 hspeed和 vspeed， speed 没有与之相关的方向，因为这是由实例的 direction的值，但它 可以有一个负值，在这种情况下，实例将以与 direction 函数设置的方向相反的方向移动（即： direction - 180°）。请注意，设置 speed 和/或 direction, 也会修改 hspeed 和 vspeed 变量的值，并且 gravity, gravity_direction 和 friction当它们在你的游戏中使用时，都可以修改这个变量的值。

语法。

speed;

返回。

(single precision floating point value)

例子。

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将使用方向键来设置实例的速度和方向。
