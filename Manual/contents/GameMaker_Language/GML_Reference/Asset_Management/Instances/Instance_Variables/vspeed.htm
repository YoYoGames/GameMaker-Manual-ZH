速率

vspeed 是所有实例都具有的 "内置 "属性之一，它定义了实例的垂直移动速度（沿Y轴），单位是每步的像素。因此， vspeed 为 3 意味着每步向底部移动 3 个像素 (+y)每一步，而 vspeed 为-3则意味着每一步向顶部（-y）移动3个像素。

请注意，如果你设置了 speed 和/或 direction 变量，那么 vspeed 的值就会自动更新以反映这些变化。 反映这些变化，同样，改变 vspeed 的值也会影响 speed 和 direction 的值。还要注意的是 gravity, gravity_direction和 friction都可以在你的游戏中使用时修改这个变量的值。

语法。

vspeed;

返回。

(single precision floating point value)

例子。

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

上面的代码将根据哪个键被按下而改变垂直速度。
