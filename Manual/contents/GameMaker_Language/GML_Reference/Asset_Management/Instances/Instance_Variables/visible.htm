可见

通过将这个变量设置为 true （可见）或 false （不可见），可以将一个实例标记为 visible 或不标记。它的作用是告诉GameMaker跳过这个实例的绘制事件，所以在使用时必须注意，因为这意味着在实例不可见时，放置在绘制事件中的代码将不会被运行。还要注意，如果 layer的实例被标记为不可见，将此变量设置为 true 将不会有任何影响，直到该层本身也被标记为可见。

语法。

visible;

返回。

例子。

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

上述代码将在碰撞事件中检查 "另一个 "实例的可见flag ，如果它被设置为 true ，则将该实例移回其先前的位置。
