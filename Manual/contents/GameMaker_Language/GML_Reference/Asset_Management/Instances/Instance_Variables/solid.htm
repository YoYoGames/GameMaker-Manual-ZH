固体

一个实例可以通过对象编辑器中的object 属性标记为实体，或者通过改变这个内置变量的值。如果 solid 被设置为 true ，那么当检测到碰撞时，碰撞的实例会自动返回到它在碰撞前一步的位置（然后在碰撞事件中运行任何代码或动作）。如果它被设置为 false ，所有的定位都必须通过碰撞事件来处理。

语法。

solid;

返回。

例子。

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

上述代码将检查碰撞中的 "另一个 "实例的实体flag ，如果它被设置为 true ，则将该实例移回其先前的位置。
