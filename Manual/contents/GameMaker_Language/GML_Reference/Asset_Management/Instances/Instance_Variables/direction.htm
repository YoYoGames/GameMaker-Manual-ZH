direction

GameMaker中的所有实例都有某些 "内置 "属性，你可以使用和设置这些属性来管理它们的外观和行为。 Direction 是这些属性之一，可以用来设置实例的运动方向。 实例的运动方向，当该实例有一个 speed请注意，GameMaker中的方向通常被计算为0°为右，90°为上，180°为左，270°为下，而 gravity和 gravity_direction变量在你的游戏中使用时可以修改 direction 。

语法。

direction

返回。

(single precision floating point value)

例子。

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将根据哪个键（左或右）被按下而改变实例的运动方向。
