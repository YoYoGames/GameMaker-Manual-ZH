重力方向

gravity_direction 是所有实例拥有的 "内置 "属性之一，可以用来设置运动方向，当实例的 gravity注意，GameMaker中的方向通常被计算为0°为右，90°为上，180°为左，270°为下。

语法。

gravity_direction;

返回。

(single precision floating point value)

例子。

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

如果玩家object ，遇到 "obj_switch "的一个实例，上述代码将改变重力方向。
