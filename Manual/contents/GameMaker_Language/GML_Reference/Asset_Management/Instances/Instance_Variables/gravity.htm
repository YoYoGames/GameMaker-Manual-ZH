重力

gravity 是所有实例都有的 "内置 "变量之一，当被设置时，将在实例中应用一个恒定的力。 gravity_direction中施加一个恒定的力，同时影响实例 speed 和 direction 。请注意，重力是一个累积的力，如果你选择不对最终的速度设置上限，就会加速object ，通常你会将这个变量设置为小数点后的数值，比如0.01。如果你把 gravity 设置为0，那么就不会对实例施加重力（这是默认值）。

语法。

gravity;

返回。

(single precision floating point value)

例子。

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

上面的代码只有在实例下面没有找到任何 "obj_Ground "的实例时才会应用重力。
