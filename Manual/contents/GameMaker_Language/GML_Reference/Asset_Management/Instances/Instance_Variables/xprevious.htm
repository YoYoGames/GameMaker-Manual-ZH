xprevious

这个内置变量返回实例的前一个X位置。这个变量将在begin step事件开始前被设置，但它也可以通过代码在任何时候被设置，这意味着你可以在必要时给它自己的自定义值。

语法。

xprevious;

返回。

(single precision floating point value)

例子。

if x != xprevious || y != yprevious
{
    moved = true;
}

上面的代码检查 xprevious 和 yprevious 值与当前的x和y值之间是否有任何差异。如果有，它将一个变量设置为真。
