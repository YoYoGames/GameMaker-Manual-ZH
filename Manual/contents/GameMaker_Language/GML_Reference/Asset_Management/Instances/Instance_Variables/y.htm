y

一个实例的y值是在当前room ，以像素为单位的垂直位置。这个值可以是0，也可以是正值或负值，其中0是room 的顶部，向下移动增加y，向上移动减少y（y的负值意味着 y的负值意味着该实例已经超出了room 的顶部）。你甚至可以给y一个真实的值，比如12.345，但由于你不能得到0.345的像素，实例可能看起来没有移动，尽管这些值将被维护并被内部使用 GameMaker内部维护和使用，并且是完全有效的。

object 通过设置y（和x）值，你可以让room ，跳到你选择的位置，或者你可以对它们进行较小量的加减，以产生运动的错觉，而不需要实际使用内置的 speed 和 direction功能。在制作游戏时应考虑到这一点，因为通常一种类型的运动比另一种类型的运动更适合某种特定的游戏风格。

语法。

y;

返回。

(single precision floating point value)

例子。

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

上面的代码将使用方向键来增加或减少Y的位置，因此创造了运动的错觉。
