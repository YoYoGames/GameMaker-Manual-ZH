x开始

这个变量存储了实例在room 中首次创建时的初始X位置。这不是一个只读的变量，可以设置，也可以读取。

语法。

xstart;

返回。

(single precision floating point value)

例子。

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

上述代码将检查是否与 "obj_spike "的实例发生碰撞，如果有，则从分数中扣除100，并将实例移回其起始位置。
