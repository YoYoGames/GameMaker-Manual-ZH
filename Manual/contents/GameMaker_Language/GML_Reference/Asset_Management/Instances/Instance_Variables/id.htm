身份证

这个只读的变量保存着实例的唯一识别号码。你创建的每一个实例--无论是通过代码还是通过在room 编辑器中把它们添加到Room ，都会被赋予一个编号，这个编号在内部用来识别这个实例，而变量 id 是你可以用来引用它。当使用 instance_create_layer() 或 instance_create_depth() ，以及其他实例函数创建一个实例时， id 也会被返回（并且可以存储在一个变量中）。

这个变量现在是对一个实例的引用，而不仅仅是一个真实的数字。你可以通过调用 real()或 int64()就可以得到实例的ID号。

注意这个变量的值与实例在room 编辑器中的标识符不一样，也与包含所有当前活动实例的ID的 instance_id数组，后者包含了所有当前活动实例的ID。

语法。

id;

返回。

Instance ID

例子。

for (var i = 0; i < instance_count; i++;)
{
    if instance_id[i] != id
    {
        instance_id[i].scr += 5;
    }
}

上面的代码在 "scr "变量中增加了5，除了运行代码的那个，room 中的每个活动实例。它是通过循环查看所有活动实例（使用 "instance_id "数组来返回每个活动实例的ID），并将它们与内置的 "id "变量进行比较来实现的，这个变量是运行代码的原始实例的ID。
