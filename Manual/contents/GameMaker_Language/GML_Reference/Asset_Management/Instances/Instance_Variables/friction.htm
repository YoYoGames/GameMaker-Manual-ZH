摩擦力

Friction 是实例可以拥有的 "内置 "属性之一，可以用来减缓实例的速度，当 speed为零以外的时候，可以用来降低实例的速度。它的工作原理很简单，从每一步的速度中减去一个量，直到object 的速度为0，所以如果摩擦力被设置为，例如，0.1，而实例的速度为1（每一步1像素），它将在10步过后减速并停止。还要注意的是，摩擦力同样适用于正数和负数的速度，最终的结果是object ，在给定的时间后，速度为0。

语法。

friction;

返回。

(single precision floating point value)

例子。

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

上述代码只有在实例的绝对速度高于0时才会应用摩擦力。
