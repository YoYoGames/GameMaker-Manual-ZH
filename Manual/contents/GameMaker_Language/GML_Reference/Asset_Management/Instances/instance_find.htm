instance_find

所有的实例都有一个唯一的标识符(id)，可以用来在游戏运行时修改和操作它们，但是你可能并不总是知道某个特定实例的ID是什么，所以这个函数可以帮助你，因为你可以用它来遍历所有的实例，找到你需要的东西。你指定你想找到的实例的object 和一个数字，如果在实例列表的那个位置有一个实例，那么这个函数就会返回该实例的id，如果没有，就会返回特殊的关键字 noone 。你也可以使用关键字 all 来遍历一个room 中的所有实例，以及一个父object 来遍历属于该父/子层次结构的所有实例，你甚至可以指定一个实例（如果你有它的 id ）作为检查，看它是否真的存在于当前room 。请注意，由于实例是以任意方式排序的，所以这个函数对实例的检查没有特定的顺序，任何实例都可以在任何位置。

在这个函数中，"n "的最大值将是

  • 对于关键词 all 。 instance_count - 1
  • 对于一个object 指数。 instance_number(OBJ) - 1

语法。

instance_find(obj, n);

争论 类型 描述
obj 对象资产 The object to find the nth instance of
n 真实的 The number of the instance to find.

返回。

Instance ID or noone

例子。

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

上面的代码将使用 for loop ，遍历 "obj_Enemy "的所有实例，并将其id存储在数组 "enemy[]"中。
