清理标签

通过这个函数，你可以从asset 浏览器中清除任何asset 上存在的所有标签。你可以提供asset 名称（作为一个string ）或其asset 索引，如果你提供一个asset 索引值，那么你将需要提供可选的asset 类型参数（一个常数），因为不同类型的assets 可以有相同的索引，尽管它们不能有相同的名称。可用的asset 类型在下表中列出。

恒定 描述
asset_object 给定的名称指的是一个对象。
asset_sprite 给定的名称指的是一个精灵。
asset_sound 这个名字指的是一种声音。
asset_room 这个名字指的是一个 room.
asset_tiles 给定的名称指的是一个瓦片组。
asset_path 给定的名称指的是一个路径。
asset_script 这个名字指的是一个 script.
asset_font 该名称指的是一种字体。
asset_timeline 这个名字指的是一条时间线。
asset_shader 给定的名称指的是一个着色器。
asset_animationcurve 该名称指的是一个动画曲线。
asset_sequence 给定的名称指的是一个Sequence。

如果函数成功地清除了标签，它将返回 true ，否则将返回 false 。

语法。

asset_clear_tags(name,[asset_type]);

争论 类型 描述
name_or_index 字符串或资产 The name of the asset (a string) or its index value (an integer).
[asset_type] 资产类型不变 OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

返回。

Boolean

例子。

asset_clear_tags(obj_Enemy_Parent, asset_object);

上述代码将清除给定的object asset 中的所有标签。

返回。资产和标签
