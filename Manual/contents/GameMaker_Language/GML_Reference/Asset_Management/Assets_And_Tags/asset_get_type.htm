资产_获取类型

通过这个函数，你可以从它的名字（一个asset ）中获得被引用的string 的类型。下面列出的常数之一将被返回。

语法。

asset_get_type(name);

争论 类型 描述
name 字符串 The name of the game asset to get the type of (a string).

返回。

Asset Type Constant

恒定 描述
asset_object 给定的名称指的是一个对象。
asset_sprite 给定的名称指的是一个精灵。
asset_sound 这个名字指的是一种声音。
asset_room 这个名字指的是一个 room.
asset_tiles 给定的名称指的是一个瓦片组。
asset_path 给定的名称指的是一个路径。
asset_script 这个名字指的是一个 script.
asset_font 该名称指的是一种字体。
asset_timeline 这个名字指的是一条时间线。
asset_shader 给定的名称指的是一个着色器。
asset_animationcurve 该名称指的是一个动画曲线。
asset_sequence 给定的名称指的是一个Sequence。
asset_unknown 该名称指的是一个asset ，它要么不存在，要么不是上述所列的一个。

例子。

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

上面的代码检查动态创建的asset 名称，看asset 的类型是否正确。如果不是，那么将显示一条调试信息，否则，asset 名称将被用于分配给实例的asset 。

返回。资产和标签
