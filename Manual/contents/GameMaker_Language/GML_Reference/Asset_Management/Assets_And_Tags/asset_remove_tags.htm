资产_删除标签

通过这个函数，你可以从strings 浏览器中删除一个或多个标签asset 到任何asset 。你可以提供asset 的名称（作为一个string ）或其asset 的索引，以及一个单一的标签string 或一个数组，其中每个项目是一个单一的标签string 。如果你提供一个asset 索引值，那么你需要提供可选的asset 类型参数（一个常数），因为不同类型的assets 可以有相同的索引，尽管它们不能有相同的名字。可用的asset 类型在下表中列出。

恒定 描述
asset_object 给定的名称指的是一个对象。
asset_sprite 给定的名称指的是一个精灵。
asset_sound 这个名字指的是一种声音。
asset_room 这个名字指的是一个 room.
asset_tiles 给定的名称指的是一个瓦片组。
asset_path 给定的名称指的是一个路径。
asset_script 这个名字指的是一个 script.
asset_font 该名称指的是一种字体。
asset_timeline 这个名字指的是一条时间线。
asset_shader 给定的名称指的是一个着色器。
asset_animationcurve 该名称指的是一个动画曲线。
asset_sequence 给定的名称指的是一个Sequence。

如果函数成功地移除标签，它将返回 true ，否则将返回 false 。

语法。

asset_remove_tags(name_or_index, tags, [asset_type]);

争论 类型 描述
name_or_index 字符串或资产 The name of the asset (a string) or its index value (an integer).
tags 字符串或字符串数组 A single asset tag string or an array with various asset tags.
[asset_type] 资产类型不变 OPTIONAL! The type of asset to remove the tags from, only used when supplying an index for the first argument.

返回。

Boolean

例子。

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

上述代码将创建一个标签数组，然后将它们从给定的object 。

返回。资产和标签
