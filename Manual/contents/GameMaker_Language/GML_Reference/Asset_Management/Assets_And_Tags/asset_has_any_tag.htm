资产_有任何标签

通过这个函数，你可以检查一个或多个标签strings 是否被分配给任何asset 浏览器的asset 。你可以提供asset 的名称（作为一个string ）或其asset 的索引，以及一个单一的标签string 或一个数组，其中每个项目是一个单一的标签string 。如果你提供一个asset 索引值，那么你需要提供可选的asset 类型参数（一个常数），因为不同类型的assets 可以有相同的索引，尽管它们不能有相同的名字。可用的asset 类型在下表中列出。

恒定 描述
asset_object 给定的名称指的是一个对象。
asset_sprite 给定的名称指的是一个精灵。
asset_sound 这个名字指的是一种声音。
asset_room 这个名字指的是一个 room.
asset_tiles 给定的名称指的是一个瓦片组。
asset_path 给定的名称指的是一个路径。
asset_script 这个名字指的是一个 script.
asset_font 该名称指的是一种字体。
asset_timeline 这个名字指的是一条时间线。
asset_shader 给定的名称指的是一个着色器。
asset_animationcurve 该名称指的是一个动画曲线。
asset_sequence 给定的名称指的是一个Sequence。

如果该函数成功，并且一个或多个标签存在于asset ，那么它将返回 true ，否则它将返回 false 。如果你需要检查与任何给定的标签或标签集的精确匹配，那么使用函数 asset_has_tags().

语法。

asset_has_any_tag(name_or_index, tags, [asset_type]);

争论 类型 描述
name_or_index 绳子或 资产 The name of the asset (a string) or its index value (an integer).
tags 字符串或 阵列字符串的 A single asset tag string or an array with various asset tags.
[asset_type] 资产类型不变 OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

返回。

Boolean

例子。

var _a = array_create(3);
_a[0] = "boss";
_a[1] = "main_boss";
_a[2] = "final_boss";

if asset_has_any_tag(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

上述代码将创建一个标签数组，然后检查是否有任何标签应用于给定的object ，如果有，它将创建另一个object 的实例。

返回。资产和标签
