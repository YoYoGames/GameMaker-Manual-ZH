tag_get_asset_ids

通过这个函数，你可以得到一个给定类型的、具有给定标签的所有assets 。你可以提供一个单一的标签（作为string ）或一个数组，其中数组中的每一项都是一个标签（作为string ），以及要检查的asset 的类型。asset 的类型应该是以下常数之一。

恒定 描述
asset_object 给定的名称指的是一个对象。
asset_sprite 给定的名称指的是一个精灵。
asset_sound 这个名字指的是一种声音。
asset_room 这个名字指的是一个 room.
asset_tiles 给定的名称指的是一个瓦片组。
asset_path 给定的名称指的是一个路径。
asset_script 这个名字指的是一个 script.
asset_font 该名称指的是一种字体。
asset_timeline 这个名字指的是一条时间线。
asset_shader 给定的名称指的是一个着色器。
asset_animationcurve 该名称指的是一个动画曲线。
asset_sequence 给定的名称指的是一个Sequence。

该函数将返回一个数组，其中数组中的每一项都是一个asset 索引值。如果没有具有给定标签的assets ，将返回一个空数组。

语法。

tag_get_asset_ids(tags, asset_type);

争论 类型 描述
tags 字符串或字符串数组 A single asset tag string or an array with various asset tags.
asset_type 资产类型不变 An asset type constant (listed above)

返回。

Array

例子。

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

上面的代码使用标签 "敌人 "找到所有带有该标签的path assets ，然后随机选择一个并将其分配给运行该代码的实例。

返回。资产和标签
