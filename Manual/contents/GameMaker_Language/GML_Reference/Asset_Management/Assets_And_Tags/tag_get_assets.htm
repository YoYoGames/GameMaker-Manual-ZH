tag_get_assets

通过这个函数，你可以检索所有被赋予给定标签的assets 的名称。你可以提供一个单一的标签string ，也可以提供一个数组，数组中的每一项都是一个标签string 。该函数将返回一个数组，其中每个条目是带有给定标签的string 的名称（作为一个asset ）。如果你需要asset 的唯一索引，那么你可以使用这个函数 asset_get_index()和返回的名称一起使用。如果没有带有给定标签的assets ，或者出现错误（例如：给定的标签不存在），那么将返回一个空数组。

语法。

tag_get_assets(tags);

争论 类型 描述
tags 字符串或字符串数组 A single asset tag string or an array with various asset tags.

返回。

Array

例子。

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

上述代码创建了一个列表，然后检索所有带有 "background "标签的assets 的名称。它通过返回的名称数组loops ，检查其中是否有sprite assets ，如果有，则将asset 的唯一索引值添加到列表中，以供将来使用。
