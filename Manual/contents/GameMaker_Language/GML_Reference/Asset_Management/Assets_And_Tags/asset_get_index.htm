资产_获取_索引

你可以使用这个函数从一个游戏的名称中获得唯一的识别索引asset 。如果没有找到asset ，该函数将返回一个值-1，否则它将返回被检查的asset 的唯一索引id。然后，这个ID可以在其他函数中使用，就像你使用任何其他索引值一样，如 sprite_index或 path_index，例如。请注意，尽管这个函数可以用来从assets （见下面的例子）引用strings ，但在使用它之前，你应该始终确保asset 的存在，否则你可能得到错误，使你的游戏崩溃。

语法。

asset_get_index(name);

争论类型 描述
name The name of the game asset to get the index of (a string).

返回。

Asset (any asset type)

例子。

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

上面的代码将从一个object ，获得一个string 的索引，如果这个索引存在，就在游戏中创建一个object 的实例。

返回。资产和标签
