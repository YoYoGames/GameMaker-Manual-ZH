font_replace_sprite_ext

这个函数将取代你之前用函数创建的sprite font 。 font_add_sprite_ext().它使用一个 "sprite strip"，其中每个子图像是一个单独的符号或字母，图像根据函数的参数 "string_map "被映射到字母。这个参数是一个string ，你可以用它来告诉GameMaker，sprite font 的子图像是什么顺序，它将在编写文本时自动映射这些图像。因此，例如，如果你有一个 "AaBbCcDdEeFfGgHh "的字符串映射，你的sprite font ，必须有子图像以这种方式排序。

你还可以将font 的间距设置为按比例 (true) 或不按比例 (false) ，其中按比例的font 是根据字母的实际宽度来确定间距的（例如，字母 "i "比字母 "w "所需的room ），而不按比例的font 将根据子图像的宽度来确定间距，就像单行线font 。最后，你可以定义书写时每个字母之间留出的空间，这可以是任何整数值，0表示没有空间（如果是按比例书写，字母会 "接触"）。该函数返回一个索引值，该索引值应储存在一个变量中，因为在所有进一步引用该font 。

如果函数失败，那么它将默默地失败，而不替换给定的sprite font 。

注意 这个功能要求你以前用sprite font 。 font_add_sprite()，因为它将取代它，不能用于作为游戏资源一部分的fonts 。

语法。

font_replace_sprite_ext(font, spr, string_map, prop, sep);

争论 类型 描述
font 字体资产 The font to replace.
spr 雪碧资产 The sprite to add a font based on.
string_map 字符串 String from which sprite sub-image order is taken.
prop 布尔型 Set as proportional font or not.
sep 真实的 The space to leave between each letter.

返回。

N/A

例子。

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

上面的代码将把索引在变量 "global.Font "中的sprite font asset 替换为索引在变量 "spr_CalcFont "中的sprite 图像。

返回。字体
