font_add_enable_aa

这个功能可以用来启用或禁用抗锯齿(AA)的功能，为fonts 。 font_add().在你添加任何fonts 之前需要调用这个函数，将其设置为 true 将启用AA，将其设置为 false 将禁用。默认情况下，AA是启用的。请注意，这对在调用该函数之前已经添加的fonts 没有影响，该函数只需要在添加font 的时候调用一次，而不是在使用font 的每一次绘制/步骤中调用。

语法。

font_add_enable_aa(enable);

争论类型 描述
enable Whether to enable (true)或禁用(false)AA来增加fonts 。

返回。

例子。

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

上述代码检查添加的抗锯齿的状态fonts ，如果没有启用，我们就启用它。

返回。字体
