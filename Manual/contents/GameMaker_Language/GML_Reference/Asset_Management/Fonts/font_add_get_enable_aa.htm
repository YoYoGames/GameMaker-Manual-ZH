font_add_get_enable_aa

这个函数可以用来检查使用fonts ，是否启用了抗锯齿（AA）。 font_add().如果启用了AA，该函数将返回 true ，如果没有，则返回 false 。请注意，AA是默认启用的，但你可以使用函数来改变添加的fonts 的AA状态。 font_add_enable_aa()，只要在添加font之前调用该函数即可。

语法。

font_add_get_enable_aa();

返回。

例子。

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

上述代码检查添加的抗锯齿的状态fonts ，如果没有启用，我们就启用它。
