font_get_size

通过这个函数，你可以得到任何font 资源的大小，也就是font 资源对话所显示的点值。

语法。

font_get_size(ind);

争论类型 描述
ind Index of the font to get the size of.

返回。

例子。

sz = font_get_size(font0);

这将获得由 "font0 "变量索引的font 的大小，并将其存储在变量 "sz "中。

返回。字体
