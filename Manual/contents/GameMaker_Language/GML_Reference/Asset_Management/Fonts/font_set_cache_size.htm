font_set_cache_size

在进行图像混合和着色时，HTML5 无法以可执行文件的方式动态地进行。因此，GameMaker必须临时保存图像的混合副本，并在需要时加载 他们在需要时载入。这个函数设置了在旧的副本被覆盖之前可以缓存多少个给定的font 的混合副本。默认值是4。

注意 对于sprite fonts ，你应该使用与之相当的函数sprites 。 sprite_set_cache_size().

语法。

font_set_cache_size(ind, max);

争论类型 描述
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

返回。

例子。

font_set_cache_size(fnt_MainMenu, 2);

font cache 这将把变量 "fnt_MainMenu "中索引的font ，设置为2份。

返回。字体
