font_replace_sprite

有了这个函数，你可以用一个 "精灵条 "来替换之前创建的sprite font asset ，其中每个子图像将是一个单独的符号或字母。索引是要使用的第一个字符的UTF8地图编号，使用的范围与正常的font ，以"！"开始，但请记住，你应该只包括你需要的东西，你应该使用 ord()函数来获取起始字母的正确UTF8值。

你还可以将font 的间距设置为按比例 (true) 或不按比例 (false) ，其中按比例的font 是根据字母的实际宽度来确定间距的（例如，字母 "i "比字母 "w "所需的room ），而不按比例的font 将根据子图像的宽度来确定间距，就像单行线font 。最后，你可以定义书写时每个字母之间留出的空间，这可以是任何整数值，0表示没有空间（如果是按比例书写，字母会 "接触"）。

如果函数失败，那么它将默默地失败，而不替换给定的sprite font 。

值得注意的是，字符图中不能有任何空格，因此，如果你，例如，使用'.'和'0'，但不使用'/'，你需要包括'/'，或者在这两个使用的字符之间留下一个空白子图像。下面的图片显示了一个典型的sprite font 。

语法。

font_replace_sprite(ind, spr, first, prop, sep);

争论 类型 描述
ind 字体资产 The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr 雪碧资产 The sprite to add a font based on.
first 真实的 The index of the first available character.
prop 布尔型 Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep 真实的 The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

返回。

N/A

例子。

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

上述代码将用外部图像文件中的一个新的font asset ，替换全局变量 "Font "中索引的font asset 。
