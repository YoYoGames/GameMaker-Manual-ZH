字体_存在

这个函数返回具有指定索引的font 是否存在。你可以检查从font 浏览器中定义的Asset 指数，或使用 font_add() 等函数添加的fonts 。

语法。

font_exists(ind);

争论类型 描述
ind Index of the font to check.

返回。

例子。

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

这将把活动的绘图font ，如果它存在的话，就设置为 fnt_Main 。

返回。字体
