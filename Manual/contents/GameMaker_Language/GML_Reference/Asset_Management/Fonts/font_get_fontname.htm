font_get_fontname

通过这个函数，你可以得到给定的font asset 的实际系统名称。这个函数返回一个字符串，而不是一个索引，返回的名称将取决于正在使用的font 。例如，你可能在font 浏览器中有一个名为"fnt_Main"的asset Asset ，而font 本身可能是Windows 系统font Arial。在这种情况下，该函数将返回"Arial"，因为这是该font 的系统名称。如果你需要出现在Asset 浏览器中的名称，使用 font_get_name().

上面描述的行为只适用于通过font 添加的assets IDE 。当用于在font （使用 font_add() ）加载的runtime ，该函数将返回该path 文件的完整font ，而不是其font 名称。

语法。

font_get_fontname(ind);

争论类型 描述
ind Index of the font to check.

返回。

String

例子。

fnt_Name = font_get_fontname(font0);

上述代码将获得一个font 资源的系统名称，并将其作为一个string ，存储在变量 "fnt_Name "中。

返回。字体
