font_get_uvs

font texture 该函数返回一个数组，其中包含texture 页面上的UV坐标，在数组中填充以下数值。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部

然后，这个值可以在其他绘图函数中使用，特别是在使用基元以及着色器函数时的一般绘图。

语法。

font_get_uvs(font);

争论类型 描述
font The index of the font to use.

返回。

例子。

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上面的代码将把给定背景的UV坐标存储在一个本地数组中，然后把这些值分配给实例变量。

返回。字体
