font_get_texture

这个函数为font texture 页面返回一个特殊的指针。然后这个值可以在其他绘图函数中使用，特别是在使用基元以及Shader函数时的一般绘图。你可以使用这里找到的不同的 texture_ 函数获得更多关于返回的texture 页面的信息。

语法。

font_get_texture(font);

争论 类型 描述
font 字体资产 The index of the font to use.

返回。

Texture

例子。

tex = font_get_texture(fnt_Main);

上面的代码将获得texture 指针，用于索引为 "fnt_Main "的font 。

返回。字体
