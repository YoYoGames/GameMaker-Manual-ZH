font_get_name

这个函数可以用来获取string ，当它作为一个font ，被添加到GameMaker asset 浏览器时，它的名字（作为一个Asset ）。请注意，这仅仅是一个 string 而不能用来直接引用font - 为此，你需要字体索引。但是，你可以使用这个string ，用返回的string ，加上 函数 asset_get_index().

语法。

font_get_name(ind);

争论类型 描述
ind Index of the font to check.

返回。

例子。

fnt_Name = font_get_name(font0);

上述代码将获得一个font 资源的名称，因为它出现在Asset 浏览器中，并将其作为string ，存储在变量 "fnt_Name "中。

返回。字体
