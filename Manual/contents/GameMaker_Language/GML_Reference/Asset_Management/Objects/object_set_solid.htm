object_set_solid

flag 通过这个函数，你可以将一个特定的object ，作为实体或非实体（关于 "实体 "flag 的更多信息，请参见 "对象编辑器"部分）。这意味着在改变了实体后创建的这个object 的所有实例将以这个新的状态创建，而已经在room 的实例可能不会受到影响。

请注意，这不是一个实例函数!flag 你可以使用 solid 变量来设置单个实例的实体object ，这样就有十个实例都被标记为实体，即使有关的flag 在IDE 中被设置为假，即使你使用这个函数将flag 的实体object 改为真，目前在room 中的所有实例将保持原样，只有在调用这个函数后创建的实例才会开始被标记为真。

语法。

object_set_solid( index, solid );

争论 类型 描述
index 对象资产 The index of the object to change.
solid 布尔型 The new solidity of the object (true=固体， false=非固体）。

返回。

N/A

例子。

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

上述代码检查flag "obj_Block "的实体object ，如果它是假的，那么它将被改为标记为真。

返回。物体
