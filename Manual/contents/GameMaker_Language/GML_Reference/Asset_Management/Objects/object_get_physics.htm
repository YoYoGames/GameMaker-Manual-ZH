对象_get_physics

这个函数将告诉你，你正在检查的object 是否被标记为 "物理学启用"--在这种情况下，它将返回 true ，或者没有--在这种情况下它将返回false 。

语法。

object_get_physics(obj);

争论类型 描述
obj The index of the object to check.

返回。

例子。

if object_get_physics(object_index)
{
    phy_active = true;
}

上面的代码将检查运行它的实例，看看它所创建的object 是否启用了物理功能，如果是的话，它将激活实例的物理模拟。

返回。物体
