对象_存在

这个函数返回指定的值是否是一个object asset (true)或不是(false)。请注意，该函数检查object 浏览器中是否存在一个Asset ，而不是实际存在于游戏中的room 。对于这一点，你应该使用函数 instance_exists().

语法。

object_exists(obj);

争论类型 描述
obj The index of the object to check.

返回。

例子。

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

上面的代码从一个通过网络收到的buffer ，读取数据。由于它包含一个object ID（用于催生实例），我们首先需要确保这个值不是无效的，并且object 实际存在。为此，我们在一个条件中调用 object_exists() ，如果是真的，我们就在收到的位置创建一个该object 的实例。

返回。物体
