对象_获取_斯普利特

这个函数将告诉你，你要检查的object 是否有一个sprite ，如果有，它将返回该sprite 的索引，如果没有，则返回-1。请注意，这不是一个实例函数!你可以有一个没有object 的sprite ，而同一个object 的实例可以有一个，反之亦然，或者它们甚至可以有不同的sprites 。你可以使用实例变量来设置一个单独的实例sprite 。 sprite_index实例变量。

语法。

object_get_sprite(obj);

争论类型 描述
obj The index of the object to check

返回。

Sprite Asset

例子。

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

上面的例子将检查实例的 sprite_index 与实例的 object_index 的sprite 。如果它们不一样，那么它将为实例分配与sprite 索引相同的object 。

返回。物体
