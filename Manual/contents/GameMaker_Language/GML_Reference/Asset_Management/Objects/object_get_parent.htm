对象_获取母体

这个函数将得到已经分配给指定的object 的任何父类的object 索引，否则返回-100，表明object 没有分配给它的父类，如果被检查的object 不存在，则返回-1。关于父类的更多信息，请参见 "对象编辑器"部分。

语法。

object_get_parent(obj);

争论类型 描述
obj The index of the object to check

返回。

Object Asset

例子。

par = object_get_parent(object_index);

上面的代码将获得运行代码的实例的object 索引的父本，并将返回值存储在变量 "par "中。

返回。物体
