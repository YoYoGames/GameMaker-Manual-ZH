对象_获取掩码

这个函数将告诉你，你要检查的object 是否有一个掩码索引，如果有，那么它将返回该掩码的索引（它是一个sprite asset ），如果没有，则返回-1。请注意，这不是一个实例函数!你可以有一个没有掩码的object ，而同一个object 的实例可以有一个，反之亦然，或者它们甚至可以有不同的掩码。你可以使用 "实例 "变量来设置单个实例的掩码索引。 mask_index实例变量来设置单个实例的掩码索引。

语法。

object_get_mask(obj);

争论类型 描述
obj The index of the object to check

返回。

Sprite Asset

例子。

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

上面的例子将检查实例的掩码索引与实例的object_index的掩码。如果它们不一样，那么它将为实例分配与object index所具有的相同的掩码。

返回。物体
