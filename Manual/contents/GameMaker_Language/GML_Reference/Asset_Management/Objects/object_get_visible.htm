对象_获取可见性

这个函数将告诉你，你正在检查的object 是否被标记为 "可见"（运行其绘制事件）或不（不运行其绘制事件）。请注意，这不是一个实例函数!所以，你可以有一个可见的object 和一个不可见的同一object 的实例，反之亦然。你可以使用实例变量来设置单个实例的可见性。 visible实例变量来设置单个实例的可见性。

语法。

object_get_visible(obj);

争论类型 描述
obj The index of the object to check.

返回。

例子。

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

上述代码将检查正在运行的实例，看它是否可见，以及检查实例的object 索引，看它是否被标记为可见。如果该实例不可见，但object 索引被标记为开启，它将为该实例设置 "可见 "为真。

返回。物体
