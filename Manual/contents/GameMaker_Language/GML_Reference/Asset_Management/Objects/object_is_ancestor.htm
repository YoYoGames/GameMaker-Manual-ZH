对象_是股东

这个函数可以用来检查一个object ，看它是否是另一个object 的祖先。因此，两个参数都需要是object 的索引，而不是实例的id，而且第一个参数总是你想找出的object ，如果它是第二个参数的孩子，它总是想检查的object ，作为祖先（父母）。

语法。

object_is_ancestor(obj, par);

争论 类型 描述
obj 对象资产 The object that is being checked as the child.
par 对象资产 The object that is being checked as the ancestor (parent).

返回。

Boolean

例子。

if object_is_ancestor(object_index, obj_Enemy)
{
    instance_destroy();
}

上面的代码检查运行代码的实例是否是object "obj_Enemy "的子代，如果是的话，它就被销毁。

返回。物体
