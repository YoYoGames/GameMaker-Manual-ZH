object_index

这个只读变量返回实例所创建的object 的索引。这与object 的名称不同，后者是一个string ，可以用 object_get_name()，因为这个函数返回索引号，它是GameMaker在创建时分配给每个object 的唯一值。

语法。

object_index

返回。

Object Asset

例子。

obj_name = object_get_name(object_index);

上面的代码将使用 object_index ，以找到当前实例所创建的object 的名称。

返回。物体
