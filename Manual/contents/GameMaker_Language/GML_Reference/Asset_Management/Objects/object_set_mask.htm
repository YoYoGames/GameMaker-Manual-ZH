对象_set_mask

通过这个函数，你可以设置一个特定的object （关于掩码的更多信息，请参见TheObject Editor）的mask_index。这意味着这个object 的所有实例在掩码被改变后，将以这个新的掩码来创建，而已经在room 的实例可能不受影响。你可以将此设置为-1，以删除一个掩码sprite ，因此默认为sprite （如果没有选择object ，则没有掩码）的sprite 所定义的掩码。

请注意，这不是一个实例函数!你可以使用变量来设置单个实例的掩码索引，因此有十个实例的掩码都与它们创建的 不同。 mask_index变量来设置单个实例的掩码索引，因此有十个实例的掩码都与它们所创建的object 不同，即使你用这个函数改变了object 的掩码索引，目前在room 中的所有实例都将保持原样，只有在调用这个函数后创建的实例才会以新的掩码开始。

语法。

object_set_mask(index, spr);

争论 类型 描述
index 对象资产 The index of the object to change.
spr 雪碧资产 The new sprite to use as the object's mask.

返回。

N/A

例子。

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

上面的代码检查全局变量的值，然后改变object "obj_Platform "的掩码索引，如果它等于10。
