object_set_sprite

sprite 通过这个函数，你可以设置一个特定的object 的索引。这意味着在改变了sprite_index之后创建的这个object 的所有实例将以这个新的sprite_index创建，而已经在room 的实例可能不受影响。

请注意，这不是一个实例函数!你可以使用变量来设置单个实例的sprite 索引，这样就会有10个实例都有与它们创建的 不同的索引。 sprite_indexsprite 变量，因此有十个实例的object 与它们所创建的sprite 不同，即使你用这个函数改变了object 的索引，所有目前在room 的实例都将保持原样，只有在调用这个函数后创建的实例才会以新的sprite 开始。

语法。

object_set_sprite( index, spr );

争论 类型 描述
index 对象资产 The index of the object to change.
spr 雪碧资产 The sprite to assign to the object.

返回。

N/A

例子。

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

object 上面的代码将根据运行代码的实例当前所处的sprite "obj_Player"room 索引设置为不同的值。然后它创建一个 "obj_Player "的实例。

返回。物体
