事件_表演对象

这个函数的工作原理与 event_perform()除了这次你可以指定来自另一个object 的事件。这里有许多选项，可以完全模拟所有可能的事件，但请注意，这实际上只是执行了事件中的所有代码，游戏不会修改任何东西来使它自己手动触发，例如，如果你选择执行一个键盘按压事件，事件将被触发，但相关的键不会被识别为已按下。或者如果你执行一个警报事件，警报计数器不会被设置为-1，而是继续倒数。你可以在该函数的页面上找到该函数所需的可用常数的完整列表 event_perform().

注意用这个函数调用的事件中的动作适用于当前的实例，而不是给定的object 的实例。

语法。

event_perform_object(obj, type, numb);

争论 类型 描述
obj 对象资产 The object that should have its event triggered.
type 事件类型常数 The type of event to perform.
numb 实数或事件数常数 The specific event, if one is necessary (otherwise, just use 0).

返回。

N/A

例子。

event_perform_object(obj_Player, ev_keypress, ord("W"));

这将执行与当前实例中object "obj_Player "的键盘检查>W键相关的事件。

返回。生成Object 事件
