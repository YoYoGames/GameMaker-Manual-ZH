事件_继承的

这个函数将调用实例的父object 的当前事件。通常情况下，当一个实例有一个父object ，它会自动inherits 所有与父相同的事件，但是如果（例如）你的父object 有一个创建事件，而你给你的子object 添加了一个，子object 的所有实例将运行你添加的新的创建事件，而不是父object 中的那个。如果你需要同时使用同一类型的父object 事件和子object 事件，你应该使用这个函数，因为它将在继续执行子事件包含的其他代码或动作之前运行父object 事件。

语法。

event_inherited();

返回。

例子。

event_inherited();
switch (sprite_index)
{
    case spr_Enemy_1: dmg += 2; break;
    case spr_Enemy_4: dmg -= 1; break;
    case spr_Enemy_10: dmg +=10; break;
}

上面的代码调用了继承的父事件（其中我们初始化了变量 "dmg" ），然后继续修改 "dmg" 变量。如果没有为运行这段代码的实例指定父级，我们会得到一个 "未知变量 "的错误，因为 dmg 没有被定义。

返回。生成Object 事件
