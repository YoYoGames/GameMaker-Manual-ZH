事件_编号

这个只读变量返回当前被调用的事件的编号，这里的编号实际上是指事件的 "子事件"，即：对于步骤事件，事件编号可以是常数ev_step_normal、ev_step_begin或ev_step_end中的任意一个。关于特定子事件可用的常数的完整列表，请参见 event_perform()，如果你应该需要知道主事件本身，你应该检查 event_type.

语法。

event_number;

返回。

Event Number Constant

例子。

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

上述代码可以在script ，用于显示调试信息，告知你当前正在触发哪个事件。
