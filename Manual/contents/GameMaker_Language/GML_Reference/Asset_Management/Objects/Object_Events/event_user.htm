event_user

通过这个函数，你可以告诉实例运行被放置在16个用户定义的事件中的动作或代码。这些事件只能以这种方式被调用，或者使用 event_perform函数。

语法。

event_user(numb);

争论 类型 描述
numb 真实的 The number of User Event to call, between 0 and 15.

返回。

N/A

例子。

event_user(4);

这将调用用户定义的事件4。

返回。生成Object 事件
