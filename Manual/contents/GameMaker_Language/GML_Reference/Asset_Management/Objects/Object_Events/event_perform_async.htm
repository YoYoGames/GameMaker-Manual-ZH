event_perform_async

这个函数被用来执行GameMaker中提供的任何一个异步事件。你提供了Async事件常量（如下表所示）和一个DS地图，它将在 async_load 变量中被调用的Async事件中使用。

非异步事件可以用以下方式调用 event_perform().

请注意，第二个参数中指定的DS地图不需要手动销毁，因为它将在Async事件执行后被函数自动销毁。

语法。

event_perform_async(type, ds_map);

争论 类型 描述
type 异步事件类型常数 The type of event to perform (see the table above).
ds_map DS地图ID The DS map to use as async_load 在被调用的事件中。

返回。

N/A

例子。

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

上面的代码创建了一个DS地图，并用自定义条目填充它，以便在Async事件中读取。然后，它用新创建的地图作为 async_load ，执行Async Social事件。

返回。生成Object 事件
