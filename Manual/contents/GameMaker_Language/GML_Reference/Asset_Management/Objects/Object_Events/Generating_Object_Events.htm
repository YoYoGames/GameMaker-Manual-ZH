产生Object 事件

GameMaker完全是事件驱动的，所有的行动都是由某些事件发生的结果，这些事件是按照特定的顺序或由游戏中的特定事件触发的。有许多不同的事件，每一个都有特定的任务或时间，其中一些甚至被分成独立的 "子 "事件。通常情况下，这些事件在GameMaker检测到游戏中的触发器时运行，或者每一个游戏时间（如步骤和绘图事件），但你也可以在一段代码中对当前room 中的一个实例应用一个事件。

存在以下处理生成事件的函数。

  • event_inherited
  • event_perform
  • 事件_执行_async
  • event_perform_object
  • event_user

还有一些只读变量，可以与上述函数一起使用，这对调试特别有用。

  • event_action
  • event_number
  • event_object
  • event_type

有关事件的更多信息，请参见 "对象事件"部分。
