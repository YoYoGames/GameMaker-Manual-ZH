事件类型

这个只读变量返回当前正在执行的事件的类型，它可以是以下常量之一。

  • 创作（ev_create
  • 销毁
  • 阶梯
  • 呼叫中心
  • 键盘（Ev_keyboard
  • 钥匙扣
  • 钥匙释放
  • 鼠标
  • ev_collision
  • 其他
  • 姿态
  • 绘制


关于与不同事件相关的常量的完整列表，见 event_perform()，如果你应该需要事件编号（即：子事件），你应该检查 event_number.

语法。

event_type;

返回。

Event Type Constant

例子。

show_debug_message("Current Event = " + string(event_type));

上面的代码将显示一个调试信息，其中有当前正在执行的事件类型。
