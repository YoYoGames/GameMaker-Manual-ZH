对象_set_visible

flag 通过这个函数，你可以将一个特定的object ，作为可见或不可见（关于 "可见 "flag 的更多信息，请参见 "对象编辑器"部分）。这意味着在改变了visible之后创建的这个object 的所有实例将以这个新的状态创建，而已经在room 的实例可能不会受到影响。

请注意，这不是一个实例函数!你可以使用变量来设置单个实例的可见flag ，从而使十个实例都被标记为可见，即使有关的 ，在 中被设置为false，即使你改变了可见 。 visible变量，从而使十个实例都被标记为可见，即使有关的object 在flag 中的IDE 被设置为假，即使你用这个函数将flag 的可见object 改为真，目前在room 中的所有实例都将保持原样，只有在调用这个函数后创建的实例才会开始被标记为真可见。

语法。

object_set_visible( index, vis );

争论 类型 描述
index 对象资产 The index of the object to change.
vis 布尔型 The new visibility of the object (true=可见， false=不可见）。

返回。

N/A

例子。

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

上述代码检查flag "obj_Block "的可见性object ，如果它是假的，那么它将被改为标记为真。
