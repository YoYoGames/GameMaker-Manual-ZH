object_get_name

这个函数将返回指定的object 的名称，即一个字符串。这个名字是在GameMaker主窗口的object 浏览器中为Asset 指定的那个名字。请注意，这只是一个字符串 而不能用来直接引用object - 为此你需要对象的索引。然而，你可以使用这个string ，使用返回的string 和函数来获得对象索引。 asset_get_index().

语法。

object_get_name(obj);

争论类型 描述
obj The index of the object to check.

返回。

例子。

str = object_get_name(object_index);

上述代码将获得运行该代码的实例的object 索引的名称，并将返回值存储在变量 "str "中。

返回。物体
