时间线_最大值_时刻

这个函数将返回timeline 执行其最后动作的时刻值。因此，如果你有一个timeline ，有8个不同的动作，相隔20个时刻，这个函数将返回该timeline 中最后一个动作的时刻值，也就是160。这个函数是测试一个timeline 在运行时是否已经过了最后的活动时刻的好方法，因为无论是否有进一步的行动，timeline 的位置在游戏的每一步都会继续递增。

语法。

timeline_max_moment(ind);

争论类型 描述
ind The index of the timeline to get the last moment of.

返回。

例子。

if timeline_position > timeline_max_moment(timeline_index)
{
    timeline_running = false;
}

上述代码将对照最大活动力矩检查当前timeline 的位置，如果大于此值，timeline 将被停止。

返回。时间轴
