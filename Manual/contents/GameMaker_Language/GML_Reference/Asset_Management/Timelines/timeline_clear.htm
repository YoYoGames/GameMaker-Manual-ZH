时间线_清空

通过这个功能，你可以清除一个特定时间线的 "时刻"，删除该时间线的所有代码和动作，使其成为空白。

语法。

timeline_clear(ind);

争论 类型 描述
ind 时间轴资产 The index of the timeline to clear.

返回。

N/A

例子。

if timeline_position > 200
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

上述代码将清除由变量 "global.tl "索引的指定时间线中的所有时刻，当一个特定时刻已经过去。

返回。时间轴
