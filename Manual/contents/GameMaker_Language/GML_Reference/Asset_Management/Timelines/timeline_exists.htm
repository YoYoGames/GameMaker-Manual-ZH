时间线_存在

通过这个函数，你可以检查并查看一个时间线是否存在（返回 true ）或不存在（返回 false ）。这在使用函数动态创建Timelines 时特别有用。 timeline_add()函数动态创建时间线时特别有用，但你应该注意，如果你搜索一个未初始化的变量（否则将被分配给时间线的索引），就会出现错误。

语法。

timeline_exists(ind);

争论类型 描述
ind The index of the time line to check for.

返回。

例子。

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

上述代码检查一个时间线是否存在并被索引在全局变量 "tl "中，如果是，它就会删除该时间线。

返回。时间轴
