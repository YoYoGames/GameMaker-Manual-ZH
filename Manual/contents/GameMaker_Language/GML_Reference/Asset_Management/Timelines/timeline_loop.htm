Timeline_loop

这个变量将返回时间线是否在循环（真）或不循环（假）。你可以改变这个变量，switch 循环的开启或关闭，并且它与负的时间线速度一起工作（如果时间线的位置低于0，它将在最后定义的时刻重新开始）。

语法。

timeline_loop;

返回。

例子。

if !timeline_loop
{
    timeline_loop = true;
}

上面的代码检查当前分配给一个实例的时间线是否会loop ，如果不会，则将其设置为true。

返回。时间轴
