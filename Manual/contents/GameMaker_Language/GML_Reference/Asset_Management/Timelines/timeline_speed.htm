时间线_速度

通常情况下，在每一步中，时间线中的位置会增加1，但是你可以通过将这个变量设置为不同的值来改变这个数量。你可以使用实数（例如0.5，或2.4），如果数值大于1，在同一时间步骤中可以发生几个时刻（它们都将按照为时间线定义的相同顺序执行，所以没有动作会被跳过）。

语法。

timeline_speed;

返回。

(single precision floating point value)

例子。

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

上述代码将检查并查看实例是否正在运行其相关的时间线，如果没有，它将启动时间线以半速运行。

返回。时间轴
