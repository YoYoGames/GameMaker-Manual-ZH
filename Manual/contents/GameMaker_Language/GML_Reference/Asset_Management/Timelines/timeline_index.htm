timeline_index

这个变量持有当前与实例相关的timeline 的索引。你可以将其设置为一个特定的时间线来使用该时间线，或者将其设置为-1来停止使用该实例的时间线（如果没有为该实例定义时间线，也会返回-1）。请注意，这并不能启动时间线--为此，请使用变量 timeline_running.

语法。

timeline_index

返回。

Timeline Asset

例子。

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

上面的代码将给运行代码的实例分配一个在变量 "global.tl "中索引的时间线，如果该时间线存在。

返回。时间轴
