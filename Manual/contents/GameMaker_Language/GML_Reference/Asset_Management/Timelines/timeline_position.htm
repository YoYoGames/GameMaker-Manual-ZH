时间线_位置

这个变量保持时间线目前所处的位置（时刻）。你可以改变这个值来跳过时间线的某些部分，或者重复某些部分或从头开始时间线。

语法。

timeline_position;

返回。

(single precision floating point value)

例子。

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

上面的代码将检查实例是否正在运行时间线，如果没有，那么它将重置指定的时间线，在第一时间启动，然后启动它。

返回。时间轴
