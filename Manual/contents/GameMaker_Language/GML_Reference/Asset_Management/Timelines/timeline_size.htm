时间线_大小

通过这个函数，你可以得到一个timeline （一个 "活动 "的时刻是一个有代码或GML Visual添加到它的时刻）的活动时刻总数。这在创建动态timelines ，使用 timeline_moment_add_script()和 timeline_moment_clear()函数来创建动态的timeline ，因为你可以检查一个给定的 是否有正确数量的活动时刻，或者根本就没有。

语法。

timeline_size(ind);

争论类型 描述
ind The index of the timeline get the information from.

返回。

例子。

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

上述代码检查给定的timeline 大小，如果它返回0（即：timeline 没有活动时刻），它将添加一个script 函数，在全局变量 "tl "中索引的timeline 的随机时刻使用。

返回。时间轴
