Timeline_get_name

这个函数可以用来获取一个时间线的名称，string 。如果时间线是用 timeline_add()函数动态创建的，返回的名称将具有 格式"__newtimelineN"，其中"N"是时间线的编号（从0开始）。请注意，这只是一个string ，不能用来直接引用timeline 。 为此，你需要时间线的索引。然而，你可以使用这个string ，用返回的string 和函数来获得时间线的索引asset_get_index().

语法。

timeline_get_name(ind);

争论类型 描述
ind The index of the time line to check the name of.

返回。

例子。

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

上述代码获取当前分配的timeline ，并将其输出到控制台。

返回。时间轴
