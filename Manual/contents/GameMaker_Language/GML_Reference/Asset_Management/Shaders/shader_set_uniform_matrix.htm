shader_set_uniform_matrix

通过这个函数，你可以将一个shader 常数的值（或多个值）设置为当前的变换矩阵（如使用矩阵函数设置的）。你必须事先用函数获得该常数的 "句柄"。 shader_get_uniform().

注意：所有的制服必须在调用函数后设置 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_matrix(handle);

争论 类型 描述
handle 着色器统一处理 The handle of the shader constant to set.

返回。

N/A

例子。

shader_set(shader_Glass);
shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

上述代码将获得shader 常数 "u_vMatrix "的句柄，然后将该常数设置为当前的变换矩阵。

返回。着色器
