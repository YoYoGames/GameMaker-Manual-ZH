着色器_设置

通过这个函数，你可以将绘图目标设置为给定的shader ，所有进一步的绘图都将使用该目标进行。你可以用函数结束shader 的使用 shader_reset().

语法。

shader_set(shader);

争论类型 描述
shader The index of the shader to use.

返回。

例子。

shader_set(shader_Glass);
draw_self();
shader_reset();

上面的代码将设置一个用于绘图的shader ，然后用它来绘制当前用于实例的sprite 。

返回。着色器
