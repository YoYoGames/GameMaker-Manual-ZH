shader_set_uniform_i

通过这个函数，你可以设置一个shader 常数的值（或数值）。你必须事先用函数获得该常量的 "句柄"，并且你必须知道它是什么类型的常量，以便将正确的数值传递给它。 shader_get_uniform()的 "句柄"，而且你必须知道它是什么类型的常量，以便向它传递正确数量的整数值，即：如果你有一个 vec2 ，你将需要向该函数传递两个值。

注意：所有的制服必须在调用函数后设置 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

争论 类型 描述
handle 着色器统一处理 The handle of the shader constant to set.
value1 ... value4 真实的 The integer value (or values) to set the shader constant to.

返回。

N/A

例子。

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

上面的代码将获得shader 常量 "u_vParams "的句柄（a vec3 ），然后将该常量设置为给定的整数值。

返回。着色器
