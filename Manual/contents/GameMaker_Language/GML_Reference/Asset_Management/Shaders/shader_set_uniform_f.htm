shader_set_uniform_f

通过这个函数，你可以设置一个shader 常数的值（或数值）。你必须事先用函数获得该常量的 "句柄"，并且你必须知道它是什么类型的常量，以便向它传递正确的浮点数。 shader_get_uniform()的 "句柄"，并且你必须知道它是什么类型的常数，以便向它传递正确数量的浮点值，即：如果你有一个 vec2 ，你将需要向该函数传递两个值。

上述规则的一个例外是在处理32位颜色值的时候。你可以将最多8个32位的颜色值传入shader ，它们将自动为你转换为0-1的vec4值。然而，为此你必须使用shader 常量名称 in_Colour ，用于单个颜色，或 in_Colour0 ， in_Colour1 ，等等...直至 in_Colour7 ，用于多个输入值。下面的例子显示了在shader 本身的情况。

    //Single colour
    attribute vec2 in_Position;
    attribute vec4 in_Colour;
    attribute vec2 in_TextureCoord;

    //Multiple colours
    attribute vec2 in_Position;
    attribute vec4 in_Colour0;
    attribute vec4 in_Colour1;
    attribute vec2 in_TextureCoord;

还要注意的是，如果你想使用4字节的颜色格式做压缩法线等，那么你也必须使用上述相同的 "in_Colour "格式。

注意：所有的制服必须在调用函数后设置 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);

争论 类型 描述
handle 着色器统一处理 The handle of the shader constant to set.
value1 ... value4 真实的 The floating point value (or values) to set the shader constant to.

返回。

N/A

例子。

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

上面的代码将获得shader 常量 "u_vParams "的句柄（一个 vec3 ），然后将该常量设置为给定的值。

返回。着色器
