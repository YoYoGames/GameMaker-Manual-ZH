shader_enable_corner_id

有了这个功能，你可以为所有正在使用的shaders 设置一个全局状态，当启用时，shader 从输入的颜色值中 "窃取 "2个比特。第一个是来自红色颜色值的低位，第二个是来自蓝色颜色值的低位。 第二位是蓝色颜色值的低位。这些值可以在shader ，以找出你正在处理的vertex （即：哪个角落）。

语法。

shader_enable_corner_id(enable);

争论类型 描述
enable Enable (true) 或禁用 (false) 该功能。

返回。

例子。

shader_enable_corner_id(true);

上述代码将使所有shaders ，为角落的id使用颜色位。

返回。着色器
