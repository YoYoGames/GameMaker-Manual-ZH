shader_get_sampler_index

由于你不能在shader 本身改变一个shader 采样器的值，你必须在调用shader 之前用一个可用的统一设置函数来设置它。然而，要做到这一点，你必须首先调用这个函数，以获得你要设置的shader 采样器的 "句柄"。

注意尽管一个shader 是由两个独立的程序组成的（vertex 和 fragment），这个函数不会区分这两个程序，并将返回其中一个程序的shader 的句柄。

语法。

shader_get_sampler_index(shader, uniform);

争论 类型 描述
shader 着色器资产 The index of the shader to use.
uniform 字符串 The shader sampler to get the handle of (a string).

返回。

Shader Sampler Handle

例子。

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

上面的代码将在shader 中获取索引为 "GlassShader "的采样器的句柄，然后将该shader 常量设置为给定的sprite texture 。

返回。着色器
