shader_is_compiled

这个函数将在运行时检查一个shader ，以确保它已经被成功编译。如果它已经被编译，那么该函数将返回 true ，否则它将返回 false 。这个函数应该在游戏开始时使用，以确保运行你的游戏的平台已经成功地编译了任何使用的shaders （特别是在Windows ，有些计算机可能使用DX9和Shader 2.0级，而不是使用shader 3.0级的后期版本）。

如果你的shader 还没有被编译，而你调用了 shader_set()游戏就会崩溃，所以当你使用除简单的GLSL ESshaders 之外的任何东西时，都值得进行某种检查。

语法。

shader_is_compiled(shader);

争论类型 描述
shader The shader to check.

返回。

Boolean

例子。

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

上述代码将设置一个全局变量为 false ，然后如果被检查的两个shaders 都正确编译了，它将被设置为 true 。

返回。着色器
