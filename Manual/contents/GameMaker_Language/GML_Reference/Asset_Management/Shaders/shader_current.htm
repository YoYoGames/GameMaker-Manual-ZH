shader_current

该函数将返回当前用于渲染的shader 的索引ID值，如果没有使用shader ，则返回-1。

语法。

shader_current();

返回。

Shader Asset or -1 (if no shader is assigned)

例子。

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

上面的代码将检查当前的shader ，如果它返回-1（没有使用shader ），那么就设置一个shader 。

返回。着色器
