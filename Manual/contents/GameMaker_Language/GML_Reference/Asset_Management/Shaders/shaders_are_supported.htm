着色器是否支持

这个函数将做一个检查，看看所选择的目标平台是否支持shaders ，如果支持则返回 true ，如果不支持则返回 false 。值得注意的是，在Android上，如果项目没有定义任何shader 资源，那么该函数将总是返回 false，而不管设备是否支持shaders 。

语法。

shaders_are_supported();

返回。

例子。

global.GFX = false;
if shaders_are_supported()
{
    if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)
    {
        global.GFX = true;
    }
}

上述代码将设置一个全局变量为 false ，然后如果平台支持shaders ，并且被检查的两个shaders 都已正确编译，它将被设置为 true 。

返回。着色器
