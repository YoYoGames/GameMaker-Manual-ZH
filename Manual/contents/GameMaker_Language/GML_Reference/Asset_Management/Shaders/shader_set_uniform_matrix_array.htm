shader_set_uniform_matrix_array

通过这个函数，你可以设置一个shader 常数来保存一个矩阵值的阵列。你必须事先用函数获得该常数的 "句柄"。 shader_get_uniform()得到常数的 "句柄"，并且你必须事先将数组初始化为一个浮点值数组，其中每个连续的16个浮点是一个4x4矩阵。

注意：所有的制服必须在调用函数后设置 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_matrix_array(handle, array);

争论 类型 描述
handle 着色器统一处理 The handle of the shader constant to set.
array 阵列 A previously initialised array of floating point values.

返回。

N/A

例子。

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

上面的代码将获得shader 常量 "cMatrixArray "的句柄，然后将该常量设置为给定的数组。

返回。着色器
