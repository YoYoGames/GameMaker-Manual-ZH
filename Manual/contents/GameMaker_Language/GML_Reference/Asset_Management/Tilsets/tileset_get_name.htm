tileset_get_name

这个函数将返回指定的tile set asset 的名称，即一个字符串。这个名字是在GameMaker主窗口的tile set 浏览器中为Asset 指定的名字。请注意，这只是一个string ，不能用来直接引用tile set - 为此你需要瓷砖集的索引。然而，你可以使用这个string ，使用返回的string 和函数来获得瓷砖集的索引。 asset_get_index().

语法。

tileset_get_name(index);

争论 类型 描述
index 瓷砖套装资产 The index of the tile set to get the name of.

返回

String

例子。

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

上述代码将获得给定层的tile set 索引的名称，将返回的string 储存在变量 tileset_name 。

返回。瓷砖套装
