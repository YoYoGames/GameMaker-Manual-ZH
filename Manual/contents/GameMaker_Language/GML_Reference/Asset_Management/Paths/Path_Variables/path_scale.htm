路径_刻度

这个值可以用来获取或设置当前分配给实例的path 的比例（由函数 path_start() 设置），默认值为1。这是一个标度值，所以 1是1:1的比例，而设置为2，例如，将是两倍的比例，设置为0.5将是减半的比例。

语法。

path_scale;

返回。

例子。

path_scale = 1 + random(2);

上面的代码将把当前路径的比例设置为1（其原始大小）和3（其原始大小的3倍）之间的随机大小。

返回。路径变量
