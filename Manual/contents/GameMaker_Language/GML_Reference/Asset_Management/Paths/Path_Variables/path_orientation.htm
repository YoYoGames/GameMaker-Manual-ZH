path_orientation

这个变量保存了在调用函数 path_start() 时分配给该实例的path 的当前方向。当一个path 被创建时，它的方向是默认的 0度，但你可以使用这个变量将这个值设置为你想要的任何东西。记住，在GameMaker中（除非你使用物理学），角度是逆时针计算的，所以把path 的方向设置为 为90°将使path 向左旋转。

语法。

path_orientation

返回。

例子。

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

上面的代码复制了一个随机的、预先制作好的path asset 到变量 "mypath "中。然后启动这个新的path ，并将其旋转90°。
