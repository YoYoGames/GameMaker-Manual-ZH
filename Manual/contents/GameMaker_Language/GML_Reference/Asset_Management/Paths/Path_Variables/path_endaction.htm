路径_行动

这个变量可以用来获取或改变一个实例的反应，当它到达当前path 。通常情况下，你会在启动path 的时候设置这个变量。 path_start()但你可能希望根据你游戏中的任何事件来改变这一行为。可用的值用以下常数表示。

路径终点行动常数
恒定 描述
path_action_stop 停止路径
path_action_restart 从起始位置继续，如果path ，则跳到起始位置。
path_action_continue 从当前位置再次启动path
path_action_reverse 将path （反向运行path ）的速度逆转。

语法。

path_endaction;

返回。

Path End Action Constant

例子。

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

上述代码将检查path 的结束动作，如果它被设置为停止，那么结束动作将被改为反向。

返回。路径变量
