path_positionprevious

这个变量可以用来获取或设置一个实例在上一步中沿其当前path 的位置，它是一个介于0和1之间的标准化值，即：0是path 的起始位置，1则是终点位置。它类似于 xprevious和 yprevious变量的工作方式相似，只是它是专门针对paths 。它对一些事情很有用，比如在有东西挡住的情况下暂时停止一个path 跟随者（见下面的示例代码）。

语法。

path_positionprevious

返回。

例子。

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(xx, yy, 16, obj_Player, false,true))
{
    path_position = path_positionprevious;
}

上述代码检查实例前面的一个区域是否与object "obj_Player "发生碰撞，如果有的话，它将实例设置回它在当前path ，由函数 path_start() 。

返回。路径变量
