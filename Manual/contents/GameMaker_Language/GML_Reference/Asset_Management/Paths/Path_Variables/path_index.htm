路径_索引

GameMaker中的所有资源都有一个唯一的识别号， path_index 是一个只读变量，它保存了使用 path_start() 函数分配给一个实例的特定path asset 的索引值。如果该实例没有分配path ，该变量将被设置为-1。

语法。

path_index;

返回。

Path Asset

例子。

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

上面的代码检查是否有一个path ，如果没有，就启动一个新的path （同时将其分配给 path_index ）。

返回。路径变量
