path_get_name

这个函数将返回作为path ，被引用的string 的名称。这个名称是在编辑器中用来定义path ，或者（如果path 是通过代码函数创建的），它将返回一个string ，格式为" _newpathXX "，其中" "是生成的 的编号，从0开始递增。 _newpathXX"，其中 "XX"是生成的path 的编号，从0开始，每创建一个新的path 就递增一次。请注意，这只是一个 string 而不能用来直接引用path - 为此你需要路径索引ID。但是，你可以使用这个string ，用返回的string ，再加上函数的作用，得到路径索引ID。 和函数 asset_get_index().

语法。

path_get_name(index);

争论类型 描述
index The index of the path to check.

返回。

例子。

path_name = path_get_name(path_array[0]);

这将把 "path_name "设置为给定数组中索引在0位置的path 的名称。

返回。路径
