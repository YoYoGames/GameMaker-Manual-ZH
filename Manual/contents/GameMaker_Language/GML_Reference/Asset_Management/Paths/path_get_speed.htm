path_get_speed

这个函数返回path 。位置应该是一个介于0和1之间的值（你可以使用 path_position，例如，如果你需要实例的当前位置），0是起始位置，1是结束位置。返回的值将是实际设置 path_speed 的系数。例如，在一个只有0和1两个点的path ，其中0点的速度系数是100，1点的速度系数是0，函数 path_get_speed(path, 0.5) ，将返回50。

语法。

path_get_speed(ind, pos);

争论 类型 描述
index 路径资产 The index of the path to check.
pos 真实的 How far through the path to check. Between 0 (start) and 1 (end), can be decimal.

返回。

Real

例子。

speed = path_get_speed(pth_Patrol, 0.5) / 20;

这将把调用实例的速度设置为正好在给定的path ，除以20的点的速度系数。因此，如果找到的速度系数是默认的100，它将把调用实例的速度设置为5。

返回。路径
