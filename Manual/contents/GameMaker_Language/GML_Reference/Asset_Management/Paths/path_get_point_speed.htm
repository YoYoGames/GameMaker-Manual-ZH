path_get_point_speed

通过这个函数，你可以得到点的速度（在路径编辑器中定义的，或者当你用动态添加path 点的时候 path_add_point())以百分比表示。因此，如果你在path 编辑器中把一个path 点设置为50，这个函数在使用时将返回50。

语法。

path_get_point_speed(index, n);

争论 类型 描述
index 路径资产 The index of the path to check.
n 真实的 The point number to check.

返回。

Real

例子。

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

path 上述代码检查由 "pth_Patrol "索引的path 的 "0 "点，看其速度是否不等于100。如果不是，则将该点的速度设置为100。

返回。路径
