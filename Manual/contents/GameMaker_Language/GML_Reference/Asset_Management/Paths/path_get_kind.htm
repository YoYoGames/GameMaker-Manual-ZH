路径

路径可以是平滑的，也可以是笔直的（平滑的path ，围绕定义点计算一个弯曲的path ，而笔直的path ，只是从一个点到另一个点的直线）。这个 函数可以用来找出给定的path 是平滑的(true)还是不平滑的(false)。

语法。

path_get_kind(index);

争论类型 描述
index The index of the path to check.

返回。

例子。

path_kind = path_get_kind(pth_Patrol);

这将把 "path_kind "变量设置为 true 或 false ，取决于给定的path 索引是哪种类型的path 。

返回。路径
