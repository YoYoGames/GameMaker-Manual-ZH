路径_获取精度

这个函数返回给定的path 已经被 "平滑 "的精度，它将是一个从1到8的整数值。尽管你可以为一个直线path 得到（和设置）这个值，但它不会影响一个实例如何使用path ，因为它只在path 类型被设置为 "平滑 "时才相关。

语法。

path_get_precision(index);

争论类型 描述
index The index of the path to check.

返回。

例子。

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

上述代码检查 "pth_Patrol "中索引的path ，看它是什么类型的，如果它是平滑的，那么它就检查平滑的精确程度。如果它小于8，那么它就被设置为8的值。
