path_get_x

通过这个函数，你可以得到任何给定的path 上某一位置的x坐标。0是path 的开始，1是path 的结束，在这两者之间的任何位置都相当于通过path 。这不需要是path 的定义点，它可以是它的任何位置。

语法。

path_get_x(ind, pos);

争论 类型 描述
index 路径资产 The index of the path to check.
pos 真实的 How far through the path to check. Between 0 (start) and 1 (end).

返回。

Real

例子。

x = path_get_x(pth_WallWalk, 0.5);

这将把调用实例的x设置为恰好在给定的path 的一半处的点的x坐标。

返回。路径
