path_get_y

通过这个函数，你可以得到任何给定的path 上某一位置的y坐标。0是path 的起点，1是path 的终点，而在这两者之间的任何位置都相当于沿着path 的长度的归一化距离。这个值不需要是path 的定义点，它可以是它的任何位置。

语法。

path_get_y(ind, pos);

争论 类型 描述
index 路径资产 The index of the path to check.
pos 真实的 How far through the path to check. Between 0 (start) and 1 (end).

返回。

Real

例子。

y = path_get_y(pth_WallWalk, 0.5);

这将把调用实例的y设置为正好在给定的path 中的点的y坐标。

返回。路径
