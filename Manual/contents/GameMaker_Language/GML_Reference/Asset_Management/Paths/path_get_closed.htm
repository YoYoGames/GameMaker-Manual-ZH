路径_封闭

这个函数可以用来返回path 是否被标记为封闭（真）或开放（假），即path loops 或是否有确定的开始和结束。

语法。

path_get_closed(index);

争论 类型 描述
index 路径资产 The index of the path to check.

返回。

Boolean

例子。

state = path_get_closed(pth_Patrol);

这将根据 "pth_Patrol "中索引的path 的关闭状态，将 "state "设置为 true 或 false 。

返回。路径
