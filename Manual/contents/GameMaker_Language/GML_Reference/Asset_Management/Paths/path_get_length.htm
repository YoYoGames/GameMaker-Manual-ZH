路径的长度（path_get_length

你可以用这个函数得到一个path 的精确长度，单位是像素。这不是一个从点到点的近似长度，而是一个沿着path 的形状的精确长度，即使path 是平滑的，有很高的弯曲精度。 高的弯曲精度。

语法。

path_get_length(index);

争论类型 描述
index The index of the path to measure.

返回。

例子。

path_len = path_get_length(pth_AI);

这将把 "path_len "设置为 "pth_AI "中索引的path 的总长度，单位为像素。

返回。路径
