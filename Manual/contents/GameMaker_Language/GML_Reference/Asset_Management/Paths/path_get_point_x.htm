path_get_point_x

这个函数将返回你为你索引的room 所输入的点的x位置（以path 坐标）。如果该点超出了path 的范围（即：一个path 有8个点，而你要求的是第10点的X位置），那么将返回一个0的值。

语法。

path_get_point_x(index, n);

争论 类型 描述
index 路径资产 The index of the path to check.
n 真实的 The point number to check.

返回。

Real

例子。

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

上面的代码将检查一个实例是否在path 。如果是，它将在path ，然后随机选择一个点，将实例移动到该点。

返回。路径
