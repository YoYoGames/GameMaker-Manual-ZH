路径_翻转

这个函数将所有的path ，并将它们沿横轴翻转。这个函数改变了实际的path 资产，因此将永久地影响游戏中所有实例对path 的使用方式，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用类似 path_duplicate() 的函数先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_flip(index);

争论类型 描述
index The index of the path to flip.

返回。

例子。

path_flip(mypath);

这将沿横轴翻转变量 "mypath "中索引的path 。
