路径_指定

通过这个功能，你可以把path 数据从一个path 复制到另一个。被复制到的 path 将首先被清除（如果它有任何path 点），并被被复制的path 完全覆盖。 复制过来的。在这个过程中，任何一个path 都不会被删除，结果是两个paths ，有两个不同的索引，但形式和点都完全一样。一般来说，你想在一个用 path_add() 创建的path 上使用这个函数。 因为如果你在一个path asset ，它将永久地影响游戏中所有实例的path ，从使用该函数的那一刻起直到游戏结束。

语法。

path_assign(index, path);

争论类型 描述
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

返回。

例子。

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

上述代码将在变量 "mypath "中创建一个新的、空的path ，然后从三个可用的path 资源中的一个复制过来path 数据。
