path_clear_points

通过这个函数，你可以从一个path 中删除所有的点定义，有效地形成一个 "空 "path 。这并没有删除path ，为此你应该使用 path_delete()但需要注意的是，这个函数改变了实际的path 资产，因此将永久影响游戏中所有实例对path 的使用，从使用这个函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，首先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_clear_points(index);

争论类型 描述
index The index of the path to clear.

返回。

例子。

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

上述代码检查path 上是否有变量 "mypath "中索引的点，如果有，则清除path 。
