路径_插入点

通过这个函数，你可以在一个path 中插入一个新的点（这个path 可以在path 编辑器中创建，或者通过代码使用 path_add()).该点将被添加到path ，在函数中指定的点 "n "之前。

语法。

path_insert_point(index, n, x, y, speed);

争论 类型 描述
index 路径资产 The index of the path to insert the point into.
n 真实的 The defining point to insert the new point BEFORE.
x 真实的 The x coordinate (relative to the path) of the new point.
y 真实的 The y coordinate (relative to the path) of the new point.
speed 真实的 The speed factor of the point.

返回。

N/A

例子。

path_insert_point(mypath, 0, 50, 50, 100);

这将在变量 "mypath "中索引的path 的最开始插入一个点，在（50,50），速度系数为100%。
