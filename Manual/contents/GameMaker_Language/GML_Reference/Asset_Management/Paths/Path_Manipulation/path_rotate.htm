路径_旋转

你可以使用这个函数将一个给定的path 围绕其中心旋转。请记住，在GameMaker中（除非你使用物理学），角度是逆时针计算的，所以将path 旋转90度会将path 向左旋转。这个函数改变了实际的path 资源，因此将永久地影响游戏中所有实例对path 的使用，从使用这个函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

注意：一旦一个path 被分配给一个实例，使用 path_start()函数分配给一个实例，你也可以使用 path_orientation 变量来改变path 的旋转。

语法。

path_rotate(index, angle);

争论 类型 描述
index 路径资产 The index of the path to flip.
angle 真实的 The angle of rotation degrees.

返回。

N/A

例子。

path_rotate(pth_Patrol, 90);

这将使给定的东西逆时针旋转90度。
