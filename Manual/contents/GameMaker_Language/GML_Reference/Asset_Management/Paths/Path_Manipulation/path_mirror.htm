路径_mirror

这个函数将所有的path ，并将它们沿纵轴进行镜像。这个函数改变了实际的path 资产，因此将永久地影响游戏中所有实例对path 的使用，从使用该函数的那一刻起直到游戏结束。 被使用，直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，首先创建一个path 的副本，然后在这个副本上调用这个函数 复制的asset （不要忘了在不再需要asset 的时候调用 path_delete() ）。

语法。

path_mirror(index);

争论类型 描述
index The index of the path to mirror.

返回。

例子。

path_mirror(mypath);

这将反映出path ，沿着纵轴在变量 "mypath "中的索引。
