path_set_precision

通过这个函数，你可以改变path 的 "平滑度"。这个值必须在1到8之间，低值会在点与点之间产生更直的边缘和更清晰的曲线，而高值会使点变圆，使path 更加 "曲折"。请注意，如果在path 编辑器中没有将path 设置为平滑，或者使用函数 ，这个函数将没有可见的效果。 path_set_kind().这个函数改变了实际的path 资产，因此将永久地影响游戏中所有实例对path 的使用方式，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用像 path_duplicate() 这样的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_set_precision(index, prec);

争论 类型 描述
index 路径资产 The index of the path to change.
prec 真实的 The precision of the path. Must be an integer between 1 and 8.

返回。

N/A

例子。

path_set_precision(pth_Patrol, 2);

这将把给定路径的精度设置为2。
