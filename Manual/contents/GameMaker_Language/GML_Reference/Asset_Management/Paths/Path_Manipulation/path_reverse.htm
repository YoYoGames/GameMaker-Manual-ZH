path_reverse

通过这个功能，你可以颠倒各个path 点的编号顺序，因此，例如，如果path 有5个点，0点将变成4点，1点将变成3点，2点则不会改变。点的实际位置不变 点的实际位置保持不变，只是它们被处理的顺序被改变。这个函数改变了实际的path 资产，因此将永久影响游戏中所有实例对path 的使用，从使用这个函数的那一刻起直到游戏结束。 函数被使用直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数来先创建一个path 的副本，然后在复制的 上调用这个函数 在复制的asset （别忘了在不再需要的时候在asset 上调用 path_delete() ）。

语法。

path_reverse(index);

争论类型 描述
index The index of the path to change.

返回。

例子。

path_reverse(pth_AI);

这将扭转给定路径上所有点的处理顺序。
