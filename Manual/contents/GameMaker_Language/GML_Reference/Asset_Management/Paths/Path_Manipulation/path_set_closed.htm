路径_封闭

flag 这个函数可以用来将一个给定的path ，作为开放（假）或封闭（真）。一个封闭的path ，其起点与终点相连，形成一个loop ，而一个开放的path ，有一个明确的、不相连的起点和终点。这个函数改变了实际的path 资产，因此将永久地影响游戏中所有实例对path 的使用，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_set_closed(index, closed);

争论 类型 描述
index 路径资产 The index of the path to change.
closed 布尔型 Whether the path is closed (true) or not (false).

返回。

N/A

例子。

path_set_closed(pth_Patrol, true);

这将使给定的path ，成为一个封闭的path 。
