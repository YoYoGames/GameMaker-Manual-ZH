路径_存在

该函数返回具有给定索引的path 是否存在。注意，如果你通过一个尚未声明的变量来检查一个path 的存在，这将引发一个错误。

语法。

path_exists(index);

争论 类型 描述
index 路径资产 The index of the path to check for.

返回。

Boolean

例子。

if (path_exists(my_path))
{
    path_delete(my_path);
}

这段代码检查给定的变量是否存储了一个path 的索引，如果是，那么path ，从内存中删除。
