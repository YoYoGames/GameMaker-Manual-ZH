路径_移位

通过这个函数，你可以沿水平和垂直轴移动一个path 。请注意，如果被移动的path 后面的实例相对地（而不是绝对地）跟随它，这在游戏中不会有明显的效果，因为路径的 位置会被忽略。这个函数改变了实际的path 资产，因此将永久地影响游戏中所有实例对path 的使用，从使用这个函数的那一刻起直到游戏结束。如果这 不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记调用 path_delete()当它不再需要时，再调用asset ）。

语法。

path_shift(index, xshift, yshift);

争论类型 描述
index The index of the path to change.
xshift The number of horizontal pixels to shift the path. Negative=left, positive=right.
yshift The number of vertical pixels to shift the path. Negative=up, positive=down.

返回。

例子。

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

上面的代码复制了一个path asset ，并将其ID值存储在一个变量中。然后用这个变量来转移复制的path 的位置，然后实例启动path 。
