路径_重复

这个函数接收一个path ，并将其复制到一个新的path 。在这个过程中，新的path ，并返回其索引，以便在所有进一步的调用中使用这个新path 。

语法。

path_duplicate(index);

争论类型 描述
index The index of the existing path to duplicate.

返回。

Path Asset

例子。

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

上面的代码从四个path 资源中选择一个并复制它，将新path 的索引存储在变量 "mypath "中。
