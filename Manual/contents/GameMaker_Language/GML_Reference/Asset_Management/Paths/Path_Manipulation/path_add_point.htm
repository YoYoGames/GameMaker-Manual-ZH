路径_添加点

通过这个函数，你可以在指定的path ，并设置其速度系数。这个点会被添加到path 的末尾，不管这个点的位置如何，系数等于下面的实例到达这个点时实际走的百分比。 path_speedpath如果你想把一个path 的点放在其他位置，而不是终点，你应该用 path_insert_point().

如果在path 浏览器的Asset 上使用，请注意该函数将改变实际资产，因此将永久影响游戏中所有实例对path 的使用方式，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_add_point(index, x, y, speed);

争论 类型 描述
index 路径资产 The index of the path to add the point to.
x 真实的 The x coordinate of the new point.
y 真实的 The y coordinate of the new point.
speed 真实的 The speed factor of the point (default value is 100).

返回。

N/A

例子。

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

上述代码将在运行该代码的实例的X/Y位置周围创建一个圆形path 。
