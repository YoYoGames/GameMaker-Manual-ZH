path_delete

你可以使用这段代码从内存中删除一个path 。如果这个path 是用动态的方式创建的 path_add()创建的，持有path 索引的变量将不再对访问path 有效，因为它不再存在，如果path 是用路径编辑器创建的，那么path 在整个游戏中就不能再被访问了，因为你正在永久地删除它。

语法。

path_delete(index);

争论类型 描述
index The index of the path to delete.

返回。

例子。

var t_path = path_add();
if (mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1))
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

上面的代码将创建一个path ，并将其索引存储在一个局部变量中。这个path ，然后用来存储一个 mp_grid_path()生成的path ，如果它成功地找到了目标，就会被分配到 "mypath "中索引的path 。最后 "t_path "被删除。
