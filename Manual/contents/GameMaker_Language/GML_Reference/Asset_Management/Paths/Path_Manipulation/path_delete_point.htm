路径_delete_point

通过这个函数，你可以从指定的path 。Path 的点是从0开始编号的，所以第一个点总是第0个点，最后一个path 的点将是1。 path_get_number()- 如果你在一个path asset 上调用这个函数，那么这个点的移除将是永久性的，将来所有分配给path 的实例都将缺少这个被移除的点。如果这不是你所需要的，那么你应该使用像 path_duplicate() 这样的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要时在asset 上调用 path_delete() ）。

语法。

path_delete_point(ind, n);

争论 类型 描述
index 路径资产 The index of the path to delete a point from.
n 真实的 The defining point to delete.

返回。

N/A

例子。

var _col = true;
while (_col)
{
    var _num = path_get_number(mypath) - 1;
    if (_num > 2)
    {
        var _x1 = path_get_point_x(my_path, _num);
        var _y1 = path_get_point_y(my_path, _num);
        var _x2 = path_get_point_x(my_path, _num -1);
        var _y2 = path_get_point_y(my_path, _num -1);
        if (collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false))
        {
            path_delete_point(mypath, _num);
            _num -= 1;
        }
        else _col = false;
    }
    else _col = false;
}

上面的代码生成了一个loop ，该代码从末端向后检查与给定的object 上的点是否有碰撞。如果发现有碰撞，就删除一个path ，直到没有碰撞，或者path ，只有2个点长。
