路径_添加

通过这个函数，你可以在GameMaker中创建一个path ，而不需要使用path 编辑器。这个函数将返回path 的索引，这个索引应该存储在一个变量中，并从那时起作为该path 的参考。请注意，创建的path 是空的，即：它没有定义任何点，所以你将不得不使用其他可用的函数来添加点到path ，或使用MP网格来生成path 。一旦你完成了使用path ，或希望创建一个新的，并将其索引存储在同一变量中，你应该首先删除旧的path ，用 path_delete以防止内存泄漏，因为内存泄漏最终会使你的游戏崩溃。

语法。

path_add();

返回。

Path Asset

例子。

global.newpath = path_add();

这将创建一个新的path ，并将其索引分配给global.newpath。
