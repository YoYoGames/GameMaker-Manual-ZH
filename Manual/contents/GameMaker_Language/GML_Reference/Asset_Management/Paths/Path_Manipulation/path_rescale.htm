path_rescale

这个函数可以用来沿着纵轴和横轴重新调整给定的path ，基本上是将path 的每一个点移动到与这个比例相对应的新位置，围绕着path 。这个函数改变实际的path 资产。 因此将永久地影响游戏中所有实例对path 的使用，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_rescale(index, xscale, yscale);

争论类型 描述
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

返回。

例子。

path_rescale(pth_AI, 3, 3);

这将把给定的path ，重新放大到原来的三倍。
