路径_附加

通过这段代码，你可以将一个path 附加到另一个上，有效地将它们连接在一起，为了使其发挥作用，两个paths 必须先前存在。一般来说，你想在使用 path_add() 创建的paths 上使用这个函数，因为如果你在path asset 上使用这个函数，从使用这个函数的那一刻起到游戏结束，它将永久性地影响游戏中的实例path 。

注意这不会从游戏中删除正在附加的path 。它仍然存在，只是它现在没有积分，如果你不再想使用它，你应该用函数删除它。 path_delete().

语法。

path_append(index, path);

争论类型 描述
index The new index of the path.
path The index of the path that will be appended to 'index'.

返回。

N/A

例子。

path_append(mypath, mppath);
path_delete(mppath);

上述代码将变量 "mppath "中索引的path 数据追加到变量 "mypath "中索引的path ，然后将 "mppath "从内存中删除。
