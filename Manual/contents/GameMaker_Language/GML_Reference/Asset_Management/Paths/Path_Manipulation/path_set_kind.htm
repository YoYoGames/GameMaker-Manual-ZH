path_set_kind

这个函数可以用来设置你希望指定资源是哪种path 。这可以是一个直线的path （设置为0）或一个平滑的path （设置为1），在这种情况下，path 的精度也必须被考虑在内（精度 也可以用 path_set_precision()).这个函数改变了实际的path 资源，因此将永久地影响游戏中所有实例对path 的使用，从使用该函数的那一刻起直到游戏结束。 如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个path 的副本，然后在复制的 上调用这个函数。 在复制的asset （别忘了在不再需要的时候在asset 上调用 path_delete() ）。

语法。

path_set_kind(index, val);

争论类型 描述
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

返回。

例子。

path_set_kind(pth_Patrol, true);

这将把给定的path ，成为一个 "平滑 "的path 。
