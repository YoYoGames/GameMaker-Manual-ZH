路径_改变点

通过这个功能，你可以改变之前为path （path ）定义的任何点的位置和/或速度系数（path 编辑器中可以创建，或者通过代码使用 path_add()).如果在path 浏览器的Asset 上使用，请注意该函数将改变实际的资产，因此将永久影响游戏中所有实例对path 的使用方式，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，先创建一个path 的副本，然后在复制的asset 上调用这个函数（不要忘记在不再需要asset 的时候调用 path_delete() ）。

语法。

path_change_point(index, n, x, y, speed);

争论 类型 描述
index 路径资产 The index of the path to change a point of.
n 真实的 The defining point to change the attributes of.
x 真实的 The new x coordinate (relative to the path) of the point.
y 真实的 The new y coordinate (relative to the path) of the point.
speed 真实的 The new speed factor of the point.

返回。

N/A

例子。

for (var i = 0; i < path_get_number(path0); ++i;)
{
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
}

上面的代码loops ，在path ，索引为 "path0 "的所有点，并将所有的点重新定位到128x128像素的区域内的随机位置。
