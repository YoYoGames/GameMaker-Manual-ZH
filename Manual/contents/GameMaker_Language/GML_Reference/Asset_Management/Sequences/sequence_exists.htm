序列_存在

通过这个函数，你可以检查一个sequence object 是否存在。你可以提供sequence object struct （如函数所返回的 sequence_create()或 sequence_get())或sequence ID(如函数返回的 layer_sequence_get_sequence()或从sequence 实例struct 属性 sequence ，或从asset 浏览器的索引），如果sequence object 存在，该函数将返回 true ，如果不存在则返回 false 。

语法。

sequence_exists(sequence_struct_or_id);

争论 类型 描述
sequence_struct_or_id 序列资产或序列Object 结构 The sequence to check for, can be the asset reference itself or its Sequence object struct.

返回。

Boolean

例子。

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

上面的代码检查给定的sequence object 是否存在，如果存在，就销毁它。

返回。序列
