序列_创建

通过这个函数，你可以创建一个新的sequence object ，然后向其添加轨道。该函数返回一个结构，然后你可以访问该结构来设置你所创建的新sequence 。该struct 的内容在本页面上有详细介绍。sequence object struct ，然后可以使用该函数在sequence 层上创建room 的实例。 layer_sequence_create().注意，当以这种方式创建sequence objects 时，你应该在不再需要它们时再次调用函数来删除它们。 sequence_destroy()来删除它们，否则你会有一个内存泄漏，这可能会使你的游戏变慢并最终崩溃。

语法。

sequence_create();

返回。

Sequence Object Struct

例子。

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

上述代码创建了一个新的sequence object struct ，并设置其播放长度和loop 模式。

返回。序列
