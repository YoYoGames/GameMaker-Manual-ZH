序列_获取

通过这个函数，你可以从一个Object sequence 索引检索序列asset 结构。asset 你提供在sequence 浏览器中定义的asset 的索引，该函数将返回可以被访问的sequence object struct ，如果sequence 不存在或者给出的asset 索引不是sequence ，则返回-1。

语法。

sequence_get(sequence_index);

争论 类型 描述
sequence_index 序列资产 The sequence asset as defined in the Asset Browser.

返回。

Sequence Object Struct or -1

例子。

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

上述代码从sequence object "seq_Loop "中检索struct sequence object ，然后为loop 设置sequence 模式（所有随后创建的sequence 实例现在将使用这个loop 模式）。

返回。序列
