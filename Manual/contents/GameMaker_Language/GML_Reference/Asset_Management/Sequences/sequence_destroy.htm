序列_销毁

通过这个函数，你可以销毁一个已经动态创建的sequence object 。你可以提供sequence object struct （由函数返回 sequence_create())或sequence ID(由函数返回 layer_sequence_get_sequence()或从sequence 实例struct 属性 sequence ）。只要不再需要动态创建的sequence ，就应该使用这个函数，以释放与之相关的内存。

语法。

sequence_destroy(sequence_struct_or_id);

争论 类型 描述
sequence_struct_or_id 序列资产或序列Object 结构 The sequence object struct or ID to destroy

返回。

N/A

例子。

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

上面的代码检查给定的sequence object 是否存在，如果存在，就销毁它。

返回。序列
