序列_instance

这是一个内置变量，是为你游戏中每个object 实例创建的实例变量的一部分。如果该实例由sequence 控制，该变量将持有控制该实例的Sequence 的序列实例结构，否则它将是 undefined 。这是一个只读的变量，不能被改变。

注意，在控制的Sequence 结束后，这个变量将变成undefined ，即使它的Sequence 元素仍然存在，如果该struct 元素被重放，将再次得到Sequence 。

语法。

sequence_instance

返回。

Sequence Instance Struct

例子。

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

上面的代码检查 in_sequence变量，如果它是 true （意味着该实例被一个sequence 控制），那么它将把该Sequence 的速度刻度改为2。

返回。序列
