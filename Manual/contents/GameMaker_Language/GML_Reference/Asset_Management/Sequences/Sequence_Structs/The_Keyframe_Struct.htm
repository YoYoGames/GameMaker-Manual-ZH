关键帧结构

每个轨道（如涵盖轨道结构的页面所定义）将有一个或多个keyframe 结构分配给它（你可以从轨道的 keyframes 属性中获得struct ），每个结构将有以下属性。

序列Keyframe 结构
变化的 类型 描述
frame 真实的 timeline ，沿着keyframe 的位置（以帧为单位）。 默认值为0。
length 真实的 keyframe 的长度。默认值为1，当设置为更大的值时，那么keyframe 所指的轨道属性将在给定长度的持续时间内保持初始值。请注意，如果设置为 true ， stretch 属性将覆盖这一点。
stretch 布尔型 如果此属性被设置为 true ，那么keyframe ，就会延伸到轨道的下一个keyframe ，如果是最后一个keyframe ，则延伸到轨道的末端。你可以获得或设置这个值，默认值是 false 。
channels 阵列的 序列Keyframe 数据结构s 此属性允许访问轨道的通道的关键帧数据结构列表。获取此属性时，会返回一个keyframe 数据的数组 structs ，而设置此属性时，应指定一个keyframe 数据的数组structs 。
