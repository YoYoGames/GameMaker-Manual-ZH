Sequence 实例结构

一个sequence ，当放在一个room ，被作为一个元素放在层上，这个元素包含一个序列实例。这个sequence 实例在其结构中会有以下属性。

序列实例结构
变化的 类型 描述
sequence 序列Object 结构 这是序列对象 struct 的ID，sequence 实例是根据这个ID创建的（更多细节请看这里）。你可以获取或设置这个struct ，从而改变实例所引用的基础sequence 。
headPosition 真实的 这是该sequence 实例的当前播放头的位置（以帧为单位）。你可以获得或设置这个值，但要注意，将它设置为一个大于或小于当前定义的播放区域的值，会产生不同的效果，这取决于为sequence 设置的播放类型。 layer_sequence_headpos()获取更多信息。
headDirection 序列方向常数 这是该sequence 实例的当前播放方向。你可以使用下表中给出的常数来获取或设置这个值（注意，你可以使用层函数 layer_sequence_get_headdir()和 layer_sequence_headdir()来获取或设置这个值，而不需要直接访问struct ）。
speedScale 真实的 此属性可用于获取或设置播放速度比例。速度比例是一个乘数，其中1是默认的播放速度，小于1的值会减慢播放速度，大于1的值会加快播放速度，例如：0.5的值会是一半的播放速度，而2的值会是两倍的播放速度。注意，你可以使用层函数 layer_sequence_get_speedscale()和 layer_sequence_speedscale()来获得或设置这个值，而不必直接访问struct 。
paused 布尔型 你可以检查这个属性，看一个sequence 是否已经暂停，如果已经暂停，则为 true ，否则为 false 。这是一个只读的属性，但你可以使用层函数 layer_sequence_pause()来暂停播放，并在需要时 layer_sequence_play()如果需要的话，可以再次恢复播放。你也可以用layer函数来检查这个属性 layer_sequence_is_paused()而不是直接检查struct 中的属性。
finished 布尔型 你可以检查这个属性，看一个sequence 是否已经播放完毕，如果它已经播放完毕，则返回 true ，否则返回 false 。这是一个只读属性，对于没有设置为loop 或 ping-pong 的轨道，只会返回 true 。你也可以使用layer函数来检查这个属性 layer_sequence_is_finished()而不是直接检查struct 中的属性。
elementID 序列元素ID 此属性持有sequence 元素的ID。这个ID是一个简单的识别值，它与room 编辑器中的一个层相关联，然后它可以与适当的层函数一起使用，以找到sequence 被分配到的层，或者改变某些sequence 的属性，而不必处理struct 。
activeTracks 阵列的 序列活动轨迹结构s 此属性将持有一个 "评估 "structs 的数组，其中包含sequence （图形、sequence 、音频等）中每个资产轨道的当前状态信息。这些是顶级的轨道，返回的数组是只读的。关于每个数组条目中返回的asset 轨道的内容struct ，请看下面的章节。

序列方向常数
恒定 描述 价值
seqdir_right sequence ，将按照从左到右的递增顺序播放帧。 1
seqdir_left sequence ，将按照从右到左的递减顺序播放帧。 -1

下面的列表包含了在访问活动轨道struct （如包含在Sequence 实例struct 的 activeTracks 数组中，上面有解释）时，你可能可用的属性。每个asset 轨道struct ，都可以有以下任何属性，这取决于该轨道使用的asset 的类型（注意，你可以修改所有这些属性，除非指定为只读），所有返回的值都是针对当前播放头的位置。

序列活动轨道结构
变化的 类型 描述
matrix 矩阵阵列 这是用于绘制轨道的变换矩阵（sprite ，实例和sequence 轨道）。
posx 真实的 asset ，在当前播放头位置的轨道上，sequence 沿X轴的位置（所有assets ）。
posy 真实的 asset sequence 中的位置，在当前播放头位置的轨道沿Y轴的位置（所有 ）。assets
scalex 真实的 asset sequence 中的比例，沿X轴为当前播放头位置的轨道（ ，实例和 轨道）。sprite sequence
scaley 真实的 asset sequence 中的比例，沿Y轴为当前播放头位置的轨道（ ，实例和 轨道）。sprite sequence
xorigin 真实的 asset ，轨道的X原点（所有assets ）。
yorigin 真实的 asset ，轨道的Y原点（所有assets ）。
width 真实的 asset 的宽度（像素），用于轨道（sprite ，实例和sequence 轨道）。
height 真实的 asset 的高度（像素），用于轨道（sprite ，实例和sequence 轨道）。
imageindex 真实的 在当前播放头位置的asset ，sequence 的图像索引（sprite 和实例轨道）。
imagespeed 真实的 asset ，在当前播放头位置的sequence 的轨道上的图像速度（sprite 和实例轨道）。
colourMultiply 颜色 在当前播放头位置的asset 中，sequence 的颜色倍增值（sprite ，实例和sequence 轨道）。这个值将是一个由四个ARGB值组成的数组，格式为[A, R, G, B]。请注意，每个分量的值都表示在0和1之间，其中0对应的是HEX值#00，1对应的是HEX值#FF（0-255，如在Sequences 编辑器中的颜色倍增轨道的颜色选择器中所示）。
emitterIndex 音频发射器ID 属于该轨道（音轨）的音频发射器的索引。
gain 真实的 当前播放头位置的轨道的增益（音量）（音频轨道）。
pitch 真实的 当前播放头位置的轨道的音高（音频轨道）。
frameSizeX 真实的 文本框（文本轨道）的水平尺寸。
frameSizeY 真实的 文本框（文本轨道）的垂直尺寸。
characterSpacing 真实的 字符间距值（文本轨道）。
lineSpacing 真实的 行距值（文本轨道）。
paragraphSpacing 真实的 段落间距值（文本轨道）。
track 序列轨道结构 该轨道所基于的轨道结构（所有assets ，是一个只读值）。
parent 序列实例结构 父级sequence ，轨道的实例ID（所有assets ，只读值）。
activeTracks 阵列的 序列轨道结构s 这是一个数组，包含了structs 轨道所包含的每个参数轨道的评估asset （所有assets ，是一个只读值）。数组中每个struct 的内容都列在参数轨道结构部分。
