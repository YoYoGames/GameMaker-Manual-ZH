序列_实例_覆盖_对象

通过这个函数，你可以用另一个object 覆盖（替换）所有用于sequence 的实例。sequence 你提供struct 实例sequence ID（如在room 中创建room 实例时返回的ID，或通过使用object 层函数之一--见这里），以及你想覆盖的asset 索引（如在object 浏览器中定义）。最后，你给出一个object 索引或一个实例ID，作为要覆盖object 的sequence （提供一个实例ID将简单地使用该实例所创建的object 作为覆盖）。请注意，这只能在sequence 实例上进行（而不是sequence objects ），而且必须在sequence 开始播放之前进行，否则将无法工作。

语法。

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

争论 类型 描述
sequence_instance_struct 序列实例结构 The sequence instance struct to modify.
object_id 对象资产 The object index of the object within the sequence to override.
instance_or_object_id 对象资产或实例ID The object index or instance ID to use to override the sequence objects.

返回。

N/A

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

上面的代码在给定的图层上创建了一个新的sequence 实例，然后对其进行修改，使object "obj_Trees_Winter "的所有实例被object "obj_Trees_Summer "的实例取代。

返回。序列
