序列_获取对象

通过这个函数，你可以检索一个数组，其中包括所有在给定的object 中有实例被创建的sequence 指数。你可以提供sequence object struct （如函数所返回的 sequence_create()或 sequence_get())或sequence ID(如函数返回的 layer_sequence_get_sequence()或从sequence 实例struct 属性 sequence ），该函数将返回一个数组，其中数组中的每一项是一个 object_index为不同的objects ，被sequence 用于创建实例。

语法。

sequence_get_objects(sequence_struct_or_id);

争论 类型 描述
sequence_struct_or_id 序列资产或序列Object 结构 The sequence object struct or ID to get the objects from.

返回。

Array

例子。

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

上面的代码为一个struct sequence ，得到object ，然后检索它所使用的objects ，并将该数组存储在一个变量中。

返回。序列
