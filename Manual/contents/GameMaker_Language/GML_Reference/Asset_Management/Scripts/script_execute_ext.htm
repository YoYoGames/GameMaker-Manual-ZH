脚本_执行_ext

这个函数的工作原理与函数script_execute()类似，只是你可以提供一个包含被调用的函数/脚本所需参数的数组。你还可以向该函数提供两个可选的参数，以指定从数组中获取参数的偏移量，以及从数组中使用的参数数量（这必须是array_length - offset的最大值）。

语法。

script_execute_ext(scr, array_args, [offset], [num_args]);

争论 类型 描述
scr 脚本资产或脚本功能 The function/script that you want to call
array_args 阵列 The array containing the arguments for the function/script
[offset] 真实的 OPTIONAL 参数数组中的偏移量
[num_args] 真实的 OPTIONAL 要使用的参数的数量（从偏移量开始）。

返回。

Variable (Will depend on the return value from the function/script being called)

例子。

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

上面的示例代码将使用 script_execute_ext 来调用给定的函数，提供一个包含四个参数的数组，但只随机使用其中的一个参数。

注意该功能不能与内置功能一起使用。

返回。脚本
