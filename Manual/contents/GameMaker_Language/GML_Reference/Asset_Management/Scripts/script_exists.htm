script_exists

这个函数将返回 true 或 false ，这取决于具有给定索引的script 或脚本函数是否存在。注意，这不是一个string ，而是asset 的名称，它持有每个script 的唯一索引（因为它将出现在IDE ）或命名的script 函数，正如在script asset 中定义的那样（注意，这对方法变量不起作用）。关于scripts 的更多信息，请参见 Script 编辑器。

语法。

script_exists(scr);

争论 类型 描述
scr 脚本资产 The script index that you want to check.

返回。

Boolean

例子。

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

上面的例子将两个script 函数和一个值添加到一个数组中，然后继续获得一个随机数，并使用这个随机数来选择一个script 函数来运行，除非选择-1，在这种情况下什么也不会发生。
