脚本名称

这个函数将返回指定的script 的名称，即一个字符串。这个名字是在GameMaker主窗口的script 浏览器中为Asset 指定的名字。有关scripts 的更多信息，请参见TheScript Editor。

语法。

script_get_name(scr);

争论 类型 描述
scr 脚本资产 The index of the script that you want to get the name of.

返回。

String

例子。

scr_name = script_get_name(Help_File);

上面的示例代码将在变量 "scr_name "中存储指定的script index的名称。

返回。脚本
