延长线_获取选项_值

该函数返回在extension ，给定名称的选项的值。

如果提供的extension 名称无效，该函数将返回未定义。

语法。

extension_get_option_value(extension_name, option_name);

争论 类型 描述
extension_name 字符串 The name of your extension asset as a string
option_name 字符串 The name of the option to read

返回。

Any

例子。

var _enabled = extension_get_option_value("MyExtension", "enabled");

这段代码在MyExtension extension 中获得启用的选项的值。

返回。延长线
