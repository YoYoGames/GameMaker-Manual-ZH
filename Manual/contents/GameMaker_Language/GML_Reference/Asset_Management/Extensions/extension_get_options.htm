延长线_获取选项

该函数返回一个struct ，其中包含所有的选项，以及它们的值，用于给定名称的extension 。

如果提供的extension 名称无效，该函数将返回未定义。

语法。

extension_get_options(extension_name);

争论 类型 描述
extension_name 字符串 The name of your extension asset as a string

返回。

Struct

例子。

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

这段代码为一个struct ，获取选项extension ，并从中读取一个值。

返回。延长线
