extension_exists

这个函数告诉人们，具有给定名称的extension 是否存在（真）或不存在（假）。

语法。

extension_exists(extension_name);

争论 类型 描述
extension_name 字符串 The name of your extension asset as a string

返回。

Boolean

例子。

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

这段代码检查MyExtension extension 是否存在，如果存在，则将其选项作为struct ，并从中读取一个值。
