animcurve_get_channel

该函数将返回包含animation curve asset 中指定的通道数据的结构。你提供animation curve ID或struct ，以及通道名称或索引，该函数将返回一个格式如下的struct 。

变量名称 数据类型 描述
name 绳子 这是频道的名称。
type 动画曲线插值类型常数 这将是一个常数 animcurvetype_linear ，用于点之间的线性插值， animcurvetype_catmullrom ，用于使用catmull-rom插值的点之间的 "平滑 "插值，或者 animcurvetype_bezier ，用于Bezier插值。
iterations 整数 如果通道使用catmull-rom("平滑")插值，这个值表示为曲线的每一段生成了多少个点(注意，这些额外的点是函数内部的，只用于runtime 计算)。如果通道使用的是线性插值，这个值仍然存在，但可以忽略，因为它对曲线的插值方式没有影响。
points 阵列指针 这是一个数组，数组中的每一项都是一个点struct 。


animation curve ID是在animation curve 浏览器中定义的Asset 的名称，或由函数返回的struct 指针。 animcurve_create().通道名称是一个string ，指的是动画曲线asset 中定义的通道，或者你可以提供一个索引值，从0到n，其中n是曲线asset 中的最后一个通道（例如：如果一个animation curve 有4个通道，它们将被索引为0到3）。请注意，传递一个索引值比传递一个通道名称需要更少的处理。如果函数失败（即：没有给定名称或索引的通道存在），你将得到一个错误。

单个通道上的点以structs ，存储在一个数组中，数组中的每一项都是一个点struct 。该点struct ，有以下变量。

变量名称 数据类型 描述
posx 真正的 该点在时间上的位置（从0到1的正常化）。
value 真正的 点的价值。

语法。

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

争论 类型 描述
curve_struct_or_id 动画曲线结构 The ID or struct pointer of the animation curve to target
channel_name_or_index 字符串或 真实 The channel name (a string) or the channel index (an integer).

返回。

Animation Curve Channel Struct

例子。

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

上面的代码在曲线struct "ac_ButtonTween "中为通道0检索通道asset ，然后loops ，通过通道曲线上的点，在它们的值上加一。
