animcurve_channel_evaluate

这个函数可以用来从一个通道结构中获取特定时间点的值。你提供该通道的struct 指针（如函数所返回的 animcurve_get_channel()或从函数animation curve struct 。 animcurve_get())和要评估的沿曲线通道的 "x"（时间）位置。这个位置应该在0到1之间，函数将返回该位置的曲线值，如果提供的通道struct 无效，则返回0。

语法。

animcurve_channel_evaluate(channel_struct, posx);

争论 类型 描述
channel_struct 动画曲线通道结构 The struct pointer for the channel to evaluate.
posx 真实的 The position in time to check (from 0 to 1).

返回。

Real

例子。

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

上面的代码为struct animation curve "ac_AlphaCurve "获取通道asset 。然后它使用返回的评估值来设置实例的图像alpha。
