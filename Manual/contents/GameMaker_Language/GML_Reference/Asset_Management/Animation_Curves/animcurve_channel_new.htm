animcurve_channel_new

这个函数可以用来创建一个新的animation curve channel结构。一个通道struct ，有以下的变量。

动画曲线通道结构
变量名称 数据类型 描述
名称 绳子 这是频道的名称。
类型 动画曲线插值类型常数 这将是下表中给出的常数之一。
迭代 整数 如果通道使用catmull-rom("平滑")插值，这个值表示为曲线的每一段生成了多少个点(注意，这些额外的点是函数内部的，只用于runtime 计算)。如果通道使用的是线性插值，这个值仍然存在，但可以忽略，因为它对曲线的插值方式没有影响。
点 阵列 这是一个数组，其中数组中的每一项都是一个 动画曲线点结构.

动画曲线插值类型常数
恒定 描述
animcurvetype_linear 用于点与点之间的线性内插。
animcurvetype_catmullrom 用于使用catmull-rom插值在各点之间进行平滑插值。
animcurvetype_bezier 用于点之间的贝塞尔插值。


默认情况下，当你创建一个新的通道struct ，"名称 "变量将是一个空的string ，"类型 "将是 animcurvetype_linear ，"迭代次数 "将是16。所有这些变量都可以设置为你所需要的值（注意，"迭代 "值对线性曲线类型没有影响）。一旦创建完毕，你需要为曲线添加点，这可以通过在 "点"阵列中添加不同的点structs 。这些点structs ，可以用函数创建 animcurve_point_new().

语法。

animcurve_channel_new();

返回。

Animation Curve Channel Struct

例子。

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的animation curve struct ，并将其存储在变量 "my_curve "中。这个struct ，然后用一个名称和一个通道数组来填充。通道数组包含一个有三个点的单一通道。
