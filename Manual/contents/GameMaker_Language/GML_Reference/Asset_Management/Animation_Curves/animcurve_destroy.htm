animcurve_destroy

这个函数可以用来销毁任何以前用函数创建的animation curves 。 animcurve_create().调用这个函数将从内存中删除animation curve ，并同时清理它所包含的任何通道或点（这些从来不需要手动清理）。注意，你不能销毁在animation curves 浏览器中创建的任何asset ，只能销毁在runtime 。

语法。

animcurve_destroy(curve_struct);

争论 类型 描述
curve_struct 动画曲线结构 The pointer to the curve struct to destroy.

返回。

N/A

例子。

animcurve_destroy(my_curve);

上述代码将销毁（先前创建的）animation curve struct ，并在变量 "my_curve" 中建立索引。
