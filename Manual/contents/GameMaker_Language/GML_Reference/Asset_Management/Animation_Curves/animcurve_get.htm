动画curve_get

这个函数将返回一个包含给定animation curve 的所有数据的结构。你提供animation curve asset ID（如在Asset 浏览器中定义的），该函数将返回一个包含以下变量的struct 。

动画曲线结构
变量名称 数据类型 描述
name 字符串 这是animation curve 的名称。
channels 阵列的 动画曲线通道结构s 这是一个数组，数组中的每一项都是一个通道struct 。


channels struct 中的变量是一个数组，每个数组项是一个 动画曲线通道结构含有与曲线中的一个通道有关的数据。本页对通道struct 进行了解释。

与通道一样，单个通道上的点以structs 的形式存储在一个数组中，数组中的每一项都是一个点struct 。本页对点struct 进行了解释。

请注意，如果函数失败（例如，给定的动画曲线asset 不存在），那么该函数将返回-1。

语法。

animcurve_get(curve_id);

争论 类型 描述
curve_id 动画曲线资产 The asset browser ID (index) of the animation curve to get.

返回。

Animation Curve Struct or -1

例子。

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
{
    _channel.type = animcurvetype_linear;
}

上面的代码为曲线animation curve "ac_ButtonTween "检索struct asset ，然后检查第一个通道的曲线类型，如果它不是线性的，就把它设置为线性。
