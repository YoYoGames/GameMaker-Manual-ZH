创建动画曲线

这个函数将创建一个空的animation curve 结构，准备让你填充通道数据。该函数将返回一个指向struct 的指针，然后用这个指针向animation curve 添加通道和其他数据。你也可以将这个指针提供给像 animcurve_get()等函数，以便以后从曲线上获取数据。

当你使用这个函数时，创建的struct ，里面没有任何数据，要使用它，你应该至少添加一个通道，而且这个通道应该有点添加到里面。要添加一个通道，见函数 animcurve_channel_new()，要向通道添加点，见函数 animcurve_point_new().此外，你可以通过设置 "名称 "变量给animation curve struct 一个名称（如下面的示例代码所示）。添加到一个新的animation curve 的通道应该作为一个数组添加，数组中的每一项都引用一个通道struct ，而通道中的点也应该作为一个数组添加，数组中的每一项都引用一个点struct 。

请注意，以这种方式创建的animation curves ，在不再需要时必须删除，否则会占用内存空间，这可能导致内存泄漏，使游戏速度变慢并最终崩溃。你可以在不再需要的时候使用函数删除一个animation curve 。 animcurve_destroy().你不需要清理曲线中的任何通道或点的数据，因为当曲线本身被销毁时，垃圾收集器会自动进行清理。

语法。

animcurve_create();

返回。

Animation Curve Struct

例子。

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的animation curve struct ，并将其存储在变量 "my_curve "中。这个struct ，然后用一个名称和一个通道数组来填充。通道数组包含一个有三个点的单一通道。
