animcurve_point_new

该函数可用于创建一个新的点结构，以添加到animation curve 通道。当创建一个点struct 是空的，你需要设置以下变量来生成点数据。

动画曲线点结构
变量名称 数据类型 描述
posx 真正的 该点在时间上的位置（从0到1的正常化）。
value 真正的 点的价值。

你创建的每个点struct ，都应该被添加到一个数组中，这就是被添加到animation curve channelstruct "channels "变量中的数组（如下例所示）。

语法。

animcurve_point_new();

返回。

Animation Curve Point Struct

例子。

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的animation curve struct ，并将其存储在变量 "my_curve "中。这个struct ，然后用一个名称和一个通道数组来填充。通道数组包含一个有三个点的单一通道。
