动画曲线_存在

这个函数将返回true，如果给定的 动画曲线资产或 动画曲线结构存在并且是一个有效的动画曲线，则该函数将返回true；如果不是，则返回false。

语法。

animcurve_exists(curve_struct_or_id);

争论 类型 描述
curve_struct_or_id 动画曲线结构或 动画曲线资产 The Animation Curve that will be checked.

返回。

Boolean

例子。

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

上面的代码检查存储在自定义spring_curve变量中的动画曲线是否存在。如果存在，它将运行一个自定义方法来使用该动画曲线。
