animcurve_get_channel_index

这个函数将返回任何给定的animation curve 通道的索引值。你提供animation curve ID或struct ，其中曲线ID是在animation curve 浏览器中定义的Asset 的名称，或由函数 animcurve_create() 返回的struct 指针。然后你提供通道的名称，作为string ，函数将返回与该通道相关的索引值。注意，如果曲线或通道不存在，你将得到一个错误。

语法。

animcurve_get_channel_index(curve_struct_or_id, channel_name);

争论 类型 描述
curve_struct_or_id 动画曲线结构 The ID or struct pointer of the animation curve to target
channel_name 字符串 The channel name (a string).

返回。

Real

例子。

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

上面的代码在曲线struct "ac_ButtonTween "中为名为 "x_pos "的通道检索通道asset ，然后loops ，通过通道曲线上的点，为它们的值加一。
