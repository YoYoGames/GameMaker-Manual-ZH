sha1_file

在密码学中，SHA-1是一个由美国国家安全局设计的密码学哈希函数，并在几个广泛使用的应用程序中被采用，其中它被用来检查文件的变化，以及协议TLS和SSL、PGP、SSH、S/MIME和IPsec。这个函数将接受一个输入文件，并返回一个160比特的ASCII格式的消息摘要，该文件是唯一的，可在以后用于完整性验证。

注意：SHA-1不是完全安全的，可以被破解。更多信息请见本页面。

语法。

sha1_file(filename)

争论 类型 描述
filename 字符串 The file to generate the sha1 hash for.

返回。

String

例子。

hash = sha1_file(working_directory + "game_data.ini")

上面的代码将为指定的文件生成一个sha1哈希值，并将返回值存储在变量 "hash "中。
