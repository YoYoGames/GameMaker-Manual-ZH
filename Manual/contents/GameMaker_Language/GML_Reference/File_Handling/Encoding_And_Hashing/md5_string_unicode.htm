md5_string_unicode

在密码学中，MD5（Message-Digest algorithm 5）是一个广泛使用的密码学哈希函数，具有128位的哈希值，并被广泛用于各种安全应用。它也常用于检查文件的完整性和strings 。 这个函数将接受一个输入的unicodestring （每个字符为16位），并返回32个字符的十六进制MD5散列，该散列对该string 。通过这种方式，你可以生成一个安全的密钥，该密钥可以被存储并用于检查被发送到（或发送到）的信息的完整性。 被发送到（或从）一个外部server （例如）的信息的完整性。

注意 MD5编码有两种格式，UTF-8和unicode。提供这两种格式是为了方便与不同的server ，但最常用的是unicode。

注意：MD5不是完全安全的，可以被破解。更多信息请见本页面。

语法。

md5_string_unicode(string)

争论类型 描述
string The string to hash.

返回。

例子。

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

上述代码将对一个string 进行base64编码，然后生成一个MD5哈希值。最后，哈希值和编码后的string 都被发送到一个server 。
