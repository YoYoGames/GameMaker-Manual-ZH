base64_encode

这个函数将把一个string 转换成base64格式编码的string 。这是一个常用的编码方案，通常用于任何需要在互联网上以文本形式存储或传输的媒体，并使输出的内容对人眼来说不可读。 人眼无法阅读。

语法。

base64_encode(string)

争论类型 描述
string The string to encode.

返回。

例子。

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

上述代码将把存储在 "game_data "中的string 转化为base64编码的string ，然后存储在一个外部文本文件中。
