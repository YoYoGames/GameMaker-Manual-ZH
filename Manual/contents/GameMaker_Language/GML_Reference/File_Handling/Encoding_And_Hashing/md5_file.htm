md5_file

在密码学中，MD5（Message-Digest algorithm 5）是一个广泛使用的密码学哈希函数，其哈希值为128位，已被广泛用于各种安全应用。它也常用于检查文件的完整性和strings 。 这个函数将采取给定的文件，并为该文件生成一个唯一的MD5，然后可以储存起来供以后使用。

注意：MD5不是完全安全的，可以被破解。更多信息请见本页面。

语法。

md5_file(filename)

争论类型 描述
filename The file to generate the MD5 hash for.

返回。

例子。

hash = md5_file(working_directory + "game_data.ini")

上面的代码将为指定的文件生成一个MD5哈希值，并将返回值存储在变量 "hash "中。
