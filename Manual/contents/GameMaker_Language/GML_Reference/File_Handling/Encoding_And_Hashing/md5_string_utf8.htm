md5_string_utf8

在密码学中，MD5（Message-Digest algorithm 5）是一个广泛使用的密码学哈希函数，其哈希值为128位，已被广泛用于各种安全应用。它也常用于检查文件的完整性和strings 。 这个函数将接受一个输入的UTF-8string （每个字符的字节数是可变的），并返回该string 的唯一的32个字符的十六进制MD5散列。通过这种方式，你可以生成一个安全的密钥，它可以被存储并用于 来检查被发送到（或从）外部server （例如）的信息的完整性。

注意：MD5编码有两种格式，UTF-8和unicode。提供这两种格式是为了方便与不同的server 设置进行通信，但最常用的是unicode。

注意：MD5不是完全安全的，可以被破解。更多信息请见本页面。

语法。

md5_string_utf8(string)

争论类型 描述
string The string to hash.

返回。

例子。

var hash, str;
str = json_encode(hiscore_map);
hash = md5_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)
