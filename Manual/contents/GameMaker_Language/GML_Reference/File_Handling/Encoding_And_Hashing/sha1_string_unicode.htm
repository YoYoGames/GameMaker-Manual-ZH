sha1_string_unicode

在密码学中，SHA-1是一个由美国国家安全局设计的密码学哈希函数，并在一些广泛使用的应用程序和协议中被采用，如流行的Git，它被用来检查文件变化。 这个函数将接受一个输入的unicodestring ，并返回一个ASCII格式的160位消息摘要，例如。

sha1 input string = "The quick brown fox jumps over the lazy dog"
sha1 output hash = "2fd4e1c6 7a2d28fc ed849ee1 bb76e739 1b93eb12"

通过这种方式，你可以生成一个安全的密钥，它可以被存储并用于检查被发送到（或从）外部server （例如）的信息的完整性。

注意：sha1编码有两种格式，UTF-8和unicode。提供这两种格式是为了方便与不同的server 设置进行通信，但最常用的是unicode。

注意：SHA-1不是完全安全的，可以被破解。更多信息请见本页面。

语法。

sha1_string_unicode(string)

争论类型 描述
string The string to hash.

返回。

例子。

var hash, str;
str = base64_encode(game_data);
hash = sha1_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

上述代码将对一个string 进行base64编码，然后生成一个sha1哈希值。最后，哈希值和编码后的string 都被发送到一个server 。
