zip_unzip

这个函数将打开一个存储的ZIP文件，并将其内容提取到给定的目录中。注意，如果你没有提供ZIP目录的完整path ，那么将使用当前的驱动器根目录，如果你想把它放在游戏捆绑工作目录的一个相对path ，那么你应该使用 working_directory变量作为path 的一部分（使用". "或"... "的相对paths 将不会像预期的那样工作，所以应该避免）。还要注意的是，压缩包必须是游戏包的一部分（即：包含的文件），或者已经用http_get_file()下载到存储区。

该函数将返回一个值，表明被提取的文件数量，如果提取失败，它将返回0或更少。

语法。

zip_unzip(zip_file, target_directory)

争论类型 描述
zip_file The zip file to open
target_directory The target directory to extract the files to

返回。

例子。

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

上述代码将打开存储在 "downloads "目录下的压缩文件，并将其内容解压缩到 "extracted "目录下（如果该目录不存在，则创建该目录），然后检查解压缩是否正确，如果失败则显示调试信息。
