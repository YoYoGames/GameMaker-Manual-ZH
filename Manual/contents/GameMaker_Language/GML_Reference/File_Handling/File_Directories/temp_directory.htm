临时目录

这可以用来返回每次运行时为你的游戏创建的临时目录（根目录不包含最后的"/"）。这个目录将保存文件，在游戏运行时可以被访问，但当游戏关闭时，它将被删除（连同其包含的所有文件）。 但当游戏关闭时，它将被删除（连同它所包含的所有文件）。

警告!由于GameMaker是沙盒式的，这个功能可能不会像你预期的那样工作!更多信息请参见文件系统部分。

语法。

temp_directory

返回。

例子。

ini_open(temp_directory + "\temp_ini.ini");

这将在游戏的临时目录中打开一个ini文件（如果它不存在，则创建它）。
