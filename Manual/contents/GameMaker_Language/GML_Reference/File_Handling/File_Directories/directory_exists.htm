directory_exists

如果指定的目录存在，这个函数将返回 true ，如果不存在，则返回 false 。指定的名称必须包括完整的path ，而不是相对的path ，默认情况下，你不能从游戏包以外的地方访问任何目录，因为所有的游戏都是沙盒的（更多信息见文件系统部分）。

语法。

directory_exists(dname)

争论类型 描述
dname The name of the directory to look for.

返回。

例子。

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

这将检查指定的目录是否存在，如果存在，就去那里并返回找到的第一个 "只读 " doc 文件。
