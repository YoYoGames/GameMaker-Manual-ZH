文件目录

在处理文件时，有的时候你需要引用文件存储的目录，GameMaker有一些特殊的函数和变量，与你可能需要的不同目录有关。

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory
  • game_save_id（来自一般游戏控制）。

注意你不能在HTML5上创建或销毁目录。
