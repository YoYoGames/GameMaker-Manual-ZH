文件_text_open_write

该函数打开指定文件名的文本文件，仅用于写入（如果文件不存在，则创建），返回文件的唯一ID，该ID应存储在一个变量中，因为它将用于所有与该文件有关的进一步操作。

注意，如果文件不能被创建（例如，由于非法文件名），该函数将返回-1。

如果文件已经存在，使用这个函数将导致它在保存数据时被覆盖。如果你想向一个文件添加数据，同时保持其现有内容不变，可以使用 file_text_open_append().

注意：你在任何时候最多只能打开32个文件。你也应该在完成后总是关闭文件，因为这样可以写入信息并释放与文件相关的内存。

警告!由于GameMaker是沙盒式的，这个功能可能不会像你预期的那样工作!更多信息请参见文件系统部分。

语法。

file_text_open_write(fname);

争论 类型 描述
fname 字符串 The name of the file to write to.

返回。

Text File ID or -1

例子。

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

上述代码将打开 "level.txt "文件进行写入，然后写入存储在变量 "level_data "中的string ，最后再次关闭文件。

返回。文本文件
