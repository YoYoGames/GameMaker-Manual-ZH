file_text_write_real

通过这个函数，你可以把一个数字写到先前打开的文本文件中。注意，由于要写入的数值可以是一个实数，所有的小数都将以". "点作为分隔符写入。如果文件中已经包含了信息，这些信息将被清除，string ，并写在文件的开头，除非你已经用 file_text_open_append().你也可以通过使用 file_text_readln()函数和 file_text_eof()函数来loop ，直到文件的内容，然后开始写入。

需要注意的是，当使用这个函数将非常大的数字写入文本文件时，它可能会被翻译成标准的简化格式，如 "6.6624e+003"，无法正确读回GameMaker。为了防止这样的问题，你应该把数值转换为string ，然后使用函数 file_text_write_string()来代替。

语法。

file_text_write_real(fileid, val);

争论类型 描述
fileid The id of the file to edit.
val The real value to write to the file.

返回。

例子。

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

上面的代码打开一个文件进行写入，然后loops ，通过两个数组，将每个数组的值写入文件的一个新行中。当loop 完成后，该文件被关闭。

返回。文本文件
