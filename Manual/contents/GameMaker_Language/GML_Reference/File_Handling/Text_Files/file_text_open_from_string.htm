从字符串打开的文本

这个函数将从string ，创建一个文本文件，并打开它进行阅读，返回文件 "句柄"，该句柄应在所有进一步的文件函数调用中使用，以从该文件读取。请注意，这个文件是临时的、只读的，因此它将在关闭时从内存中删除。

注意：你在任何时候最多只能打开32个文件。你也应该在结束时总是关闭文件，因为这样可以写入信息并释放与文件相关的内存。

警告!由于GameMaker是沙盒式的，这个功能可能不会像你预期的那样工作!更多信息请参见文件系统部分。

语法。

file_text_open_from_string(string);

争论类型 描述
string The string to create the file from.

返回。

File ID or -1

例子。

file = file_text_open_from_string(reset_str);

上面的代码把存储在变量 "reset_str "中的string ，用它来创建一个只读文本文件。然后，这个文件的 "句柄 "被存储在变量 "file "中，供所有其他文件函数使用。

返回。文本文件
