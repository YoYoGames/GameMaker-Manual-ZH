ini_open_from_string

这个函数将从string 创建一个临时的 ini 文件，并打开它进行读/写。string 应该被正确地格式化为一个ini文件（即：有章节、换行键和数值），否则ini文件将不能被正确创建。请注意，这个ini文件是临时性的，一旦关闭就会从内存中删除，会丢失存储在其中的任何信息。 ini_close()函数返回一个完整的ini文件的string ，然后可以保存到server 或磁盘上。

语法。

ini_open_from_string(string);

争论类型 描述
string The string containing all the ini information.

返回。

N/A

例子。

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

上面的代码设置了一个临时的ini来保存来自string "str "的信息，然后在再次关闭之前从ini中读取信息。
