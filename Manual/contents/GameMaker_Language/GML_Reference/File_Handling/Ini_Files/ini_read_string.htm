ini_read_string

你可以使用这个函数从一个ini数据文件中读取string （文本）。Ini文件被分割成若干部分，然后每个部分随后被分割成键-值对。因此，一个典型的ini文件看起来会是这样的。

语法。

ini_read_string(section, key, default);

争论 类型 描述
section 字符串 The section of the .ini to read from.
key 字符串 The key within the relevant section of the .ini to read from.
default 字符串 The string to return if a string is not found in the defined place (or the .ini file does not exist). Must be a string.

返回。

String

例子。

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

这将打开 "savedata.ini "并将global.name设置为其中 "save1">"Name "下的string ，然后再次关闭.ini文件。如果 "save1">"Name "下没有string ，或者没有 "savedata.ini "文件存在，global.name将被设置为 "Player1"。
