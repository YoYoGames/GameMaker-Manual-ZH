伊尼文件

INI文件是小而轻的文件，与大多数平台兼容。它们是存储小块信息的理想选择，如界面偏好、本地高分、关卡数据等......因此是你开始保存游戏信息的一个伟大而简单的方法。如果你正在寻找更高级的方法来保存信息到文件，你可能想看看其他一些文件功能，如文本文件、二进制文件和缓冲器。

重要的是!所有GameMaker项目都有一个名为""的文件。options.ini"，所以你不能用这个名字命名任何文件（当你试图编译你的游戏项目时，你会得到一个UID失败）。

请注意，对于带有本地化的游戏来说，可以从外部文件读取重音字母是非常重要的。这意味着你应该先创建UTF8格式的ini文件，然后把它作为一个包含的文件添加到GameMaker中，这样它就会在运行游戏时被导出，并代替默认的ANSI格式ini文件，如果之前没有提供任何文件，ini函数会创建这个文件。通过这种方式，你可以正确地读写它，所有的重音和非罗马字母都被保留。

ini文件有以下功能。

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
