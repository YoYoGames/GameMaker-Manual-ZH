ini_key_exists

这个函数检查一个键是否存在于当前打开的ini中，如果存在，将返回 true ，否则返回 false 。这不是一个防止错误的必要检查，因为当一个键不存在时，从一个不存在的键中读取将只是返回一个默认值。然而，它对于查看一个ini文件是否保存了特定的数据和其他一些东西是有用的。

语法。

ini_key_exists(section, key);

争论类型 描述
section The section in the open .ini file to check a key in.
key The key to check for.

返回。

例子。

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
    global.name = "Player1";
}
ini_close();

如果不存在 "name "这样的键，这将把变量global.name设置为 "Player1"。
