ini_close

这个函数应该在你完成对任何打开的ini文件的读或写的时候调用。如果你在使用了任何ini写入函数后没有使用该函数，那么将不会有任何东西被写入磁盘，因为文件信息被保留在内存中，直到该函数被调用，从而强制写入。如果你试图打开一个ini，而之前没有关闭另一个（或同一个），你也会得到一个错误。

该函数还将返回一个string ，并将ini文件编码进去。这个string ，然后可以保存到一个server ，和/或与该函数一起再次使用 ini_open_from_string()来重新创建ini文件。

语法。

ini_close();

返回。

String

例子。

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开 "savedata.ini "并从 "save1 "部分读取一个键为 "score "的值，然后关闭ini文件。如果 "save1 "下没有名为 "score "的值，或者没有 "savedata.ini "文件，将返回0并创建一个新的ini文件。
