ini_write_string

你可以使用这个函数将string （文本）写入一个ini数据文件中。Ini文件被分割成若干部分，然后每个部分随后被分割成键-值对。因此，一个典型的ini文件看起来会是这样的。

值得注意的是，在写inistrings 时使用引号和转义字符可能会在以后读回数据时造成问题，因为它可能被过早地截断了。例如，写这个将是一个问题。

ini_write_string("test2", "section", "hello \"Fritz\"");

语法。

ini_write_string(section, key, value);

争论 类型 描述
section 字符串 The section of the .ini to write to.
key 字符串 The key within the relevant section of the .ini to write to.
value 字符串 The string to write to the relevant destination.

返回。

N/A

例子。

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

上述代码打开一个ini文件进行读写，然后将存储在全局变量 "名称 "中的string ，写入键为 "播放器 "的 "保存 "部分，然后再次关闭文件。
