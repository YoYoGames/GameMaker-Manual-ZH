ini_section_exists

这个函数检查当前打开的ini文件中是否存在一个部分，如果存在，将返回 true ，否则返回 false 。这不是一个防止错误的必要检查，因为当一个部分不存在时，从一个不存在的部分读取将只是返回一个默认值，然而它对于查看一个ini文件是否保存了特定的数据很有用。

语法。

ini_section_exists(section);

争论类型 描述
section The section in the open .ini file to check for.

返回。

例子。

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

如果没有找到 "save1"，这将把变量 global.savegame 设置为 "1"。有了forloop ，这可以作为计算一个游戏有多少个保存游戏的基本方法（即：为一个游戏创建一个同时保存多个游戏的系统）。
