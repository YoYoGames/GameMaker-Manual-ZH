ini_open

这将打开一个 ini_file 以供阅读和/或写入。如果在你检查的位置不存在ini_file，GameMaker可能会创建一个，但只有当你向它写入数据时。如果你只从ini文件中读取信息，那么 读取函数的默认值将被返回，但ini文件实际上不会被创建。

请注意，你在任何时候都只能打开一个ini文件，并记得在完成对.ini文件的读写后使用 ini_close()一旦你完成了对.ini文件的读/写，因为在那之前，信息实际上并没有存储到磁盘上（在文件关闭之前，它也存储在内存中）。 磁盘上（在文件关闭之前，它也会存储在内存中）。

警告!由于GameMaker是沙盒式的，这个功能可能不会像你预期的那样工作!更多信息请参见文件系统部分。

语法。

ini_open(name);

争论类型 描述
name The filename for the .ini file.

返回。

例子。

ini_open("Settings/savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开 "savedata.ini"，读取 "save1 "部分的分数值，其中有 "score "键，然后再次关闭.ini文件。如果 "save1 "下没有 "score "的值，或者没有 "savedata.ini "文件，score将被设置为0。 文件，score将被设置为0（默认值）。请注意，该ini文件已被放置在 "设置 "子目录下，该目录是存放Asset 浏览器所含文件中的ini文件的文件夹。
