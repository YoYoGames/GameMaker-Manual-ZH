文件删除

这个函数将从系统中删除指定的文件。需要注意的是，这个函数只删除GameMaker能够创建和解析的那些文件：ini文件、文本文件和二进制文件，或者那些用来存储游戏创建的资源的文件，如sprites 或表面。然而，它不会删除任何其他文件。如果文件被成功删除，该函数还将返回 true ，或者在任何其他情况下返回 false 。

警告!由于GameMaker是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统限制一节以了解更多信息。

语法。

file_delete(fname);

争论类型 描述
fname The name of the file to delete.

返回。

例子。

if file_exists("level.txt")
{
    file_delete("level.txt");
}

这将检查一个文件，如果它存在，就将其删除。

返回。文件系统
