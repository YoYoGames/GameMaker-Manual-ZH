文件系统

在GameMaker中存在一些功能，使你能够以不同的方式处理各种类型的文件。

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

以下功能可用于改变文件名。

注意这些函数并不对实际的文件起作用!它们只处理与不同strings 元素有关的path ，并应与其他文件功能一起使用。

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

最后，有一些特殊的函数用于获取文件名，可以用文件类型过滤器加载和保存文件。这些函数将打开一个资源管理器窗口，并因此导致游戏失去焦点。这意味着游戏所依赖的任何输入可能不会被正确注册，因此我们建议（如果在键盘事件中使用这些函数），你应该使用上键事件，而不是按下或放下事件。

注意这些功能仅适用于Windows和MacOS。

警告：由于GameMaker是沙盒式的，这些功能可能不会像你预期的那样工作。请参阅文件系统一节以了解更多信息。

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
