文件名_名称

使用该函数返回指定文件的名称部分，包括 extension ，但不包括path 。

语法。

filename_name(fname);

争论类型 描述
fname The file to use.

返回。

String

例子。

name = filename_name(file_find_first("C:/Games/*.doc", 0));

上述代码获得了在指定目录中发现的第一个 "doc "类型文件的名称（作为string ）。

返回。文件系统
