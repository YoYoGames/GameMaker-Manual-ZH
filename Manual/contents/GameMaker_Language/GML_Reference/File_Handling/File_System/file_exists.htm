文件存在

如果指定的文件存在，该函数将返回 true ，如果不存在则返回 false 。注意，该函数只能用于检查本地文件，而不能用于检查存储在远程server 的任何文件。

语法。

file_exists(fname);

争论类型 描述
fname The name of the file to check for.

返回。

例子。

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

这将检查一个文件，如果它存在，将被打开进行阅读。

返回。文件系统
