文件属性

你可以用这个函数来检查一个给定文件的属性。你可以把下面的常数加起来，看看你要的文件类型。

恒定 描述
fa_readonly 只读文件
fa_hidden 隐藏的文件
fa_sysfile 系统文件
fa_volumeid 卷标文件
fa_directory 目录
fa_archive 归档文件

注意：这只是一个Windows 的功能。

语法。

file_attributes(fname, attr);

争论 类型 描述
fname 字符串 The name of the file to check.
attr 文件属性常数 The attributes to check for.

返回。

Boolean

例子。

if !file_attributes(file, fa_hidden)
{
    file_delete(file);
}

这将检查一个文件，看它是否被隐藏，如果不是，则被删除。

返回。文件系统
