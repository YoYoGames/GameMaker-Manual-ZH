文件名_修改_文本

该函数返回指定的文件名，并将extension （包括点）改为新的extension 。通过使用一个空的string 作为新的extension ，可以将extension 部分全部删除。

语法。

filename_change_ext(fname, newext);

争论类型 描述
fname The file to use.
newext The new extension to use.

返回。

例子。

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

上述代码得到的文件名（作为string ）中去掉了extension 部分。

返回。文件系统
