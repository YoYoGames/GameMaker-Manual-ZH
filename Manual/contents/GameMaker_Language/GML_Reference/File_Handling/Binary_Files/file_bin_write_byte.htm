file_bin_write_byte

这个函数将把一个字节的数据写到当前写入位置的文件ID所标识的文件中。你提供文件ID值，正如该函数所返回的那样 file_bin_open()和要写入的数据的字节。

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_write_byte(binfile, byte);

争论类型 描述
binfile The ID of the file to write to.
byte The data to write.

返回。

例子。

file_bin_write_byte(file, data);

这将向选定的文件写入一个字节。

返回：二进制文件
