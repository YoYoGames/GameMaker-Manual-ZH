file_bin_read_byte

这个函数将从文件中的当前位置返回一个字节的数据，并给出文件ID。你提供的文件ID值，由函数返回 file_bin_open().

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_read_byte(binfile);

争论类型 描述
binfile The ID of the file to get the data from.

返回。

例子。

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

这将从与游戏相同的目录中打开一个文件，并在再次关闭它之前从其中获取一个字节的数据。

返回：二进制文件
