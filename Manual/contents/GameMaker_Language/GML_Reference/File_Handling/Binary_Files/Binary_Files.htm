二进制文件

在少数情况下，你可能需要从二进制文件中读取数据。这些函数可以用于这个目的，尽管一般来说，我们建议你使用专用的缓冲区函数来创建、保存和加载二进制数据，而不是使用这些函数。

注意：当目标模块为HTML5时，这些功能不起作用。

有以下低级例程可用于此。

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
