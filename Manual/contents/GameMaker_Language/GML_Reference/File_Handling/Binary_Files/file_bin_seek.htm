file_bin_seek

这个函数将文件中的当前读取位置移动到指定位置。你提供文件的ID值，正如函数所返回的那样 file_bin_open()，并且要追加一个文件，在写之前将位置移动到 文件的大小，然后再写入。

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_seek(binfile, pos);

争论类型 描述
binfile The ID of the file to read from.
pos The position in the file to move to.

返回。

例子。

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量 "file"，然后得到文件的大小并将下一个写入位置设置为该大小。

返回：二进制文件
