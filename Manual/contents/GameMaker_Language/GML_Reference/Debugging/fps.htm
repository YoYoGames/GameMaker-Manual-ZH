fps

在GameMaker中，有两种主要方式可以用来告诉你游戏的运行速度。一种是 game speed(在游戏选项中指定）和 fps （每秒帧数）。这些值经常被混淆，但基本上一个是GameMaker应该在一秒钟内完成的游戏步骤，而另一个（fps）是GameMaker在一秒钟内实际完成的CPU步骤，最高值为room 速度本身。要获得真正的fps，即每个游戏步骤的实际cpu步数，使用 fps_real变量。

这个只读变量以整数形式返回当前的fps值。请注意，这个函数在你的游戏中每走一步都会更新一次，所以可能会出现从一个值到另一个值的 "跳跃"，但这是很正常的。

语法。

fps

返回。

例子。

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

上面的代码将检查游戏是否处于调试模式，如果是，它将在屏幕上显示当前的fps。
