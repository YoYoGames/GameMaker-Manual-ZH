fps_real

在GameMaker中，有两种主要方式可以用来告诉你游戏的运行速度。一种是 game speed(在游戏选项中指定）和 fps （每秒帧数）。这些数值经常被混淆，但基本上一个是GameMaker应该在一秒钟内完成的游戏步骤数（游戏速度），而另一个是GameMaker在一秒钟内实际完成的CPU步骤数（真正的fps），这个数值通常比游戏速度高得多，但会随着你的游戏变得更加复杂，使用更多的处理能力来维持设定的room 。

这个只读变量以整数形式返回当前的fps值。请注意，这个函数在你的游戏中每走一步都会更新一次，所以可能会出现从一个值到另一个值的 "跳跃"，但这是很正常的。

注意：在HTML5目标上，这个变量将返回显示器刷新率的倍数，因为GameMaker必须依靠浏览器来计时和发送。

语法。

fps_real

返回。

例子。

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps_real));
}

上面的代码将检查游戏是否处于调试模式，如果是，它将在屏幕上显示当前的真实帧数。

返回。调试
