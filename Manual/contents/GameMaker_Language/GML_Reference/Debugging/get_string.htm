get_string

这将创建一个弹出式窗口，显示一条标准信息，有一个标有 "OK "的按钮，提示用户输入一个string 。该函数将返回输入的string ，如果没有输入，则返回默认值。

注意这个函数只在Windows 目标上用于调试，但在所有其他目标上已被废弃。

语法。

get_string(str, def);

争论 类型 描述
str 字符串 The string to show in the pop-up message.
def 字符串 The default string in the text box.

返回。

String

例子。

global.test_name = get_string("Test highscore name:", "Anonymous");

上述代码将提示用户给出一个名字，然后将其存储在全局变量 "test_name "中。如果没有输入任何信息，用户只是按了 "OK"，那么将返回默认值 "匿名"。
