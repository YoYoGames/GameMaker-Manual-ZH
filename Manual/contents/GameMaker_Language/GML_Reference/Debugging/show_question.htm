显示_问题

这个函数创建了一个弹出式消息框，有 "是 "和 "否 "的两个按钮。它返回 true 或 false ，取决于用户按两个按钮中的哪一个。

注意这个函数只在Windows 目标上用于调试，但在所有其他目标上已被废弃。

语法。

show_question(str);

争论 类型 描述
str 字符串 The string to show in the pop-up question.

返回。

Boolean

例子。

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

上述代码将检查分数，如果超过500分，它将询问用户是否希望继续，如果点击 "是 "按钮，它将进入另一个room ，但如果选择 "否 "按钮，它将结束游戏。
