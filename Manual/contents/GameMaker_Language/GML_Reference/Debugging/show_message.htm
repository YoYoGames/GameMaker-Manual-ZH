显示_消息

该函数创建一个弹出式消息框，显示给定的string ，并有一个标有 "OK "的按钮来关闭它。

注意这个函数只在Windows 目标上用于调试，但在所有其他目标上已被废弃。

语法。

show_message(str);

争论 类型 描述
str 字符串 The string to show in the pop-up message.

返回。

N/A

例子。

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

上述代码将loop ，通过存储在数组 "inv "中的值，并将它们添加到变量 "tot "中，然后显示一个包含总数的消息。
