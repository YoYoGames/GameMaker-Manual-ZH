get_integer

这将创建一个弹出式窗口，显示一个自定义信息，并有一个标有 "OK "的按钮，提示用户输入一个整数值。该函数将返回输入的整数，如果没有输入，则返回默认值。

注意这个函数只在Windows 目标上用于调试，但在所有其他目标上已被废弃。

语法。

get_integer(str, def);

争论 类型 描述
str 字符串 The string to show in the pop-up message.
def 真实的 The default value in the text box.

返回。

Real

例子。

global.level = get_integer("Level to test?", 1);

上述代码将显示一条信息，提示用户选择一个级别进行测试。返回值将被存储在全局变量 "global.level "中。
