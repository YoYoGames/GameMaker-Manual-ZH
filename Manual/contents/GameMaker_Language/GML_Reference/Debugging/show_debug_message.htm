show_debug_message

这个函数将创建一个自定义的调试信息，在编译器窗口中显示：runtime 。请注意，该信息可以是一个string ，也可以是一个实数，但如果你需要这两个数字，则必须先用函数将其转换为string 。 string()函数转换为（见下面的例子），如果数字有两个以上的小数位，那么你应该用 string_format()来显示它们，因为默认情况下，小数被四舍五入到最接近的两个小数位（所以 "1.2468 "在输出窗口中显示为 "1.25"）。

用这个函数显示的调试信息将显示在IDE 底部的编译器输出窗口，以及在调试模式下运行游戏时debugger 的图形视图中。如果你只想在调试模式下看到信息，那么你可能应该使用 debug_event()来代替。

语法。

show_debug_message(string);

争论类型 描述
string The custom debug message to show.

返回。

例子。

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

上述代码检查实例是否存在，如果不存在，编译窗口会显示一条调试信息并结束游戏。
