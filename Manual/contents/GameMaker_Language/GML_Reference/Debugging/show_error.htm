show_error

这个函数将显示一个自定义的string 作为错误信息。

重要提示第二个参数的存在只是为了向后兼容老项目，对显示的错误对话框没有影响。

注意该功能仅用于调试。

语法。

show_error(str, abort);

争论 类型 描述
str 字符串 The string to show in the pop-up message.
abort 布尔型 UNUSED 该错误是否应该中止游戏（真）或允许玩家忽略它（假）。

返回。

N/A

例子。

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

上面的代码将检查当前room 是否是rm_Dungeon，如果不是，将显示错误信息。

返回。调试
