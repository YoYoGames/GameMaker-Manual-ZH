Network_send_raw

通过这个函数，你可以通过网络发送一个 "原始 "数据包。该函数需要通过套接字ID进行连接，然后你必须提供包含要发送的数据的缓冲区ID（关于buffers 的更多信息，见参考-缓冲区），最后是数据包的大小（字节）。发送的数据不会被GameMaker以任何方式格式化，接收设备将获得数据流，这意味着你必须自己处理它。该函数将返回发送的数据字节数，如果发送失败，则返回一个小于0的数字。

选项争论

最后一个参数是可选的，并且只用于WebSockets。它允许你选择发送二进制或文本数据。这两个常数中的任何一个都可以在这个参数中指定。

网络发送类型常数
恒定 描述
network_send_binary 发送一个二进制信息
network_send_text 发送短信

一些平台的API在使用WebSockets时只接受文本信息（例如Twitch），所以在这种情况下可以使用network_send_text常量。如果没有指定这个参数，默认情况下会发送二进制数据。

语法。

network_send_raw(socket, bufferid, size, [options]);

争论 类型 描述
socket 网络插座ID The id of the socket to use.
bufferid 缓冲区ID The id of the buffer to get the data from.
size 真实的 The size (in bytes) of the data.
options 网络发送类型常数 OPTIONAL 用于WebSockets在文本和二进制数据之间进行选择；如果不指定，则发送二进制数据。

返回。

Real

例子。

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

上述信息将先前保存的buffer 数据加载到内存中，并返回buffer id，将其存储在变量 "buff "中。这个完整的buffer ，然后使用变量 "sock "标识的套接字在网络上作为原始数据包发送。
