网络_创建_服务器_raw

这个函数用来为你的游戏创建一个新的网络server ，使用允许的连接协议之一（见下面列出的常数）。你提供server ，然后给它一个使用的端口，最后是在任何时候都应该允许的最大连接数server （注意，这个数字由你决定，但是太多的连接客户端会使网络饱和，或者设备的CPU无法处理这么多的玩家，所以要谨慎使用）。该函数返回一个唯一的ID，该ID应该用来存储在一个变量中，并在所有进一步的网络功能中用来识别server ，如果连接失败则返回一个小于0的值。

由于这创建了一个 "原始 "server ，它将不接受也不使用任何内置的GameMaker 数据头进行通信，因此你应该使用函数 network_send_raw()和 network_send_udp_raw()来发送未格式化的数据到创建的server 。

语法。

network_create_server_raw(type, port, max_client);

争论类型 描述
type The type of server to create (see the constants listed below).
port The port that the server will use.
max_client The maximum number of clients that can connect at once.

返回。

Network Socket ID

例子。

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

上面的代码将尝试使用TCP通过6510端口创建一个server 。如果该端口不可用，它将通过loop ，找到一个可用的端口。

返回。联网
