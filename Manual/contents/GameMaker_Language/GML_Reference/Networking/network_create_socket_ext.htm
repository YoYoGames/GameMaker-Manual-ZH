网络_创建_socket_ext

这个函数用来为你的游戏创建一个新的客户端套接字，以便通过网络进行通信。你必须定义套接字类型（见下面的常量列表），并给出一个要使用的端口，该函数将返回一个唯一的ID，该ID应在该套接字的所有后续函数调用中使用，如果连接失败则返回一个小于0的值。

语法。

network_create_socket_ext(protocol, port);

争论 类型 描述
protocol 插座类型恒定 The network protocol to use
port 真实的 The port to use

返回。

Network Socket ID

例子。

client = network_create_socket_ext(network_socket_udp, 6510);

上面的代码将在6510端口创建一个新的UDP套接字。

返回。联网
