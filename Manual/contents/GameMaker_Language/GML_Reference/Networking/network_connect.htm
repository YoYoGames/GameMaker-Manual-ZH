网络_连接

通过这个函数，你可以发送一个连接到server 的请求。这个函数接收要通过的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址(string)以及要连接的端口，如果连接失败，将返回一个小于0的值。连接使用了一个特殊的协议，确保只有GameMaker游戏可以相互连接，然而如果你需要连接到一个不是GameMaker游戏的server ，你可以使用 network_connect_raw().注意，在默认情况下，该功能是同步的，这意味着你的游戏在连接时可能会出现 "挂起"。你可以为连接设置一个超时值，或者使用函数 network_set_config() ，或者使用函数 network_connect_async() ，使创建异步。

注意你不能在HTML5上使用这个函数。对于WebSockets，请使用Async函数。

语法。

network_connect(socket, url, port);

争论 类型 描述
socket 网络插座ID The id of the socket to use.
url 字符串 The URL or IP to connect to (a string).
port 真实的 The port to connect to.

返回。

Real or Network Socket ID

例子。

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

上述代码将创建一个新的TCP套接字，然后试图通过该套接字连接到6510端口的指定IP地址。

返回。联网
