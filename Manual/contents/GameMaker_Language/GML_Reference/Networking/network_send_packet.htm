network_send_packet

通过这个函数，你可以通过网络发送一个数据 "包"。该函数接收要连接的套接字ID，然后你必须提供包含要发送的数据的缓冲区ID（关于buffers 的更多信息，见参考-缓冲区），最后是数据包的大小（字节）。用这个函数发送的数据包的格式是这样的：接收数据的GameMaker游戏可以正确地 "分割 "数据包，这个函数将返回发送的数据字节数，如果发送失败则返回一个小于0的数字。值得注意的是，这个函数返回的正在发送的数据的最终大小也将包括GameMaker 头信息，这是一个额外的12字节。

语法。

network_send_packet(socket, bufferid, size);

争论类型 描述
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

返回。

Real

例子。

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

上述信息将先前保存的buffer 数据加载到内存中，并返回buffer id，将其保存在变量 "buff "中。这个完整的buffer ，然后使用变量 "sock "标识的套接字作为数据包在网络上发送。

返回。联网
