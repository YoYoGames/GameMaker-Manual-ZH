network_create_server

这个函数用来为你的游戏创建一个新的网络server ，使用允许的连接协议之一（见下面列出的常数）。你提供server ，然后给它一个使用的端口，最后是在任何时候都应该允许的最大连接数server （注意，这个数字由你决定，但是太多的连接客户端会使网络饱和，或者设备的CPU无法处理这么多的玩家，所以要小心使用）。该函数返回一个唯一的ID，该ID应该用来存储在一个变量中，并在所有进一步的网络功能中用来识别server ，如果连接失败则返回一个小于0的值。

语法。

network_create_server(type, port, max_client);

争论 类型 描述
type 插座类型恒定 The type of server to create (see the constants listed above).
port 真实的 The port that the server will use.
max_client 真实的 The maximum number of clients that can connect at once.

返回。

Network Socket ID

例子。

var port = 6510;
server = network_create_server(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

上面的代码将尝试使用TCP通过6510端口创建一个server 。如果该端口不可用，它将通过loop ，找到一个可用的端口。

返回。联网
