network_connect_async

通过这个函数，你可以发送一个连接到server 的请求。该函数接收要连接的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址()以及要连接的端口。 string ）以及连接的端口，如果连接失败，将返回一个小于0的值。该连接使用了一个特殊的协议，确保只有GameMaker游戏可以相互连接，但是如果你需要 连接到一个不是GameMaker游戏的server ，你可以使用 network_connect_async_raw().注意，这个函数是异步的，生成一个类型为network_type_non_blocking_connect 的异步网络事件。

语法。

network_connect_async(socket, url, port);

争论类型 描述
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

返回。

例子。

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

上面的代码将创建一个新的TCP套接字，然后试图通过该套接字建立一个异步连接到6510端口的指定IP地址。
