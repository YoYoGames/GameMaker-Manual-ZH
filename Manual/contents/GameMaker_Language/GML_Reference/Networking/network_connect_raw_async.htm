网络_connect_raw_async

通过这个函数，你可以发送一个连接到server 的请求。该函数接收要连接的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址()以及要连接的端口。 string ）以及连接的端口，如果连接失败，将返回一个小于0的值。这个函数和 network_connect_async()的区别是，这个 函数可以连接到任何server ，对原始数据不做任何处理，这意味着你必须在server ，自己实现协议。注意，这个函数是异步的，生成一个类型为network_type_non_blocking_connect 的异步网络事件。

语法。

network_connect_raw_async(socket, url, port);

争论类型 描述
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

返回。

例子。

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

上述代码将创建一个新的TCP套接字，然后试图通过它创建一个 "原始 "异步连接到6510端口的给定URL。
