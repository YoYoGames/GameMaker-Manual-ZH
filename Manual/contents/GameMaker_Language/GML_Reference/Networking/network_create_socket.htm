network_create_socket

这个函数用来为你的游戏创建一个新的客户端套接字，以便通过网络进行通信。你必须定义套接字类型（见下面的常量列表），该函数将为该套接字返回一个唯一的ID，该ID应在该套接字的所有后续函数调用中使用，如果连接失败则返回一个小于0的值。

提示你可以使用IP "127.0.0.1 "来连接回正在运行游戏的同一设备。

语法。

network_create_socket(type);

争论 类型 描述
type 插座类型恒定 The type of socket connection to create (see the constants listed above).

返回。

Network Socket ID

例子。

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

上述代码将检查游戏是否在浏览器中运行，并创建一个新的TCP或Web套接字，然后尝试通过该套接字连接到指定端口的指定IP地址。

返回。联网
