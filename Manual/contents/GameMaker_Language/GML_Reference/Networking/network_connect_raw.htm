网络_connect_raw

通过这个函数，你可以发送一个连接到server 的请求。这个函数接收要通过的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址(string)以及要连接的端口，如果连接失败，将返回一个小于0的值。这个函数和 network_connect()的区别是，这个函数可以连接到任何server ，对原始数据不做任何处理，这意味着你必须在server 端自己实现协议。注意，默认情况下该函数是同步的，这意味着你的游戏在连接时可能会出现 "挂起"。你可以为连接设置一个超时值，或者使用函数使创建过程异步化。 network_set_config()或者使用函数 network_connect_async_raw()来代替。

注意你不能在HTML5上使用这个函数。对于WebSockets，请使用Async函数。

语法。

network_connect_raw(socket, url, port);

争论 类型 描述
socket 网络插座ID The id of the socket to use.
url 字符串 The URL or IP to connect to (a string).
port 真实的 The port to connect to.

返回。

Real or Network Socket ID

例子。

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

上述代码将创建一个新的TCP套接字，然后试图通过该套接字创建一个 "原始 "连接到6510端口的指定URL。

返回。联网
