可点击_添加

通过这个函数，你可以创建一个自定义的、可点击的图标，作为游戏画布之外的 "按钮"，但在浏览器窗口本身。该函数将返回一个ID值，这个ID值必须存储在一个变量中，以便在所有进一步引用新的可点击的按钮函数中使用。这个函数非常有用，可以防止浏览器在点击按钮时创建一个弹出式窗口，因为这一切都通过GameMaker本身完成。

你必须首先设置位置，这是相对于浏览器窗口的（0,0）（左上角），然后给texture 页面条目，你希望按钮图标有sprite （见。 sprite_get_tpe()).然后你可以指定一个要打开的URL或一个要打开或运行的script （来自GameMaker的Asset Browser）。你指定的 "目标 "参数与你使用 open() 方法时的标准JavaScript "名称 "值相同（请注意，除了'_self'以外的所有参数可能会导致浏览器屏蔽，或询问用户是否愿意允许），而 "params "参数与控制新窗口/标签应显示哪些属性的JavaScript "规格 "参数相同（并非所有浏览器都可能支持所有功能）。如果你使用图标来运行一个内部script ，这最后两个参数可以留作空strings ""。

有效的目标是。

目标 描述
_blank 在一个新的窗口或标签中打开链接的文件。
亲自出马 在被点击的同一框架内打开链接的文件（这是默认的）。
父/母 打开父框架中的链接文档。
_top 在整个窗口中打开链接的文件。

有效参数为：。

参数 描述
'高度=[px]' 窗口的高度，最小值为100。
'宽度=[px]' 窗口的宽度，最小值为100。
'left=[px]' 窗口的左边位置。
'top=[px]' 窗口的顶部位置（仅限IE）。
'location=[boolean]' 是否显示地址栏（默认为1）。
'menubar=[boolean]' 是否显示菜单栏（默认为1）。
'resizable=[boolean]' 窗口是否可以调整大小（默认为1）。
'scrollbars=[boolean]' 是否显示滚动条（默认为1）。
'status=[boolean]' 是否添加状态栏（默认为1）。
'titlebar=[boolean]' 是否显示标题栏。除非调用的应用程序是一个HTML应用程序或一个受信任的对话框，否则这一点会被忽略（默认为1）。
'toolbar=[boolean]' 是否显示浏览器工具条（默认为是）。


还值得注意的是，如果你传入的URL参数是一个脚本函数的名称（即：不是以 "http://"开头），并且script 被称为 gmcallback_* ，那么你可以直接进入html5事件。如果GameMaker的脚本函数名称以 gmcallback_ 开始，则不会被混淆。

语法。

clickable_add(x, y, tpe, url, target, params);

争论 类型 描述
x 真实的 The x position within the window.
y 真实的 The y position within the window.
tpe 纹理页条目 The texture page entry for the sprite to be used.
url 字符串 The URL (website address) to link to, or the script to be run.
target 字符串 This is the target area to open the URL in (see description).
params 字符串 The various parameters for the icon (see description).

返回。

Clickable ID

例子。

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

上述代码在游戏画布运行的页面的位置（32，32）创建了一个可点击的DOM图标。sprite 该图标使用从texture 页面引用的 "spr_MS_Home"，当点击该图标时，将为指定的URL打开一个新窗口，并为该窗口定义属性。
