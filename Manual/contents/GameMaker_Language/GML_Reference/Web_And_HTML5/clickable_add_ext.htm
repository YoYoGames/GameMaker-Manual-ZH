可点击_添加_文本

通过这个函数，你可以创建一个自定义的、可点击的图标，作为游戏画布之外的 "按钮"，但在浏览器窗口本身。该函数将返回一个ID值，这个ID值必须存储在一个变量中，以便在所有进一步引用新的可点击的按钮函数中使用。这个函数非常有用，可以防止浏览器在点击按钮时创建一个弹出式窗口，因为这一切都通过GameMaker本身完成。

这个函数的工作原理与 clickable_add()texture 函数，你必须设置相对于浏览器窗口（0,0）的位置，给出你希望按钮图标具有的sprite 页面条目（见。 sprite_get_tpe())，指定一个要打开的URL或一个要打开或运行的script （来自GameMaker的Asset 浏览器），然后设置要使用的目标和参数。然而，这个函数还有两个额外的选项，允许你设置图标的alpha从0（完全透明）到1（完全不透明），以及图标的比例，这是一个应用于sprite 的乘数值。因此，1的值是默认的比例，而（例如）0.5将是一半的比例，2将是两倍。

还值得注意的是，如果你传入的URL参数是一个脚本函数的名称（即：不是以 "http://"开头），并且script 被称为 gmcallback_* ，那么你可以直接进入html5事件。如果GameMaker的脚本函数名称以 gmcallback_ 开始，则不会被混淆。

语法。

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

争论 类型 描述
x 真实的 The x position within the window.
y 真实的 The y position within the window.
tpe 纹理页条目 The texture page entry for the sprite to be used.
url 字符串 The URL (website address) to link to, or the script to be run.
target 字符串 This is the target area to open the URL in (see description).
params 字符串 The various parameters for the icon (see description).
alpha 真实的 The image alpha of the icon (default 1).
scale 真实的 The scale of the icon (default 1).

返回。

Clickable ID

例子。

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

上面的代码在游戏画布运行的页面的位置（32，32）创建了一个可点击的DOM图标。sprite 该图标使用从texture 页面引用的 "spr_MS_Home"，当点击该图标时，将为指定的URL打开一个新窗口，并为该窗口定义属性。该按钮也将被缩放为原始sprite 大小的两倍，图像alpha为1的一半。
