启用webgl_enabled

这个只读变量将返回你的游戏是否启用了WebGL (true)或没有(false)。它只对那些通过浏览器运行的游戏有效（即：HTML5），对于所有其他平台，它将返回true。

语法。

webgl_enabled

返回。

例子。

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

上述代码检查WebGLflag ，然后相应地设置全局变量 "质量"。
