可点击_存在

这个函数返回一个可点击的DOM图标是否已经创建，指定的索引是否存在。请注意，用于检查的值必须在之前被初始化过，否则你会得到一个错误，导致GameMaker关闭。

语法。

clickable_exists(index);

争论 类型 描述
index 可点击的ID The index of the clickable to check.

返回。

Boolean

例子。

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

上面的代码检查变量 "home_but "是否已经存在，如果不存在，就在游戏画布运行的页面的位置（32，32）创建一个可点击的DOM图标。sprite 该图标使用从texture 页面引用的 "spr_MS_Home"，当点击该图标时，将为指定的URL打开一个新窗口，并为该窗口定义属性。
