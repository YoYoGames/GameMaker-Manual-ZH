浏览器_输入_捕捉

通过这个函数，你可以设置浏览器窗口是否应该捕捉所有的输入（设置为 false ）或者游戏是否应该捕捉输入（设置为 true ）。注意，这个函数只适用于HTML5模块的使用。

语法。

browser_input_capture(enable);

争论 类型 描述
enable 布尔型 Enable or disable game input capture

返回。

N/A

例子。

browser_input_capture(true);

上述代码将游戏设置为捕捉HTML5目标平台上的所有浏览器输入。
