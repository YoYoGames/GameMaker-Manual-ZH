clickable_set_style

这个函数让你通过提供的键/值对为给定的按钮设置CSS样式属性。 DS Map.你需要在之前创建按钮元素（使用 clickable_add())和 DS Map ，并提供每个元素的存储索引作为参数。

语法。

clickable_set_style(index, map)

争论 类型 描述
index 可点击的ID The index of the clickable icon to style.
map DS地图ID The index of the DS Map 来设置样式。

返回。

N/A

例子。

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

上面的代码将改变可点击按钮的样式，其索引存储在变量 "button "中。
