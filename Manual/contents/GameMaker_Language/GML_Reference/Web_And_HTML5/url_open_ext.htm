url_open_ext

这将在所选择的目标设备的浏览器上打开指定的URL，或者，如果你使用HTML5模块，则在当前打开的浏览器中打开。当你使用 open() 方法时，你指定的 "目标 "参数与标准的JavaScript "名称 "值相同，你应该注意，除了"_self"之外，其他都可能导致浏览器屏蔽，或询问用户是否愿意允许。

有效的目标是。

目标 描述
_blank 在一个新的窗口或标签页中打开链接的文件（如果用户阻止了弹出式窗口，这将不起作用，在这种情况下，你可以使用 clickable_*功能代替）。
亲自出马 在被点击的同一框架内打开链接的文件（这是默认的）。
父/母 打开父框架中的链接文档。
_top 在整个窗口中打开链接的文件。

语法。

url_open_ext(url, target);

争论 类型 描述
url 字符串 The URL (website address) to link to.
target 字符串 This is the target area to open the URL in.

返回。

N/A

例子。

url_open_ext("http://gamemaker.io", "_blank");

这将在一个新窗口中打开GameMaker的主页。
