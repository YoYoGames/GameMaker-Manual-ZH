url_open_full

这将在选定的目标设备的浏览器上打开指定的URL，或者，如果你使用HTML5模块，则在当前打开的浏览器中打开。你指定的 "目标 "参数与你使用 open() 方法时的标准JavaScript "名称 "值相同（请注意，除"_self"外的所有参数可能会导致浏览器屏蔽，或询问用户是否愿意允许），"选项 "与JavaScript "规格 "参数相同，用于控制新窗口/标签应显示哪些属性（并非所有浏览器都可能支持所有功能）。

有效的目标是。

目标 描述
_blank 在一个新的窗口或标签页中打开链接的文件（如果用户阻止了弹出式窗口，这将不起作用，在这种情况下，你可以使用 clickable_*功能代替）。
亲自出马 在被点击的同一框架内打开链接的文件（这是默认的）。
父/母 打开父框架中的链接文档。
_top 在整个窗口中打开链接的文件。

有效的选项是。

参数 描述
'高度=[px]' 窗口的高度，最小值为100。
'宽度=[px]' 窗口的宽度，最小值为100。
'left=[px]' 窗口的左边位置。
'top=[px]' 窗口的顶部位置（仅限IE）。
'location=[boolean]' 是否显示地址栏（默认为1）。
'menubar=[boolean]' 是否显示菜单栏（默认为1）。
'resizable=[boolean]' 窗口是否可以调整大小（默认为1）。
'scrollbars=[boolean]' 是否显示滚动条（默认为1）。
'status=[boolean]' 是否添加状态栏（默认为1）。
'titlebar=[boolean]' 是否显示标题栏。除非调用的应用程序是一个HTML应用程序或一个受信任的对话框，否则这一点会被忽略（默认为1）。
'toolbar=[boolean]' 是否显示浏览器工具条（默认为是）。

语法。

url_open_full(url, target, options);

争论 类型 描述
url 字符串 The URL (website address) to link to.
target 字符串 This is the target area to open the URL in (see description).
options 字符串 Standard browser options (see description).

返回。

N/A

例子。

url_open_full("http://gamemaker.io", "_blank", "resizable=0, height=200, scrollbars=0");

这将在一个不能调整大小的新窗口中打开GameMaker主页，高度为200像素，没有滚动条。
