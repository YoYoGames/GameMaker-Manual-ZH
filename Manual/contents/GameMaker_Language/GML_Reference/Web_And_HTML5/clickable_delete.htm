可点击_删除

这个函数必须被用来从游戏窗口中移除之前用 clickable_add()创建的可点击图标。

语法。

clickable_delete(index);

争论 类型 描述
index 可点击的ID Index of the clickable icon to remove.

返回。

N/A

例子。

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

上述代码将全局变量 "Help_Icon "中索引的可点击图标从游戏窗口中移除，如果它存在的话。
