可点击的变化_文本

通过这个功能，你可以改变以前用以下方法创建的可点击图标的sprite 和位置 clickable_add().请记住，位置是基于窗口，而不是画布，(0,0)的位置，而且sprite ，必须直接从texture 页面引用（见。 sprite_get_tpe()).这个函数还允许你改变所使用的sprite 的比例（作为一个乘数，所以1是默认的，0.5是一半，2是两倍），以及屏幕上最终图标的alpha值从0（完全透明）到1（完全不透明）。

语法。

clickable_change_ext(index, tpe, x, y, alpha, scale)

争论 类型 描述
index 可点击的ID The index of the clickable icon to change.
tpe 纹理页条目 The texture page entry for the sprite to be used
x 真实的 The new x position within the window.
y 真实的 The new y position within the window.
scale 真实的 The scale of the icon (default 1).
alpha 真实的 The image alpha of the icon (default 1).

返回。

N/A

例子。

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

上述代码将根据当前room ，改变变量 "global.Help "中索引的可点击图标的图像索引和位置，同时改变图标的比例和alpha。
