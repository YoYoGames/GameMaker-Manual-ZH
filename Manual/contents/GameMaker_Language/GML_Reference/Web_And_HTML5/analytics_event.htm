分析_事件

这个函数将发送指定的文本到你通过HTML5游戏选项设置的分析提供者。这个函数可以用来创建一个自定义分析，以跟踪正在使用的供应商范围之外的东西。

语法。

analytics_event(string);

争论类型 描述
string A string to send to the provider

返回。

例子。

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

上述代码将检查一个变量，看它是否是 true ，如果是，那么将触发一个特殊的分析事件，并指定string 。
