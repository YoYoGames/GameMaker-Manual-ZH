device_mouse_dbclick_enable

这个功能可以用来设置设备检测双击 mb_left （鼠标左键）作为 mb_right （鼠标右键）的输入。

默认情况下，这被设置为 true ，这意味着每次用户连续快速点击设备屏幕两次时，返回值与点击鼠标右键的情况相同。当这一点被打开时，第一次点击将被检测为 mb_left ，第二次为 mb_right ，所以确保你使用的任何代码都考虑到这一点。

语法。

device_mouse_dbclick_enable(bool);

争论 类型 描述
bool 布尔型 Set double-click detection on (true) 或关闭(false)。

返回。

Boolean

例子。

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

上面的代码检查运行游戏的设备是Windows PC还是Mac，如果是其中之一，它就会禁用双击功能。

返回。设备输入
