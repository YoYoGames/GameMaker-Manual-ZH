device_mouse_x_to_gui

这个函数返回设备上触摸的X位置与GUI层的关系。由于GUI层可以被设置为独立于屏幕分辨率和 room 的不同尺寸，所以有时要获得用户触摸屏幕时在GUI层上的准确位置是很复杂的。有了这个函数，你可以根据实际的GUI尺寸得到这个位置，使GUI按钮和HUD元素更容易创建和互动。如果你在HTML5或PC和Mac模块上运行这个功能，那么只要设备（通常是鼠标）被插入，这个值就会持续更新，但是对于移动设备，这个值只会在屏幕被触摸时更新，注意，可以检测到的最大触摸次数在很大程度上取决于使用的设备和运行的操作系统。

语法。

device_mouse_x_to_gui(device);

争论类型 描述
device The device (from 0 - n) that is being checked.

返回。

例子。

if device_mouse_check_button(0, mb_left)
{
    if (device_mouse_x_to_gui(0) > 0 && device_mouse_x_to_gui(0) < 32)
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

上面的代码检查设备的鼠标是否被按下，如果是，它就在GUI层上轮询设备的X位置，看它是否在参数之内。如果是，它将变量 "pressed "设置为 true ，否则将设置为 false 。
