设备_is_keypad_open

这个功能是检查设备是否有键盘，如果有的话，返回 true ，否则返回 false 。请注意，这个功能主要是用于Android 设备。

对于那些使用索尼Xperia Play的用户，在GameMaker内有一个设定的按钮/按键地图设置，所以你可以使用键盘常量 vk_up,vk_down,vk_left,vk_right作为操纵板按键，三角形是 ord("T") ，方形是 ord("S") ，圆形是vk_alt+vk_backspace，而返回按钮只是 vk_backspace ，十字是vk_space，选择是vk_return，开始是vk_rshift，L触发器是 ord("L") ，R触发器是 ord("R") 。

语法。

device_is_keypad_open

返回。

例子。

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

上面的代码检查一个键盘，然后根据返回的值改变一个全局变量。
