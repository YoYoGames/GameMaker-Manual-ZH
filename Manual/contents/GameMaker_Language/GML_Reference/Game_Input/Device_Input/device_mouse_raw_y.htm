设备_鼠标_raw_y

这个函数返回设备上触摸的原始y位置。这意味着它返回被触摸的实际设备定义的y位置，而不是 GameMaker的定义，因此将忽略诸如视图位置和缩放的东西。请注意，可以检测到的最大设备触摸次数在很大程度上取决于所使用的设备和它所运行的操作系统。

注意：这个函数在很大程度上取决于设备，你应该先用所需的目标模块和设备进行实验，看看到底会返回什么。

语法。

device_mouse_raw_y(device);

争论类型 描述
device The device (from 0 - n) that is being checked.

返回。

例子。

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));
    y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1));
}

上面的代码检查设备1和设备2是否被按下，如果被按下，则计算每个按压点之间的x/y坐标的平均位置。
