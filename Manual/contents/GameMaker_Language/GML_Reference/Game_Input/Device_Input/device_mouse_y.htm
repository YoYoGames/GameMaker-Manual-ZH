设备_mouse_y

这个函数返回设备上的触摸的y位置。如果你在HTML5或PC和Mac模块上运行这个函数，那么只要设备（通常是鼠标）被插入，这个值就会持续更新，但是对于移动设备，这个值只会在屏幕被触摸时更新，并且注意，可以检测到的最大触摸次数在很大程度上取决于正在使用的设备和它所运行的操作系统。

语法。

device_mouse_y(device);

争论类型 描述
device The device (from 0 - n) that is being checked.

返回。

例子。

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

上面的代码检查设备是否被按下，如果是，它就会轮询设备的Y位置，看它是否在参数范围内。如果是，它将变量 "pressed "设置为 true ，否则将设置为 false 。
