设备_鼠标_检查_按钮_按下

这个函数返回 true 或 false ，取决于你指定的设备是否被 "触摸"（点击）了。设备参数指的是触摸数，可以从0到n，可以检测到的最大触摸数在很大程度上取决于所使用的设备和它所运行的操作系统（大多数设备至少支持4次同时触摸）。这个函数只被实际的按压动作触发一次 ，本页列出的常数可以用来检查鼠标按键。请注意， mb_right ，只有在检测到双击触摸并在第二次点击时保持的情况下才会被检测到（这个行为可以用函数 device_mouse_dbclick_enable())。

语法。

device_mouse_check_button_pressed(device, button);

争论类型 描述
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

返回。

例子。

if device_mouse_check_button_pressed(0, mb_left)
{
    press = true;
}

上面的代码检查设备是否被按下，如果被按下则将一个变量设置为真。

返回。设备输入
