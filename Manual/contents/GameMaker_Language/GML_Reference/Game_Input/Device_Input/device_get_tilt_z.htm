装置_获取倾斜度_z

这个函数根据设备的 "倾斜 "角度返回一个在-1和1之间的值。倾斜度和返回值之间的实际关联性取决于设备和它所使用的操作系统，但一般来说，1或-1的值与+/-90°相同。 -1与+/-90°是一样的。下面的图片显示了每个可用功能与设备的关系。

语法。

device_get_tilt_z()

返回。

例子。

y += sign(device_get_tilt_z());

上面的代码使用倾斜值沿y轴移动玩家。
