鼠标清空

这个函数将清除给定的鼠标按钮的当前状态。这意味着对它是否被按住的检查将不会返回 true ，直到玩家释放按钮并再次按下它（但如果在按住鼠标按钮时进行清除，则仍然会检测到释放状态）。

你可以提供本页所列的一个鼠标按键常数。

语法。

mouse_clear(button);

争论 类型 描述
button 鼠标按钮常数 Which mouse button constant to check for.

返回。

Boolean

例子。

mouse_clear(mb_any);

上面的代码将清除所有鼠标按钮的下降状态。

返回。鼠标输入
下一步。 mouse_lastbutton
