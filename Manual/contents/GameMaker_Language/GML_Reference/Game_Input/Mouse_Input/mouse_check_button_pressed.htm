鼠标_检查_按钮_按下

如果被检查的鼠标按钮被按下，这个函数将返回 true ，如果没有，则返回 false 。对于任何一个鼠标按钮，当它第一次被按下时，这个函数将只被触发一次，要再次触发它，该按钮需要被释放并再次被按下。请注意，在该步骤的持续时间内，它将被认为是被触发的，对于有任何鼠标事件或使用该相同函数的所有实例，也是如此。

你提供的鼠标按钮要从以下常数之一检查。

语法。

mouse_check_button_pressed(numb);

争论 类型 描述
numb 鼠标按钮常数 Which mouse button constant to check for.

返回。

Boolean

例子。

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

上面的代码将检查鼠标左键是否被按下，如果被按下，它将在分数上增加50分。

返回。鼠标输入
下一步。 mouse_check_button_released
