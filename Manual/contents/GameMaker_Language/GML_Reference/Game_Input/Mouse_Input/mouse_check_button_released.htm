鼠标_检查_按钮_释放

如果被检查的鼠标按钮已经被释放，这个函数将返回 true ，如果没有，则返回 false 。对于任何一个鼠标按钮，当它被释放时，这个函数将只被触发一次，要再次触发这个函数，需要再次按下并释放该按钮。你可以从以下常数中选择要检查的鼠标按钮。

语法。

mouse_check_button_released(numb);

争论 类型 描述
numb 鼠标按钮常数 Which mouse button constant to check for.

返回。

Boolean

例子。

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

上面的代码将检查鼠标右键是否被释放，如果被释放，它将把实例的速度设置为当前x/y位置与鼠标x/y位置之间距离的十分之一。

返回。鼠标输入
