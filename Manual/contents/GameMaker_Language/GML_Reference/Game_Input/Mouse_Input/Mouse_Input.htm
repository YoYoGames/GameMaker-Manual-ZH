鼠标输入

鼠标输入在所有平台上都被接受（在移动设备上，它被接受为单屏触摸--如果你需要使用多点触摸，你应该使用设备的特定功能），并有一些常量，用来指定被按下的按钮。这些常数在下表中显示。

标准鼠标按钮控制存在以下功能。

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

注意，还有一组与在桌面目标上使用鼠标有关的窗口功能。

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
