鼠标_lastbutton

这个变量返回最后一个被按下的鼠标按钮，并且可以返回任何一个特殊的鼠标常量，除了 mb_any （你也可以把这个变量设置为其中一个常量）。

语法。

mouse_lastbutton;

返回。

Mouse Button Constant (except mb_any)

例子。

if mouse_lastbutton = mb_left
{
    x -= 1;
}

如果最后按下的按钮是鼠标左键，这将使当前实例向左移动。

返回。鼠标输入
下一步。 mouse_wheel_up
