keyboard_key_release

通过这个函数，你可以模拟键盘上任何键的释放。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)。 或主键盘输入页面上列出的任何 vk_* 常数。

语法。

keyboard_key_release(key);

争论类型 描述
key The key to simulate a release of.

返回。

例子。

keyboard_key_release(vk_space);

这将模拟一个空格键的释放。

返回。键盘输入
