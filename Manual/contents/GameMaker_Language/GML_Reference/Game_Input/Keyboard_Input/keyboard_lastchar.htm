keyboard_lastchar

这个变量存储了最后一次按下的键的string 。这个变量不是只读的，你可以改变它，例如把它设置为""（一个空的string ），如果你已经处理了它。

语法。

keyboard_lastchar;

返回。

例子。

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

上面的代码检查lastkey变量是否大于-1，如果是，就把最后一个键作为string ，添加到变量 "str"。
