keyboard_check_direct

这个函数通过直接检查硬件，如果带有特定键码的键被按下，将返回 true ，如果没有，则返回 false 。它允许更多的检查，特别是你可以使用键码vk_lshift,vk_lcontrol,vk_lalt, vk_rshift,vk_rcontrol和vk_ralt来检查左或右shift, control或alt键是否被按下。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何 vk_* 常量。

注意 这个功能只适用于标准的Windows 目标，其结果与哪个应用程序有焦点无关。

语法。

keyboard_check_direct(key);

争论类型 描述
key The key to check the down state of.

返回。

例子。

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

上面的代码将检查左或右alt键是否被按下，如果被按下，则将变量 "crouch "设置为true。

返回。键盘输入
