keyboard_set_numlock

你可以用这个功能switch ，开启或关闭键盘数字锁（设置为true为开启，false为关闭）。

注意 该功能仅在Windows 目标构建中可用，在任何其他设备上都不会发挥作用。

语法。

keyboard_set_numlock(value);

争论 类型 描述
value 布尔型 Set this to true for "on" and false for "off"

返回。

N/A

例子。

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

上面的示例代码将获得数字锁键的状态，如果它是开（真），它将把它设置为关（假），反之亦然。

返回。键盘输入
