keyboard_check

有了这个功能，你可以检查一个键是否被按住。不像 keyboard_check_pressed()或 keyboard_check_released()函数只在按键被按下或释放时触发一次，这个函数在按键被按下的每一步都被触发。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何 vk_* 常数。

语法。

keyboard_check(key);

争论 类型 描述
key 虚拟钥匙常数（vk_*）。 The key to check the down state of.

返回。

Boolean

例子。

if keyboard_check(vk_left)
{
    x -= 5;
}

上面的代码将检查是否有方向键被按下，并在每一步返回真时向左移动实例5像素。
