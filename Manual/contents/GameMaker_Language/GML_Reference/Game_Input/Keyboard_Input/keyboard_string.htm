键盘字符串

这个变量持有一个string ，其中包含在键盘上输入的最后（最多）1024个字符。这个string ，只包含输入的可打印字符，但它会正确响应按退格键，删除最后一个字符。这个变量不是只读的，你可以改变它，例如把它设置为""（一个空的string ），如果你已经处理了它，你可以使用字符串函数来操作它。注意，当使用屏幕上的虚拟键盘时，只有这个变量会随着键盘的输入而被更新。

语法。

keyboard_string;

返回。

例子。

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

上述代码将把键盘string 的长度限制为15个字符，通过把前15个字符复制到变量中来删除那些超过这个限制的字符。

返回。键盘输入
