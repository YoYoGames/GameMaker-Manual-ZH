键盘_检查_按下

通过这个功能，你可以检查一个键是否被按下。不像 keyboard_check()函数，这个函数在每次按键被按下时只运行一次，所以要想再次触发，必须先释放按键，然后再按下。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何 vk_* 常数。

语法。

keyboard_check_pressed(key);

争论 类型 描述
key 虚拟钥匙常数（vk_*）。 The key to check the pressed state of.

返回。

Boolean

例子。

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

如果玩家按下键盘上的任何一个键，上述代码将推进到下一个room （工作方式类似于 "按任何键继续 "的提示）。

返回。键盘输入
下一步。 keyboard_check_released
