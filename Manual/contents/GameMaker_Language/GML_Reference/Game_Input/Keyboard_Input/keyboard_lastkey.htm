键盘_最后键

这个变量指的是 keyboard_key在前一帧中的值，返回该键的键码（所有标准的键码常数都被返回）。这个变量不是只读的，你可以改变它，例如，如果你已经处理了它，可以把它设置为-1。

语法。

keyboard_lastkey;

返回。

Virtual Key Constant

例子。

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

上面的代码检查lastkey变量是否不等于-1，如果是，就把最后一个键作为string ，然后重设 keyboard_lastkey ，以接受进一步的输入。

返回。键盘输入
下一步。 keyboard_lastchar
