keyboard_key_press

通过这个函数，你可以模拟按下键盘上的任何键。当使用时，该键将被标记为被按下，直到相应的释放函数被调用（见 keyboard_key_release()). 该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何 vk_* 常数。

请注意，在调用此函数后，如果键盘上的键被实际按下，此按下事件将不会被再次注册，直到该键被实际释放（触发释放事件并停止此函数）。 或者调用了相应的释放键函数，并再次按下该键。

语法。

keyboard_key_press(key);

争论类型 描述
key The key to simulate a press of.

返回。

例子。

keyboard_key_press(vk_space);

这将模拟按空格键的情况。

返回。键盘输入
