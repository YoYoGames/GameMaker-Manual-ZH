键盘_清空

通过这个函数，你可以清除当前的键盘状态，这基本上意味着，如果按键被按住，它将不再被识别，直到它被再次释放（这时也不会触发键盘按键释放事件）。 这时也不会触发键盘按键释放事件）并再次按下。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何 vk_* 常数。 主键盘输入页面上列出的任何常量。

语法。

keyboard_clear(key);

争论类型 描述
key The key to clear.

返回。

例子。

keyboard_clear(vk_space);

上面的代码清除了空格键的状态。

返回。键盘输入
