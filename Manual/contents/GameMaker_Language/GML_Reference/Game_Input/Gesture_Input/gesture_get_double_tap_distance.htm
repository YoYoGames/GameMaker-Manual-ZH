手势_get_double_tap_distance

该功能用于获取在单次点击后必须再次触摸/点击屏幕以触发双击手势的距离。该距离以英寸为单位，默认值为0.1。

语法。

gesture_get_double_tap_distance();

返回。

(inches)

例子。

if gesture_get_double_tap_distance() != 0.1
{
    gesture_double_tap_distance(0.1);
}

上述代码检查手势的双击距离是否设置为0.1英寸，如果不是，则将其设置为该值。
