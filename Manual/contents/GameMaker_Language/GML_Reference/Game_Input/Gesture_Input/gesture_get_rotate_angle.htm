手势_get_rotate_time

这个函数用于获得一对触摸必须超过的角度，以触发旋转开始手势。该角度以度为单位，默认值为5°。

语法。

gesture_get_rotate_angle();

返回。

(inches)

例子。

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

上面的代码检查手势的旋转角度是否被设置为5°，如果不是，就将其设置为这个值。
