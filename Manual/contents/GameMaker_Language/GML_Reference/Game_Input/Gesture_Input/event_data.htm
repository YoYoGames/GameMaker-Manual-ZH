event_data

这个变量是全局范围的，当用于适当的事件（如手势事件和广播信息）时，用于保存DS地图，其他时间为-1。DS地图的实际内容将取决于触发它的事件类型，所以请参考这些事件的个别章节。

语法。

event_data

返回。

DS Map ID

例子。

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

上面的代码是从Drag End手势事件中提取的，它检查通风口是否是一个 "轻弹 "事件，如果是，它从 event_data DS地图中提取所需的数据，并使用它来设置一些变量。如果没有检测到 "轻弹 "事件，那么同样的变量被设置为0。

返回。全局变量
