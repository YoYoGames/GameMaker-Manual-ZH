手势_double_tap_time

此功能用于设置两次触摸/点击之间触发双击手势事件的时间。时间的单位是秒，默认值是0.16。

语法。

gesture_double_tap_time(time);

争论 类型 描述
time 真实的 The time (in seconds) to set for double tap gesture event detection.

返回。

N/A

例子。

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

上面的代码检查手势的双击时间是否被设置为0.16秒，如果不是，则将其设置为该值。
