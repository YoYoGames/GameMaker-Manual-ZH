手势_get_flick_speed

该函数用于获取在释放触摸或点击时触发"轻触手势"事件所需的速度。速度的单位是英寸/秒，默认值是2.0。

语法。

gesture_get_flick_speed();

返回。

(inches per second)

例子。

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

上面的代码检查手势的滑动速度是否被设置为每秒2英寸，如果不是，则将其设置为该值。
