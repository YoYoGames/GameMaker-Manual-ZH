gesture_get_drag_time

此功能用于获取拖动开始手势事件被触摸或点击触发的时间。这个时间也会影响轻敲事件的触发方式，因为在这个时间结束前的触摸/点击和释放将被视为轻敲。该时间以秒为单位，默认值为0.16。

语法。

gesture_get_drag_time();

返回。

(time in seconds)

例子。

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

上面的代码检查手势的拖动时间是否被设置为0.16秒，如果不是，则将其设置为该值。
