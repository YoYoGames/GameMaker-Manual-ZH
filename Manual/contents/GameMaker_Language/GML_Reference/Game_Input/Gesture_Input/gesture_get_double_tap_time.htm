手势_get_double_tap_time

该函数用于获取两次触摸/点击触发双击手势事件的时间。时间的单位是秒，默认值是0.10。

语法。

gesture_get_double_tap_time();

返回。

Real (seconds)

例子。

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

上面的代码检查手势的双击时间是否设置为0.16秒，如果不是，就将其设置为该值。
