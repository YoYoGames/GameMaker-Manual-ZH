游戏输入

使游戏成为游戏的基本要素是与玩家的互动。这可以通过各种方式实现，在电脑上通过键盘和鼠标，在手机上通过触摸屏和设备倾斜，甚至在所有的平台上通过游戏手柄！这就是游戏的魅力。

在GameMaker中，首先你会在PC上使用专用的鼠标或键盘事件，或在手机上使用手势事件，但有时你需要比这些事件提供更多的控制，或者你想包括游戏板控制或多个不同的控制选项。为此，你可以使用不同的游戏输入功能来获得哪些按钮或按键被按下或释放，以及检索屏幕上的触摸或游戏板操纵杆被移动了多少。

在你的游戏中，有大量的游戏输入和控制功能，因此它们被分成与特定输入法有关的部分，以便更容易找到你需要的东西。可用的部分是：。

  • 键盘输入
  • 鼠标输入
  • 设备输入
  • 手势输入
  • 游戏板输入
  • 虚拟按键和虚拟键盘
