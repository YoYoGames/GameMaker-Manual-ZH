gamepad_button_check_pressed

这个函数将返回 true 或 false ，取决于给定的游戏板按钮是否被检测到被按下。注意，这个函数只在按钮第一次被按下时触发一次。要想再次触发，必须先释放按钮，然后再重新按下。如果你需要检查连续按下的按钮，你应该使用函数 gamepad_button_check().

如果你正在检查一个模拟按钮，那么检查会考虑到阈值的设置，只有当原始输入值超过给定的阈值时才会返回 true （你可以用函数 gamepad_button_value() 来获得这个原始值）。

语法。

gamepad_button_check_pressed(device, button);

争论 类型 描述
device 真实的 Which gamepad device "slot" to check.
button 游戏手柄按钮常数 Which gamepad button constant to check for.

返回。

Boolean

例子。

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

上述代码将检测连接到设备 "插槽 "0的游戏手柄的 "开始 "按钮是否被按下，如果被按下，则改变room 。

返回。游戏手柄输入
