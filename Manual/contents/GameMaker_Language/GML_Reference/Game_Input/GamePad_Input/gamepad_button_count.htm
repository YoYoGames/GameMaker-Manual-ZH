gamepad_button_count

这个函数将返回连接到给定设备 "插槽 "的游戏板的可用按钮总数。

语法。

gamepad_button_count(device);

争论类型 描述
device Which gamepad device "slot" to check.

返回。

例子。

b_num = gamepad_button_count(0);

上述代码将返回插入设备 "插槽 "0的游戏板上可用的按钮数量，并将该值存储在变量 "b_num "中。

返回。游戏手柄输入
