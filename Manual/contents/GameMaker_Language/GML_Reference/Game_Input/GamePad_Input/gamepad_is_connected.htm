gamepad_is_connected

这个函数将返回一个游戏手柄是否连接到给定的 "插槽"（返回 true ）或没有（返回 false ）。你通常会把这个函数和 gamepad_get_device_count()函数来获取正确的可用游戏板和/或游戏板 "插槽 "的数量。请注意，在用户连接游戏板和GameMaker检测到它被连接之间可能会有一点延迟（在处理蓝牙连接的控制器时尤其如此）。

注意：在某些平台上--特别是游戏机--这个功能在连接/选择游戏板后可能立即返回错误，可能需要在几帧（步骤）后的警报中检查，以便正确检测游戏板。

语法。

gamepad_is_connected(numb);

争论类型 描述
device Which gamepad "slot" to check.

返回。

例子。

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

上面的代码loops ，通过可用的游戏板（或游戏板插槽），然后检查每个游戏板的连接情况。然后，返回的值被用来设置一个全局数组 true 或 false ，以便在未来的检查中使用。
