gamepad_get_guid

通过这个函数，你可以检索连接到给定插槽索引的游戏手柄的GUID。你提供要检查的游戏板插槽（从0-11），该函数将返回以下字符串之一。

  • 如果没有GUID或者没有连接游戏手柄，则为"无"。
  • 如果游戏板插槽索引超出范围，则"设备索引超出范围"。
  • GUIDstring ，类似于 "050000005e040000fd020000ff3f00"，如果插槽有一个分配的游戏板，并且可以得到它的GUID。

这个函数通常与 gamepad_get_description()来生成SDLstring 的部分内容，这些内容是在Windows,Android 和macOS 上重新映射控制器所需要的。

平台的具体说明
平台 注意事项
Opera GX 在Opera GX目标平台上，对于非直接输入设备，该函数将始终返回"无"。

语法。

gamepad_get_guid(index);

争论类型 描述
index Which gamepad "slot" index to check

返回。

例子。

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

上述代码获得GUID和名称描述strings ，然后将它们连接起来，并将最终的string 在一个全局变量中，以供将来参考。
