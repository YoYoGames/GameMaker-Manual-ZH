gamepad_hat_count

这个函数将返回连接到给定设备 "插槽 "的游戏板可用的帽子总数。帽子一般指的是上/下/左/右按钮，注意在Windows 目标上，帽子只对直接输入的 控制器（所以，从插槽4开始）。

语法。

gamepad_hat_count(device);

争论类型 描述
device Which gamepad device "slot" to check.

返回。

例子。

h_num = gamepad_hat_count(4);

上述代码将返回插入设备 "插槽 "4的游戏板上可用的帽子数量，并将该值存储在变量 "h_num "中。

返回。游戏手柄输入
