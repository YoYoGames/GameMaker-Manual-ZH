gamepad_get_option

这个函数可以用来检索给定的选项值。你提供要检查的游戏板 "插槽"，以及选项string ，该函数将返回相关的值。可用的选项string 将取决于项目运行的平台，如下所示。

选项 平台 描述 返回
"allow_rotation" tvOS 返回Siri遥控器是否会被旋转锁定（即：只能垂直使用）。当解锁时，旋转控制器将改变输入，例如向左旋转90度将使触摸板的 "右 "侧返回 "上"，等等......这样，所有的输入将与设备的方向相对。 布尔型
"dpad_absolute" tvOS 返回Siri Remote触摸板是接受绝对输入，还是相对输入。在绝对模式下，触摸板的中心被认为是(0,0)的位置--即模拟摇杆静止时的位置或dpad的中心位置--围绕这个位置的移动将产生输入。当设置为相对时，(0,0)位置将被认为是在触摸板上的初始位置，所有的输入将从这个位置产生，而不是触摸板的中心。 布尔型

语法。

gamepad_get_option(device, option_name);

争论类型 描述
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

返回。

Boolean, Real, String, or N/A (depends on platform and option)

例子。

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

上述代码loops ，通过所有的游戏板插槽，检查tvOS 平台上的 "Siri遥控器"。如果检测到一个，它就会将遥控器设置为允许输入旋转。

返回。游戏手柄输入
