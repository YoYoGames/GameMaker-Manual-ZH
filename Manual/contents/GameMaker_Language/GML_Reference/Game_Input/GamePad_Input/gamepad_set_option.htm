gamepad_set_option

这个函数可以用来设置任何可用的游戏板选项。你提供要设置选项的游戏板 "插槽"，以及要设置的选项string 和使用的值。可用的选项string ，取决于项目运行的平台，如下表所示。

选项 平台 描述 价值
"allow_rotation" tvOS 设置Siri遥控器是否会被旋转锁定（即：只能垂直使用）。当解锁时（数值为 true ），旋转控制器将改变输入，例如向左旋转90度将使触摸板的 "右 "侧返回 "上"，等等......这样，所有的输入都是相对于设备的方向。当锁定时（值为 false ），那么输入将不会相对于控制器的位置而改变，并将始终被视为控制器处于垂直位置。 布尔型
"dpad_absolute" tvOS 设置Siri Remote触摸板是接受绝对输入，还是接受相对输入。在绝对模式下（数值为 true ），触摸板的中心被认为是(0,0)的位置--即：模拟摇杆静止的位置或dpad的中心位置--围绕这个位置的移动将产生输入。当设置为相对（数值为 false ）时，（0,0）位置将被认为是在触摸板上的初始位置，所有的输入将从这个位置产生，而不是触摸板的中心。 布尔型

语法。

gamepad_set_option(device, option_name, value);

争论 类型 描述
device 真实的 Which gamepad device "slot" to set.
option_name 字符串 The name of the option to set (a string, see the table above)
value 实数，布尔值，或字符串 The value to set the option to (can be boolean, real or string - see the table above)

返回。

N/A

例子。

for(var i = 0; i < 12; ++i;)
{
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
    {
        if gamepad_get_option(i, "allow_rotation") == false
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

上述代码loops ，通过所有的游戏板插槽，检查tvOS 平台上的 "Siri遥控器"。如果检测到一个，它就会将遥控器设置为允许输入旋转。

返回。游戏手柄输入
