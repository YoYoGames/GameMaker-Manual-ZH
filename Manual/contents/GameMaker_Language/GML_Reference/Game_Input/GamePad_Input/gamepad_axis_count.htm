gamepad_axis_count

这个函数将返回被检查设备上的 "轴 "控制的数量。这些控制是大多数控制器上的模拟方向 "拇指摇杆"。

语法。

gamepad_axis_count(device);

争论类型 描述
device Which gamepad device "slot" to check.

返回。

例子。

axis = gamepad_axis_count(0))

上述代码将连接到设备 "插槽 "0的游戏板的可用轴数存储在变量 "axis "中。

返回。游戏手柄输入
