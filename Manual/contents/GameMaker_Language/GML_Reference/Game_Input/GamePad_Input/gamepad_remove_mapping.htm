游戏手柄_删除映射

这个函数可以用来从给定的游戏板插槽索引中移除当前设备映射。一旦被调用，如果你希望能够使用游戏板常量来检测输入，就需要用 gamepad_test_mapping()函数重新映射，如果你想能够使用游戏板常量来正确检测输入（直接输入总是可以使用 gamepad_axis/button/hat_count() 和 gamepad_axis/button/hat_value() 函数来检索）。

语法。

gamepad_remove_mapping(index);

争论类型 描述
index Which gamepad index "slot" to remove the mapping from.

返回。

例子。

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

上面的代码将根据一个变量的值从给定的游戏手柄索引槽中删除映射。

返回。游戏手柄输入
