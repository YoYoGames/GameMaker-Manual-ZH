gamepad_button_value

通过这个函数你可以得到一个模拟按钮的当前值，从0到1，其中0是无压力，1是全压力。你提供要检查的游戏板插槽索引，以及一个按钮常数（如这里列出的），或者一个在0和gamepad_button_count() - 1之间的整数。

请注意，这将返回按钮的原始值，而不考虑阈值的设置（关于阈值的更多信息请看这里）。

语法。

gamepad_button_value(device, button);

争论 类型 描述
device 真实的 Which gamepad device "slot" to check.
button 游戏手柄按钮常数 Which gamepad button constant to check for.

返回。

Real

例子。

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

上述代码使用插入设备 "插槽 "0的游戏手柄的模拟触发值来设置实例的速度。

返回。游戏手柄输入
