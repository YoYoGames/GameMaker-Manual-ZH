gamepad_set_colour

这个函数可以用来设置PlayStation控制器中的LEDs的颜色。你指定要设置的设备插槽，然后给一个颜色，可以是任何颜色常数或使用特定颜色函数创建的颜色值或HEX值（如$FFFFFFF）。

语法。

gamepad_set_colour(device, colour);

争论类型 描述
device Which gamepad device "slot" to set.
colour The colour to use.

返回。

例子。

if health < 10
{
    gamepad_set_colour(0, c_red);
}

如果 health 变量低于10，上述代码将把PlayStation控制器的LED灯设置为红色。

返回。游戏手柄输入
