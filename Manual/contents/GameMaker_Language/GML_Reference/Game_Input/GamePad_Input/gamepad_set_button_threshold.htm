gamepad_set_button_threshold

这个功能可以用来设置一个给定设备的模拟按钮的当前阈值设置。所有模拟按钮的默认阈值是0.5，范围是0到1。

阈值定义了在什么时候按钮被认为是 "被按下"，用于需要它们作为数字按钮的游戏。

这个函数将影响模拟按钮的检查、按下和释放状态，但不会影响函数返回的值 gamepad_button_value()所返回的值，它将始终返回按钮的原始值。

语法。

gamepad_set_button_threshold(device, threshold);

争论 类型 描述
device 真实的 Which gamepad device "slot" to check.
threshold 真实的 The new threshold value (from 0 - 1, default 0.5).

返回。

N/A

例子。

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

上面的代码检查连接到设备 "插槽 "0的游戏板的模拟按钮阈值，如果它不是默认值0.5，它就被设置为这个值。

返回。游戏手柄输入
