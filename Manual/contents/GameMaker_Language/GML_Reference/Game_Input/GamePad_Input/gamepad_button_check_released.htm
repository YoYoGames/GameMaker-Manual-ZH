gamepad_button_check_released

这个函数将返回 true 或 false ，取决于给定的游戏手柄按钮是否被检测到已经释放。请注意，这个函数只在按钮被释放的瞬间触发一次。要想再次触发，必须先按下按钮，然后再释放一次。如果你检查的是一个模拟按钮，那么检查将考虑到阈值的设置，只有当原始输入值低于给定的阈值时才会返回 true （你可以用函数 gamepad_button_value() 来得到这个原始值）。

语法。

gamepad_button_check_released(device, button);

争论 类型 描述
device 真实的 Which gamepad device "slot" to check.
button 游戏手柄按钮常数 Which gamepad button constant to check for.

返回。

Boolean

例子。

if gamepad_button_check_released(0, gp_select)
{
    audio_play_sound(snd_Button, 0, false);
    global.Pause = !global.Pause;
}

上述代码将检测连接到设备 "插槽 "0的游戏手柄的 "选择 "按钮是否被按下，并切换全局 "暂停 "变量。

返回。游戏手柄输入
