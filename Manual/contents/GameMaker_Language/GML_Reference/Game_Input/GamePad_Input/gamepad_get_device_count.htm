gamepad_get_device_count

这个功能要么给你连接的游戏板的数量，要么给游戏板连接的可用 "槽 "的数量。

实际的返回值将取决于平台和该平台的内部配置，因此该函数应与函数 gamepad_is_connected()来确保任何时候连接的焊盘的确切数量。

语法。

gamepad_get_device_count();

返回。

Real

例子。

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i)
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

上面的代码loops ，通过可用的游戏板（或游戏板插槽），然后检查每个游戏板的连接情况。然后，返回的值被用来设置一个全局数组 true 或 false ，以便在未来的检查中使用。
