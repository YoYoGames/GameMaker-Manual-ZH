gamepad_axis_value

你可以使用这个函数从一个给定的游戏手柄插槽中获得不同轴的值。你提供要检查的游戏板插槽索引，以及一个轴索引常数（如这里列出的），或一个在 0 和 gamepad_axis_count()-1 之间的整数值。对于每个可用的水平和垂直轴，返回的值将在 -1 和 1 之间。

请注意，返回的值是正常化的，考虑到轴的死区设置（使用函数 gamepad_set_axis_deadzone() ），所以总是在-1和0，以及0和1之间。例如，将死区设置为0.2将意味着只有当原始轴值超过0.2时，向右推杆才会开始返回0-1的数值--因此，例如，当原始值为0.2时，返回值为0，当原始值为0.5时，返回值为0.375，或者如果原始值为0.9，那么返回值将为0.875。

语法。

gamepad_axis_value(device, axisIndex);

争论 类型 描述
device 真实的 Which gamepad device "slot" to check.
axisIndex 游戏手柄轴常数 The axis index to check (see the constants list).

返回。

Real

例子。

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

上述代码根据插入设备 "插槽 "0的游戏手柄的左模拟控制器的返回值，设置实例的速度和方向。

返回。游戏手柄输入
