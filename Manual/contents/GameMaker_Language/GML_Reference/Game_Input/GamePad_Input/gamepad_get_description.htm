gamepad_get_description

这个函数将返回一个带有给定游戏板描述的string ，例如，PS3控制器可能返回一个类似于 "PLAYSTATION(R)3 Controller "的string ，或者XBox360控制器可能返回 "Xbox 360 Controller (XInput STANDARD GAMEPAD)"。这个string ，与硬件有关，返回值取决于插入被检查的设备 "插槽 "的游戏手柄。

语法。

gamepad_get_description(device);

争论类型 描述
device Which gamepad "slot" to get the name of.

返回。

例子。

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i)
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(i));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "No Gamepad Connected");
    }
}

上面的代码将loop ，检查所有的游戏板插槽，检查是否有连接的设备，然后根据游戏板是否连接到插槽上，在屏幕上画一些文字。

返回。游戏手柄输入
