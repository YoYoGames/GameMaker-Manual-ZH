virtual_key_delete

如果你的游戏有不同的rooms 或实例来控制菜单和游戏玩法以及其他东西，那么你很可能需要在某些时候改变你的虚拟按键的位置和按键图。为此，你可以使用这个函数在创建新的虚拟按键之前删除旧的虚拟按键（如果需要）。该函数要求你提供要删除的虚拟键的ID--如通过以下方式返回的 virtual_key_add().

请注意，当room ，任何虚拟钥匙将自动从room ，所以这个功能只有在你希望在room 本身完成之前手动删除钥匙时才有必要。

语法

virtual_key_delete(index);

争论 类型 描述
index 虚拟钥匙ID The index of the virtual key to delete.

返回。

N/A

例子。

virtual_key_delete(global.Left);

上述代码删除了全局变量 "global.Left "中索引的虚拟键。
