键盘_虚拟高度

此函数将根据显示器的大小，返回虚拟键盘的当前高度（像素）。如果键盘不可见，将返回0。

语法

keyboard_virtual_height();

返回。

例子。

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

上面的代码将检查操作系统虚拟键盘的状态，如果它是可见的，就设置一个变量为键盘的高度。
