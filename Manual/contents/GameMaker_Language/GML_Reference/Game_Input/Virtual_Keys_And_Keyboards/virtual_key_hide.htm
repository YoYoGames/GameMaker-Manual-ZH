虚位以待

一旦你为你的设备创建了虚拟钥匙，你可能需要调试和测试它是否正确定位和正常工作，为此你将调用函数 virtual_key_show().然而，你可能需要切换这个视图，所以你也可以用这个函数隐藏虚拟键，使用你想隐藏的虚拟键的索引（之前用 virtual_key_add()).一旦你对事情感到满意，建议你在其区域内创建并放置你自己的图形。

语法

virtual_key_hide(index);

争论 类型 描述
index 虚拟钥匙ID The index of the virtual key to hide.

返回。

N/A

例子。

if global.Debug
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

上述代码检查全局变量 "global.Debug"，如果测试为真，那么变量 "global.Left "中索引的虚拟键将被画在屏幕上，如果测试为假，那么该键将被隐藏。
