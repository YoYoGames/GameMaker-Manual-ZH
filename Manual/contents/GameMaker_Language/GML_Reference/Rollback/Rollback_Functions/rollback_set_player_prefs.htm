滚回_设置玩家参数

启用偏好后，这个函数被用来设置本地播放器的偏好。这意味着该函数必须依赖本地输入，而不是Rollback输入。

首选项可以是任何类型的。如果设置了默认偏好，请确保你使用相同的数据类型和结构。

当任何玩家设置其偏好时，游戏中的所有人都会收到一个回滚事件。

如果使用，这个函数必须在连接/创建函数之前运行。

请阅读。玩家偏好

语法。

rollback_set_player_prefs(preferences);

争论 类型 描述
preferences 任何 The preferences to set for the local player.

返回。

N/A

例子。

rollback_set_player_prefs({
character: obj_customizer.character_selected,
color: obj_customizer.color_selected
});

上面的代码为播放器设置偏好，从 "自定义器 "object ，它将是非管理的（本地）。
