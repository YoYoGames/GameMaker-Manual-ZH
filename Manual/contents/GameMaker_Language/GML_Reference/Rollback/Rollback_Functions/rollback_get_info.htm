滚回_获取信息

这个函数返回一个struct ，其中包含关于一个球员的信息。这应该只在手动管理球员实例时使用。如果你使用的是 rollback_define_player()，那么每个球员实例将自动收到这些信息作为变量。

该函数将玩家的ID作为参数，并返回一个带有以下变量的struct 。

变化的 类型 描述
player_avatar_url 字符串 指向玩家头像的URL。
player_avatar_sprite 雪碧资产 含有玩家头像的精灵
player_name 字符串 球员的名字
player_type 字符串 说明玩家是否登录了GX.games。

访客用户为"Guest"，已登录的用户为"User"。
player_user_id 字符串 GX.games上玩家的ID。
player_prefs 任何 为播放器设置的偏好

语法。

rollback_get_info(player_id);

争论 类型 描述
player_id 真实的 The ID of the player for which information will be returned.
