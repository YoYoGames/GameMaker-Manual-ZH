rollback_start_game

这个函数强制当前的多人游戏开始，在所有玩家都加入之前。你必须先用以下方法创建一个多人游戏 rollback_create_game().

系统通常会等待所有玩家加入，然后自动开始游戏。这个功能可以用来提前开始游戏，在所有玩家都加入之前。

你可以调用 rollback_use_manual_start()来让Rollback等待这个函数，即使在所有玩家都加入之后。

调用此函数将触发回滚开始事件。

语法。

rollback_start_game();

返回。

N/A

例子。

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

如果回滚游戏没有运行，而按下回车键，这将启动游戏。

这使得玩家可以在所有玩家加入之前手动开始游戏。
