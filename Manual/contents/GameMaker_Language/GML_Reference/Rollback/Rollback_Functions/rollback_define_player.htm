滚回_定义玩家

这个函数定义了在多人游戏中应该为玩家使用的object 。这是可选的，当使用时，系统将自动为连接的玩家创建实例，并为断开连接的玩家销毁实例。

默认情况下，这将在"Instances "层创建玩家实例，该层必须存在于你的游戏中room 。如果你想使用另一个层，把它的名字作为第二个参数传过来。

关于使用或不使用此函数创建游戏的信息，请参阅定义一个玩家对象。如果使用，这个函数必须在join/create函数之前运行。

用这个函数创建的每个实例都会得到本页所列的实例变量。

语法。

rollback_define_player(object, [layer_name]);

争论 类型 描述
object 对象资产 The object that will be used for creating player instances.
layer_name 字符串 OPTIONAL 将创建播放器实例的层的名称。默认为"实例"。

返回。

N/A

例子。

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

上面的代码定义了一个玩家object ，然后试图加入一个游戏。如果没有加入，它就开始一个新的游戏，最多有2个玩家，并禁用同步测试。
