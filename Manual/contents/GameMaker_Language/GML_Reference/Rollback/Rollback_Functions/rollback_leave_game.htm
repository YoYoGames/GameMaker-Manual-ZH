rollback_leave_game

该函数离开当前的多人游戏会话，允许玩家再次加入或创建一个新游戏。

当一个room 的主人离开时，该room 的所有权将转移给另一个玩家，因此游戏可以继续进行。只有当所有玩家都离开后，游戏才会结束。

避免错误

在玩家离开多人游戏会话后，GameMaker ，继续执行所有的游戏代码，然而它返回到 "预启动 "状态，你不能创建或修改托管实例。

这意味着你需要在离开游戏时手动销毁你的玩家实例，并在一个条件下使用rollback_game_running暂停任何其他游戏逻辑，以避免错误。

见。回滚开始前的状态

触发的事件

当一个玩家离开你所在的游戏时，回滚事件被触发。更多信息见回滚事件。

语法。

rollback_leave_game();

返回。

N/A

例子。

if (keyboard_check_pressed(vk_escape))
{
    rollback_leave_game();
}

上面的代码使玩家在按下Escape键时离开游戏。
