回滚_使用播放器_预设功能

这个功能可以启用玩家偏好，并允许你为所有玩家设置默认偏好。

默认偏好可以是任何类型的。如果你不指定默认偏好，那么所有玩家都需要在游戏开始前设置好偏好。

调用这个函数将禁用自动启动，所以你必须手动启动游戏。见。手动与自动启动

如果使用，这个函数必须在连接/创建函数之前运行。

请阅读。玩家偏好

语法。

rollback_use_player_prefs([default]);

争论 类型 描述
default 任何 OPTIONAL 为所有球员设置的默认偏好。

返回。

N/A

例子。

Rollback_use_player_prefs({
字符：0。
color: c_white
});

上面的代码启用了播放器的偏好，并在struct.NET中设置了默认的偏好。
