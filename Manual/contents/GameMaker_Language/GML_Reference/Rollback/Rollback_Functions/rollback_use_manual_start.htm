回滚_使用_手动_开始

默认情况下，当所有玩家都加入后，多人游戏会自动开始。

调用这个函数可以禁用自动启动行为，因此游戏会等待你手动启动。

如果使用，这个函数必须在连接/创建函数之前运行。

语法。

rollback_use_manual_start();

返回。

N/A

例子。

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

上述代码启用手动启动，然后加入或创建一个新的多人游戏。
