滚回_使用_随机输入

当你使用同步测试进行游戏时，所有 "远程 "玩家都会收到他们定义的输入的随机值，作为一种基本的测试方式。你可以用false作为参数调用这个函数来禁用这个行为，并通过指定true再次启用它。

语法。

rollback_use_random_input(enabled);

争论 类型 描述
enabled 布尔型 Specifies whether random input should be enabled (true）或禁用（false）。默认情况下，它是启用的。

返回。

N/A

例子。

rollback_use_random_input(false);

上述代码在同步测试期间禁用了随机输入。
