回滚_定义_输入

这个函数用来为所有球员定义自定义输入。它接收一个struct ，其中应包含所有的输入及其分配的输入常量。

要知道如何分配自定义输入，请看自定义控制。

如果使用，这个函数必须在连接/创建函数之前运行。

语法。

rollback_define_input(input_struct);

争论 类型 描述
input_struct 结构 A struct containing input names as variables, and their assigned inputs as values.

返回。

N/A

例子。

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

上面的代码为所有球员定义了自定义输入。最后两个输入使用了多个控件，通过将它们列在一个数组中。

关于一个扩展的例子，见定义输入。
