滚回_显示_事件

默认情况下，当你运行游戏时，你会在游戏的左上角看到调试信息，显示发生的多人事件。你可以用false作为参数调用这个函数来禁用这些信息，并通过指定true再次启用它们。

语法。

rollback_display_events(enabled);

争论 类型 描述
enabled 布尔型 Specifies whether event display should be enabled (true）或禁用（false）。默认情况下，它是启用的。

返回。

N/A

例子。

rollback_display_events(false);

上面的代码禁用了事件的显示。
