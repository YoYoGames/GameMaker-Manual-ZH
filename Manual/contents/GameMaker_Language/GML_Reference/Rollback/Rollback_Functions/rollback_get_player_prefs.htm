滚回_获取玩家设置

此函数返回给定玩家ID的偏好设置。

如果没有指定球员的ID，该函数必须在一个球员实例内调用，在这种情况下，该球员的偏好将被返回。

你也可以使用实例变量 player_prefs 。

这个函数返回给定播放器最后同步的偏好，所以设置偏好不会立即更新这个函数返回的偏好。

请阅读。玩家偏好

语法。

rollback_get_player_prefs([player_id]);

争论 类型 描述
player_id 真实的 OPTIONAL 返回偏好的玩家的ID。

返回。

Any

例子。

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

上面的代码获得了为当前玩家设置的偏好，并将一个偏好应用于一个实例变量。
