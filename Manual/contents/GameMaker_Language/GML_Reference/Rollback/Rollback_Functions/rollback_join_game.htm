回退_加入游戏

这个函数试图加入一个游戏。你必须在一个包含虚拟room 的ID的URL上才能加入。

如果加入了room ，则返回真，如果没有加入游戏，则返回假。

当一个新的玩家加入到你已经存在的游戏中时，回滚事件被触发。更多信息请参见回滚事件。

dry_run参数是可选的，当设置为true时，允许你在不实际加入游戏的情况下运行该函数。这是用来检查是否可以加入一个游戏而不实际加入它。

语法。

rollback_join_game([dry_run]);

争论 类型 描述
dry_run 布尔型 Disabled by default (false).当启用（真）时，该函数不会加入游戏，只是在可以加入游戏时返回真，否则返回假。

这样，你可以用这个函数只检查玩家是否被邀请。

返回。

Boolean

例子。

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_create_game(2, false);
}

上面的代码定义了一个玩家object ，然后试图加入一个游戏。如果没有加入，它就创建一个新的游戏，最多有2个玩家，并禁用同步测试。

关于一个扩展的例子，见创建一个多人游戏。
