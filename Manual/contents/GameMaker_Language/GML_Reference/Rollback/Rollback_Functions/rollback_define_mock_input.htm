滚回_定义_模拟输入

这个函数可以在同步测试期间使用，为指定的播放器分配临时的 "模拟 "输入。这覆盖了为该播放器定义的控制和来自同步测试的任何随机输入。

有关其使用的信息，见模拟输入。

语法。

rollback_define_mock_input(player_id, input_struct);

争论 类型 描述
player_id 真实的 The ID of the player to assign mock input to.
input_struct 结构 A struct containing input names as variables, and their assigned inputs as values.

返回。

N/A

例子。

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: ord("A"),
    right: ord("D")
});

rollback_define_mock_input(1, {
    fire: vk_control,
    interact: vk_shift,
    left: ord("J"),
    right: ord("L")
});

上面的代码首先定义了球员的输入。然后它为玩家1定义了模拟输入，保持相同的输入名称，但分配不同的输入。这样，两个人就可以在本地使用键盘玩游戏了。
