回滚_聊天

这个功能是用来向其他玩家发送string 信息。

你指定要发送的string ，最大尺寸为128字节。

当没有指定第二个参数时，信息将被发送到所有玩家，包括发送者。

你可以在第二个参数中指定一个玩家ID，这是将收到信息的玩家。

如何使用？

这个函数是为了在 "本地 "使用的，所以会产生本地输入的结果，而不是Rollback输入。

见底部的使用例子。听取回滚事件中收到的消息。

语法。

rollback_chat(message, [to]);

争论 类型 描述
message 字符串 The message to send, as a string.
to 真实的 OPTIONAL 消息将被发送到的玩家的ID。不指定这个参数，或者指定-1，就会把消息发送给所有玩家。

返回。

N/A

例子。

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

如果在本地按了回车键，那么 keyboard_string值就会发送给所有玩家。该string ，然后被清除，因此可以输入新的信息。

这将在所有客户端（包括发送方）收到消息时触发回滚事件。

回滚事件

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}

返回。回滚功能
