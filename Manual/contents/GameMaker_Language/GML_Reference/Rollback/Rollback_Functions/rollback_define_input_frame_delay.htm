滚回_定义_输入帧_延迟

这个函数定义了本地播放器输入的固定帧延迟。

默认情况下，你的游戏将使用一个自适应的输入延迟，它是基于玩家之间的延迟。这几乎适用于所有的游戏，然而，一些特定类型的游戏（如格斗游戏）可能更喜欢使用固定的输入延迟，而不是可能随时改变的自适应延迟。

除非你清楚地知道你在做什么，否则不建议改变这个。你可以在多人游戏前和游戏中的任何时候调用这个函数。

指定的延迟值必须是帧数。如果你指定-1，游戏将恢复到使用自适应延迟。你可以设置的最大延迟是10帧。

语法。

rollback_define_input_frame_delay(frames);

争论 类型 描述
frames 真实的 The number of frames to set as the fixed input delay, or -1 将其重置为默认值。

返回。

N/A

例子。

// 'Options' script
global.frame_delay = 5;

// obj_game: Create event
rollback_define_input_frame_delay(global.frame_delay);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

你可以将你的帧延迟值定义为一个全局变量，并在回滚游戏开始前调用rollback_define_input_frame_delay()时使用。
