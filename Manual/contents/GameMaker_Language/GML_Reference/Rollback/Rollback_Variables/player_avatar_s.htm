玩家_阿凡达_斯普利特

这个变量存储当前玩家实例的sprite 。

这个变量只存在于由 rollback_define_player()创建的球员实例中，并且对每个球员来说都是不同的。

访客玩家将显示一个占位符sprite 。

语法。

player_avatar_sprite;

返回。

Sprite Asset

例子。

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

这将显示运行代码的玩家实例的头像，在64x64的区域内，在实例的上方256px处。
