回滚_当前帧

这是一个全局变量，用于存储多人游戏开始后的帧数。

应该用它来代替 current_time 和其他类似的变量，如时间变量中所解释的。

语法。

rollback_current_frame

返回。

Real

例子。

image_angle = dsin(rollback_current_frame) * 90;

上面的代码使用 rollback_current_frame 与 dsin()来计算一个介于1和-1之间的振荡值，这个振荡值乘以90并应用于实例的角度。
