滚回游戏运行中

这个全局变量表示回滚游戏目前是否正在运行。如果为真，意味着回滚游戏已经开始，并且目前正在运行。在所有其他时间，它将是假的。

语法。

rollback_game_running

返回。

Boolean

例子。

if (!rollback_game_running) return;

x += move_x;
y += move_y;

上述代码将在玩家的Step事件中运行object 。当回滚游戏没有运行时，它将退出该事件，所以在回滚游戏开始之前，玩家不能被控制。
