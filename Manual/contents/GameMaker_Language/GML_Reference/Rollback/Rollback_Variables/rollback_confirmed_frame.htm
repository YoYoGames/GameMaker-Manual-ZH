回滚_确认的框架

这个全局变量存储所有玩家的输入被同步和确认的最后一帧号码。

你可以将其与rollback_current_frame ，以检查当前帧是否已确认所有玩家的输入。

你也可以使用函数 rollback_sync_on_frame() ，检查当前帧是否同步。

语法。

rollback_confirmed_frame

返回。

Real

例子。

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

这段代码通过检查当前帧值是否等于确认的帧值，来检查当前帧是否有所有玩家的确认输入。

如果是真的，它就会运行一个函数来检查游戏结束条件是否得到满足，这样它就可以结束游戏。

这样做是为了避免基于预测的状态结束游戏，并在试图检查游戏结束条件之前等待一个完全同步的帧。
