玩家名称

这个变量存储了当前播放器实例的名称。

这个变量只存在于由 rollback_define_player()创建的球员实例中，并且对每个球员来说都是不同的。

访客玩家将返回一个占位符名称。

语法。

player_name;

返回。

String

例子。

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

这将在其实例上方100px处画出玩家的名字，并在中间位置对齐。
