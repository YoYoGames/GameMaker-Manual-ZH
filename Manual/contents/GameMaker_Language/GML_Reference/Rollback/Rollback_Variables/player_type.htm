玩家类型

这个变量告诉我们玩家实例是一个已经登录的GX.games用户，还是一个访客。它可以返回以下任何一个字符串。

字符串 描述
"Guest" 玩家没有在GX.games中签名
"User" 该玩家已登录GX.games

访客用户获得占位的头像和名字。

这个变量只存在于由 rollback_define_player()创建的球员实例中，并且对每个球员来说都是不同的。

语法。

player_type;

返回。
