表面_重设目标

通过这个函数，你可以将所有进一步的绘图从目标表面重置到屏幕上。请注意，要开始在一个曲面上绘图，你必须先调用函数 surface_set_target()并在完成后再调用这个函数，否则屏幕上不会有任何东西被绘制，因为所有进一步的绘制（甚至在其他情况下）都会在曲面上完成。你也应该意识到，如果曲面本身没有在实例的绘制事件中被绘制到屏幕上，那么什么都不会被看到。

注意：如果你之前没有用函数设置一个渲染目标 surface_set_target()来设置渲染目标，那么使用这个函数将默默地（即：没有任何错误信息）结束该事件的所有后续代码执行。

语法。

surface_reset_target();

返回。

例子。

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

上面的代码将检查当前正在绘制的视图，如果是view[0]，它将把绘制目标设置为一个曲面，并在再次重置绘制目标之前绘制所有object "obj_Effect "的实例。如果视图不是view[0]，则将曲面绘制到屏幕上。
