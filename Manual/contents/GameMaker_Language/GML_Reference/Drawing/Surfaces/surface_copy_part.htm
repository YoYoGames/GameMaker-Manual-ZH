surface_copy_part

这个函数只是把一个曲面上的图像复制到另一个曲面上的指定局部位置（其中（0,0）位置是目标曲面的左上角）。你可以指定要复制的局部X和Y位置，以及该部分的宽度和高度。请注意，这些坐标是基于曲面的尺寸，而不是基于曲面在room 。如果目标曲面已经有了信息，这些信息将被复制覆盖，该函数不会以任何方式改变源曲面。

注意：在处理表面时，由于它们被储存在texture ，它们有可能在任何时候都不存在了。你应该在直接引用表面之前，先用 surface_exists()来检查曲面是否存在，然后再直接引用它们。

语法。

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

争论 类型 描述
destination 表面标识 The ID of the surface to copy the other surface to.
x 真实的 The x position to copy to.
y 真实的 The y position to copy to.
source 表面标识 The ID of the surface to be copied.
xs 真实的 The x position in the source surface to copy from.
ys 真实的 The y position in the source surface to copy from.
ws 真实的 The width of the area in the source surface to copy from.
hs 真实的 The height of the area in the source surface to copy from.

返回。

N/A

例子。

if view_current == 0
{
    var _cam1_x = camera_get_view_x(view_camera[1]);
    var _cam1_y = camera_get_view_y(view_camera[1]);

    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, _cam1_x - mouse_x, _cam1_y - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

上面的代码将检查当前正在绘制的视图，如果是view[0]，它将把变量 "temp_surf "中索引的表面复制到变量 "surf "中索引的表面。复制的区域对应于鼠标在曲面中的相对位置所形成的矩形，因为它将在视图[1]中被绘制。如果当前的视图不是view[0]，那么曲面 "surf "将被绘制到屏幕上。
