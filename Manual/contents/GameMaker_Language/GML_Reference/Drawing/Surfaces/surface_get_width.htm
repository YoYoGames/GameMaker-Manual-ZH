表面_获取宽度

这个函数简单地返回索引表面的宽度，单位是像素。需要注意的是，如果你在使用这个函数改变了 application_surface ，然后立即调用这个函数来检查它的大小。 surface_resize() 后，它不会返回新的值，因为这个变化需要一两步才能完全处理。在等待一个步骤后，它应该正确地返回新的尺寸。

注意：在处理表面时，由于它们被储存在texture ，它们有可能在任何时候都不存在了。你应该在直接引用表面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_get_width(surface_id);

争论类型 描述
surface_id The ID of the surface to get the width of.

返回。

例子。

sw = surface_get_width(surf);

上面的代码将把变量 "surf "中索引的表面的宽度存储到变量 "sw "中。
