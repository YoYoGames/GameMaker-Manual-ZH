创建表面

这个函数用来创建一个曲面，并将返回该曲面的索引，该索引应保存在一个变量中，以便将来调用函数。当曲面第一次被创建时，它可能包含 "噪音"，因为基本上它只是为这个目的预留的一个内存区域（而且这个内存可能仍然包含信息），所以你可能想在使用函数之前清除这个曲面，比如 draw_clear_alpha().

强烈建议所有表面的大小都是2的幂，例如：16、128、512或1024像素的大小。这在某些平台（如Windows 和 MacOS）上并不是完全必要的，但它肯定会提高这些目标的兼容性，而对于HTML5和设备来说，它是必不可少的，非常重要的是你要记住这一点，否则你以后可能会遇到问题。

语法。

surface_create(w, h);

争论类型 描述
w The width of the surface to be created.
h The height of the surface to be created.

返回。

Surface ID

例子。

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
    surface_set_target(surf);
    draw_clear_alpha(c_black, 0);
    surface_reset_target();
    view_surface_id[0] = surf;
}

上面的代码检查一个表面是否存在，如果不存在，它将创建一个1024像素宽和1024像素高的表面，将索引分配给变量 "surf"。然后将绘图目标设置为新的表面，在将绘图目标重置为显示器之前，将其清空并使其透明。最后，这个曲面被分配到一个视图中。
