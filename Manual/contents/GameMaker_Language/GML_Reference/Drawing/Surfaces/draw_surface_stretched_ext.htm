绘画_表面_伸展_扩展

这个函数的功能与 draw_surface_stretched()函数一样，但增加了在绘制表面时设置颜色混合和alpha值的功能（类似于函数 draw_surface_ext()).

语法。

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

争论 类型 描述
id 表面标识 The unique ID value of the surface to draw.
x 真实的 The x position of where to draw the surface.
y 真实的 The y position of where to draw the surface.
w 真实的 The width at which to draw the surface.
h 真实的 The height at which to draw the surface.
colour 颜色 The colour with which to colour the surface.
alpha 真实的 The alpha with which to blend the surface.

返回。

N/A

例子。

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

这将在实例的X/Y位置绘制给定的表面，它将被拉伸到占据200x200像素的区域，没有混合，但部分透明。
