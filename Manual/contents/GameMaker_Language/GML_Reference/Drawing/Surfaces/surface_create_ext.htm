surface_create_ext

这个功能允许你把一个表面附加到你的网页中已经存在的画布元素上，这意味着你可以有效地把你的游戏的一部分分割开来，在页面中的不同地方进行绘制。为此，你必须在游戏的 *.html 页面中正确定义画布元素，使用正确的尺寸和名称来对应你想创建的表面。因此，你会有你的 "主 "画布，然后是你的次要表面画布元素，这些元素将使用这个功能分配给正确的表面。下图是一个有三个画布元素的页面如何设置的例子。

页面布局可能很棘手，特别是如果你希望所有的元素都能正确排队，但一旦完成了创建布局的艰巨任务，你就可以使用游戏选项的HTML5标签将这个 html 文件作为游戏的默认页面文件。接下来你应该做的是设置你的room 和视图，因为每个表面都需要与一个特定的视图相关联以 "捕捉 "游戏图像（见视图变量 view_surface_id[0...7]).下面的图片显示了上述画布例子的游戏room ，将如何设置。

最后，你将使用这个函数来创建曲面，名称与画布元素的名称相同，尺寸与该画布的尺寸相对应。这个函数将返回曲面的索引，这个索引应该保存在一个变量中，以便以后调用函数。当表面第一次被创建时，它可能包含 "噪音"，因为基本上它只是一个被搁置的内存区域（而且该内存可能仍然包含信息），所以你可能想在使用类似的函数之前清除表面 draw_clear_alpha().

注意：这个功能只适用于HTML5模块。

语法。

surface_create_ext(name, w, h);

争论类型 描述
name The name of the canvas element to link the surface to.
w The width of the surface to be created.
h The height of the surface to be created.

返回。

Surface ID

例子。

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

上述代码创建了两个不同大小的表面，将每个表面分配给不同的画布元素，然后将这些表面分配给两个视图，这样就可以捕捉到room 的正确部分。
