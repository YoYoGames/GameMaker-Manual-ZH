应用程序_表面_使能

通过这个函数，你可以启用或禁用应用程序的表面。默认情况下，应用程序表面被启用，所有的绘图事件都将在该表面上完成，而表面则被绘制到屏幕上。然而，在一些旧的设备和特定的芯片组上，这可能会导致较差的性能。在这些情况下，你可以switch ，关闭对应用程序表面的绘制，然后所有的绘制事件将直接在屏幕上完成buffer 。

你可以在游戏中的任何时候使用这个功能来启用或禁用应用面，但你应该注意，它需要至少一帧（步）的变化才会生效。将其关闭 将switch ，关闭所有长宽比校正，在目标平台的游戏选项中设置。这意味着你应该有一些方法，将游戏缩放到所需的设备、窗口或屏幕尺寸。如果你再次switch ，那么游戏选项将在下一步再次生效。

建议你谨慎使用这个功能，因为它将影响到所有东西在屏幕上的绘制方式。一般来说，你会希望把它作为游戏菜单中的一个选项，或者在游戏开始时只调用一次，并围绕启用或禁用它来编写代码。

语法。

application_surface_enable(enable);

争论类型 描述
enable Set to enable (true) 或禁用(false)应用程序表面。

返回。

例子。

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

上述代码检查按键，并根据其状态（如选项菜单）切换应用程序表面的开启或关闭。
