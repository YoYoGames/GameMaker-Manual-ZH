surface_get_texture

这个函数返回曲面的特殊指针 texture 。然后这个值可以在其他绘制函数中使用，特别是在一般的三维和一些二维原始函数中。

注意在处理表面时，由于它们被储存在texture 内存中，它们有可能在任何时候都不存在。你应该在直接引用表面之前，用 surface_exists()来检查曲面是否存在，然后再直接引用它们。

语法。

surface_get_texture(surface_id);

争论 类型 描述
surface_id 表面标识 The ID of the surface to get the width of.

返回。

Texture

例子。

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

上面的代码将绘制一个4vertex 的三角形条状图，其纹理是由 "tex "变量中的texture ，而这个变量本身就是取自先前创建的表面。
