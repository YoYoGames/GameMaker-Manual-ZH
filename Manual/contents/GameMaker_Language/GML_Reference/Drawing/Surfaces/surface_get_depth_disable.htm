表面_获取深度_禁用

这个函数检查是否启用了为曲面自动生成深度buffer 。通常情况下，所有的表面都有深度buffers ，所以如果你给它们绘制3Dobjects ，那么它就会按深度正确排序。然而，分配深度buffers ，基本上会使表面的大小翻倍，这可能是一个过度和不必要的开销，特别是如果你的游戏是非常密集的内存或主要是2D。在这些情况下，你可以用这个函数来检查，如果需要的话，可以用这个函数来禁用表面的深度buffer 。 surface_depth_disable().

语法。

surface_get_depth_disable();

返回。

例子。

if (!surface_exists(surf))
{
    if surface_get_depth_disable() == false
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

上面的代码会检查给定的曲面是否存在，如果不存在，就会检查曲面深度的当前状态buffer ，如果它是启用的，就会改成禁用，最后才会创建这个曲面。
