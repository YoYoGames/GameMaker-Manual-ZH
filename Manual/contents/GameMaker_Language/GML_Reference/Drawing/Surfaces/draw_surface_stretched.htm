绘画_表面_伸展

这个函数只是取一个表面，并将其拉伸到给定的宽度和高度，使其占据该区域。与 draw_surface()你可以指定一个曲面，然后指定曲面在room ，最后指定宽度和高度（必须是像素值）。

语法。

draw_surface_stretched(id, x, y, w, h);

争论 类型 描述
id 表面标识 The unique ID value of the surface to draw.
x 真实的 The x position of where to draw the surface.
y 真实的 The y position of where to draw the surface.
w 真实的 The width at which to draw the surface.
h 真实的 The height at which to draw the surface.

返回。

N/A

例子。

draw_surface_stretched(surf, 10, 10, 100, 100);

这将画出变量 "surf "中索引的曲面，其左角在(10,10)。它的宽度和高度都被设置为100，这是它所占据的空间，与曲面的实际宽度和高度无关。
