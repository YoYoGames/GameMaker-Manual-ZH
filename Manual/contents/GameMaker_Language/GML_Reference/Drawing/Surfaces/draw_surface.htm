绘制表面

用这个函数可以在room ，在指定的位置画出一个曲面，曲面的左上角在指定的x/y位置。

语法。

draw_surface(id, x, y);

争论 类型 描述
id 表面标识 The unique ID value of the surface to draw.
x 真实的 The x position of where to draw the surface.
y 真实的 The y position of where to draw the surface.

返回。

N/A

例子。

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

上面的代码在与摄像机视图[0]相同的位置绘制了索引为 "surf "的表面。
