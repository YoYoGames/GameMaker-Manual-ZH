surface_set_target_ext

这个函数是与着色器函数一起使用的，为本地级别的shaders （DX9、DX11、OpenGL）设置MRT（0 - 3）。

注意：在处理表面时，由于它们被储存在texture ，它们有可能在任何时候都不存在了。你应该在直接引用表面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_set_target_ext(index, surface_id);

争论类型 描述
index The render target index to use (from 0 to 3).
surface_id The ID of the surface to use.

返回。

例子。

surface_set_target_ext(0, surf);

上面的代码将把渲染目标0设置为变量 "surf "中索引的表面ID。
