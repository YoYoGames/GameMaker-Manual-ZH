绘画_表面_倾斜

这个函数会取一个曲面，然后在整个tile ，从你在函数中给出的坐标开始，反复地进行room 。

语法。

draw_surface_tiled(id, x, y);

争论 类型 描述
id 表面标识 The unique ID value of the surface to draw.
x 真实的 The x position of where to draw the surface.
y 真实的 The y position of where to draw the surface.

返回。

N/A

例子。

draw_surface_tiled(surf, x, y);

这将在实例自己的x和y位置上绘制以 "surf "为索引的表面，并在每个方向上铺设room 。
