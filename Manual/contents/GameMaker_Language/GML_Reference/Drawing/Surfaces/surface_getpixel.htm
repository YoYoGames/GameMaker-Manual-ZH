surface_getpixel

这个函数可以用来从一个表面获得一个特定像素的颜色，使用表面的局部坐标，其中（0,0）是左上角。这个函数不应该经常使用，因为它非常慢，可能会导致游戏的暂停。

注意：在处理表面时，由于它们被储存在texture ，它们有可能在任何时候都不存在了。你应该在直接引用表面之前，先用 surface_exists()来检查曲面是否存在，然后再直接引用它们。

语法。

surface_getpixel(surface_id, x, y);

争论类型 描述
surface_id The ID of the surface.
x The x position on the surface from which to get the pixel.
y The y position on the surface from which to get the pixel.

返回。

Colour Constant

例子。

col = surface_getpixel(surf, 56, 78 );

这将返回变量 "surf "中索引的表面的坐标（56,78）处的像素的颜色。
