表面_存在

由于表面的不稳定性，这个功能在处理表面时是必不可少的。曲面总是保存在texture 内存中，这意味着它们可以在不同的时刻被销毁（例如，当窗口的屏幕保护程序启动时，或者在Android 设备上最小化时），所以在对曲面做任何事情之前，你应该总是检查它是否存在（这包括将它画到屏幕上）。下面的示例代码显示了在实例的绘制事件中对该命令的典型使用，以检查表面并在它被删除时重新创建它（注意，该表面最初是在object 的创建事件中创建的）。

语法。

surface_exists(surface_id);

争论类型 描述
surface_id The ID of the surface to check.

返回。

例子。

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
}
else
{
    if view_current = 1
    {
        draw_surface(surf,0,0);
    }
}

上面的代码将检查变量 "surf "中索引的曲面是否存在，如果不存在，它将重新创建它。如果它确实存在，它将检查当前正在绘制的视图，如果是视图[1]，它将绘制这个曲面。
