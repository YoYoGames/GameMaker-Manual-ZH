绘画_表面_内容

这个函数会像在函数中一样绘制给定的曲面 draw_surface()中的做法，但会有额外的选项来改变被绘制的曲面的比例、混合、旋转和阿尔法。改变这些值并不以任何方式修改资源（只修改它的绘制方式）。

语法。

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

争论 类型 描述
id 表面标识 The unique ID value of the surface to draw.
x 真实的 The x position of where to draw the surface.
y 真实的 The y position of where to draw the surface.
xscale 真实的 The horizontal scale.
yscale 真实的 The vertical scale.
rot 真实的 The rotation or angle to draw the surface.
col 颜色 The colour with which to blend the surface.
alpha 真实的 The alpha transparency for drawing the surface.

返回。

N/A

例子。

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

上面的代码将在room ，在(0,0)的位置画出一个在变量 "surf "中索引的表面，其比例是原来的两倍，混合红色和半透明。
