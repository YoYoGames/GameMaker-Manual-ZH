应用程序_表面处理_绘图_启用

你可以使用这个功能来启用或禁用应用程序表面的自动绘制。默认情况下是启用的，但在很多情况下，你会希望在什么时候以及如何绘制表面（例如，当使用shaders ），在这种情况下 你可以用这个函数把它设置为false，这样你就可以在你需要的时候自己绘制。值得注意的是，当你在switch 关闭自动绘图，并自己绘制应用程序的表面时，你可能会看到某些问题 sprites 和表面本身的 alpha 分量的问题。这是因为GameMaker在开启自动绘图时，会在没有alpha混合的情况下绘制应用程序的表面。如果你switch 关闭自动绘图，那么你需要 自己来处理这个问题，使用下面的代码（例如）。

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

语法。

application_surface_draw_enable(flag);

争论类型 描述
flag The flag will be either true (启用，默认值）或 （禁用）。 false

返回。

例子。

application_surface_draw_enable(false);

上述代码将switch ，关闭应用面的自动绘制。
