surface_free

当你在使用曲面时，你应该在使用完后随时使用这个功能。表面会占用内存空间，因此需要被删除，通常是在room ，但也可以在任何时候进行，这取决于你对它们的使用。如果不这样做，会导致内存泄漏，最终会使你的游戏变慢并崩溃。

注意：在处理表面时，由于它们被储存在texture ，它们有可能在任何时候都不存在了。你应该在直接引用表面之前，先用 surface_exists()来检查曲面是否存在，然后再直接引用它们。

语法。

surface_free(surface_id);

争论类型 描述
surface_id The ID of the surface to be freed.

返回。

例子。

if keyboard_check_pressed(vk_escape)
{
    surface_free(surf);
    room_goto(rm_Menu);
}

上面的代码检查是否有按键，如果检测到有按键，它将释放为变量 "surf "中索引的表面保留的内存，然后改变room 。
