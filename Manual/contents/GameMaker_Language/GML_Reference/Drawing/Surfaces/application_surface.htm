application_surface

这个全局的内置变量可以用来访问应用程序的表面，用于任何表面函数。这个表面是永久可用的，是GameMaker绘制游戏的地方。

常规的绘图事件（绘图开始、绘图和绘图结束）默认是在这个表面上绘图。其他的绘制事件（如前/后绘制和绘制GUI）不会在应用程序表面绘制。关于这些事件如何工作的更多信息，请参见绘图事件。

语法。

application_surface

返回。

Surface ID

例子。

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

上述代码将调整应用程序表面的大小，使其与GUI层的比例为1：1。
