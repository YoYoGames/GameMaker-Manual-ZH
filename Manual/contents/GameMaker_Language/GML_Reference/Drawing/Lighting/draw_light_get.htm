draw_light_get

  • [0] = 启用/禁用 (true / false)
  • [1] = 光类型常数(见下表)
  • [2] = X位置
  • [3]=y位置
  • [4] = Z位置
  • [5] = 光线半径（仅适用于点状灯光，定向灯光将为0）。
  • [6]=浅色（一个真实的）。

光的类型（元素[1]）可以是以下两个常数之一。

灯光类型恒定
恒定 描述
lighttype_dir 该灯是一种定向灯
lighttype_point 该灯是一个点光源

语法。

draw_light_get(ind);

争论 类型 描述
ind 真实的 The index number of the light (from 0 to 7)

返回。

Array

例子。

light_a = draw_light_get(1);
if (light_a[5] < 200)
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

上述代码将获得用于定义灯光的数值，索引为 "1"，然后检查半径，如果它小于200，它将被增加，灯光半径被设置为新的数值。
