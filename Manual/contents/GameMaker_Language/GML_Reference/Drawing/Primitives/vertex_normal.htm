顶点_正常

此函数会将表面法线数据添加到当前正在为自定义基元定义的vertex 。您可以提供要写入数据的buffer 以及法线的 x、y 和 z 分量部分。

语法。

vertex_normal(buffer, nx, ny, nz);

争论 类型 描述
buffer 顶点Buffer ID The buffer to write the information to.
nx 真实的 The x component of the surface normal.
ny 真实的 The y component of the surface normal.
nz 真实的 The z component of the surface normal.

返回。

N/A

例子。

vertex_normal(buff, 0, 1, 1);

上述代码将设置被定义的vertex 的表面法线。
