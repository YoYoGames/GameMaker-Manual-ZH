顶点_位置_三维

此函数会将三维位置数据添加到当前正在为自定义基元定义的vertex 。您提供了要将数据写入的buffer ，以及用于绘图的x、y和z位置。

语法。

vertex_position_3d(buffer, x, y, z);

争论 类型 描述
buffer 顶点Buffer ID The buffer to write the information to.
x 真实的 The x position.
y 真实的 The y position.
z 真实的 The z position.

返回。

N/A

例子。

vertex_position_3d(buff, x - 100, y - 125, 0);

上述代码将设置被定义的vertex 的位置。
