draw_primitive_begin_texture

必须在定义纹理基元的顶点之前调用此函数。您必须给出要使用的基元的种类（更多信息见 draw_primitive_begin()更多信息）和要使用的texture 的ID，该 ID 可以是sprite 或背景图像asset 。这个asset id可以从函数中得到 sprite_get_texture()例如（使用-1表示没有texture ）。

注意对于一个texture ，它的大小必须是2的幂，即：32x32，128x128，等等。

语法。

draw_primitive_begin_texture(kind, tex)

争论 类型 描述
kind 原始类型常数 The kind of primitive you are going to draw.
tex 纹理指针 The texture to use with the primitive.

返回。

N/A

例子。

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

上述代码将绘制一个4vertex 的三角形条（构成一个矩形），用 "tex "变量中持有的texture 进行纹理处理，整个texture ，以覆盖完成的基元。
