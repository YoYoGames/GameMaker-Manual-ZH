vertex_create_buffer

通过这个函数，你可以创建一个新的vertex buffer 。这是由GameMaker创建的一个特殊的增长 buffer ，它被预先格式化，以便在构建基元时使用（例如，与shaders ，使用）。该函数将返回buffer 的索引，然后在所有对它的进一步调用中使用。

使用用此函数创建的vertex buffer 时，您只需调用 数据来开始定义您的自定义基元。 vertex_begin()开始向其分配vertex 数据，以开始定义您的自定义基元，然后将其保存在buffer 中，以备提交给shader 。必要时可重复使用buffer （除非您使用了 vertex_freeze()函数），每次调用 vertex_begin()擦除先前的buffer 数据，以便接受新数据。

语法。

vertex_create_buffer();

返回。

Vertex Buffer ID

例子。

v_buff = vertex_create_buffer();

上面的代码将创建一个新的vertex buffer ，并将其句柄存储在变量 "v_buff "中。
