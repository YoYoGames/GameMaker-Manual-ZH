顶点_从缓冲区创建缓冲区_ext

与函数 vertex_create_buffer()，此函数将创建一个新的vertex buffer ，只是现在它所存储的vertex 数据是从被指定为源的常规buffer 中复制的。buffer 已经预设了用于构建基元的vertex 格式，以便与（例如）shaders 一起使用，你还可以在源buffer 中提供一个要复制的偏移量以及最终buffer 应该有的顶点数。请注意，如果顶点数量与被复制的顶点数量不一致，你可能会得到损坏的vertex 数据。

语法。

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

争论 类型 描述
buffer 缓冲区ID The buffer to create the vertex buffer from.
format 原始类型常数 The primitive vertex format to use.
src_offset 真实的 The offset within the the source buffer to copy from.
vert_num 真实的 The number of vertices the buffer should have.

返回。

Vertex Buffer ID

例子。

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

上面的代码将创建一个新的vertex 格式，然后从先前创建的常规vertex 创建一个新的buffer buffer ，将自定义的vertex 格式应用于它，偏移量为0。该函数告诉新的vertex buffer ，它应该期待512个顶点。
