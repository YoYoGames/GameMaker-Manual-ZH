顶点删除缓冲区（vertex_delete_buffer

这个函数可以用来从系统内存中删除一个先前创建的vertex buffer （见 vertex_create_buffer()）从系统内存中删除。这也会为其包含的任何自定义基元删除所有vertex 数据。

语法。

vertex_delete_buffer(buffer);

争论 类型 描述
buffer 顶点Buffer ID The vertex buffer to be removed.

返回。

N/A

例子。

vertex_delete_buffer(v_buff);

上述代码将从内存中删除变量 "v_buff "中索引的buffer 。
