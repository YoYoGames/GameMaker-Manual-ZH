顶点_端

通过此函数，您可以结束对自定义基元的构建。一旦调用此命令，该基元就可在 vertex_submit()函数中使用，以便在shader 中使用，或者可以使用buffer 冻结(使vertex buffer 只用于只读，速度更快)。 vertex_freeze().

语法。

vertex_end(buffer);

争论 类型 描述
buffer 顶点Buffer ID The buffer to end writing to.

返回。

N/A

例子。

vertex_end(buff);

上述代码将结束对自定义基元的定义。
