vertex_format_add_position_3d

告诉GameMaker接受三维位置数据（X、Y和Z）作为正在创建的新vertex 格式的一部分。

语法。

vertex_format_add_position_3d();

返回。

例子。

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position_3d();
my_format = vertex_format_end();

上述代码将创建一个只有颜色和位置值的新的vertex 格式，然后将格式ID存储在变量 "my_format "中。
