顶点提交（vertex_submit

您可以使用此函数将vertex buffer 的内容提交给图形管道，以便与shader 一起使用。您提供要使用的buffer 索引、要使用的基本基元类型（参见下面的常数）和要使用的texture 。基本基元类型仅用于分配您存储在buffer 中的顶点的绘制和连接顺序，但每个顶点使用的实际数据将是您在创建vertex buffer 时定义的数据。

注意：这个功能只能在抽奖活动中使用。

关于可用的不同基础基元的直观例子，见下图。

恒定 描述
pr_pointlist 一个由点的列表组成的基元。
pr_linelist 一个由列表中的单个行组成的基元。
pr_linestrip 一个由连续的线条组成的原始物。
pr_trianglelist 一个由列表中的单个三角形组成的基元。
pr_trianglestrip 一个由连续的三角形条带组成的原始物。

语法。

vertex_submit(buffer, primitive, texture);

争论 类型 描述
buffer 顶点Buffer ID The buffer to use.
primitive 原始类型常数 The primitive base type.
texture 纹理指针 The texture to use (or -1 for no texture).

返回。

N/A

例子。

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

vertex 上述代码将提交在变量 "buff "中索引的buffer shader ，以便用texture "shader_prim "进行绘制，使用一个三角形列表作为基础基元，并使用运行代码的实例的sprite 。
