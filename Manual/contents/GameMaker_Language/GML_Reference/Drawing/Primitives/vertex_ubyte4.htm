vertex_ubyte4

这个函数将在vertex 数据中加入四个无符号字节值（0 - 255）。vertex 的格式必须正确，以接受这个用 vertex_format_add_custom()函数。

语法。

vertex_ubyte4(buffer, byte, byte, byte, byte);

争论 类型 描述
buffer 顶点Buffer ID The buffer to write the information to.
byte 真实的 The first input value.
byte 真实的 The second input value.
byte 真实的 The third input value.
byte 真实的 The fourth input value.

返回。

N/A

例子。

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

上面的代码将向正在定义的vertex 数据添加四个值。
