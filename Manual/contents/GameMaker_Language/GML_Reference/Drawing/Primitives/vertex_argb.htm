顶点_argb

此函数将为当前正在为自定义基元定义的vertex 设置ARGB值。您提供了要写入数据的buffer 以及希望作为单个 32 位无符号整数使用的红、绿、蓝和阿尔法值 - 阿尔法样本在最高 8 位，其次是红样本、绿样本，最后是蓝样本在最低 8 位。你可以使用十六进制符号（$AARRGGBB）一个实数或使用任何 make_colour_*() 函数来定义颜色值。

语法。

vertex_argb(buffer, argb);

争论 类型 描述
buffer 顶点Buffer ID The buffer to write the information to.
argb 颜色 The ARGB value to set.

返回。

N/A

例子。

vertex_argb(buff, $FFFFFFFF);

上述代码将把被定义的vertex 的ARGB值设置为白色。
