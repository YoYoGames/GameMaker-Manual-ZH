绘制原始数据的开始

在你定义任何基元之前，必须调用这个函数。有6种类型的基元可以定义为以下常量。

原始类型常数
恒定 描述
pr_pointlist 一个点的清单
pr_linelist 一条线列表
pr_linestrip 一条线条
pr_trianglelist 一个三角形的列表
pr_trianglestrip 一个三角形的带子
pr_trianglefan 一个三角形的风扇

下面的图片基本上说明了这些应该是什么样子的，还有你应该定义顶点的顺序。

请注意，在一些平台上（Windows, UWP, XBox）， pr_trianglefan 类型不是原生支持的，因此GameMaker在编译游戏时做了一个转换，以使它们工作。这意味着在这些平台上， pr_trianglefan 类型的使用速度会比其他平台慢很多。这也要注意，要在HTML5上使用这个功能，你必须在游戏选项中启用WebGL。

语法。

draw_primitive_begin(kind)

争论 类型 描述
kind 原始类型常数 The kind of primitive you are going to draw.

返回。

N/A

例子。

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

上述代码将画出由基元构成的四分之三的圆。
