vertex_float1

这个函数将在vertex 数据中增加一个浮点值。vertex 的格式必须正确，以接受这个用 vertex_format_add_custom()函数。

语法。

vertex_float1(buffer, float);

争论 类型 描述
buffer 顶点Buffer ID The buffer to write the information to.
float 真实的 The input value.

返回。

N/A

例子。

vertex_float1(buff, 0.05);

上面的代码将为正在定义的vertex 数据添加一个浮点值。
