vertex_format_add_normal

告诉GameMaker接受表面法线数据（nx、ny和nz）作为正在创建的新vertex 格式的一部分。

语法。

vertex_format_add_normal();

返回。

例子。

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_normal();
my_format = vertex_format_end();

上面的代码将创建一个新的vertex 格式，其中只有texture 和表面法线值，然后将格式ID存储在变量 "my_format "中。
