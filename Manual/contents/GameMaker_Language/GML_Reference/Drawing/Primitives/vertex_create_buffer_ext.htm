顶点_创建缓冲区_ext

与函数 vertex_create_buffer()，此函数将创建一个新的vertex buffer 。这是由GameMaker创建的一个特殊的增长 buffer ，它被预先格式化，以便在构建基元时使用shaders 。你可以为buffer 指定一个初始的起始大小（以字节为单位），它将返回一个buffer 的索引，然后应该在所有进一步调用buffer 。

语法。

vertex_create_buffer_ext(size);

争论类型 描述
size The initial size of the buffer (in bytes).

返回。

Vertex Buffer ID

例子。

v_buff = vertex_create_buffer_ext(1024 * 1024);

上述代码将创建一个新的vertex buffer ，最初大小为1MB，并将其句柄存储在变量 "v_buff "中。
