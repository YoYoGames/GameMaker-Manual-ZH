顶点格式添加顶点坐标（Vertex_format_add_texcoord

告诉GameMaker接受texture 位置数据（u和v）作为正在创建的新vertex 格式的一部分。

语法。

vertex_format_add_texcoord();

返回。

例子。

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

上面的代码将创建一个新的vertex 格式，只有texture 和位置值，然后将格式ID存储在变量 "my_format "中。
