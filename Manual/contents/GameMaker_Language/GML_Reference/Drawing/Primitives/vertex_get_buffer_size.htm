顶点_get_buffer_size

通过这个函数，你可以得到给定的vertex buffer ，以字节为单位的大小。

语法。

vertex_get_buffer_size(buffer);

争论 类型 描述
buffer 顶点Buffer ID The buffer to get the size of.

返回。

Real

例子。

bufferBytes = vertex_get_buffer_size(buff);

上述代码将获得所给的vertex buffer ，并将该值存储在变量 "bufferBytes "中。
