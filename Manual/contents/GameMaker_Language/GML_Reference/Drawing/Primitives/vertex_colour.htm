顶点颜色

如果您定义的vertex 格式需要一个颜色值，您可以使用此函数将该数据添加到正在为当前基元定义的vertex 。该函数需要一个buffer 来存储数据，并将接受一个颜色常数或一个十六进制值（使用 BGR 的标准GameMaker格式，例如：$FF0000 表示蓝色）以及一个从 0（透明）到 1（完全不透明）的 alpha 值。

语法。

vertex_colour(buffer, colour, alpha);

争论 类型 描述
buffer 顶点Buffer ID The buffer to write the information to.
colour 颜色 The colour for this vertex (can be a constant or a hex value).
alpha 真实的 The alpha value for the vertex (from 0 to 1).

返回。

N/A

例子。

vertex_colour(b, c_white, 1);

上面的代码将把被定义的vertex 定为白色，alpha值为1。
