顶点冻结（vertex_freeze

此函数可用于 "冻结 "一个vertex buffer 。此buffer 成为只读，这意味着如果您需要更改它，您必须删除整个buffer 并重新创建它。冻结的buffer 可以比正常的、动态的shader 更快地提交给buffer ，对于那些需要在关卡或游戏期间保持不变的自定义基元的效果，推荐使用此函数。

该函数在成功冻结指定的vertex buffer ，将返回0，但是如果由于任何原因失败，它将返回-1。

语法。

vertex_freeze(buffer);

争论 类型 描述
buffer 顶点Buffer ID The vertex buffer to freeze.

返回。

Real (0 for success, -1 for failure)

例子。

// Create event
vertex_freeze(vbuff);

上面的代码将在Create事件中冻结给定的vertex buffer ，所以它可以在其他事件中作为 "静态 "buffer ，以便更快地访问。
