顶点格式的删除

每当你使用完任何创建的矢量格式，都必须调用这个函数。你提供格式的ID值（如由函数 vector_format_end())，这个函数将释放与之相关的内存。注意，如果你在调用这个函数后试图再次使用这个格式，你会得到一个错误。

语法。

vertex_format_delete(formatID);

争论 类型 描述
formatID 顶点格式ID The vertex format to delete

返回。

N/A

例子。

vertex_format_delete(myFormat);

上述代码将从内存中删除在变量 "myFormat "中创建的vertex 格式。
