绘画_设置对齐方式

该函数用于沿水平轴对齐文本，改变水平对齐方式将改变所有进一步绘制文本的位置和方向，默认值为 fa_left 。可以接受以下常数。

恒定 统一口径
fa_left
fa_center
fa_right

语法。

draw_set_halign(halign);

争论 类型 描述
halign 水平对准常数 Horizontal alignment constant (from the table above).

返回。

N/A

例子。

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

上面的代码将在同一条线上画出两个strings ，分数是左对齐的，健康是右对齐的。
