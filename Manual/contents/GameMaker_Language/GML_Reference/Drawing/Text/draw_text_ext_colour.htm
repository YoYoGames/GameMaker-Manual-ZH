draw_text_ext_colour

这个函数是由基础 draw_text()函数与 draw_text_ext()和 draw_text_colour()函数的组合，允许你为文本定义梯度颜色，以及定义行距和每行最大宽度。

注意 除非启用WebGL，否则梯度混合对HTML5目标是不可用的，尽管你仍然可以设置混合颜色，它将把font 与第一个给定的颜色混合。然而，所有以这种方式进行的混合都会创建一个重复的font ，然后存储在cache ，并在需要时使用，这远非最佳状态，如果你使用多种颜色变化，会降低游戏性能。如果有必要，你可以设置font cache 的大小来尝试限制这种情况，使用函数 font_set_cache_size().

语法。

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

争论 类型 描述
x 真实的 The x coordinate of the drawn string.
y 真实的 The y coordinate of the drawn string.
string 字符串 The string to draw.
sep 真实的 The distance in pixels between lines of text.
w 真实的 The maximum width in pixels of the string before a line break.
c1 颜色 The colour for the top left of the drawn text.
c2 颜色 The colour for the top right of the drawn text.
c3 颜色 The colour for the bottom right of the drawn text.
c4 颜色 The colour for the bottom left of the drawn text.
alpha 真实的 The alpha for the text.

返回。

N/A

例子。

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

string 上述代码将绘制用户在键盘上输入的任何文本，每当该行的长度超过300像素时，就将其分割成新的一行。如果出现这种情况，代码还将在各行之间保持5像素的间隔。每行文字将使用蓝色梯度着色，顶部是浅蓝色，底部是深蓝色。
