绘制字体

这个函数将设置font ，以用于所有进一步的文本绘制。这个font ，必须已经被添加到游戏的字体资产中，或者已经用 font_add()或 font_add_sprite().

语法。

draw_set_font(font);

争论类型 描述
font The name of the font to use.

返回。

例子。

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

上述代码将使用变量 "fnt_Game "中索引的font ，以蓝色绘制给定文本。

返回。文本
