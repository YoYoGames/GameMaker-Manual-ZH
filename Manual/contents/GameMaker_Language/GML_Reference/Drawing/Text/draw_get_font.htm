绘制字体

该函数将获得当前为绘制文本分配的font 。如果没有设置font ，该函数将返回-1，或者返回分配给font asset 的ID值（一个正整数）。

语法。

draw_get_font();

返回。

Font Asset or -1

例子。

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

上面的代码得到了当前应用的font ，并对其运行了一个switch 语句，根据font ，应用了一个不同的Y偏移值。然后在绘制一些文本时使用该偏移值。

返回。文本
