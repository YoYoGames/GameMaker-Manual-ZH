draw_text

通过这个函数，你可以在string 内的任何位置绘制任何room （对于绘制实数，你应该使用 string()函数将其转换为文本）。要组合strings ，你可以使用+（见下面的例子），你也可以在一个string 内使用\n来添加一个换行符，这样它就会被画在多行上（关于如何正确格式化一个string 和你可以使用什么转义字符的信息，请看这里）。文本的颜色和阿尔法是由当前的基本阿尔法和颜色值决定的，由 draw_set_alpha()和 draw_set_colour().

注意：文本的实际位置将受到由以下参数设置的对齐值的影响 draw_set_halign()和 draw_set_valign().

语法。

draw_text(x, y, string);

争论类型 描述
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

返回。

例子。

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

上述代码将在实例的x/y位置绘制一个string ，它将使用存储在全局变量 "名称 "中的string ，并将其分成两行。

返回。文本
