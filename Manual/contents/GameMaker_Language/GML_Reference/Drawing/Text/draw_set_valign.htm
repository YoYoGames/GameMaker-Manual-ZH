draw_set_valign

这个函数用于沿垂直轴对齐文本，改变垂直对齐将改变所有进一步绘制文本的位置和方向，默认值为 fa_top 。可以接受以下常数。

恒定 统一口径
fa_top
fa_middle
fa_bottom

语法。

draw_set_valign(valign);

争论 类型 描述
valign 垂直对准常数 Vertical alignment constant (from the table above).

返回。

N/A

例子。

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

上面的代码将以文本的最中心为中心绘制分数。

返回。文本
