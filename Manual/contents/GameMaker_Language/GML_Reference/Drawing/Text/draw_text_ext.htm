绘图_文本_文本

这个函数将以类似于 draw_text()只是现在你可以设置每行文本之间的空间--如果文本占据了超过一行的话--并限制每行的宽度（像素） string 的宽度（像素），这样，如果任何一行超过这个值，GameMaker将自动将文本分割到下一行最近的可用白位（如果文本没有白位，那么它将超过这个最大的 宽度值）。请注意，由于这个原因，任何放置在string 开始的空白将在被解析绘制之前被剥离出来。还要注意的是，如果行间距参数的值为-1，则默认为基于 "M "字符高度的间距。 所选font 中的 "M "字符的高度。

语法。

draw_text_ext(x, y, string, sep, w);

争论类型 描述
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

返回。

例子。

draw_text_ext(100, 50, keyboard_string, 3, 300);

上述代码将绘制用户在键盘上输入的任何文字，每当该行的string 长度超过300像素时，就将其分割到新的一行。如果发生这种情况，代码还将在行与行之间保持3像素的间隔。

返回。文本
