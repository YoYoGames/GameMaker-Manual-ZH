文本

有许多函数可供你用来绘制各种形式的文本。应该注意的是，其中一些函数在绘制时以动态方式改变文本的颜色和大小，但在使用HTML5或设备时，这些函数可能会降低你的游戏速度或看起来很差，因此，拥有各种大小的font ，而不是使用按比例绘制，或者拥有大型font ，然后按比例绘制，而不是按比例绘制，可能会更好。另外，如果你需要给大块的文字上色，最好是使用 draw_set_colour()和 draw_set_alpha()函数和简单的 draw_text ，而不是那些有颜色成分的扩展函数。

以下函数用于设置绘制时使用的font ，以及绘制时文本的定位。

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

这些函数可以用来在屏幕上绘制文本，可以是简单的，也可以是带有变换和颜色的。

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

以下函数也存在，专门用于绘制使用内置高分系统创建的高分值。

  • draw_highscore
