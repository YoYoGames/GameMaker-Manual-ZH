绘画_获取阿尔法（draw_get_alpha

该函数返回绘制alpha的当前值，其范围在0（完全透明）和1（完全不透明）之间。绘制alpha会影响所有绘制函数的透明度，可以用 draw_set_alpha() 函数来设置。

语法。

draw_get_alpha()

返回。

例子。

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

上面的代码将当前的绘图alpha存储到一个局部变量中，并根据一个实例变量改变绘图alpha。绘制完一些文本后，它将alpha值重新设置为存储在本地变量中的值。
