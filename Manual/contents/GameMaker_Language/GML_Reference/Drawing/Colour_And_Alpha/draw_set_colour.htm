绘画_设置颜色

通过这个功能，你可以设置游戏的基本绘图颜色。

这将影响fonts 、窗体、基元和3D的绘制，但是它不会影响sprites （手动或由实例绘制）。如果任何受影响的图形是用它们自己的颜色值绘制的，这个值将被忽略。

语法。

draw_set_colour(col);

争论 类型 描述
col 颜色 The colour to set for drawing.

返回。

N/A

例子。

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上面的代码将在指定的位置绘制一些文字，并通过修改绘制的alpha和颜色来创建阴影效果。
