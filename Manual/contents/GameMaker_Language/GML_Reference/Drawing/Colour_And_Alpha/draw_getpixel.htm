draw_getpixel

通过这个函数，你可以得到正在绘制到当前渲染目标的任何像素的颜色值。这意味着结果将取决于调用该函数的事件，也取决于正在使用的目标表面。注意，这个函数不会返回任何alpha值，为此你应该使用 draw_getpixel_ext().

重要提示 这个功能会造成巨大的性能冲击，所以只有在绝对必要的时候才可以使用。

语法。

draw_getpixel(x, y);

争论类型 描述
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

返回。

Colour Constant

例子。

draw_set_colour(draw_getpixel(500,200));

上述代码将把所有文本、表格和基元的绘图颜色设置为游戏屏幕上(500,200)处像素的颜色。
