绘画_清空

这个函数可以用来清除整个屏幕（没有阿尔法混合）到给定的颜色，并且只在一个实例的绘制事件中使用（如果在任何其他事件中使用，它将不会显示）。它也可以用于在新创建的表面时清除它们。

语法。

draw_clear(col);

争论类型 描述
col The colour with which the screen will be cleared

返回。

例子。

draw_clear(c_blue);

这将清除屏幕上的蓝色。
