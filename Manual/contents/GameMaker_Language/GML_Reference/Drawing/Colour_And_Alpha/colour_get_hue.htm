colour_get_hue

这个函数将返回给定颜色的色调。这是 "纯 "色调，是定义颜色的色调、饱和度和值（亮度）方法的一部分。下面的图片说明了这个值如何对应于 HSV色阶。

语法。

colour_get_hue(col);

争论类型 描述
col The colour to check

返回。

例子。

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

上面的代码得到了用于制作颜色常量 "c_teal "的色调，然后以50:50的比例与白色合并，将得到的颜色存储在变量 "col "中。
