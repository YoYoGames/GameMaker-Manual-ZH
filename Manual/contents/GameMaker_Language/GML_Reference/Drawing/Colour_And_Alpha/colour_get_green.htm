颜色_获取_绿色

这个函数返回用于制造给定颜色的绿色量，其值在0和255之间，其中0是没有绿色，255是全部绿色。下面的图片说明了这一点。

语法。

colour_get_green(col);

争论类型 描述
col The colour to check

返回。

例子。

g_comp = colour_get_green(c_teal);

上面的代码将获得颜色常数 "c_teal "的绿色分量，并将该值存储在变量 "g_comp "中。
