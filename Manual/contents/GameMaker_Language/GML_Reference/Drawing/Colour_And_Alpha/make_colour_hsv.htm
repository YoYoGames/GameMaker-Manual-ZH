make_colour_hsv

GameMaker提供了这个功能（以及其他功能），允许用户制作自己的颜色。这个特殊的函数需要三个组成部分，色调、饱和度和值（也被称为 "亮度"）来创建所需的颜色。这些值在0和255之间，所以你可以用它来制作16,777,216(256*256*256)种颜色!下面你可以看到这些组件分离后的图像。

左边的图片是函数的各个组成部分的分解，然后右边是改变这些组成部分如何影响最终颜色的说明。正如你所看到的，即使不改变色调（它保持在0，红色），我们也可以通过其他两个设置极大地改变颜色，饱和度使色调从灰色（0）变成纯色（255），而数值使它从更深（0）变成更浅（255）。

语法。

make_colour_hsv(hue, sat, val);

争论类型 描述
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

返回。

Colour Constant

例子。

col = make_colour_hsv(100, 145, 255);

上面的代码使用该函数来创建一个颜色，并将其值存储在变量 "col "中，以便以后使用。
