draw_set_alpha

通过这个函数，你可以设置游戏的基本绘制alpha。这个值可以从0到1设置，0是完全透明的，1是完全不透明的（默认值），它将影响所有进一步的绘制，包括背景、sprites 、fonts 。 基元和三维。

请注意，如果你正在绘制任何assets ，并且作为函数的一部分提供了一个alpha参数（例如，当使用 draw_sprite_ext())，那么这里设置的alpha值将被 覆盖，所以使用 draw_sprite_ext() 绘制一个alpha参数为1的sprite ，将绘制完全不透明的sprite ，而不考虑 draw_set_alpha() 的alpha设置（这在使用 draw_self() 和设置 image_alpha的时候也是如此）。还要注意的是，当一个实例是默认绘制时（即：它有一个sprite ，但没有绘制事件），那么这个 image_alpha 变量将被钳制在用 draw_set_alpha() 设置的alpha上，这样， image_alpha 为1，绘图alpha为0.5，将以alpha 0.5绘制sprite ，而 image_alpha 为0.25，绘图alpha为 0.5将以0.25的α值绘制sprite 。

语法。

draw_set_alpha(alpha);

争论类型 描述
alpha The alpha to set (between 0 and 1)

返回。

例子。

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上面的代码将在指定的位置绘制一些文字，并通过修改绘制的alpha和颜色来创建阴影效果。
