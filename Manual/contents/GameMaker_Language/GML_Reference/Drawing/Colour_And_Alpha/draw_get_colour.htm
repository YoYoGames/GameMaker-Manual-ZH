绘画_获取颜色

此函数返回当前的绘制颜色，用于绘制表格、文本、基元和无纹理的三维模型。这可以通过 draw_set_colour()函数设置。

语法。

draw_get_colour()

返回。

Colour Constant

例子。

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

上面的代码将当前的绘制颜色存储到一个局部变量中，并根据一个实例变量改变绘制颜色。在绘制完一些文本后，它将颜色重新设置为存储在本地变量中的值。
