视频_set_volume

这个函数改变当前加载的视频的音频音量。它的取值在0和1之间，其中0为无声，1为最大音量。

确保只在调用 video_open() ，但在调用 video_close() 之前调用，否则它不会做任何事情（因为不会有视频加载）。

在一些平台上，这个功能只有在视频完全加载后才会起作用，所以要使用Async Callbacks来找出视频的准备时间，然后改变音量。

语法。

video_set_volume(value);

争论 类型 描述
value 真实的 Volume value between 0 and 1

返回。

N/A

例子。

my_video = video_open("splash.mp4");
video_set_volume(0.5);

上面的代码从游戏的Included Files中加载splash.mp4，并将其音量改为0.5（其最大音量的50%）。

返回。视频播放
