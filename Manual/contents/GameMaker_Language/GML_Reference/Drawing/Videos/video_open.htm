视频_打开

这个函数使用指定的path ，打开一个视频文件。path 可以指你项目中包含的文件，用户设备上的本地文件，或从互联网上加载文件的URL（可能不在所有平台上工作）。

由于你一次只能加载一个视频，这个函数将指定的视频加载到唯一可用的 "视频槽 "中，然后通过其他视频函数进行操作。

有与视频播放相关的异步回调。更多信息请参见异步回调。

语法。

video_open(path);

争论 类型 描述
path 字符串 Path to the video file

返回。

N/A

例子。

my_video = video_open("splash.mp4");

上面的代码从游戏的Included Files中加载splash.mp4。

返回。视频播放
