视频_is_looping

这个函数返回当前加载的视频是否被设置为loop （真）或不（假）。你可以通过调用 video_enable_loop() ，将一个视频设置为loop 。

语法。

video_is_looping();

返回。

Boolean

例子。

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

上面的代码检查当前加载的视频是否在循环播放，在这种情况下，通过设置一个自定义实例变量为false，禁用播放器的控制。

返回。视频
