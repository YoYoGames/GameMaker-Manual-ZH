视频_get_volume

该函数返回当前加载的视频的音量，它是一个介于0和1之间的值。你可以用 video_set_volume() 来改变加载视频的音量。

语法。

video_get_volume();

返回。

Real

例子。

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

上述代码将在一个步骤事件中运行。它获取当前正在播放的视频的音量，如果它低于1，它将增加0.02，逐渐增加音量，直到达到最大水平。

返回。视频
