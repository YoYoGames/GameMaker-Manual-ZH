视频_get_duration

该函数返回当前加载的视频的持续时间，单位是毫秒。

语法。

video_get_duration();

返回。

Real

例子。

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

上述代码在按下右方向键时，将正在播放的视频向前移动2秒，而且新位置小于视频的持续时间。然而，如果它不在持续时间内，视频就会关闭。

在这个例子中， video_get_duration() 在一个条件中使用，以检查新的位置是否小于返回的期限。
