视频_get_status

该函数返回当前加载的视频的状态。状态可以是下列常数中的任何一个。

视频状态常数
恒定 描述
video_status_closed 当前没有加载任何视频，或者视频被关闭了。 video_close()
video_status_preparing 视频目前正在准备，还没有开始播放
video_status_playing 目前正在播放视频
video_status_paused 视频暂停了（见 video_pause() ）。

语法。

video_get_status();

返回。

Video Status Constant

例子。

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

上面的代码获取视频的状态，然后检查播放器是否按下了Space。在这种情况下，如果视频正在播放，它就暂停，如果暂停了，它就恢复。

返回。视频
