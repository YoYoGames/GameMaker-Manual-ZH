draw_enable_drawevent

通过这个函数，你可以选择启用（true ）或禁用（false ）游戏中所有实例的绘制事件，从而让你控制事物的绘制方式和时间，如果你想实现 "跳帧 "技术，这很有用。请注意，这不仅仅是阻止实例绘制到屏幕上，它完全抑制了绘制事件，这意味着应该注意，因为任何存在于该事件中的游戏逻辑也将不会被运行。

关于这个函数，需要了解的一件事是，如果你在游戏开始时，在初始帧被渲染之前（即：游戏的第一个object ，第一个room 的创建事件）调用它，那么游戏窗口将不会被渲染。这对于需要一个不需要渲染任何东西的专用server 的多人项目可能很有用，但是请记住，这并不能使它成为无头的，因为GameMaker 不支持 -- 所以你将不能在没有图形界面的server 上运行它。

语法。

draw_enable_drawevent(enable);

争论类型 描述
Enable Set to true 或 。 false

返回。

例子。

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

上面的代码检查一个变量，如果它返回 true ，则启用绘图事件，否则绘图事件被禁用。

返回。绘画
