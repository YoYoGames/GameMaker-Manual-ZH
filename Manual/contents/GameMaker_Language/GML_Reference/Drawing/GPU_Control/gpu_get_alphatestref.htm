gpu_get_alphatestref

你可以使用这个函数来找到阿尔法测试参考的当前值（默认为0，但你可以用 gpu_set_alphatestref()来设置这个值，而不是这个值）。

语法。

gpu_get_alphatestref();

返回。

(from 0 - 255)

例子。

if gpu_get_alphatestenable()
{
    if (gpu_get_alphatestref() < 254)
    {
        gpu_set_alphatestref(254);
    }
}

上面的代码检查是否启用了阿尔法测试，如果启用了，它就会检查当前的阿尔法测试参考值，如果它小于254，就将其设置。
