gpu_set_cullmode

此函数可用于设置背面裁剪模式。一个多边形（基元）有一个正面和一个背面。正面据说是顶点按逆时针顺序定义的那一面。通常情况下，两边都会被画出来，但是如果你做的是一个封闭的形状（比如金字塔），那么这就是在浪费处理能力，因为三角形的背面永远不会被看到。正是在这种情况下，你可以在switch ，因为这可以节省大约一半的绘制时间，尽管这意味着你要以正确的方式定义你的多边形，以防止出现问题。

使用的剔除模式值必须是下列常数之一（默认值是 cull_noculling ）。

恒定 描述
cull_noculling 将不进行杀戮
cull_clockwise 所有顺时针方向的三角形都将被剔除
cull_counterclockwise 所有逆时针方向的三角形都将被剔除

语法。

gpu_set_cullmode(cullmode);

争论 类型 描述
cullmode 剔除模式不变 The cull mode to use (see list above)

返回。

N/A

例子。

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

上面的代码得到了当前的剔除模式，如果它不是 cull_clockwise ，就被设置为该常量。
