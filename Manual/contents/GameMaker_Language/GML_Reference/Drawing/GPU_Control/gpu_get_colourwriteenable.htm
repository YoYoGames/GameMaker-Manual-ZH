gpu_get_colourwriteenable

这个函数可以用来检索当前颜色的可写值。该函数返回一个4个元素的一维数组，其中的元素是 true （启用）或 false （禁用）。在默认情况下，所有的颜色写入都被设置为 true 。

  • [0] = 红色通道启用/禁用
  • [1] = 绿色通道启用/禁用
  • [2] = 启用/停用蓝色通道
  • [3] = 启用/停用Alpha通道

语法。

gpu_get_colourwriteenable();

返回。

(4 elements only; see above for details)

例子。

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上面的代码得到了当前的颜色写入值，然后将alpha分量设置为 false 。
