gpu_set_blendenable

这个函数可以用来切换阿尔法混合的开启和关闭。基本上，如果你把它设置为 false ，所有正在绘制的图像将被绘制成100%不透明的，这意味着sprite 或背景的任何透明或半透明区域将以纯色显示。 将以纯色显示。尽可能关闭阿尔法混合是个好主意（尤其是在为移动设备开发时），因为这将大大增加绘制速度。

语法。

gpu_set_blendenable(enable);

争论类型 描述
enable Enable or disable alpha blending value (true 或 ）。 false

返回。

例子。

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

上面的代码关闭了阿尔法混合来绘制背景sprite ，然后再打开它继续绘制。
