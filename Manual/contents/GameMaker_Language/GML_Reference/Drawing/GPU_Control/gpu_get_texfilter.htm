gpu_get_texfilter

通过这个函数，你可以检查texture 过滤（线性插值）是否启用（返回 true ）或不启用（返回 false ）。关于texture 滤波的更多信息，见函数 gpu_set_texfilter().

注意：在HTML5目标上，这个功能只有在启用WebGL的情况下才能发挥作用。

语法。

gpu_get_texfilter();

返回。

Boolean

例子。

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

上述代码检查texture 过滤是否打开或关闭，并进行相应的切换。
