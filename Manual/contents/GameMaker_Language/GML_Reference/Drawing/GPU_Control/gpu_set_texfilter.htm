gpu_set_texfilter

这个函数可以用来设置游戏屏幕上绘制的所有图像的texture 过滤（线性插值）。当这个功能被启用时 (true)，所有textures 在绘制时将被平滑化（这包括sprites ，因为它们也被认为是textures ），这意味着当缩放或移动时，如果没有1:1的像素比例，那么在不同的像素上将会出现 "污点"，这可能使图像看起来很模糊，这取决于使用的美术风格。如果这个功能被禁用（false ），那么在缩放或移动时，图像将根据最近的像素来绘制，这可能会导致 "块状 "图像。默认值是 false ，这也可以在全局游戏选项中针对各个目标平台进行更改。

注意:这个设置将覆盖任何texture 阶段插值的设置，shaders ，使用函数 gpu_set_texfilter_ext().

注意：在HTML5目标上，这个功能只有在启用WebGL的情况下才能发挥作用。

语法。

gpu_set_texfilter(enable);

争论类型 描述
enable Enable or disable texture filtering (true / ) false

返回。

例子。

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

上述代码检查texture 过滤是否打开或关闭，并进行相应的切换。
