gpu_get_blendmode_ext

该函数可用于检索当前用于绘图的扩展混合模式。该函数返回一个2个元素的一维数组，其中有以下元素。

  • [0] = 来源 混合模式系数常数(默认为 bm_src_alpha)
  • [1] = 目的地 混合模式系数常数(默认为 bm_inv_src_alpha)

数组的值将是以下常数之一（"s "表示从源头取值，而 "d "表示从目的地取值）。

混合模式系数常数
恒定 混合系数（红、绿、蓝、Alpha）。
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (如、如、如、如)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (广告、广告、广告、广告)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) 其中f = min(As, 1-Ad)

注意，你可以改变这些值，并将完整的数组作为一个参数传递给 gpu_set_blendmode_ext()函数作为一个参数。

语法。

gpu_get_blendmode_ext();

返回。

Array (2 elements only; see above for constants)

例子。

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

上面的代码得到了当前的扩展混合模式，修改了源，然后再次设置扩展混合模式。
