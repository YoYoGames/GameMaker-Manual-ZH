gpu_get_blendenable

这个函数可以用来检索alpha-blending的状态。如果它返回 true ，那么alpha-blending是启用的，如果它返回 false ，则是禁用的。默认情况下，它是打开的，因此该函数将返回 true

语法。

gpu_get_blendenable();

返回。

例子。

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

上面的代码检查阿尔法混合的状态，如果它被禁用，就会再次启用。
