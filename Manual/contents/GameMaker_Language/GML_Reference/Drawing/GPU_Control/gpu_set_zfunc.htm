gpu_set_zfunc

这个函数可以用来设置z-buffer测试比较模式（见 gpu_set_ztestenable()获取更多信息）。可供使用的值是以下任何一个常数（默认值是 cmpfunc_lessequal ）。

恒定 描述
cmpfunc_never 从不
cmpfunc_less 更少
cmpfunc_equal 平等
cmpfunc_lessequal 少于或等于
cmpfunc_greater 更多
cmpfunc_notequal 不平等
cmpfunc_greaterequal 大于或等于
cmpfunc_always 始终如一

语法。

gpu_set_zfunc(cmp_func);

争论 类型 描述
cmp_func 比较函数常数 The comparison mode to use (see list above)

返回。

N/A

例子。

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

上面的代码在绘制背景sprite ，然后再把它关掉，继续绘制，就开启了z-buffer测试，并设置了其比较模式。
