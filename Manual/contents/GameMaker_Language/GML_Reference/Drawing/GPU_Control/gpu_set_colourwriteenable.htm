gpu_set_colourwriteenable

有了这个功能，你可以switch ，为所有进一步的绘图打开或关闭颜色通道和alpha通道。例如，你可以用这个函数来创建只有阿尔法通道的表面（switch ，在使用表面之前关闭RGB书写），或者在向屏幕绘图时创建特殊效果。每个组件的默认值是 true ，可以作为四个唯一的参数提供，也可以作为一个4个元素的一维数组提供，其中的元素将是 true （启用）或 false （禁用）。

  • [0] = 红色通道启用/禁用
  • [1] = 绿色通道启用/禁用
  • [2] = 启用/停用蓝色通道
  • [3] = 启用/停用Alpha通道

语法。

gpu_set_colorwriteenable(red, green, blue, alpha);

争论 类型 描述
red 布尔型 Enable/disable the red channel
green 布尔型 Enable/disable the green channel
blue 布尔型 Enable/disable the blue channel
alpha 布尔型 Enable/disable the alpha channel

返回。

N/A

例子。

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上面的代码得到了当前的颜色写入值，然后将alpha分量设置为 false 。
