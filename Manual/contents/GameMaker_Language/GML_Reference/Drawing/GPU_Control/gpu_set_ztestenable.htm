gpu_set_ztestenable

这个函数可以用来切换z-buffer测试的开或关，影响事物的绘制方式（一般来说，只有在处理3D项目时才会用到）。基本上，默认情况下，当z-testing关闭时，你有两个objects 绘制在同一个空间。 两个 objects ，不管其中一个是否会过度绘制另一个，都会被渲染，导致不必要的绘制调用。switch 如果你把这个功能打开，那么Z-buffer会被测试，看一个object 是否是 "可见的"，如果不是，就不画。 不可见。请注意，这是默认行为，但你可以通过改变用于z-buffer测试的比较类型来改变它（见函数 gpu_set_zfunc().

默认情况下，z-buffer测试是关闭的(false)。

语法。

gpu_set_ztestenable(enable);

争论类型 描述
enable Enable or disable z-buffer testing (true 或 ）。 false

返回。

例子。

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

上面的代码开启了z-buffer测试，以绘制一个背景sprite ，然后再将其关闭，继续绘制。
