gpu_set_alphatestenable

这个功能将为你的游戏启用或禁用阿尔法测试（默认情况下这是禁用的）。开启阿尔法测试后，你可以使用配套的函数 gpu_set_alphatestref()来设置 "截止 "值，在这个值上所有的阿尔法值都将被设置为0。 下图显示了阿尔法测试开启或关闭时可以看到的差异。

注意：该功能可能对iOS 和安卓设备的性能产生负面影响。

语法。

gpu_set_alphatestenable(enable);

争论类型 描述
enable Enable or disable alpha testing (true / ) false

返回。

例子。

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

上面的代码将检查是否启用了alpha测试，如果没有，它将在alpha测试中switch ，并将测试阈值设置为128（只有alpha值超过0.5的像素会被绘制）。
