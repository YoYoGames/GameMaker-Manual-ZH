gpu_set_texrepeat

此函数可用于向GameMaker指示，当用于模型和基元时，textures 是否应该重复(true)或不重复(false)。通常情况下，texture 的宽度和高度位于 0 - 1 的范围内，但如果你指定一个大于 1 的值，默认情况下，其余部分不会被绘制（整个texture 将被绘制一个 "空 "的部分，对应于超过 1 的百分比。因此，texture 的宽度或高度的值为2，将在一半的空间内绘制texture ，而将另一半留空）。然而，通过使用这个函数将重复设置为真，texture 会在所有模型和基元上自动重复。这些函数可能的用例是在三维中重复一个texture ，但是为了使其发挥作用，并且不从texture 页面的其他部分提取图像，需要在sprite 编辑器中把所使用的Texture 标记为在一个 "独立的Sprite 页面 "上。

注意：Sprites 和背景的绘制总是不重复的，所以一旦你绘制了sprite 或背景，这个值就会被重置为 false 。这个默认行为不会影响texture 的阶段设置（即：调用函数会，但内部设置为false不会）。

注意：在使用功能时，此设置将覆盖任何texture 阶段的重复设置，shaders 。 gpu_set_texrepeat_ext().

语法。

gpu_set_texrepeat(enable);

争论类型 描述
enable Enable or disable texture filtering (true / ) false

返回。

例子。

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

上面的代码检查texture 重复是开还是关，并进行相应的切换。
