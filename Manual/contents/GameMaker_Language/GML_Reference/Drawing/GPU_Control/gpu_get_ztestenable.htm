gpu_get_ztestenable

该函数可用于检索是否启用了z-testing（该函数返回 true ）或未启用（该函数返回 false ）。默认值是z-testing被禁用，所以该函数将返回 false 。

语法。

gpu_get_ztestenable();

返回。

例子。

if gpu_get_ztestenable() == false
{
    gpu_set_ztestenable(true);
}

上面的代码检查Z测试是否被启用，如果被禁用，则再次启用。
