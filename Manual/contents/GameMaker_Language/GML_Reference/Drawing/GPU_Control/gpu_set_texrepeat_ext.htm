gpu_set_texrepeat_ext

在GameMaker中使用Shaders时，这个函数可以用来设置单个采样器 "槽 "是否重复指定的texture 。如果uv坐标不在0-1范围内，设置为 true ，将重复texture ，而设置为 false ，则意味着不重复。这些功能的可能使用情况是在3D中重复一个texture ，但是为了让它工作，并且不从texture 页面的其他部分拉取图像，所使用的sprite 需要在Texture 编辑器中被标记为在 "独立的Sprite 页面"。

注意 这个设置将被调用函数时设定的值所覆盖。 gpu_set_texrepeat().

语法。

gpu_set_texrepeat_ext(sampler_id, enable);

争论类型 描述
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / ) false

返回。

例子。

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

上面的代码检查特定采样器ID（存储在一个局部变量中）的texture 过滤是否开启或关闭，并相应地进行切换。
