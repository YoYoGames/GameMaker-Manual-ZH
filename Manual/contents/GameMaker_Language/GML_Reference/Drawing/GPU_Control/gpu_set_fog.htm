gpu_set_fog

这个功能可以用来启用或禁用绘制雾气。在3D游戏中，雾可以用来使远处的实例看起来褪色甚至不可见，这有助于创造气氛以及掩盖你没有绘制远处的实例的事实。

你可以设置它是启用（true ）还是禁用（false ），雾应该使用哪种颜色进行混合，以及开始和结束的绘制距离。距离值本质上是以像素为单位的深度值，并且是相对于摄像机的位置而言。

例如，0的位置是离摄像机最近的，每增加一次，它就会变得更远。默认情况下，一个视图的摄像机放置在-16000深度，所以如果你想让雾气从0深度开始，在1000深度结束，你要把开始距离设置为16000，结束距离为17000。

该函数可以接受四个单独的参数（上面给出的），也可以接受一个具有以下结构的单一数组（下面的示例代码显示了这种方法）。

  • [0] = 启用切换（一个布尔值，要么是 true ，要么是 false ），默认为 false
  • [1] = 颜色（实数），缺省 c_black
  • [2] = 启动距离（实数），默认为0
  • [3] = 末端距离（实数），默认为1

语法。

gpu_set_fog(enable, colour, start, end);

争论 类型 描述
enable 布尔型 Enable or disable fog
colour 颜色 The fog colour
start 真实的 The distance to start applying fog from (relative to the camera)
end 真实的 The distance to end the fog (relative to the camera)

返回。

N/A

例子。

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上面的代码得到了当前的雾气设置，然后将数组中的颜色元素设置为 c_red ，然后使用改变后的数组再次设置雾气。
