gpu_get_zfunc

这个函数可以用来检索z的比较模式。返回的值将是以下常数之一（默认值是 cmpfunc_lessequal ）。

比较函数常数
恒定 描述
cmpfunc_never 从不
cmpfunc_less 更少
cmpfunc_equal 平等
cmpfunc_lessequal 少于或等于
cmpfunc_greater 更多
cmpfunc_notequal 不平等
cmpfunc_greaterequal 大于或等于
cmpfunc_always 始终如一

语法。

gpu_get_zfunc();

返回。

Comparison Function Constant (see table above)

例子。

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

上面的代码检查z测试方法是否被设置为 cmpfunc_greater ，如果不是则被设置为该常数。
