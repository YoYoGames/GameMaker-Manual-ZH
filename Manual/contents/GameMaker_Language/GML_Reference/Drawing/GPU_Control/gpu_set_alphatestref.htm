gpu_set_alphatestref

这个函数将设置阿尔法测试的参考值（当它已经用 gpu_set_alphatestenable()).这是一个 "截止 "阈值，在这个阈值上，带有阿尔法的像素将不会被绘制。

基本上，如果你把这个值设置为0，并且你打开了线性插值（无论是在游戏选项中还是用 gpu_set_texfilter())，你仍然会在重叠的边缘出现视觉错误objects 。这是因为你可以有几乎完全透明的区域，阻挡在它们后面绘制的东西，但如果你把这个参考值设置为，例如254（即比最大值低一个），这保证只有100%不透明的像素会被绘制。你也可以把texture 插值关闭，这通常可以让你使用默认值，没有任何问题，根本不需要使用这个功能，然而这在很大程度上取决于源艺术作品。

注意：这是一个高级绘图功能，所以如果你不确定你是否需要它，或者不知道它的具体作用，你可能最好不要使用它，因为它可能会严重破坏你的游戏的绘图方式。

语法。

gpu_set_alphatestref(val);

争论类型 描述
val Sets the reference value for alpha testing from 0 to 255 (default value is 0)

返回。

例子。

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

上面的代码将检查是否启用了alpha测试，如果没有，它将在alpha测试中switch ，并将测试阈值设置为128（只有alpha值超过0.5的像素会被绘制）。
