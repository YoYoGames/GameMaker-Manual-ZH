gpu_set_state

这个函数将使用传入的DS Map来设置当前的GPU状态。所提供的地图可以通过以下函数创建 gpu_get_state().

语法。

gpu_set_state(ds_map);

争论类型 描述
ds_map The GPU state to set as a DS Map.

返回。

例子。

gpu_set_state(gpu_map);

上面的代码使用变量 "gpu_map "中提供的地图来设置GPU状态。
