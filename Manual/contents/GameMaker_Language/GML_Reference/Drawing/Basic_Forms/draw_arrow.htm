绘制_箭头

这个函数将从点（x1,y1）到点（x2,y2）画一个箭头。箭头的茎是沿着这些点画的，实际的箭头是在末端画的，其中箭头的大小由参数 "size "定义，并且 被计算为箭杆的一部分，因此端点总是与x2,y2定义的位置对齐。箭头的宽度是按照长度的比例自动计算的。

注意：如果你想用shader ，你应该知道，大多数shaders ，希望有以下输入。vertex,texture, 颜色。然而，当使用这个函数时，只有vertex 和颜色数据被传入，所以 shader 可能不会画出任何东西（或画出一些东西但不正确）。如果你需要以这种方式画出形状，那么shader 应该考虑到这一点进行定制。

语法。

draw_arrow(x1, y1, x2, y2, size);

争论类型 描述
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line (where the arrowhead ends).
y2 The y coordinate of the end of the line (where the arrowhead ends).
size The length of the arrow in pixels.

返回。

例子。

draw_arrow(x, y, mouse_x, mouse_y, 10);

上面的代码将从实例的位置到鼠标的位置画一个箭头，箭头的长度为10像素。
