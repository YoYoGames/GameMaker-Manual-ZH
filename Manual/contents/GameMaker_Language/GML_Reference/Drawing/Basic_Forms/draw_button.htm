绘制按钮

这个函数将使用当前选择的绘制颜色和阿尔法绘制一个非常简单的矩形 "按钮"，其中向上参数定义了斜边效果的外观，如下图所示。

注意：如果你想用shader ，你应该知道，大多数shaders ，希望有以下输入。vertex,texture, 颜色。然而，当使用这个函数时，只有vertex 和颜色数据被传入，因此shader 可能不会画出任何东西（或画出东西但不正确）。如果你需要以这种方式绘制形状，那么在定制shader 的时候就应该考虑到这一点。

语法。

draw_button(x1, y1, x2, y2, up);

争论 类型 描述
x1 真实的 The x coordinate of the left of the button
y1 真实的 The y coordinate of the top of the button
x2 真实的 The x coordinate of the right of the button
y2 真实的 The y coordinate of the bottom of the button
up 布尔型 Whether the button is up (true) 或向下 (false)

返回。

N/A

例子。

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

这将画出一个按钮，如果按住鼠标左键，这个按钮就会显示被按下。
