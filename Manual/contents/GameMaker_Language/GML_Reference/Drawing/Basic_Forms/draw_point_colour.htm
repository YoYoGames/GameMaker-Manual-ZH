draw_point_colour

通过这个函数，你可以在屏幕的任何地方用你定义的颜色画一个单一的像素。该颜色设置将覆盖用函数设置的基本颜色。 draw_set_colour().

注意：如果你想用shader ，你应该知道，大多数shaders ，希望有以下输入。vertex,texture, 颜色。然而，当使用这个函数时，只有vertex 和颜色数据被传入，所以 shader 可能不会画出任何东西（或画出一些东西但不正确）。如果你需要以这种方式画出形状，那么shader 应该考虑到这一点进行定制。

语法。

draw_point_colour(x, y, col1);

争论类型 描述
x The x coordinate of the point.
y The y coordinate of the point.
col1 The colour of the point.

返回。

例子。

draw_point_colour(50, 50, c_red);

这将在(50,50)处画一个红色像素。
