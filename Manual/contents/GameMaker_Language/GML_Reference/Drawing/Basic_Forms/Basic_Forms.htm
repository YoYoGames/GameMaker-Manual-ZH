基本形式

GameMaker有一整套可用的函数来绘制不同的形式和形状。这些函数只能在实例的绘制事件中使用（你可以在其他地方使用它们，但你不会看到任何东西！），并创建纯粹的图形效果，这意味着它们不会产生碰撞事件，因为那些只能通过分配给实例的sprite 或碰撞掩码来完成。所有这些功能都会受到当前绘制颜色、alpha和混合模式的影响（见颜色一节）。

重要的是!线条、点和表格总是以1:1的屏幕分辨率绘制，无论它们是在什么情况下绘制的。

重要的是!这些函数会破坏顶点批处理，并增加游戏中texture 的交换次数，所以在一个绘制帧中多次调用其中任何一个函数都可能导致性能问题。

以下是用于绘制基本形状的绘图函数。请注意，这些函数的调用速度很慢，只作为基本绘图工具提供。

  • draw_arrow
  • draw_circle
  • draw_circle_colour
  • draw_ellipse
  • draw_ellipse_colour
  • draw_line
  • draw_line_colour
  • draw_line_width
  • draw_line_width_colour
  • draw_point
  • draw_point_colour
  • draw_rectangle
  • draw_rectangle_colour
  • draw_roundrect
  • draw_roundrect_colour
  • draw_roundrect_ext
  • draw_roundrect_colour_ext
  • draw_triangle
  • draw_triangle_colour

GameMaker中还包括以下函数，用于设置绘图精度和绘制一些非常特殊的东西。

  • draw_set_circle_precision
  • draw_button
  • draw_healthbar
  • draw_path
