draw_line_colour

用这个功能你可以画一条1像素宽的线，颜色在第一点的颜色1和第二点的颜色2之间混合。颜色设置将覆盖用函数设置的基础颜色。 draw_set_colour(). 请注意，由于各种支持的平台之间的差异，被绘制的线可能需要不同的值（x，y上的+/-1）来绘制所需的尺寸。

注意：如果你想用shader ，你应该知道，大多数shaders ，希望有以下输入。vertex,texture, 颜色。然而，当使用这个函数时，只有vertex 和颜色数据被传入，所以 shader 可能不会画出任何东西（或画出一些东西但不正确）。如果你需要以这种方式画出形状，那么shader 应该考虑到这一点进行定制。

语法。

draw_line_colour(x1, y1, x2, y2, col1, col2);

争论类型 描述
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
col1 The colour of the start of the line.
col2 The colour of the end of the line.

返回。

例子。

draw_line_colour(50, 50, 300, 50, c_red, c_blue);

这将画出一条从(50,50)到(300,50)的水平线，有一个平滑的红色到蓝色的渐变。
