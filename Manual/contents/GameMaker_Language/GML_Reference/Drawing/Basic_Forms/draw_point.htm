绘制点

通过这个函数，你可以在屏幕的任何地方画一个单一的像素，使用当前设置的绘制颜色和alpha。

语法。

draw_point(x, y);

争论类型 描述
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

返回。

例子。

draw_set_colour(c_yellow);
draw_point(100,100);

这将在位置(100,100)画一个黄色像素。
