draw_set_circle_precision

为了优化画圆时的性能，GameMaker基本上是画一个多边形，有足够的边，使它看起来是圆形的。然而，根据你需要的圆圈的大小，你可能会发现改变 这个值可以帮助提高游戏的性能，或者使圆圈看起来更好。你可以输入这个函数的精度值是圆形多边形的边数，这个数字可以是4到64之间的任何数字。 但必须是一个能被4整除的数字，默认值为24。

注意：如果你想用shader ，你应该知道，大多数shaders ，希望有以下输入。vertex,texture, 颜色。然而，当使用这个函数时，只有vertex 和颜色数据被传入，所以 shader 可能不会画出任何东西（或画出一些东西但不正确）。如果你需要以这种方式画出形状，那么shader 应该考虑到这一点进行定制。

语法。

draw_set_circle_precision(precision);

争论类型 描述
precision The precision of all circles drawn after this function is called.

返回。

例子。

draw_set_circle_precision(12);

上面的代码将使用12边的多边形绘制所有的圆。
