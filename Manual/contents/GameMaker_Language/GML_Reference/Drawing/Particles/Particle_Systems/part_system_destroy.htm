销毁系统

通过这个函数，你可以销毁一个给定的粒子系统并将其从内存中移除。这个函数应该总是在不再需要这个系统的时候被调用，比如在一个room ，或者在一个实例的销毁事件中，否则你可能会导致粒子出现在以后的rooms ，而没有办法删除它们，同时也会导致内存泄漏，最终使你的游戏崩溃。

注意：这个函数也将销毁和删除任何可能已经创建并与被销毁系统相关的发射器。

语法。

part_system_destroy(ind);

争论 类型 描述
ind 粒子系统ID The index of the particle system to destroy.

返回。

N/A

例子。

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

上述代码检查全局变量的值是否小于或等于零，如果是，那么它就会破坏给定变量中引用的粒子系统，然后进入下一个room 。
