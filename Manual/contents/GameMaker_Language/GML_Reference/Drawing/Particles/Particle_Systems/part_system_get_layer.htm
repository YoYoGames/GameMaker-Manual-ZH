部分系统_获取图层

这个函数可以用来检索给定粒子系统的唯一层ID值。你提供粒子系统的ID值（如函数所返回的 part_system_create_layer())，该函数将返回该层的ID值。

语法。

part_system_get_layer(ind);

争论 类型 描述
ind 粒子系统ID The particle system ID value to get the layer ID of

返回。

Layer ID or 0 (if the layer is an internally managed one)

例子。

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

上面的代码将检查一个粒子系统，看它在哪个层上，如果它不在给定的层上，它将被改变。
