部分系统_更新

通过这个函数，你可以强制粒子系统在屏幕上绘制所有的当前粒子。如果 part_system_automatic_draw()关闭，那么这个函数将在实例的绘制事件中显示粒子，或者可以在绘制目标被设置为一个表面时使用（使用 surface_set_target())来将系统中的粒子绘制到该表面。

语法。

part_system_drawit(ind);

争论 类型 描述
ind 粒子系统ID The index of the particle system to draw the particles of.

返回。

N/A

例子。

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

上述代码检查变量 "surf "中索引的表面是否存在，如果存在，则将绘图目标设置为该表面，用存储在全局变量中的ID绘制粒子系统，然后重置绘图目标，使所有正常的绘图再次出现在屏幕上。
