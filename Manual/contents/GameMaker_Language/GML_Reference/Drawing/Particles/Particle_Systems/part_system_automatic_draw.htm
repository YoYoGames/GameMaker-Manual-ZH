part_system_automatic_draw

这个功能可以用来switch 关闭粒子系统的绘制，这样就不会看到对该系统的任何更新（自动或其他）。这是一个纯粹的视觉选项，当设置为 false ，你将无法看到粒子，因为它们没有被绘制出来，但它们仍然存在，并且正在改变位置、颜色等......就像它们正常的那样。当自动绘制关闭时，你可以明确命令GameMaker使用函数来绘制粒子系统的当前状态。 part_system_drawit()，如果你再把这个函数设置为 true ，你就可以switch ，把自动绘制功能重新打开。

有一点需要注意的是，如果你使用的是由函数创建的简单效果 effect_create_above()或 effect_create_below()那么你可以使用0（用于下面的效果）或1（用于上面的效果）的值作为粒子系统的索引，因此也可以切换这些粒子的自动绘制（这也适用于切换GML Visual粒子效果的绘制）。

语法。

part_system_automatic_draw(ind, automatic);

争论 类型 描述
ind 粒子系统ID The index of the particle system to change.
automatic 布尔型 Whether automatic drawing is on (true) or not (false).

返回。

N/A

例子。

part_system_automatic_draw(global.Sname, false);

上述代码关闭了全局变量 "Sname "中索引的粒子系统的自动绘图。
