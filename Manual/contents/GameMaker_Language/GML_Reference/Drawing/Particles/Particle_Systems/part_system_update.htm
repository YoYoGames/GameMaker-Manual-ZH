部分系统_更新

这个函数将在游戏时间内将给定的粒子系统推进一步，可以用来模拟自动更新关闭时的粒子系统行为，也可以用来将一个粒子系统一步推进到一个特定的点。例如，如果你有一个雪的粒子系统，你想让雪从玩家进入room ，那么你可能希望在一个实例的创建事件中把雪的粒子提前几秒钟，这样在它们第一次被绘制时，屏幕上到处都是粒子（下面的代码例子显示了这个）。当系统的自动更新被函数关闭时，你也可以使用这个函数 part_system_automatic_update()来推动系统的发展。

语法。

part_system_update(ind);

争论 类型 描述
ind 粒子系统ID The index of the particle system to update.

返回。

N/A

例子。

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

上述代码将使全局变量 "RainSys "中索引的粒子系统提前3秒。
