部分_系统_层

这个函数可以用来将一个粒子系统从它的当前层转移到一个新的层中，switch 。你提供粒子系统的ID值（由函数返回 part_system_create_layer())，然后提供唯一的图层ID(由函数返回) layer_create()或作为string 的层的名称--例如 "instance_layer"--在room 编辑器中定义），然后系统将被转移到新的层。

语法。

part_system_layer(ps, layer);

争论 类型 描述
ps 粒子系统ID The particle system ID
layer 字符串 或层ID The new layer ID

返回。

N/A

例子。

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

上面的代码将检查一个粒子系统，看它在哪个层上，如果它不在给定的层上，它将被改变。
