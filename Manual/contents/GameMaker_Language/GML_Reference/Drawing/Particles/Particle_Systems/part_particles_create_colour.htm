粒子创建颜色

这个功能对于那些不需要粒子发射器所提供的任何功能的效果来说是非常理想的（例如，从导弹中产生烟雾，或者一个简单的爆炸效果），因为它允许你在游戏中的任何位置快速而容易地创建粒子room 。你也可以在用这个功能创建粒子时 "即时 "为其上色，这个颜色将超越粒子的预定义颜色，但它不会在粒子的生命周期内混合这个颜色。注意，你必须事先创建好粒子系统和粒子类型，才能使用这个函数。

语法。

part_particles_create_colour(ind, x, y, parttype, colour, number);

争论 类型 描述
ind 粒子系统ID The index of the particle system.
x 真实的 The x coordinate of where to create the particles.
y 真实的 The y coordinate of where to create the particles.
parttype 粒子类型ID The index (type) of the particles to be created.
colour 颜色 The colour blending for the particles.
number 真实的 The number of particles to create.

返回。

N/A

例子。

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

上述代码将在具有该代码的实例的速度大于1的每一步产生一个粒子。这些粒子将是一个随机的颜色。
