创建系统

这个函数用来创建一个新的粒子系统，并将返回一个唯一的索引号，这个索引号应该被储存起来，并用于所有与该系统有关的后续函数。该系统将被分配到一个托管层，并将被设置为深度为0。托管层不能被用户访问，当使用深度而不是层时，只用于内部管理。通常情况下，你会使用函数 part_system_create_layer()而不是这个函数。

语法。

part_system_create();

返回。

Particle System ID

例子。

global.Sname = part_system_create();

这将创建一个新的粒子系统，并在全局变量 "Sname "中存储索引。
