栏目组_创建

这个功能对于那些不需要粒子发射器提供的任何功能的效果来说是非常理想的（例如，从导弹中产生烟雾，或者一个简单的爆炸效果），因为它允许你在游戏中的任何位置快速和容易地创建粒子room 。注意，你必须事先创建好粒子系统和粒子类型，才能使用这个功能。

语法。

part_particles_create(ind, x, y, parttype, number);

争论 类型 描述
ind 粒子系统ID The index of the particle system.
x 真实的 The x coordinate of where to create the particles.
y 真实的 The y coordinate of where to create the particles.
parttype 粒子类型ID The index (type) of the particles to be created.
number 真实的 The number of particles to create.

返回。

N/A

例子。

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

上面的代码检查鼠标按钮是否被按下，如果返回 true ，则在鼠标位置生成5个粒子。
