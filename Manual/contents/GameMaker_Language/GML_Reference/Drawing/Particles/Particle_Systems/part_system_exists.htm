部分系统存在

通过这个函数，你可以检查给定的粒子系统的索引是否存在于游戏中。请注意，如果被检查的变量是一个未初始化的变量（否则粒子系统会将其索引分配给该变量），这将引发一个错误。

语法。

part_system_exists(ind);

争论 类型 描述
ind 粒子系统ID The index of the particle system to check for.

返回。

Boolean

例子。

if !part_system_exists(global.Sname)
{
    global.Sname = part_system_create();
}

上面的代码检查全局变量中引用的粒子系统是否存在，如果不存在就会被创建。
