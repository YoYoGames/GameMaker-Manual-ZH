部件_系统_位置

通过这个函数，你可以为粒子系统设置相对于room （0,0）位置的基础位置，这意味着所有与这个系统有关的进一步的粒子函数现在都将相对于新的位置绘制。默认情况下，这个位置总是(0,0)，但是在一些非常特殊的情况下，你可能希望将这个位置改为其他的位置。

注意：这个函数会改变粒子系统内的一切，所以如果你在位置(100,100)有一个发射器，然后将粒子系统的位置设置为(0,100)，那么这个发射器现在会在(100,200)绘制。同样，如果你移动系统，然后创建发射器，即使你在(100,100)处创建它，它也会在(100,200)处被绘制。

语法。

part_system_position(ind, x, y);

争论 类型 描述
ind 粒子系统ID The index of the particle system to change.
x 真实的 The new x coordinate of the particle system.
y 真实的 The new y coordinate of the particle system.

返回。

N/A

例子。

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

上述代码将检查鼠标按钮的按下情况，如果检测到，全局变量 "Sname "中索引的粒子系统将被转移到鼠标的X/Y位置。
