部件_系统_自动更新

这个函数控制GameMaker是否应该自动更新粒子系统，通常你不需要使用这个函数，因为默认值 true （自动更新开启）是你希望发生的。然而，对于特殊效果或暂停游戏，你可以将其设置为 false ，它将阻止在给定的粒子系统上进行任何更新，除非通过使用该函数明确指示 part_system_update()或者你再次使用这个函数来打开自动更新。不更新意味着在这个系统中创建的粒子将不再改变位置、颜色或任何其他参数，发射器也将停止工作，将系统 "冻结 "在自动更新被设置为 false 的确切位置。

有一点需要注意的是，如果你使用的是由函数创建的简单效果 effect_create_above()或 effect_create_below()那么你可以使用0（用于下面的效果）或1（用于上面的效果）的值作为粒子系统的索引，因此也可以切换这些粒子的自动更新（这也可以用于切换GML Visual粒子效果的更新）。

语法。

part_system_automatic_update(ind, automatic);

争论 类型 描述
ind 粒子系统ID The index of the particle system to change.
automatic 布尔型 Whether automatic updating is on (true）或不（假）。

返回。

N/A

例子。

if global.Pause
{
    part_system_automatic_update(global.Sname, false);
}

如果全局变量 "Paused "为真，上述代码将switch 关闭粒子的更新。
