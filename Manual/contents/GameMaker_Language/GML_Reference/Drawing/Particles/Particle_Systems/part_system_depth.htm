部分系统深度

通过这个功能，你可以设置粒子系统的绘制深度，就像你可以在room 内设置不同层的渲染深度一样，低的绘制深度意味着它将出现在所有绘制深度较高的事物之上，而高的绘制深度则将其置于所有绘制深度较低的事物之下。

语法。

part_system_depth( ind, depth );

争论 类型 描述
ind 粒子系统ID The index of the particle system to change.
depth 真实的 The depth at which to set the particle system.

返回。

N/A

例子。

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

上述代码将创建一个粒子系统，并将其索引存储在全局变量 "Sname "中。然后这个系统被赋予-1000的低深度，这意味着它将出现在所有具有更高绘制深度的事物之上。
