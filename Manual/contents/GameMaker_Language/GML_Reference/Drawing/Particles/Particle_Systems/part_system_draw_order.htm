栏目中的 "系统 "和 "绘图 "的顺序

通过这个函数，你可以设置在屏幕上创建粒子时的绘制方式。默认系统使用旧>新的外观（该函数被设置为 true ），其中旧粒子的绘制深度高于新粒子，因此出现在新粒子的 "下面"，但通过将该函数设置为 false ，你可以颠倒这个顺序，让新粒子的绘制深度更高，因此出现在旧粒子的 "下面"。下面的图片说明了这一点，左边的图片是默认值 true ，右边的图片是 false 。

注意：当用加法混合模式绘制粒子时，这个功能的效果可能不一定明显。

语法。

part_system_draw_order(ind, oldtonew);

争论 类型 描述
ind 粒子系统ID The index of the particle system to change.
oldtonew 布尔型 Whether old particles should be drawn behind newer ones (true) or vice versa (false).

返回。

N/A

例子。

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

这将创建一个新的粒子系统，其索引为"mysystem"，然后它设置粒子，将较新的粒子画在较旧的粒子之上。
