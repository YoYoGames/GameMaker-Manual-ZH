栏目组_发射器_创建

这个函数用于在给定的粒子系统中创建一个新的粒子发射器。该函数将返回新发射器的索引值，该索引值必须存储在一个变量中，并在所有进一步引用该发射器的函数中使用，当不再使用该发射器时，发射器本身必须被销毁，以防止内存泄漏（这可以通过特定的发射器销毁函数或销毁该发射器所属的整个粒子系统来实现）。

注意这个函数返回的发射器ID可以重复使用，例如，如果你首先创建了三个粒子发射器，它们的ID将被分配为0、1和2。如果你随后用这个函数销毁了第二个发射器（它的ID是1），然后创建了另一个发射器，那么这个发射器就会被分配到0、1和2。 part_emitter_destroy然后创建另一个发射器，那么这个发射器将重新使用1的ID。 由于发射器的ID 2正在使用，下一个发射器将得到3的ID，以此类推。

语法。

part_emitter_create(ps);

争论 类型 描述
ps 粒子系统ID The particle system to create the emitter in.

返回。

Particle Emitter ID

例子。

p_emit1 = part_emitter_create(Sname);

这将创建一个新的粒子发射器，并在变量 "p_emit "中存储其索引。
