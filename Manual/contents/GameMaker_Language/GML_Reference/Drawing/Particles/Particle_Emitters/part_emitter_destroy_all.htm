辐射器的全部内容

这个函数将从给定的系统中删除所有定义的发射器，并从内存中清除它们（这也将停止任何由给定的发射器产生的粒子，但它不会将它们从room ）。这个函数应该总是在系统不再需要发射器时被调用，以防止内存泄漏和错误。

语法。

part_emitter_destroy_all( ps );

争论类型 描述
ps The particle system to destroy all emitters from.

返回。

例子。

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

上述代码检查内置的全局变量 "lives"，如果它是0，就会销毁所有粒子发射器，然后改变room 。
