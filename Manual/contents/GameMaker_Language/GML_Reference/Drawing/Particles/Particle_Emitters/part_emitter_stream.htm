栏目组_发射器_流

这个函数允许你将发射器设置为特定类型的粒子流，通常用于创建事件，因为它是一个一次性的代码，可以将发射器设置为在此后的每一步游戏中产生指定数量的粒子。粒子是按照函数设定的分布、形状和位置发射出来的。 part_emitter_region().你可以将粒子流的数量设置为零，然后再次调用这个函数来 "关闭 "粒子流，这个函数也会接受负数的数量，在这种情况下，发射器将根据随机机会产生粒子。例如，如果你把粒子数设置为-5，那么每一步都有1:5的机会产生一个粒子。如果你需要粒子一次出现，而不是每一步都出现，你应该使用函数 part_emitter_burst().

语法。

part_emitter_stream(ps, ind, parttype, number);

争论 类型 描述
ps 粒子系统ID The particle system that the emitter is in.
ind 粒子发射器ID The index of the emitter to burst from.
parttype 粒子类型ID The index (type) of the particles to be created.
number 真实的 The number of particles to create per step.

返回。

N/A

例子。

part_emitter_stream(global. Sname, p_emit1, p1, 1);

上述代码在游戏的每一步都会流转1个粒子，直到发射器被摧毁或流转设置为0。
