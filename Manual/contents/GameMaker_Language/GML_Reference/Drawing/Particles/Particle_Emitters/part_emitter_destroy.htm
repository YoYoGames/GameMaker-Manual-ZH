part_emitter_destroy

这个函数将从给定的系统中移除指定的发射器，并从内存中清除它（这也将停止任何由给定的发射器产生的粒子，但它不会将它们从room ）。当系统不再需要指定的发射器时，应该总是调用这个函数，以防止内存泄漏和错误。

语法。

part_emitter_destroy( ps, ind );

争论 类型 描述
ps 粒子系统ID The particle system to destroy the emitter from.
ind 粒子发射器ID The index of the emitter to destroy.

返回。

N/A

例子。

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit1);
}

上述代码将检查变量 "p_emit "中索引的粒子发射器是否存在于给定的粒子系统中，如果存在，则将其销毁。
