部分发射器存在

通过这个函数，你可以检查给定的粒子发射器的索引是否存在于给定的系统中。请注意，如果被检查的变量是一个未初始化的变量（否则粒子发射器会被分配索引），这将会产生一个错误。

语法。

part_emitter_exists(ps, ind);

争论 类型 描述
ps 粒子系统ID The particle system to check for an emitter.
ind 粒子发射器ID The index of the emitter to search for.

返回。

Boolean

例子。

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

上述代码将检查变量 "permit "中索引的发射器，如果它存在，它将从它那里爆出30个粒子。
