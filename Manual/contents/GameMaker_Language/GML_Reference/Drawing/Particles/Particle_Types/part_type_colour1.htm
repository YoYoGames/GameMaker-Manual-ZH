部件_类型_颜色1

该函数用于将一个粒子类型设置为在每个单独的粒子寿命的总时间内为单一颜色。

语法。

part_type_colour1(ind, colour1);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
colour1 颜色 The single colour to make the particle type.

返回。

N/A

例子。

part_type_colour1(global.Snow_Part, c_white);

上述代码将把所有以全局变量 "Snow_Part "为索引的粒子类型创建的粒子设置为仅为白色。
