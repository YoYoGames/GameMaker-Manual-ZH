部分_类型_混合

通过这个函数，你可以让你的粒子用加法混合模式（true ）或不用（false ）绘制。加法混合是一种特殊的混合模式，它在每个粒子重叠时增加其亮度值，因此浅色会在重叠时逐渐变亮（直到呈现白色），而深色会变得越来越透明，黑色几乎看不见。这个功能可以带来一些漂亮的粒子效果，但不一定需要。例如，一个烟迹会关闭加法混合以保持灰色效果，但一个火焰效果可能会打开加法混合以使粒子更加透明和明亮。

语法。

part_type_blend(ind, additive);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
additive 布尔型 Whether the particles should be blended additively (true) or normally (false).

返回。

N/A

例子。

part_type_blend(part_Fire, true);

上述代码将把所有以 "part_Fire "变量为索引的粒子类型创建的粒子设置为具有加法混合模式。
