毁坏的部分_类型

通过这个函数，你可以从游戏中删除指定的粒子类型。当你使用这个函数时，所有给定类型的粒子将从room ，而粒子本身也会从内存中删除，所以这个函数应该只在你不再需要粒子的时候使用。

语法。

part_type_destroy(ind);

争论 类型 描述
ind 粒子类型ID The index of the particle type to destroy.

返回。

N/A

例子。

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

上述代码检查一个粒子系统，看当前是否有粒子在room ，如果没有，粒子、系统和实例就会被销毁。
