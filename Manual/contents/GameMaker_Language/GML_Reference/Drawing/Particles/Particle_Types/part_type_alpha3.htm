栏目组_类型_alpha3

这个函数可以用来为每个创建的给定类型的粒子设置三个alpha（透明度）值梯度。第一个阿尔法值是所有粒子开始时的阿尔法值，第二个阿尔法值是在其生命周期的一半时将被混合的阿尔法值，第三个阿尔法值是粒子将结束时的阿尔法值。在粒子的生命周期中，从一个字母到另一个字母会发生平滑的梯度变化。

语法。

part_type_alpha3(ind, alpha1, alpha2, alpha3);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
alpha1 真实的 The starting alpha of the particle.
alpha2 真实的 The halfway point alpha of the particle.
alpha3 真实的 The ending alpha of the particle.

返回。

N/A

例子。

part_type_alpha3( part_Health, 0.5, 1, 0);

上述代码将使所有在变量 "part_Health "中索引的类型创建的粒子在其生命周期中具有从0.5到1到0的α混合。
