栏目组_type_clear

通过这个函数，你可以 "重置 "一个粒子，将与该粒子有关的每个函数（生命、颜色、阿尔法、方向等）的所有值返回到它们的默认值。请注意，这个函数不会从屏幕上删除当前在room 中可见的任何粒子，为此，你应该使用 part_particles_clear().

语法。

part_type_clear(ind);

争论 类型 描述
ind 粒子类型ID The index of the particle type to clear.

返回。

N/A

例子。

part_type_clear(global.explode_part);

上述代码将把全局变量 "explode_part "中索引的粒子类型清除为其默认值。
