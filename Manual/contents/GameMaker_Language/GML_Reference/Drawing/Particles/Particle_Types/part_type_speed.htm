part_type_speed

这个函数用于设置粒子创建时的移动速度，也可以用于使粒子在其生命周期内增加或减少其速度。最小和最大的速度值默认为1，这意味着粒子每一步将移动1个像素，如果它们被设置为相同的值，所有粒子的初始速度将是相同的，但如果设置为不同的值，每个粒子将从给定范围内的一个随机速度开始。例如，如果你的最小值是3，最大值是6，那么所有的粒子将以每步3到6个像素之间的速度创建。

你还可以设置一个增量值，这个增量值将使粒子在其生命周期内移动得更快（如果是正数）或更慢（如果是负数），每一步都会在粒子速度上增加给定的数量。这个值可以是最小的（+/-）0.01。

最后，你可以设置 "摇摆 "因子。这是一个值，它将在粒子寿命的每一步中随机加入或减去速度。显然，大的数值比小的更明显，这个数值甚至可以是一个负数，最大范围在-20和20之间。

语法。

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
speed_min 真实的 The minimum speed the particle can start at.
speed_max 真实的 The maximum speed the particle can start at.
speed_incr 真实的 How much the particle speed should increase or decrease per step.
speed_wiggle 真实的 How much should randomly be added or subtracted from the particle's speed per step.

返回。

N/A

例子。

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

上述代码将设置各种粒子值，包括设置最小速度为5，最大速度为10的速度，每一步的速度会以-0.2的数值减慢。速度也会在每一步中以0到20之间的随机值进行摆动。
