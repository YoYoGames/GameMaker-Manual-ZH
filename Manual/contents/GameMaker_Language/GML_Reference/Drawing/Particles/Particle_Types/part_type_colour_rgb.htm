part_type_colour_rgb

通过这个函数，你可以为所有创建的给定类型的粒子设置红色、绿色和蓝色的混合颜色。你为这三种成分中的每一种提供一个最小值和一个最大值，创建的粒子将根据给定的参数范围拥有一个随机的颜色。所有数值必须在0到255之间。

语法。

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
rmin 真实的 The minimum the final colour's red component can be.
rmax 真实的 The maximum the final colour's red component can be.
gmin 真实的 The minimum the final colour's green component can be.
gmax 真实的 The maximum the final colour's green component can be.
bmin 真实的 The minimum the final colour's blue component can be.
bmax 真实的 The maximum the final colour's blue component can be.

返回。

N/A

例子。

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

上述代码将全局变量 "Blood_Part "中索引的粒子类型所发射的每个粒子设置为只有不同的红色。
