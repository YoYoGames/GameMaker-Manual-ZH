部分_类型_颜色_混合

通过这个函数，你可以将给定的粒子类型设定为两种颜色的随机混合。

语法。

part_type_colour_mix(ind, colour1, colour2);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
colour1 颜色 The first colour the blend will take from.
colour2 颜色 The second colour the blend will take from.

返回。

N/A

例子。

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

上述代码将把全局变量 "P_Damage "中索引的粒子类型的每个粒子发射的颜色设置为红色和黄色之间的随机混合色。
