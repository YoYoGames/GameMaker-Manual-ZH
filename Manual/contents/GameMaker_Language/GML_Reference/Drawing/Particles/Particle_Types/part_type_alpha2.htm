栏目组_类型_alpha2

这个函数可以用来为每个创建的给定类型的粒子设置一个alpha值（透明度）梯度。第一个阿尔法值是所有粒子开始时的阿尔法值，第二个阿尔法值是粒子结束时的阿尔法值，在粒子的一生中，阿尔法值将从一个到另一个发生平滑的梯度变化。这可以从0（透明）到1（不透明）。

语法。

part_type_alpha2(ind, alpha1, alpha2);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
alpha1 真实的 The start alpha of the particle.
alpha2 真实的 The end alpha of the particle.

返回。

N/A

例子。

part_type_alpha2(global.Snow_Part, 0, 1);

上述代码将设置所有创建的粒子类型的粒子在全局变量 "Snow_Part "中的索引，使其具有0的alpha值（透明），然后在其生命周期结束时淡化为1的alpha（不透明）。
