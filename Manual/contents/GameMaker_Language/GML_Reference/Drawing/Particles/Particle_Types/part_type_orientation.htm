零件类型方向

这个函数用来确定粒子的方向sprite ，当它被创建时，也可以用来使粒子的方向在其生命周期中增加或减少。最小和最大的方向值默认为0，但是这些值可以按照GameMaker的标准方向来改变方向，0度为右，90度为上，180度为左，270度为下。如果你将它们设置为相同的值，那么粒子将以相同的方向被创建。

你也可以设置一个增量值，它将在方向的有效期内增加（如果是正数）或减少（如果是负数）一定的度数。这个值可以是最小的（+/-）0.01。

你也可以设置 "摆动 "因子，和其他粒子函数一样。这是一个值，它将在粒子寿命的每一步中随机地增加或减少方向。很明显，大的数值比小的更明显，这个数值甚至可以是一个负数，最大范围在-20和20之间。

最后，你可以选择是否有相对的方向，这意味着当粒子有一个方向（和速度）时，粒子sprite ，将围绕这个矢量定向。请记住，如果你将粒子的速度设置为降低，并且达到了0，缺乏速度会将方向设置为默认值0°，因此相对方向会导致粒子sprite "跳 "到一个不同的角度。

语法。

part_type_orientation( ind, ang_min, ang_max, ang_incr, ang_wiggle, ang_relative );

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
ang_min 真实的 The minimum starting angle of the particles.
ang_max 真实的 The maximum starting angle of the particles.
ang_incr 真实的 The increase (or decrease if set negative) of the angle in degrees per step.
ang_wiggle 真实的 The amount the angle wiggles clockwise/counterclockwise per step.
ang_relative 真实的 Whether to set its angle relative to the direction of the particle's motion (true) or not (false).

返回。

N/A

例子。

part_type_shape(particle2, pt_shape_spark);
part_type_size(particle2, 0.10, 0.50, 0.01, 0);
part_type_scale(particle2, 0.30, 0.30);
part_type_colour1(particle2, 8454143);
part_type_alpha1(particle2, 0.50);
part_type_speed(particle2, 4, 4, -0.07, 1);
part_type_direction(particle2, 0, 359, 0, 20);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 5);

上述代码将设置各种粒子值，包括方向，这将是0º和359º之间的随机值。每一步也会有一个0到20之间的随机量加入其中，方向是相对于运动方向的。
