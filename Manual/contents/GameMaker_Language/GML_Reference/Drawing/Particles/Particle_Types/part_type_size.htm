part_type_size

这个函数用来确定粒子在创建时的大小，也可以用来使粒子在其生命周期中增加或减少大小。最小和最大的尺寸值默认为1（意味着在设置了比例之后，尺寸与基础粒子的比例相同）。 part_type_scale()如果它们被设置为相同的值，那么所有粒子的初始尺寸将是相同的，但是如果设置为不同的值，每个粒子将从范围内的随机尺寸开始，例如，如果你的最小值是0.5，最大值是2，那么所有的粒子将以基本比例的一半到两倍的尺寸被创建。

你还可以设置一个增量值，使粒子sprite ，在粒子的生命周期内增长（如果是正数）或缩小（如果是负数），每一步都将给定的数量加入到粒子的尺寸中。这个值可以是最小的（+/-）0.01。

最后，你可以设置 "摇摆 "因子。这是一个值，它将在粒子寿命的每一步中随机地增加或减少其大小。显然，大的数值比小的更明显，这个数值甚至可以是一个负数，最大范围在-20和20之间。

语法。

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
size_min 真实的 The minimum size the particle can start at.
size_max 真实的 The maximum size the particle can start at.
size_incr 真实的 How much the particle should increase or decrease per step.
size_wiggle 真实的 How much should randomly be added or subtracted from the particles size per step.

返回。

N/A

例子。

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

上面的代码将设置各种粒子的值，包括在每个粒子被创建时被设置为两倍的大小，然后每一步将大小增加0.01，也有0.1的摇摆。
