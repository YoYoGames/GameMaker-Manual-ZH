部分_类型_方向

这个函数用于确定粒子创建时的方向，也可以用来使粒子在其生命周期内增加或减少其方向。最小和最大的方向值默认为0，这意味着粒子将向右移动，没有变化，但这些值可以改变，以限制运动方向，遵循GameMaker的标准方向，即0是右，90是上，180是左，270是下。如果你把它们设置成相同的值，那么粒子将只向一个方向移动。

你还可以设置一个增量值，它将在方向的有效期内增加（如果是正数）或减少（如果是负数）一定的度数。这个值可以是最小的（+/-）0.01。

最后，你可以设置 "摇摆 "因子。这是一个值，它将在粒子寿命的每一步中随机加入或减去方向。显然，大的数值比小的更明显，这个数值甚至可以是一个负数，最大范围在-20和20之间。

语法。

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
dir_min 真实的 The minimum direction the particle can start at.
dir_max 真实的 The maximum direction the particle can start at.
dir_incr 真实的 How much the particle direction should increase or decrease per step.
dir_wiggle 真实的 How much should randomly be added or subtracted from the particle's direction per step.

返回。

N/A

例子。

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

上述代码将设置各种粒子值，包括方向，它将是0到359度之间的随机值。它也会每一步上升1度，并且每一步都会有一个0到20之间的随机量加入。
