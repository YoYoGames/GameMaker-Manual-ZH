栏目组_type_colour_hsv

通过这个函数，你可以为所有给定类型的粒子设置一个色调、饱和度和数值范围。你为这三种成分中的每一种提供一个最小值和一个最大值，创建的粒子将根据给定的参数范围具有随机颜色。通过这种方式，你可以创建具有相同色调但不同饱和度的粒子，或者具有不同色调但相同数值（亮度）的粒子等等。所有的值必须在0到255之间。

语法。

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
hmin 真实的 The minimum the final colour's hue component can be.
hmax 真实的 The maximum the final colour's hue component can be.
smin 真实的 The minimum the final colour's saturation component can be.
smax 真实的 The maximum the final colour's saturation component can be.
vmin 真实的 The minimum the final colour's value component can be.
vmax 真实的 The maximum the final colour's value component can be.

返回。

N/A

例子。

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

上述代码将全局变量 "Stars "中索引的粒子类型所发射的每个粒子设置为具有不同的颜色和饱和度，但具有相同的值（发光度）。
