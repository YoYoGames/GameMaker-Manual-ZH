部分_类型_存在

通过这个函数，你可以检查给定的粒子类型的索引是否存在于游戏中。请注意，如果被检查的变量是一个未初始化的变量（否则粒子类型将被分配其索引），这将引发一个错误。

语法。

part_type_exists(ind);

争论 类型 描述
ind 粒子类型ID The index of the particle type to check for.

返回。

Boolean

例子。

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

上面的代码检查全局变量 "p1 "是否存储了一个粒子类型的索引，如果没有，就创建一个。
