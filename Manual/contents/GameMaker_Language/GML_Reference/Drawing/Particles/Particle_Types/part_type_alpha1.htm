栏目组_类型_alpha1

这个函数用来设置一个粒子类型，使其在每个单独的粒子寿命的总时间内有一个单一的α值（透明度），这个值可以从0（透明）到1（不透明）。

语法。

part_type_alpha1(ind, alpha1);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
alpha1 真实的 The alpha of the particle.

返回。

N/A

例子。

part_type_alpha1(global.Snow_Part, 0.5);

上述代码将设置所有创建的粒子类型的粒子在全局变量 "Snow_Part "中的索引，使其具有0.5的alpha值（半透明的）。
