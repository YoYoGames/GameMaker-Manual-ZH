形状_类型_形状

通过这个函数，你可以设置sprite 粒子类型的形状，以下常数可以接受。

粒子形状常数
恒定 描述
pt_shape_pixel 一个1x1的像素。(这是默认设置）。
pt_shape_disk 一个被填满的圆。
pt_shape_square 一个被填满的正方形。
pt_shape_line 一条8px宽的水平线。
pt_shape_star 一颗五点填充的星。
pt_shape_circle 一个3PX轮廓的圆。
pt_shape_ring 一个向内发光的圆圈（看起来像一个气泡）。
pt_shape_sphere 一个向外发光的圆圈 ' 中间是实心的，向外发光。
pt_shape_flare 一个刺眼的发光点（在夜里看起来像一颗真正的星星）。
pt_shape_spark 火花效应'就像一个有多个点的星星逐渐消失。
pt_shape_explosion 一个方形的烟云'需要多种颜色，以类似于爆炸。
pt_shape_cloud 稀薄的云，需要向上扩展和多个粒子，以类似于云。
pt_shape_smoke 爆炸效果的平滑版本。使用多个来创造一个烟雾云。
pt_shape_snow 一个通用的雪花形状。

语法。

part_type_shape(ind, shape);

争论 类型 描述
ind 粒子类型ID The index of the particle type to set.
shape 粒子形状常数 The shape to choose.

返回。

N/A

例子。

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

上述代码将全局变量 "p1 "中索引的粒子类型设置为正方形的形状，然后设置其他各种粒子属性。
