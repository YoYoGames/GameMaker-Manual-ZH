部件_类型_颜色2

这个函数可以用来为每个创建的给定类型的粒子设置一个双色渐变。第一种颜色是所有粒子开始时的颜色，第二种颜色是粒子结束时的颜色，在粒子的生命周期中，颜色会从一种颜色到另一种颜色发生平滑的渐变。

语法。

part_type_colour2(ind, colour1, colour2);

争论 类型 描述
ind 粒子类型ID The index of the particle type to change.
colour1 颜色 The colour the particle will start at.
colour2 颜色 The colour the particle will end at.

返回。

N/A

例子。

part_type_colour2( part_Health, c_red, c_white);

上述代码将使所有以 "part_Health "变量索引的类型创建的粒子在其生命周期内有一个从红色到白色的混合颜色。
