gpu_set_tex_mip_enable

通过这个功能，你可以改变mipmapping是否被关闭，是否对所有东西都打开，或者只对Texture Group Manager中选择的texture 组打开。该功能需要下面列出的常数之一。

恒定 描述
mip_off Mipmapping被禁用。
mip_on 所有textures 的Mipmapping已经启用。
mip_markedonly 对于在textures 组选项中启用了Mipmapping的Texture （默认）。

语法。

gpu_set_tex_mip_enable(setting);

争论 类型 描述
setting Mipmapping常数 The mipmap setting (a constant, default: mip_markedonly)

返回。

N/A

例子。

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

上面的代码将检查是否启用了mipmapping，如果没有，它将启用它。

返回。Mipmapping
