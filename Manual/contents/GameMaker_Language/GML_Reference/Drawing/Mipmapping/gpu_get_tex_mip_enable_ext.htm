gpu_get_tex_mip_enable_ext

通过这个函数，你可以得到mipmapping是否关闭，是否对所有东西都打开，或者只对texture 取样器上的Texture Group Manager中选择的shader 组打开。你提供shader 取样器的索引值（如函数所返回的）。 shader_get_sampler_index()的索引值，该函数将返回下面列出的常数之一。

语法。

gpu_get_tex_mip_enable_ext(sampler_index);

争论 类型 描述
sampler_index 着色器采样器手柄 The index of the shader sampler to get

返回。

Constant

恒定 描述
mip_off Mipmapping被禁用。
mip_on 所有textures 的Mipmapping已经启用。
mip_markedonly 对于在textures 组选项中启用了Mipmapping的Texture 。

例子。

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上面的代码为给定的shader texture 采样器启用了mipmapping，如果它还没有被启用。

返回。Mipmapping
