gpu_set_tex_min_mip

通过这个函数，你可以设置当前使用的最小mipmap级别，其中0的值是最高分辨率，1是使用第一个mipmap，2是第二个等等。

语法。

gpu_set_tex_min_mip(minmip);

争论类型 描述
minmip The minimum mipmap level to use

返回。

例子。

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

上面的代码将检查当前的最小mipmap级别，如果不是0，就设置为0。

返回。Mipmapping
