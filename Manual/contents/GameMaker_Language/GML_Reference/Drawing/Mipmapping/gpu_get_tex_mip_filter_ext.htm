gpu_get_tex_mip_filter_ext

通过这个函数，你可以得到一个给定的shader 采样器的当前mip过滤模式。你提供shader 采样器的索引值（如函数所返回的 shader_get_sampler_index()，该函数将返回下面列出的模式值常数之一。

语法。

gpu_get_tex_mip_filter_ext(sampler_index);

争论类型 描述
sampler_index The index of the shader sampler to get

返回。

Mipmapping Filter Constant (listed below):

恒定 描述
tf_point 这意味着mipmap级别之间的混合被禁用，这可能会导致可见的texture ，但能提供最佳性能。
tf_linear 这意味着mipmap级别之间的混合被启用（这也被称为三线过滤），这使texture ，但它会对性能产生轻微的影响。
tf_anisotropic 这意味着各向异性过滤被启用，这大大改善了texture ，并能减少其他过滤模式下可见的模糊现象，但它对性能的打击最大。

例子。

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_filter_ext(_sampleIndex) != tf_point
{
    gpu_set_tex_mip_filter_ext(_sampleIndex, tf_point);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上面的代码将给定的shader texture 采样器的mip过滤模式设置为 tf_point （禁用mipmapping），如果它还没有被设置。

返回。Mipmapping
