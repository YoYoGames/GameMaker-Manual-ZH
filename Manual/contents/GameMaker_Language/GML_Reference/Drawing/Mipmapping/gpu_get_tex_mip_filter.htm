gpu_get_tex_mip_filter

通过这个函数你可以得到当前的mip过滤器模式。该函数返回的模式值将是下面列出的常数之一。

语法。

gpu_get_tex_mip_filter();

返回。

Mipmapping Filter Constant (listed below):

Mipmapping过滤器常数
恒定 描述
tf_point 这意味着mipmap级别之间的混合被禁用，这可能会导致可见的texture ，但能提供最佳性能。
tf_linear 这意味着mipmap级别之间的混合被启用（这也被称为三线过滤），这使texture ，但它会对性能产生轻微的影响。
tf_anisotropic 这意味着各向异性过滤被启用，这大大改善了texture ，并能减少其他过滤模式下可见的模糊现象，但它对性能的打击最大。

例子。

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

上面的代码检查键盘，如果指定的键被按下，它就会得到当前的mip过滤器，并将其值切换到下一个，在不同的模式中循环往复。

返回。Mipmapping
