gpu_set_tex_mip_filter

通过这个函数，你可以将当前的mip滤波模式设置为支持的三种类型之一。你给出的常数指的是你所需要的mip滤波模式，你可以在点滤波（默认设置，意味着没有滤波）、线性滤波（这也被称为三线性滤波）或各向异性滤波之间进行选择（见下面的常数表）。

恒定 描述
tf_point 这将禁用mipmap级别之间的过滤，这可能导致可见的texture ，但能提供最佳性能。
tf_linear 这将启用mipmap级别之间的线性滤波（这也被称为三线滤波），这将平滑texture ，但它会对性能产生轻微的影响。
tf_anisotropic 这将在mipmap级别之间启用各向异性过滤，这将大大改善texture ，并可以减少其他过滤模式下的模糊，但它对性能的影响最大。

语法。

gpu_set_tex_mip_filter(filter);

争论 类型 描述
filter Mipmapping过滤器常数 The mip filter mode to use (a constant, default: tf_point).

返回。

N/A

例子。

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

上面的代码检查键盘，如果指定的键被按下，它就会得到当前的mip过滤器，并将其值切换到下一个，在不同的模式中循环往复。

返回。Mipmapping
