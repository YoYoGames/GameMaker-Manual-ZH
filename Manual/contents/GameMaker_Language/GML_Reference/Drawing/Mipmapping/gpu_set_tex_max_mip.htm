gpu_set_tex_max_mip

通过这个函数，你可以设置当前设置的要使用的最大mipmap级别，其中0的值是最高分辨率，1是使用第一个mipmap，2是第二个等等。请注意，这对于避免在渲染textures 时出现出血现象非常有用，例如，将texture 页面边界设置为8px，然后将最大mipmap级别设置为3，将确保你在较大的渲染距离上不会出现任何出血问题。

语法。

gpu_set_tex_max_mip(maxmip);

争论类型 描述
maxmip The maximum mipmap level (default: 16)

返回。

例子。

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

上面的代码将检查当前的最大mipmap级别，如果不是4，就设置为4。

返回。Mipmapping
