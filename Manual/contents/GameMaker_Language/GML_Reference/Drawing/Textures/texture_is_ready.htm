纹理_is_ready

这个函数可以用来检查一个特定的texture 页面是否已经被解压并可以使用，或者一组texture 页面是否已经被解压并可以使用。你提供唯一的纹理页ID（由 texturegroup_* 函数返回）或texture 组IDstring （在Texture 组编辑器中定义），如果它们已经被解包，该函数将返回 true ，否则返回 false 。

语法。

texture_is_ready(tex_id);

争论 类型 描述
tex_id 纹理指针或字符串 The texture page pointer or a texture group name (a string)

返回。

Boolean

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

上述代码将检索texture "MainMenu "组的texture 页面ID，然后检查它们是否被解压，如果没有，它们就被预取到VRAM中。
