通过这个功能，你可以从视频内存中删除所有的textures ，然后它们将在第一次使用时被重新加载。texture 当你的游戏有多个texture 页面时，这是管理视频内存的唯一有效方法。 你应该在游戏的各关之间冲洗内存，并使用texture 组功能组织图形，以确保使用最少的textures 。

值得注意的是，在所有目标上，默认情况下，texture 页面只在需要时才会被加载，你可以使用sprites 的各种预取和刷新函数（在这里找到） 来管理事情，也可以使用这个函数。

语法。

draw_texture_flush();

返回。

例子。

draw_texture_flush();

上面的代码刷新了视频内存中的texture 数据，可能会被放在object 的第一个实例的创建事件中，放在room 中。
