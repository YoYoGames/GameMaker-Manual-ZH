纹理_get_height

返回具有给定id的texture 的高度，它总是一个在0 - 1范围内的值。然后，这可以在将textures 映射到模型或基元时使用。

语法。

texture_get_height(tex);

争论 类型 描述
tex 纹理指针 The texture page asset pointer to use

返回。

Real

例子。

tex_h = texture_get_height(surface_get_texture(global.Surf));

上面的代码将获得从先前创建的表面中提取的texture 的高度。
