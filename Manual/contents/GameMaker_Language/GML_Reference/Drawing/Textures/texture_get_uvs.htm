纹理_get_uvs

这个函数返回一个一维数组，其中4个元素代表给定的texture 指针的UV坐标，用以下数值填充数组。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部
  • [4] =asset 编译器从sprites 左侧修剪的像素数量( 仅限sprite assets )
  • [5] =asset 编译器从sprites 顶面修剪的像素数量( 仅限sprite assets )
  • [6] = 已保存到sprites 页面的原始texture 宽度的像素数据的归一化百分比( 仅限sprite assets )。
  • [7] = 已保存到sprites 页面的原始texture 高度的像素数据的归一化百分比 ( 仅限sprite assets )

然后，这个值可以在其他绘图函数中使用，特别是在一般的三维和一些二维原始函数，以及Shader 。如果你需要UVS用于sprite ，那么你可以使用 sprite_get_uvs() ，而对于font, font_get_uvs().

语法。

texture_get_uvs(texid)

争论 类型 描述
texid 纹理 The texture pointer to get the UVS for

返回。

Array (4 - 8 elements)

例子。

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

上面的代码首先检索存储在surf_back中的表面的texture ，然后得到该texture 的UV坐标。然后它从返回的数组中检索出左、上、右和下的UV坐标，并将它们存储在本地变量中。
