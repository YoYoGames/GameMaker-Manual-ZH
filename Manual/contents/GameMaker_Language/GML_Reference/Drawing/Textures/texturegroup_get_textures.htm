纹理组_get_textures

这个函数可以用来检索构成一个texture 组的各个页面的纹理页ID。你提供texture 组IDstring （如在Texture 组编辑器中定义的），该函数将返回一个一维数组，数组中的每个条目是一个单独的texture 页面ID。如果该函数失败--即：给出了一个无效的组，或者该组没有分配给它的texture ，那么该数组将是空的（0长度）。

语法。

texturegroup_get_textures(tex_id);

争论 类型 描述
tex_id 字符串 The name of the texture group to check (a string)

返回。

Array of Texture Pointers

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

上面的代码将检索texture 组 "MainMenu "的texture 页面ID，然后检查它们是否被解压，如果是，它们将被放入VRAM。

返回。纹理
