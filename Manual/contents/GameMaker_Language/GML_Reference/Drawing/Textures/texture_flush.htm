texture_flush

这个函数可以用来 "刷新 "一个texture 页面或一组texture 页面，即：在不再需要时从VRAM中删除它们。你可以提供唯一的纹理页ID（由 texturegroup_* 函数返回）来刷新单个页面，或者你可以提供一个纹理组名称（如在纹理组编辑器中定义的）来刷新该组中所有的texture 页面。

语法。

texture_flush(tex_id);

争论 类型 描述
tex_id 纹理指针或字符串 The texture page pointer or a texture group name (a string)

返回。

N/A

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_flush(_tex_array[i]);
}

上述代码将刷新texture 组 "MainMenu "下的所有texture 页面。

返回。纹理
