texture_get_texel_height

texel或纹理元素是计算机图形中使用的texture 空间的基本单位。Textures 由texel数组表示，就像图片由像素数组表示一样，这个函数返回所使用的图像texture 页面中单个texel的高度asset 。

语法。

texture_get_texel_height(tex);

争论 类型 描述
tex 纹理指针 The texture page asset pointer to use

返回。

Real

例子。

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

texture 上面的代码将从sprite asset ，得到texel的宽度和高度。

返回。纹理
