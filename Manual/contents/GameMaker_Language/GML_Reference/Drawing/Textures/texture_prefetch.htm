texture_prefetch

这个函数可以用来 "预取 "一个texture 页面或一组texture 页面，即：在需要时将它们加载到VRAM中。你提供唯一的纹理页ID（由 texturegroup_* 函数返回）来预取单个页面，或者你可以提供一个纹理组名称（如在纹理组编辑器中定义的）来预取该组中的所有texture 页面。

语法。

texture_prefetch(tex_id);

争论 类型 描述
tex_id 纹理指针或字符串 The texture page pointer or a texture group name (a string)

返回。

N/A

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_prefetch(_tex_array[i]);
}

上述代码将预取texture "MainMenu "组下的所有texture 页面。

返回。纹理
