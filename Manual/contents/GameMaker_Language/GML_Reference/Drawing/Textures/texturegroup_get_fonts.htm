纹理组_get_fonts

font 通过这个函数，你可以检索在给定的fonts 组内分配给texture 页面的每个texture 的索引。texture 你提供string 组IDtexture （如在font 组编辑器中定义的），该函数将返回一个1D数组，其中每个条目包含font 资源的索引。如果该函数失败--即：给出了一个无效的组，或者该组没有分配给它的texture ，那么该数组将是空的（0长度）。

语法。

texturegroup_get_fonts(tex_id);

争论 类型 描述
tex 字符串 The name of the texture group to check (a string)

返回。

Array

例子。

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
}

font 上面的代码将检索texture 组 "MainMenu "的索引，然后在控制台输出窗口显示这些ID。
