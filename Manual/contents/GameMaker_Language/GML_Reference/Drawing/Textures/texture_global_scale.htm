纹理_global_scale

这个函数允许你控制从WAD文件加载时texture 页面的比例，该文件是你最终游戏可执行文件的一部分。输入值必须是一个2的幂值，并且将在以下情况下工作。

  • 要将texture 页面设置为1:1，请使用 texture_global_scale(1)
  • 对一半textures ，一直使用 texture_global_scale(2)
  • 要将它们分成四份，请使用 texture_global_scale(4) ，等等。

通过这种方式，你可以在有低内存问题的平台上更好地控制texture 页面的内存使用。请注意，textures 页面是根据WAD的需求创建的，所以你可以在游戏开始时调用它来加载所有的textures 。 也可以在特定的时间调用它来加载特定的textures ，在这种情况下，你可能想在调用这个函数之前调用 draw_texture_flush()在这种情况下，你可能想在调用这个函数之前，使用预取函数来获取正确的 texture 页进入内存。

语法。

texture_global_scale(pow2integer);

争论类型 描述
pow2integer The scale factor to use (1, no scale, 2, half scale, 4, quarter scale, etc...)

返回。

例子。

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

上述代码将从内存中刷新所有的textures ，然后将texture 缩放比例设置为2（因此texture 页面为一半大小），最后预取texture "spr_Trees "中包含的sprite 页面。

返回。纹理
