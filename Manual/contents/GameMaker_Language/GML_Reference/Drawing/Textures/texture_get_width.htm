纹理_获取宽度

返回具有给定id的texture 的宽度，该id总是在0-1范围内的一个值。然后在将textures 映射到模型或基元时，可以使用这个值。

语法。

texture_get_width(tex);

争论 类型 描述
tex 纹理指针 The texture page asset pointer to use

返回。

Real

例子。

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

texture 上面的代码将获得从sprite asset 的宽度。
