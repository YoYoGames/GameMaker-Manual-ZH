纹理_调试_消息

这个函数可以用来启用或禁用texture 调试信息。当启用时（设置为 true ），关于texture 页面使用的额外信息将被发送到输出窗口。设置为 false ，可以再次禁用这种输出。

语法。

texture_debug_messages(enable);

争论类型 描述
enable Enable or disable the texture debug messages

返回。

例子。

if global.debug
{
    texture_debug_messages(true);
}

上述代码将检查一个全局变量的值，如果它被设置为 true ，那么texture 调试信息将被启用。

返回。纹理
