纹理_set_stage

这个函数将给定的阶段 "槽 "设置一个texture ，以便使用。可用的阶段 "槽 "的数量取决于你所编译的平台，对于Windows 、Mac和Linux ，最多有8个，但在较低端的Android 设备上（例如），这个数字可能低至2个。还值得注意的是，第一个阶段槽（1）总是由GameMaker自动使用。

注意：这个函数在运行中的shader 的上下文之外不会做任何事情!更多信息请参见着色器 。

语法。

texture_set_stage(stage, tex);

争论 类型 描述
stage 真实的 The texture "slot" to use.
tex 纹理指针 The texture to use.

返回。

N/A

例子。

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

上面的代码将在shader 中获取索引为 "GlassShader "的采样器的句柄，然后将该shader 常量设置为给定的sprite texture 。

返回。纹理
