纹理

下面的函数都与 "纹理 "有关，即：存储在VRAM中的图像，在你的游戏中被用作sprites 或其他东西。第一组可用的函数是用来检索特定图像的信息和它们在纹理页上的位置或大小，以及设置某些texture 功能。

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

GameMaker还有一些与管理与单个textures 以及整个texture 组相关的VRAM有关的功能。

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

最后，我们有以下函数，主要用于调试项目和确保有效使用texture 内存。

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
