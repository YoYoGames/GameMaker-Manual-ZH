texturegroup_get_tilesets

通过这个函数，你可以检索在给定的tilesets 组内分配给texture 页面的每个texture 的瓦片集索引。你提供texture 组IDstring （如在texture 组编辑器中定义），该函数将返回一个1D数组，其中每个条目包含一个瓷砖组资源的瓷砖组索引。如果该函数失败--即：给出了一个无效的组，或者该组没有分配给它的texture ，那么该数组将是空的（0长度）。

语法。

texturegroup_get_tilesets(tex_id);

争论 类型 描述
tex 字符串 The name of the texture group to check (a string)

返回。

Array

例子。

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
}

上述代码将检索texture "MainMenu "组的瓷砖组索引，然后在控制台输出窗口显示这些ID。
