绘制瓷砖图

这个函数可以用来在room 中的任何地方绘制一个给定的瓦片图。你给瓦片图元素ID（当你用 "创建 "瓦片图元素时得到的ID），或者当你使用函数时得到的ID。 layer_tilemap_create()或当你使用函数 layer_tilemap_get_id()），然后给出绘制的X/Y位置（在room ）。注意，这将只是在指定的点上绘制瓷砖图，使用调用该函数的实例的层深度。它不会移动瓦片图--也不会以任何方式改变它--也不管瓦片图是否被标记为可见。

语法。

draw_tilemap(tilemap_element_id, x, y);

争论 类型 描述
tilemap_element_id 瓦片地图元素ID The unique ID value of the tilemap element to draw
x 真实的 The x position within the room to draw at
y 真实的 The y position within the room to draw at

返回。

N/A

例子。

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

上面的代码得到了图层ID，然后用它来得到一个特定的瓷砖图ID，这又被用来在鼠标的位置绘制瓷砖图。
