draw_tile

通过这个函数，你可以从一个tile ，画出任何给定的tile set - 与旋转、翻转和镜像竞争 - 设置框架（如果是动画）和在room 。tile set ID值是在tile set 中设置的IDE 索引，tile 数据是数据 "blob"，包含所有关于tile 索引和已经应用的变换的信息。你可以使用函数 tilemap_get()来获得tile 数据，然后可以使用瓦片函数来操作，或者你可以使用以下tile 常量和要创建的tile 的索引值来自己构建它。

恒定 描述
tile_rotate 用于设置/获取一个tile 数据块的旋转位。
tile_mirror 用于设置/获取一个tile 数据块的镜像位。
tile_flip 用于设置/获取一个tile 数据包的翻转位。
tile_index_mask 一个特殊的常数，用于与tile 数据blob进行 "和"，以提取tile 索引。


你将这些常量与tile 索引值进行比特 "或"，以获得最终的平铺数据。

var tiledata = 8 | tile_rotate | tile_flip;

上面的代码简单地创建了一个局部变量来存储tile 上索引为8的tile set 的平铺数据，并对其进行了旋转和翻转。然后，这可以用来从你指定的tile ，以该索引绘制tile set 。这不与任何瓷砖图层相联系，tile ，将在进行绘制的实例的层深度，在你在room 中指定的x/y位置绘制。注意，你也可以设置绘制的帧。如果被索引的tile 是一个动画集的一部分，那么你可以在这里设置帧数（从0到帧数-1），被索引的tile 将仅仅是你正在使用的动画tile set 的指示器，而它将是被绘制的指定帧。如果tile 不是一个动画集的一部分，那么帧应该被设置为0。

语法。

draw_tile(tileset, tiledata, frame, x, y)

争论 类型 描述
tileset 瓷砖套装资产 The ID of the tile set to use
tiledata 瓦片数据 The tile data to use
frame 真实的 The frame number for the animated tile to use (default 0 for non-animated tiles)
x 真实的 The x position within the room to draw at
y 真实的 The y position within the room to draw at

返回。

Real

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

上面的代码将从给定的瓷砖图上获得tile ，然后设置 "翻转 "flag ，并使用这些数据绘制在实例位置翻转的tile 。

返回。精灵和瓷砖
