draw_set_swf_aa_level

这个函数可以用来设置SWF格式矢量sprites 的抗锯齿（AA）水平。这可以是一个从0到1的实值，将 "平滑 "这些sprites 的边缘。请注意，要看到这种效果，你必须先用函数启用AA draw_enable_swf_aa().

语法。

draw_set_swf_aa_level(AA);

争论类型 描述
AA The anti-aliasing value to use from 0 to 1.

返回。

例子。

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上述代码将检查SWF格式的AA值sprites ，如果它是0，它将启用AA并将值设置为0.5。

返回。精灵和瓷砖
