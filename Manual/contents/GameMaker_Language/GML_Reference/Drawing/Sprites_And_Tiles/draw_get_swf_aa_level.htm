draw_get_swf_aa_level

这个函数可以用来获取SWF格式矢量sprites 的抗锯齿（AA）水平。返回值将在0和1之间，显示这些sprites 的边缘将被绘制的 "平滑 "程度。你可以用这个函数来设置AA级别 draw_set_swf_aa_level().

语法。

draw_get_swf_aa_level();

返回。

例子。

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上述代码将检查SWF格式的AA值sprites ，如果它是0，它将启用AA并将值设置为0.5。

返回。精灵和瓷砖
