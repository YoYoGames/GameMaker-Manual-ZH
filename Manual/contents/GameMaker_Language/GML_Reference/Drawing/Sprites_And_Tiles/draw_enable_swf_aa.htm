draw_enable_swf_aa

通过这个功能，你可以为SWF格式的矢量sprites ，启用或禁用抗锯齿（AA）。AA只是将矢量图像的边缘平滑化，使其看起来更漂亮。使用的AA量将取决于使用该函数设置的值 draw_set_swf_aa_level().默认情况下这是禁用的。

语法。

draw_enable_swf_aa(enable);

争论类型 描述
enable Enable (true)或禁用(false)所有SWF的AA，sprites 。

返回。

例子。

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上述代码将检查SWF格式的AA值sprites ，如果它是0，它将启用AA并将值设置为0.5。

返回。精灵和瓷砖
