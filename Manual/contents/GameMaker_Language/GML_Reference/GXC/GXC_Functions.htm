Opera GX和GXC

本节概述了为Opera GX目标平台提供的功能。本页列出的功能允许您记录屏幕，并将其与您的挑战分数一起上传到GXC，但请注意，您需要使用GXC API库来实现这一目的。请参阅YoYo Games Helpdesk上的Opera GX部分，了解有关实施GXC API库和使用挑战的说明。

提供以下功能用于上传有挑战的视频记录。

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
