gxc_stop_movie_recording

这个函数用来停止先前用gxc_start_movie_recording() 开始的GXC录音。它也将录音提交给参数中指定的URL，但行为是不同的，取决于指定的string 。

  • 如果你指定了一个文件名，录音将以该名称保存在你的下载文件夹中。
  • 如果你指定了一个URL，录音将被上传到该URL。

当需要将录音上传到GXC时，应使用URL方法，文件应保存在本地，只用于检查录音是否成功。

语法。

gxc_stop_movie_recording(filename);

争论 类型 描述
filename 字符串 The file name or URL where the recording is to be saved or uploaded

返回。

N/A

例子。

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

上述代码使用GXC API库向当前活动的挑战赛上传分数，当在其回调方法中收到响应时，它检索视频的上传URL并向其上传录音。
