gxc_start_movie_recording

此函数用于在GXC上开始录制您的挑战。您可以指定录制的宽度、高度和帧率，以及是否录制音频的布尔值。

语法。

gxc_start_movie_recording(width, height, fps, record_audio);

争论 类型 描述
width 真实的 The width of the recording
height 真实的 The height of the recording
fps 真实的 The frame rate (frames-per-second) of the recording
record_audio 布尔型 Whether audio should be recorded (true) 或不(false)

返回。

N/A

例子。

gxc_start_movie_recording(320, 240, 4, false);

这将开始以320x240的分辨率和每秒4帧的速度进行视频录制，但不记录任何音频。
