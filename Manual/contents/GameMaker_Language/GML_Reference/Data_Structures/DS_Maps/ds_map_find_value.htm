ds_map_find_value

通过这个函数，你可以从一个指定的键中获取值。该函数的输入值是要使用的（先前创建的）DS地图和要检查的键。

注意：如果没有这样的键存在，那么该函数将返回 undefined 。你应该总是使用 is_undefined()函数

语法。

ds_map_find_value(id, key)

争论类型 描述
id The id of the map to use.
key The key to find.

返回。

Variable or undefined

例子。

amount = ds_map_find_value(inventory, "food");

或者，使用地图访问器"？"。

amount = inventory[? "food"];

上面的代码将获得键 "食物 "的值，并将其存储在变量 "金额 "中。

返回。DS地图
