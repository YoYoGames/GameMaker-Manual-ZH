ds_map_replace_map

通过这个函数，你可以用之前创建的另一个地图替换已经存储在给定 "键 "中的 DS Map 。这个函数是为创建JSON兼容的地图而设计的，然后你可以使用 json_encode()编码的JSON兼容地图，并且只能与该功能一起使用。

语法。

ds_map_replace_map(id, key, value)

争论 类型 描述
id DS地图ID The id of the ds_map to use.
key 字符串 The key to replace.
value DS地图ID The id of the ds_map to use to replace the one previously stored in the given key.

返回。

N/A

例子。

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

上面的代码将创建一个 DS Map ，并在替换以前存储在 DS Map "j_map "中的地图之前，用一个数组和一个 DS List ，来填充它。

返回。DS地图
