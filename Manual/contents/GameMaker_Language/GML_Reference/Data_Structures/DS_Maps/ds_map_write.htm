ds_map_write

这个函数将把指定地图的DS地图数据变成string 格式，然后可以写到*.ini或*.txt文件中，以方便存储。这个string ，然后可以在以后读回一个新的ds_map，使用 ds_map_read().

注意：返回的string ，不是人类可读的string ，而是数据结构内容的转储。

语法。

ds_map_write(id);

争论类型 描述
id The id of the data structure to use

返回。

例子。

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

上面的代码打开了一个准备写入的ini文件。然后，它使用 ds_map_write() ，生成一个string ，存储在临时变量 "t_string "中。最后，在关闭ini文件之前，它将该string 写入该文件。

返回。DS地图
