ds_map_set

通过这个函数，你可以在一个给定的DS地图中设置一个键的值。你提供DS地图的ID值（由函数返回 ds_map_create())，然后给出你想设置的键和要设置的值。键可以是整数或strings ，如果给定的键不存在，那么将为你创建并设置为该值。这个函数与使用DS map访问器来设置/创建一个地图键/值对是一样的。该函数不返回任何东西，所以如果你需要检查键值是否被替换或者新的键被创建，那么你应该使用函数 ds_map_replace() 。

语法。

ds_map_set(id, key, value)

争论类型 描述
id The id of the map to use.
key The key to set.
value The value to set the key to.

返回。

例子。

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

上面的代码将检查给定的键是否存在，如果不存在，那么它将被创建和设置。

返回。DS地图
