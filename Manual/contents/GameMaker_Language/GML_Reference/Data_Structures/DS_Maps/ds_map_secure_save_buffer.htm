ds_map_secure_save_buffer

这个函数将先前创建的DS地图保存到一个buffer 。你提供DS地图的ID值（由函数返回 ds_map_create())和要写入的buffer 的ID(由函数返回) buffer_create()).注意，如果被保存的DS地图包含一个数组，那么在保存时将被转换为一个DS列表。

重要的是!安全保存的DS地图文件只能在创建它的设备上加载，如果你试图加载保存在不同设备上的文件，那么它将无法工作。

注意：HTML5和UWP上不支持此功能。

语法。

ds_map_secure_save_buffer(filename);

争论 类型 描述
id DS地图ID The DS map ID value.
buffer 缓冲区ID The buffer to save to.

返回。

N/A

例子。

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

上面的代码将创建一个buffer 和一个DS地图，然后用一些值填充地图，并在删除地图之前将其写入buffer 。

返回。DS地图
