ds_map_delete

通过这个函数，你可以从给定的、先前创建的DS地图中删除任何给定的键（和它的相应值）。

语法。

ds_map_delete(id, key);

争论 类型 描述
id DS地图ID The id of the map to change.
key 字符串 The key to delete (along with its associated value).

返回。

N/A

例子。

ds_map_delete(inventory, "shield");

上述代码将从DS地图（库存）中删除键 "盾牌"（以及与之配对的值）。

返回。DS地图
