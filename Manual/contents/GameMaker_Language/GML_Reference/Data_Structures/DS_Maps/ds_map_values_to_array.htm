ds_map_values_to_array

通过这个函数，你可以检索一个DS地图中每个键所包含的所有值。你提供DS地图的ID来获取键值（如由 ds_map_create())，该函数将返回一个数组，数组中的每个条目都是DS地图中的一个值（例如，如果地图有10个键，那么将返回一个10个条目的数组，每个条目对应一个键的值）。该函数有一个可选的第二个参数，你可以提供一个你已经创建的数组，在这种情况下，地图的键值数据将被附加到数组中的任何现有数据上。请注意，该函数将直接修改所提供的数组，但也将返回它的一个引用（如果没有提供，则返回一个新数组的引用）。

语法。

ds_map_values_to_array(id, [array])

争论 类型 描述
id DS地图ID The id of the map to use.
[array] 阵列 OPTIONAL 用于追加DS地图值的数组。

返回。

Array

例子。

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

上面的内容是从DS地图中为每个键检索所有的值，并将它们作为一个数组存储在一个本地变量中。然后，loops ，将每个值添加到一个本地变量中，然后用来显示所有值的总数。

返回。DS地图
