ds_map_secure_load

这个函数将从给定的文件中加载一个先前保存的DS地图。这个文件必须是以前用 ds_map_secure_save()函数创建的，并且在加载时，该函数将返回从加载的数据中创建的DS地图的索引。这个DS地图的索引应该存储在一个变量中，并用于对这个地图的所有进一步的函数调用。请注意，如果正在加载的DS地图是以数组作为键值之一保存的，那么在加载时这个数组将被转换成DS列表。

重要的是!安全保存的文件的一个特点是，它被锁定在创建它的设备上，所以你不能把保存在一个设备上的文件加载到另一个设备上运行的项目中。

语法。

ds_map_secure_load(filename);

争论类型 描述
filename The name of the file to load the map data from.

返回。

DS Map ID

例子。

p_map = ds_map_secure_load("p_data.dat");

上面的代码将加载一个安全保存的DS地图，并将其索引值存储在一个变量中供将来使用。

返回。DS地图
