ds_map_create

这个函数用于创建一个新的、空的DS地图，并将返回其id，然后在所有其他DS地图函数中用于访问数据结构。

重要的是!当你创建一个数据结构时，识别它的索引值是一个从0开始的整数。这意味着不同类型的数据结构可以有相同的索引值，所以如果有疑问，你应该在访问它们之前使用 ds_exists()函数来访问它们。还要注意的是，索引是可以重复使用的，所以一个被破坏的数据结构索引值可能会被之后新创建的数据结构使用。

语法。

ds_map_create();

返回。

DS Map ID

例子。

inventory = ds_map_create();

上面的代码将创建一个新的、空的DS地图，并将其id索引存储在变量 "库存 "中。

返回。DS地图
