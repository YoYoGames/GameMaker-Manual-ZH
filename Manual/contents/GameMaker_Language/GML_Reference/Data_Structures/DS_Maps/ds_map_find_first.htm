ds_map_find_first

这个函数返回存储在DS地图中的第一个键。这并不是按照你添加的顺序的第一个键!DS地图不是以线性形式存储的，为此要使用DS列表，所以这个函数所做的只是找到计算机存储的第一个键。如果你必须在DS地图中反复寻找某个东西，这可能很有用，但如果可能的话应该避免，因为它可能很慢。

注意，如果给定的DS地图是空的，这个函数将返回 undefined 。

语法。

ds_map_find_first(id);

争论类型 描述
id The id of the map to use.

返回。

Variable or undefined

例子。

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if key != "gold"
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

上面的代码创建了一些临时变量，然后得到DS地图的大小，并找到计算机存储在地图中的第一个键。然后，它使用 for loop ，遍历DS地图，寻找键值 "金"。如果它找到了，它就把loop 。

返回。DS地图
