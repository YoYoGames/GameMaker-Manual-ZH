ds_map_add_list

通过这个函数，你可以将一个（先前创建的）DS列表分配给给定DS地图中的一个键。这个函数是为创建JSON兼容的地图而设计的，然后你会使用 json_encode()编码的JSON兼容地图，并且只能与该功能一起使用。如果一个DS地图以这种方式添加了一个列表，销毁父地图也将销毁所包含的列表并释放它们的内存，并且在父地图上调用 ds_map_clear()也会销毁和清理任何标记的列表。

语法。

ds_map_add_list(id, key, value)

争论 类型 描述
id DS地图ID The id of the map to use.
key 字符串 The key for the added list.
value DS列表ID The id of the list to add.

返回。

N/A

例子。

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

上面的代码将创建一个列表，并用全局变量的各种值来填充它。然后这个列表被 "嵌套 "在一个DS地图中，然后地图被编码成JSONstring ，然后地图被销毁，从内存中删除它和它包含的任何列表。

返回。DS地图
