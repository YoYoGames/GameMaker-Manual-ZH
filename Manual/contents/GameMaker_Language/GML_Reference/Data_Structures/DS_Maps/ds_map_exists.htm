ds_map_exists

如果指定的键存在于（先前创建的）DS地图中，这个函数将返回 true ，如果不存在，则返回 false 。

语法。

ds_map_exists(id, key);

争论 类型 描述
id DS地图ID The id of the data structure to check
key 字符串 The key to check for

返回。

Boolean

例子。

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

上述代码将检查变量 "库存 "中索引的DS地图中的 "药水 "键，如果它不存在，它将把它添加到地图中。

返回。DS地图
