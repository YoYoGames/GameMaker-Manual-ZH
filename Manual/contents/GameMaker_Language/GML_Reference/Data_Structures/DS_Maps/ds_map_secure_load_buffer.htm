ds_map_secure_load_buffer

这个函数将从一个buffer 中加载一个安全保存的DS地图。你必须事先将buffer 加载到内存中（使用 buffer_load())，然后将其传递给这个函数，将返回一个由buffer 的内容填充的DS地图。注意，buffer 必须是用函数 ds_map_secure_save_buffer() 创建的，这样才能正常工作，同时注意，如果被加载的DS地图包含一个数组，在加载时将被转换为一个DS列表。

重要的是!安全保存的文件的一个特点是，它被锁定在创建它的设备上，所以你不能把保存在一个设备上的文件加载到另一个设备上运行的项目中。

注意：HTML5和UWP上不支持此功能。

语法。

ds_map_secure_load_buffer(buffer);

争论 类型 描述
buffer 缓冲区ID The buffer ID of the buffer to load the map data from.

返回。

DS Map ID

例子。

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

上面的代码将从buffer ，加载一个安全保存的DS地图，并将其索引值存储在一个变量中供将来使用。

返回。DS地图
