ds_map_read

这个函数将接收一个先前由函数创建的string 。 ds_map_write()创建的地图，然后将其读入先前创建的DS地图。如果string 被读入的地图包含任何键/值对，在重新构建保存的地图之前，这些键/值对将首先被清除掉。

注意，如果指定的DSstring 是由GameMaker:Studio 1.2.xruntime （或更早），你应该指定可选的参数 "legacy"，将其设置为 true ，因为string 的格式在那之后发生了变化。

语法。

ds_map_read(id, str, [legacy]);

争论 类型 描述
id DS地图ID The id of the data structure to read the string into
str 字符串 The string to read
legacy 布尔型 OPTIONAL 可以是 ，也可以是 ，或者完全省略。 true false

返回。

N/A

例子。

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

上面的代码创建了一个新的DS地图，并将其id索引存储在变量 "invent "中。然后，它打开一个ini文件，从该文件中读取一个string 到临时变量 "t_string"。最后，它检查以确保string 是有效的（而不是默认的ini值""），如果是的话，它就把string 读取到新创建的DS地图中，然后再次关闭ini文件。

返回。DS地图
