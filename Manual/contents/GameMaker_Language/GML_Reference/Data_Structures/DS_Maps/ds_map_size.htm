ds_map_size

通过这个函数，你可以找到（先前创建的）DS地图包含多少个键/值对。

语法。

ds_map_size(id);

争论类型 描述
id The id of the data structure to check.

返回。

例子。

if (ds_map_size(inventory) > 49)
{
    full = true;
}

上述代码将检查DS地图的大小（即：键/值对的数量），如果它大于49，它将变量 "full "设置为 true 。

返回。DS地图
