ds_map_copy

你可以用这个功能把一张地图的内容复制到另一张你以前用 ds_map_create() 。如果被复制的DS地图不是空的。 那么这个函数会在复制前先清除它。原始的DS地图在这个过程中保持不变。

语法。

ds_map_copy(id, source);

争论类型 描述
id The id of the map you are copying to
source The id of the map you are copying from

返回。

例子。

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

上面的代码将创建一个新的地图，并把它分配给变量 "inventory_2"。然后，它将把DS地图的内容复制到变量 "inventory_1 "的索引中，并复制到这个新地图中。

返回。DS地图
