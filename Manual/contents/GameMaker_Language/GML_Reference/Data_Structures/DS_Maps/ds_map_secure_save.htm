ds_map_secure_save

这个函数将把给定的DS地图的内容保存到一个与它所创建的设备相联系的文件中（意味着如果转移到任何其他设备上，它就不能被读取）。文件本身可以有几乎任何extension （例如， *.dat, *.json, *.bin, 等...），并将被混淆并存储到目标平台的本地存储。然后你可以使用函数重新加载ds_map ds_map_secure_load().注意，如果 被保存的DS地图包含一个数组，这个数组在保存时将被转换为DS列表。

重要的是!安全保存的文件的一个特点是，它被锁定在创建它的设备上，所以你不能把保存在一个设备上的文件加载到另一个设备上运行的项目中。

语法。

ds_map_secure_save(map, filename);

争论类型 描述
map The id of the data structure to use
filename The name of the file to save the map to

返回。

例子。

ds_map_secure_save(purchase_map, "p_data.dat");

上述代码将把变量 "p_data "中索引的DS地图保存到给定的文件中，以便以后检索。

返回。DS地图
