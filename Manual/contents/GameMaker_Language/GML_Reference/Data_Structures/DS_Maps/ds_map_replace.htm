ds_map_replace

通过这个函数，你可以改变DS地图中给定键的值。你提供地图的索引（如函数所返回的 ds_map_create()) 然后提供要替换的键 -string 或一个整数 - 以及要替换的键值。如果给定的键不 存在，那么它将为你创建，如果它存在，那么当前值将被新的 值替换。如果键值存在并且被替换，该函数将返回 true ；如果键值不存在并且创建了一个新的键值，则返回 false 。

语法。

ds_map_replace( id, key, val );

争论类型 描述
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

返回。

例子。

ds_map_replace(inventory, "torso", 55);

上面的代码在DS地图中查找键 "torso"，当它找到它时（如果不存在，则创建它），当前值被替换成指定的值。

返回。DS地图
