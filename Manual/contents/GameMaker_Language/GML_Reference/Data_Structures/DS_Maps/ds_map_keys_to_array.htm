ds_map_keys_to_array

通过这个函数，你可以检索一个DS地图包含的所有键。你提供DS地图的ID来获取键（如由 ds_map_create())，该函数将返回一个数组，数组中的每个条目都是DS地图中的一个键。该函数有一个可选的第二个参数，你可以提供一个你已经创建的数组，在这种情况下，地图的键数据将被附加到数组中的任何现有数据上。请注意，该函数将直接修改所提供的数组，但也将返回它的一个引用（如果没有提供，则返回一个新数组的引用）。

语法。

ds_map_keys_to_array(id, [array])

争论 类型 描述
id DS地图ID The id of the map to use.
[array] 阵列 OPTIONAL 要追加DS地图键的数组。

返回。

Array

例子。

map_keys = ds_map_keys_to_array(inventory);

上面的代码检索了一个DS地图的键，然后将它们存储为一个数组，以供将来使用。

返回。DS地图
