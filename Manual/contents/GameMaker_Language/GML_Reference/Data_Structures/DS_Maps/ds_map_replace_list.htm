ds_map_replace_list

通过这个函数，你可以用之前创建的另一个列表替换已经存储在给定 "key "中的DS列表。这个函数是为创建JSON兼容的地图而设计的，然后你可以使用 json_encode()编码，因此只能与该功能一起使用。

语法。

ds_map_replace_list(id, key, value)

争论 类型 描述
id DS地图ID The id of the map to use.
key 字符串 The key to replace.
value DS列表ID The id of the ds_list to use to replace the one previously stored in the given key.

返回。

N/A

例子。

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

上面的代码将创建一个 DS List ，并在替换先前存储在 DS Map "j_map "中的列表之前，用各种全局变量的值来填充它。

返回。DS地图
