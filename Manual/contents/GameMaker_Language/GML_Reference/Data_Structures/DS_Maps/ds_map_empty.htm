ds_map_empty

如果指定的（先前创建的）DS地图包含任何键/值对，这个函数将简单地返回 false ，如果不包含，则返回 true 。

语法。

ds_map_empty(id);

争论类型 描述
id The id of the data structure to check.

返回。

例子。

if ds_map_empty(inventory)
{
    weight = 0;
}

上述代码检查变量 "库存 "中索引的DS地图是否有任何键/值对，如果没有，则将变量 "重量 "设置为0。

返回。DS地图
