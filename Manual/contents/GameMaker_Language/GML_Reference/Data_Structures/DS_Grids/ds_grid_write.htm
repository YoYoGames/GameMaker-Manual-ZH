ds_grid_write

这个函数可以用来将给定的ds_grid转换为string ，然后将其存储在一个外部文件中（例如）。你可以使用函数将该函数返回的string 读回ds_grid中。 ds_grid_read().

注意 返回的string ，不是人类可读的string ，而是数据结构内容的转储。

语法。

ds_grid_write(index);

争论类型 描述
index The index of the grid to write.

返回。

例子。

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

上面的代码将打开一个ini文件（如果它不存在，则创建它），然后将给定的ds_grid作为string ，写入该文件。
