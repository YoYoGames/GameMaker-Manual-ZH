ds_grid_read

这个函数可以用来将之前由函数创建的string ，转换为DS网格。 ds_grid_write()创建的网格，再转换为DS网格。该DS网格必须是以前创建的（见下面的例子）。

注意，如果指定的DSstring 是由GameMaker:Studio 1.2.xruntime （或更早），你应该指定可选的参数 "legacy"，将其设置为 true ，因为string 的格式在那之后发生了变化。

语法。

ds_grid_read(index, string [, legacy]);

争论 类型 描述
index DS网格ID The index of the grid to read.
string 字符串 The string to read into the DS grid.
legacy 布尔型 OPTIONAL 可以是 ，也可以是 ，或者完全省略。 true false

返回。

N/A

例子。

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

上述代码根据room （每个32x32的正方形像素代表一个网格单元）的大小创建了一个DS网格，然后从ini文件中读取先前保存的一组网格数据到新的DS网格中。
