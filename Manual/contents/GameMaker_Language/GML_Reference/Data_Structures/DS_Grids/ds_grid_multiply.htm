ds_grid_multiply

这个函数将把给定的网格单元的值乘以指定的数量。

注意这个函数只对实数起作用，而不是strings 。

语法。

ds_grid_multiply(index, x, y, val);

争论 类型 描述
index DS网格ID The index of the grid.
x 真实的 The x position of the cell in the grid.
y 真实的 The y position of the cell in the grid.
val 真实的 The value to multiply with the cell.

返回。

N/A

例子。

ds_grid_multiply(mygrid, 5, 5, 2)

上述代码将把存储在给定的DS网格单元的值乘以2。
