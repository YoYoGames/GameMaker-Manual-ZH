ds_grid_shuffle

这个函数可以用来随机化一个网格内所有单元格中所有数值的位置。

注意由于GameMaker每次都会生成相同的初始随机种子，使debugging ，因此每次运行游戏时，这个函数都会将网格单元洗到相同的位置。为了避免 这种行为，请在游戏开始时使用 randomise()在你的游戏开始时使用。这只是在测试和debugging ，因为最终可执行的 包将不会显示这种行为，每次游戏都是随机的。

语法。

ds_grid_shuffle(index);

争论类型 描述
index The index of the grid to shuffle.

返回。

例子。

ds_grid_shuffle(grid)

这将采取DS网格中的所有数值，在变量 "grid "中进行索引，并对它们进行洗牌以获得不同的顺序。
