ds_grid_multiply_disk

这个函数将在DS网格的一个给定的圆盘状区域内获取所有的数值，并将每个数值乘以给定的数量。

注意这个函数只对实数起作用，而不是strings 。

语法。

ds_grid_multiply_disk(index, xm, ym, r, val);

争论 类型 描述
index DS网格ID The index of the grid.
xm 真实的 The x position of the disk on the grid.
ym 真实的 The y position of the disk on the grid.
r 真实的 The radius of the disk on the grid.
val 真实的 The value to multiply the cells within the disk with.

返回。

N/A

例子。

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

上面的代码将把在圆形网格区域内发现的所有数值，逐一乘以2。
