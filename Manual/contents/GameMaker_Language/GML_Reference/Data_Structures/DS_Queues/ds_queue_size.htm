ds_queue_size

这个函数将返回queue 的 "大小"，即：已经排到它的项目数量。

语法。

ds_queue_size(id);

争论类型 描述
id The id of the data structure to check.

返回。

例子。

if !ds_queue_empty(control_queue)
{
    num = ds_queue_size(control_queue);
}

上面的代码检查一个DS队列，看它是否为空，如果不是，它就得到它所包含的项目数，并将该值存入一个变量。
