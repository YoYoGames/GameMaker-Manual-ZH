DS队列

DSqueue 有点类似于DS堆栈，但它是以先入先出（FIFO）的方式工作。首先放入queue 的值也是第一个被移除的。你可以把它想象成商店里用于付款的queue ，先进入queue 的人先付款，然后离开商店。Queues 通常用于存储仍然需要由实例完成的动作或行为（例如，以scripts 的形式），但也有许多其他用途。

注意：与所有的动态资源一样，数据结构会占用内存，所以当不再需要时，应该总是销毁，以防止内存泄漏，这将减缓并最终使你的游戏崩溃。

注意：如果你需要检查一个数据结构是否存在，你可以使用 ds_exists()函数。

  • ds_queue_read
  • ds_queue_write
