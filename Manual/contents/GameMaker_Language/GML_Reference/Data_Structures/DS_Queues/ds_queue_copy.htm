ds_queue_copy

这个函数可以用来将一个queue 的内容复制到另一个。请注意，这不会删除原始queue 中的内容，也不会破坏原始queue 。当使用这个功能时，被复制的queue 必须是以前创建的，如果它在复制之前包含任何项目，那么这些项目将首先被清除（意味着这些信息将被丢失）。

语法。

ds_queue_copy(id, source);

争论 类型 描述
id DSQueue ID The id of the new queue.
source DSQueue ID The original queue to copy from.

返回。

N/A

例子。

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

上述函数创建了一个新的实例，然后在该实例中创建了一个新的DS队列，并将运行该代码块的实例中的queue ，复制到新创建的实例中queue 。
