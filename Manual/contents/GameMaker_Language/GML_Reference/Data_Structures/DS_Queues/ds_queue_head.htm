ds_queue_head

这个函数将只读取 queue 的第一个值（即 "头部 "的值）。它不会取消该值，这意味着它仍然可以在将来被这个函数或 ds_queue_dequeue().如果queue 是空的，那么该函数将返回常数 undefined ，否则它将返回string 中包含的实数或queue 值。

语法。

ds_queue_head(id);

争论类型 描述
id The id of the data structure to read from.

返回。

(Data type value stored in the queue)

例子。

num = ds_queue_head(control_queue);

上述代码将从变量 "control_queue "中索引的queue 读取数值，并将返回值存储在变量 "num "中。
