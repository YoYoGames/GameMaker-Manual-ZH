ds_queue_tail

这个函数将只读取 queue 的最后一个值（即 "在尾部 "的值）。它不会取消该值，这意味着它仍然可以在将来被这个函数或 ds_queue_dequeue().

语法。

ds_queue_tail(id);

争论类型 描述
id The id of the data structure to read from.

返回。

(Data type value stored in the queue)

例子。

num = ds_queue_tail(control_queue);

上述代码将从变量 "control_queue "中索引的queue 读取数值，并将返回值存储在变量 "num "中。
