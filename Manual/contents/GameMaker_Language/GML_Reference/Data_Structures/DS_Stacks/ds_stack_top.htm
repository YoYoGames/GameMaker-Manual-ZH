ds_stack_top

这个函数将只读取堆栈的第一个值（即 "在上面的"）。它不会弹出该值，这意味着它仍然可以在将来被这个函数或 ds_stack_pop().如果堆栈是空的，那么该函数将返回常数 undefined ，否则它将返回堆栈中包含的实数或string 值。

语法。

ds_stack_top(id);

争论类型 描述
id The id of the data structure to read from.

返回。

Variable (Data type value that is stored in the stack) or undefined

例子。

num = ds_stack_top(control_stack);

上述代码将从变量 "control_stack "索引的堆栈中读取数值，并将返回值存储在变量 "num "中。
