ds_stack_write

该函数返回一个string ，然后可以用该函数存储或转移到另一个数据结构中。 ds_stack_read()函数来存储或转移到另一个数据结构中。

注意：返回的string ，不是人类可读的string ，而是数据结构内容的转储。

语法。

ds_stack_write(id);

争论类型 描述
id The id of the data structure to write.

返回。

例子。

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

上面的代码打开了一个ini文件，然后写了一个string ，其中包含存储在DS堆栈中的信息，这些信息的索引在变量 "堆栈 "中。然后清空堆栈，关闭ini文件。
