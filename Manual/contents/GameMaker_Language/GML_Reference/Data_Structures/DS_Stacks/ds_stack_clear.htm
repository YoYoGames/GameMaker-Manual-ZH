ds_stack_clear

通过这个函数你可以清除给定堆栈数据结构中的所有数据。这并不破坏数据结构（为此你应该使用 ds_stack_destroy())，它只是清除其中的所有数据，并返回一个空栈。

语法。

ds_stack_clear(id);

争论类型 描述
id The id of the data structure to clear.

返回。

例子。

if (ai_count = 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

上面的代码检查一个变量，看它是否达到了一个特定的值，如果达到了，它就清除变量 "AI_stack "中索引的DS栈，设置一个警报，并将变量重置为0。
