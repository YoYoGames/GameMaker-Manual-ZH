ds_stack_copy

这个函数可以用来将一个堆栈的内容复制到另一个堆栈。请注意，这不会从原栈中删除内容，也不会破坏原栈。当使用这个函数时，被复制的堆栈必须是先前创建的，如果它在复制之前包含任何项目，那么这些项目将首先被清除（意味着这些信息将被丢失）。

语法。

ds_stack_copy(id, source);

争论 类型 描述
id DS堆栈ID The id of the new stack.
source DS堆栈ID The original stack to copy from.

返回。

N/A

例子。

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    stack = ds_stack_create();
    ds_stack_copy(stack, other.stack);
}

上述函数创建了一个新的实例，然后在该实例中创建了一个新的DS堆栈，并将运行代码块的实例中的堆栈内容，复制到新创建的实例堆栈中。
