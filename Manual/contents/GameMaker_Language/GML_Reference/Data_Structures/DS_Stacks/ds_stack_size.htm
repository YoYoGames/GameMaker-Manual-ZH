ds_stack_size

这个函数将返回堆栈的 "大小"，即：被推入堆栈的项目数量。

语法。

ds_stack_size(id);

争论类型 描述
id The id of the data structure to check.

返回。

例子。

if !ds_stack_empty(control_stack)
{
    num = ds_stack_size(control_stack);
}

上面的代码检查一个DS堆栈，看它是否为空，如果不是，就得到它所包含的项目数，并将该值存入一个变量。
