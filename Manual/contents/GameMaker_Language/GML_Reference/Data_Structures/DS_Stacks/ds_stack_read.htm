ds_stack_read

通过这个函数，你可以重新创建一个已保存的DS堆栈（一个以前被写成string ，使用 ds_stack_write()).你必须首先创建一个新的DS堆栈来读取string ，如果DS堆栈已经存在并且有信息存储在里面，那么在读取之前会被清除掉。在为你的游戏创建保存/加载机制时，这个功能是非常重要的。

注意，如果指定的DSstring 是由GameMaker:Studio 1.2.xruntime （或更早），你应该指定可选的参数 "legacy"，将其设置为 true ，因为string 的格式在那之后发生了变化。

语法。

ds_stack_read(id, str [, legacy]);

争论 类型 描述
id DS堆栈ID The id of the data structure to read into.
str 字符串 The string to read from.
legacy 布尔型 OPTIONAL 可以是 ，也可以是 ，或者完全省略。 true false

返回。

N/A

例子。

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
{
    ds_stack_read(stack, str);
}
ini_close();

上面的代码创建了一个堆栈，并将索引存储在变量 "stack "中。然后，它打开一个 ini 文件，并从中读取一个string ，首先检查以确保string 不被返回为空。这个string ，然后读到新创建的DS堆栈中。
