ds_stack_destroy

这个函数将从内存中删除给定的堆栈数据结构，释放它所使用的资源并删除它所包含的所有值。这个函数应该总是在你结束使用DS堆栈时使用，以防止内存泄漏，这可能会减慢和崩溃你的游戏。

重要的是!当你创建一个数据结构时，识别它的索引值是一个从0开始的整数。这意味着不同类型的数据结构可以有相同的索引值，所以如果有疑问，你应该在访问它们之前使用 ds_exists()函数来访问它们。还要注意的是，索引是可以重复使用的，所以一个被破坏的数据结构的索引值可能会被之后新创建的数据结构使用，所以我们建议在破坏之后一定要将持有DS索引的变量设置为-1。

语法。

ds_stack_destroy(id);

争论类型 描述
id The id of the data structure to remove.

返回。

例子。

if (lives == 0)
{
    ds_stack_destroy(AI_stack);
    AI_stack = -1;
    room_goto(rm_Menu);
}

上述代码将检查内置的全局变量 "lives "的值，如果它是0，它将销毁在变量 "AI_stack "中索引的DS栈，然后改变rooms 。
