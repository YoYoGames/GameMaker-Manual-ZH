ds_list_sort

通过这个函数，你可以对一个列表中的所有数值进行排序，可以按升序或降序排序。如果列表中包含strings ，这些值将按照英文26个字母组成的字母表进行排序。

语法。

ds_list_sort(id, ascend);

争论类型 描述
id The id of the list to sort.
ascend Whether the values should be ascending (true)或降序(false)的顺序。

返回。

例子。

if newgame
{
    ds_list_sort(name_list, true);
}

如果变量 "newgame "被标记为 true ，上述代码将对变量 "name_list "中索引的列表进行排序。

返回。DS列表
