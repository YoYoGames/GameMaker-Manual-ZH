ds_list_replace

这个函数将用另一个值替换给定位置的值。

语法。

ds_list_replace(id, pos, val);

争论 类型 描述
id DS列表ID The id of the list to change.
pos 真实的 The position to replace the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val 变化的 The new value to replace the given value with.

返回。

N/A

例子。

ds_list_replace(n_list, 3, name);

上述代码将把存储在列表中第3位的数值替换为存储在变量 "name "中的数值。

返回。DS列表
