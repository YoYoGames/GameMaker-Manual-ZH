ds_list_set

这个函数可以用来设置一个先前添加的列表条目。你给列表 ID（如你创建列表时返回的）和列表中要设置的位置，以及要设置的值。注意，如果被设置的条目超出了列表的范围（即你设置了列表条目 20，但当前列表只包含 10 个条目），那么列表将被填充到给定的位置，每个条目将被设置为 0。

语法。

ds_list_set(id, pos, val);

争论 类型 描述
id DS列表ID The id of the list to add to.
pos 变化的 The position within the list to set.
[val2, ... max_val] 变化的 The value to set in the list.

返回。

N/A

例子。

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

上述代码将把存储在 "score "变量中的值添加到变量 "sc_list "索引的列表中。

返回。DS列表
