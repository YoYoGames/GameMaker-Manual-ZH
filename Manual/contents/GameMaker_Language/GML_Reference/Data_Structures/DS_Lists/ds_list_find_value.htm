ds_list_find_value

通过这个函数，您可以检查给定的列表位置，并返回该位置在列表中的值。请注意，如果您给出的位置超出了给定的列表大小（例如：10个值的列表中的第11个位置），那么该函数可能会返回 undefined 或0。这是因为当您创建列表时，在内部将列表中的前几个条目设置为 0，以减少最初向列表添加项目时的性能问题（尽管该 ds_list_size()函数仍然会在一个新创建的列表中返回 0）。如果你想确保列表在创建时是 "真正的 "空的，那么你应该在创建列表后调用 ds_list_clear()这将意味着任何未填充的列表槽的返回值将是 undefined ，然后你可以使用 is_undefined()函数来检查。

语法。

ds_list_find_value(id, pos);

争论 类型 描述
id DS列表ID The id of the list to use.
pos 真实的 The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

返回。

Variable or undefined

例子。

val = ds_list_find_value(list, ds_list_size(list) - 1);

上面的代码检查变量 "list "中索引的列表中的最后一个位置，并将返回值存储在变量 "val "中。

返回。DS列表
