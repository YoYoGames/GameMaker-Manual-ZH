ds_list_insert

这个函数将把给定的值添加到列表的给定位置。如果列表在给定位置之后包含更多的值，它们的位置将被上移一个，使room ，使列表大一个。

语法。

ds_list_insert(id, pos, val);

争论 类型 描述
id DS列表ID The id of the list to add to.
pos 真实的 The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val 变化的 The value to add to the list.

返回。

N/A

例子。

ds_list_insert(list, 9, score);

上述代码将把存储在变量 "score "中的值添加到由变量 "list "索引的列表的第10个位置（列表从0开始计算，所以一个10个值的列表的编号是0-9）。

返回。DS列表
