ds_list_empty

通过这个函数，你可以检查给定的DS列表是否为空（返回 true ）或不为空（返回 false ）。

语法。

ds_list_empty(id);

争论 类型 描述
id DS列表id The id of the data structure to check.

返回。

例子。

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

上面的代码检查一个变量，看它是否达到了一个特定的值，如果达到了，就清除变量 "command_list "中索引的DS列表，设置一个警报，并将变量重置为0。

返回。DS列表
