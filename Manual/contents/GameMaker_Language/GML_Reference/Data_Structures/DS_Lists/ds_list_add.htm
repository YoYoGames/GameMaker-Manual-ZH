ds_list_add

这个函数可以用来向列表中添加一个新的值（实数或string ），该值将被添加到列表的最后。该函数可以接受更多的可选参数（根据你的需要），允许你在一次调用中连续向列表中添加多个值。

语法。

ds_list_add(id, val1 [, val2, ... max_val]);

争论 类型 描述
id DS列表ID The id of the list to add to.
val 变化的 The value to add to the list.
[val2, ... max_val] 变化的 Optional values to be added to the list.

返回。

N/A

例子。

ds_list_add(sc_list, score);

上述代码将把存储在 "score "变量中的值添加到变量 "sc_list "索引的列表中。

返回。DS列表
