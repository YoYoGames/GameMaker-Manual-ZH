ds_list_find_index

通过这个函数，你可以检查给定列表中的一个值，并返回该值在列表中的位置。请注意，如果列表中存在多个具有相同值的条目，可以返回其中任何一个条目的位置，如果该值不存在，则将返回-1。

语法。

ds_list_find_index(id, val);

争论 类型 描述
id DS列表ID The id of the list to use.
val 变化的 The value to find.

返回。

Real

例子。

pos = ds_list_find_index(list, "Player1");

上述代码检查变量 "list "中索引的列表的值 "Player1"，并将返回的位置存储在变量 "pos "中。

返回。DS列表
