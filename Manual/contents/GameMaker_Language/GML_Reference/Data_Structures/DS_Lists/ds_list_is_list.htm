ds_list_is_list

通过这个函数，你可以检查在一个DS列表中的给定位置是否存放有另一个DS列表。如果给定的位置包含一个DS列表的ID，那么该函数将返回 true ，否则它将返回 false 。

请注意，这将只检测到使用 ds_list_mark_as_list() 功能手动标记的列表。

语法。

ds_list_is_list(id, pos);

争论类型 描述
id The id of the list to check.
pos The position within the list to check.

返回。

例子。

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if ds_list_is_list(ships, i)
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

上面的代码loops 一个DS列表，并检查是否有条目包含其他的列表ID。如果它们有，那么这些列表就会被销毁，然后在loop 结束后，主列表就会被销毁。

返回。DS列表
