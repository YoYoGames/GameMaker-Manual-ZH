ds_list_mark_as_list

这个函数将 "标记"（或 "标志"）一个先前创建的DS列表中的给定位置，作为持有另一个DS列表。这个功能在对JSONstrings （你可以用JSON创建）进行编码时是必需的。 json_encode())，但在嵌套数据结构时也很有用，因为当父级DS列表被销毁时，以这种方式标记的项目将自动被垃圾收集（销毁）。这意味着你不必在销毁 "父 "级列表之前手动浏览列表内容并单独销毁被标记的数据结构。然而，如果你单独删除列表位置，它所包含的DS列表将不会被垃圾收集，你应该在删除父列表位置上的项目之前调用相应的DS列表销毁函数。还要注意，如果你在一个列表上调用函数 ds_list_clear()时，任何被标记为列表的项目在列表被清除时也会被销毁。

同样值得注意的是，如果被标记的位置在列表中发生了变化，"标记 "也会随之移动，所以如果你在列表中标记了位置3（例如），然后在它下面再插入2个项目，使它上升到位置5，它仍然会被标记为一个列表。

语法。

ds_list_mark_as_list(id, pos);

争论类型 描述
id The id of the list to mark.
pos The position within the list to mark.

返回。

DS List ID or -1 (if it fails)

例子。

var j_list = ds_list_create();
var sub_list = ds_list_create();
ds_list_add(sub_list, health);
ds_list_add(sub_list, lives);
ds_list_add(sub_list, score);
ds_list_add(j_list, sub_list);
ds_list_mark_as_list(j_list, 0);

上面的代码创建了两个DS列表，然后用全局变量的各种值填充其中一个。然后将这个列表添加到第二个列表中，并将这个位置 "标记 "出来，以便以后能够正确地进行编码。

返回。DS列表
