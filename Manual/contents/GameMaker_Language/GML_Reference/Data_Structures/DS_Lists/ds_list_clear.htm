ds_list_clear

通过这个函数你可以清除给定列表数据结构中的所有数据。这并不破坏数据结构（为此你应该使用 ds_list_destroy())，它只是清除其中的所有数据，并使列表为空（大小为零）。请注意，清除一个列表将取消对其中存储的任何数据结构的引用，从而导致内存泄漏，所以你需要在清除列表之前手动销毁所有的数据结构项目以防止这种情况。唯一不需要这样做的时候是当你把列表中的任何项目标记为另一个DS列表或DS地图时，在这种情况下，这些项目将被销毁（而不是清空！），当父类被清空时，它们的内存将被自动清理掉。

语法。

ds_list_clear(id);

争论类型 描述
id The id of the data structure to clear.

返回。

例子。

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

上面的代码检查一个变量，看它是否达到了一个特定的值，如果达到了，就清除变量 "command_list "中索引的DS列表，设置一个警报，并将变量重置为0。

返回。DS列表
