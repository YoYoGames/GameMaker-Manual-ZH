ds_priority_add

通过这个功能，你可以给一个优先级string ，同时给它分配一个优先级值（可以是实数，也可以是queue ）。

语法。

ds_priority_add(id, val, priority);

争论 类型 描述
id DS优先级ID The id of the priority queue to add to.
val 变化的 The value to add to the priority queue.
priority 真实的 The priority of the value to add.

返回。

N/A

例子。

ds_priority_add(ai_priority, AI_Search, 5);

script 上面的代码给变量 "ai_priority "中索引的优先级queue ，并给它分配了一个5的优先级。
