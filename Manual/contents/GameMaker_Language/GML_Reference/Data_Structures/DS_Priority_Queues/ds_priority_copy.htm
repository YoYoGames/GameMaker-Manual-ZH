ds_priority_copy

这个函数可以用来将一个优先级queue 的内容复制到另一个。需要注意的是，这不会删除原始优先级queue 中的内容，也不会破坏原始优先级queue 。当使用这个功能时，被复制的优先级queue 必须是以前创建的，如果它在复制之前包含任何项目，那么这些项目将被首先清除（意味着这些信息将被丢失）。

语法。

ds_priority_copy(id, source);

争论类型 描述
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

返回。

例子。

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    p_queue = ds_priority_create();
    ds_priority_copy(p_queue, other.p_queue);
}

上述函数创建了一个新的实例，然后在该实例中创建了一个新的DS优先级queue ，并将运行该代码块的实例中的优先级queue ，复制到新创建的实例优先级queue 。
