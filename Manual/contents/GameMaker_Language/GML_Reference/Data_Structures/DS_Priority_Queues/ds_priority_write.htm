ds_priority_write

该函数返回一个string ，然后可以用该函数存储或转移到另一个数据结构中。 ds_priority_read()函数来存储或转移到另一个数据结构中。

注意：返回的string ，不是人类可读的string ，而是数据结构内容的转储。

语法。

ds_priority_write(id);

争论类型 描述
id The id of the data structure to check.

返回。

例子。

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

上面的代码打开了一个ini文件，然后写了一个string ，其中包含存储在DS优先级queue ，并在变量 "p_queue "中索引的信息。然后清空优先级queue ，并关闭ini文件。
