ds_priority_find_max

通过这个函数，你可以找到存储在优先级queue ，优先级最高的值，如果有多个值具有相同的优先级，那么它们中的任何一个都可以按任何顺序被返回。然而，与之不同的是 ds_priority_delete_max()，这个函数不会将值从queue 。

语法。

ds_priority_find_max(id);

争论类型 描述
id The id of the priority queue to use.

返回。

Real or String

例子。

if ai_move
{
    script_execute(ds_priority_find_max(ai_priority));
}

上面的代码检查了一个实例变量，如果它返回 true ，它将执行一个script ，在优先级queue ，优先级值最高的函数。
