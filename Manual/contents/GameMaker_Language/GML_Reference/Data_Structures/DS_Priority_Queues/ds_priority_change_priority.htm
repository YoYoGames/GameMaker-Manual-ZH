ds_priority_change_priority

这个函数将接受一个给定的值，并改变它在参考的优先级queue 。给定的值应该已经存在于优先权中queue 。

语法。

ds_priority_change_priority(id, val, priority);

争论 类型 描述
id DS优先级ID The id of the priority queue to change.
val 变化的 The value to change the priority of.
priority 真实的 The new priority of the value.

返回。

N/A

例子。

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

上述代码检查一个全局变量，如果它低于某个值，那么它将改变优先级script 函数索引中持有的优先级queue 。
