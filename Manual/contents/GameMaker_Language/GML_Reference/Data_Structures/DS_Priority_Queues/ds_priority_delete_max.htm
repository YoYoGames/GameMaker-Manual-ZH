ds_priority_delete_max

这个函数将返回在queue 中具有最高优先级的值，然后从数据结构中删除该值（和优先级）。如果有多个值具有相同的优先级，那么它们中的任何一个都可以按任何顺序返回，但所有其他具有相同优先级的值仍然会在queue 。

语法。

ds_priority_delete_max(id);

争论类型 描述
id The id of the priority queue to use.

返回。

(Data type stored in the priority)

例子。

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

上面的代码检查了一个实例变量，如果它返回 true ，它将执行一个script ，索引在优先级queue ，优先级值最高，然后将该script ，从queue 。
