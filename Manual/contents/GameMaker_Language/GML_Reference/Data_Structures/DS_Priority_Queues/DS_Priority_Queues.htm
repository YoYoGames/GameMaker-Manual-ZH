DS优先队列

一个DS优先级queue 是一个特殊的数据结构，类似于一个 DS queue一样，只是在这里，queue 的值是按照这些值被分配的优先级（总是实数）来排序的。这种结构对于构建排行榜或信息列表非常有用，其中每个条目的重要性可以被加权。

注意：与所有的动态资源一样，数据结构会占用内存，所以当不再需要时，应该总是销毁，以防止内存泄漏，这将减缓并最终使你的游戏崩溃。

注意：如果你需要检查一个数据结构是否存在，你可以使用 ds_exists()函数。

存在以下处理DS优先级队列的功能。

  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
