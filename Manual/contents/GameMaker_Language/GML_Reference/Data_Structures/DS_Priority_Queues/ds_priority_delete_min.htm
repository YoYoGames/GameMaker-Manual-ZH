ds_priority_delete_min

这个函数将返回queue 中优先级最低的值，然后从数据结构中删除该值（和优先级）。如果有多个值具有相同的优先级，那么它们中的任何一个都可以按任何顺序返回，但所有其他具有相同优先级的值仍然会在queue 。

语法。

ds_priority_delete_min(id);

争论类型 描述
id The id of the priority queue to use.

返回。

(Data type stored in the priority)

例子。

if ai_move
{
    script_execute(ds_priority_delete_min(ai_priority));
}

上面的代码检查了一个实例变量，如果它返回 true ，它将执行一个script ，该函数的索引是优先级最低的queue ，然后将该script 从queue 。
