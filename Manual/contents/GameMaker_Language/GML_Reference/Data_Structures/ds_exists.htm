ds_exists

通过这个函数，你可以检查一个给定类型的数据结构是否存在。你提供 "索引 "值（在一个变量中持有）和DS "类型"，它可以是下面列出的任何常数，如果数据结构存在，该函数将返回 true ，否则返回 false 。

DS型常数
恒定 描述
ds_type_map 一个地图数据结构
ds_type_list 一个列表数据结构
ds_type_stack 一个堆栈数据结构
ds_type_grid 一个网格数据结构
ds_type_queue 一个队列数据结构
ds_type_priority 一个优先权数据结构

注意你不能用这个函数来检查一个数据结构的类型，因为同一个索引号可能被不同类型的多个数据结构使用。

语法。

ds_exists(ind, type);

争论 类型 描述
ind 任何DS ID The variable index to check for the data structure
type DS型常数 The type of data structure to check for (see the list of constants above)

返回。

Boolean

例子。

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

上面的代码检查（先前初始化的）变量 "ai_grid"，看它是否索引了一个DS网格类型的数据结构，如果没有，就创建一个，并将其索引存入变量。
