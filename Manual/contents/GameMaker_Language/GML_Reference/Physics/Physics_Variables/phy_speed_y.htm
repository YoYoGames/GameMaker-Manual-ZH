phy_speed_y

这个变量可以用来获取或改变实例的线性速度矢量的Y分量，并以每步的像素数定义（关于每秒的像素数，见 phy_linear_velocity_y).对于一个静态实例（即：密度为0的实例）来说，改变这个变量将把它变成一个运动学实例。

语法。

phy_speed_y;

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

if phy_speed_y != 0
{
    phy_speed_y = 0;
}

上述代码将检查线性速度矢量的Y分量，如果它不是0，就将其设置为0。
