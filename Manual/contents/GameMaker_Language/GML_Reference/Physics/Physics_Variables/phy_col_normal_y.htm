phy_col_normal_y

这个只读变量返回碰撞法线的y分量，对应于 phy_collision_y数组的值。对于每个接触点，都有一个相关的接触法线（通常对于碰撞中的所有接触点都是同一个法线）。这个接触法线是一个单位向量，从碰撞中的一个实例指向另一个实例，例如，可以用来计算正确的 "推 "的方向来解决碰撞问题

注意：这个变量只在启用了物理学的实例的碰撞事件中可用。

语法。

phy_col_normal_y;

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

上面的代码使用接触法线来设置在两个启用物理的实例之间的碰撞事件中创建的实例的运动方向。
