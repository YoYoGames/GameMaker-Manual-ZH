湿度

这个只读的变量返回实例的质量，单位是公斤。这个值是根据分配的灯具的表面积和它们的密度值自动计算出来的，但它可以通过函数来改变 physics_mass_properties().

语法。

phy_mass;

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

上面的代码来自于实例与另一个实例的碰撞事件，它比较了每个实例的质量，并摧毁了质量较小的那个。
