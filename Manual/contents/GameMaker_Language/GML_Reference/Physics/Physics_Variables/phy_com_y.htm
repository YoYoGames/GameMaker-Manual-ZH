phy_com_y

这个只读变量将返回实例质量中心的y位置。这是根据实例的密度、惯性和质量自动计算出来的，由相应的函数定义

语法。

phy_com_y;

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

xx = phy_com_x;
yy = phy_com_y;

上述代码将两个变量设置为实例质心的x和y位置。
