phy_speed

这个只读变量返回启用物理的实例的当前速度，定义为每步像素。如果你需要改变这个值，你必须通过改变X和Y向量来实现，使用变量 phy_speed_x和 phy_speed_y.

语法。

phy_speed;

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

上面的代码检查物理学启用实例的速度，然后根据返回值改变线性阻尼。
