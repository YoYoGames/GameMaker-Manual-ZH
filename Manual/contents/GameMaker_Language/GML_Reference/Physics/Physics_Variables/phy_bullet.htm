phy_bullet

这个变量定义了实例是否是极速移动的（例如子弹）。默认值是 false ，但如果设置为 true ，这就告诉GameMaker，实例将以如此高的速度移动，它将需要更昂贵的碰撞检测，以确保它不会穿过其他实例而不被察觉。

语法。

phy_bullet

返回。

(or undefined if the instance is not physics enabled)

例子。

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

上面的代码创建了一个新的实例，然后将其定义为物理世界中的 "子弹"，然后给它一个沿y轴的巨大冲力。
