phy_rotation

这个变量可以用来获取（或设置）实例灯具的角度，单位是度，类似于设置或获取. image_angle.但是请注意，在物理世界中，旋转的计算方式与普通GameMaker游戏世界的计算方式相反，这意味着矢量函数如 point_direction()应该修改它们的返回值（简单地将正数改为负数就可以解决这个问题）。

语法。

phy_rotation

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

上述代码检查线性速度，如果任一矢量不为0，它就会计算出实际速度，并使用该速度来设置旋转。
