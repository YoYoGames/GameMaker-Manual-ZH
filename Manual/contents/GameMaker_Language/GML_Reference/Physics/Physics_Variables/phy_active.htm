phy_active

这个变量控制实例当前是否处于 "活动 "状态。将其设置为 false ，将阻止实例参与物理世界，而将其设置为 true ，将使其再次参与。请注意，这并不等同于停用该实例，因为该实例在屏幕上仍然是可见的，仍然可以通过代码进行更改，而这个函数只是阻止它参与物理模拟。

语法。

phy_active

返回。

(or undefined if the instance is not physics enabled)

例子。

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause;
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

上面的代码将检测到一个字母 "P "的按键，然后将全局变量 "Pause "从 true 切换到 false ，再切换回来。然后这个变量被用来设置物理学在变量 "obj_Parent "中索引的object 的子实例中是否被激活。
