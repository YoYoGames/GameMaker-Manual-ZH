phy_sleeping

这个只读变量返回实例当前是否在 "睡眠"（true ）或不在（false ），一个 "睡眠 "的实例是指没有积极从事任何物理模拟的实例。当一个实例处于休息状态并且没有与另一个实例发生碰撞时，GameMaker将使objects ，以节省模拟周期。

语法。

phy_sleeping;

返回。

(or undefined if the instance is not physics enabled)

例子。

if phy_sleeping
{
    instance_destroy();
}

上面的代码检查object 是否被主动模拟，如果不是，则被销毁。
