phy_position_x

这个变量可以用来获取（或设置）实例在游戏中的X位置room 物理世界。请注意，当直接设置这个变量来移动实例时，物理世界可能会出现错误，因为它将中断 连续模拟。这个变量相当于物理学中的实例变量 x.

语法。

phy_position_x;

返回。

(single precision floating point value, or undefined if the instance is not physics enabled)

例子。

xx = phy_position_x;
yy = phy_position_y;

上述代码将实例的x和y位置存储在两个变量中。
