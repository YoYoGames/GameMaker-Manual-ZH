物理学_暂停_启用

通常情况下，为了使物理模拟工作，它必须是连续的，不能停止和启动，或者在room ，让实例突然从一个地方移到另一个地方。然而，有些时候你需要暂停模拟，因为（例如）设备os已经暂停了，所以你会使用这个功能。如果flag 被设置为 true ，它就会暂停模拟，在flag 再次被设置为 false 之前，不会再进行物理计算。

注意：如果你想停用一个room 中的所有实例，这一点特别有用，因为即使停用了一个物理体，仍然会继续在物理世界中进行计算和模拟。

语法。

physics_pause_enable(flag)

争论类型 描述
flag This can be set to true 暂停模拟，或者 ，再次启动模拟。 false

返回。

例子。

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

上面的代码检查操作系统是否已经暂停，如果已经暂停，那么它就暂停物理世界，停用除自己以外的所有东西，然后创建一个暂停菜单实例。
