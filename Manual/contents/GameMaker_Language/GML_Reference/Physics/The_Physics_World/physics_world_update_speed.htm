物理学_世界_更新_速度

GameMaker根据room 速度来更新东西，所以如果你把它设置为例如30，GameMaker将在一秒钟内运行30步。然而，对于物理功能来说，这可能是不够的，你可能希望事情以稍快的速度更新，以提高稳定性或精度。为此，我们使用了函数 physics_world_update_speed() ，它为物理系统设置了独立于room 的更新速度。这意味着，你可以有一个30的room 速度，但将物理系统设置为60，与步骤更新的速度相比，物理系统更新和执行计算的速度有效地增加了一倍。

注意：你目前不能将其设置为任何低于room 的速度，但未来的更新可能会改变这一点。

语法。

physics_world_update_speed(speed)

争论 类型 描述
speed 真实的 The number of times per second that the physics system updates

返回。

N/A

例子。

physics_world_update_speed(room_speed * 2);

上面的代码将物理系统的更新速度设置为room ，是速度的两倍。
