物理学_粒子_set_group_flags

通过这个函数，你可以改变一组粒子的组flags 。 group 的值是你使用函数创建粒子组时返回的值。 physics_particle_group_end()，而flags 是以下常数的组合的返回值。

恒定 描述
phy_particle_group_flag_solid 固体粒子组可以防止其他固定装置在其内部停留。如果有什么东西穿透了它，固体颗粒组就会把违规的夹具推回它的表面，使固体颗粒组具有特别强的排斥力。
phy_particle_group_flag_rigid 刚性粒子组是指其形状不会改变的粒子，即使它们与其他固定装置相撞。

这些组flags ，使用位屏蔽来创建一个最终的输出值，然后检查来设置组的不同基本属性，除了那些由粒子flags 。你可以通过使用位法或"|"来屏蔽相应的位，来使用这些常量中的任何一个、一个或两个（如下面的例子所示）。

语法。

physics_particle_set_group_flags(group, flags)

争论 类型 描述
group 物理学粒子组ID The particle group to set.
flags 物理学 粒子组Flag 常数(s) The flags to set on the particle group.

返回。

N/A

例子。

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

上述代码将创建一个变量来存储flags ，然后用它来设置由变量 "group "索引的组中创建的所有粒子的flags 。
