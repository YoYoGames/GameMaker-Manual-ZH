physics_particle_get_data

这个函数使用给定的flags 检查，返回物理模拟中每个粒子的各种信息。所使用的buffer ，必须是先前用函数创建的。 buffer_create()创建的，并且应该是 "增长 "类型的，其大小与预期返回的数据差不多。flags 是使用下面给出的任何一个常数来设置的，你将使用位法或"|"来创建一个单一的flag 值来获得所需的信息。

物理学 粒子数据Flag 常数
恒定 描述 数据类型
phy_particle_data_flag_typeflags 粒子的flags 值。 buffer_u32
phy_particle_data_flag_position 粒子的X和Y位置。 2 x buffer_f32
phy_particle_data_flag_velocity 水平和垂直速度。 2 x buffer_f32
phy_particle_data_flag_colour 颜色和阿尔法值（十六进制）。 buffer_f32
phy_particle_data_flag_category 粒子类别（在你创建粒子或其所属的组时定义的）。 buffer_u32

语法。

physics_particle_get_data(buffer, flags)

争论 类型 描述
buffer 缓冲区ID The (previously created) buffer to use to store the data.
flags 物理学 粒子数据Flag 常数(s) The flags to use to extract data about specific particle types.

返回。

N/A

例子。

var count = physics_particle_count();
var flags = phy_particle_data_flag_position | phy_particle_data_flag_colour;
if (count > 0)
{
    var buffer = buffer_create(count * 12, buffer_grow, 4);
    physics_particle_get_data(buffer, flags);
    for (var n = 0; n < count; n++;)
    {
        var xx = buffer_read(buffer, buffer_f32);
        var yy = buffer_read(buffer, buffer_f32);
        var argb = buffer_read(buffer, buffer_u32);
        var alpha = (argb >> 24) & 255;
        draw_sprite_ext(sprBlob, 0, xx, yy, 1, 1, 0, c_green, alpha);
    }
    buffer_delete(buffer);
}

上面的代码得到了粒子的数量，并且用数据创建了一个变量flags ，以进行检查，然后检查room 。如果有，就创建一个buffer ，然后用粒子数据填充，经过检查，在粒子位置画一个sprite 。
