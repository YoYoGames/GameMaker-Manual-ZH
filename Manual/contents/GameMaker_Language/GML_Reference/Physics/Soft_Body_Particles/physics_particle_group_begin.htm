physics_particle_group_begin

通过这个函数，你可以在一个room 。 粒子组是指创建一个特定形式所需的粒子作为一个组被一次性创建，允许你创建模拟的软体，并由所使用的flags ，设置各种属性。标志 "是粒子常数的组合的返回值（详见函数 physics_particle_create()的细节），而 "groupFlags "是以下组常数的组合（或者0表示没有flags ）。

物理学 粒子组Flag 常数
恒定 描述
phy_particle_group_flag_solid 固体粒子组可以防止其他固定装置在其内部停留。如果有什么东西穿透了它，固体颗粒组就会把违规的夹具推回它的表面，使固体颗粒组具有特别强的排斥力。
phy_particle_group_flag_rigid 刚性粒子组是指其形状不会改变的粒子，即使它们与其他固定装置相撞。

这些组flags ，使用位屏蔽来创建一个最终的输出值，然后检查来设置组的不同基本属性，除了那些由粒子flags 。你可以通过使用位法或"|"来屏蔽相应的位，来使用这些常量中的任何一个、一个或两个（如下面的例子所示）。

除了flags ，你还可以在room ，设置创建粒子群的位置（给出的位置将是完成的粒子群的质量中心），初始水平和垂直速度，以及粒子群的角速度，允许你创建移动的、旋转的粒子群。

你还可以设置组中所有粒子的初始颜色和阿尔法，这些值在使用函数绘制粒子时被使用。 physics_particle_draw().之后，你可以设置组中粒子的内聚强度，这是另一种说法，即它们彼此 "粘 "在一起的程度。强度可以是一个介于0和1之间的值，其中0代表没有内聚力，1代表最大内聚力。

最后，你必须为你的粒子组给出类别。这个值是一个任意的整数，你给每个你想拥有类似属性的粒子组，它可以用来在以后为该特定类别设置flags ，以及其他一些事情。

注意：你可以使用任何整数值作为类别，但0除外，因为0被模拟保留，用于选择其他功能中的所有类别（如用于绘图）。

需要注意的是，调用这个函数并不会在物理学room 中创建组。这个函数只是开始了这个过程，并为该组设置了基本的粒子属性。要真正创建粒子组，你需要给它一个形式（见下面的例子），然后调用函数 physics_particle_group_end()，这时它将被创建。

注意：你设置的粒子类型flags ，也会影响到粒子组的内聚力，例如，如果你flag 粒子是水的类型（例如），当该组与另一个夹具碰撞时，它就会破裂，而如果你flag 粒子是弹簧类型，那么它在碰撞时将保持内聚力。

语法。

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

争论 类型 描述
flags 物理学 粒子Flag 常数(s) The flags to set on the particle.
groupflags 物理学 粒子组Flag 常数(s) The group flags to set on the particle group (see table at the top).
x 真实的 The x position to create the particle group at.
y 真实的 The y position to create the particle group at.
ang 真实的 The angle (in degrees) to rotate the group to on create.
xv 真实的 The initial horizontal velocity.
yv 真实的 The initial vertical velocity.
ang_velocity 真实的 The initial angular velocity.
col 颜色 The base colour to use for the particle group.
alpha 真实的 The base alpha to use for the particle group.
strength 真实的 Set the cohesion strength between particles in the group.
category 真实的 The user defined category to which the group belongs.

返回。

N/A

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

上面的代码在变量中存储了flags 粒子类型和粒子组属性，然后使用这些变量在鼠标位置创建一个半径为100px的圆形粒子组。
