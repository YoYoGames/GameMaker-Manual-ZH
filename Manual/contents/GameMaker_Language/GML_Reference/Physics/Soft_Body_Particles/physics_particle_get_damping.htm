物理学_粒子_获取阻尼

通过这个函数，你可以找出物理模拟中的粒子当前的线性阻尼是多少（你可以用以下方法设置这个值 physics_particle_set_damping()).

语法。

physics_particle_get_damping()

返回。

例子。

if (physics_particle_get_damping() < 1)
{
    physics_particle_set_damping(physics_particle_get_damping() + 0.01);
}

上述代码将检查系统中所有粒子的当前阻尼值，如果它小于1，那么它将增加0.01。
