physics_particle_draw

这个函数将在所有共享相同的用户定义的类别值的粒子的位置上绘制一个sprite ，并且这些粒子的flags ，与为掩码设置的相一致。遮罩值的定义方式与你在创建粒子时定义flags 的方式相同（见函数 physics_particle_create()为例），即：使用位法或"|"来屏蔽每个flag 。

注意：如果用户类别相同，该函数将绘制物理模拟中的所有粒子，flags ，与掩码的粒子相吻合，无论它们是单个粒子还是群体。

绘制的粒子将使用你创建它时设置的颜色和阿尔法值，并将随着粒子的角动量旋转。然而，如果你希望缩放sprite ，或者改变它的任何其他绘制属性，你应该使用函数 physics_particle_draw_ext().

语法。

physics_particle_draw(typemask, category, sprite, subimg)

争论 类型 描述
typemask 物理学 粒子Flag 常数(s) The mask to compare the particles flags with.
category 真实的 The user defined category to draw (use 0 to draw all categories).
sprite 雪碧资产 The sprite to use for each particle.
subimg 真实的 The image_index 的精灵来绘制。

返回。

N/A

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

上述代码将为用户定义的类别 "2 "中的所有粒子绘制一个sprite ，这些粒子具有与给定掩码相同的flags 。
