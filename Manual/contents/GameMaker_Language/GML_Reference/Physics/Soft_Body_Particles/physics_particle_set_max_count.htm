物理学_粒子_设置最大计数

这个函数将设置物理模拟中允许的粒子总数。如果你把这个值设置为0，那么创建的粒子就没有上限，而把它设置为任何其他的值，当粒子总数等于这个值时，就会停止创建任何进一步的粒子。 粒子总数等于该值时，将停止进一步创建粒子。请注意，你在一次模拟中可以创建的最小粒子数是128，所以将这个值设置得再低也不会有什么影响（除非设置为0）。

语法。

physics_particle_set_max_count(count)

争论类型 描述
count The maximum number of particles to permit.

返回。

例子。

physics_particle_set_max_count(500);

上述代码将设置物理模拟的粒子上限为500。
