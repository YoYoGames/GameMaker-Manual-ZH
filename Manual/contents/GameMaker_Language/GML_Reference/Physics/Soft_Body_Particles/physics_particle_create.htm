物理学_粒子_创建

通过这个函数，你可以在游戏的任何地方创建一个单一的粒子room ，设置某些flags 和属性。这些 "标志 "是以下常数的组合的返回值。

物理学 粒子Flag 常数
恒定 描述
phy_particle_flag_water 软体粒子的默认属性。
phy_particle_flag_zombie 僵尸粒子是指在一个步骤之后就会被销毁的粒子，所有其他的粒子都以这种方式标记。
phy_particle_flag_wall 这将粒子定义为静态的，本质上是在物理模拟中把它创建为一个不可移动的object ，因为无论什么东西与它们碰撞，它们都将保持在一个固定的位置。你应该使用这个flag ，而不是将密度设置为0。
phy_particle_flag_spring 弹簧粒子产生了相互连接的效果，就像通过弹簧一样。用这种flag 创建的粒子是成对 "连接 "的，每个粒子与创建时离它最近的那个粒子相连。一旦成对，粒子不会改变 "伙伴"，而且外力将它们拉得越远，当外力被移除时，它们碰撞的力量就越大。请注意，无论成对的粒子离彼此有多远，它们之间的联系都不会断裂。
phy_particle_flag_elastic 弹性粒子变形，当它们与物理模拟中的其他刚性体碰撞时，也可能会反弹。
phy_particle_flag_viscous 粘性颗粒是一种表现出 "粘性 "或 "粘性 "的颗粒，如油。粘性颗粒会更多地结块和粘在一起。
phy_particle_flag_powder 粉末颗粒会产生散射效果，就像你可能看到的沙子或灰尘一样。
phy_particle_flag_tensile 拉伸粒子被用来产生表面张力的效果，或液体体表面的紧绷曲率。例如，它们可能被用来产生你在一滴水上看到的表面张力。一旦张力被打破，粒子就会像有弹性一样反弹，但也会继续相互吸引。因此，颗粒在反弹时往往会形成团块。
phy_particle_flag_colourmixing 混色粒子会吸收与之碰撞的其他粒子的一些颜色。请注意，如果两个碰撞的粒子中只有一个是混色粒子，另一个粒子就会保留其碰撞前的颜色。

这些flags ，使用位掩码来创建一个最终的输出值，然后通过检查来设置粒子的不同基本属性（基本属性总是 phy_particle_flag_water ）。例如，如果你想模拟具有表面张力的粘稠液体，你将使用位法或"|"来flag 适当的位，如下面的例子中所示。通过这种方式，你可以为创建的每个单独的粒子设置不同的属性（除全局属性外）。

除了flags ，你还可以在room ，设置创建粒子的位置、初始水平和垂直速度、初始颜色和alpha，以及用户类别。颜色和阿尔法值只在使用函数绘制粒子时使用。 physics_particle_draw()的时候使用，但是类别值会在其他多个函数中使用。这个值是一个任意的整数，你可以给每个你想拥有类似属性的粒子一个整数，它可以用来在以后为这个特定的类别设置flags ，以及其他一些东西。例如，如果你定义了一个 "水 "和一个 "油 "的粒子，你将给所有那些标记为 "水 "的粒子以1的用户类别，给所有那些创建为 "油 "的粒子以2的用户类别，允许你以后针对单个类别来改变它的flags ，或者获得特定的数据。

注意：你可以使用任何整数值作为类别，但0除外，因为0被模拟保留，用于选择其他功能中的所有类别（如用于绘图）。

该函数为粒子返回一个唯一的索引（或ID）值，该值可以存储在一个变量中，以便在其他直接引用该粒子的函数中使用。

语法。

physics_particle_create(flags, x, y, xv, yv, col, alpha, category)

争论 类型 描述
flags 物理学 粒子Flag 常数(s) The flags to set on the particle.
x 真实的 The x position to create the particle at.
y 真实的 The y position to create the particle at.
xv 真实的 The initial horizontal velocity.
yv 真实的 The initial vertical velocity.
col 颜色 The base colour to use for the particle.
alpha 真实的 The base alpha to use for the particle.
category 真实的 The user defined category to which the particle belongs.

返回。

Physics Particle ID

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var x_vel = lengthdir_x(5, image_angle);
var y_vel = lengthdir_y(5, image_angle);
var p = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1);

上述代码将创建变量来设置粒子类型和它的初始速度，然后用这些变量在实例的X/Y位置创建一个粒子。
