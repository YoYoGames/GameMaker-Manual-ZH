物理学_粒子_删除_区域_保利

通过这个函数，你可以在当前room ，从物理模拟中删除（移除）所有属于定义的多边形区域范围内的粒子。该函数接收一个（先前创建的） ds_list()包含多边形每个点的x/y位置，列表中的偶数位置是x坐标，奇数位置是y坐标，即：对于一个三角形，你的列表将有六个条目，条目0、2和4是x坐标，1、3和5是y坐标。然后，多边形将被用来删除所有落在定义区域内的粒子。

请注意，所定义的多边形必须至少有三个点，最多有八个点。

语法。

physics_particle_delete_region_poly(pointList)

争论类型 描述
pointList A ds_list of points to use to create the polygon.

返回。

例子。

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

上述代码将删除所有在由添加到 DS List 的点定义的多边形区域中发现的粒子。
