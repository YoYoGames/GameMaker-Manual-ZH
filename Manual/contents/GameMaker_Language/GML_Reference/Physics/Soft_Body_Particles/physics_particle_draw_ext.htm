物理学_粒子_绘图_文本

这个函数将在所有共享相同的用户定义的类别值的粒子的位置上绘制一个sprite ，并且这些粒子的flags ，与为掩码设置的相一致。掩码值的定义方式与你在创建粒子时定义flags 的方式相同（见函数 physics_particle_create()为例），即：使用位法或"|"来掩盖每个flag 的适当位。

注意：如果用户类别相同，该函数将绘制物理模拟中的所有粒子，flags ，与掩码的粒子相吻合，无论它们是单个粒子还是群体。

绘制的粒子将忽略你创建它时设置的颜色和阿尔法值，而是使用这个函数中给出的比例、角度、颜色和阿尔法参数来绘制。请注意，这纯粹是为了在粒子的位置上绘制一个sprite ，绝不影响粒子的行为。

语法。

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

争论 类型 描述
typemask 物理学 粒子Flag 常数(s) The mask to compare the particles flags with.
category 真实的 The user defined category to draw (use 0 to draw all categories).
sprite 雪碧资产 The sprite to use for each particle.
subimg 真实的 The image_index 的精灵来绘制。
xscale 真实的 The scale factor to apply to the sprite along the x axis.
yscale 真实的 The scale factor to apply to the sprite along the y axis.
ang 真实的 The angle (in degrees) to draw the sprite at.
col 颜色 The colour to blend with the sprite (default is c_white).
alpha 真实的 The alpha value to use when drawing (0 - 1).

返回。

N/A

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

上述代码将为用户定义的类别 "2 "中的所有粒子绘制一个sprite ，这些粒子具有与给定掩码相同的flags ，沿两轴缩放为2，α值为0.5。
