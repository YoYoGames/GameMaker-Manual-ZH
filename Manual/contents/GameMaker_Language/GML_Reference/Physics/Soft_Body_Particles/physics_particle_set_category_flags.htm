物理学_粒子_set_category_flags

通过这个函数，你可以改变某一类粒子的flags 。 category ，这个值是你在函数中定义的 physics_particle_create()或（如果你创建了一个组） physics_particle_group_begin()，而flags 是以下常数的组合的返回值。

恒定 描述
phy_particle_flag_water 软体粒子的默认属性。
phy_particle_flag_zombie 一个僵尸粒子是一个将在一个步骤中被摧毁的粒子，所有其他的粒子都以这种方式被标记。
phy_particle_flag_wall 这将粒子定义为静态的，本质上是在物理模拟中把它创建为一个不可移动的object ，因为无论什么东西与它们碰撞，它们都将保持在一个固定的位置。你应该使用这个flag ，而不是将密度设置为0。
phy_particle_flag_spring 弹簧粒子产生了相互连接的效果，就像通过弹簧一样。用这种flag 创建的粒子是成对 "连接 "的，每个粒子与创建时离它最近的那个粒子相连。一旦成对，粒子不会改变 "伙伴"，而且外力将它们拉得越远，当外力被移除时，它们碰撞的力量就越大。请注意，无论成对的粒子离彼此有多远，它们之间的联系都不会断裂。
phy_particle_flag_elastic 弹性粒子变形，当它们与物理模拟中的其他刚性体碰撞时，也可能会反弹。
phy_particle_flag_viscous 粘性颗粒是一种表现出 "粘性 "或 "粘性 "的颗粒，如油。粘性颗粒会更多地结块和粘在一起。
phy_particle_flag_powder 粉末颗粒会产生散射效果，就像你可能看到的沙子或灰尘一样。
phy_particle_flag_tensile 拉伸粒子被用来产生表面张力的效果，或液体体表面的紧绷曲率。例如，它们可能被用来产生你在一滴水上看到的表面张力。一旦张力被打破，粒子就会像有弹性一样反弹，但也会继续相互吸引。因此，颗粒在反弹时往往会形成团块。
phy_particle_flag_colourmixing 混色粒子会吸收与之碰撞的其他粒子的一些颜色。请注意，如果两个碰撞的粒子中只有一个是混色粒子，另一个粒子就会保留其碰撞前的颜色。

这些flags ，使用位屏蔽来创建一个最终的输出值，然后通过检查来设置粒子类别的不同基本属性（基本属性总是 phy_particle_flag_water ）。例如，如果你想模拟具有表面张力的粘稠液体，你可以使用位法或"|"来掩盖适当的位，如下例所示。通过这种方式，你可以为创建的每一类粒子设置不同的属性（除全局属性外）。

语法。

physics_particle_set_category_flags(category, flags)

争论 类型 描述
category 真实的 The particle category to set.
flags 物理学 粒子Flag 常数(s) The flags to set on the particle category.

返回。

N/A

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_set_category_flags(1, flags);

上述代码将创建一个变量来存储flags ，然后用它来设置所有创建的类别值为1的粒子的flags 。
