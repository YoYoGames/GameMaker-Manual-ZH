物理学_粒子_获取半径

通过这个函数，你可以找出物理模拟中的粒子的当前半径（以像素为单位）（你可以用以下方法设置这个值 physics_particle_set_radius()).

语法。

physics_particle_get_radius()

返回。

例子。

if (physics_particle_get_radius() < 32)
{
    physics_particle_set_radius(physics_particle_get_radius() + 1);
}

上述代码将检查模拟中粒子的当前半径，如果它小于32，那么它将把它们的大小增加1。
