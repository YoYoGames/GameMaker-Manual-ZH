物理学_粒子_设置_重力尺度

通过这个函数，你可以为物理模拟中的粒子设置重力比例因子。该函数旨在帮助防止物理模拟中的不稳定性，特别是在使用非常小的粒子时，这些粒子的行为可能是不可预知的 (即打破动量守恒)的情况下，如爆炸。通过减少重力尺度来减缓这些粒子的速度可以稳定它们的行为。这个函数是全局性的，因为它将改变重力尺度，而不仅仅是 对于改变后产生的新粒子，而且对于那些已经存在于模拟中的粒子。

注意：调整每一步的更新迭代次数（使用函数 physics_world_update_iterations() 也可以影响重力对粒子的影响。较大的迭代大小赋予了对重力的更大抵抗力。

语法。

physics_particle_set_gravity_scale(scale)

争论类型 描述
scale The scaling factor to be applied to gravity for all particles.

返回。

例子。

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

上述代码将为模拟中的所有粒子设置基本属性。
