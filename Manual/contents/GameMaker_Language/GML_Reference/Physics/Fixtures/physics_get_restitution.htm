physics_get_restitution

当你使用 "Action "将一个灯具绑定到一个实例上时，它会为绑定的灯具返回一个 "ID"。 physics_fixture_bind()返回绑定灯具的 "id"。你可以使用这个id来获得绑定灯具的恢复（"弹跳 "属性） 值，而不是"基本 "灯具，在任何时候使用这个函数。

语法。

physics_get_restitution(fixture)

争论类型 描述
fixture the id of the bound fixture

返回。

例子。

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上面的代码为实例的绑定物理属性获取当前的恢复值，然后将它们设置为不同的值。

返回。固定装置
