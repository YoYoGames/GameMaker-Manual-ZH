物理学_set_restitution

当你使用 "Action "将一个灯具绑定到一个实例上时，它会为绑定的灯具返回一个 "ID"。 physics_fixture_bind()返回一个绑定灯具的 "id"。你可以在任何时候使用这个ID来设置绑定的灯具的恢复值，而不是"基本 "灯具的恢复值。恢复值通常被设置为0到1之间的值，但是如果需要，你可以使用更高的值，尽管结果可能是不可预测的。

语法。

physics_set_restitution(fixture, restitution)

争论类型 描述
fixture the id of the bound fixture
restitution the new restitution value to apply

返回。

例子。

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上面的代码为实例的绑定物理属性获取当前的恢复值，然后将它们设置为不同的值。

返回。固定装置
