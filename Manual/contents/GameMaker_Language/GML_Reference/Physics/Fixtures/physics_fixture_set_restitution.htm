physics_fixture_set_restitution

在物理学中，恢复被定义为 "一个object 或系统在弹性变形后恢复到原来的状态"，但由于GameMaker中的夹具是真正的刚性体，不能变形，恢复是指 实际上是说夹具的 "弹跳 "程度。这个设置将影响一个object 与其他objects 碰撞时的 "弹跳 "程度，并且与作用在实例上的其他力量（如重力和摩擦力）共同相关。 通常是一个介于0和1之间的值（可以使用更高的值，但可能会产生不可预知的结果）。下面是一个关于它如何工作的说明。

语法。

physics_fixture_set_restitution(fixture, restitution)

争论类型 描述
fixture The index of the fixture
restitution The restitution of the fixture (usually between 0 and 1)

返回。

例子。

physics_fixture_set_restitution(fix_Ball, 0.9);

上面的代码将把 "fix_ball "中索引的夹具的还原度设为0.9。

返回。固定装置
