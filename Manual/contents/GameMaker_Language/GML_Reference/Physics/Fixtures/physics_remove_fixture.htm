物理学_移除灯具

这个函数从一个或多个实例中移除（或 "解除绑定"）一个灯具。它需要绑定的灯具的唯一 "id"（由函数返回）。 physics_fixture_bind()它将删除该实例的所有当前定义的物理属性，允许你重新定义一个新的夹具并将其绑定到该实例。通过这种方式，你可以改变实例的物理属性，而不需要破坏和重新创建它。

语法。

physics_remove_fixture(id, fixture)

争论 类型 描述
id 实例ID The ID of the instance to remove the fixture from
fixture 物理夹具ID The ID of the fixture that is to be removed from the instance

返回。

N/A

例子。

physics_remove_fixture(id, my_fix);

上面的代码将从实例中删除存储在变量 "my_fix "中的 "id "的灯具。

返回。固定装置
