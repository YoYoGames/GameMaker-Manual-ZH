physics_set_friction

当你使用 "Action "将一个灯具绑定到一个实例上时，它会为绑定的灯具返回一个 "ID"。 physics_fixture_bind()返回一个绑定灯具的 "ID"。你可以在任何时候使用这个ID来设置绑定灯具的摩擦力值，而不是"基本 "灯具。注意，摩擦力通常被设置为0到1之间的值，但是如果需要，你可以使用任何非负值。

语法。

physics_set_friction(fixture, friction)

争论类型 描述
fixture the id of the bound fixture
friction the new friction value to apply

返回。

例子。

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

上面的代码为实例的绑定物理属性获取当前的摩擦力值，然后将它们设置为不同的值。

返回。固定装置
