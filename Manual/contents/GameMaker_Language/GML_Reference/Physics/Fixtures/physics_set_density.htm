物理学_set_density

当你使用 "Action "将一个灯具绑定到一个实例上时，它会为绑定的灯具返回一个 "ID"。 physics_fixture_bind()返回绑定灯具的 "id"。你可以使用这个id来设置绑定灯具的密度值，而不是使用这个函数来设置 "基本 "灯具，在任何时候使用这个函数。

语法。

physics_set_density(fixture, density)

争论类型 描述
fixture the id of the bound fixture
density the new density value to apply

返回。

例子。

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

上面的代码得到了实例的绑定物理属性的电流密度值，然后将它们设置为不同的值。

返回。固定装置
