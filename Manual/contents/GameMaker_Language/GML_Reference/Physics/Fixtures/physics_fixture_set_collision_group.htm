物理学_灯具_设置碰撞组

一旦你创建了你的灯具，你可以把它定义为碰撞组的一部分。这意味着你可以告诉具有该灯具的实例总是与其他具有相同碰撞组的灯具绑定的实例发生碰撞或从不发生碰撞。可以分配的值是基于正数或负数的，0是默认值（意味着具有这个灯具设置的实例被忽略，除非为它创建一个碰撞事件）。这里有一张图片来帮助解释。

正如你所看到的，设置碰撞组对于创建复杂的物理体是非常有用的，它们与周围的物理世界相互作用，但不是相互作用。然而，应该注意的是，由于基于这样的组计算碰撞的复杂性，建议你在任何一个room ，尽量将使用的组的数量降到最低。

碰撞组的数值范围为-32,768至32,767（包括两者）。

语法。

physics_fixture_set_collision_group(fixture, group);

争论 类型 描述
fixture 物理夹具ID The fixture that is to be bound
group 真实的 The group that this fixture is assigned to (either a positive value, 0, or a negative value)

返回。

N/A

例子。

physics_fixture_set_collision_group(fix_Wheel, -1);

上面的代码将设置变量 "fix_Wheel "中索引的灯具为"-1 "组的一部分。

返回。固定装置
