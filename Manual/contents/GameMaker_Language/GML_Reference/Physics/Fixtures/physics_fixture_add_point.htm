physics_fixture_add_point

这个函数定义了一个多边形的点，这个多边形之前已经被设置为夹具，使用了 physics_fixture_set_polygon_shape()函数定义一个多边形的点，该多边形之前已经被设置为夹具，或者可以使用函数将点添加到一个边缘链 的固定装置链上添加点，使用函数 physics_fixture_set_chain_shape().对于一个多边形形状，这些点是相对于夹具的原点而言的，x/y轴的0,0位置是中心。 随后的坐标在GameMaker中照常计算，向下/右为+x,+y，向上/左为-x,-y。这里有一个插图来帮助你直观地了解这个过程。

正如你所看到的，任何简单的凸形多边形都可以用这种方式建模，但是 凹形多边形目前还不支持。最后，为了关闭多边形，你必须将夹具绑定到一个实例上。你还应该注意，在将多边形绑定到一个实例之前，你还必须为其定义至少三个点。 您还应该注意，在将多边形绑定到一个实例之前，您还必须为您的多边形定义至少三个点，否则您将得到一个错误，而且向一个先前没有设置多边形形状的夹具添加一个点也将导致一个错误。还要注意的是，一个夹具允许的最大点数是八个。

对于链状形状（这只是一系列连接的边缘形状），这个函数将设置一个点作为 "连接"，将边缘链在一起。因此，在将夹具绑定到实例之前，你需要至少定义两个点。 否则你会得到一个错误。然而，与多边形不同的是，你不被限制在最多8个点的范围内，你可以根据需要添加更多的点来创建链条，而且这些点是以实际的room ，而不是相对于实例的坐标来定义。 实例。

语法。

physics_fixture_add_point(fixture, xpos, ypos)

争论类型 描述
fixture the index of the fixture
xpos the x position of the point (relative to the fixture origin for polygon shapes)
ypos the y position of the point (relative to the fixture origin for polygon shapes)

返回。

例子。

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

上面的代码将对变量 "fix_Ship "中索引的夹具应用一个多边形形状，然后定义三个点来创建一个三角形形状。

返回。固定装置
