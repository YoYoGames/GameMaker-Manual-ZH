物理学联合常数

在GameMaker GML 中包含了大量的常量，这些常量是专门针对物理世界中固定装置之间的关节的。这些常量可以与许多不同的关节函数一起使用，以便在物理模拟运行时实时地设置或获取信息。但是，你应该知道，当你调用这些函数时，会进行复杂的计算，所以应该谨慎使用，只有在必要时才使用，并注意许多函数是针对特定类型的接头的。还要注意，虽然你可以用适当的函数获得所有这些值，但你只能设置那些被标记为非只读的值。

一般来说，这些常数将与下列函数一起使用。

  • physics_joint_get_value
  • physics_joint_set_value

以下常数可以应用于任何可用的接头类型。

物理学联合常数
恒定 描述 只读
phy_joint_anchor_1_x 房间内接头的第一个锚点的X坐标。 Yes
phy_joint_anchor_1_y 房间内接头的第一个锚点的y坐标 Yes
phy_joint_anchor_2_x 房间内接头的第二个锚点的X坐标。 Yes
phy_joint_anchor_2_y 房间内接头的第二个锚点的y坐标 Yes
phy_joint_reaction_force_x 这是施加在X锚位置的关节的第二个实例上的反作用力。 Yes
phy_joint_reaction_force_y 这是施加在Y锚位置的关节的第二个实例上的反作用力 Yes
phy_joint_reaction_torque 这是在锚定位置的关节中被施加到第二实例的扭矩 Yes


这些常数是针对那些有电机连接的关节（渐开线、棱柱线、轮子）。

物理学联合常数
恒定 描述 只读
phy_joint_max_motor_force 创建关节时指定的最大马达力的数值。 No
phy_joint_max_motor_torque 创建接头时为最大电机扭矩指定的数值。 No
phy_joint_motor_force 当前的马达力 Yes
phy_joint_motor_speed 当前的电机速度 No
phy_joint_motor_torque 当前的电机扭矩 Yes

物理学联合常数
恒定 描述 只读
phy_joint_translation 获取锚点x/y坐标与本地x/y坐标之间的距离。 Yes

对于一个距离、焊接和轮子的接头，你可以使用以下常数（以及滑轮接头的常数）。

物理学联合常数
恒定 描述 只读
phy_joint_damping_ratio 阻尼比是非维度的，定义了关节的 "弹性"。这个常数的值通常在0和1之间，但可以更大，在1时，阻尼是关键的，意味着所有的振荡都应该消失。 No
phy_joint_frequency 这将返回（或设置）关节的振荡频率，单位为赫兹，通常该频率应小于时间步长的一半，由函数设置。 physics_world_update_speed(). No
phy_joint_length_1 这将返回从第一个局部X/Y坐标到第一个锚点X/Y坐标的关节长度（仅限距离关节，只能读取）。 Yes
phy_joint_length_2 这将返回从第二个局部X/Y坐标到第二个锚点X/Y坐标的关节长度（仅限距离关节，只能写入）。 No
