物理学_关节_删除

一旦两个具有物理表征的实例被一个联合体绑定，这可以在任何时候再次删除。通常情况下，当两个实例中的一个被摧毁时，或者当room 结束时，这将自动发生，但有时你可能希望手动完成。在这些情况下，你会使用这个函数。

还应该注意的是，每当作为齿轮接头一部分的实例被销毁时，齿轮接头应该在参与形成齿轮的任何实例之前使用这个函数被删除（但其余的接头将被自动删除），例如在实例的销毁事件中。

语法。

physics_joint_delete(joint)

争论 类型 描述
joint 物理学联合身份证 The index of the joint that you wish to delete

返回。

N/A

例子。

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
{
    reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
    if reactionForce > 2
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

上面的代码检查变量 "shipJoint "是否持有一个关节索引，如果是，它就用两个常数计算应用于该关节的力。最后，如果总的力大于2，则删除该关节。
