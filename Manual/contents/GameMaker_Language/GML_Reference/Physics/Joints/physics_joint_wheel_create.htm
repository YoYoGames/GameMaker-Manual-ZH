physics_joint_wheel_create

一个轮子关节简单地结合了一个活塞和一个渐开线关节，就像一个安装在汽车减震器上的轮子。你指定关节的锚点以及两个物理学启用的实例，第一个实例将作为关节的 "主体"，而第二个实例将是 "车轮"，并被允许围绕关节轴自由移动。你还可以设置一个轴向量，这是一条假想线，关节将沿着这条假想线充当 "弹簧"，允许车轮在遇到障碍物时沿着它上下弹跳（很像活塞式关节）。下面的图片说明了这一点。

如果你选择启用电机，那么第二个实例将围绕锚点位置旋转，你可以设置用于实现所需电机速度的最大电机扭矩（N/m），以及电机应该以何种速度转动。由于轮子关节也有一个轴，它可以沿着这个轴移动，你可以设置振荡频率（Hz）以及关节的阻尼比--你可能需要对这些数值进行微调，建议你从较小的数值开始，然后增加它们，直到得到你想要的效果。

和所有的关节一样，如果你把 "col "值设置为 true ，那么这两个实例可以相互作用和碰撞，但只有在它们有碰撞事件的情况下，然而如果它被设置为 false ，它们无论如何都不会发生碰撞。

语法。

physics_joint_wheel_create(inst1, inst2, anchor_x, anchor_y, axis_x, axis_y, enableMotor, max_motor_torque, motor_speed, freq_hz, damping_ratio, col)

争论 类型 描述
inst1 实例ID The first instance to connect with the joint
inst2 实例ID The second instance to connect with the joint
anchor_x 真实的 The x coordinate where the joint is anchored, within the game world
anchor_y 真实的 The y coordinate where the joint is anchored, within the game world
axis_x 真实的 The x component of the wheel axis vector
axis_y 真实的 The y component of the wheel axis vector
enableMotor 布尔型 Whether the motor should be active (true) or not (false)
max_motor_torque 真实的 Sets the maximum motor torque used to achieve the desired motor speed (in Newtons per meter)
motor_speed 真实的 This is the speed at which the motor should rotate
freq_hz 真实的 This is the oscillation frequency for the joint, in hertz
damping_ratio 真实的 This damping ratio for the joint
col 颜色 Whether the two instances can collide (true) or not (false)

返回。

Physics Joint ID

例子。

var mainFixture = physics_fixture_create();
physics_fixture_set_box_shape(mainFixture, 128, 32);
var wheelFixture = physics_fixture_create();
physics_fixture_set_circle_shape(wheelFixture, 32);
var o_id = instance_create_layer(x+25, y + 100, "Instances", obj_Wheel);
physics_fixture_bind(mainFixture, id);
physics_fixture_bind(wheelFixture, o_id);
physics_joint_wheel_create(id, o_id, o_id.x, o_id.y, 0, 1, true, 10, 1, 15, 6, false);
physics_fixture_delete(mainFixture);
physics_fixture_delete(wheelFixture);

上面的代码创建并定义了两个新的夹具。然后，这些夹具被绑定到调用的实例和一个新的实例上，这个新的实例在应用轮子连接它们之前被创建，之后，这些夹具从内存中被删除。
