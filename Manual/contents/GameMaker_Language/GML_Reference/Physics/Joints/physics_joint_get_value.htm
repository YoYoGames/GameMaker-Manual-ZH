物理学_联合体_获取值

通过使用一系列预定义的常量，你可以要求GameMaker告诉你关于任何给定关节状态的一些事情。这非常有用，因为它让你有能力删除关节或改变一个实例的行为，这取决于你当时的需要。有许多常量可以用于这个功能，它们可以在这里找到。物理学关节常量，但是要注意，当你调用这些常量时，会进行复杂的计算，所以应该谨慎使用，只有在必要时才使用，而且要注意，许多常量是特定类型的关节所特有的。

如果该属性不存在（例如，如果你检查一个滑轮接头的棱镜翻译），那么0将是返回值。

语法。

physics_joint_get_value(joint, value)

争论 类型 描述
joint 物理学联合身份证 The index of the joint that you wish to test
value 物理学联合常数 The constant for the joint property that you wish to test

返回。

Real

例子。

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if reactionForce > 2
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

上面的代码检查变量 "shipJoint "是否持有一个关节索引，如果是，它就用两个常数计算应用于该关节的力。最后，如果总的力大于2，则删除该关节。
