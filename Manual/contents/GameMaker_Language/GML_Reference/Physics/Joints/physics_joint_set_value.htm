physics_joint_set_value

某些关节属性甚至可以在创建关节后进行更改和设置。有一些常量可以在这个功能中使用，它们可以在这里找到。物理学关节常数。

语法。

physics_joint_set_value(joint, field, value)

争论 类型 描述
joint 物理学联合身份证 The index of the joint that you wish to change
field 物理学联合常数 The constant for the joint property that you wish to change
value 真实的 The new value for the joint property

返回。

N/A

例子。

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

上述代码检查关节的最大电机扭矩是否被设置为小于2，如果是，则将其设置为2。
