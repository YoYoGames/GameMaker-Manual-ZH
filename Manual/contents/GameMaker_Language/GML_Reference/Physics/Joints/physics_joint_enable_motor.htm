物理学_关节_启用电机

当你有一个带马达的关节（棱柱形或渐开线）时，你可能希望能够根据游戏中的变量和条件来开启或关闭马达switch 。switch 为此，你需要事先将关节的索引存储在一个变量中，然后你可以通过使用这个函数并将 "马达 "参数设置为 true 或 false ，来开启或关闭马达。

语法。

physics_joint_enable_motor(joint, motor)

争论 类型 描述
joint 物理学联合身份证 The joint that you wish to enable or disable the motor on
motor 布尔型 Whether you wish to turn the motor on (true) or off (false)

返回。

N/A

例子。

var t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, sprite_get_width(sprite_index) / 2);
var o_id=instance_create_layer(x+300, y, "Instances", obj_Door);
physics_fixture_bind(t_fix, id);
physics_fixture_bind(t_fix, o_id);
perma_joint = physics_joint_revolute_create(id, o_id, x+25, y, -90, 90, 1, 10, 2, 0, 0);
physics_joint_enable(perma_joint, 1);
physics_fixture_delete(t_fix);

上面的代码创建并定义了一个新的夹具，然后创建了一个 "obj_Door "的实例，将创建的夹具绑定到两个实例上。然后，它们被一个没有马达的渐开线连接起来，角度被限制在+/-90度的摆动，我们将连接的索引存储在变量 "perma_joint "中。然后我们使用这个变量switch ，最后从内存中删除夹具，因为它不再需要了。
