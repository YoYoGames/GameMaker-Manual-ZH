physics_apply_local_impulse

你不仅可以用GameMaker中的物理学将力和重力应用到object ，而且还可以应用冲力。这与力略有不同，因为当它被应用时，它将立即影响速度，以及可能影响object 的扭矩（或 "旋转"），特别是如果选择应用脉冲的点有一个与质心不一致的矢量（注意：质心不一定与原点相同！）。这个函数将一个脉冲局部应用到一个实例上。这意味着冲力的强度和方向是根据实例的原点（如果没有sprite ，则是位置）来计算的，而不考虑它在游戏中可能具有的方向或旋转room 或物理世界。下面是一个例子。

正如你所看到的，图像中的青蛙从下面得到一个脉冲，使它 "跳 "起来。确切的冲力是由我们从与xpos/ypos坐标有关的组件ximpulse/yimpulse得到的矢量定义的--这简单地意味着冲力被计算为从xpos/ypos到ximpulse/yimpulse的距离，单位是牛顿，而方向是我们从xpos/ypos到ximpulse/yimpulse的角度。

注意：物理世界的所有计算都使用米，每米的像素比例在room 编辑器中定义或使用 physics_world_create().

语法。

physics_apply_local_impulse(xpos, ypos, ximpulse, yimpulse)

争论类型 描述
xpos The x coordinate relative to the origin where the impulse will be applied
ypos The y coordinate relative to the origin where the impulse will be applied
ximpulse the x component of the impulse vector (relative to the xpos coordinate)
yimpulse the y component of the impulse vector (relative to the ypos coordinate)

返回。

例子。

if jump
{
    physics_apply_local_impulse(0, 20, 0, 50);
}

如果变量 "跳跃 "为真，上面的代码将应用一个冲力为30牛顿的脉冲，其方向是相对于实例原点从下往上。

返回。力量
