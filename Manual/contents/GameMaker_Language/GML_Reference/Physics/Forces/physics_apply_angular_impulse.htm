物理学_应用_angular_impulse

这个函数将给一个启用了物理学的实例提供一个角度脉冲。这个脉冲将按给定的数量设置角度旋转，忽略当前的扭矩，本质上是设置一个灯具的 "旋转 "量。如果你想用扭矩给一个实例施加一个角力，那么你应该使用函数 physics_apply_torque().

语法。

physics_apply_angular_impulse(impulse)

争论类型 描述
impulse The impulse (in Newton metres) to be applied

返回。

例子。

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

上面的代码将在按下一个键时给物理学上的灯具一个角度脉冲。

返回。力量
