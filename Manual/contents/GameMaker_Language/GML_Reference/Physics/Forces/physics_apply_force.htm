物理学_应用_力

当使用物理学时，你不仅需要将重力应用于objects ，而且还需要将力应用于。力是任何导致object 发生速度变化、方向变化或形状变化的影响，在GameMaker物理世界中的定义与我们定义重力一样，通过从两个x力/y力分量中获得矢量。应该注意的是，力不是施加在object 的质量中心，而是施加在room 的某个点上，它们将改变方向和速度以及扭矩（旋转），这取决于作用在object 的任何其他力（如重力）。下面是一个帮助说明的例子图片。

在上面的例子中，底部的弹簧object ，在球上产生了一个力object ，使它向不同的方向移动。确切的力是由我们从与xpos/ypos坐标相关的组件xforce/yforce得到的矢量定义的--这简单地意味着力被计算为从xpos/ypos到xforce/yforce的距离，单位为牛顿（牛顿被定义为以每秒一米的平方速度加速一公斤的质量所需的净力），方向是我们从xpos/ypos到xforce/yforce的角度。像上面这样的碰撞的代码将在弹簧的碰撞事件中进行object ，看起来就像下面的例子中提供的那样。

注意：物理世界的所有计算都使用米，每米的像素比例在room 编辑器中定义或使用 physics_world_create().

语法。

physics_apply_force(xpos, ypos, xforce, yforce)

争论 类型 描述
xpos 真实的 The x coordinate in the room where the force will be applied
ypos 真实的 The y coordinate in the room where the force will be applied
xforce 真实的 The x component of the force vector
yforce 真实的 The y component of the force vector

返回。

N/A

例子。

with (other)
{
    physics_apply_force(x, y, 0, -30);
}

上面的代码将对碰撞中的其他 object ，力为30牛顿，方向为30牛顿。

返回。力量
