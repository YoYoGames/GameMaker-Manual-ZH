物理学_应用_局部力

在物理世界中使用力的另一种方式是将它局部应用于一个实例。这意味着力的强度和方向是根据实例的原点（如果没有sprite ，则是位置）计算的，而不考虑它在游戏room 或物理世界中可能具有的方向或旋转。应该注意的是，在这个函数中，力不是施加在object 的质心上，而是施加在相对于实例的某一点上，而且它们不会立即被施加，因为它们取决于作用在object 上的任何其他力（如重力）。这个插图展示了局部力是如何工作的。

room 正如你所看到的，xlocal和ylocal坐标的计算是基于轴的位置总是被认为是实例的原点（0,0），所以无论object ，xlocal和ylocal总是要一样的。然后你必须决定力的矢量分量（xforce和yforce）相对于xlocal/ylocal的位置应该在哪里。确切的力是由我们从xforce/yforce分量中得到的与这些坐标有关的矢量定义的--这简单地意味着，力被计算为从xlocal/ylocal到xforce/yforce的距离，单位是牛顿，而方向是我们从xlocal/ylocal到xforce/yforce的角度。这似乎有点令人困惑，但你只需看一下插图就知道这是怎么一回事了，它为你省去了一些复杂的数学运算!

注意：物理世界的所有计算都使用米，每米的像素比例在room 编辑器中定义或使用 physics_world_create().

语法。

physics_apply_local_force(xlocal, ylocal, xforce, yforce)

争论类型 描述
xlocal The x coordinate relative to the origin where the force will be applied
ylocal The y coordinate relative to the origin where the force will be applied
xforce the x component of the force vector
yforce the y component of the force vector

返回。

例子。

if keyboard_check(vk_right)
{
    physics_apply_local_force(-25, 0, -50, 0);
}

上面的代码将在按下右方向键时，对实例施加一个相对于原点从左到右的50牛顿的力。

返回。力量
