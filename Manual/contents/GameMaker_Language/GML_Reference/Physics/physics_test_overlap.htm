物理学_测试_重叠

这个函数可以用来检查和查看一个物理体（即：一个实例的夹具）在旋转和放置在room 。"角度 "参数是调用的实例在要检查的位置具有（或将具有）的旋转角度，而 "obj "参数可以是单个实例的id，和object 索引或 关键字全部或其他。

语法。

physics_test_overlap(xpos, ypos, angle, obj);

争论 类型 描述
xpos 真实的 The x position in the room to check
ypos 真实的 The y position in the room to check
angle 真实的 The angle to check (of the calling instance)
obj 对象资产或实例ID The object to check for

返回。

Boolean

例子。

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

上面的代码将检查一个物理夹具重叠的位置，如果有的话，它会设置一个变量和一个警报。

返回。物理学
