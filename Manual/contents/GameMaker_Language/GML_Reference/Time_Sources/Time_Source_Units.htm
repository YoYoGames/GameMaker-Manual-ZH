时间来源单位

时间来源单位常数
恒定 描述 价值
time_source_units_seconds 使用秒作为时间源周期（与帧无关）。 0
time_source_units_frames 使用时间源周期的帧（取决于帧）。 1

这些常数被用于 time_source_create()和 time_source_reconfigure()中使用，以设置时间源的周期所使用的单位，并由 time_source_get_units().

如果你使用秒，你的时间源将独立于游戏的帧速率运行。

如果你使用帧，你的时间源将取决于游戏的帧速率。

你也可以使用BPM（每分钟节拍）作为单位，通过调用 time_bpm_to_seconds()将你的BPM值转换成以秒为单位的周期，然后使用time_source_units_seconds 作为单位。
