时间来源_停止

这个函数停止给定的时间源并重置其计时器。你不能停止一个没有启动或处于完成状态的时间源。

语法。

time_source_stop(id);

争论 类型 描述
id 时间源ID The Time Source 阻止

返回。

N/A

例子。

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

上面的代码将在一个时间源还没有停止的时候停止它。

返回。时间来源
