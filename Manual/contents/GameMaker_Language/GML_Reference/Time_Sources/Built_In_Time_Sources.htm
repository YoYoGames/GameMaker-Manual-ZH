内置的时间源

时间源常数
恒定 描述 价值
time_source_global 全球时间源 0
time_source_game 游戏时间来源 1

你可以使用全球时间源或游戏时间源作为你的自定义时间源的父源。这两种时间源都是全球可用的。

全局时间源在主游戏之外运行loop ，而游戏时间源作为游戏的一部分运行loop 。

全局/游戏之间的选择并不影响时间源是否受到游戏帧率的影响；这将取决于你使用的单元。

秩序

继承自全局时间源的时间源 在游戏时间源 之前被处理。

国家

游戏时间源有 状态，意味着它可以 paused.全局时间源没有状态，不能暂停。

你可以将全局时间源用于无论游戏状态如何都应该运行的计时器，而将游戏时间源用于与你的主要游戏相联系的计时器loop 。然后你可以通过暂停游戏时间源来暂停所有与游戏相关的计时器。
