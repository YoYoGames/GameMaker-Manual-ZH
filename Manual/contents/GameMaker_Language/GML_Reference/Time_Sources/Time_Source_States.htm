时间源状态常数
恒定 描述 价值
time_source_state_initial 时间源还没有开始 0
time_source_state_active 时间源已经启动，正在倒计时。 1
time_source_state_paused 时间来源是 paused 2
time_source_state_stopped 时间源被停止了，或者它完全过期了 3

一个时间源可以有一个状态，它可以是上面的任何一个常数。新创建的时间源的默认状态是time_source_state_initial。

可以用以下方法改变状态 time_source_start(), time_source_stop()，和 time_source_pause().

时间源的状态是用 time_source_get_state().
