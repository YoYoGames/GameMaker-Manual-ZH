时间源到期类型

时间源到期常数
恒定 描述 价值
time_source_expire_nearest 时间源将在离其过期时间最近的帧上过期。 0
time_source_expire_after 时间源将在其过期时间后的第一帧过期。 1

以秒为单位的时间源可能不总是在游戏帧上过期。它们可能在一个游戏帧被处理后，但在一个新的游戏帧开始处理前过期。

然而，时间源的回调只能在游戏帧上运行，所以时间源的过期类型控制其回调的运行时间，以防它在两个帧的中间过期。

当使用time_source_expire_nearest时，时间源的回调将在离其到期时间最近的帧上运行，这可能是它之前的帧，也可能是它之后的帧。

当使用time_source_expire_after时，时间源的回调将总是在其到期时间之后的帧上运行。
