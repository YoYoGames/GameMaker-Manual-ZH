时间来源_get_parent

这个函数返回给定时间源的 时间源ID时间源的父源的ID。

返回值可能是自定义时间源的ID，也可能是内置时间源之一，取决于给定时间源的创建方式。

语法。

time_source_get_parent(id);

争论 类型 描述
id 时间源ID The Time Source 的父类，以获得

返回。

Time Source ID

例子。

var _parent = time_source_get_parent(time_source);

这段代码获得了一个时间源的父级，并将其存储在一个局部变量中。

返回。时间来源
