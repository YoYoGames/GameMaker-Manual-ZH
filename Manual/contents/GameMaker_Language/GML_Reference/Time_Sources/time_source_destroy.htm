时间源的销毁

这个函数将销毁给定的时间源。

你可以选择指定是否所有子时间源也要被删除。

语法。

time_source_destroy(id, [destroy_tree]);

争论 类型 描述
id 时间源ID The Time Source 摧毁
destroy_tree 布尔型 OPTIONAL 是否也要销毁子时间源，默认为false

返回。

N/A

例子。

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

创建事件代码创建了一个新的时间源，每秒过期一次，无限期地重复。

Clean Up事件代码销毁了该时间源，因为在实例被销毁后就不需要它了。
