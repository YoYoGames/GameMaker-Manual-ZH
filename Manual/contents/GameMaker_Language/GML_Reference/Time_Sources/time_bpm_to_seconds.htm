时间_bpm_to_seconds

这个函数接受一个每分钟的节拍值，并返回每个节拍的长度（秒）。

这可以在创建时间源时使用，以便为时间源使用一个BPM值。重要的是，这样的时间源使用秒作为它的单位。

语法。

time_bpm_to_seconds(bpm);

争论 类型 描述
bpm 真实的 The beats-per-minute value to convert into seconds-per-beat

返回。

Real

例子。

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

这段代码将90BPM的数值转换成秒数，并使用它来创建一个无限期运行的时间源。

在每个节拍中，它都会在输出日志中打印出"BEAT!"的信息。

返回。时间来源
