Time_source_exists

这个函数返回给定的ID是否是一个有效的时间源。

如果时间源被摧毁，这将返回false，然而如果时间源被停止或者它只是过期了，它将继续存在，这意味着这个函数返回true。

语法。

time_source_exists(id);

争论 类型 描述
id 时间源ID The Time Source 来检查

返回。

Boolean

例子。

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

这段代码检查一个时间源是否存在。如果存在，它将检查是否按下了空格键。

在这种情况下，它破坏了时间源。

返回。时间来源
