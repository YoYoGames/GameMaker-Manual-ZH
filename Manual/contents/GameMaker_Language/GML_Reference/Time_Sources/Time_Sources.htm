时间来源

一个"时间源"是你创建的一个自定义计时器。它在给定的时间段内运行，并在结束时过期。

一个时间源（TS）有能力在过期时调用一个自定义方法（"回调"）。

它也能重复一定数量的次数，或永远重复。一个时间源的每一次重复都被称为 "代表"。

如何使用时间源？

创建并启动一个调用自定义方法的时间源。

  • 调用 time_source_create()/创建时间源来创建一个时间源
    • 指定这个inherits ，是来自全局时间源、游戏时间源还是自定义时间源。
    • 指定它运行的时间段，以及使用的单位（秒或帧）。
    • 指定一个自定义的方法或script ，并将参数数组传递给该方法。
    • 可以选择，指定它应该重复的次数
      • 默认情况下，它只运行一次
  • 调用 time_source_start()/启动时间源以立即启动该时间源，或在以后需要启动时调用它

这样做将导致时间源的运行，在指定的时间段后，它将过期并调用你指定的方法。

内置的时间源

一个时间源可以从两个内置的时间源中的任何一个inherit ，也可以从一个自定义的时间源。
