时间来源_复位

这个函数重置了给定的时间源，重置了它的定时器和重复信息。

调用这个函数后，你需要再次启动 时间源，让它进行倒计时。

语法。

time_source_reset(id);

争论 类型 描述
id 时间源ID The Time Source 重置

返回。

N/A

例子。

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

每当一个新的room 开始时，这段代码将重置给定的时间源，然后再次启动时间源。

返回。时间来源
