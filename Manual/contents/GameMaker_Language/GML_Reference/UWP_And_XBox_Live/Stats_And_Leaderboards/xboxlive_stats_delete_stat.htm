xboxlive_stats_delete_stat

这个函数可以用来从统计管理器中为给定的用户ID删除一个统计。你提供由（例如）函数返回的用户 ID，然后是要删除的统计信息 。 xboxlive_get_user()所返回的用户 ID，然后提供要删除的统计信息string 。这将清除状态值并将其从状态管理器中删除，这意味着它将不再被函数 xboxlive_stats_get_stat_names()或 xboxlive_stats_get_stat().

如果出现错误或用户ID无效，该函数将返回-1，如果函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_delete_stat(user_id, stat);

争论类型 描述
user_id The user ID pointer to delete the stat of
stat The statistic to delete (a string)

返回。

例子。

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

上面的代码loops ，通过连接的用户，从每个用户的统计管理器中删除指定的统计。

返回。统计资料和排行榜
