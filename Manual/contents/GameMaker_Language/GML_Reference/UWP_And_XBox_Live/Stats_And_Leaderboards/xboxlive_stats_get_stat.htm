统计数据

这个函数可以用来从统计管理器中为一个给定的用户检索一个单一的统计值。你提供由（例如）函数返回的用户ID xboxlive_get_user()所返回的用户 ID，然后提供你在使用 xboxlive_stats_set_stat_* 函数之一创建它时定义的 statstring 。返回值可以是一个string ，也可以是一个实数（取决于正在检查的状态），如果状态不存在或出现错误，则返回常数GML undefined 。

语法。

xboxlive_stats_get_stat(user_id, stat);

争论类型 描述
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

返回。

Real, String or undefined

例子。

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

上面的代码检查一个变量，如果它是 true ，那么它将检查 "PercentDone "状态的值。如果这个值小于100，那么就会产生一个值，并将状态设置为这个值。

返回。统计资料和排行榜
