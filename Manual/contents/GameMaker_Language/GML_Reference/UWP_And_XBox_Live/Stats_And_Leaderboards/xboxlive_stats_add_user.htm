xboxlive_stats_add_user

这个函数可以用来将一个给定的用户ID指针添加到统计管理器中。这必须在使用任何其他统计函数之前完成，以便自动与XBox Liveserver 同步游戏并检索最新的数值。你提供的用户ID是由（例如）函数返回的 xboxlive_get_user()，如果出现错误或用户ID无效，该函数将返回-1，如果函数被成功调用，则返回任何其他值。

该函数将产生一个回调，它将触发一个系统异步事件。这个事件将有一个特殊的DS地图 async_load然后应该对以下键进行解析。

  • "id" - 将保持常数 achievement_stat_event
  • "eventname" - 将持有string "LocalUserAdded"
  • "userid" - 与请求相关的用户ID。
  • "error" - 如果成功则为0，如果出现错误则为其他值
  • "errormessage" - 一个带有错误信息的string ，如果有的话。

语法。

xboxlive_stats_add_user(user_id);

争论类型 描述
user_id The user ID (a pointer) to add

返回。

例子。

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
}

上述代码将获得可用的用户账户数量，然后loop ，并将账户ID分配给一个数组，同时在统计管理器中注册用户。

返回。统计资料和排行榜
