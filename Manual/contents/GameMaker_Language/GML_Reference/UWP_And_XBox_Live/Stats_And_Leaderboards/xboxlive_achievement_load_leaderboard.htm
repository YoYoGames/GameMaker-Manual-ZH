xboxlive_achievement_load_leaderboard

这个函数将向server 发送请求，以获取给定排行榜的信息。该排行榜必须是之前在你的游戏的开发者仪表板上创建的，并且给定的ID是你必须提供的，作为函数的第一个参数。它将触发一个回调的社会异步事件，其中包括 async_load地图填充了相关的键/值对。这个 DS Map 的id键用于识别正确的回调（对于任何给定的异步事件，可以有一个以上的触发函数），并将与常数 xboxlive_achievement_leaderboard_info ，以及每个玩家的一些其他键/值对配对。该地图的具体内容如下所示。

  • "id" - 对于这个函数，它应该是 xboxlive_achievement_leaderboard_info
  • "leaderboardid" - 在提供者仪表板上定义的排行榜的唯一ID。
  • "numentries" - 你在排行榜上获得的条目数。
  • "PlayerN" - 玩家的名字，其中 "N "是一个整数值，对应于他们在排行榜条目列表中的位置。
  • "PlayeridN" - 玩家的唯一用户ID，"N"。
  • "RankN" - 玩家 "N "在排行榜上的排名。
  • "ScoreN" - 棋手 "N "的得分。

你可以通过给出一个最小和最大的指数值来指定要显示的排行榜的确切部分，还可以要求对结果进行过滤，只检索你的朋友的分数或使用以下过滤常数的所有玩家发布的分数。

扩展实例。

以下代码可能会在玩家登录他们的游戏账户后被调用，以获得所有朋友在游戏中的排行榜位置列表。

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

这将发送一个关于给定排行榜信息的请求，并生成一个异步回调，其中包含特殊的 async_load DS地图，包含以下数据。

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_leaderboard_info
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

上述代码检查社交异步事件中返回的 DS Map ，如果它的 "id "与被检查的常数相匹配，它就会检查以确保被发送的排行榜与最初的函数调用相匹配，然后循环浏览地图并将所有不同的值存储在一些数组中。
