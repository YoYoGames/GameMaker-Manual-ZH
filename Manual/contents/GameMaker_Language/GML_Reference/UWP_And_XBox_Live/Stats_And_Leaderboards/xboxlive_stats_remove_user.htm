xboxlive_stats_remove_user

这个函数可以用来从统计管理器中删除（unregister）一个给定的用户，执行统计数据的刷新到实时server 。根据XBox的文档，游戏不需要将用户从统计管理器中删除，因为XBox操作系统会定期刷新统计资料。你提供由（例如）函数返回的用户ID。 xboxlive_get_user()，如果出现错误或用户ID无效，该函数将返回-1，如果函数被成功调用，则返回任何其他值。

注意：如果在该用户上设置的统计信息无效（比如统计信息名称包含非阿尔法数字字符），那么删除该用户会返回一个错误。

该函数将产生一个回调，它将触发一个系统异步事件。这个事件将有一个特殊的DS地图 async_load然后应该对以下键进行解析。

  • "id" - 将保持常数 achievement_stat_event
  • "eventname" - 将持有string "LocalUserRemoved"
  • "userid" - 与请求相关的用户ID。
  • "error" - 如果成功则为0，如果出现错误则为其他值
  • "errormessage" - 一个带有错误信息的string ，如果有的话。

注意，如果你想在任何时候冲刷统计数据到实时server ，而不删除用户，你可以使用函数 xboxlive_stats_flush_user().

语法。

xboxlive_stats_remove_user(user_id);

争论类型 描述
user_id The user ID (a pointer) to remove

返回。

例子。

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

上述代码loop ，通过存储的用户账户ID，从统计管理器中删除（取消注册）每个用户。

返回。统计资料和排行榜
