xboxlive_stats_set_stat_int

这个函数可以用来为给定的用户ID设置一个状态的值。你提供由（例如）函数返回的用户ID xboxlive_get_user()返回的用户ID，然后提供要设置的状态字符串 来设置（如果statstring 不存在，那么将创建一个新的stat并设置为给定的值）和一个值（一个整数）来设置该stat。注意，状态名称必须是字母数字，不能有符号或空格。

当设置状态值时，任何以前的值都会被覆盖，因此，在设置之前，你要通过比较当前的状态值和新的状态值来确定状态值是否应该被更新（即检查高分是否真正是最高分）。 新的状态值，然后再设置它。

如果出现错误或用户ID无效，该函数将返回-1，如果该函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_set_stat_int(user_id, stat, value);

争论类型 描述
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (an integer)

返回。

例子。

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

上面的代码检索了状态 "GamesPlayed "的值，然后将同一状态设置为该值加1。

返回。统计资料和排行榜
