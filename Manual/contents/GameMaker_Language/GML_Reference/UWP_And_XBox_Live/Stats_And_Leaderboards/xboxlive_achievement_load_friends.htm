胜利后的朋友们

这个函数将向server ，以获取所有登录用户朋友的信息，并将触发一个回调社会异步事件，其中包含了 async_load地图填充了相关的键/值对。这个 DS Map 的id键用于识别正确的回调（对于任何给定的异步事件，可以有一个以上的触发函数），并将与常数 xboxlive_achievement_friends_info 以及每个朋友的其他一些键/值对配对。该地图的具体内容如下所示。

  • "id" - 对于这个函数，它应该是 xboxlive_achievement_friends_info
  • "FriendN" - 朋友的名字，其中 "N "是一个整数值，对应于他们在朋友列表中的位置。
  • "FriendidN" - 朋友的唯一用户ID，"N"。

语法。

xboxlive_achievement_load_friends();

返回。

扩展实例。

以下代码可能会在玩家登录他们的游戏账户后被调用，以获得所有用户的朋友列表。

xboxlive_achievement_load_friends();

这将发送一个关于用户朋友信息的请求，并生成一个异步回调，其中包含特殊的 async_load DS地图，包含以下数据。

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_friends_info
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

上面的代码检查社交异步事件中返回的 DS Map ，如果它的 "id "与被检查的常量相匹配，它就会通过地图loops ，将所有不同的值存储在一些数组中。
