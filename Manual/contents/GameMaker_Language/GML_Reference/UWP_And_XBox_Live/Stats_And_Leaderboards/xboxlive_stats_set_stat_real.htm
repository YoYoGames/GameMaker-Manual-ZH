xboxlive_stats_set_stat_real

这个函数可以用来为给定的用户ID设置一个状态的值。你提供由（例如）函数返回的用户ID，然后提供要设置的状态。 xboxlive_get_user()返回的用户ID，然后提供要设置的状态string （如果状态string 不存在，那么将创建一个新的状态并设置为给定的值）和一个要设置状态的值（一个实数）。注意，状态名称必须是字母数字，不能有符号或空格。

当设置状态值时，任何以前的值都会被覆盖，因此，在设置状态值之前，你要通过与当前的状态值和新的状态值进行比较来确定是否应该更新状态值（即检查高分是否真的是最高分）。

如果出现错误或用户ID无效，该函数将返回-1，如果该函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_set_stat_real(user_id, stat, value);

争论类型 描述
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

返回。

例子。

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

上面的代码检查一个变量，如果它是 true ，那么它将检查 "PercentDone "状态的值。如果这个值小于100，那么就会产生一个值，并将状态设置为这个值。
