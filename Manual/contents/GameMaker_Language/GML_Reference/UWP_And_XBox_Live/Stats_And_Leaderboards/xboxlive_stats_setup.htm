xboxlive_stats_setup

在你使用其他Xbox Live统计函数之前，需要调用这个函数，它只是初始化了系统中所需要的库。参数 "user_id "是由函数返回的原始用户ID。 xboxlive_get_user()，而 "service_config "和 "title_id "是你的游戏在Xbox Live开发中心的唯一ID。

这个功能只适用于基于事件的（2013）统计。

语法。

xboxlive_stats_setup(user_id, service_config_id, title_id);

争论 类型 描述
user_id Xbox用户ID The ID pointer of the user to check.
service_config_id 字符串 This is the config_id string for the game.
title_id 字符串 The unique ID for your game on the Xbox Dev Center.

返回。

N/A

例子。

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

上面的代码为给定的用户ID初始化统计系统。
