xboxlive_achievement_show_achievements

这个函数将打开所选平台的成就页面。请注意，这是一个异步功能，即：当成就页面被显示时，你的游戏将继续在后台运行。因此，你应该小心使用这个功能，并确保暂停游戏，或者只允许在游戏中不干扰游戏的地方显示这个功能。

语法。

xboxlive_achievement_show_achievements()

返回。

例子。

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

上面的代码检查一个自定义变量，看成就是否被启用，如果被启用，它就设置一个全局变量，并在调用函数在屏幕上显示成就之前创建一个控制器 "暂停 "实例。
