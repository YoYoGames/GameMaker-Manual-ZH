xboxlive_stats_set_stat_string

这个函数可以用来为给定的用户ID设置一个状态到string 。你提供由（例如）函数返回的用户ID，然后提供要设置的状态。 xboxlive_get_user()，然后提供要设置的状态string （如果状态string 不存在，那么将创建一个新的状态并设置为给定的值），然后提供要设置状态的string 。注意，状态名称必须是字母数字，没有符号或空格，但你要设置的string 值可以有空格和特殊字符。

当设置状态string ，任何以前的string 将被覆盖，因此由你来决定是否应该更新状态string ，在设置之前将当前的状态string 与新的状态进行比较。

如果出现错误或用户ID无效，该函数将返回-1，如果该函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_set_stat_string(user_id, stat, value);

争论 类型 描述
user_id Xbox用户ID The user ID pointer to set the stat for
stat 字符串 The statistic to set (a string)
value 字符串 The value to set the stat to (a string)

返回。

Real

例子。

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

上面的代码将状态 "LevelName "设置为存储在给定全局变量中的string 。
