uwp_was_closed_by_user

通过这个函数，你可以发现游戏是否在玩的时候被用户关闭了（也就是说，它被正确地终止了，而不是被系统终止）。如果是的话，该函数将返回 true ，否则将返回 false 。

语法。

uwp_was_closed_by_user();

返回。

例子。

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

上面的代码检查应用程序是否以前被用户终止过，如果是，它将设置一个全局变量。
