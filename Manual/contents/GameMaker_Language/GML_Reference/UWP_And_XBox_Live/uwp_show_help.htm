uwp_show_help

这个函数启动了Xbox One帮助应用程序，它将反过来显示你的游戏帮助文件（这是一个HTML格式的文件，你之前已经上传至XBox开发者门户网站）。请注意，在调用这个函数时要指定一个用户ID，以便在显示时可以使用用户的地域信息。你可以用以下函数检索用户ID xboxlive_get_user(), xboxlive_get_activating_user()，或 uwp_license_trial_user().

语法。

uwp_show_help(user_id);

争论类型 描述
user_id The User ID pointer to open the helpfile for.

返回。

例子。

if gamepad_button_check_pressed(0, gp_start)
{
    if (uwp_license_trial_version())
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

上面的代码检查是否有按钮被按下，然后检查应用程序，看它是否在使用试用许可证运行。如果是，那么它就会获得许可的用户ID，并为该用户打开帮助文件。
