xboxlive_set_savedata_user

这个函数指定未来在游戏保存区操作的文件操作（即所有的文件写入，以及从不在捆绑区的文件中读取）将与指定的用户ID指针相关。这可以根据需要经常调用，将保存数据重定向到适当的用户，或者你可以使用常量 pointer_null ，保存到通用的机器存储区。

语法。

xboxlive_set_savedata_user(user_id);

争论 类型 描述
user_id Xbox用户ID或pointer_null The user ID (a pointer) to set for saving, or pointer_null

返回。

N/A

例子。

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

上面的代码检查一个用户当前是否被指定为保存目标，如果不是则被指定。
