xboxlive_get_savedata_user

该函数返回当前与文件保存相关的用户ID指针（如果当前没有使用用户ID，则返回常数 pointer_null ）。详情见 xboxlive_set_savedata_user()以了解更多细节。

语法。

xboxlive_get_savedata_user();

返回。

Xbox User ID or pointer_null

例子。

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

上面的代码检查一个用户当前是否被指定为保存目标，如果不是则被指定。
