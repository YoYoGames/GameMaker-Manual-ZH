uwp_was_terminated

通过这个函数，你可以发现游戏之前是否被系统关闭过（也就是说，这不是游戏的第一次运行）。如果是的话，该函数将返回 true ，否则将返回 false 。一个使用的例子是检查它，看游戏是否以前被系统终止过，然后（如果它返回 true ）加载玩家在游戏中的前一个阶段。如果针对XBox One ，那么如果检测到这一点，游戏所需的行为在不同版本的认证文档中会有所不同，所以值得检查最新的版本，看看游戏应该怎么做。

语法。

uwp_was_terminated();

返回。

例子。

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

上面的代码检查应用程序之前是否被系统终止，如果是，它就加载一个buffer ，并设置一个全局变量。
