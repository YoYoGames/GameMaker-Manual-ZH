uwp_is_suspending

这个函数将在单帧时返回 true （否则返回 false ），其方式类似于 os_is_paused().然后，游戏在被系统终止前有一秒钟的时间来保存游戏状态。一旦游戏完成了数据保存，它应该调用 uwp_suspend()来干净利落地暂停游戏。游戏可能也应该进入暂停状态，这样如果系统恢复它（如果它还没有完全终止），玩家就不会被直接扔回游戏环境中。

语法。

uwp_is_suspending();

返回。

例子。

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

上面的代码检查应用程序是否进入暂停状态，如果是，则调用script ，在暂停游戏之前保存游戏数据。
