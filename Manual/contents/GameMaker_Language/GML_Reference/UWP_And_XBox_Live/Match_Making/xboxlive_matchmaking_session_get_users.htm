xboxlive_matchmaking_session_get_users

这个函数将创建并填充一个DS地图，其中包含指定会话中用户的详细信息，如果有错误，则为-1。你可以从创建或找到会话时在异步社交事件中返回的 async_loadDS地图中获得，该地图在你创建或找到一个会话时在异步社交事件中返回（见 xboxlive_matchmaking_create()了解更多细节）。返回的DS地图将有以下键/值对。

  • "num_results" - 包含会话中的用户数
  • "userId<index> " - 包含用户的ID（<index>是一个从0到 "num_results"-1的值）。
  • "userIsOwner<index>" - 如果用户是主机，则包含1，否则包含0（<index>是一个从0到 "num_results"-1的值）。

语法。

xboxlive_matchmaking_session_get_users(session_id);

争论类型 描述
session_id The session ID pointer to use

返回。

Async Request ID

例子。

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
{
    for (var i = 0; i < session_map[? "num_results"]; i++)
    {
        if (session_map[? "userIsOwner" + string(i)] == 1)
        {
            global.OwnerID = session_map[? "userId" + string(i)];
        }
    }
}

以上将检索所有参与匹配会话的用户数据，并将一个全局变量设置为会话所有者的ID。
