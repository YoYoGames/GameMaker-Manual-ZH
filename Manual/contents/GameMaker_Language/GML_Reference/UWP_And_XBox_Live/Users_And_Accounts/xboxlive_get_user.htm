xboxlive_get_user

通过这个函数，你可以检索到被索引用户的用户ID指针。如果该用户不存在，该函数将返回常数 pointer_null 。你可以用该函数找到当前登录的用户数 xboxlive_get_user_count().

重要提示：该功能只能与 xboxlive_get_user_count() ，不要与游戏手柄一起使用。请使用 xboxlive_user_for_pad() 。

语法。

xboxlive_get_user(index);

争论 类型 描述
index 真实的 The index (an integer) to get the User ID from.

返回。

Xbox User ID or pointer_null

例子。

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

上述loops ，通过所有已签入的用户，并将其唯一的ID指针存储在一个数组中。

返回。用户和账户
