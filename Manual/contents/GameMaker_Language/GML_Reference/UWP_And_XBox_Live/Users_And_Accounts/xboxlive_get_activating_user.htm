xboxlive_get_activating_user

通过这个函数，你可以检索到从控制台启动游戏的用户的用户ID指针。

注意：你如何使用这个指针将取决于你在游戏中想要做什么，但你不应该依赖它作为当前正在玩的用户ID，因为激活的用户可以在游戏运行时退出。

语法。

xboxlive_get_activating_user();

返回。

Xbox User ID

例子。

global.activUser = xboxlive_get_activating_user();

上面将激活的用户ID指针存储在一个全局变量中

返回。用户和账户
