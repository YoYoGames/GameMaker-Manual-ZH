盒子里的小键盘

Xbox用户可以为他们的用户ID分配各种垫子，每一个垫子都充满了一个 "槽"。这些槽从0开始编号，可以容纳任何垫子的索引，所以槽0可以分配给垫子3，槽1可以分配给垫子2。这个函数可以用来获得分配给任何给定槽的垫子，如果用户在一定时间内没有输入游戏板，这个函数就特别有用，因为它将switch ，所以你可以使用这个函数检查任何给定的用户ID，以确保他们有一个活动的游戏板，如果他们没有，就通过（例如）暂停你的游戏做出相应的反应。如果游戏板处于活动状态，它的索引值将被返回，但如果不是，该函数将返回-1。

注意，如果任何槽中的垫子变得不活跃，后面的槽将 "滑入 "空缺的位置，所以如果你有三个活跃的垫子分配给槽0、1和2，而槽0中的垫子不活跃，槽1和to现在将变成槽0和1，所以槽0应该总是包含一个活跃的垫子，除非所有的垫子都关闭（在这种情况下，它将返回-1）。

语法。

xboxlive_pad_for_user(user_id, slot);

争论 类型 描述
user_id Xbox用户ID The User ID (a pointer) to check
slot 真实的 The index (an integer) of the gamepad slot to check

返回。

Xbox Pad ID

例子。

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
    for(var i = 0; i < gamepad_get_device_count(); i++;)
    {
        var u_id = xboxlive_user_for_pad(i);
        if (u_id == -1)
        {
            async_ID = xboxlive_pad_for_user(i, u_id);
        }
    }
}

上面的代码检查给定用户ID指针的0号槽，看是否有游戏板被分配，如果没有则分配一个。

返回。用户和账户
