xboxlive_set_rich_presence

这个函数将为给定的用户设置丰富的存在string 。Rich Presencestring 在用户正在玩的游戏名称后面显示用户的游戏活动，用连字符分开。这个string ，在 "朋友和俱乐部 "列表中显示在玩家的游戏名下，也显示在玩家的Xbox Live用户资料中。

当使用这个函数时，你需要为用户提供用户ID指针，然后你可以flag ，作为当前游戏中的活动或不活动（使用 true/false ）。如果设置为 false ，那么丰富的存在string 将被附加上"/afk "或其他适当的东西。下一个参数是要显示的丰富的存在string ID，最后你可以（选择性地）提供一个 service_config_id string 。这个参数是可选的，当没有提供时，它使用Xbox游戏选项中指定的SCID。

语法。

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

争论 类型 描述
user_id Xbox用户ID The ID pointer of the user to check.
is_user_active 布尔型 Flag the user as active or not.
rich_presence_string 字符串 The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id 字符串 OPTIONAL 这是游戏的config_id 。string

返回。

N/A

例子。

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

上面的代码得到分配给游戏板[0]槽的用户的用户ID指针，然后为该用户设置丰富的存在string 。

返回。用户和账户
