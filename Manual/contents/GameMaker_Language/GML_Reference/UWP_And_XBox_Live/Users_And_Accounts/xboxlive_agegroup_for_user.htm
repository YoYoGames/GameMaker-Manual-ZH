xboxlive_agegroup_for_user

这个函数将返回四个常数之一（如下所示），以表示分配给指定用户ID指针的年龄范围。

语法。

xboxlive_agegroup_for_user(user_id);

争论 类型 描述
user_id Xbox用户ID The user ID (a pointer) to check

返回。

Xbox Age Group Constant

Xbox年龄组常数
恒定 描述
xboxlive_agegroup_unknown 用户的年龄组不详
xboxlive_agegroup_child 用户年龄段在8至12岁之间（含）。
xboxlive_agegroup_teen 用户年龄段为13至17岁（含），在韩国则为13至19岁。
xboxlive_agegroup_adult 用户年龄段为18岁或以上（或在韩国为20岁以上）。

例子。

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
{
    global.ContentControl = true;
}
else global.ContentControl = false;

上述代码检查激活用户的年龄组，并根据返回的常数将一个全局变量设置为 true 或 false 。

返回。用户和账户
