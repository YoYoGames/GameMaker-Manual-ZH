xboxlive_show_account_picker

该功能将打开XBox账户选择器屏幕，如果用户还没有登录，他们可以登录（如果他们已经登录，那么该功能将不做任何事情）。请注意，这个功能只适用于在PC上使用UWP构建，而不是Xbox，因为要在Xbox上启动游戏，用户必须已经登录了。

语法。

xboxlive_show_account_picker();

返回。

例子。

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

上面的代码检查用户是否已经登录XBox Live，如果他们已经登录，代码将获得用户的游戏账号，并将其存储在一个全局变量中供将来使用。如果他们没有登录，那么代码会检查他们是否正在登录，在这种情况下不会发生进一步的事情，如果他们没有登录，那么它将打开账户选择器，提示他们登录。

返回。用户和账户
