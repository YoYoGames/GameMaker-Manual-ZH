xboxlive_get_user_count

通过这个函数，你可以找到当前登录到XBox系统的用户总数。返回值将是一个整数，从0-N，其中N是登录用户的数量-1。

语法。

xboxlive_get_user_count();

返回。

例子。

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

上述loops ，通过所有已签入的用户，并将其唯一的ID指针存储在一个数组中。

返回。用户和账户
