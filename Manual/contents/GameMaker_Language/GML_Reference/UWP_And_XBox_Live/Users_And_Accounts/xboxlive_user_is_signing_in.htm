xboxlive_user_is_signing_in

该函数将检查一个用户是否在签到过程中，如果是这样，则返回 true ，否则返回 false 。

语法。

xboxlive_user_is_signing_in();

返回。

例子。

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

上面的代码检查用户是否登录了XBox Live，如果他们登录了，代码将获得用户的gamertag并存储在一个全局变量中供将来使用。如果他们没有登录，那么代码会检查他们是否正在登录，在这种情况下不会发生进一步的事情，如果他们没有登录，那么它将打开账户选择器，提示他们登录。

返回。用户和账户
