uwp_is_constrained

如果游戏在资源受限模式下运行，这个函数将返回 true ，如果正常运行，则返回 false 。当游戏没有焦点时，资源受限模式可以被激活，在这种模式下，游戏的GPU资源减少。对此的反应通常应该是暂停游戏（因为用户不再有控制权）。另外，如果在这种模式下观察到速度减慢，禁用任何特别费力的效果可能是一个好主意。请注意，与 uwp_is_suspending()函数不同，只要游戏处于受限模式，这个函数就会持续返回 true （而不仅仅是第一次发生时的一次）。

语法。

uwp_is_constrained();

返回。

例子。

if xboxliveis_constrained()
{
    if !instance_exists(obj_PauseMenu)
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

上面的代码检查应用程序是否受到约束，如果是，它就会创建一个object （用于暂停游戏）的实例。
