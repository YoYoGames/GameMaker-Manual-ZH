screen_save

通过这个功能，你可以保存游戏窗口中当前出现的游戏截图。

该文件将被保存到游戏的工作目录中，在windows下是 <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ ，在Mac下是 ~/Library/Application Support/[Game Name]/ 。捕获的图像将是游戏应用表面的最终渲染，或者（如果应用表面被禁用）它将是整个显示屏或窗口的图像。

值得注意的是，该函数被设计为在Draw GUI End Event中调用，这应该在所有平台上得到一致的结果。在任何其他事件中使用它可能不会像预期的那样工作，并且可能在不同的目标甚至设备上得到不同的结果。

如果你需要对被保存的内容进行更多的控制，你应该使用函数 screen_save_part().

注意 此功能在HTML5目标上不起作用。

语法。

screen_save(fname);

争论 类型 描述
fname 字符串 The filename for the screenshot to save

返回。

N/A

例子。

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

当用户按下 "S "键时，上述代码会拍摄一张游戏的屏幕截图。这个屏幕截图被保存到一个特定的目录，并使用一个变量进行编号，然后递增。
