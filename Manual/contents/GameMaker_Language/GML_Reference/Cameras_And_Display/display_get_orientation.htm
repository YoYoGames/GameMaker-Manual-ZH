display_get_orientation

这个函数将返回GameMaker的四个常数之一，以告诉你运行游戏的设备是以横向还是纵向模式持有的（见下表）。

请注意，在HTML5目标模块中使用该功能时，可能无法正确检测设备的方向。然而，通过使用下面的脚本可以很容易地模仿这一点。

return (browser_width < browser_height);

这样一个函数将返回 true （纵向）和 false （横向）。

语法。

display_get_orientation()

返回。

Display Orientation Constant

显示方向不变
恒定 描述
display_landscape 设备被水平持有，即：最长的边缘从左到右，而菜单按钮在右边。
display_landscape_flipped 和上面一样，只是现在菜单按钮在左边。
display_portrait 设备被垂直地拿着，即：最长的边缘是从上到下，而菜单按钮在底部。
display_portrait_flipped 如上所述，只是现在菜单按钮在顶部。

例子。

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

上面的代码检查设备的方向，并根据返回的值设置一个全局变量。
