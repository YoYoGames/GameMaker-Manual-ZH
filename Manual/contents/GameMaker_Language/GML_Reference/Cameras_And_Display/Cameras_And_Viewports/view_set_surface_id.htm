查看_set_surface_id

通过这个变量，你可以设置视口的内容，将其绘制到曲面上。在处理曲面时，经常需要将屏幕上的整个可见区域捕捉到曲面上，因此你会用这个函数将其分配给一个视口。这意味着在所选择的视口中显示的所有内容现在都将被绘制到指定的曲面上，而该视口的内容将不再显示，这意味着你需要：1:

  • 启用一个新的视图，只在该视图中绘制曲面（用 view_current来检查哪个视图正在被绘制）。
  • 在实例的Draw GUI或Post Draw事件中绘制曲面，因为这些事件与视图无关。

当使用这个函数时，你要给出视图端口的索引（从0到7）和一个表面的索引（要么是 application_surface或由函数返回的唯一索引值 surface_create())，或者，如果之前已经分配了一个曲面，而你想移除它，那么就给一个-1的值。 更多关于设置视图端口到一个曲面的例子，请看变量 view_surface_id.

语法。

view_set_surface_id(view_port, surf)

争论 类型 描述
view_port 真实的 The view port to target (0 - 7)
surf 表面标识 The surface to apply to the view

返回。

Real

例子。

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

上面的代码将检查是否有一个表面被分配到视图端口[0]，如果没有，就会分配一个。
