摄像机_获取视图_矩阵

此函数可用于检索视图矩阵。该函数返回矩阵数组，然后可以在其他矩阵函数中使用，或者用于设置另一台摄像机的视图矩阵（使用 camera_set_view_mat()).请注意，此函数通常用于用以下方法创建的摄像机 camera_create()创建的摄像机，但它也可以用于在room 编辑器中创建的摄像机（或用 camera_create_view())--在这种情况下，返回的矩阵只有在给定的摄像机第一次在视口中使用时才有效。

语法。

camera_get_view_mat(camera_id)

争论类型 描述
camera_id The unique camera ID value returned when you created the camera

返回。

Matrix Array

例子。

mat = camera_get_view_mat(camera_view[0]);

上面的代码得到分配给view port[0]的摄像机的视图矩阵。
