相机_获取默认值

这个函数可以用来检索默认摄像机（GameMaker在游戏中没有摄像机视图或端口激活时使用的摄像机room ）的唯一摄像机ID值。

语法。

camera_get_default()

返回。

Camera ID

例子。

var def = camera_get_default();
view_camera[0] = def;

上面的代码得到了默认摄像机的ID，并为port[0]设置了视图摄像机，以使用它。
