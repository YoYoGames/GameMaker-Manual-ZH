view_set_visible

这个函数可以用来设置给定视口的可见性。该函数接收视图端口索引（从0到7）和一个布尔值 true ，如果它是可见的，如果它是不可见的 false 。

语法。

view_set_visible(view_port, visible)

争论类型 描述
view_port The view port to target (0 - 7)
visible Visibility toggle (true 是可见的， 是不可见的) false

返回。

N/A

例子。

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

上面的代码将检查视图端口[0]是否可见，如果不可见，则设置为可见。
