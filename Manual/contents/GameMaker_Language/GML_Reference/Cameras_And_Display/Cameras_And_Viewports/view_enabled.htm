view_enabled

这个变量控制在room 中可见的任何视口是否被启用。如果你把视口设置为可见，然后禁用这个选项，整个room 将按窗口大小的比例绘制到屏幕上，而不是通过视口绘制不同的摄像机。

语法。

view_enabled

返回。

(true: enabled, false: disabled)

例子。

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

上面的代码检查是否为room ，如果没有，它确保视图端口[0]是可见的，然后为room 。
