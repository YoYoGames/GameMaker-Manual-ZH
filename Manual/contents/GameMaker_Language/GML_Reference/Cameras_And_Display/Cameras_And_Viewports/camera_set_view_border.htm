摄像机_set_view_border

你可以使用这个函数在room 内设置视图摄像机的跟随边界。你给出唯一的摄像机ID值（由不同的 camera_create 函数返回），然后给出x边界大小和y边界大小（水平和垂直）。 和垂直）。这些值只有在摄像机被分配了一个要跟随的实例目标时才会被使用（无论是在Room 编辑器中还是使用函数 camera_set_view_target())，并且涉及到 实例离视图的边界有多远，视图才会更新其位置来跟随。

语法。

camera_set_view_border(camera_id, x_border, y_border)

争论类型 描述
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

返回。

例子。

camera_set_view_border(view_camera[0], 64, 64);

上面的代码将为分配给view port[0]的摄像机设置视图摄像机边界。
