毁坏相机

通过这个函数，你可以销毁任何摄像机。在调用该函数时，你要提供唯一的摄像机ID值，你可以从 camera_create_* 函数中获得，如果使用 编辑器设置视图端口和视图，则可以从 view_camera数组中获得，如果使用Room 编辑器来设置视口和视图。除非你在同一步骤中为该视图分配一个新的摄像机，否则你永远 不应该销毁当前分配给可见视图的摄像机，当你的游戏不再需要时，你应该永远销毁你通过代码创建的任何摄像机，以防止内存泄漏，如果你在Room 编辑器中分配了任何摄像机，你也可以销毁默认摄像机，但你应该为视图端口分配一个新的摄像机（或禁用它），否则你会得到奇怪的结果。

语法。

camera_destroy(camera_id)

争论类型 描述
camera_id The unique camera ID value returned when you created the camera.

返回。

N/A

例子。

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

上面的代码销毁了当前分配给视图端口[0]的摄像机，然后创建了一个新的摄像机并将其ID分配给该端口。
