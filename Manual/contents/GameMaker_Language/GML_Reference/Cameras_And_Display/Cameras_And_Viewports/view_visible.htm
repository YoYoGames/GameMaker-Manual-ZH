查看_可见

这个变量可以用来了解某个特定的视图端口当前是否可见。你也可以将这个变量设置为 true （可见）或 false （不可见），从而有效地 "打开 "或 "关闭 "一个视图。注意，即使你将视图端口设置为可见，如果视图端口没有被启用（使用内置变量 view_enabled或在Room 编辑器中启用它们），那么它们就不会被绘制到屏幕上。

语法。

view_visible[0 ... 7];

返回。

(true: enabled, false: disabled)

例子。

if !view_visible[0]
{
    view_visible[0] = true;
}

上面的代码检查view[0]是否可见，如果不可见，则将其设置为可见。
