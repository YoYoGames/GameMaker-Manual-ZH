查看当前

这个只读变量只在 绘制事件并返回当前正在渲染的视图端口。当你有不同的视图时，返回值会在draw事件中发生变化，因为draw事件会对每个视图端口连续调用一次。因此，当（例如）你在游戏中使用view port[0]和view port[1]时room ，所有实例的draw事件将被运行两次，每个端口一次，通过这个变量你可以检查出当前正在绘制的视图。一般来说，当多个视图端口在room 中同时可见时，这被用来只对单个端口渲染特定细节。请看下面的示例代码。

语法。

view_current;

返回。

Real (view port index from 0 to 7)

例子。

if view_current == 0
{
    var xx = camera_get_view_x(view_camera[0]);
    var yy = camera_get_view_y(view_camera[0]);
    draw_text(xx + 32, yy + 32, "Player 1");
}
else
{
    var xx = camera_get_view_x(view_camera[1]);
    var yy = camera_get_view_y(view_camera[1]);
    draw_text(xx + 32, yy + 32, "Player 2");
}

上述代码检查当前正在绘制的视图，然后根据返回值为每个视图绘制不同的文本。
