camera_set_view_target

您可以使用此函数在room 内设置视图摄像机的跟随目标。您可以给出唯一的摄像机ID值（由不同的 camera_create()函数返回的），然后给出你希望设置摄像机视图的目标实例或object ID。请注意，如果你选择了一个object ID，而在object 中有多个该room 的实例，GameMaker没有办法知道你想跟随哪个实例，所以可能是其中的任何一个。

语法。

camera_set_view_target(camera_id, instance_id/object_id)

争论 类型 描述
camera_id 摄像机ID The unique camera ID value returned when you created the camera
instance_id/object_id 对象资产或实例ID Instance or object to have the camera target for following

返回。

N/A

例子。

camera_set_view_target(view_camera[0], id);

上述代码将把分配给view port[0]的摄像机的视图摄像机目标设置为运行代码的实例的目标。
