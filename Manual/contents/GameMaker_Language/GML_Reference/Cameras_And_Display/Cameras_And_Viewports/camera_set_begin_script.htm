摄像机_set_begin_script

这个函数可以用来设置一个脚本函数，在摄像机被分配到一个可见和活动的视口的每个游戏帧的开始，都会被调用。你给出唯一的摄像机ID值（由不同的 camera_create() 函数返回）和要调用的函数的ID。scripts 连接到摄像机和摄像机视图的实际渲染的顺序如下。

  • 所有可见和活动视图端口的摄像机的更新函数都被称为
  • 然后对于每个视图。
    • 为该视图的摄像机调用开始函数
    • 执行该视图的绘制事件
    • 摄像机的结束函数 被调用
  • 移动到下一个活动的可见视图并重复...

语法。

camera_set_begin_script(camera_id, script)

争论 类型 描述
camera_id 摄像机ID The unique camera ID value returned when you created the camera.
script 脚本功能 The script function to run each game frame

返回。

N/A

例子。

camera_set_begin_script(view_camera[0], beginCamera);

上面的代码为分配给视口[0]的摄像机设置了开始script 功能。
