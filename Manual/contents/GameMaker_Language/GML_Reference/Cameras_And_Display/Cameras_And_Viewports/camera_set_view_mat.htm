摄像机_set_view_mat

此函数将为一个给定的摄像机设置视图矩阵。你要给出唯一的摄像机ID值（由不同的 camera_create()函数返回的）和一个要应用的视图矩阵。你可以从矩阵函数一节中了解到更多关于创建视图矩阵的信息，特别是 matrix_build_lookat().

object 请注意，如果你的摄像机做了自动object 追踪--即：它是用 camera_create_view() ，索引/实例ID不是-1，或者你正在设置一个在Room 编辑器中定义的摄像机，或者你正在设置默认摄像机--那么这个矩阵将在下一个游戏帧被覆盖。

语法。

camera_set_view_mat(camera_id, matrix)

争论类型 描述
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

返回。

例子。

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建了一个新的摄像机并将其分配给view port[0]。然后这个相机的视图和投影矩阵被设置。
