camera_apply

这个函数将把给定的摄像机设置立即应用于正在渲染的当前摄像机视图。虽然你可以使用特定的 camera_set_* 函数来设置任何视图的摄像机，但新的设置要到下一个游戏帧才会被用于渲染，但通过这个函数你可以立即进行这些改变。如果你使用的是多个视图，那么这个函数应该在绘图事件中使用，你可以使用变量 view_current来选择性地将给定的摄像机只应用于当前正在渲染的摄像机视图。请注意，这不会改变摄像机的ID值，只会改变该摄像机的设置，如果你在默认的摄像机上使用这个函数（即：GameMaker在没有视图摄像机的情况下使用的内部摄像机），那么在下一个游戏帧中，这些改变将被绘制room 的内部设置所覆盖。

语法。

camera_apply(camera_id)

争论类型 描述
camera_id The unique camera ID value returned when you created the camera.

返回。

例子。

if view_current == 0
{
    camera_apply(cutscene_cam);
}

上面的代码（在绘图事件中）检查当前正在渲染的摄像机视图，如果是摄像机视图[0]，那么变量 "cutscene_cam "所引用的摄像机的设置就会应用到它。
