display_get_sleep_margin

这个函数可以用来获取当前用于游戏渲染计时的睡眠保证金值，并将返回一个毫秒值。关于显示计时的更多信息，请参见 display_set_timing_method().

语法。

display_get_sleep_margin();

返回。

例子。

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

上面的代码将检查计时方法，然后如果它没有被设置为 tm_sleep ，它将被设置为该值，并将睡眠保证金设置为20。
