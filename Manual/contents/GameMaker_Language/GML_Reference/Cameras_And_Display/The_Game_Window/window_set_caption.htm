窗口_set_caption

通过这个功能，你可以改变或设置你当前所处的room 的窗口标题。当游戏不在全屏模式时，这个标题会出现在窗口的顶部，在游戏图标的旁边。

语法。

window_set_caption(caption);

争论类型 描述
caption The new caption.

返回。

例子。

if window_get_caption() != ""
{
    window_set_caption("");
}

上面的代码将检查窗口的标题，看它是否有文字，如果有，就把它设置为空的string ，这样就不会显示标题了。

返回。游戏窗口
