窗口_手柄

通过这个函数你可以得到内部Windows 的id值（HWND，一个指针）。 window handle 这个函数实际上只对那些需要extension ，以便在DLL代码中调用Windows API的编写者有用（返回的指针应该被铸成一个string ，然后在C++中只是把它铸成一个HWND）。下表显示了支持的平台以及它们返回的内容。

平台 返回
窗户 窗口HWND
macOS NSWindow类
Ubuntu (Linux) XWindow手柄
HTML5 帆布的ID

语法。

window_handle();

返回。

Pointer

例子。

win_id = windows_handle();

上面的代码将在一个变量中存储游戏窗口的ID。

返回。游戏窗口
