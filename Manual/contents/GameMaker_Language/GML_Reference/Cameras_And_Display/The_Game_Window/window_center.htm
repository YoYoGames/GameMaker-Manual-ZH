window_center

当目标模块是Windows,Ubuntu (Linux) 或macOS 时，用这个功能可以使游戏窗口在显示器中居中，如果目标模块是HTML5，也可以在浏览器中居中。这个功能对其他设备没有任何影响。

注意如果你在当前步骤中调整了游戏窗口的大小（例如，从全屏切换到视窗或使用 window_set_size() ），并希望将新的窗口居中，这个函数应该至少晚一步调用（例如，通过使用 alarm)，否则它将无法正常工作。

语法。

window_center();

返回。

例子。

if !window_get_fullscreen()
{
    window_center();
}

如果不是全屏模式，上述代码将使游戏窗口在显示屏内居中。

返回。游戏窗口
