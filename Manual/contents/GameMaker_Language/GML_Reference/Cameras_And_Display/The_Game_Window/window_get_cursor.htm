窗口_获取游标

通过这个函数，你可以得到游戏窗口中当前使用的光标，返回值是下面列出的任何一个常量（要设置当前使用的光标，你可以使用函数 window_set_cursor()来设置正在使用的光标，它也需要这些常量）。

游标常数
恒定 游标
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

语法。

window_get_cursor();

返回。

Cursor Constant

例子。

if mouse_check_button_pressed(mb_left)
{
    if window_get_cursor() != cr_drag window_set_cursor(cr_drag);
}

上面的代码将把窗口光标改为标准的windows拖动光标，如果鼠标左键被按下并且之前还没有被改变的话。

返回。游戏窗口
