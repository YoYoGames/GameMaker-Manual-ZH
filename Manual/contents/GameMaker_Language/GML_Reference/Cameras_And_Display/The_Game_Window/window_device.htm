window_device

这个函数将返回当前的d3d设备指针，然后你可以（例如）将其传递给Windows 和macOS 上的DLL或Dylib。

注意在GameMaker中，这个函数已被废弃，改为使用 os_get_info().

语法。

window_device();

返回。

Pointer

例子。

gfx_pointer = window_device();

上面的代码将把d3d设备的指针储存在一个变量中。

返回。游戏窗口
