window_set_max_height

这个函数可以用来为你的游戏设置一个最大的窗口高度。如果你在目标平台的游戏选项中启用了窗口调整选项，那么玩家可以将游戏窗口调整到他们希望的任何尺寸，然而通过使用这个函数 你可以将最大高度限制在你指定的尺寸。如果你希望回到默认行为（即：没有最小值），那么使用-1的值。

注意：该功能仅在Windows、macOS和Linux平台上可用。

语法。

window_set_max_height(height);

争论类型 描述
height The maximum height in pixels for the game window.

返回。

例子。

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

上面的代码将为游戏窗口设置最小和最大的宽度和高度。

返回。游戏窗口
