window_set_fullscreen

通过这个功能，你可以将游戏窗口设置为全屏(true)或不设置(false)。请注意，对于macOS目标，你必须取消勾选 "在全屏下开始 "选项，并在游戏选项中勾选 "允许玩家调整游戏窗口大小 "选项，否则该功能将失败。还要注意的是，这个功能在HTML5上是不工作的，除非它被添加到 "可点击 "回调中（更多细节见这里）。

语法。

window_set_fullscreen(full);

争论类型 描述
full Whether to set the screen to fullscreen (true) or not (false).

返回。

例子。

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

上面的代码检查鼠标按键是否被按下，如果还没有被按下，则将窗口设置为全屏，如果已经被按下，则将其设置为窗口。

返回。游戏窗口
