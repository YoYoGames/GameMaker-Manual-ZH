display_set_ui_visibility

这个函数可以用来显示或隐藏Android系统的用户界面。该函数要求你提供一个或多个系统标志的整数值。当使用一个以上的flag ，这些需要用位法 "或 "来合并，如下面的例子所示。你可以在这里找到一个Android 系统flags 的列表。

语法。

display_set_ui_visibility(flags);

争论 类型 旗帜
flags 真实的 The Android SYSTEM flags to use (as an integer value).

返回。

N/A

例子。

var flags = 1024|4096;
display_set_ui_visibility(flags);

上述代码将使用Android 系统flags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY "和 "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN "来设置显示器的可见性。
