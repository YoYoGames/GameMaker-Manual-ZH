screen_save_part

通过这个功能，你可以保存游戏中某一部分的截图，就像它目前在游戏显示屏中显示的那样。选择坐标是基于显示器（如果不是全屏，则为窗口）的绝对尺寸，所以（0,0）总是屏幕（或窗口）的左上角。下面是一张图片，显示了这个功能的工作原理。

该文件将被保存到游戏的工作目录中。值得注意的是，该函数被设计为在Draw GUI End Event中调用，它应该在所有平台上得到一致的结果。在任何其他事件中使用它可能不会像预期的那样工作，并且可能在不同的目标甚至设备上得到不同的结果。

注意 该功能在HTML5目标上不起作用。

语法。

screen_save_part(fname,x,y,w,h);

争论类型 描述
fname The filename for the screenshot to save
x The x coordinate of the top left corner of the area of the screen to capture
y The y coordinate of the top left corner of the area of the screen to capture
w The width of the area of the screen to capture
h The height of the area of the screen to capture

返回。

例子。

if keyboard_check_pressed(ord("S"))
{
    screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100)
}

上面的代码在用户按下 "S "键时，对游戏屏幕上100像素的部分进行截图。这个屏幕截图被保存到一个特定的文件夹，并使用一个变量进行编号，然后递增。
