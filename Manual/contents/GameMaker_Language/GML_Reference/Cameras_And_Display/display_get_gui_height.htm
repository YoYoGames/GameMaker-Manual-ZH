display_get_gui_height

通过这个函数，你可以得到在Draw GUI Event中使用的GUI的高度（像素）。

语法。

display_get_gui_height();

返回。

Real

例子。

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

上面的代码将设置一个广告显示在显示屏的左下角。
