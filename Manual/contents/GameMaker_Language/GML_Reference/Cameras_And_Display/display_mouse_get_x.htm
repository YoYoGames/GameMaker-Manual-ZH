display_mouse_get_x

这个函数将返回鼠标在屏幕中的X位置。应该注意的是，这个函数只有在Windows 和macOS 目标上使用时才能正常工作。

它也可以用于HTML5，但只返回相对于画布本身的0,0的值，当鼠标在画布之外时，不会返回任何值。

对于其他设备，它将返回0，而你应该使用 device_mouse_raw_x()和 device_mouse_raw_y()函数来代替。

语法。

display_mouse_get_x();

返回。

Real

例子。

my_x = display_mouse_get_x();

这将把my_x设置为鼠标相对于显示器的x坐标。
