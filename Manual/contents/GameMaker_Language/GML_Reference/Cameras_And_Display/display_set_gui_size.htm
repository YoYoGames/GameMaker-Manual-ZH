display_set_gui_size

这个函数只影响在Draw GUI事件中绘制的东西，可以用来为该事件中的所有GUI组件设置一个特定的宽度和高度，并且无论用什么 显示屏或窗口的大小来显示GUI，它都会被正确缩放以适应。GUI的定位（即：(0,0)位置）将取决于你是否使用了该函数 display_set_gui_maximise() 函数与否。如果你不使用这个函数（默认设置），那么GUI层将被定位，其(0,0)点与应用表面（你的游戏被绘制的区域）的位置相同，宽度和高度将被拉伸以适应这个区域。 被拉伸以适应这个表面，例如，当应用表面是512px时，用这个函数设置一个256px的GUI宽度将拉伸这256个像素来适应，有效地使像素变成两倍大。这对那些游戏来说是很理想的 这对那些在全局游戏设置中设置了长宽比校正的游戏是非常理想的。

如果你使用了 display_set_gui_maximise() 函数，那么GUI层的(0,0)位置将默认为游戏窗口的左上角，或者显示器（在全屏或设备上时），其缩放系数由该函数设置。 和这个函数设置的宽度和高度。你可以在任何时候将GUI层重置为默认配置，方法是用-1作为宽度和高度调用这个函数--这将把(0, 0)位置设置为应用程序表面，宽度 和高度也将被重置为与表面1:1。

语法。

display_set_gui_size(width, height);

争论类型 描述
width The width of the GUI
height The height of the GUI

返回。

例子。

display_set_gui_size(768, 1024);

上述代码将把绘制GUI事件锁定在给定的宽度和高度上，使用该比例缩放所有组件以适应应用程序表面或显示器。
