display_get_dpi_x

每英寸点数（DPI）是对空间印刷或视频点密度的衡量，特别是在1英寸（2.54厘米）范围内，可以放在一条线上的单个点的数量。当在移动设备上工作时（特别是Android 设备），这是一个需要考虑的重要因素，因为可能适合于一种显示分辨率的东西，可能不适合于另一种。例如，您可能有两个分辨率相同的显示器，即400 x 800，但显示器1的dpi为60，显示器2的dpi为30。在这种情况下，显示在显示器2上的任何文本或图像都会显得大得多，尽管实际分辨率是一样的。

此函数将获得设备显示屏沿x轴的dpi（此值也取决于设备的方向）。请注意，Mac和iOS ，不返回具体的dpi设置，但似乎返回与操作系统相同的值，这是不正确的（但将不得不这样做），因为苹果没有给出正确的值。

语法。

display_get_dpi_x();

返回。

例子。

dpx = display_get_dpi_x();

这将把变量 "dpx "设置为x轴的dpi值。
