display_set_sleep_margin

这个函数可以用来设置用于游戏渲染计时的睡眠保证金值，需要一个毫秒值。默认值如下。

平台 毫秒
窗户 10
macOS 10
乌班图 10
HTML5 10
安卓 4
基础设施 4
Windows UWP 10
宇宙飞船 10
淘宝网 10
任天堂开关 10

注意，即使计时方法被设置为使用vsync计时，睡眠余量也会对游戏的渲染产生一定的影响，因此在设置这个值时应该注意。关于显示计时的更多信息，请参见 display_set_timing_method().

注意 Windows ：除了睡眠保证金，你还可以在runtime ，通过调整线程调度器的分辨率，进一步控制你的性能和功耗。

语法。

display_set_sleep_margin(milliseconds);

争论 类型 描述
milliseconds 真实的 The number of milliseconds for the sleep margin

返回。

N/A

例子。

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if (display_get_sleep_margin() != 20)
    {
        display_set_sleep_margin(20);
    }
}

上面的代码将检查计时方法，然后如果它没有被设置为 tm_sleep ，它将被设置为该值，并将睡眠保证金设置为20。
