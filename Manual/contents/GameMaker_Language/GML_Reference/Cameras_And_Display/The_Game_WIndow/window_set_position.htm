窗口_设置位置

通过这个功能，你可以将游戏窗口设置到显示器内的特定位置（在macOS 、Linux(Ubuntu)和Windows ）或在浏览器内（HTML5）。

注意：如果你的HTML5游戏使用了一个自定义的 index.html ，并且将画布设置为固定位置，那么这个函数将对窗口位置没有影响。

语法。

window_set_position(x, y);

争论类型 描述
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

返回。

例子。

window_set_position(0, 0);

上述代码将把游戏窗口定位在浏览器或显示器的左上角（取决于正在使用的目标模块）。

返回。游戏窗口
