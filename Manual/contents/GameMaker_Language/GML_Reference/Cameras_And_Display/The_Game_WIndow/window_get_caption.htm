窗口_获取标题

这个函数返回窗口的标题（这是出现在窗口顶部的文字，在它的图标旁边），默认情况下，它显示的是你当前所处的room 的标题。

语法。

window_get_caption();

返回。

例子。

if window_get_caption() != ""
{
    window_set_caption("");
}

上面的代码将检查窗口的标题，看它是否有文字，如果有，就把它设置为空的string ，这样就不会显示标题了。
