窗口_设置大小

通过这个功能，你可以改变游戏窗口的大小。在桌面平台上，这通常指的是窗口的客户端区域，不包括标题栏和边框。

如果你用这个函数来改变游戏窗口的大小，窗口的内容将以1:1的比例与背景画布相适应，这将保持图像的清晰明了（尽管是缩放的，如果窗口与room 或视图大小不同）。例如，如果你有一个200×150像素的窗口，而玩家通过拖动边框拉伸窗口，最终的窗口尺寸可能是400×300像素，但绘图画布仍然是200×150，这就会产生模糊的图像，因为现在的像素比是1:2。然而，如果你使用这个函数来改变窗口的大小，1:1的比例就会被保持，即使图像被缩放以适应新的窗口大小。

语法。

window_set_size( w, h );

争论类型 描述
w The new width of the window.
h The new height of the window.

返回。

N/A

例子。

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

上面的代码将在按下鼠标按钮时根据变量 "global.scale "的值改变窗口的大小。

返回。游戏窗口
