窗口_有焦点

通过这个函数你可以轮询窗口（或标签）的状态，如果它失去了焦点，该函数将返回 false ，否则将返回 true 。在大多数情况下，你可以简单地使用 os_is_paused()函数来测试，但在一些非常特殊的情况下（例如Chrome应用程序上的游戏），该函数将不会触发，在这种情况下，你应该使用这个函数。

注意：这个功能只对HTML5、Windows 、MacOS有效。

语法。

window_has_focus();

返回。

例子。

if !window_has_focus()
{
    PauseGame();
}

上面的代码将检查游戏窗口是否处于焦点位置，如果该函数返回 false ，将调用一个函数。

返回。游戏窗口
