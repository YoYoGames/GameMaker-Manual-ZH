游戏窗口

你创建的游戏发生在一个窗口中（即使是全屏），这个窗口有很多属性，比如位置、大小、是否全屏等等。这些细节通常是根据room ，结合 目标平台 的游戏选项的设置，自动为你设置的，但你可以在游戏过程中使用本页面列出的功能来改变它们。

注意这些功能只适用于Windows、Ubuntu(Linux)、macOS 和HTML5目标模块，在其他任何设备上可能无法使用。

下面的图片说明了一些一般的窗口功能是如何相互联系和相互作用的。

请注意，上图显示的是在浏览器中绘制的游戏窗口，但对于其他目标，你可以用显示器代替浏览器，所以像 window_get_y() 这样的函数将返回游戏窗口顶部相对于显示器的位置。
