display_get_timing_method

这个函数可以用来检索用于渲染游戏的计时方法。该方法可以是下面列出的常数之一。

有关不同计时方法的更多信息，请参见 display_set_timing_method().

语法。

display_get_timing_method();

返回。

Display Timing Method Constant

显示定时方法常数
恒定 描述
tm_sleep 睡眠余量值是主要的计时方法
tm_countvsyncs Vsync计时是主要的计时方法（所有支持的平台都默认）。

例子。

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

上面的代码将检查计时方法，然后如果它没有被设置为 tm_sleep ，它将被设置为该值，并将睡眠保证金设置为20。
