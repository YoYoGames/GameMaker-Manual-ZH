display_get_frequency

这个函数返回正在玩游戏的显示器的频率（或刷新率）。它将返回一个真实的数值，即每秒钟的帧数，因此，如果你的显示器是60赫兹，你将得到60，如果它以144赫兹运行，你将得到144，以此类推。

注意HTML5上不支持此功能。

语法。

display_get_frequency();

返回。

例子。

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

上面的代码得到了显示器的频率，并运行一些条件来设置游戏运行在240、120、60或30帧。这意味着，如果你的显示器是90hz，游戏将以60帧运行，如果是144hz，那么将以120帧运行，等等。当然，你可以把显示器的频率直接传到 game_set_speed() ，这样它就被用作游戏的帧率。
