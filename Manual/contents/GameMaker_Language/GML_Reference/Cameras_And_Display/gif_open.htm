gif_open

通过这个函数，你可以创建一个空的GIF格式图像，准备好向其添加数据。当你调用这个函数时，你需要指定GIF的宽度和高度（以像素为单位，除了可用的内存外，大小没有上限），该函数将返回唯一的ID值，用于在随后的函数中识别礼物，如果GIF不能被初始化（例如，如果宽度/高度对可用的内存太大），它将返回-1。你还可以指定一个可选的参数来设置GIF的 "透明色"。这是一个RGB值（没有阿尔法成分），在向GIF添加任何东西之前，会将GIF清除为这个颜色。如果你不提供清除颜色的参数，将使用默认的黑色。

注意，在使用这个函数时，你必须在打开另一个文件进行录制之前调用 gif_save()来结束GIF的创建，然后再打开另一个文件进行录制（所以每一个 gif_open() ，必须有一个相应的 gif_save() ）。

语法。

gif_open(width, height, [clear_colour]);

争论 类型 描述
width 真实的 The width of the gif to create
height 真实的 The height of the gif to create
[clear_colour] 颜色 OPTIONAL 要清除gif的颜色（RGB）。

返回。

GIF ID

例子。

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

上面的代码将创建一个GIF图像文件，其中有30个从应用程序表面拍摄的帧，然后将其保存。
