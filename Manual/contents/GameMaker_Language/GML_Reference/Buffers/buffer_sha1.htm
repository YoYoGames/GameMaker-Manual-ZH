buffer_sha1

在密码学中，SHA-1是一个由美国国家安全局设计的密码散列函数，被用于一些广泛使用的应用程序和协议中，如流行的 Git，它被用来检查文件的变化。这个函数将接受来自buffer 的输入数据，并以ASCII格式返回160位消息摘要。通过这种方式，你可以生成一个安全的密钥，它可以被存储并用于检查 被发送到（或从）一个外部server （例如）的信息的完整性。buffers 当使用这个函数将其应用于buffer 时，你必须指定要使用的buffer id，然后指定一个偏移值（以字节为单位）作为开始的位置，然后是 大小（同样以字节为单位），以确定要被散列的区域。

注意：SHA-1不是完全安全的，可以被破解。更多信息请见本页面。

语法。

buffer_sha1(buffer, offset, size);

争论类型 描述
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

返回。

例子。

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

上面的代码将为存储在buffer ，由变量 "buff "索引的完整数据创建一个sha1哈希值，并将返回的哈希值存储在变量 "check_string "中。
