buffer_crc32

这个函数将从一个buffer ，并返回一个crc32校验和散列的输入数据。buffer 你指定要使用的buffer ID，然后指定一个偏移值（以字节为单位）作为开始的位置，然后指定要散列的区域的大小（同样以字节为单位），并且 该函数将为该区域返回一个32位的整数值。

语法。

buffer_crc32(buffer, offset, size);

争论类型 描述
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

返回。

例子。

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

上述代码将为存储在buffer ，由变量 "buff "索引的完整数据创建一个crc32校验和哈希值，并将返回的整数哈希值存储在变量 "check_val "中。
