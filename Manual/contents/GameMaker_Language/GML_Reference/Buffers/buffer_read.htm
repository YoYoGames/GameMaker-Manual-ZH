缓冲区_读

这个函数可以用来从一个先前创建的buffer 中读取数据。返回值将取决于你所读取的数据类型，其本身由以下常数定义。

恒定 描述
buffer_u8 一个无符号、8位的整数。这是一个从0到255的正值。
buffer_s8 一个有符号的、8位的整数。这可以是一个从-128到127的正值或负值（0被认为是正值）。
buffer_u16 一个无符号、16位的整数。这是一个从0-65,535的正值。
buffer_s16 一个有符号的、16位的整数。这可以是一个从-32,768到32,767的正值或负值（0被认为是正值）。
buffer_u32 一个无符号、32位的整数。这是一个从0到4,294,967,295的正值。
buffer_s32 一个有符号的、32位的整数。这可以是一个从-2,147,483,648到2,147,483,647的正值或负值（0被归类为正值）。
buffer_u64 一个无符号的64位整数。
buffer_f16 一个16位的浮点数。这可以是+/-65504范围内的一个正值或负值。(目前不支持！)
buffer_f32 一个32位的浮点数。这可以是+/-16777216范围内的一个正值或负值。
buffer_f64 一个64位的浮点数。
buffer_bool 一个布尔值。只能是1或0 (true 或 false)
buffer_string 一个string ，无论大小。
buffer_text 一个任意大小的string ，没有最后的空结尾字符。

如果函数成功，它将返回一个给定类型的值，但是如果失败，那么它将引起一个运行错误。

注意对正在读取的数据使用不正确的数据类型将导致返回错误的值。

语法。

buffer_read(buffer, type)

争论 类型 描述
buffer 缓冲区ID The index of the buffer to read from.
type 缓冲区数据类型常数 The type of data that is to be read from the buffer (see the list of constants above).

返回。

Real, Boolean or String

例子。

var cmd = buffer_read(buff, buffer_s16);

上面的代码从buffer ，id存储在变量 "buff "中，读取一个有符号的16位值到本地变量 "cmd"。
