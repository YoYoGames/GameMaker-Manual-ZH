buffer_decompress

通过这个函数，你可以用zlib压缩法解压一个先前压缩的buffer 。你提供要解压的buffer ID，该函数将返回一个新的buffer ID，其中包含未压缩的数据。如果解压失败（例如，你提供的是一个没有被压缩的buffer ），那么这个函数将返回一个小于0的值。

语法。

buffer_decompress(buffer);

争论类型 描述
buffer The index of the buffer to decompress.

返回。

Buffer ID

例子。

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

上面的代码将首先加载一个保存的buffer ，然后解压，最后从解压后的string 读取buffer 数据到一个全局变量。
