buffer_base64_decode

通过这个函数，你可以将一个base64编码的string （用该函数创建）解码为一个 。 buffer_base64_encode()buffer该函数将创建buffer （作为一个1字节对齐的 "增长 "缓冲区），并返回buffer 的唯一索引，该索引应在所有后续函数调用中使用。

注意：当你不再需要这样的动态创建的资源时，必须将其从内存中删除，以防止内存泄漏，所以当你完成了你所创建的buffer ，你应该再次使用释放它。 buffer_delete().

语法。

buffer_base64_decode(string);

争论类型 描述
string The base64 encoded string to decode

返回。

Buffer ID

例子。

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

上面的代码将打开一个ini文件，然后从其中读取一个string ，进入解码函数。该函数将返回一个buffer 索引，该索引被存储在变量 "buff "中，其中包含先前编码和保存的数据。然后，ini文件被关闭。
