buffer_get_address

通过这个函数，你可以得到一个指向原始的、对齐的 buffer 地址的指针。这主要是为了与extensions ，因为你可以把这个值传递给他们，使他们能够访问buffer 数据。请注意，有一些事情 当从扩展访问时，你不能对buffer 进行操作。

  • 你不能释放记忆。
  • 你不能调整内存的大小。
  • 你不能写超过buffer 地址加上buffer 大小（你可以使用 buffer_get_size()函数），否则你会得到一个越界的错误。

语法。

buffer_get_address(buffer);

争论类型 描述
buffer The index of the buffer to use.

返回。

例子。

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

上面的代码将获得在变量 "buff_model "中索引的buffer 的内存地址，然后使用 buffer_get_size() 函数获得buffer 的末尾的内存地址，将这两个值存储在本地变量中，以便进一步使用。 进一步使用。
