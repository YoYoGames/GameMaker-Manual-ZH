缓冲区_保存

通过这个功能，你可以把buffer 的内容保存到一个文件中，准备用这个文件读回内存。 buffer_load()函数读回内存。

注意：在HTML5上，使用此功能时，buffer 的内容将被保存为base64编码的strings 。

语法。

buffer_save(buffer, filename);

争论类型 描述
buffer The index of the buffer to save.
filename The name of the file to save as.

返回。

例子。

buffer_save(buff, "Player_Save.sav");

将buffer 的当前内容与存储在变量 "buff "中的id保存到一个文件中。
