buffer_base64_encode

这个函数将把给定的buffer 中的数据转换成base64格式的编码string 。这是一个常用的编码方案，通常用于任何需要在互联网上以文本形式存储或传输的媒体，并使 输出，使人眼无法阅读。要使用这个，你需要指定一个已经创建的buffer ，偏移值（这是你希望开始编码的buffer 内的点）以及要编码的buffer 内存的大小（字节）。

语法。

buffer_base64_encode(buffer, offset, size);

争论类型 描述
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

返回。

例子。

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

上述代码将创建编码存储在由变量 "buff "索引的buffer 中的完整数据，并将返回的string 存储在本地变量 "b_str "中。
