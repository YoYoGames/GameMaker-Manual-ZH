buffer_base64_decode_ext

通过这个函数，你可以将一个base64编码的string （用该函数创建）解码为一个 。 buffer_base64_encode()函数创建的）解码成一个buffer 。与函数不同的是 buffer_base64_decode(), 这不会为你创建一个buffer ，而是你应该已经创建了buffer （见 buffer_create())，然后你会用这个函数来使用它的ID。偏移量 "是指 偏移量 "是在buffer 中解码给定的string 的位置（以字节为单位）。

语法。

buffer_base64_decode_ext(buffer, string, offset);

争论类型 描述
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

返回。

例子。

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

上述代码将创建一个buffer ，并将其唯一的ID存储在变量 "buff "中，然后打开一个ini文件，并从其中读取一个string 到本地变量 "str"。这个string ，然后被解码到新创建的buffer ，然后再关闭ini文件。 再次关闭ini文件。
