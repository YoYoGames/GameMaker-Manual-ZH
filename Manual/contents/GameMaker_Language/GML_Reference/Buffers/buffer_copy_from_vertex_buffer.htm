buffer_copy_from_vertex_buffer

该函数可用于将一个vertex vertex 中存储的部分（或全部）buffer 数据复制到一个先前创建的普通buffer 。当用这个函数从一个vertex buffer 拷贝到一个普通的buffer 时，两个buffers 都必须是以前创建的（使用 vertex_create_buffer()和 buffer_create()函数，例如）。你可以指定你想复制到vertex 的buffer 数据范围，其中开始vertex 可以是0到顶点数-1之间的任何地方，你可以给出从该点开始要复制的顶点数。你可以使用函数 vertex_get_number()vertex buffer 来获得存储的顶点总数。最后，你给出buffer 的索引来复制vertex 的数据，以及一个数据偏移量来定义复制vertex 的数据到目标buffer 的位置。

语法。

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

争论 类型 描述
vertex_buffer 顶点Buffer ID The index of the vertex buffer to copy from.
start_vertex 真实的 The starting vertex.
num_vertices 真实的 The total number of vertices to use.
dest_buffer 缓冲区ID The index of the buffer to copy to.
dest_offset 真实的 The offset position to copy the data to (in bytes).

返回。

N/A

例子。

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

vertex 上述代码将复制存储在vertex buffer 中的、以 "model_buffer "为索引的数据，然后将其粘贴到以 "player_buffer "为索引的buffer 。
