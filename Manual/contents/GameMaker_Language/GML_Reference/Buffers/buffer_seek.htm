这个函数可以用来在一个buffer ，找到开始、结束或相对于读或写数据时最后使用的位置。偏移 "值是在给定的寻找位置上增加的偏移量（以字节为单位），例如，如果基数是相对的，偏移量是4，那么buffer 位置将从其当前位置沿4字节移动。请注意以下几点。

  • 你可以使用负值的偏移量，通过buffer ，也可以使用正值的偏移量来回寻。
  • 如果buffer 是 "缠绕 "类型的，并且你偏移超过了buffer 的终点，那么寻找的位置也会被缠绕。
  • 如果buffer 不是 "包裹 "类型，寻址将夹在buffer 的开头或结尾，即使偏移量会使寻址超出buffer 的限制。

以下常数被接受为寻求的 "基础 "参数。

缓冲区寻找常数
恒定 描述
buffer_seek_start 缓冲区的起点
buffer_seek_relative 相对于当前读/写位置的一个位置
buffer_seek_end 缓冲区的末端

语法。

buffer_seek(buffer, base, offset);

争论 类型 描述
buffer 缓冲区ID The index of the buffer to use.
base 真实的 The base position to seek.
offset 真实的 The data offset value.

返回。

N/A

例子。

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

上面的代码找到了buffer ，其id存储在变量 "buff "中，他们向其写入了一系列有符号的16位整数值。
