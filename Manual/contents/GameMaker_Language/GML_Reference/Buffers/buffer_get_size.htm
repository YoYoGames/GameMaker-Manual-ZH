buffer_get_size

通过这个函数，你可以得到给定的buffer ，以字节为单位的大小。

语法。

buffer_get_size(buffer);

争论 类型 描述
buffer 缓冲区ID The index of the buffer to get the size of.

返回。

Real

例子。

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

上述代码将创建一个新的buffer ，并将其索引存储在本地变量"_temp "中，这个新的buffer 的大小与变量 "player_data "中索引的buffer 相同。
