緩衝區_刪除

通过这个函数，你可以从内存中删除一个以前用函数创建的buffer 。 buffer_create()从内存中删除，释放用于创建它的资源并删除它目前可能包含的任何数据。

注意：当你不再需要任何动态创建的资源时，一定要将其从内存中删除，以防止内存泄漏，这一点很重要。

重要的是!当你创建一个buffer ，用来识别它的索引值是一个从0开始的整数。这些索引被GameMaker ，所以一个被破坏的buffer 的索引值可能被之后新创建的 使用。 我们建议你在销毁buffer 后，将任何持有缓冲区索引的变量设置为-1。

语法。

buffer_delete(buffer)

争论类型 描述
buffer The index of the buffer to delete.

返回。

例子。

buffer_delete(player_buffer);
player_buffer = -1;

上面的代码将删除先前创建的buffer ，其id存储在变量 "player_buffer "中，然后将该变量设置为-1。
