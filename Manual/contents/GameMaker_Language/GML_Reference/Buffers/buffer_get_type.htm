buffer_get_type

通过这个函数，你可以检索出被buffer ID值引用的buffer 的类型。返回值将是以下常数之一。

恒定 描述
buffer_fixed 一个固定尺寸的buffer 。
buffer_grow 一个buffer ，它将随着数据的增加而动态地 "增长"。
buffer_wrap 一个buffer ，其中的数据将被 "包裹 "起来。当正在添加的数据达到buffer 大小的极限时，覆盖的数据将被放回buffer 的起点，并从该点继续写下去。
buffer_fast 特殊的 "剥离 "buffer ，读/写速度极快。只能与 buffer_u8 数据类型一起使用，并且必须是1字节对齐。
buffer_vbuffer 一个vertex 缓冲器

语法。

buffer_get_type(buffer);

争论类型 描述
buffer The index of the buffer to check.

返回。

Buffer Type Constant

例子。

type = buffer_get_type(buff);

上述代码将从变量 "buff "中索引的值中获得buffer 的类型，并将其存储在一个变量中。
