buffer_sizeof

这个函数将返回给定的缓冲区数据类型常数（这里列出）的大小（以字节为单位）。

语法。

buffer_sizeof(type);

争论 类型 描述
type 缓冲区数据类型常数 The type of data that is to be checked (see the list of constants here).

返回。

Real

例子。

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

上面的代码首先通过无符号8位数据类型乘以12来计算要创建的buffer 的大小（因为我们将用buffer 来保存12个数据），然后用这个值来设置一个固定的buffer ，使之达到正确的大小。
