buffer_set_surface

通过这个函数你可以把信息从一个buffer 写到一个指定的表面。被读取的buffer 和被写入的表面都必须是以前创建的，你可以提供一个偏移量到buffer ，以便开始读取。请注意，阅读将总是从buffer 的开头加上偏移值开始，而不是从当前的寻找位置加上偏移值开始。

注意：从GameMaker的2.3.1版本开始，这个函数已经发生了变化。以前的版本使用常量 buffer_surface_copy ，现在已经废弃了。

语法。

buffer_set_surface(buffer, surface, offset);

争论类型 描述
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

返回。

例子。

buffer_set_surface(buff, application_surface, 0);

上面的代码将把存储在buffer ，在变量 "buff "中索引的所有数据复制到应用表面，没有偏移。
