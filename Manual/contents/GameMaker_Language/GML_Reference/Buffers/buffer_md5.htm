buffer_md5

在密码学中，MD5（Message-Digest algorithm 5）是一个广泛使用的密码散列函数，具有128位的哈希值，并被广泛用于各种安全应用。 它也常用于检查数据的完整性。这个函数将从一个给定的buffer （或部分buffer ）中获取输入数据，并返回对该数据唯一的32个字符的十六进制MD5散列。通过这种方式，你可以生成 一个安全密钥，它可以被存储并用于检查被发送到（或从）外部server （例如）的信息的完整性。buffers 当使用这个函数将其应用于buffer ，你必须指定要使用的buffer id，然后是一个偏移值（单位：%）。 使用，然后指定一个偏移值（以字节为单位）作为开始，然后指定一个大小（同样以字节为单位）作为要散列的区域。

注意：MD5不是完全安全的，可以被破解。更多信息请见本页面。

语法。

buffer_md5(buffer, offset, size);

争论类型 描述
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

返回。

例子。

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

上述代码将为存储在buffer ，由变量 "buff "索引的完整数据创建一个md5哈希值，并将返回的哈希值存储在变量 "check_string "中。
