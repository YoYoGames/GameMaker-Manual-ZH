clipboard_has_text

如果剪贴板包含文本，该函数将返回 true ，如果不包含则返回 false 。

语法。

clipboard_has_text();

返回。

Boolean

例子。

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

上述代码检查剪贴板中的文本，如果它包含任何文本，则将其作为string ，读入变量 "str"。最后，通过将剪贴板设置为一个空的string ，来清除剪贴板。

返回。弦乐
