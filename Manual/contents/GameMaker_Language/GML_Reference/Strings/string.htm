绳子

通过这个函数，你可以把任何实数变成string 。如果实数是一个整数，它将被保存为无小数位，否则，它将被保存为两个小数位。如果你需要更多的小数位，那么请使用函数 string_format().还要注意的是，在存储数组、数据结构或结构的变量上使用这个函数，将把这些变量的内容转换成string ，然后可以输出到控制台或保存到文件中，以便debugging 。

语法。

string(val);

争论 类型 描述
val 真实的 The real number to be turned into a string.

返回。

String

例子。

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

上面的代码使用 string 函数将实数和strings 一起画出来（draw只接受string 或实数，而不是两者）。

返回。弦乐
