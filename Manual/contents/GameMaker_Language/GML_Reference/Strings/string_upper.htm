通过这个函数，你可以强迫一个string ，只包含大写字母的字符。

注意该功能只检测A-Z的26个英文字母。

语法。

string_upper(string);

争论类型 描述
string The string to convert to uppercase.

返回。

例子。

str1 = "Hello World";
str2 = string_upper(str1);

上述代码将设置变量 str2 为 "HELLO WORLD"。
