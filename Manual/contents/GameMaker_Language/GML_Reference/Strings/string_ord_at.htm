弦外之音_at

你可以使用这个函数来返回一个string 中特定位置的特定字符代码，索引从1开始，为第一个字符。如果没有找到任何字符或者string 比给定的索引值短，则返回-1。

语法。

string_ord_at(str, index);

争论类型 描述
str The string to check.
index The position to get the character code from.

返回。

例子。

str = "Hello World";
char_code = string_ord_at(str, 7);

这将获得string "str "中第七个字符（其中 "H "算作第一个）的字符代码，并将其存储在变量 "char_code "中。

返回。弦乐
