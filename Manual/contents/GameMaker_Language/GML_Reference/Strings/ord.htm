条例

这个函数接受一个单一的字符输入string ，并返回该字符的Unicode（UTF8）值。请注意，当与 keyboard_check*函数时，输入的string 只能是一个字符的长度，并且只能是0到9的数字或A到Z的大写罗马字符。

语法。

ord(string);

争论 类型 描述
string 字符串 The string with which to find the Unicode value.

返回。

Real

例子。

if keyboard_check(ord("W"))
{
    y -= 4;
}

如果按住W键，这将使调用实例向上移动四个像素。

返回。弦乐
