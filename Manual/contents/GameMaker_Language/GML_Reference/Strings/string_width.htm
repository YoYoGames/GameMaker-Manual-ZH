弦的宽度

这个函数将返回输入string 的宽度（像素），同时考虑到文本可能有的断行。它对于根据组成字符串的字母的总宽度计算文本元素之间的距离非常方便。 因为它可以用 draw_text()使用当前定义的font 。

语法。

string_width(string);

争论类型 描述
string The string to measure the width of.

返回。

例子。

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

上面的代码将获得给定的string 的宽度，然后绘制两行文字，使用返回的string 宽度作为分隔符。

返回。弦乐
