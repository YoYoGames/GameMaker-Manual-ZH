符号(Ansi_char)

该函数返回一个string ，其中包含有原始BYTE值设置的字符。这不会也不应该被显示出来，但它会正确地保存在磁盘上，供编码时使用。

语法。

ansi_char(val);

争论类型 描述
val The raw byte value.

返回。

(Single character)

例子。

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

上述代码从原始字节数据中创建了一个string ，并将其写入一个（先前打开的）文件。

返回。弦乐
