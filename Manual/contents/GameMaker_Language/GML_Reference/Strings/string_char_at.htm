符号

你可以使用这个函数来返回一个string 中特定位置的特定字符，索引从1开始，为第一个字符。如果没有找到任何字符或者string 比给定的索引值短，则返回一个空的string ""，然而如果给定的索引等于或小于0，则返回string 的第一个字符。

语法。

string_char_at(str, index);

争论类型 描述
str The string to check.
index The position to get the character from.

返回。

例子。

str1 = "Hello World";
str2 = string_char_at(str1, 7);

这将把 str2 中的第七个字符（"H "算作第一个）设置为string str1 ，在这里是 "W"。

返回。弦乐
