字符串_format

将一个真实的数字变成一个string ，使用你自己的格式，你可以选择多少个 "位 "被保存到string ，以及多少个小数位也被保存。这两种方法都很方便，有些游戏喜欢将分数显示为一个固定的数字，而控制小数点后的位数可以达到很高的精确度，而小数点后的两位数则无法提供。 string()的两位小数不能提供。如果指定的位数大于要显示的数值和/或指定的数值加上小数点后的位数小于总的位数，那么在数值前会加上空格来补足差额（见下面的例子）。如果给定的数值小于总数和要包括的小数位数，小数点右边将添加零。默认的格式是左边没有额外的空格，右边只有两个小数位，例如 "265.73"。

语法。

string_format(val, tot, dec);

争论 类型 描述
val 真实的 The real number to be turned into a string.
tot 真实的 The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec 真实的 The number of decimal places to be included.

返回。

String

例子。

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

这将设置 str1 为 " 1234"， str2 为 " 3.1415926535"， str3 为 " 3.14159"。

返回。弦乐
