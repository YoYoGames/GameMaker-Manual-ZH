string_delete

这个函数用于删除给定的string 的一部分。你提供输入的string ，在该string 中要删除的字符的起始位置（索引从1开始）和要删除的字符数。该函数将返回一个没有该部分的新string 。

语法。

string_delete(str, index, count);

争论类型 描述
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

返回。

String

例子。

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

这将把 str2 "Hello World"，因为它删除了 "Hello "中多余的 "o"（存在于位置5）。

返回。弦乐
