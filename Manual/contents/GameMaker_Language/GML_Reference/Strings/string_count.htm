弦外之音_count

这个函数将返回给定子串在一个特定的string 。通过这种方式，你可以检查一个字母或一个短语在一段存储文本中重复出现了多少次。

语法。

string_count(substr, str);

争论类型 描述
substr The substring to check the string for.
str The string to check.

返回。

例子。

str1 = "Hello World";
ocount = string_count( "o", str1 );

这将把变量 ocount 设置为 str1 中的 "o "的数量，在这里是2。

返回。弦乐
