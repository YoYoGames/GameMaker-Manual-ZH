字符串_length

这个函数返回由一个给定的string 组成的字符数。它对一些事情很有用，比如什么时候限制一个自定义文本条目的字符长度（例如：将一个球员的名字限制在10个字符以内）。请记住，这与 string_width()不同的是，它测量的是string 中的字符数，而不是屏幕上显示的像素宽度。

语法。

string_length(string);

争论 类型 描述
string 字符串 The string to measure the number of characters of.

返回。

Real

例子。

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

这将检查姓名的string 是否大于十个字符，如果是，它将只复制和使用前十个字符。

返回。弦乐
