string_byte_length

该函数返回string 中的字节数。由于它们被保持为UTF8，这将不等于string 中的字符数。

语法。

string_byte_length(string);

争论 类型 描述
string 字符串 The string to measure the number of bytes of.

返回。

Real

例子。

bytesize = string_byte_length("Hello World");

这将把bytesize设置为 "Hello World "中的字节数。

返回。弦乐
