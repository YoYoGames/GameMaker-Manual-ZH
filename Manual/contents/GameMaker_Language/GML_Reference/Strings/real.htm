real

这个函数可以用来把一个给定的string ，变成一个实数。当使用这个函数时，string 中的数字、减号、小数点和指数部分会被考虑在内，而其他字符（如字母）将导致错误 被抛出。如果你知道或怀疑一个string 可能有其他字符，那么你可以使用 string_digits()来删除所有非数字字符，然后再使用这个函数将得到的字符串 变成一个实数。

语法。

real(string);

争论类型 描述
string The string to be converted to a real value.

返回。

例子。

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将采取输入string ，剥离除数字以外的所有字符，然后设置变量 "age"，以保持返回string 的真实数字值。

返回。弦乐
