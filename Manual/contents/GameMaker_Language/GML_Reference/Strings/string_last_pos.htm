弦乐_最后的位置

这个函数将返回一个string 中的子串实例的字符位置，从string 的末尾搜索到开头（因此与 string_pos().如果没有找到搜索的string ，该函数将返回0，如果找到，则返回搜索的string 的第一个字符的位置。请记住，对于传统的支持，strings 是从1开始索引的，所以1是string 的第一个位置，而不是你可能期望的0。

语法。

string_last_pos(substr, str);

争论类型 描述
substr The substring to look for in the string.
str The string.

返回。

例子。

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

上面的代码在变量 "text "中搜索string ，从string 的末尾开始搜索。如果找到一个，就在该位置插入子串 "name"。

返回。弦乐
