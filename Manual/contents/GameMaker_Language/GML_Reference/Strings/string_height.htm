弦乐_高鹰生殖中心

这个函数将返回输入string 的高度（像素），同时考虑到文本可能有的行间距和任何断行。在计算文本元素之间的距离时，它非常方便，因为它是根据组成 的字母中最高的那个来计算的。 组成的string ，因为它将使用当前定义的 。 draw_text()使用当前定义的font 。

语法。

string_height(string);

争论类型 描述
string The string to measure the height of.

返回。

例子。

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

上面的代码将获得给定的string 的高度，然后绘制两行文字，使用返回的string 高度作为分隔符。

返回。弦乐
