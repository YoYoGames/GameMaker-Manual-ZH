字符串

通过这个功能，你可以删除所有不属于字母的字符。这对于防止人们在文本输入中输入不需要的字符（如输入 "#"强制换行）是很方便的。

注意该功能只检测并返回A-Z（大写或小写）26个字母的英文字母。

语法。

string_letters(string);

争论类型 描述
string The string to convert to letters.

返回。

例子。

username = string_letters(username);

上面的代码将设置 "用户名 "变量，只保留原始string 的字母字符（如果原始string 是 "MacSw33ny"，返回的string 将是 "MacSwny"）。

返回。弦乐
