字符串_digits

你可以使用这个函数来解析一个给定的string ，并从中获得任何数字。例如，假设你有这样的文字--"我已经81岁了"。使用这个函数，你会得到一个 "81 "的返回string 。

语法。

string_digits(string);

争论类型 描述
string The string to get the digits from.

返回。

例子。

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将接收输入的string ，剥离除数字以外的所有字符，然后设置变量 age ，以保持返回的string 的实数值（因此，如果输入的string 是--例如--"我是18"，该 函数将返回 "18"）。

返回。弦乐
