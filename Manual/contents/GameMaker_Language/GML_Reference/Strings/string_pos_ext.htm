词条_position_ext

该函数将返回一个string 中子串实例的字符位置，从给定的起始位置开始向前搜索string 。如果没有找到搜索的string ，该函数将返回0，如果找到，则返回搜索的string 的第一个字符的位置。请记住，对于传统的支持，strings 是从1开始索引的，所以1是string 中的第一个位置，而不是你可能期望的0。

语法。

string_pos_ext(substr, str, start_pos);

争论类型 描述
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

返回。

例子。

if (string_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_pos_ext(",", text, 20));
}

上面的代码在存储在变量 "text "中的string ，寻找第20个字符后的逗号，如果找到，就在该位置插入子串 "name"。

返回。弦乐
