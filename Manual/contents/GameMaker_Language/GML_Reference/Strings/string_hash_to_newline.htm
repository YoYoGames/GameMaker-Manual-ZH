string_hash_to_newline

该函数返回一个string ，其中 "#"符号已被转换为一个新行。因此，一个string ，其格式为：

"Hello#World"

将被打印到屏幕上，成为。

Hello
World

重要的是!这个功能是为GameMaker和以前版本的软件之间的导入兼容性而提供的，因此，除非有必要复制过时的功能，否则不应使用这个功能。相反 你应该使用转义字符来标记一个新行。关于导入兼容性的更多信息，请看这里。

语法。

string_hash_to_newline(string);

争论类型 描述
string The string to convert over multiple lines

返回。

例子。

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

上面的代码将带有哈希符号的string ，转换成两行分割的string ，然后将其打印到屏幕上。

返回。弦乐
