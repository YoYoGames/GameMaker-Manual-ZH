string_lower

通过这个函数，你可以强制一个string ，使其只包含小写字符。

注意该功能只检测A-Z的26个英文字母。

语法。

string_lower(string);

争论类型 描述
string The string to convert to lowercase.

返回。

例子。

str1 = "Hello World";
str2 = string_lower(str1);

上述代码将设置 str2 为 "hello world"。
