clipboard_set_text

这个函数将设置剪贴板来容纳定义的string 。你可以把它设置为一个空的string ""，以有效地清除剪贴板上的文本。

语法。

clipboard_set_text(string);

争论 类型 描述
string 字符串 The text (as a string) to set the clipboard to hold.

返回。

N/A

例子。

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

上述代码检查剪贴板中的文本，如果它包含任何文本，则将其作为string ，读入变量 "str"。最后，通过将剪贴板设置为一个空的string ，来清除剪贴板。
