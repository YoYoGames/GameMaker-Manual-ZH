植入字符串

通过这个功能，你可以创建一个新的string ，由两个strings ，其中一个被插入到另一个的指定位置。例如，在预定义的文本中加入一个用户名，从而使你的游戏的玩家感觉到 更多的参与到行动中。在计算要插入的位置时要记住，strings 是从1开始索引的，所以string 中的第一个字符是1，而不是你可能期望的0。

语法。

string_insert(substr, str, index);

争论类型 描述
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

返回。

例子。

str2 = string_insert(username, "Hello, , how are you?", 8);

这将把 "用户名 "变量中的string 插入给定的短语中，产生的string 看起来像这样："你好，NAME，你怎么样？"。

返回。弦乐
