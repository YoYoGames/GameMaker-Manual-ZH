chr

该函数返回一个string ，其中包含与输入的Unicode代码有关的字符，用于显示。这个字符取决于当前绘制的fonts 字符集代码页，如果没有设置font ，它将使用机器的默认代码页。 机器的默认代码页。

语法。

chr(val);

争论类型 描述
val The Unicode code value to get the string from

返回。

例子。

mystring = chr(53) + chr(48);

这将把 mystring 设为 "50"（一个string ，不是一个整数），因为chr(53)是 "5"，chr(48)是 "0"。

返回。弦乐
