gc_get_target_frame_time

通过这个函数，你可以为垃圾收集器检索到当前的目标帧值。返回的值是以微秒为单位的(其中1,000,000微秒等于1秒），默认的目标帧时间是100微秒。如果你想改变这个值，那么你应该使用函数gc_target_frame_time() 。

语法。

gc_get_target_frame_time(time);

返回。

例子。

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

上面的代码检查了垃圾收集器的当前帧时间目标，如果它不是50微秒，那么它就被设置为这个值。
