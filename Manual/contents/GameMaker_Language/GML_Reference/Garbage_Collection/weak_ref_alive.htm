弱反射

通过这个函数，你可以检查一个struct 的弱引用，看它是否仍然 "活着"。你提供要检查的弱引用（如函数所返回的 weak_ref_create())，如果struct 仍然在某处被引用，该函数将返回 true ，如果不是，并且已经被垃圾回收，则返回 false 。注意，如果你提供的值不是一个弱引用，函数将返回 undefined 。

语法。

weak_ref_alive(weak_ref);

争论 类型 描述
weak_ref 结构弱化参考 The weak reference to the struct you want to check.

返回。

Boolean (or undefined)

例子。

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

上面的代码检查了对一个struct 的弱引用，如果它还活着，引用它的变量被设置为-1，取消对struct 的引用，使它能够被垃圾回收。
