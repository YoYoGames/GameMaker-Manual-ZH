gc_enable

通过这个函数你可以启用或禁用垃圾收集器。用 true 作为参数调用该函数可以启用它，用 false 可以禁用它（不推荐）。默认情况下，它是启用的。

语法。

gc_enable(enable);

争论类型 描述
enable enable (true) 或禁用 (false) 垃圾收集器。

返回。

例子。

if (global.debug == true)
{
    gc_enable(false);
}

如果给定的全局变量是 true ，上述代码将禁用垃圾收集。
