weak_ref_create

通过这个函数，你可以创建一个对struct 的弱引用，然后可以用来检查这个struct 在游戏中是否仍然 "活着"（被引用）。你提供你想跟踪的struct 的引用，该函数将返回另一个struct ，它是该struct 的弱引用。请注意，你可以通过使用函数 instanceof() 来检查一个引用是 "强 "还是 "弱"，因为一个强引用将返回 "struct" 或创建该struct 的构造函数的名称，如果是一个弱引用，则返回 "weakref"。还要注意的是，弱引用struct 将有一个变量 "ref" 可以被访问，以获得有关struct 的强引用，除非它已经被垃圾回收，在这种情况下它将返回 undefined 。

语法。

weak_ref_create(struct_to_track);

争论 类型 描述
struct_to_track 结构 The struct that you want to create a weak reference for

返回。

Struct Weak Reference

例子。

inventory_ref = weak_ref_create(inventory);

上面的代码创建了一个对struct 的弱引用，并将其存储在一个实例变量中供以后使用。
