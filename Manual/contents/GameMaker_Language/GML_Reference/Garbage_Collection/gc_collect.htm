采集

通过这个函数，你可以触发垃圾收集器，迫使它在当前帧（步）的末端运行。值得注意的是，垃圾收集器不 需要处于激活状态就可以工作。在禁用垃圾收集器后调用这个函数（使用函数 gc_enable() ）将启用垃圾收集器一帧，在垃圾收集器再次被禁用前，所有被标记为收集的objects ，将被从内存中删除。

语法。

gc_collect();

返回。

例子。

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

上面的代码检查了一个全局变量和一个被按下的键，如果这些是 true ，那么就会在帧（步骤）结束时触发垃圾收集。
