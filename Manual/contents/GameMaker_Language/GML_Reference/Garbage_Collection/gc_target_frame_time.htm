gc_target_frame_time

通过这个函数，你可以向垃圾收集器表明，它应该致力于使每一帧的运行时间不超过指定的时间。该函数接受一个以微秒为单位的时间值（1,000,000微秒等于1秒），默认的目标帧时间是100微秒。

请注意，这只是一个 目标值，因为有些操作仍然需要在一帧内完全运行，有时可能需要比目标时间更长的时间。还要注意的是，增加目标时间会使垃圾收集器对内存使用量的快速变化做出更多的反应，并会导致内存被更快地释放，尽管在实践中在大多数情况下不太可能需要这样做。还要注意的是，将目标帧时间设置为0并不会 导致收集器完全停止--它仍然会在每一帧做最小的工作。要完全禁用收集器，请使用 gc_enable() 函数。

语法。

gc_target_frame_time(time);

争论类型 描述
time The target time - in microseconds -  that the garbage collector should work each frame

返回。

N/A

例子。

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

上面的代码检查了垃圾收集器的当前帧时间目标，如果它不是50微秒，那么它就被设置为这个值。
