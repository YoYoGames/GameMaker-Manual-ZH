array_push

通过这个函数，你可以在不知道数组长度的情况下将一个（或多个）值推到数组的末端。该函数要求你提供一个存放数组的变量，以及至少一个要推送的值，尽管你可以选择提供更多的参数，它们都将以连续的顺序被推送到数组上。

语法。

array_push(variable, value, [value], [value], [etc...]);

争论 类型 描述
variable 阵列 The variable that holds the array.
value 变化的 The value to push onto the end of the array
[value], [value], [etc...] 变化的 OPTIONAL 要推送到数组中的更多数值

返回。

N/A

例子。

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

上面的代码将把四个值推到给定数组的末端。

返回。可变的功能
