is_method

这个函数可以用来检查，看一个变量是否是一个方法变量（它将返回 true ）或不是（在这种情况下，它将返回 false ）。

语法。

is_method(n);

争论 类型 描述
n 变化的 The variable to check.

返回。

Boolean

例子。

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

上面的代码检查了一个变量，看它是否是一个方法，如果函数返回 true ，那么就会向控制台输出一条调试信息。

返回。可变的功能
