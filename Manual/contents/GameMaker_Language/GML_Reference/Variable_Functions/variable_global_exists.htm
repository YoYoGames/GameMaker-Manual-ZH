变量_global_exists

通过这个函数你可以检查一个全局范围的变量是否存在。你提供要检查的全局变量名称为一个字符串（见下面的示例代码），如果一个给定名称的全局变量存在，该函数将返回 true ，否则返回 false 。

语法。

variable_global_exists(name);

争论 类型 描述
name 字符串 The name of the global variable to check for (as a string)

返回。

Boolean

例子。

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

上面的代码将检查名为 "enemy_num "的全局变量是否存在，如果不存在，将创建它并设置一个值。

返回。可变的功能
