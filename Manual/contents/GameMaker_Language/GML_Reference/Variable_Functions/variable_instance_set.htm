variable_instance_set

通过这个函数，你可以在一个实例中设置一个给定变量的值。你提供唯一的实例ID值（可以从room 编辑器中的实例属性中找到，或者在你调用该函数时被返回 instance_create_layer())以及要设置值的字符串变量的名称（见下面的示例代码），最后是要设置的值（可以是任何有效的数据类型）。如果该变量在实例中不存在，它将被创建，然后被赋值。

语法。

variable_instance_set(instance_id, name, val);

争论 类型 描述
instance_id 实例ID The unique ID value of the instance to use
name 字符串 The name of the variable to set (as a string)
val 变化的 The value to set the variable to

返回。

N/A

例子。

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

上面的代码将检查一个实例变量是否存在于调用的实例中，如果不存在，那么它将被创建并设置为0。

返回。可变的功能
