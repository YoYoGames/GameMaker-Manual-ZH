变量_实例_存在

通过这个函数你可以检查一个实例范围变量是否存在。你提供唯一的实例ID值（可以从room 编辑器中的实例属性中找到，或者在你调用该函数时返回 instance_create_layer())以及要检查的变量名称作为一个字符串（见下面的示例代码）。如果实例中存在一个具有给定名称的变量，该函数将返回 true ，否则返回 false 。

语法。

variable_instance_exists(instance_id, name);

争论 类型 描述
instance_id 实例ID The unique ID value of the instance to check
name 字符串 The name of the variable to check for

返回。

Boolean

例子。

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

上面的代码将检查运行代码的实例中是否存在名为 "shields "的变量，如果不存在，那么它将被创建并初始化为0。

返回。可变的功能
