变量_实例_获取名称

通过这个函数，你可以检索到一个数组，该数组中包含了一个实例的变量名称，或者一个游戏的全局变量。当你传入一个实例ID值时，数组中的每个条目将是一个string ，该变量名称对应于该实例中已经创建的实例范围变量。然而，如果你传入关键字 global ，数组中的每个条目将是对应于全局范围变量的变量名的string 。

语法。

variable_instance_get_names(instance_id/global);

争论 类型 描述
instance_id/global 实例ID或 global The unique ID value of the instance to check or the keyword global

返回。

Array (each entry is a string)

例子。

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

上面的代码将为运行代码块的实例检索一个所有实例范围变量的数组，然后在调试输出中显示这些变量及其值。

返回。可变的功能
