is_real

这个函数返回一个给定的变量是否是一个实数（单数、双数或整数）。在某些情况下，你想检查并查看一个变量是否持有一个实数，这时你就会使用这个函数。它不返回实数，而是返回 true 或 false ，因此，一个值，例如，"fish "将返回 false ，然而一个值为200将返回 true 。

注意：该函数将返回false 。 enum值，因为它们被存储为int64s 。检查该数据类型的正确函数是is_int64() 。

语法。

is_real(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if is_real(val)
{
    score += val;
}

上面的代码检查变量 "val "是否是一个实数，如果是，就把它加到分数上。
