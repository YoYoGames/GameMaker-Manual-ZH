is_int64

这个函数返回一个给定的变量是否是64位整数。在某些情况下，你想检查并查看一个变量在GameMaker中持有什么数据类型，这时你就会使用这个函数。它返回 true 或 false ，取决于该值是否是int64。

语法。

is_int64(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if !is_int64(val)
{
    show_debug_message("Not a 64bit integer!");
}

上述代码检查变量 "val "是否包含int64，如果不包含，则在调试控制台显示一条信息。

返回。可变的功能
