variable_instance_get

通过这个函数，你可以从一个给定的命名变量中获取数值。你提供唯一的实例ID值（可以从room 编辑器的实例属性中找到，或者在你调用该函数时返回 instance_create_layer())以及变量的名称来获取字符串的值（见下面的示例代码）。该函数将返回该变量的值，如果该变量不存在，则返回 undefined 。

重要的是!如果你得到的变量不存在，那么该函数将返回关键字 undefined ，你可能会得到错误，从而使游戏停止运行，所以如果有疑问，请使用函数 variable_instance_exists首先。

语法。

variable_instance_get(instance_id, name);

争论 类型 描述
instance_id 实例ID The unique ID value of the instance to use
name 字符串 The name of the variable to get (as a string)

返回。

Variable (any data type) or undefined (if the named variable does not exist)

例子。

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

上面的代码将检查一个变量是否存在，如果存在，那么它的值将被检索并存储在一个局部变量中。如果它不存在，本地变量将被设置为-1。

返回。可变的功能
