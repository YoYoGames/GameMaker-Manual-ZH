变量_结构_获取

通过这个函数，你可以在一个struct ，从一个给定的命名变量中获取数值。你提供struct ，以及要获取数值的字符串的变量名称（见下面的示例代码）。该函数将返回该变量所持有的值，如果命名的变量不存在，则返回 undefined 。

重要的是!如果你得到的变量不存在，那么该函数将返回关键字 undefined ，你可能会得到错误，从而使游戏停止运行，所以如果有疑问，请使用函数 variable_struct_exists()首先。

语法。

variable_struct_get(struct, name);

争论类型 描述
struct The struct reference to use
name The name of the variable to get (as a string)

返回。

Variable (any data type) or undefined (if the named variable does not exist)

例子。

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

上面的代码将检查一个变量是否存在于给定的struct ，如果存在，那么它的值将被检索并存储在一个局部变量中。如果它不存在，则本地变量被设置为-1。

返回。可变的功能
