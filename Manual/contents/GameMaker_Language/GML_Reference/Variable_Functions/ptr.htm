这个函数将尝试把一个给定的值转换成一个指针数据类型，其中该值必须是一个 real ，一个 string ，一个 int64 ，一个 int32 ，或者一个 ptr 。其他的都会导致游戏崩溃并出现错误信息。

语法。

ptr(n);

争论 类型 描述
n 实数，字符串，或指针 The value to convert.

返回。

Pointer

例子。

if !is_ptr(val)
{
    val = ptr(application_surface);
}

上面的代码检查变量 "val "是否包含一个指针，如果不包含，就给它分配一个指针。

返回。可变的功能
