is_array

这个函数可以用来检查并查看一个变量是否持有一个数组（它将返回 true ）或不持有（在这种情况下，它将返回 false ）。

语法。

is_array(n);

争论 类型 描述
n 变化的 The variable to check.

返回。

Boolean

例子。

if is_array(a)
{
    a = -1;
}

上面的代码检查一个变量，看它是否是一个数组，如果函数返回 true ，则通过将该变量设置为-1来删除该数组。

返回。可变的功能
