变量函数

本页中的函数是为了在你的游戏中使用GameMaker Language ，处理所有不同的变量和变量类型。关于变量的更多信息，请参见 "变量和变量范围"页面。

可使用以下变量功能。

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • 变量_结构_删除
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d（已废弃）。
  • array_length_2d(已弃用)
  • array_height_2d（已废弃）。
