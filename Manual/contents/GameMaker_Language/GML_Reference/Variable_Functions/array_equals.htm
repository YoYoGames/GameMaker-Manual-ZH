array_equals

通过这个函数，你可以检查两个数组是否相等（相等或相同）。你给出要检查的两个数组，如果它们相等（相等或相同），该函数将返回 true ，如果不相等，则返回 false 。请注意，这与使用==检查两个数组是否相同不同，后者不会检查两个数组是否持有相等的值，而只是检查两个数组是否引用了相同的初始数组。比如说

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

注意，有一些常数可能不等于自己，这可能使这个函数失败。下面是一个例子。

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

更多信息见平等表。

语法。

array_equals(var1, var2);

争论 类型 描述
var1 阵列 The first array.
var2 阵列 The second array.

返回。

Boolean

例子。

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

上面的代码将检查两个数组，看它们是否持有相等的值，如果没有，那么代码将把数组 "inventory_array "的全部内容复制到数组 "item_array "中。

返回。可变的功能
