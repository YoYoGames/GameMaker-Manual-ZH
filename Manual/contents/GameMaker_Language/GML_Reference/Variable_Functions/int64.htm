int64

这个函数将尝试把一个给定的值转换成一个有符号的64位整数，其中该值必须是一个实数、一个string 、一个 int64 、一个 int32 、或一个 ptr 。其他任何东西都会导致游戏崩溃并出现错误信息。你可以使用函数来检查一个变量是否持有一个int64。 is_int64().

语法。

int64(val);

争论 类型 描述
val 真实, 实数，或 指针 The value to convert.

返回。

int64 (signed 64-bit integer)

例子。

steam_handle = int64(global.fileReadString);

上面的代码将全局变量中的值转换为64位整数。

返回。可变的功能
