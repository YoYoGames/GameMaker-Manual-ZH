变量_结构_删除

通过这个函数，你可以从一个struct 中删除一个变量。你提供要删除变量的struct ID和要删除的变量的string 名称。

语法。

variable_struct_remove(struct, name);

争论 类型 描述
struct 结构 The struct reference to remove the variable from
name 字符串 The name of the variable to remove (as a string)

返回。

N/A

例子。

if variable_struct_exists(mystruct, "shields")
{
    variable_struct_remove(mystruct, "shields");
}

上面的代码将检查给定的变量是否存在于给定的struct ，如果存在，则将其删除。

返回。可变的功能
