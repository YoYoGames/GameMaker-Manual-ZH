method_get_index

通过这个函数，你可以检索到定义该方法的script 的脚本索引。如果该方法没有在script ，那么该函数将返回-1，否则它将返回script 的索引值。

语法。

method_get_index(method);

争论 类型 描述
method 方法 The method variable to check

返回。

Script Asset or Real

例子。

var _index = method_get_index(light_setup);
if _index != -1
{
    show_debug_message(script_get_name(_index));
}

上面的代码将检查script 方法 "light_setup "的索引，然后如果它不是-1，它将向控制台输出script 名称。

返回。可变的功能
