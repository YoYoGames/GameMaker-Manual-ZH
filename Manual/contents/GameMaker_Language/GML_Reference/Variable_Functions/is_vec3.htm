is_vec3

在某些情况下，你想检查并查看一个变量在GameMaker中持有什么数据类型，这时你会使用这个函数。它返回 true 或 false ，这取决于该值是否是一个3值向量。

语法。

is_vec3(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if !is_vec3(val)
{
    show_debug_message("Not a 3 value vector!");
}

上述代码检查变量 "val "是否包含vec3，如果不包含，则在调试控制台显示一条信息。

返回。可变的功能
