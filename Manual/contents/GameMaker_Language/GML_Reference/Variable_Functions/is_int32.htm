is_int32

这个函数返回一个给定的变量是否是32位整数。在某些情况下，你想检查并查看一个变量在GameMaker中的数据类型，这时你会使用这个函数。它返回 true 或 false ，取决于该值是否是int 32。

语法。

is_int32(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if !is_int32(val)
{
    show_debug_message("Not a 32 bit integer!");
}

上面的代码检查变量 "val "是否包含int32，如果不包含则在调试控制台上显示一条信息。

返回。可变的功能
