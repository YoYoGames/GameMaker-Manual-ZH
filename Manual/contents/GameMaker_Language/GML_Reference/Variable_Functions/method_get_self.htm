method_get_self

通过这个函数，你可以检索实例ID或struct 引用，这是调用方法时使用的上下文。 self当方法被调用时使用的上下文。如果该变量不是一个方法，那么该函数将返回 undefined 。请注意，该函数也可能返回常量 pointer_null ，在这种情况下，调用时使用的是当前的自我。

语法。

method_get_self(method);

争论类型 描述
method The method variable to check.

返回。

Instance ID, Struct, undefined, or pointer_null

例子。

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

上面的代码获得了给定方法变量的自我上下文并将其输出到控制台。

返回。可变的功能
