变量结构设置

通过这个函数，你可以在struct 中设置一个给定的变量的值。你提供struct 引用以及要设置值的字符串的变量名称（见下面的示例代码），然后最后是要设置的值（可以是任何有效的数据类型）。如果该变量在struct 中不存在，它将被创建，然后被赋值。

语法。

variable_struct_set(struct, name, val);

争论 类型 描述
struct 结构 The struct reference to set
name 字符串 The name of the variable to set (as a string)
val 变化的 The value to set the variable to

返回。

N/A

例子。

if !variable_struct_exists(mystruct, "shields")
{
    variable_struct_set(mystruct, "shields", 0);
}

上面的代码将检查给定的变量是否存在于给定的struct ，如果不存在，那么它将被创建并设置为0。

返回。可变的功能
