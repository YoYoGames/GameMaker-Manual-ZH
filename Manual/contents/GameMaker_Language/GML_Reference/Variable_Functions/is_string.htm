is_string

这个函数返回一个给定的变量是否是string 数字。在某些情况下，你想检查并查看GameMaker中的一个变量是否持有一个string ，而不是一个实数，这时你就会使用这个函数。它不返回string ，而是返回 true 或 false ，因此，例如n的值为 "fish"，将返回 true ，但n的值为200将返回 false 。

语法。

is_string(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if is_string(val)
{
    name = "Player: " + val;
}

上面的代码检查变量 "val "是否包含一个string ，如果包含，则将其添加到另一个string ，然后分配给变量 "name"。

返回。可变的功能
