variable_global_get

通过这个函数，你可以从一个指定的全局变量中获取数值。你提供全局变量的名称，以获取其值的字符串（见下面的示例代码），该函数将返回全局变量所持有的值，如果该变量不存在，则 undefined 。

重要的是!如果你得到的全局变量不存在，那么该函数将返回关键字 undefined ，你可能会得到错误，从而使游戏停止运行，所以如果有疑问，请使用该函数 variable_global_exists首先。

语法。

variable_global_get(name);

争论类型 描述
name The name of the global variable to get (as a string)

返回。

Variable (any data type) or undefined (if the named variable does not exist)

例子。

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

上面的代码将检查一个全局变量是否存在，如果存在则输出到控制台。

返回。可变的功能
