is_bool

这个函数返回一个给定的变量是否是布尔值（true ior false ）。在某些情况下，你想检查并查看GameMaker中的一个变量是否持有布尔值，这时你会使用这个函数。

语法。

is_bool(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

上面的代码检查变量 "val "是否是一个实数，如果是，就用它来设置一个全局变量。

返回。可变的功能
