is_struct

这个函数检查所提供的值是否是一个struct 。如果是，则返回 true ，否则返回false 。

请注意，方法变量也将返回 true ，而对象实例将返回 false 。

语法。

is_struct(val);

争论 类型 描述
val 变化的 The value to check.

返回。

Boolean

例子。

if is_struct(a)
{
    delete(a);
}

上面的代码检查一个变量是否是struct ，如果函数返回 true ，则删除struct 。

返回。可变的功能
