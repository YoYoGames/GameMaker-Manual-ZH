array_set

通过这个函数，你可以将一个数组中的一个索引的值设置为一个值。该函数要求你提供一个存放数组的变量，以及要设置的索引和要设置的值。这个函数也可以用于多维数组，只要你在提供数组索引时指定你要设置的维度，遵循这个模式。

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

语法。

array_set(variable, index, value);

争论 类型 描述
variable 阵列 The variable that holds the array.
index 真实的 The index of the array element to set the value for.
value 变化的 The value to set.

返回。

N/A

例子。

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

上面的代码将把给定数组中的前10项设置为一个值。

返回。可变的功能
