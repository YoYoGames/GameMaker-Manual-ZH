typeof

该函数将任何给定变量的数据类型作为string 。可能的返回值列于下表。

字符串 描述
"number" 该变量持有一个实数（浮点数）--这可以包括 NaN 和 infinity
"string" 该变量持有一个字符串
"array" 该变量引用了一个数组
"bool" 该变量持有一个布尔值 (true / false)
"int32" 该变量持有一个32位的整数
"int64" 该变量持有一个64位的整数
"ptr" 该变量持有一个指针
"undefined" 该变量未被定义
"null" 该变量持有一个空值（这在正常情况下不应该出现）。
"vec3" 该变量持有一个3值向量
"vec4" 该变量持有一个4个值的向量
"method" 该变量持有一个函数引用
"struct" 该变量持有一个struct 参考
"ref" 该变量持有一个实例引用
"unknown" 值是未知的。这不应该被看到，它标志着在最基本的层面上出了问题，如内存被覆盖。

请注意，在有些情况下，这个函数可能不会为一个方法返回正确的值。考虑以下两个函数的定义。

a = function()
{
    // something
}

function b()
{
    // Something
}

b 从技术上讲，这两个都被认为是方法，因为它们是将一个函数绑定到一个实例范围的变量上，然而在函数 typeof() ，将返回 "数字 "而不是 "方法"，而在 a ，将返回 "方法"。这是由于像为 b 创建的方法被分配了script 索引（是整数值），因为这是编译器识别脚本函数的方式，即使该函数没有定义在script 。

语法。

typeof(variable);

争论 类型 描述
variable 任何 The variable to get the data type of.

返回。

String (see table above)

例子。

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

上面的代码得到了给定的全局变量所持有的数据类型，并将string ，然后用于向控制台输出一条信息。

返回。可变的功能
