instanceof

这个函数可以用来获取用于创建struct 的构造函数的名称。struct 本身应该是用 new操作符创建的，同时也是构造函数本身。你提供带有struct 引用的变量来检查，该函数将返回一个带有构造函数名称的string 或 undefined 。

请注意，如果你向该函数传递一个struct 字面（即：一个没有使用构造函数而创建的struct ），那么它将简单地返回string "结构"。

这个函数也可以用来检查一个struct 引用是否是一个 弱引用或不是，在这种情况下，该函数将返回string "weakref"，而不是创建该struct 的函数的名称。更多信息，请参阅函数 weak_ref_create().

如果构造函数不是在一个script 中创建的，返回的构造函数名称将包括额外的文本。如果它是在一个object 中创建的，它的名称将包括object 和创建的事件。例如，在Object1的Create事件中创建的名为test_constructor的构造函数将被返回为"test_constructor_gml_Object_Object1_Create_0"。为了避免这种情况，请将构造函数创建在 script.

语法。

instanceof(struct_id);

争论 类型 描述
struct 结构 The struct reference to use.

返回。

String or undefined

扩展实例。

在这个例子中，我们必须首先定义将被用作我们struct 的构造函数。下面这个函数是在script 资产中定义的。

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

然后，这个函数可以和 new 操作符一起使用，以创建一个struct ，并将变量设置为函数中使用的参数值来填充它。

mystruct = new init_struct(10, 100, "Hello World");

然后我们可以像这样获得所使用的构造函数的名称。

var _name = instanceof(mystruct);
if (is_string(_name))
{
    show_debug_message(_name);
}

这将在输出日志中打印string "init_struct"，这是构造函数的名称，定义在其script 。
