is_infinity

这个函数返回一个给定的变量是否是 infinity （一个无限的数字），如果是则返回 true ，如果不是则返回 false 。

语法。

is_infinity(n);

争论 类型 描述
n 变化的 The argument to check.

返回。

Boolean

例子。

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

上面的代码检查全局变量 "value"，看它是否是无限的，如果是，则显示一个调试信息。

返回。可变的功能
