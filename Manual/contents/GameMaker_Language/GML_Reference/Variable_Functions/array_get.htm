array_get

通过这个函数，你可以从一个数组中的一个索引中获取数值。该函数要求你提供一个保存数组的变量和从数组中获取数值的索引。注意，如果给定的数组索引超出了范围，那么游戏将崩溃并出现错误。这个函数也可以用于多维数组，只要你在提供数组索引时指定你想得到的维度，按照这个模式。

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

语法。

array_get(variable, index);

争论 类型 描述
variable 阵列 The variable that holds the array.
index 真实的 The index of the array element to get the value from.

返回。

Variable (Any valid data type that an array can hold)

例子。

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

上面的代码将把给定的前10项输出到控制台。

返回。可变的功能
