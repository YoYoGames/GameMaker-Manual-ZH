variable_global_set

通过这个函数，你可以设置一个给定的全局变量的值。你提供要设置值的全局变量的名称作为一个字符串（见下面的示例代码），然后提供要设置的值（可以是任何有效的数据类型）。如果该全局变量在游戏中不存在，它将被创建，然后被赋值。

 

语法。

variable_global_set(name, val);

争论 类型 描述
name String The name of the global variable to set (as a string)
val Any The value to set the global variable to

 

返回。

N/A

 

例子。

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

上面的代码将检查一个全局变量是否存在，如果不存在，那么它将被创建并设置为0。
