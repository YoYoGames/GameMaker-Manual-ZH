HTTP

本节列出了GameMaker中所有不同的异步HTTP函数。这些函数将在所有拥有它的实例中产生一个异步HTTP事件。

  • http_request
  • http_get
  • http_get_file
  • http_post_string

请注意，由于跨域安全问题，上述 http_* 功能可能无法发挥预期作用。这意味着对你的server 的请求或从跨域加载资源的尝试会被阻止，并可能出现返回空白结果或404错误。你可以绕过这个问题的方法之一是有一些server 边的PHP，允许某些域访问你的server （这也是一种保护你的资源和阻止不包括在PHP允许列表中的servers 的方法）。

下面是一个可以用于此的PHP代码的例子。

$http_origin = $_SERVER['HTTP_ORIGIN'];
if ($http_origin == "http://127.0.0.1:51268")
{
    header("Access-Control-Allow-Origin: *");
}

对于确定或设置其跨源类型很重要的图像加载请求，存在以下功能。

  • http_get_request_crossorigin
  • http_set_request_crossorigin

在大多数情况下，应该不需要使用这些功能，但如果游戏存储在一个安全的server --其中某些assets 可能需要基本认证才能访问，并且在加载时产生安全错误--将图像请求的跨源类型设置为 "使用认证 "可能是必要的（与默认的 "匿名 "设置相反）。
