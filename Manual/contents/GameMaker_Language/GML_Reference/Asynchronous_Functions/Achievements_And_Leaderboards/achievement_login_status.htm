成就_登录_状态

这个函数目前只对Android 平台上的Google Play有用。它将检查用户是否已经登录到谷歌服务，在这种情况下，它将返回真，或者不返回，返回假。如果它返回 true，就不需要调用这个 achievement_login()函数。

语法。

achievement_login_status();

返回。

Boolean

例子。

if !achievement_login_status()
    {
    achievement_login();
    }

上述代码将使你登录到所选择的排行榜和成就服务。

返回。成就和排行榜
