成就重置

这个函数将重置所有成就，使其回到游戏的初始值。这个函数是作为一个调试函数提供的，不建议你允许最终用户在你的游戏中这样做。

语法。

achievement_reset()

返回。

N/A

例子。

if debug_mode
    {
    if keyboard_check_pressed(vk_space)
        {
        achievement_reset()
        }
    }

上面的代码检查游戏是否处于调试模式，如果是的话，按 "空格 "会重置所有的成就。
