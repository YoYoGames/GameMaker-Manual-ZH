成就_帖子

你可以使用这个函数将你的成就发送到选定的排行榜和成就服务中。在Android ，你需要将成就的ID作为string （这是你设置成就时分配的唯一成就ID）。 而在所有其他平台上，你需要提供成就名称（string ），然后给出你为获得该成就而完成的百分比（0-无，100-完成）。该函数将触发一个社交异步事件，返回的 异步加载DS地图将有以下键/值对。

  • "类型" - 这是已经触发的事件的类型，这将是这个函数的string "achievement_post_result"。
  • "id" - 这是事件的ID，对于这个函数，它将返回一个GML 常数GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT。
  • "achievement_id"--这是被更新的成就的唯一ID。
  • "status" - 如果成就被成功更新，这将是1，如果请求失败，则是0。

语法。

achievement_post(achievement, percent)

争论 描述
achievement 成就的名称/ID。
percent 成绩完成的百分比。

返回。

N/A

例子。

if achievement_available()
    {
    achievement_post(global.Achievement[0], 100);
}

上述代码检查所选择的排行榜和成就服务是否可用，如果可用，则发布一个成就。

返回。成就和排行榜
