成绩_增加值

游戏可以有没有完成值的成就，所以你可以用这个函数把这些成就增加一个给定的数量。在Android ，你将成就的ID作为string （这是在设置成就时分配的唯一成就ID）。 而在所有其他平台上，你要把定义好的成就名称作为string ，然后给出要递增的实际成就值。

该函数将触发一个社交异步事件，返回的 async_load DS地图将有以下键/值对。

  • "类型" - 这是已经触发的事件的类型，这将是这个函数的string "achievement_increment_result"。
  • "id" - 这是事件的ID，对于这个函数，它将返回一个GML 常数GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT。
  • "achievement_id" - 这是成就的唯一ID，已经被增加。
  • "status" - 如果成就被成功增加，这将是1，如果请求失败，则是0。
  • "完成" - 如果这个增量的成就已经完成，这将是1，如果请求失败，这将是0。

语法。

achievement_increment(name, value)

争论 描述
name 成就的名称/ID。
value 要作为增量发送的值。

返回。

N/A

例子。

achievement_increment(global.Achievement[0], score);

上面的代码将当前玩家的分数添加到成就中，其ID存储在全局数组中。

返回。成就和排行榜
