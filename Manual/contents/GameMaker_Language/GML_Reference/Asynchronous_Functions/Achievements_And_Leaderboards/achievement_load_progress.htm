成就_加载_进度

这个函数将向server 发送请求，以获取所有可用成就的信息。它将触发一个回调社会异步事件，其中包括 的 异步_加载地图填充了相关的键/值对。这个DS地图的id键被用来识别正确的回调（可以有一个以上的触发函数）。 对于任何给定的异步事件，可以有一个以上的触发函数），并将与常数 achievement_info以及每个玩家的一些其他键/值对配对。该地图的具体内容 的具体内容如下所示。

注意： 该功能仅适用于iOS。

  • "id" - 对于这个函数，它应该是achievement_achievement_info。
  • "numentries" - 列表中成就的数量。
  • "AchN"--成就的名称，其中 "N "是一个整数，对应于它在条目列表中的位置。
  • "AchCompN" - 成就 "N "的完成程度，百分比值从0到100（一个string ）。

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_achievement_info
