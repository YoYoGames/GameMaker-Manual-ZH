淘宝网上的评价

这个功能打开一个依赖于操作系统的对话框，你可以要求用户在一个特定的页面上发表评价或评论。你可以定义出现在对话中的文本，以及你希望出现在两个按钮上的文本和必须张贴评论的URL。

注意：该功能仅适用于Android 和iOS 目标。

重要的是!在iOS上，当使用TestFlight时，在开发过程中会显示提示，但你不能实际做任何事情。一旦你的应用进入App Store，这个按钮就会被激活，用户就可以选择写评论或留下评价。注意，如果你通过TestFlight发布游戏，这个窗口甚至可能不会显示。

语法。

shop_leave_rating(text, yes_string, no_string, url)

争论 类型 描述
text 字符串 Text that appears on the dialog.
yes_string 字符串 Text that appears on the "yes" button.
no_string 字符串 Text that appears on the "no" button.
url 字符串 The full URL where the comment has to be sent.

返回。

N/A

例子。

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

上面的代码将要求用户在变量 "计时器 "倒计时到0的时候留下评论。
