云储蓄

在制作游戏时，通常需要将游戏状态的信息存储在某种类型的文件中，但将这些信息存储在设备上并不总是最好的选择，因为如果玩家删除了游戏，然后重新安装，这些信息可能会丢失。为了防止这种情况，你可以使用各种不同的云服务，这些服务通过互联网提供数据存储，以便在任何时候检索和修改。你应该注意，这个功能仅限于每个游戏的一个单一数据 "blob"，所以每次你向云服务发送一个新的保存，无论是string ，还是一个文件，它将覆盖任何先前存储的数据。

注意 目前只有安卓目标使用这些功能，你需要在Android 游戏选项中的社交部分勾选启用谷歌云存储复选框。这将提示你下载谷歌服务 extension ，这是在Android 上进行云存储的必要条件。关于云端保存的更多信息，请参见以下帮助台文章。安卓云存储

GameMaker通过一些简单的函数来支持这一点，这些函数与异步云事件一起工作。这些功能在下面几页中进行了解释。

  • cloud_synchronise
  • cloud_string_save
  • cloud_file_save
