angle_difference

这个函数将返回两个指定角度之间的最小差值，其中的差值是从源角度向目的角度计算的。返回的值将在-180和180度之间（其中正的角度是逆时针，负的角度是顺时针）。

源 "和 "目的 "角度之间的关系是这样的：你通过调用angle_difference(dest, src) 得到的差值，再加上src 的值，就得到了目的角度（尽管具体数值可能不同）。你可以使用类似的技术将源角度逐步向目的角度移动，比如说，你将返回的差值的10%加到src上，使其一点点向目的角度移动（这在下面的例子中有所演示）。

语法。

angle_difference(dest, src)

争论类型 描述
dest The destination or target angle
src The source angle

返回。

例子。

var _dir = point_direction(x, y, mouse_x, mouse_y);
var _diff = angle_difference(_dir, image_angle);
image_angle += _diff * 0.1;

上面的代码将得到从实例到鼠标光标的方向，然后得到该角度与当前角度的差值。 image_angle的差值，用这个值来逐步将实例向鼠标方向旋转。
