达尔鑫

返回x的反正弦，在如果 dsin(x)=n ， darcsin(n)=x ，得到的数字将在-90和90之间。

注意：这只接受-1和1之间的数字（其他的都会出错）。

注意：这个函数返回的是度数，而不是弧度。

语法。

darcsin(x);

争论类型 描述
val The value to return the inverse sine of.

返回。

例子。

val = darcsin(-1);

这将把 "val "设置为-90。
