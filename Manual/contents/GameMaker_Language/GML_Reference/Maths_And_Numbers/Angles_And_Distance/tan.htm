棕褐色

在一个直角三角形中，tan(val)=对角/邻角，其中val是三个角中的一个。

注意 这个函数的取值单位是弧度而不是度。

注意：Pi/2, (3 pi/2), -pi/2和其他大量的数值会因为返回无穷大而与这个函数产生误差，图形表示会在这些数值上产生渐近线。

语法。

tan(val);

争论类型 描述
val The angle (in radians) to return the tangent of.

返回。

例子。

val = tan( pi );

这将把val设置为0。

返回。角度和距离
