辩证法

返回x的反切值。这将接受任何数字，因为与 tan() 不同， arctan() 渐近线在y轴上，所以它只是意味着你永远不会得到返回的数字 大于pi/2或小于-pi/2的数字。

注意：这个函数的取值单位是弧度，而不是度。

语法。

arctan(x);

争论类型 描述
x The angle (in radians) to return the inverse tangent of.

返回。

例子。

val = arctan( 0 );

这将把val设置为0。
