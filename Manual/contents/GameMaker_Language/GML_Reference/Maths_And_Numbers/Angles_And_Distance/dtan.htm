dtan

在一个直角三角形中，dtan(val)=对角/邻角，其中 "val "是三个角中的一个角。

注意：大量的数值（例如90，或-90）会因为返回无穷大而在这个函数中出错，这种情况的图形表示会在这些数值上产生渐近线。

注意：这个函数的取值单位是度，而不是弧度。

语法。

dtan(val);

争论 类型 描述
val 真实的 The angle (in degrees) to return the tangent of.

返回。

Real

例子。

val = dtan(45);

这将设置 "val "为1。

返回。角度和距离
