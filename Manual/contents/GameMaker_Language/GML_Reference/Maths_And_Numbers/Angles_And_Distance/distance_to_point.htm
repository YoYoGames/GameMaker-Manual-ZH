到点的距离

这个函数计算从调用实例的边界框的边缘到room 中指定的x/y位置的距离，返回值为像素。注意，如果调用的object ，没有定义sprite ，或者没有定义掩码，结果将是不正确的。

语法。

distance_to_point(x, y);

争论类型 描述
x The x position to check.
y The y position to check.

返回。

例子。

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    canshoot = true;
}

上面的代码将检查到播放器object x/y位置的距离，如果它小于存储在变量 "范围 "中的值，变量 "canshoot "将被设置为真。
