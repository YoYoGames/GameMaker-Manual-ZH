达尔科斯

返回x的反余弦，在如果 dcos(val)=n ， darccos(n)=val ，得到的数字将在180和0之间。

注意：这只接受-1和1之间的数字（其他的都会出错）。

注意：这个函数返回的是度数，而不是弧度。

语法。

darccos(val)

争论类型 描述
x The value to return the inverse cosine of.

返回。

例子。

val = arccos(-1);

这将设置 "val "为180。
