distance_to_object

这个函数计算从调用实例的边界框的边缘到指定的最近的实例的最近边缘的距离object 。object 可以是一个object 索引，也可以是一个特定的实例ID，以及 关键字另一个，并且距离是以像素为单位返回的。注意，如果objects 中没有定义sprite 或没有定义掩码，结果将是不正确的。

语法。

distance_to_object(obj);

争论 类型 描述
obj 对象资产或实例ID The object to check for.

返回。

Real

例子。

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

上面的代码将检查到播放器的距离object ，如果它小于存储在变量 "范围 "中的值，变量 "canshoot "将被设置为真。

返回。角度和距离
