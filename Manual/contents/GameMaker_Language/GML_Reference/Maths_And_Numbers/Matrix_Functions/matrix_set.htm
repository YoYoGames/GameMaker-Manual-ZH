Matrix_set

这个函数接收一个由16个值组成的数组，对应于给定的4x4矩阵类型，其中元素[0-3]将是第1行，元素[4-7]将是第2行，等等。(见主页上的图片)。你可以用以下方法创建这样的矩阵 matrix_build()或 matrix_get()函数来创建这样的矩阵，或者干脆自己建立一个数组并将其传入函数。可用的矩阵类型有视图、投影和世界，对于这些类型，你可以使用以下常数之一。

恒定 描述
matrix_view 当前视图矩阵
matrix_projection 当前的投影矩阵
matrix_world 当前的世界矩阵

语法。

matrix_set(type, matrix);

争论 类型 描述
type 矩阵类型常数 The type of matrix to get the values of (see the constants listed above)
matrix 矩阵阵列 The matrix data as an array

返回。

N/A

例子。

matrix_set(matrix_world, m_array);

上述代码将把当前世界矩阵的值设置为存储在数组矩阵 "m_array "中的值。
