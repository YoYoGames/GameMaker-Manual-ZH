矩阵功能

矩阵是一个数字的集合，排列成固定数量的行和列。通常这些数字是实数，但也不一定是这样。下图大致显示了一个四行四列的矩阵的构造。

最上面的一行是第1行，最左边的一列是第1列，这个矩阵是一个4x4矩阵，因为它有四行和四列（其他大小的矩阵也可以构造）。在描述矩阵时，格式总是行/列，组成矩阵的每个数字都被称为矩阵的一个元素。矩阵中的元素有特定的位置，由其行和列的位置描述，其中矩阵的左上角是第1行第1列。在上面的矩阵例子中，第1行第1列的元素是数值 "1"。第2行第3列的元素是数值 "4.8"。

矩阵对很多事情都非常重要，但在处理三维空间时用得最多，例如，它们可以用来设置摄像机视角，或者用来平移或转换模型。以下是GameMaker中处理矩阵的函数。
