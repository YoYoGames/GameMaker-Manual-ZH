矩阵_堆栈_设置

这个函数用指定的矩阵覆盖了当前矩阵栈的顶部。

语法。

matrix_stack_set(matrix);

争论类型 描述
matrix The matrix index to use.

返回。

例子。

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

上面的代码将建立一个新的矩阵，并将产生的矩阵索引存储在变量 "m "中，然后用它替换矩阵栈的顶部。
