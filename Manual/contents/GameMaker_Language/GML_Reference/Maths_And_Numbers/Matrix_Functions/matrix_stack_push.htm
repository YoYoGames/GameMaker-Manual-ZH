matrix_stack_push

这个函数用来将一个新的矩阵推到当前矩阵栈的顶部。它首先会将给定的矩阵与当前堆栈顶部的矩阵相乘，然后将所得的矩阵推到堆栈中。

这个功能对于将多个矩阵转换应用于你的视觉效果非常有用，而不需要手动将各种矩阵相乘。

例如，你可以有一个基本的矩阵被推到堆栈中，将图形绘制到某个矩形区域（例如，一个窗口）。在使用该矩阵绘制一些图形后，你可以将另一个矩阵推入堆栈，用于在一个子区域内绘制图形（如窗口中的一个按钮）。在那个子区域画完东西后，你可以调用 matrix_stack_pop() ，把它的矩阵从堆栈中移除，继续在主窗口区域画视觉效果。

语法。

matrix_stack_push(matrix);

争论类型 描述
matrix The matrix to push to the stack

返回。

例子。

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

上面的代码将建立两个新的矩阵，然后把它们推到矩阵栈中。然后世界矩阵被设置到堆栈的顶部，画出一个sprite ，然后从堆栈中弹出变换。
