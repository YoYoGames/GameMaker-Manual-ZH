date_is_today

如果给定的日期值是被检查的日子（即：今天），该函数将返回 true ，否则返回 false 。对于游戏中的复活节彩蛋或解锁季节性内容来说，这是一个很方便的功能。请注意，这个函数将受到所设置的时区的影响（默认为当地时间），你可以用 date_set_timezone()函数来改变。

语法。

date_is_today(date);

争论类型 描述
date The datetime to use.

返回。

例子。

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

上面的代码将检查存储在全局变量 "Halloween "中的日期时间，看它是否是今天，如果是，它就将另一个全局变量设置为一个新值。

返回。日期和时间
