date_set_timezone

设置基本时区，用于所有其他的日期和时间功能。这个时区可以是本地时间（由系统设置），也可以是UTC时间，你可以使用下面的一个常数来定义使用哪一个（默认是本地时间）。

恒定 描述
timezone_local 使用系统设置的本地时区
timezone_utc 使用协调世界时

语法。

date_set_timezone(timezone);

争论 类型 描述
timezone 时区常数 The time zone to use for the base time.

返回。

N/A

例子。

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

这段代码检查游戏的基本时区设置，如果它不是UTC，就改变它。
