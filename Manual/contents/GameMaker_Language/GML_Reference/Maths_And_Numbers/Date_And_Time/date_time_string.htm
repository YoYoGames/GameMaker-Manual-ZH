date_time_string

通过这个函数，你可以创建一个包含给定时间的string ，格式化为函数调用时正在运行游戏的系统或设备。

语法。

date_time_string( date );

争论类型 描述
date The datetime to use.

返回。

例子。

str = date_time_string(date_current_datetime());

这将根据系统对日期/时间显示的设置和当前的日期和时间，将给定的变量设置为类似 "11:36.00 "的内容。

返回。日期和时间
