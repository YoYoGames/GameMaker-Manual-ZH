日期_小时_跨度

通过这个函数，你可以得到两个日期之间的小时数。这个值总是正的，不完整的小时将以分数的形式返回。

语法。

date_hour_span(date1, date2);

争论类型 描述
date1 The first datetime.
date2 The datetime to compare the first one to.

返回。

例子。

diff = date_hour_span(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

这将把 "diff "设置为2011年9月15日11:04.0与当前日期和时间之间的小时数。

返回。日期和时间
