date_get_timezone

该函数获取用于所有其他日期和时间函数的基础时区。这个时区可以是本地时区（由系统设置），也可以是UTC时区，该函数将返回以下常数之一。

时区常数
恒定 描述
timezone_local 使用系统设置的本地时区
timezone_utc 使用协调世界时

语法。

date_get_timezone();

返回。

Time Zone Constant

例子。

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

这段代码检查游戏的基本时区设置，如果它不是UTC，就改变它。

返回。日期和时间
