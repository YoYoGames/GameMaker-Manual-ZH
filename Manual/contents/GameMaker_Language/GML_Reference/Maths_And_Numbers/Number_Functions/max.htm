最大

这个函数返回输入值的最大值，其中最多可以有16个。例如， max(12, 96, 32, 75) 将返回96，因为这是所有输入值中最高的一个。

语法。

max(val1, val2, ... max_val);

争论类型 描述
val0 ... max_val The values to compare.

返回。

例子。

x = max(x, 0);

这将阻止玩家从room 的左边离开。这通过不断地将它的x设置为自己或0，以较大者为准。如果玩家离开了左边，它的X会小于0（即负数），所以它会被直接设置为 归位。

返回。数字功能
