随机

这个非常有用的函数只返回整数（整数）。因此，例如，要得到一个从0到9的随机数，你可以使用 irandom(9) ，它将返回一个从0到9的数字（含）。请注意，也可以使用浮点数，但点后的上限值将被排除，所以 irandom(9.7) ，将只返回0到9的值。该函数的上限为$7fffffffffffffLL，所以如果使用非常大的数字，应该注意。

注意由于GameMaker每次都会生成相同的初始随机种子，使debugging ，所以这个函数在每次重新运行游戏时都会返回相同的值。为了避免这种行为，请在游戏开始时使用 randomise()在你的游戏开始时使用。

语法。

irandom(n);

争论类型 描述
n The upper range from which the random number will be selected.

返回。

例子。

if (irandom(9) == 1)
{
    score += 100;
}

这将产生十分之一（因为包括0）的机会，使分数增加100。
