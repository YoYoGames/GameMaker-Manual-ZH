权力

这将返回一个数字乘以自己的 "n "倍的值。例如， power(5,3) ，将5乘以自己3次，返回125，这就等于说5*5*5=125。请注意，"x "值 (要改变的数字)不能是一个负值。

语法。

power(x, n);

争论类型 描述
x The number to change.
n How many times to multiply x by itself.

返回。

例子。

score += power(dmg, 3);

这将把变量 "dmg "中持有的数值添加到分数变量的3倍。
