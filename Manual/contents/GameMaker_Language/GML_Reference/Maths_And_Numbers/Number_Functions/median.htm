median

这个函数返回输入值的中位数，也就是中间值。当参数数为偶数时，将返回两个中间值中较小的一个，该函数可以有任意多的参数（注意，参数越多 将意味着该函数的解析速度会变慢），这些参数必须都是实值。这意味着，例如， median(43, 12, 25, 19, 6) ，将返回19，因为它是所有其他参数之间的中间值。

语法。

median(val1, val2, ... max_val);

争论类型 描述
val0 ... max_val The values to compare.

返回。

例子。

x = median( 0, x, room_width );
y = median( 0, y, room_height );

这将阻止玩家离开room 的任何一边，通过使用中位数作为一个钳制。例如，如果玩家移动到room 边界的左边，它的x会小于0。这将意味着第一个中位数的中间数 将是0，所以玩家将被跳转到（0,y）。

返回。数字功能
