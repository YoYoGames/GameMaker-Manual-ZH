这是函数power(e,n)，其中e大约是2.718281828（也称为欧拉数），n是它应该乘以自身的次数。这基本上是说 "在经过n个单位的时间后我能得到多少增长 (和100%的连续增长）？"。因此， exp(3) ，将返回20.08，这意味着在3个时间单位后，我们有20.08倍于我们开始的金额。

语法。

exp(n);

争论类型 描述
n The power of e.

返回。

例子。

val = exp(2);

这将把val设置为e^2，大约7.4。
