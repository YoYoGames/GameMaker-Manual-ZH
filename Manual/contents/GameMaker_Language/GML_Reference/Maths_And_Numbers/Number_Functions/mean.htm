mean

这个函数的工作原理是将所有的输入值相加，然后除以它们自己的数字。你可以根据你的要求有多少个参数（注意，更多的参数将意味着该函数的解析速度会更慢）。因此， mean(2, 6, 9, 32) ，返回12.25，因为2+6+9+32=49，49/4=12.25。

语法。

mean(val1, val2, ... max_val);

争论类型 描述
val0 ... max_val The values to compare.

返回。

例子。

xmiddle = mean(obj_player1.x, obj_player2.x, obj_player3.x);
ymiddle = mean(obj_player1.y, obj_player2.y, obj_player3.y);

这将设置xmiddle和ymiddle为三个玩家objects ，obj_player1，obj_player2和obj_player3的平均坐标的x和y坐标。例如，你可以用它来保持游戏摄像机聚焦在所有三个玩家身上 而不是只有一个。
