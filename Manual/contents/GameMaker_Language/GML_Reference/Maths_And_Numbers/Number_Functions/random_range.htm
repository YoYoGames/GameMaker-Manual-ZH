随机范围

这个函数返回一个在指定范围内的随机数，这个返回值不需要是一个整数。例如， random_range(20,50) 将返回一个从20到50的随机数，但该值可能是一个实数，如38.65265。 实数也可以作为输入参数使用。

注意：由于GameMaker每次都会生成相同的初始随机种子，使debugging ，所以这个函数在每次重新运行游戏时都会返回相同的值。为了避免这种行为，请使用 randomise()在你的游戏开始时使用。

语法。

random_range(n1, n2);

争论类型 描述
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

返回。

例子。

score += random_range(500, 600);

这将使总分增加500到600之间。
