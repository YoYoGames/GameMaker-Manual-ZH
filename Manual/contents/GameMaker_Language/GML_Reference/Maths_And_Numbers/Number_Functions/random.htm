随机

这个函数适用于不需要返回整数（整数）的概率。例如， random(100) ，将返回一个从0到99的值，但这个值可能是22.56473!你也可以在这个函数中使用实数而不是整数，比如 - random(0.5) ，它将返回一个0到0.4999999的值。

注意：由于GameMaker每次都会生成相同的初始随机种子，使debugging ，所以这个函数在每次重新运行游戏时都会返回相同的值。为了避免这种行为，请在游戏开始时使用 randomise()在你的游戏开始时使用。

语法。

random(n);

争论类型 描述
n The upper range from which the random number will be selected.

返回。

例子。

if (random(10) >= 9)
{
    score += 100;
}

这将产生大约十分之一的机会，使分数增加100分。
