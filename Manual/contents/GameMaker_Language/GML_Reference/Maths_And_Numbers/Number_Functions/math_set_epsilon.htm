Epsilon是一个用于确定两个受四舍五入误差影响的数字是否足够接近而被视为 "相等 "的数值。它在处理浮点数学时非常有用，因为它可以减少 "四舍五入误差"，这些误差使某些操作返回的值看起来不正确或与我们的预期相反。例如，我们可能向sprite 的图像索引添加了一个值，并期望结果是一个单一的整数，但由于浮点数学的性质，实际的最终值最终是像5.0000002，所以当我们有以下检查。

if image_index == 5
{
    //do something
}

这段代码的表现与预期不符，返回 false 。然而，如果我们将epsilon值设置为0.000001， image_index 值将被四舍五入为最接近原值的+/- 0.000001的实数，使上述比较返回真。

epsilon值将被用于以下所有的整数运算符。

  • < :少于
  • > :大于
  • == :等于
  • <= :小于或等于
  • >= :大于或等于
  • != :不等于

请注意，设置epsilon值为0将禁用所有的四舍五入，而使用1的值将产生一个错误。

语法。

math_set_epsilon(epsilon);

争论 类型 描述
epsilon 真实的 The new epsilon value (from 0 to 0.999999999).

返回。

例子。

math_set_epsilon(0.0001);

这将为所有进一步的浮点运算设置epsilon值。
