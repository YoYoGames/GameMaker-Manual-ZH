对数2

这个函数基本上是问 "你需要乘以多少个2才能得到n？"。在编程中，一个常见的例子是计算表示一个数字所需的比特数。要做到这一点，我们要计算数字的 log2(x) ，四舍五入并加1--例如 log2(100) ，返回6.643856，四舍五入后是6，再加1，就得到7。所以我们需要7个比特来表示100。它还可以用来描述指数增长或 衰减，通常被称为二进制对数。

语法。

log2(n);

争论类型 描述
n The input value.

返回。

例子。

colourbits = floor(log2(colour)) - 1;

上面的代码存储了表示存储在变量 "color "中的数值所需的比特数。

返回。数字功能
