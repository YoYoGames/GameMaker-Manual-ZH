ln

自然对数 ln(n) ，是指达到一定水平的连续增长所需的时间，其中n是达到的水平。因此，如果我们想知道我们需要多少时间单位来获得20的增长，我们将使用 ln(20) ，它的结果是 返回2.99个时间单位来获得这个增长量。

语法。

ln(n);

争论类型 描述
n The input value.

返回。

例子。

alarm[0] = ln(age) * room_speed;

上面的代码使用存储在变量 "age "中的数值的自然对数来设置警报。

返回。数字功能
