围绕

正如它所说的那样， round() ，将一个实数向上或向下取整到最接近的整数。在特殊情况下，如果提供的数字正好是一个半整数（1.5、17.5、-2.5等），该数字将被四舍五入到最接近的偶数值，例如，2.5将被四舍五入到2，而3.5将被四舍五入到4。这种类型的四舍五入被称为庄家四舍五入，在大量的迭代或使用浮点数学时，它提供了统计学上更好的四舍五入 比更传统的 "如果超过0.5就向上取整，否则向下取整 "的方法更好。

这意味着，如果一个值的分数是0.5，那么四舍五入的结果是最接近输入值的偶数整数。因此，例如，23.5变成24，24.5也是如此，而-23.5变成-24，-24.5也是如此。这种方法处理正 这种方法对称地处理正负值，因此没有符号偏差，更重要的是，对于合理的数值分布，四舍五入后的数字的预期（平均）值与原始数字是一样的。

语法。

round(n);

争论类型 描述
n The number to round.

返回。

例子。

score += round(hp / 5);

上面的代码将在得分值上添加一个四舍五入的整数。
