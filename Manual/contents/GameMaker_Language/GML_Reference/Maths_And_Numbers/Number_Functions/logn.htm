logn

这个函数类似于 log2(n)和 log10(n) 函数相似，只是你规定了对数基数。例如， logn(5,25) 将返回我们需要乘以多少个5来得到25（即2）。 来得到25（即2）。

语法。

logn(n, val);

争论类型 描述
n The log base.
val The input value.

返回。

例子。

logval = logn(5, num);

上面的代码得到存储在 "num "中的数值的 log5 。
