选择

有时你想为随机选择指定一些数字以外的东西，或者你想要的数字不按任何实际顺序或在任何设定的范围内。在这些情况下，你会使用 choose() 来生成一个随机结果。例如，假设你想创建一个object ，在开始时有一个随机的sprite ，那么你可以使用这个函数将sprite 的索引设置为一组给定的sprites 中的一个。注意，你可以有你所需要的参数（注意，更多的参数将意味着该函数的解析速度会更慢）。

注意这个函数在每次重新运行游戏时都会返回相同的值，这是因为GameMaker每次都会生成相同的初始随机种子，以使debugging 代码变得更容易完成。为了避免这种行为，请在游戏开始时使用 randomise()在你的游戏开始时使用。

语法。

choose(val0, val1, val2... max_val);

争论 类型 描述
val0... max_val 变化的 Any type of value(s).

返回。

Variable (One of the given arguments)

例子。

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

上面的代码使用选择来为实例设置一些属性。
