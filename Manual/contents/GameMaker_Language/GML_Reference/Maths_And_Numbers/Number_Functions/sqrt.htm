sqrt

如果你将一个数字与它自己相乘，你会得到这个数字的平方，但有时你想做相反的事情，得到一个数字的平方根。因此，要想知道什么数字与自己相乘后得到任何给定的正值，我们使用 这个函数。例如： sqrt(9) 将返回3，因为3*3=9。

语法。

sqrt(val);

争论类型 描述
val The number to get the square root of.

返回。

例子。

num = sqrt(exp);

上面的代码将设置变量 "num "来保存 "exp "所含数值的平方根。

返回。数字功能
