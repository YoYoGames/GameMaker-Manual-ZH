对数10

这个函数类似于 log2(n)函数，只是 log10(n) ，得到我们需要乘以10的数量来得到n。所以 log10(100) ，返回2，因为10乘以自己得到的是 100， log10(1000) 返回3，因为10x10x10=1000等等。然而，这个数字不一定要能被10整除，但仍会返回一个适当的值，例如，如果我们有 log10(101) ，我们得到2.00432137...这就是 这是101除以10的次数。

语法。

log10(n);

争论类型 描述
n The input value.

返回。

例子。

logval = log10(num);

上面的代码得到存储在 "num "中的数值的 log10 。
