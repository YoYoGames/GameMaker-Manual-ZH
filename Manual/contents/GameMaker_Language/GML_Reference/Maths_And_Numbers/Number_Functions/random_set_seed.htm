random_set_seed

为了生成一个随机数，GameMaker以一个随机种子数开始。通过这个函数，你可以将这个种子设置为一个已知的值，从而 "强制 "所有随机事件的结果在每次程序运行时都是一样的。例如，这个函数可以和以下函数一起使用 random_get_seed()来创建程序生成的内容，并保存结果，而不需要巨大的保存游戏（你只保存种子，不需要其他东西）。如果你需要真正的随机结果，你应该使用 randomise()函数。

注意虽然这个种子在每个目标平台上会有一致的结果，但由于每个目标的工作方式不同，平台之间的结果可能会有所不同。

语法。

random_set_seed(val);

争论类型 描述
val The seed to set.

返回。

N/A

例子。

if debug
{
    random_set_seed(1);
}

上述代码仅在变量 "debug "为真时才将随机种子设置为1。

返回。数字功能
