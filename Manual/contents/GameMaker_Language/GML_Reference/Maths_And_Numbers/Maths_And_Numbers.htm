数学和数字

GameMaker包含大量的函数来处理不同领域的数学、各种类型的数字以及日期和时间。这些函数中的大部分可以在下面列出的章节中找到。

  • 日期和时间
  • 数字功能
  • 角度和距离
  • 矩阵功能

有一些内置的常数也可以被评估为实数（尽管你应该总是检查常数而不是数值）。

恒定 价值
true 1（尽管任何等于或大于1的值都会被评估为 true ）。
false 0（尽管任何小于1的值也会被评估为 false ）。
pi 3.141592653589793280等......（具体数值将取决于各种因素，如目标平台。）
