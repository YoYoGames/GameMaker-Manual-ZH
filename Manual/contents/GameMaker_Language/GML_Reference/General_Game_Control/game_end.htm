游戏结束

通过这个函数，你可以结束游戏（并且游戏结束事件将被触发）。这不是瞬间发生的，而是在当前步骤结束时发生的，所以任何代码 在这个函数被调用后，你在同一步骤中的任何代码仍将运行。请注意，这个函数有以下限制。

  • 在Android 设备上，调用 game_end() 将把应用程序推到后台，但不会关闭应用程序。这必须由用户来完成。
  • 在iOS ，它将什么也不做，默默地失败。
  • 在所有的游戏机上（包括在Xbox上运行的UWP），它可能会使游戏崩溃，或者最多是默默地失败，也可能是提交失败，所以这个函数一定不能使用。
  • 在HTML5上，它将结束游戏，但让用户盯着网页上的空白画布，因此，不建议在该目标平台上使用它。
  • 在Windows （包括PC上的UWP）、Linux 和macOS ，该函数只是结束游戏并关闭游戏窗口（游戏结束事件也将被触发）。

语法。

game_end();

返回。

例子。

if keyboard_check_pressed(vk_escape) game_end();

如果玩家按下 "逃生 "键，这将结束游戏。
