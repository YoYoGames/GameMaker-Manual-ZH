游戏_项目名称

这个只读变量以目标平台的 "友好保存 "格式返回你的游戏在目标平台的显示名称。如果显示名称包含任何不允许用于文件名的字符，它们将被自动替换为"_"。 会被自动替换成"_"。显示名称可以在游戏选项中设置。请注意，由于HTML5对文件名没有限制，这个string 可能会与 与由 game_display_name.

语法。

game_project_name

返回。

例子。

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

上面的代码得到了项目名称，并使用它来打开（或创建）一个ini文件，然后将其写入。
