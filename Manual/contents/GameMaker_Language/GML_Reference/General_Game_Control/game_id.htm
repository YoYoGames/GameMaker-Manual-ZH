game_idDEPRECATED

这个只读变量返回你所创建的游戏的唯一标识符。如果你需要一个唯一的文件名，或者其他需要识别你的游戏的东西，你可以使用它。这可以在游戏选项中设置。

语法。

game_id

返回。

Real

例子。

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

上面的代码对分数进行了非常基本的加密，在保存前加上了game_id。当把它读回游戏时，你将扣除游戏ID以再次得到正确的值。

返回。一般游戏控制
