游戏保存缓冲区（game_save_buffer

这是 game_save() 功能的一个变体，所以请先阅读它的页面，因为它包含与它的使用和这个功能的相关的重要信息。

通过这个函数，你可以将游戏的当前状态保存到先前创建的 "成长 "buffer （见缓冲区），然后可以在任何时候用 game_load_buffer()函数。这个函数被设计为一次只在一个room （即：不建议在一个buffer ，然后试图从另一个room ）中做一个buffer 保存，最好只用于检查点或关卡重启。

注意：这个功能非常有限，它是为初学者设计的，以便快速建立和运行一个检查点系统，但更高级的用户可能更愿意使用文件功能来编码他们自己的系统，因为游戏不会保存任何动态资源，如数据结构、表面、添加的sprites 等。

语法。

game_save_buffer(buffer);

争论类型 描述
buffer The buffer id to save to.

返回。

例子。

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

上面的代码将设置一个全局变量为 true ，然后在按下 "S "键时将当前游戏状态保存到变量 "save_buff "中索引的buffer 。

返回。一般游戏控制
