game_save_id

这个只读变量将返回你的游戏用来保存文件的目录的完整path ID。这个目录可能对其他应用程序可见，也可能不可见，取决于平台。

在HTML5目标上，它将返回一个空的string 。

注意这个变量的存在纯粹是为了提供信息，不应该用来获取文件path 或任何类似的东西，因为它不会在所有平台上正确工作。

语法。

game_save_id

返回。

String

例子。

save_dir = game_save_id;

这将把保存文件的目录存储到一个变量中，以便将来使用。

返回。一般游戏控制
