game_load_buffer

通过这个函数，你可以加载一个之前保存的游戏状态。游戏从先前创建的 "增长 "buffer （见缓冲区）加载，并且buffer ，必须有一个游戏状态被保存到它，使用 game_save_buffer()函数。

语法。

game_load_buffer(buffer);

争论类型 描述
buffer The buffer id to load from.

返回。

例子。

if keyboard_check_pressed(ord("L"))
{
    if (global.Checkpoint) game_load_buffer(save_buff);
}

上述代码将从变量 "save_buff "中索引的buffer ，只有当全局变量是 true ，当玩家按下 "L "键时，才会加载先前保存的游戏状态。

返回。一般游戏控制
