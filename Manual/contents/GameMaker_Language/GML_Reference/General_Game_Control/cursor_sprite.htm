游标_斯普利特

设置这个变量将指示 游戏制作者使用指定的sprite 作为光标（基本上每一步都将其设置为当前鼠标的X/Y位置）。默认值是-1，即没有光标的sprite ，但你可以从游戏中指定任何sprite ，assets ，或从外部资源中导入的索引。请注意，没有办法控制动画速度或图像_索引，所以如果sprite 有子图像，这些图像将以与room 速度相同的速度进行循环。要删除光标sprite ，你可以将这个变量再次设置为-1。

还值得注意的是，这个变量并不能取代游戏窗口的光标，它仍然会被正常绘制。为了避免这种情况，你可以使用函数 window_set_cursor()并将其设置为常数 cr_none ，这将使标准光标不可见。

语法。

cursor_sprite;

返回。

Sprite Asset

例子。

cursor_sprite = spr_CustomCursor;

上面的代码将把变量 "spr_CustomCursor "中索引的sprite ，作为游戏的游标sprite 。
