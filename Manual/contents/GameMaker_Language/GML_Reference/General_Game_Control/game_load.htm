游戏加载

有了这个功能，你就可以加载一个以前保存过的游戏，用 game_save().请注意，它将恢复用于创建保存的游戏版本，所以在它之后的任何更新都将不可见。欲了解更多信息，请阅读关于 game_save().

语法。

game_load(filename);

争论 类型 描述
filename 字符串 The name of the file to load from.

返回。

N/A

例子。

if keyboard_check_pressed(ord("L"))
{
    if (global.Save) game_load("Save.dat");
}

上面的代码将加载一个先前保存的游戏，如果一个全局变量在玩家按下 "L "键时被 true 。

返回。一般游戏控制
