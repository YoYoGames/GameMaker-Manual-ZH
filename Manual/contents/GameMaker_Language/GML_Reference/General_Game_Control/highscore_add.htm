highscore_add

通过这个函数，你可以将一个名字和一个分数添加到内部全局高分列表中。不需要检查数值是否高到可以进入分数列表，因为GameMaker只会存储那些大于第十位存储的数值。

语法。

highscore_add(str, numb);

争论 类型 描述
str 字符串 The string (name) to attribute the score to.
numb 真实的 The number (score) to add.

返回。

N/A

例子。

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

上述代码将检查生命值，如果它们被设置为0，它将把当前的分数和全局变量 "名称 "中持有的string 加入高分列表，最后将分数设置为0，并将玩家送到 "rm_MainMenu "中索引的room 。

返回。一般游戏控制
