高分_姓名

通过这个函数，你可以检索到在给定位置的高分列表中已经存储的名称string 。如果没有输入名字，string "未知 "将被返回。

语法。

highscore_name(place);

争论类型 描述
place The place on the table (1-10).

返回。

例子。

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

上面的代码将loop ，通过高分列表并将所有的名字存储在一个数组中。

返回。一般游戏控制
