game_restart

通过这个功能，你可以重新启动游戏。这本质上与第一次运行游戏相同，因此将触发游戏开始事件，以及 游戏结束事件。

应该注意的是，当这个函数被调用时，某些东西将不会被重置。

  • 全局变量不会被重新初始化，除非有明确的编码--例如，内置的全局变量 score ，如果它已经在游戏中被修改过，那么在游戏重启后不会从零开始。
  • GPU的状态将不会被改变（因此，如果你已经设置了绘制颜色或alpha，例如，它将保持在改变后的值）。
  • 游戏速度将保持在你在游戏代码中设置的水平（如果你改变了它，这种改变将被延续）。
  • 任何来自asset 浏览器的Asset ，如果在游戏中的运行时间被改变了--例如，如果你改变了sprite 资源的原点或移动了path 资源的位置，将不会被重置。
  • 动态资源如buffers 、表面、数据结构或导入的sprites ，也不会被清理或删除（尽管你可能会失去对它们的引用，所以在使用这个函数时要注意为动态资源使用全局引用，或者在调用这个函数之前将它们清理掉）。

语法。

game_restart();

返回。

N/A

例子。

if keyboard_check_pressed(ord("R")) game_restart();

这将在玩家按下 "R "键时重新启动游戏。
