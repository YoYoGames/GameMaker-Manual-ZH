调度器_分辨率_设置

这个函数用来改变Windows 线程调度器的分辨率。参数中提供的分辨率值需要以毫秒为单位。

在默认情况下。 游戏制作使用1毫秒的分辨率，这是允许的最小的分辨率，并提供最佳性能。把分辨率改成更高的值可能会对性能产生负面影响。 但它也可能降低功耗。

你可以提供-1作为参数，将分辨率恢复到其默认值（由Windows ）。注意，改变调度器的分辨率会影响所有与你的游戏同时运行的进程。

注意：该功能仅适用于Windows 。

语法。

scheduler_resolution_set(milliseconds);

争论类型 描述
milliseconds The new resolution value (in milliseconds) or -1 for default

返回。

例子。

scheduler_resolution_set(2);

这个例子将Windows 线程调度器的分辨率设置为2毫秒。
