os_get_region

这个函数返回一个string ，其中包含运行游戏的操作系统的两个或三个字母的区域代码，这是由ISO3166-1标准规定的。如果该信息不可用，它将只保留一个空的string ""。

注意：这不是返回的位置区域代码，而是操作系统的区域语言代码。

语法。

os_get_region()

返回。

例子。

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

上面的代码检查操作系统的语言，如果是中文，它就检查操作系统的区域，根据返回的值打开不同的 *.ini 。
