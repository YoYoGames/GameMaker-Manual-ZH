GM_build_date

这个常量持有GameMaker构建正在运行的可执行文件时的日期和时间。该值可以使用日期和时间函数进行解析，并在编译时取自系统的UTC值。

语法。

GM_build_date;

返回。

Datetime

例子。

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

上面的代码将画出游戏的版本号和它被编译的日期和时间。
