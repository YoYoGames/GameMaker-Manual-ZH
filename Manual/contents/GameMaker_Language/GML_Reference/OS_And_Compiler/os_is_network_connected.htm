os_is_network_connected

通过这个函数，你可以检查并查看你的设备目前是否有互联网连接，如果有，它将返回 true ，如果没有，则返回 false ，根据操作系统的不同，它可能会在返回一个值之前尝试建立一个连接。

该函数有一个可选参数，用于启用/禁用它在调用时尝试建立连接，这个选项目前只适用于Nintendo Switch 。当设置为 true ，该函数将尝试建立连接，而当设置为 false ，它将仅仅限于检查连接。请注意，尝试建立连接可能会导致Switch 操作系统挂起几秒钟的时间。

注意 这个函数检查控制连接的内部设备API，因此如果有蓝牙连接、Wi-Fi连接，甚至只是一个允许上网的普通网络连接，都可能返回真。

语法。

os_is_network_connected([attempt_connection]);

争论 类型 描述
attempt_connection 布尔型 OPTIONAL 仅适用于Nintendo Switch目标。设置为 ，以便在调用时尝试操作系统级别的连接。 true

返回。

Boolean

例子。

if os_is_network_connected()
{
    global.connected = true;
}

上面的代码检查设备是否有互联网连接，如果有，就设置一个全局变量。
