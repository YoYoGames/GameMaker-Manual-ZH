种类

这个只读变量持有GameMaker的各种常量之一，告诉你游戏是为哪个操作系统创建的。请注意，这不一定与运行它的设备的操作系统相同，因为--例如--你的游戏可能运行在Amazon Fire 操作系统上，但会被建立在Android 平台上（在这种情况下， os_type 将成为 os_android ）。

可以返回以下常数。

操作系统类型常数
恒定 描述
os_windows Windows操作系统
os_uwp Windows 10 通用Windows 平台
os_operagx Opera GX
os_linux 基础设施
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos 苹果tvOS
os_android 安卓
os_ps4 索尼PlayStation 4
os_ps5 索尼PlayStation 5
os_xboxone 微软Xbox One
os_xboxseriesxs 微软Xbox系列X/S
os_switch 任天堂开关
os_unknown 未知的操作系统

语法。

os_type;

返回。

OS Type Constant

例子。

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

上述代码检查运行游戏的操作系统并相应地设置一个全局变量。
