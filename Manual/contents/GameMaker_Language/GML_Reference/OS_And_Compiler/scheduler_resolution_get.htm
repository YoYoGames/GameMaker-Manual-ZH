调度器_分辨率_获取

这个函数用来检索Windows 线程调度器的分辨率，单位是毫秒。如果调度器的分辨率被设置为默认值（由Windows 设置），该函数将返回-1。

关于线程调度器的信息，请参见scheduler_resolution_set() 。

注意：该功能仅适用于Windows 。

语法。

scheduler_resolution_get();

返回。

(or -1 for default)

例子。

scheduler_resolution = scheduler_resolution_get();

这个例子检索了Windows 线程调度器的分辨率，并将其存储在"scheduler_resolution" 变量中。
