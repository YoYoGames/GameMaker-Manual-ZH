os_get_config

该函数返回当前为你的游戏选择的配置的名称（作为string ）。关于配置的更多信息，请参见配置部分。

语法。

os_get_config()

返回。

例子。

if os_get_config() = "Free_Version"
{
    global.Ads = true;
}
else global.Ads = false;

上述代码将检查正在使用的配置，如果是名为 "Free_Version "的配置，游戏中的广告将被启用。
