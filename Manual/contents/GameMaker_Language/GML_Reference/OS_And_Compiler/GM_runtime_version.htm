GM_runtime_version

这个常数保持runtime 的版本号，在首选项中定义为用于构建项目的runtime 。该值被存储为string 。

语法。

GM_runtime_version;

返回。

String

例子。

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

上面的代码获取GM的日期、版本和runtime 常数，并将它们画到屏幕上。
