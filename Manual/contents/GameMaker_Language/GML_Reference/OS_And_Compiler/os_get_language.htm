os_get_language

这个函数返回一个string ，其中包含运行游戏的操作系统的两个字母的语言代码，由ISO639标准设定。如果该信息不可用，它将保留一个空的string ""，或 "en "表示 "英语 "语言。请注意，有些语言也有相关的区域代码，所以要区分同一国家的不同地区，请使用函数 os_get_region().

下表显示了ISO 639所定义的一些主要的双字母语言代码的例子。

语言 编码
阿拉伯语 臂膀
中国 zh
丹麦 da
英语 en
法国 冯小刚
德国 德
希腊语 el
意大利语 它
日本人 ja
挪威 不
波兰语 铅字
葡萄牙语 淘宝网
俄罗斯 ru
西班牙 es
瑞典语 sv

注意：这不是返回的位置国家代码，而是操作系统的语言代码。

语法。

os_get_language()

返回。

例子。

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

上述代码检查操作系统语言，并根据返回值打开不同的 *.ini 文件。
