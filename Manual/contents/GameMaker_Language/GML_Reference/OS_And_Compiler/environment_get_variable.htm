环境_获取变量

这个函数返回给定名称的环境变量的值（astring ）（也是astring ）。你可以通过在终端程序中输入 "env" 来获得macOS 和Ubuntu (Linux) 上的可用环境变量。 关于Windows 环境变量的信息，如果你使用的是命令提示符，那么输入 "echo %PATH%"，而使用PowerShell则是 "ls env"。请注意，在macOS 和Ubuntu (Linux) 上，"HOME "环境变量将返回 " " 。 环境变量将返回 "~/"path ，它映射到 "/Users/<username>"macOS 和 "/home/<username>"Ubuntu (Linux) 上。

注意：此功能仅适用于Windows,macOS 和Ubuntu (Linux)。

语法。

environment_get_variable(name);

争论类型 描述
name The name (a string) of the environment variable to check.

返回。

例子。

e_str = environment_get_variable("APPDATA");

上面的代码将返回环境变量"%appdata%"的完整的path ，通常是"C:\Users\{username}\AppData\Roaming"。
