对象变量

当你点击变量 按钮时，它将打开对象变量 窗口，然后你可以用它在 对象的创建事件之前 生成任何数量的变量。

在这个窗口中，你可以点击添加 按钮，将一个新的变量添加到列表中。然后，你可以为 该 以及设置其类型 和调整其属性。

变量名称必须以字母开头，只能包含字母、数字和下划线符号'_'，最大长度为64个符号。 最大长度为64个符号。因此，有效的变量是像 fish, foo_bar, num1, 而无效的变量是 6fish, foo bar , 或 *num 。一旦你给出了变量的名称，你需要设置它的类型，它可以是以下任何一种。值得注意的是，这个功能在处理父/子实例时特别有用--因为你可以在父object ，并定义一组变量，然后在子object ，简单地修改它们（详见下面的Object Variables And Child Objects一节）--以及在房间编辑器中添加实例时（详见Layers >Instance Layer一节），因为你也可以为个别实例修改这些变量。 因为你也可以为已经放置在room 中的单个实例修改这些变量。

  • 实数：实数是任何非整数的数字，可以是正数或负数。因此， 124.5, 45639.566546456, 0.9, -45.5 等...都是实数的例子。所有实数都以64位浮点值（双数）的形式存储。实数 （和整数） 。 类型有一个设置数值范围的选项，可以通过点击选项 按钮进入：当你选中这个选项时，你可以输入一个开始值和一个结束值，而不是在Object 变量窗口中为变量显示一个固定值。 而不是在变量窗口中显示一个固定的值，你将有一个与这些值相联系的滑块。
  • 整数。整数是一个整数，可以是正数或负数，例如： 30004, 19, 0, -300 。注意，你可以 注意你可以为输出的整数设置一个范围，就像上面对实数的解释一样。
  • 字符串。string 是指任何被置于双引号内的东西，例如 "fish"、"Hello World "或 "12345"。
  • 布尔值。布尔值是一个不是真就是假的值。在Object 编辑器变量窗口中，它被简单地显示为一个方框，你可以勾选为真，取消勾选为假。
  • 表达式。表达式是一个数学短语，可以包含普通数字、变量、strings ，或GML 函数以及一个或多个运算符。例如， sqrt(85 * 6) + 5.5 是一个表达式。

对象变量和子对象

Object 变量的一个重要特点是，它们被你在objects 浏览器中的任何子Asset 继承 （更多信息请参见父对象一节），这意味着你 如果你愿意，你可以选择覆盖或改变它们中的任何或全部。
