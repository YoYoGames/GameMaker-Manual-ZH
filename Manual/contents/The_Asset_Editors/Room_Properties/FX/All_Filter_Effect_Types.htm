FX类型和参数

本页列出了GameMaker中提供的所有滤镜/特效类型，以及关于每种特效的以下信息。

  • 内部名称（例如"_filter_tintfilter"），可以传入 fx_create() ，以便在运行时创建一个FX。
  • 描述
  • 示例图片
  • IDE参数，可以通过图层属性进行编辑。
  • 运行时参数，可以用来修改FX的runtime ，使用滤镜和效果函数
    • Runtime 参数也提到了它们的类型，所以你知道哪种类型的数据要应用到哪种参数。
