图像编辑器

图像编辑器 是你可以创建自己的精灵或编辑那些你之前导入或创建的精灵的地方。如果你使用过任何其他图像工具，那么你应该很熟悉，但在使用GameMaker图像编辑器时，有几件事需要注意，可能不是很明显。

  • 当你创建多个框架时，你可以在它们动画化时对它们进行绘画！这就是为什么你可以在框架上进行绘画。因此，如果你按下框架控制栏上的播放按钮 ，然后选择一个画笔并开始在编辑器窗口中画画，你会看到这些框架会在编辑器中产生动画，你所画的东西会被添加到那一刻的相应框架中。你可以尝试不同的帧率，直到你得到你喜欢的东西，但是一旦你掌握了这个工作的方法，你将很快创造出动态的sprites 和效果。
  • 你可以把图像编辑器标签从主workspace 窗口拉到它自己的窗口。例如，在使用多个显示器时，这可能很方便。
  • 当你使用复制 功能，例如，复制一个图像的选区时，那么复制的选区就会被添加到你的画笔列表中。这意味着你可以选择图像的一部分，然后在其他绘画工具中使用它来绘制线条、多边形或自由绘画。画笔也可以被缩放并与颜色混合。请注意，"粘贴 "功能并不是为了在编辑器内部使用，因为你复制的任何东西都会被添加到画笔列表中，所以可以选择以这种方式 "粘贴"。相反，粘贴只是为了让你从外部程序中抓取一些东西，并将其粘贴到图像编辑器中。
  • 由于方便的图层编辑器，你可以为每个sprite ，有多个图层。这些图层可以排序，甚至可以放在文件夹里，使创建更复杂的图像更容易。
  • 你可以通过使用键盘上的 "1" 和 "2" 键在编辑器中的子图像（框架）之间快速切换。

这些只是图像编辑器提供的一些更高级的功能。要了解全部的功能范围，请阅读下面的项目，详细说明编辑器的每一部分的功能。

框架视图

窗口的这部分显示了框架的顺序，它们将被制作成动画。你可以点击左边的 按钮来添加更多的帧--或者使用图像菜单--你也可以点击并拖动任何一个帧来改变它们在timeline 。在帧视图的左上方，你可以在洋葱剥皮 switch ，这意味着在任何sprite 动画的单个帧中，你可以选择将前面或后面的任何一个帧作为一个半透明层放在你当前正在编辑的图像帧下。这样做的好处是，每一帧都比上一帧略微浅一些，所以你可以在绘制下一帧时看到动画是如何从一帧到另一帧的进展。

启用这个功能会在帧上显示一些额外的控制，你可以点击 ，然后拖动来设置显示的前一帧的数量以及后一帧的数量。

你可以在 "视图"菜单中设置洋葱皮的某些属性，这在本页面的后面有解释。

在洋葱皮按钮下面，你有一个循环控制按钮 ，用来设置是sprite loops （当达到最后一帧时回到第一帧）还是ping-pongs（当达到最后一帧时倒退到各帧）。底部的播放按钮 ，用于启动/停止动画的预览。

你可以点击任何一个框架来选择它进行编辑，被选中的框架在顶角会有一个删除按钮 ，你可以点击这个按钮将该框架从sprite 。你也可以在多个框架上点击 / + 来一起选择它们，在这种情况下，点击删除按钮会删除所有被选中的框架。最后，你可以使用标准的键盘快捷键 "剪切"、"复制 "和 "粘贴 "来编辑选定的框架。 / + "X", / + "C", 和 / + "V"。

框架信息

此栏包含帧信息。在这里你可以用左边的输入框设置动画帧的回放速度。帧速可以是基于"每秒一帧"，也可以是基于"每游戏帧数"。可用的总帧数与当前被选中的帧数一起显示在右边，你可以通过点击左边的 图标来折叠这个栏（隐藏它）。你可以在图像编辑器的首选项中把它设置为总是开始打开或总是开始关闭。

请注意，这里用于动画速度的值也将设置sprite 在游戏中使用时的动画方式，或者作为一个图层上的sprite asset ，或者作为分配给一个实例的sprite ，尽管你可以使用适当的函数和变量修改这个runtime 。

画布控制

画布控件处理的是如何显示主画布及其内容的各个方面。这些选项是

  • Toggle Canvas Grid: 这将切换画布网格的开/关。这是GameMaker在主画布上绘制的一个网格，将其划分为几个部分，默认情况下，网格的大小被设置为1px。然而，如果你点击网格菜单图标 ，你将打开网格选项。
    这些选项允许你设置网格的颜色和阿尔法，以及沿X和Y轴的网格的单元值。你还可以在这里选择启用或禁用网格捕捉（默认情况下禁用）。
  • 画布缩放控制 ：这些按钮控制当前画布的缩放水平。你可以放大或缩小，点击 按钮将重置画布，使其与正在编辑的图像成1：1。你也可以点击Window Fit按钮 ，使整个图像画布适合当前编辑器workspace （这将酌情放大/缩小以使其适合）。请注意，你也可以使用 / 和鼠标滚轮 来放大和缩小，按 / + 将设置room 画布与图像为1：1。
  • 分割画布 。点击这个按钮将在水平分割画布和垂直分割画布之间循环，使其成为一个单一的画布。在分割画布视图中，每个画布都有自己的画布控制，因此你可以放大一个，缩小另一个，例如，你可以编辑任何一个画布，其变化反映在另一个画布上。要改变画布视图的切换，你可以点击分割选项 按钮，选择你想使用的选项，或者简单地点击分割画布按钮，直到它循环到你想要的选项。

绘制画布

这是你的图像的主要编辑窗口。在这里，你可以应用右边的任何工具在当前选定的图层上作画。你可以用鼠标左键 或右键 （并从颜色选择器中为每个颜色指定一个颜色）并使用编辑器右上方定义的任何画笔。你还可以使用 / 和鼠标滚轮 来放大或缩小，按 / + 将画布设置为与图像1:1（或者你可以使用上面说明的画布控制）。也请注意，你可以使用鼠标中键 或 + 在画布上平移。

帆布价值

底部的这个小条显示了编辑器中鼠标位置和选择工具的宽度和高度的不同值。

刷子

在这里你可以选择要画的画笔 。默认情况下，你有不同大小的方形和圆形画笔可供选择，但你可以很容易地创建自己的画笔，方法是在图像层上绘画，然后选择你希望用于新画笔的部分，并使用 / + "C" 来复制它。复制的部分将被添加到你的画笔中，你可以像其他画笔一样用它作画，这包括使用不同的绘画工具，如画线。

注意，新画笔的默认混合颜色将是白色，但如果你用颜色选择器改变颜色并使用画笔作画，它将与这个新颜色混合 。如果你想从列表中删除一个画笔，只需点击它并按下 键，它就会被删除。

注意：画笔在不同的时段不会被保留，关闭项目将删除你所创建的任何画笔。

颜色选择器

在这个窗口中，你可以使用任何可用的滑块和小工具来定义颜色，或者你可以直接输入ARGB值或HSV值。

如果你按住CTRL/CMD并点击 任何一个默认的色板，它将调出颜色选择器工具，并允许你为sprite 。在这种情况下，点击颜色选择器中的 "好 "将把调色板设置为新的颜色并与正在编辑的sprite asset 一起保存。

工具

下面我们列出了GameMaker默认提供的工具--请注意，所有的工具都可以用鼠标左键 ，右键 ，而且大多数工具都有与之相关的键盘快捷键（重要的工具在这里和工具一起列出，但完整的列表请见快捷键页面）。大多数工具也会有额外的选项，在图像编辑器的顶部可见，如下图所示。

图层编辑

你可以在图像编辑器中为图像的不同部分创建图层 ，这些图层可以使用各种画笔和工具进行操作和绘制。要创建一个新的图层，只需点击 图标，要删除它，再点击 图标。图层也可以通过点击 ，然后将图层向上或向下拖动到你需要的位置来进行排序，你还可以通过使用 按钮创建文件夹，然后将图层拖入其中来进一步组织图层。如果你在一个图层上双击 （或使用 ，并选择编辑图层属性），你就可以进入图层属性窗口。

在这里，你可以为图层命名，并设置混合模式（它应该与下面的图层 "混合 "的方式），从正常、加法、减法 或多重 混合中选择。最后，你可以用滑块设置图层的不透明度（透明度），但要注意，这是与单个像素的实际alpha值相累积的--例如，如果你有50%不透明的像素，并将图层的不透明度设置为50%，最终像素将以25%的不透明度绘制。

图片

图像菜单有以下选项（其中一些选项可能会或可能不会使用，这取决于当前的编辑器焦点）。

  • 剪切框架/选区 - 根据编辑器的焦点，从图像中剪切框架或选区。如果剪切一个选区，这将把它添加到画笔中。
  • 复制框架/选区 - 从图像中复制框架或选区，取决于编辑器的焦点。如果复制一个选区，这将把它添加到画笔中。
  • 粘贴框架/选区 - 从剪贴板粘贴框架，或从剪贴板粘贴选区到图像，这取决于当前的焦点IDE 。当粘贴到图像时，你应该注意这个选项只为使用外部程序的剪贴板数据而设计。当你在图像编辑器中从图像中剪切或复制到剪贴板时，这将被添加到编辑器右上方的画笔中，你应该选择一个画笔，用剪切/复制的选区进行绘画，而不是试图将其粘贴到图像中。还要注意的是，由于剪贴板使用的格式，一些程序在将剪贴板上的透明图像粘贴到GameMaker图像编辑器中时会出现问题。
  • 全选 - 选择完整的图像。
  • 取消选择- 清除图像中的任何选择。
  • 反转选择- 反转当前选择。
  • 添加框架- 添加一个新的框架到sprite 。
  • 插入框架- 在当前显示的框架之后插入一个新的框架。
  • 删除选定的帧- 从sprite 中删除选定的帧（你可以使用 / + 和/或 + 来选择多个帧）。
  • Import Image(s)- 从磁盘上加载一个要用作框架的图像。注意，你可以在资源管理器窗口中选择多个单独的文件来导入，从而一次加载多个框架。一旦你选择了你的文件，你将会看到下面的导入窗口。
    在这里，你可以选择如何将导入的图像添加到当前的sprite ，允许你调整整个画布的大小，裁剪或填充它，还可以设置放置新图像的 "锚 "位置。
  • 转换为帧- 这个选项将把一个单一的图像转换成若干个独立的帧。你必须先创建一个包含所有动画部分的单帧（或导入一个单条图像），如下图所示。
    由于这是一个单一的图像，我们想把它转换成单独的帧，我们可以使用这个选项，然后会打开以下窗口。
    在这里你可以设置如何分割单张图片来创建帧，设置帧的数量、帧的宽度和高度以及任何包括任何偏移值。
  • 导入带状图像- 这个选项将打开一个文件资源管理器，并提示你寻找一个单一的带状图像文件，以用于创建框架。一旦选择了一个图像，导入条状图像的窗口就会打开，与上面显示的转换为框架的选项完全一样。
  • 导出到PNG- 选择这个选项将把sprite 输出为一个单一的 PNG 图像。如果sprite 有多个图像，那么将创建一个 "带状 "sprite ，其中包括所有的图像，一个接一个，并且保存文件名将自动附加 "_stripX"，其中"X"是导出的帧数。例如，一个五帧的sprite ，将被导出为 "sprite_name_strip5.png"。
  • 调整所有帧的大小- 当你选择这个选项时，它将打开以下窗口，你可以设置你想调整组成精灵的所有动画帧大小的方式。
    你可以选择缩放 图像，或者调整画布的大小。如果你选择缩放图像，那么你只需为每一帧选择新的宽度和/或高度，是否使用相同的长宽比缩放，以及是否以像素或百分比缩放。请注意，在缩放时，你可以开启或关闭线性插值，开启线性插值会导致图像缩放时像素模糊或 "涂抹"（一般来说，对于高分辨率的艺术作品，你希望开启这个功能，但对于像素艺术作品，你希望关闭这个功能）。

    调整画布大小的选项不会改变图像中的像素，而是增加或减少图像所画的画布的大小。你可以用箭头图标设置调整大小的 "锚 "的位置，然后设置新的大小（以像素或百分比为单位），以及设置是否保持原始框架的长宽比。请注意，使框架小于原始sprite 图像将裁剪像素。
  • 裁剪所有帧到选区- 这个选项只有在你在主画布上选择了图像的某个区域时才可用。当你选择这个选项时，正在编辑的图像（以及动画中的所有其他帧）将被裁剪到选区的边界上。
  • Auto Trim All Frames- 这个选项允许你从sprite 的边缘修剪所有零阿尔法像素。这将考虑到sprite 内的所有图像帧，所以每一面将裁剪到哪一帧的最远的非零阿尔法像素的边缘。
  • 反转帧数- 选择这个将简单地扭转sprite 的帧数顺序。
  • 镜像 - 这个选项允许你沿垂直轴镜像当前帧和所有组成sprite 的帧。
  • 翻转 - 这个选项允许你沿水平轴翻转当前帧和所有组成sprite 的帧。
  • 旋转所有帧（顺时针90°）- 这个选项将把动画的所有帧顺时针旋转90°。
  • 旋转所有帧（逆时针90°）- 这个选项将把动画的所有帧逆时针旋转90°。

查看

视图菜单有以下选项。

  • 1:1- 缩放图像，使其与分辨率成1:1（即：实际尺寸），并在图像编辑窗口中居中。
  • 适合屏幕- 缩放图像以填充图像编辑窗口。
  • 显示网格- 切换像素网格。这与画布控件中的网格按钮是一样的（在本页面上进一步解释）。
  • 洋葱皮肤设置- 改变洋葱皮肤的设置。这将打开以下窗口，你可以设置洋葱头皮肤按钮所显示的框架，以及皮肤框架的不透明度和颜色。

影响

效果菜单是你可以添加自己的自定义效果插件的地方，默认有以下效果。

  • 模糊 - 对图像应用模糊效果。你可以设置强度以及使用的模糊类型（盒式或高斯式），并设置是否只沿水平轴或垂直轴模糊，或沿两者模糊。然后你可以选择只应用于当前层、可见层或所有层。
  • 灰度 - 以一个确定的数量对图像进行去饱和。你可以选择只应用于当前层、可见层或所有层。
