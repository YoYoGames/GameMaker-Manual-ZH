使用涂料表

TheDope Sheet 是显示添加到sequence 的不同轨道的数据的地方，它不仅让你了解每个轨道的概况，而且还可以编辑它们的位置以及参数变化的时间，等等。

当你添加一个新的asset 到一个sequence ，一个资产轨道将在轨道面板中被创建，一个资产键将被添加到dope sheet 在当前播放头的位置。这个键将反映asset 的播放长度，以帧为单位。

然后可以通过移动鼠标到钥匙框架的起点或终点，当光标改变时点击 ，然后拖动，来编辑asset 钥匙本身。这将移动钥匙的起点或终点，这取决于你选择的是哪一个，但都会影响钥匙的长度。请注意，你也可以通过简单地点击 ，然后向左或向右拖动，来改变一个asset 键的位置。

除了asset 键之外，dope sheet 也会显示参数键 。参数键最初是时间上的 "点"，在这里你发出信号改变它们相关的asset 的属性，sequence 将使用该点的值来改变asset 。如何做到这一点将取决于你是否在轨道面板中把键设置为线性插值。参数键，像轨迹键一样，可以通过点击 ，然后沿着timeline 拖动到一个新的位置，你还可以在拖动的同时使用 键将键扩展到一个以上的帧（光标会改变以显示这一点）。这意味着你可以在继续进行任何插值到下一个键之前，在一段时间内保持键值。

当使用dope sheet 工作时，也可以使用通常的键盘快捷键，即： / 和鼠标左键 ，以任何顺序选择一个或多个轨道， 和鼠标左键 ，选择连续的轨道，你也可以使用 / +"X" ， / + "C" 和 / + "V" 组合键进行剪切、复制和粘贴。

鼠标右键 菜单中也有不同的按键选项，详见下文。注意，当使用这些命令将按键粘贴到dope sheet ，按键将被粘贴在播放头的当前 位置。

这就是使用dope sheet 的基本知识，但下面我们将更详细地介绍每个不同部分的用途以及如何使用它们。

区域开始/区域结束

这些标记是剪辑区域的开始和结束点。这些标记可用于设置游戏运行时在sequence 中显示的room 的特定部分，并且在编辑器中预览序列时也将被用作loop 点。 在编辑器中预览序列时，也将使用这些点。这些点不能大于或小于sequence 编辑器序列控制中设置的总Sequence 长度。要改变剪辑区域标记的位置，只需点击 ，把它们拖到所需的帧上。

缩放控制

该栏用于显示dope sheet 的当前缩放，它也可以用来改变这个缩放级别。你可以点击 ，然后拖动来改变缩放的级别。

dope sheet 还要注意的是，你可以使用 / 键和鼠标滚轮放大或缩小 。


资产追踪

资产轨道在dope sheet ，表示为 "键"，它是代表asset 的位置和它在sequence 的活动时间长度的彩色条。如上所述，这些键可以根据需要移动、加长和缩短。请注意，一个asset 轨道的键比asset 本身的持续时间长--例如一个声音可能持续5帧，而你把它的键拉长到15帧--将在它的键的整个持续时间内重复。Asset 轨道和键可以用标准的键盘快捷键复制和粘贴，或用鼠标右键进一步修改 菜单选项（在本页下面进一步解释）。

如果你删除了一个asset 轨迹键，这并不会删除轨迹本身，也不会删除任何已经添加的轨迹参数，它只是从asset 中删除了dope sheet 键。如果这个轨道没有其他的键，你可以从asset 浏览器中拖动一个Asset 到dope sheet ，它将被添加为一个新的键，但要注意，你不能添加一个与原来的asset 不同的asset （即：如果你删除一个sprite 的键，你不能再添加一个声音asset 到该键的位置）。

请注意，只要在object 中创建了Sequence 元素，Sequence 中的任何room 轨道就会创建它们的实例，而不管它们的asset 键在轨道上始于何处。Sequence 控制器只是根据轨道的asset 键的位置和持续时间来切换实例的可见性，以隐藏和显示它，而不会 重复创建和销毁它。因此，当sequence 元素被创建时，实例将运行它们的创建事件，而不是它们的资产键开始时。

参数跟踪

参数轨道在dope sheet ，在它们所属的主要asset 轨道下面用彩色条表示。色条本身不能被编辑，它纯粹是参数键所应用的轨道的一种视觉表示。 而参数轨道上的各个点才是可以编辑的（见下文的参数键）。

弹头

播放头是目前正在预览的timeline 的时间位置。当把assets 投入序列画布时，这个位置被用来设置它们被添加的位置，以及其他一些事情，如添加参数或分割键。 以及其他一些事情，如添加参数或分割键。当你在sequence 编辑画布上按下 "播放 "键时，播放头会在sequence ，你也可以点击 ，然后拖动它来移动到sequence 中的特定帧，或者在多个帧上 "刷 "一下，得到一个快速的 预览，而不需要播放全部内容。


参数键

参数键是参数轨道上显示的 "点"，只要该参数发生了变化。这些点可以根据需要进行移动和扩展，使其持续超过一帧（如本页面上的进一步解释），并且 你可以在轨迹面板上编辑它们的值，或者通过改变画布上的asset （如果它是一个比例、位置或旋转参数）。请注意，所有的参数轨道 键也会反映为该参数所属的资产键 轨道上的一个点，但这些点不能编辑，当你在轨道面板中折叠不同的参数轨道时，这些点是作为一种信息帮助。


参数曲线

dope sheet 还允许你在参数轨道中使用帕里米特 曲线 而不是keyframes ，以改变参数值。这些曲线可以作为sequence asset 的一部分嵌入。 或者可以是应用于参数轨道的外部动画曲线 assets 。更多信息请参见 使用动画曲线获取更多信息。

人民币菜单

在资产轨道上的dope sheet ，点击鼠标右键 ，将弹出以下菜单选项（注意有些选项可能不可见，这取决于所点击的轨道部分是否有键）。

  • 剪切、复制和粘贴键。这些选项允许你剪切/复制/粘贴asset 。注意，任何使用这些选项粘贴到dope sheet 的键都将被粘贴到播放头的位置上。
  • 在原地添加键。这将在离鼠标点击的位置最近的帧位置，而不是 在播放头位置，向asset 轨道添加一个新的asset 键。
  • 就地分割键。当使用时，这将把鼠标下的asset 键分割成两个键，分割是在离鼠标点击的地方最近的一帧创建的，而不是在播放头位置。
  • 原地粘贴键。当你使用这个选项时，复制或剪切的键将被粘贴在最接近鼠标右键点击的帧位置，而不是 播放头位置。
  • 删除键。选择此项将删除当前选定的一个或多个键。
  • 复制到开始。这个选项将把选定的键复制到sequence （不是 剪辑区域的开始）的最开始。
  • 复制到最后。这个选项将把选定的键复制到sequence （而不是 剪辑区域的末端）的最末端。
  • StretchAsset Key:当你选择此项时，所选的asset 键将被拉伸到sequence 的末端（而不是剪辑区域的末端），或者拉伸到它与同一轨道上的另一个asset 键的起点相遇。
  • 删除Asset 键。当你使用这个功能时，所选的asset 键将被删除，然后你可以从asset 浏览器中拖动它到被删除的asset 轨道上，为该轨道添加一个新的dope sheet 。这将创建一个新的asset 键，然后可以进行编辑，同时保持与asset 相关的任何参数轨道。

在参数轨道上的dope sheet ，点击鼠标右键 ，会出现以下菜单选项（注意有些选项可能不可见，这取决于所点击的轨道部分是否有关键帧，以及该轨道是使用keyframes 还是参数曲线）。

  • 剪切、复制和粘贴键。这些选项允许你剪切/复制/粘贴参数键。注意，使用这些选项粘贴到dope sheet 的任何键将被粘贴到播放头的位置上。
  • 就地添加键。这将在离鼠标点击位置最近的帧位置，而不是 在播放头位置，向参数轨道添加一个新的参数键。
  • 就地分割键。当使用时，这将把鼠标下的参数键分割成两个keyframes ，分割是在离鼠标点击的地方最近的一帧创建，而不是在播放头位置。
  • 原地粘贴关键帧。当你使用这个选项时，复制或剪切的关键帧将被粘贴在点击鼠标右键的位置，而不是播放头的位置。
  • 删除键(s)。选择此项将删除当前选定的keyframe 或keyframes 。
  • 转换为嵌入式动画曲线。这个选项将把所选参数轨道的所有keyframes ，转换成嵌入式动画曲线（更多信息请参见使用动画曲线一节）。另外，如果在使用动画曲线的参数曲线上使用asset ，这将把曲线asset ，作为嵌入式参数曲线复制到轨道上。
  • Set To External Anim Curve:这个选项允许你将一个动画曲线asset 到参数轨道上（更多信息见使用动画曲线一节）。
  • 转换为钥匙。通过这个选项，您可以将一个animation curve 转换为单个keyframes 。请注意，如果曲线使用了catmull-rom插值，这将会丢失，而keyframes 将使用曲线上各点之间的线性插值来代替，同时也请注意，从animation curve 产生的键必须落在一个帧上。这意味着，任何具有帧间点的曲线在转换为键值时将被置于最近的整数帧值上，这可能会改变曲线的行为。
  • 导出嵌入式动画曲线。这个选项可以让你把一个嵌入的animation curve ，并把它转换成一个动画曲线资产，它将显示在Asset 浏览器中，然后可以在其他地方使用。
  • 打开编辑器。这将打开dope sheet 曲线编辑器，用于编辑已应用于参数轨道的嵌入式或外部animation curves 。这与使用dope sheet 上的Toggle Curve Mode 按钮相同。
  • 复制到开始。这个选项将把选定的关键帧复制到sequence （而不是 剪辑区域的开始）的最开始。
  • 复制到最后。这个选项将把选定的关键帧复制到sequence （而不是 剪辑区域的末端）的最末端。
  • Stretch Parameter Key:当你选择此项时，所选的asset 键将被拉伸到sequence （而不是剪辑区域的末端）或直到轨道上的下一个keyframe 。被拉伸的参数键意味着它将在被拉伸的帧数上保持它的数值。如果在参数曲线上使用，该曲线将被拉伸到sequence ，或者直到它与轨道上的另一条曲线相遇。
  • 删除参数键。当你使用这个功能时，选择的参数关键帧将被删除。
