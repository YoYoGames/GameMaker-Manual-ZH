使用动画曲线

涂料表的一个重要特点是能够将动画曲线分配给sequence 内的参数轨道，以创建参数曲线而不是 而不是参数keyframes 。有两种类型的曲线可以用于参数轨道。

  • 嵌入曲线。这些是嵌入在sequence 的曲线，对它们所使用的参数轨道来说是独一无二的。
  • 共享曲线。这些是独立于sequence ，使用动画曲线编辑器创建的曲线。

我们先解释一下如何使用动画曲线asset ，这只是创建一个新的曲线，在动画曲线编辑器中设置通道，然后将曲线asset ，拖到dope sheet ，将其应用到一个参数上。 轨迹。


你也可以使用鼠标右键 菜单选项Assign Shared Curve (Import) 来添加一个animation curve asset 到选定的参数轨道上。这将打开Asset Explorer，你可以从呈现给你的曲线中选择所需的曲线。

需要注意的是，animation curve 资产必须有正确的通道数，因为每一种参数轨道都需要不同的通道数。 每种类型的参数轨道都需要不同数量的通道，在animation curve 。任何没有足够通道的animation curves ，在asset 浏览器中会显示一个 指示器。

下面是不同的参数轨道和它们在每条曲线中所需要的通道的列表asset （通道应按照每个参数类型所给的顺序来添加）。

  • 位置：2个通道，X和Y。
  • 旋转。1个通道。
  • 颜色相乘。4个通道，Alpha、红、绿、蓝，范围必须在0到255之间。
  • 图像索引。1个通道。
  • 图像速度：1通道。
  • 原点：2个通道，X和Y。
  • 刻度：2个通道，X刻度和Y刻度，注意这个值是一个乘数，所以基准值是1（没有刻度），大于1会使图像变大，小于1会使图像变小。负值将翻转或 镜像图像，以及应用缩放。
  • 音高：1个通道，必须是最小的0，其中1是声音的默认音高。
  • 音量：1个通道，必须 最小为0，最大为1。

如果你导入的animation curve ，没有足够的通道用于参数轨道，你会被提示选择是否要将任何缺少的通道添加到动画曲线中。

如果在外部animation curve 编辑器中删除了任何所需的通道，共享曲线也可以变得无效。在这种情况下，共享曲线会在dope sheet 中显示为无效，你可以点击 "添加通道 "来创建任何缺失的通道，从而使曲线重新有效。 你可以点击 "添加通道 "来创建任何缺失的通道，这将使曲线再次有效。

默认情况下，animation curve 将沿着它所连接的asset 键的整个长度应用，然而，就像keyframes 一样，你可以点击曲线上的 并拖动它来重新定位它在timeline 上，你也可以按住 键并拖动来调整曲线的长度。 (这也适用于嵌入式曲线)。

将参数曲线添加到轨道中的另一种方法是嵌入它。要做到这一点，可以在参数轨道上点击鼠标右键 animation curve ，选择"转换为曲线"，或者点击Toggle Curve Mode 按钮，这将在完整的dope sheet ，然后你可以点击Convert To Curve按钮。

一旦你将参数keyframe 轨道转换为曲线，就可以在dope sheet ，与编辑动画曲线资产的方式完全一样。 这包括添加或删除点，改变它们的位置，改变曲线类型（线性、平滑或贝塞尔），应用预设，以及重命名通道和与之相关的颜色。 与它们相关的颜色。

重要提示：你也可以通过这种方式编辑已经应用于参数轨道的animation curve assets ，但是你所做的任何改变都会应用于所有使用该动画曲线的轨道，因为你是在编辑 曲线asset ，而不是它的一个副本。要编辑一个只应用于某一轨道的animation curve asset ，你必须先在参数轨道上使用鼠标右键菜单 ，选择嵌入曲线。这将复制animation curve asset ，并将其嵌入到参数轨道中，作为仅适用于该轨道的独立曲线。反之，你可以将一个嵌入的动画曲线 转换成一个动画曲线asset ，使用鼠标右键 菜单选项Export To Shared Curve。这将把嵌入的 曲线转换成一个新的asset ，然后可以像平常一样进行编辑，并在需要时应用于其他参数轨道。

频道列表

通道列表是你可以编辑构成参数曲线的不同通道的地方。每个通道必须有一个唯一的名称（只使用字母数字字符和"_"下划线符号）。 你还可以通过双击左边的色块 ，来设置通道的颜色，这将打开颜色选择器，让你选择一个新的颜色。 要重命名通道，你可以做一个缓慢的双击 ，然后重命名，你也可以用鼠标右键 ，打开一个菜单，你可以选择重命名、改变颜色或复制通道。通道可以通过点击名称旁边的箭头来展开，以显示通道曲线上的每一个点，这些点可以通过点击手动编辑 这些点可以通过点击当前值和键入新的值来手动编辑。你也可以在dope sheet ，通过点击 "眼睛 "图标来切换通道的可见性 。

转换控制

在这里keyframes ，可以转换为参数曲线（正如本页前面所解释的），但也有一个按钮可以将参数曲线转换为keyframes 。然而，你应该注意，在这样做的时候，有几个注意事项。关键帧 不能在 "帧之间"，所以有些曲线值需要四舍五入到最接近的帧，这反过来又会改变参数轨迹的表现方式。另外，如果参数曲线使用catmull-rom或bezier插值，这将会丢失 ，而 keyframes 之间的插值将被设置为线性插值。

曲线菜单

这个菜单包含编辑器中其他地方的选项，但都在一个地方。这些选项是

  • 设置范围。设置参数曲线的垂直范围（见下文通道范围）。
  • 转换为曲线。使用keyframes ，将选定的参数轨道转换为参数曲线。
  • 嵌入曲线。将参数轨道正在使用的共享曲线asset ，嵌入到轨道中。
  • 指派共享曲线（导入）。指定一个动画曲线asset ，作为参数轨道的共享曲线。
  • 转换为键。使用参数曲线将选定的参数轨道转换为keyframes 。
  • 导出到共享曲线。将嵌入的曲线转换成共享曲线asset 。

频道范围

通道范围按钮是用来设置通道编辑器的视觉垂直范围的。默认情况下，这将被设置为从-1到1，但点击这个按钮将打开以下窗口。

改变这些值将改变数据在通道编辑器部分的表示方式，允许你在通道曲线上有更高/更低的值，而不仅仅是-1到1。请注意，这纯粹是视觉上的，而不是钳制通道的值。 通道的值。
