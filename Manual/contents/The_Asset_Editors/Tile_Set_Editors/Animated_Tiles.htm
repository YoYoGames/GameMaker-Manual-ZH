动画瓷砖

瓷砖通常被认为是游戏中的静态单元room ，但是在GameMaker中，可以对它们进行动画处理，就像你处理一个sprite 。你问为什么不直接使用sprites ？嗯，sprites 有一定的处理开销，因为 由于它们与一个边界框相关联，以及它们被渲染的方式，但tiles 的处理开销要低得多，所以渲染起来更快。然而，这也是有代价的，因为使用tile set 动画，你被限制在 创建的动画必须是长度的2次方--即：2、4、8、16......帧--而且它们显然必须在tile 网格内创建，而且 它们必须以相同的速度（以每一格为基础）制作动画。然而，即使有这些限制，动画。 即使有这些限制，动画tiles 是一个强大的功能，可以用来为其他静态环境和背景添加生命和色彩。

要创建一个tile 动画，你必须先有一个tile set ，在一个tiles 图像中都有所需的sprite （但不是作为sprite 帧，所以一个大的sprite 帧中有所有的动画图像），然后在瓦片集编辑器中，你点击瓦片动画按钮，调出动画编辑器。

上面的例子是一个包含所有动画帧的单一图像，但应该注意的是，这个图像也可以包含其他非动画的。 也可以包含其他非动画tiles ，因为你可以使用一个单一的大型tile set ，然后 "挑 "出你想创建动画的部分。例如，你可以有一个有树木的大型景观tile set 。 大地和水，而在tile set ，有多个tiles ，以使水产生涟漪。

要创建一个tile 动画，你首先要通过点击添加动画 按钮 ，将一个动画添加到动画库中，这将会 将一个空的动画条添加到库中，并要求你选择要使用的帧数。

然后你会看到每个空的动画帧的视图，并为你突出显示初始帧。

如果你再回到编辑器左边的tile set ，你可以在任何一个tile ，点击 ，把它加入到动画中，帧会前进到下一个。在连续的tiles 上点击 ，就会填满动画。 然后你可以通过点击帧旁边的 "播放 "按钮来预览动画。

一旦你这样做了，你的动画将成为动画库的一部分，当你进入房间编辑器时，你可以选择它并把它放在room 。注意，如果你从一个动画中选择任何一个tiles ，并把它放在一个room ，该tile 将从选定的tile 开始动画化。 帧放在room ，它们都会产生动画效果。唯一需要注意的是，动画的速度将始终是相同的，因为它是为整个tile set ，而不是为库中的单个动画定义的。

关于在动画中使用的tiles ，有几件事需要注意。

  • 如果tile 有自己指定的动画（所以有一个动画，它被设置为第一个tile ），那么它将使用该动画。
  • 如果tile 没有自己的动画，但只在另一个动画中使用，那么它将从tile 在动画中出现的第一个位置开始使用该动画。
  • 如果tile ，没有自己的动画，而在其他多个动画中使用，那么它就根本不会有动画。你将需要明确地指定一个以它为首的动画tile 。
