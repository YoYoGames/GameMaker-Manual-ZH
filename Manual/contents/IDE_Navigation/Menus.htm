菜单

GameMaker IDE 的顶部被菜单栏占据。在这里你可以找到一些下拉菜单，可以用来导航 IDE ，执行任务或打开带有工具的附加窗口以帮助你制作游戏。请注意，我们在这里列出了可用的默认菜单，但一些asset 编辑器会根据上下文添加额外的菜单。这些额外的 菜单将在与创建它们的编辑器相关的页面上进行解释。

可用的默认菜单是。

  • 文件菜单
  • 编辑菜单
  • 构建菜单
  • Windows 菜单
  • 工具菜单
  • Marketplace 菜单
  • 布局菜单
  • 帮助菜单
