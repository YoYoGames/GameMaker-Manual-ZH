书签

另一种浏览workspace 的方法是使用书签。你可以在IDE 窗口或scripts 或object 事件中的代码/GML 视觉部分添加多达9个 "书签"，然后你可以使用正确的组合键快速跳过它们。

要添加一个书签，只需选择你想标记的窗口或代码行，然后在WIndows上 + + NumberKey[1 - 9] ，或在Mac上 + +NumberKey[1 - 9] ，这将设置书签，在IDE 中标记该项目，如下所示。

在上图中，我们已经将编号为1、2、4的书签分配给了IDE ，现在我们可以从这些窗口中导航，甚至完全关闭它们，因为书签将允许我们在以后的任何时候直接返回到它。这可以通过按 / + NumberKey[1 - 9] ，如果书签的项目已经关闭，它将被打开，否则workspace ，将平移到选定的书签。

重要提示 书签不会随项目一起保存，也不会在关闭IDE 。它们只对它们创建的那个会话有效。

如果你想删除一个书签，你只需要选择有书签的窗口或代码行，然后重复你用来创建它的 + / + NumberKey[1 - 9] 。
