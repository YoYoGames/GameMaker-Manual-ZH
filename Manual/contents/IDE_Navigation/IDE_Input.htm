IDE输入和导航

GameMaker IDE接受鼠标和键盘输入，许多操作可以使用其中一个或另一个，或者同时使用。一般来说，你可以点击鼠标左键 来选择任何东西，使用 / + 来选择多个项目，按住 把项目拖到不同的底座上或拖到workspaces ，还可以用 来打开上下文特定的菜单。注意，如果你在一个macOS 系统上运行GameMaker，并且使用单键鼠标，请使用 + ，以获得鼠标右键 。

还有大量的键盘快捷键，可以用来浏览不同的workspace 元素和资源编辑器，我们将快速浏览一些最重要的快捷键（你可以在这里找到一个完整的列表）。

  • / + "Z"：这将撤销前一个动作，在大多数编辑器中都适用。你也有多级撤消，所以你可以多次按这个键来 "回滚 "变化。
  • / + "Y"：这将重做之前被撤销的操作，在大多数编辑器中都可以使用。像撤销一样，你也有多个级别的重做。
  • / + "S" : 这将强制GameMaker保存你的项目。
  • / + "T" : 这将打开Goto窗口，使你能够快速搜索并进入你游戏中的任何asset ，以及搜索偏好和游戏选项。你会看到一个所有内容的列表，当你输入搜索词时，这个列表将被删减以适应你的需要，所以如果你所有的rooms ，例如以 "rm_"为前缀，输入这个词将删减列表，只显示那些以该名称开头的assets 。

  • / +: 这将打开工作区概览窗口，可用于在不同的workspaces ，快速浏览打开的项目。只要按住 或 ，该窗口就会一直打开，按一次 ，就会移动到窗口的下一个项目。松开快捷键后，将导航到被选中的项目。

    注意如果你把这个窗口的绑定重新映射为使用一个以上的修改器（如 + + B ）或一个单键（如只用 B ），那么在按下快捷键后，无论你是否还握着它，窗口都将保持打开，直到你手动关闭它或从窗口中选择一个项目。

  • :这将打开该手册。请注意，在使用GML Visual或GML 代码时，你也可以选择一个动作或一个函数（或关键词或其他），然后按 ，或点击鼠标中键 ，以打开相关页面的手册。
  • :这可以用来折叠或展开IDE 中的所有不同的停靠窗口。

笔记本电脑模式

如果你在笔记本电脑上使用GameMaker，那么你将在IDE 的顶部有一个进一步的选项，即笔记本电脑模式。

这在默认情况下是打开的，但如果你愿意，可以通过切换这个 按钮。笔记本电脑模式与IDE 中的一些工具相结合，使其在使用触摸板时有更好的体验，简化了3种主要的鼠标交互方式。 平移。 缩放和 滚动。它使用两个修改键来实现这一点。左键 / 和左 .当笔记本模式开启时。 左边 / 和 左边 被 "保留 "给鼠标互动，比如平移room 编辑器，其中的 左边 / 是缩放。 左边 是滚动和平移。笔记本模式下的滚动就像平移一样，所以它应该是直观和精确的，以至于你不需要使用滚动条。 所以它应该足够直观和精确，以至于你不需要使用滚动条。请注意，当按住这些按钮时，你只需要移动鼠标--不是点击也不是拖动，只是移动--如果有一个一般的快捷方式，通常需要 使用 / 或 键，你将使用键盘上的 键盘的右键 来代替。



触摸屏支持

最后我们要在这里提到的一点是，GameMaker的IDE ，也对触摸屏有最低限度的支持。在所有的操作系统上，你可以使用触摸屏来点击和拖动主workspace ，并且我们支持2个同时的 指针，第二次点击将执行右键点击。请注意，在Windows 8 及以上版本中，GameMaker IDE 也会支持笔类设备。
