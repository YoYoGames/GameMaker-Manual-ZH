设置Git 插件

在我们使用Git 插件之前，我们需要设置Git 可执行文件来使用，方法是进入首选项中的插件 -Source Control (Git)部分。

git.exe 如果通过Git ，path 可执行文件GameMaker 应该自动填写，但是如果该字段是空的，你可以点击 按钮，手动选择 cmd 文件夹中的Git 安装，通常在 C:\Program Files\Git\ 。

关于其他的Git ，请看插件首选项页面。

在你为你的项目使用Source Control 之前，你需要进入游戏选项并启用源代码控制。
