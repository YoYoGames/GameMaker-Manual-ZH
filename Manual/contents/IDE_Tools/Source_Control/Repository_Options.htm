存储库选项

在设置了Git 可执行文件并为你的项目启用了Source Control 之后，你现在需要将GameMaker IDE 链接到资源库，这可以通过两种方式之一完成。

  • 创建一个项目库
  • 克隆一个现有的存储库

在第一种情况下，创建一个项目仓库，这将直接在你的项目中创建一个Git 仓库。用Git 的说法，这是一个"非裸库"，所以推/拉 在这种情况下没有意义，但你仍然可以提交和跟踪项目的变化，以及在必要时恢复变化和修订。要设置这个，你只需要在主菜单上点击Source Control>Create Project Repository。

作为一个方便的功能，你也可以通过IDE克隆 一个外部版本库。同样，你需要在你的身份偏好中设置你要连接的版本库的用户名和密码，然后只需点击Source Control>Clone Repository。填写URL（使用HTTPS变量而不是SSH），并说明你想把它放到哪里。

一旦克隆完成，它将自动打开一个文件浏览器，如果你愿意，可以打开该项目。
