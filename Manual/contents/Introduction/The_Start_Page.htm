起始页

当你第一次打开GameMaker时，你会被要求登录到你的YoYo 或Opera账户（如果你还没有注册的话，请看这里）。一旦注册并登录，GameMaker的主集成开发环境将在开始页上打开。

当你第一次启动GameMaker时，将鼠标悬停在每个启动屏幕元素上，你就能看到对它的描述。

在这里，你可以点击"SKIP to Setup Wizard "按钮，在设置向导的帮助下开始创建你的第一个项目，它将带领你完成在GameMaker中创建第一个游戏的必要步骤。

你也可以点击 "关闭 "来关闭向导，继续手动使用GameMaker，如果你不希望在再次启动GameMaker时看到向导，还可以选择启用 "不要再显示我 "复选框。

起始页元素

开始页包含以下内容。

  • IDE菜单栏。这显示了你可以在整个IDE中使用的菜单；更多信息将在下面一节中给出。
  • 项目选项。项目 "部分是你可以创建、打开或导入项目的地方。这一点在本页面下面有更详细的解释。
  • 最近的项目。在这里你可以看到你可以打开的以前的项目列表。你还可以看到完整的项目名称和path ，点击鼠标左键 ，就可以打开该项目。你也可以通过按右上角的按钮在switch 模式和列表模式之间tile 本节的视图。
  • 版本细节。本节提供了当前正在使用的IDE ，以及当前Runtime版本的详细信息。你也会在这一节中得到关于IDE 或Runtime 的任何变化的通知。
  • 用户详情。这显示了你的YoYo 账户的用户名和你最后一次登录的时间。
  • 有用的资源。这显示了各种tiles ，你可以点击访问GameMaker的官方教程资源。

集成开发环境菜单

在顶部，你可以找到一般的IDE 菜单，这些菜单在手册的以下部分有解释。

  • 文件菜单
  • 编辑菜单
  • 构建菜单
  • Windows 菜单
  • 工具菜单
  • Marketplace 菜单
  • 布局菜单
  • 帮助菜单

请注意，在顶部的菜单栏中也会出现 特定的菜单选项，这取决于你目前所关注的窗口。例如，如果你关注的是图像编辑器窗口，那么你在这里会有额外的菜单项，包括"图像"、"视图"和"效果"。这些额外的菜单项在手册的相关章节中对产生它们的workspace 或窗口进行了解释。

开始一个新项目

你可以点击New 按钮来创建一个新的项目，Open 按钮来打开一个现有的项目，或者Import 按钮来打开一个压缩的 YYZ GameMaker项目文件或者一个传统的GameMaker。Studio 1.4 GMX 项目文件。打开和导入都会打开文件资源管理器，让你浏览到你需要的项目文件。

注意：GameMaker并不完全向后兼容GameMaker。Studio 1.4 项目，但导入的 1.4 项目仍然可以运行，因为过时的功能已经使用兼容性脚本自动为你重新创建。关于可能出现的问题和对GML 所做的修改的全部细节，请参见帮助中心文章《将 GMS 1.4 项目移植到GameMaker》，以及手册中关于过时功能的部分。

要创建一个新的项目，只需点击标有 "新建"的按钮，这将打开模板菜单。

选择一个模板（或从一个空白项目开始），为你的项目选择一个名称和保存位置，然后点击"开始吧！"。
