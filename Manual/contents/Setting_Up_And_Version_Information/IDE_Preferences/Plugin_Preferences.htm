插件首选项

插件首选项用于控制任何已添加到GameMaker IDE 的额外插件。默认情况下，它有一个部分专门用于源代码控制插件，用于将GameMaker项目与Git集成。

源控制 (Git)

本节涉及让Git SCM插件与GameMaker 协同工作所需的偏好。

  • Git的可执行路径。这是Git 可执行文件的位置，GameMaker 用于所有Source Control 命令。如上图所示，它将位于你安装的Git 的 cmd 文件夹中。

在这下面，你可以通过这些设置（可选）对合并工具和差分工具进行设置，Source Control 。

  • 合并（工具位置）：在这里你指定完整的文件path 到Git 合并工具。
  • 合并 (工具选项):在这个领域，你可以添加任何额外的命令，以便在你使用合并工具时运行。
  • 差异 (工具位置):这里你指定完整的文件path 到Git diff工具。
  • 差异 (工具选项):在这个领域，你可以添加任何额外的命令，以便在你使用diff工具时运行。
