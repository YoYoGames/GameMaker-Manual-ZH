字体编辑器首选项

字体编辑器首选项用于定义字体编辑器的行为方式。 下面列出了不同的选项，以及每个选项的作用描述。

  • 更新从GameMaker1 项目导入的Fonts 。当勾选此选项时，GameMaker将尝试更新fonts 并重新生成它们的texture 。这只有在以下情况下才会发生。
    • 该偏好被启用
    • font 在机器上是可用的
    • GameMaker可以在font 家族中找到一个匹配的面孔（这是最有可能导致导入时不一致的原因）。
    注意，如果GameMaker1font 没有缓存的font texture ，它也会发生。如果一个传统的font 不能被更新，那么GameMaker将默认使用缓存的 font texture 而Font 编辑器将不再允许你编辑font ，显示一个备份和更新到另一个选项。

    点击这个选项将使你能够编辑font ，要么添加缺少的font ，要么设置另一个，并且 备份的font ，作为 <font name>.yy.gms1 和 <font name>.png.gms1 ，存储在font asset 文件夹中。要恢复这些文件，请手动删除 <font name>.yy 文件，然后用这些文件替代，从文件名中删除 .gms1 extension 。 重要!所有的assets ，它们的名字都存储在 .yy 文件中，所以你必须编辑这些 并把正确的资源名称也放在 <name> 字段中（如果不同）。
  • 自动选择失败的内容再生对话框。当你去重新生成一个font asset texture ，你会看到一个警告。将此选项设置为 "OK"，意味着font ，而不显示警告。 显示。
  • 自动选择失败的font 再生对话框。当font asset texture 的再生失败时，会显示一个警告。将此选项设置为 "确定 "将跳过显示该信息。
  • 自动选择升级导入的遗留字体。当你导入一个传统项目时，你可能会得到一个关于传统fonts ，并提供升级的警告。把这个选项设置为 "确定"，就可以自动升级而不显示这个消息。 显示这个消息。
