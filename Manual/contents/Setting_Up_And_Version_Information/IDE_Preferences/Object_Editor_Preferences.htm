对象编辑器首选项

对象编辑器首选项用于定义对象编辑器窗口的某些属性。主要部分包含以下选项。

  • 点击打开事件的方法。这个选项控制你是否需要对一个object 事件进行双击或单击来打开它。默认情况下，这被设置为双击。
  • Inherited Events alpha:这个选项允许你改变任何 "继承事件 "在object 。默认情况下，它被设置为0.5（数值应该在0到1之间）。
  • 事件标签格式。在这里，您可以启用或禁用不同的可能文本，这些文本可以应用于object 编辑器事件列表中的事件标签。默认情况下，事件描述只被启用。
  • 添加默认的事件内容。默认情况下，这个选项是启用的，它所做的只是在所有新事件的代码编辑器中添加一个简单的注释，提醒你可以在那里添加你的代码。
  • 默认的事件内容。在这里你可以编辑新事件的代码窗口中默认添加的注释。

Object 编辑器也存在以下子类别。

确认对话

本节涉及到在使用Object 编辑器时可能显示的对话框，有以下选项。

  • 自动响应GML Visual转换为Gamemaker语言。你可以在GML Visualworkspace 内使用鼠标右键 ，选择将该事件的所有GML Visual转为GML 代码的选项。这里的默认动作是显示一个警告信息，询问你是否要继续，但你可以将其设置为 "好"，在转换前不会显示对话框。
  • 对超过物理vertex 限制的自动反应：这个选项可以用来控制当你在物理形状编辑器中超过允许的vertex 数量时显示的弹出消息。这里的默认动作是显示一个警告信息，但你可以将其设置为 "好"，这样就不会显示对话框。
  • 事件删除时的自动选择。这个选项可以用来控制当你在Object 编辑器事件列表中删除一个事件时弹出的信息。这里的默认动作是显示一个警告信息，但你可以将其设置为 "好"，这样就不会显示对话框了。
