瓦片组编辑器首选项

瓷砖组编辑器首选项有以下自动贴图的选项。

自动tile 背景颜色：该选项在tile set 编辑器中设置背景颜色，用于创建自动Tiles 。默认颜色是（rgba）#808080FF。

Tile Set 编辑器首选项中也有一个确认对话框 子部分。

这里的选项可以控制当你改变 的单元格大小时弹出的信息，这将影响到瓷砖图层的渲染方式（导致裁剪图层）。 tile set ，这将影响room 瓷砖图层的渲染方式（导致裁剪的图层）。默认情况下，你会被提示继续或取消操作，这个选项可以让你通过选择 "好 "来抑制这个信息。 这将继续进行tile set 单元的改变，而不考虑对room 瓦片图层的影响。默认值是 "显示信息"。
