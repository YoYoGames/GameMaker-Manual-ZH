一般偏好

常规偏好用于定义GameMaker IDE的一般行为。最普通的选项列在主页面上，而其他许多选项被分成多个子类别，在本页面的下方有进一步解释。

  • IDE语言。该选项允许你从提供的选项列表中选择IDE 的语言。默认为英语。
  • IDE字体。font 该选项允许你改变整个IDE 。 默认为Open Sans。
  • GameMaker 编辑器的最大帧率。这个选项允许你设置GameMaker IDE 更新的最大帧率，允许你设置帧率以匹配你的显示器的更新速度。默认值是60。
  • IDE皮肤。在这里你可以从不同的皮肤中选择，IDE 可以使用。默认情况下，GameMaker有两种皮肤，一种是浅色，一种是深色，初始设置使用深色皮肤。
  • 默认音频设备。在这里，你可以选择GameMaker的默认音频设备IDE ，以便使用。你可以通过点击刷新按钮刷新列表 。
  • 自动响应退出确认信息。这个选项允许你设置当你退出GameMaker时是否应该显示确认信息。默认情况下，它被设置为 "显示消息"，但也可以设置为跳过消息，就像你点击确定一样。
  • 自动响应系统Font 改变的对话框。这个选项允许你设置当一个系统font ，被添加、删除或编辑时是否应该显示一个确认信息。默认情况下，该选项被设置为 "显示信息"，但也可以设置为跳过该信息，就像你点击了确定一样。
  • 启用通知。勾选此选项将使IDE ，以显示关于新runtimes ，等的通知。这在默认情况下是打开的。
  • 通知超时（毫秒）。这设置通知显示的时间长度（以毫秒为单位）。默认值是6000。
  • 退出时注销：如果选中这个选项，那么当GameMaker关闭时，IDE 将自动注销用户会话。这主要是针对教育用户的，因为--如果选中--这意味着每当程序关闭时，用户就会被注销，从而腾出会话席位供其他用户使用。这在默认情况下是关闭的。
  • 启动时显示欢迎屏幕。默认情况下，这个选项是打开的，只是告诉GameMaker在你启动程序时显示一个简短的闪屏。取消勾选这个选项将防止显示闪屏。
  • 允许通过快捷键关闭空标签：你可以使用键盘快捷键 / +W 关闭任何打开的标签，但如果你禁用此选项，该快捷键将不再起作用。这在默认情况下是启用的。
  • 在将文件拖入IDE时打开新资源进行编辑。某些资源（如sprites 或声音）可以拖到GameMaker的主Workspace IDE ，并将被检测到并作为新资源添加到Asset 浏览器中。默认情况下，当这种情况发生时，相应的资源编辑器也会被打开进行编辑，但取消勾选这个选项将阻止编辑器被打开，你将需要手动打开它们。
  • 创建时打开新资源进行编辑。如果勾选此项，每次您在Asset 浏览器中创建一个新资源时，相应的资源编辑器将自动打开。这个选项是默认勾选的，取消勾选将意味着你必须手动打开所有新创建的资源。
  • 自动重新加载已更改的文件。如果勾选了这个选项，那么IDE 将自动扫描和检查文件，看看是否有任何文件在GameMaker之外被修改过，如果有的话，就重新加载它们。
  • 禁用IDE 的过渡动画。GameMaker IDE 有许多动画过渡，比如打开窗口等......然而它们并不符合每个人的口味，如果禁用它们，你可能会在旧的或动力不足的机器上体验到性能提升。你可以在这里禁用它们，而且它们在默认情况下是打开的。
  • 选择加入反馈调查。当这个选项被选中时，你会告诉GameMaker你希望参与反馈调查，这意味着在工作时，偶尔会显示一条关于你正在工作的区域的信息，并要求你进行评价和反馈。这个选项默认是打开的。
  • 启用DPI覆盖。这个选项可以让你用你自己的DPI设置来覆盖默认的DPI设置。你必须首先启用该选项，然后设置原始DPi覆盖值（默认值为96）或设置原始DPi的百分比（默认值为100%），一旦改变，你将被提示重新启动IDE ，这可能需要你保存当前打开的项目上正在进行的任何工作。请注意，DPi的最低百分比为50%，最高为500%，如果你把鼠标放在信息图标上，你可以看到正在使用的显示器的推荐最大设置。

    请注意，如果你超过了显示器的推荐DPi设置，你将会看到一个通知，允许你重置DPi。
  • Use US Date Format: This setting changes the format used to display dates in the IDE. When enabled, dates are displayed as e.g. "Thursday, April 27, 2023", when disabled, dates are displayed as e.g. "27 April 2023" instead. By default this preference is unchecked.

上面的选项是常规 偏好中的主要选项，但在下面列出的子类别中还有一些额外的选项供你使用。

  • 汇编

  • 归属地
  • 帮助

  • 输出窗口
  • 路径
  • 省电
  • 最近的窗口
  • 资源
  • 工作空间
