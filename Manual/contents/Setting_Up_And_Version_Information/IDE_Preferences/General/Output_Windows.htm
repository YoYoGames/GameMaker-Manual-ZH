输出窗口首选项

该 输出窗口首选项控制了IDE 中各种不同的输出窗口的不同颜色和格式化选项。每个不同的窗口都有相同的 选项的集合，这些选项列在下面。

  • 颜色：被写入窗口的文本的默认颜色。
  • 字体名称。要使用的font （可用的fonts ，取决于你系统上安装的字体）。
  • 风格。要使用的font 风格，如粗体 或斜体 （注意，不是所有的窗口都允许相同的风格，而且还取决于所使用的font ）。
  • 大小。为给定的输出窗口设置font 的尺寸。
  • 抗锯齿。在这里你可以设置输出窗口的抗锯齿模式font 渲染。你可以选择让IDE ，通过选择使用默认值来决定，或者你可以通过选择这两个选项来强制打开 或关闭。 或者通过选择这两个选项中的一个来强制打开或关闭 。
