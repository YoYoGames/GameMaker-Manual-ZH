文件观察者首选项

文件观察器是IDE 的一部分，它观察与项目有关的文件，看是否由于外部软件（例如：在外部编辑器中编辑图像）或由于内部变化（如编辑脚本）而发生任何变化。 asset 。一般来说，这些设置不需要编辑，但在某些系统上--比如那些使用固态硬盘作为项目文件的保存驱动器的系统--文件观察器可能会在你做一些事情时通知你一些变化，比如简单地保存 或者当你编辑任何项目asset ，在这种情况下你可能需要改变这些设置。

可用的设置有：。

  • 启用文件观察器。这可以用来启用或禁用文件监视系统。如果该功能被禁用， IDE 将不再接收任何由外部来源改变的文件，因此在禁用该功能时应注意 这个功能时要小心。默认情况下，文件观察器是启用的。
  • 在IDE中设置保存后MS中的文件观察者容忍度。这调整了保存文件和文件观察者检查变化之间的时间。如果你在保存你的项目时，文件观察器错误地出现说有变化，你可以调整这个值。 你可以调整这个值，如果你正在保存你的项目，而文件观察者错误地出现说有什么变化，在这种情况下，你可以增加延迟时间，直到信息不再出现。默认值是2000ms，但范围可以从1ms到10000ms。
