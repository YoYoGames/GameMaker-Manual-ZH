输入偏好

输入 首选项控制着IDE 中的鼠标和键盘输入的某些方面。可用的选项有：

  • 双击时间慢（ms）。IDE 的某些区域需要较慢的双击来正确工作，例如在重命名资源时（快速双击将打开资源编辑器，而慢速双击将允许你重命名它）。这个选项允许你设置检测速度，默认值为500ms。
  • 滚动锁定时间（ms）。这个选项涉及到GameMaker在滚动窗口重叠时应该等待的时间。基本上，当你在窗口 "A "中滚动时，光标可能会悬停在窗口 "B "中的另一个可滚动区域，所以这个选项定义了在你停止滚动 "A "后，你必须等待多长时间才能滚动 "B"，这样你就不会意外地开始滚动不同的面板。默认值是500ms。
  • 平移时将光标环绕在屏幕上。启用后，当平移时，这将使光标绕着屏幕的边界移动workspaces 。默认是启用。
  • 滚动速度：这控制了不同workspaces 内的滚动速度。默认值为1。
  • 标签拖动距离解除桌面最大限度的限制（px）。当你拖动一个workspace 标签离开它在IDE 中的默认位置时，如果当时只有一个workspace 处于活动状态，那么GameMaker窗口将取消最大化并随着鼠标移动（如果你有一个以上的workspace 处于活动状态，将创建一个新的IDE 的实例来容纳workspace ）。这个设置控制了在workspace 识别它之前你必须拖动标签的距离，默认值是50px。
  • 与窗口内容互动的最小缩放水平（%）。这个选项控制缩放的截止点，在这个截止点上，鼠标将不再能够与workspace 窗口互动（即：添加节点，拖动窗口等...）。默认值是40。
  • 鼠标滚轮缩放需要Ctrl键。当勾选这个选项时，所有的编辑器（和主workspace ）都会用 / + 鼠标滚轮 启用缩放功能，而不勾选它将只用鼠标滚轮MMB图标启用缩放。
  • Ctrl + MMB来打开继承的事件而不是手册。object 当这个被选中时，在一个 / + 鼠标中键 ，将打开objects 继承的事件（如果它有的话），而不是手册。
  • 为文本选择添加偏置，防止跳行。当这个被选中时，在任何文本窗口中选择单行时设想的行高将被填充，使其不容易出现垂直不精确的情况。这在默认情况下是启用的。
  • 在所有设备上启用笔记本模式。勾选此选项将在IDE ，使你能够根据需要switch ，无论运行GameMaker的设备是否为笔记本电脑，都可以打开或关闭笔记本电脑模式。默认情况下这是不被选中的。
  • 强制使用笔记本电脑模式。这个设置可以改变 "笔记本模式 "是否可用，并且默认为启用。你可以禁用它，以确保当你在笔记本电脑上启动IDE ，笔记本模式不会自动被启用。
