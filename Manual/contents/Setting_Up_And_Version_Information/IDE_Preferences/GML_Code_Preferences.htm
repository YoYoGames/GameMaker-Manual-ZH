GML代码首选项

GML代码首选项用于定义如何在GameMaker的IDE 中显示一些编码的语言功能。GML代码的选项如下。

  • 在状态栏中显示参数类型。这个选项与你使用你自己的自定义scripts ，并使用JSDoc符号来识别不同的参数等有关。(更多信息请看这里)。当这个选项被选中，并且你在script 中指定了参数类型，那么它们将显示在编辑器底部的状态栏中，并被括在 {} 。这在默认情况下是关闭的。
  • 在状态栏中显示参数描述。如果这个被选中，那么你用JSDoc符号给你的自定义scripts 的任何描述将显示在编辑器窗口底部的状态栏中。这在默认情况下是关闭的。
  • 为系统功能显示美式拼写。默认情况下，GameMaker使用GB英文拼写的函数，例如 draw_text_colour() ，但这可能与你的习惯不同，所以你可以使用这个选项switch ，变成美国的拼写，这样函数就变成了 draw_text_color() ，例如。这在默认情况下是关闭的。
  • 显示的最大语法错误数。这个值设定了在任何时候显示的最大语法错误数量。如果你发现IDE 是滞后的，那么你可以尝试减少这个值。默认情况下，该值被设置为50。
  • 启用变量引用警告。当启用时，这将flag 任何潜在的变量问题，比如在某处定义了一个变量，但在代码中没有实际使用。当禁用时，这些警告将不再显示。默认情况下，这个功能是启用的。
  • 启用语句折叠为区域。启用后，你的代码中的括号能够像区域一样被折叠。这在默认情况下是启用的。
