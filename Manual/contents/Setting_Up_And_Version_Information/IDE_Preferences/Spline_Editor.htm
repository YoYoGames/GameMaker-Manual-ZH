花键编辑器的偏好

花键编辑器首选项主要用于定义路径编辑器中的平滑 ，尽管它们会影响IDE的某些其他方面（如物理形状）。 主要用于定义路径编辑器中的平滑paths ，尽管它们会影响IDE的某些其他方面（如物理形状的 编辑器）。)存在以下选项。

  • 插入path 点的精度（px）。这设置了在path 上插入点的精度值（像素），即：鼠标必须在这个范围内才能插入点，而不是增加一个新的点。默认是20px。
  • 锚点宽度：设置锚点的宽度，单位为像素。默认为8px。
  • 锚点高度。设置锚点的高度，单位是像素。默认为8px。
  • 路径颜色：当你编辑时，用来显示path 的颜色。默认为(ARGB) #FFFFFFFF。
  • 突出显示的path 段落颜色：用于突出显示任何选定的path 段落的颜色。默认为（ARGB）#FFFFFFFF。
  • 无效的path 颜色：用来突出无效的path 的颜色。默认为（ARGB）#FF0000FF。
  • 锚点颜色：用于绘制path 锚点的颜色。默认为（ARGB）#FFC0C0C0。
  • 无效锚点颜色：用于绘制path 锚点的颜色。默认为（ARGB）#FF800000。
  • 选定的锚点颜色：用于绘制被高亮（选定）的path 锚点的颜色。默认为（ARGB）#FF00A4FF。
  • 选定的无效锚点颜色：用于绘制高亮但无效的锚点的颜色。默认为（ARGB）#FFA400FF。
  • 鼠标悬停的锚点颜色：用于绘制活动的锚点的颜色。默认是（ARGB）#FFFFFFFF。
  • 鼠标移过无效锚点的颜色：用于绘制激活但无效的锚点的颜色。默认为（ARGB）#FF0000FF。
  • 遮罩覆盖颜色：如果遮罩被启用，设置遮罩的覆盖颜色（见上文）。默认是（ARGB）#FF404040。
