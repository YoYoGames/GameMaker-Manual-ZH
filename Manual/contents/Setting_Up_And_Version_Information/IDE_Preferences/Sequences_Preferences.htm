序列首选项

序列首选项用于定义序列编辑器及其元素如何运作的某些属性。主要的选项有。

  • 在新创建的序列上显示画布框架。启用后，对于所有新的Sequence 编辑器的画布，将显示画布框架的智能指南sequences 。这个选项也可以根据需要从编辑器本身的画布属性中启用/禁用。默认值是启用。
  • 在新创建的序列上的画布框架宽度。在这里你可以为所有新的sequences ，设置默认的框架宽度。默认值是1366px。
  • 在新创建的序列上的画布框架高度。在这里你可以为所有新的sequences ，设置默认的框架高度。默认值是768px。
  • 在画布上点击时暂停。当这个选项被选中时，在Sequence 编辑器画布的任何地方点击将暂停sequence 的播放。这在默认情况下是启用的。
  • 重叠选择延迟，单位是毫秒。这控制了在Sequence 编辑器画布上的重叠项目时，项目工具箱需要等待多长时间才会弹出。默认值是1000ms。
  • 翻译path 点插入的余地。在Sequence 编辑器画布中，当你添加了一个asset ，然后创建了一个位置参数轨迹后，会有虚线画出，以演示path ，移动时的asset 。如果你把鼠标放在这条线附近，那么你就可以为它添加一个新的参数轨迹点，这个选项设置了鼠标离平移线的距离，在这个点的标记被显示出来之前，鼠标需要离开平移线。默认值是6px。
  • 自动创建默认参数轨道。启用后，该选项将自动为在sequence"轨迹面板 "中创建的任何新轨迹添加位置、比例、旋转和原点的参数轨迹。 默认为禁用。
  • 自动拉伸新创建的资产。该选项将拉伸所有新创建的asset 键，使之与sequence 的长度一致。 默认为禁用。
  • Dopesheet中Anim Curve轨道的高度。这个选项可以用来设置在keyframe 视图中，使用曲线的参数轨道在dopw sheet中占多少条轨道线。 默认是2，但你也可以选择1或3。
  • 新文本跟踪的默认文本。当你在string 中创建一个新的文本轨迹时，所使用的文本Sequence 。
  • 检查器选择颜色：用于突出检查器多选列表中所选择的轨道的颜色。

确认对话

确认对话框 首选项用于改变当你执行某些操作时在sequence 编辑器中显示的信息。你可以改变的信息有

  • 自动选择嵌入已编辑的曲线。如果你试图嵌入一个你已经编辑过的animation curve ，默认情况下你会看到一条信息来确认这个动作。然而，把这个选项设置为 "是"，就不会显示信息，每次都会嵌入。设置为 "否 "将阻止每次的嵌入。
  • 自动选择将曲线转换为键。如果你试图将一个animation curve 转换为keyframes ，默认情况下你会看到一条信息，询问你是否要继续。但是将这个选项设置为 "OK "就不会显示这个信息，只是不让你转换曲线。

智能指南

在这里你可以选择在Sequences 编辑器中工作时智能导引的行为。可用的选项有

  • 启用智能导引器。在这里你可以启用或禁用Sequences 编辑器画布中的智能导引。这在默认情况下是启用的。
  • 启用标尺。在这里你可以启用或禁用Sequences 编辑器画布中的智能导引。这在默认情况下是启用的。
  • 将标尺卡在边缘。assets 这个选项将使标尺 "捕捉 "到sequences 画布中的边缘（反之亦然）。默认是启用的。
  • 捕捉距离。这是一个项目需要与标尺（或标尺与项目）的距离（以像素为单位），以便它能 "捕捉 "到这个位置。默认值是6px。
