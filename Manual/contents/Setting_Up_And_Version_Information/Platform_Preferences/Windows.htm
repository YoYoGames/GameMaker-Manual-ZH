Windows首选项

Windows 偏好有以下选项。

  • Visual Studio位置。该选项用于将GameMaker指向构建PC上的微软Visual Studio 安装。
  • 默认的打包类型。当建立一个最终的Windows 可执行包时，你可以选择创建一个安装程序（使用NSIS）或一个压缩的 ZIP 文件。 文件，其中包括游戏和任何其他文件。默认情况下，你会在编译时看到一个对话框来选择这两个选项，但你可以将这个偏好设置为总是使用其中一个。注意，当这个选项被设置为 显示对话框，在编译时弹出的实际对话框窗口中，启用 "记住打包选项 "将把这个偏好设置为你选择的选项。
