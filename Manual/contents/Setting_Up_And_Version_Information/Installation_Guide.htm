安装指南

GameMaker是为在macOS 和Windows 台式电脑上使用而设计的，具有这里提到的最低规格。

您可以从YoYo Games网站上的YoYo 帐户页面下载GameMaker安装程序。安装后，第一次运行GameMaker时，可能会提示您在使用该产品之前进行更新。在安装任何更新后，您需要提供您的YoYo Games 帐户的许可证详细信息（更多信息请见这里），然后才能使用该产品，这意味着您第一次使用GameMaker时，必须有互联网连接。初次登录后，你应该能够离线使用该产品，但它会定期要求你连接以验证登录账户。

值得注意的是--根据你拥有的许可证类型--你可能需要安装额外的组件或遵循额外的步骤来设置和使用特定平台的GameMaker。关于每个平台所需的全部信息，请参见YoYo Games Helpdesk 上的这篇文章。GameMaker 所需的 SDKs。

稳定版和测试版

你从YoYo Games 网页上下载的版本应该是GameMaker当前的稳定 版本，但也可能有另一个"Beta"版本，用于展示或测试新功能，然后再进入最终的稳定版本。完整的版本细节如下。

  • 稳定的。这个版本是你用于一般开发的版本，建议与任何商业产品一起使用。它可能缺少Beta版本中的某些功能，但这是因为它们还没有经过充分的测试或需要额外的工作。
  • 测试版。GameMaker的Beta版是为那些希望在新功能被添加到稳定版之前试用这些功能的人准备的，或者为那些需要对某个问题进行修复的人准备的。我们不建议你将Beta版用于最终的游戏制作，除非它包含一个重要的错误修复，而这个错误正阻碍着你的项目。你可以从你的YoYo "下载 "页面获得GameMaker的Beta版，它可以与稳定版同时安装和运行。请注意，如果你通过Steam 运行GameMaker，那么你将无法使用Steam ，尽管你仍然可以下载和使用桌面版的产品Beta版。
