平台首选项

手册的这一部分涵盖了不同平台的特定首选项 ，在你将游戏编译到某些目标之前需要进行设置。这些通常只需要在开始时设置一次（尽管未来的运行系统更新可能需要你改变某些东西，但这些会在发行说明中提到），用来指向任何需要的SDK，并设置基本的编译选项，以便你希望GameMaker为任何特定的平台编译你的游戏项目。请注意，可用的平台将取决于你的许可证，而不是所有的平台都可用。

以下是标准许可证的所有可用目标平台的列表。

  • 安卓
  • HTML5
  • Opera GX
  • 基础设施
  • tvOS
  • macOS
  • 窗户
  • Windows UWP

请注意，除了设置这些偏好外，你还需要设置其中一些平台的目标设备，否则你可能无法编译或测试你的项目。更多信息请参见编译部分和设备管理器部分。

由于不同公司保持着不同的法律限制, 本手册没有显示各种控制台 的偏好(Xbox, PlayStation和Nintendo Switch)。如果你有控制台许可证，那么你可以从YoYo Games帮助台获得这些信息，但你必须先用你的控制台ID电子邮件联系YoYo Games ，并请求允许查看不同部分。
