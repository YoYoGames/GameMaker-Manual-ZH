Opera GX游戏选项

Opera GX游戏选项允许你修改你的游戏属性。提供了以下几个部分。

一般

这个窗口显示你的Opera GX登录状态。你可以点击 Refresh图标，用你的Opera账户登录。

然后你有以下选择。

  • 选择游戏。从这个列表中，你可以选择一个现有的游戏来更新，或者选择上传一个新的游戏。
  • 游戏名称。这是你在GX.games上的游戏名称（标题）。

    重要事项为了成功上传到GX.games，你必须在游戏名称中只使用允许的字符（默认情况下，该名称被设置为你的GameMaker 项目名称，所以确保选择一个有效的项目名称或事后手动更改游戏名称）。

    如果你上传的游戏标题有禁止的字符，你会得到这个错误。 数据不正确。检查输出

    允许的字符是。
    • a-z, A-Z（支持本地模式，如å）。
    • 0-9
    • 空白
    • : - !? ., ' & ( )
  • 版本。这是最后一次上传游戏的版本。你只能在你的游戏被上传到GX.games之前改变这个字段。
  • 下一个版本。这是你下次上传GX.games时使用的版本号。你只能在你的游戏被上传到GX.games之后才可以改变它。你不需要为每次上传手动修改，因为GameMaker 会自动为你增加版本号的最后一位数字。
  • 在GX.games上注册。点击这个按钮可以在GX.games上注册一个新条目。这只有在"选择游戏"被设置为"新游戏"时才有效。
  • 选择组别。在上传新游戏之前，你可以选择该游戏所属的工作室。
  • 在Opera上编辑游戏。这将打开GX.games DevCloud，并将你带到你的游戏的 "游戏细节 "页面。这只有在你登录并且你的游戏已经上传的情况下才可以点击。
  • 内部共享URL。只有在GX.games DevCloud上启用时，这才会打开你的游戏的私人版本。
  • 公开分享URL。只有在GX.games DevCloud上启用时，才能打开你的游戏的公开版本。

    这两个URL都可以与其他人分享，让他们玩你的游戏。

图形

在这里，你可以改变与你的游戏如何显示有关的以下细节。

  • 显示光标。控制游戏启动时鼠标光标是否可见。对那些不需要鼠标输入或使用自定义鼠标精灵的游戏很有用。
  • 在像素之间进行颜色插值。打开线性插值，这基本上是 "平滑 "像素。对于清晰的像素图形，它应该是关闭的，但如果你有漂亮的阿尔法混合和平滑的边缘图形，最好是打开。这在默认情况下是打开的。
  • 缩放。你的游戏可以被配置为自动缩放画布以保持 浏览器内的长宽比 ，或者你可以选择让它全比例运行。全比例选项不会在浏览器中全屏显示游戏，而是将绘制的内容拉伸到适合画布的尺寸，如游戏的第一个room 所定义。默认情况下，这被设置为保持长宽比。

最后是设置纹理页的大小的选项。默认（也是最兼容的）尺寸是2048x2048，但你可以从256x256到8192x8192之间的任何地方进行选择。还有一个按钮标明 预览 的按钮，它将为这个平台生成texture 页面，然后打开一个窗口，以便你可以看到它们的样子。如果你希望看到texture 页面的结构，并防止texture 页面大于（或小于）所需，这可能是非常有用的。关于texture 页面的更多信息，请看这里。
